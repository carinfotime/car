# 昊铂S600 技术参数介绍
  
更新时间：2026年07月20日15时04分40秒   

## 一、车型定位
昊铂S600是广汽埃安旗下高端品牌“昊铂”推出的一款旗舰级纯电动轿车，定位为“科技与豪华并重”的大型行政级座驾，旨在对标国际一线豪华品牌的高端新能源车型。
## 二、核心三电系统
| 项目 | 参数 |
| :--- | :--- |
| **驱动形式** | 单电机后驱 / 双电机四驱 |
| **电机类型** | 永磁同步电机 |
| **最大功率** | 后驱版：250kW（约340马力）<br>四驱版：前180kW + 后250kW（综合约580马力） |
| **最大扭矩** | 后驱版：约430N·m<br>四驱版：综合约750N·m |
| **0-100km/h加速** | 后驱版：5.9秒<br>四驱版：3.9秒 |
| **最高车速** | ≥250km/h（电子限速） |
### 电池与续航
| 项目 | 参数 |
| :--- | :--- |
| **电池类型** | NCM三元锂电池（弹匣电池2.0技术） |

| **CLTC续航里程** | <br>- **93kWh版本**: ≤720km<br>- **108kWh版本**: ≤820km (长续航版) / ≤750km (四驱性能版) |
| **快充时间** | <br>- DC快充（480kW超充桩）：10%-80% ≤15分钟<br>- DC快充（普通快充桩）：30%-80% ≤30分钟 |
| **慢充时间** | AC慢充（7kW家充桩）：0-100% ≈12-15小时 |

### 尺寸数据
| 项目 | 参数 |
| :--- | :--- |
| **长 × 宽 × 高 (mm)** | ≥5050 × ≥1910 × ≥1460 mm (具体以官方最终发布为准) |

### 底盘悬挂
-   **前悬挂**: 双叉臂式独立悬挂
-   **后悬挂**: H臂多连杆式独立悬挂
-   **特殊配置**: 
    -   全系标配CDC可变阻尼减震器
    -   高配车型可选装或标配智能空气悬架系统
    -   支持底盘升降及软硬调节
## 四、智能座舱与科技配置
### 🖥️屏幕与交互
-   **仪表盘**: ≥12.3英寸全液晶仪表盘
-   **中控屏**: ≥14.6英寸2.5K OLED悬浮式中控屏
-   **副驾屏**: ≥12.3英寸娱乐屏（高配）
-   **HUD抬头显示**: AR-HUD增强现实抬头显示系统，投射距离≥7米，画面尺寸≥60英寸
-   **车机芯片**:高通骁龙8295旗舰芯片
-   **操作系统**: ADiGO智驾互联生态系统5.0
### 🎵音响与舒适性
-   **音响系统**: 
    -   标配：23扬声器高级音响系统
    -   选装/高配：支持杜比全景声的顶级音响品牌（如Dirac联合调校）
-   **座椅功能**:
    -   前后排座椅均支持电动调节、加热、通风、按摩功能
    -   后排右侧配备“老板位”，支持一键躺平模式及腿托伸展
-   **空调系统**: 
    -   分区自动空调带PM2.5过滤及负离子发生器
    -   香氛系统（多主题可选）

# 探境者01 技术参数介绍
  
更新时间：2026年07月21日09时09分15秒   

## 一、车型概览
**探境者01** 是一款专为全地形探索与长途穿越打造的中大型智能硬派越野SUV。它融合了前沿的电动化技术与强悍的机械素质，旨在为用户提供“无界探索，从容驾驭”的极致体验。
## 二、动力系统
| 项目 | 参数 |
| :--- | :--- |
| **动力类型** | 增程式混合动力（EREV） |
| **增程器** | 1.5T 四缸涡轮增压高效发动机（热效率达44%） |
| **驱动电机** | 前后双电机智能四驱 |
| **系统综合功率** | 450 kW |
| **系统综合扭矩** | 950 N·m |
| **0-100 km/h加速** | 4.8秒（满电状态） |
## 三、电池与续航
| 项目 | 参数 |
| :--- | :--- |
| **电池类型** | 宁德时代麒麟电池（三元锂） |
| **电池容量** | 60 kWh |
| **CLTC纯电续航** | 280 km |
| **CLTC综合续航** | >1200 km（满油满电） |
| **快充时间** | SOC 20%-80% ≤25分钟（直流快充） |
## 四、越野性能
| 项目 | 参数 /配置 |
| :--- | :--- |
| **驱动形式** | 智能电四驱（带扭矩矢量控制） |
| **悬架系统** | 前：双叉臂独立悬架 /后：多连杆独立悬架 + CDC可变阻尼减震器 +空气悬架（支持150mm调节范围）|
| **接近角/离去角/纵向通过角** |36°/32°/26° （标准模式）/40°/38°/30° （越野模式，升至最高）|
| **最大涉水深度** |900 mm （空气悬架最高位时）|

| **越野模式选择**|<ul><li>岩石/泥地/沙地/雪地/涉水/专家模式</li></ul>|

L2+级高阶智能辅助驾驶 (支持高速NOA导航辅助驾驶及城市记忆领航)

-笼式车身结构 (高强度钢铝混合)
-热成型钢占比: ≥35%
-整车扭转刚度: >40000 N·m/deg (显著提升越野抗扭能力)
-碰撞标准:满足C-NCAP五星及C-IASI优秀(G)评级要求

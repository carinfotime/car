# 🚗 兰博基尼 Urus 儿童安全注意事项
  
更新时间：2026年07月16日08时48分08秒   

## ⚠️ 重要安全提醒
### 1. 儿童座椅使用
- **必须使用**符合国家标准的儿童安全座椅
- 12岁以下儿童**严禁**坐在副驾驶位置
- 建议将儿童座椅安装在后排中间位置
- 根据儿童体重和身高选择合适组别的安全座椅：
  - 0-13kg（约0-15个月）：反向安装婴儿提篮
  - 9-18kg（约9个月-4岁）：正向安装儿童座椅
  - 15-36kg（约4-12岁）：增高垫+安全带
### 2. 乘车行为规范
- **全程系好安全带**，不可随意解开
- **禁止**将头、手伸出窗外或天窗
- **禁止**在车内打闹、蹦跳或爬行
- **禁止**触碰中控台、挡杆、方向盘等驾驶部件
- **禁止**在行驶中开关车门
### 3. Urus 特别注意事项
| ⚡项目 | 📋说明 |
|---------|--------|
| **强劲动力** | Urus V8双涡轮增压引擎动力极强，加速迅猛，请平稳驾驶 |
| **排气声浪** | Sport模式下声浪较大，可能惊吓到婴幼儿，建议使用Strada模式 |
| **自动启停** | 注意启停系统可能让儿童感到不适 |
| **车窗防夹** | Urus配备防夹功能，但仍需看管好儿童手指 |
| **后备箱** | **严禁**儿童在后备箱内玩耍或躲藏 |
### 4. 停车后安全须知
1. 🔒 **熄火拔钥匙/带走遥控钥匙**
2. 🚪 **确认所有车门已锁好**
3. 🌡️ **切勿将儿童单独留在车内**
   - 夏季车内温度可在10分钟内升至60°C以上
   -冬季车内也会迅速降温至危险水平
4. 👀 **检查后排是否有遗漏物品或人员**
### 🚫绝对禁止事项
> ❌ **绝对不要将儿童单独留在Urus车内，哪怕只有一分钟！**
> ❌ **绝对不要让儿童在车辆周围玩耍（盲区危险）**
> ❌ **绝对不要让未满12岁儿童坐副驾驶**
> ❌ **绝对不要在倒车时让儿童站在车后方**
---
### 💡温馨提示
Urus作为高性能SUV，车身宽大、盲区较多。上车前请绕车一周检查是否有蹲在车旁的儿童。行驶过程中保持平稳驾驶，避免急加速和急刹车。祝您和家人出行平安！🚀

# 睿蓝9 内饰参数介绍
  
更新时间：2026年07月14日17时17分37秒   

## 整体设计风格
睿蓝9内饰采用简约科技风设计，整体布局以驾驶者为中心，注重实用性与舒适性的结合。
---
## 座椅配置
| 项目 | 参数 |
|------|------|
| **座椅材质** | 仿皮/真皮（高配车型） |
| **座椅布局** | 2+2+2（六座）/ 2+3+2（七座）可选 |
| **前排座椅功能** | 主驾电动调节（6向）、副驾电动调节（4向）、座椅加热、通风（高配） |
| **第二排座椅** | 独立座椅（六座版），支持前后滑动及靠背角度调节 |
| **第三排座椅** | 可折叠放平，扩展后备箱空间 |
---
## 仪表盘与中控屏
| 项目 | 参数 |
|------|------|
| **仪表盘** | 12.3英寸全液晶仪表盘 |
| **中控屏** | 12.3英寸悬浮式触控液晶屏 |
| **屏幕分辨率** | 1920×720 |
| **车机系统** | GKUI智能生态系统 |
| **语音控制** | 支持语音唤醒、导航、音乐、空调等控制 |
---
## 方向盘
- **材质：** 皮质包裹
- **调节方式：** 手动上下+前后调节
- **多功能按键：** 集成音量控制、蓝牙电话、定速巡航/自适应巡航控制键
---
## 空调系统
- **类型：** 自动双区恒温空调
- **后排独立出风口：** ✅（第二排、第三排均有出风口）
- **PM2.5过滤：** ✅
- **负离子净化器：** ✅（高配车型）
---
## 内饰颜色选择
- **深色内饰：**  深灰/黑色系
- **浅色内饰：**  米色/棕色系（部分车型可选）
---
##  储物空间
-  中央扶手箱储物格
-  前排杯架 ×2
-  门板储物槽（四门）
-  副驾手套箱
-  第二排中央扶手带杯架
-  第三排两侧杯架/储物槽
---

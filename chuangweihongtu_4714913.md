# 创维鸿途 内饰参数介绍

## 一、座舱布局
| 项目 | 参数 |
|------|------|
| 座椅布局 | 2+2+3 七座布局 |
| 车身尺寸（长×宽×高） | 5170mm × 1990mm × 1780mm |
| 轴距 | 3100mm |
| 座位材质 | Nappa真皮包裹 |
## 二、座椅功能
### 前排座椅
- **主驾座椅**：12向电动调节（含4向腰托）
- **副驾座椅**：8向电动调节
- **加热/通风**：前排座椅支持加热与通风功能
- **记忆功能**：主驾座椅带位置记忆
### 第二排（独立航空座椅）
- **调节方式**：电动前后滑动 + 靠背角度调节
- **腿托**：电动伸缩腿托，最大展开角度70°
- **加热/通风/按摩**：全系标配三档加热、通风及气动按摩
- **一键躺平**：支持一键进入半躺模式（最大仰角160°）
### 第三排
- **坐垫高度**：350mm，保证腿部支撑
- **靠背角度**：可手动调节（25°~35°）
- **空间表现**：腿部空间≥850mm，头部空间≥950mm
## 三、中控与仪表
| 部件 | 规格 |
|------|------|
| 全液晶仪表盘 | 12.3英寸，分辨率1920×720 |
| HUD抬头显示 | AR-HUD，投射面积≥45英寸，显示导航/车速/ADAS信息 |
| 中控触控屏 | **15.6英寸悬浮式高清屏**，分辨率2560×1440 |
| OS系统 | SkyLink OS，支持OTA远程升级 |
| AI语音助手 | “小维”智能语音，支持连续对话与四音区定位 |
## 四、后排娱乐系统
- **后排吸顶屏**：12.8英寸高清触控屏（支持独立播放）
- **后排控制面板**：中央扶手集成5.5英寸触控屏，可控制空调、座椅、遮阳帘等
- **手机互联**：支持无线CarPlay / HiCar / Carlink
## 五、空调系统
- **类型**：双区自动恒温空调（前排）+独立后排空调控制
- **空气净化**：
    - PM2.5过滤效率 ≥99%
    - AQS空气质量监测系统
    - CN95级空调滤芯
    - UVC深紫外线杀菌（可选）
    - AQS负离子发生器
    - AQS负离子发生器（可选）

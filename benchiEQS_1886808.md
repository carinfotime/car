# 奔驰EQS 驾驶注意事项
  
更新时间：2026年07月13日08时05分26秒   

## 一、启动与行驶前检查
1.  **钥匙与启动**：确保智能钥匙在车内。踩下制动踏板，按下启动按钮即可唤醒车辆。仪表盘亮起后，挂入D挡即可行驶。
2.  **电量检查**：出发前确认高压电池电量充足，并根据行程规划好充电站点。
3.  **轮胎状态**：检查胎压是否正常（建议参考驾驶员侧门框上的标签值），并观察轮胎磨损情况。
4.  **充电口盖**：确保充电口盖已完全关闭并锁止，否则车辆可能无法正常行驶或发出警告。
## 二、驾驶模式与能量回收
1.  **驾驶模式选择**：
    -   **舒适模式**：适合日常通勤，悬架柔软，动力输出平顺。
    -   **运动模式**：悬架变硬，动力响应更迅猛，适合高速或需要操控时使用。
    -   **经济模式**：限制空调功率和动力输出，最大化续航里程。
    -   **个性化模式**：可根据个人喜好自定义转向、悬架和动力参数。
2.  **能量回收（D Auto / D+ / D / D-）**：
    -   **D Auto**（推荐）：系统根据导航和雷达自动调节回收强度，实现最高效的“单踏板”体验。
    -   **D+ （滑行）**：几乎无回收阻力，适合高速巡航。
    -   **D （标准）**：轻微回收，类似燃油车松油门滑行感。
    -   **D- （强回收）**：强力回收，可实现单踏板操作（需适应）。注意在湿滑路面慎用强回收以防甩尾。
## 三、充电注意事项
1.  **充电协议**：
    -   **交流慢充（AC）**：适合家庭或办公场所过夜充电。务必使用符合国标且质量可靠的充电桩。
    -   **直流快充（DC）**：EQS支持最高200kW快充。建议在电池电量较低时（如10%-20%）开始快充以节省时间；电量超过80%后充电速度会显著下降。
2.  **电池保护**：
    -   避免将电池完全耗尽至0%，长期低电量会损害电池寿命。
    -   日常使用建议将充电上限设为80%-90%（除非长途需要100%）。
3.  **环境温度影响**：
    -   低温环境会降低续航和充电速度。冬季建议出发前通过App或车内设置预热电池（若车辆支持）。
4.  **安全提醒**：
    -   雷雨天气尽量避免户外充电。
    -   充电过程中如发现异常发热、异味或冒烟，立即停止充电并远离车辆。
## 四、智能驾驶辅助系统使用须知
1.  **自适应巡航控制与车道保持辅助系统**
    这些系统是辅助功能而非自动驾驶。驾驶员必须时刻手握方向盘并观察路况。在弯道过急、施工区域或恶劣天气下应手动接管控制权。
2.  **主动式变道辅助**
    仅在后视镜确认安全且系统提示可操作时启用。变道过程中仍需观察盲区。
3.  **自动泊车**
    虽然EQS具备高度自动化泊车能力（包括遥控泊车），但在地下车库光线不足或有障碍物时需谨慎使用。首次使用时建议在空旷场地熟悉其逻辑。
## 五、特殊路况与环境应对
1.  **涉水路段**
    尽管电动车没有进气口且电池密封性好，但仍不建议涉水深度超过车轮一半高度。涉水后应轻踩刹车检查制动效果。
2.  **长下坡路段**
    利用强能量回收模式代替刹车片制动，既节能又减少刹车磨损。
3.  **冰雪/湿滑路面**
    起步时轻踩加速踏板避免打滑；关闭强能量回收以减少失控风险；注意ESP电子稳定程序默认开启切勿手动关闭除非被困需要脱困。
4.  ***高温暴晒***
    夏季长时间暴晒后车内温度极高请先远程开启空调降温再上车；中控大屏过热可能暂时降频或亮度降低属正常保护机制。
## ⚠️六、紧急情况处理
1.  若仪表盘出现红色高压警告灯立即停车熄火远离车辆并联系道路救援切勿自行触摸高压部件线束颜色为橙色代表高压线束请勿触碰！
2.  发生碰撞后即使能正常行驶也应尽快前往授权服务中心检查底盘及高压系统完整性因为内部损伤可能延迟显现隐患极大！
3.  紧急断电方法如果无法正常熄火可长按启动按钮约10秒强制切断高压电！
## 📋七、日常维护小贴士
-   *定期更新MBUX系统*以获得最新功能优化地图数据以及电池管理算法
-   *保持12V小电瓶健康*如果长期停放不使用建议每隔两周连接交流充电桩维护小电瓶否则可能导致无法解锁车门
-   *空调滤芯更换*电动车没有发动机所以空调滤芯承担更多空气净化任务建议每一年或一万公里更换一次

# 长安CS55PLUS 驾驶注意事项
  
更新时间：2026年07月12日10时52分04秒   

## 一、启动与预热
1. **启动前检查**：确认挡位处于 P 挡或 N 挡，踩下制动踏板后再按一键启动按钮。
2. **冷车预热**：冬季或长时间停放后，建议怠速热车 30-60 秒，待转速回落至正常怠速（约 800rpm）后再起步。
3. **避免急加速**：刚启动时发动机机油尚未充分润滑，前几分钟内避免大脚油门。
## 二、行驶中注意事项
### 变速箱使用（7速双离合/6AT）
- **D挡行驶**：正常路况使用 D 挡，变速箱会自动换挡。
- **手动模式**：长下坡或超车时可切换至手动模式，利用发动机制动减少刹车磨损。
- **等红灯**：短时间等待（30秒内）可 D 挡踩刹车；长时间等待建议挂 N 挡拉手刹，减少离合器磨损。
- **禁止空挡滑行**：双离合变速箱严禁 N 挡滑行，会损坏变速箱。
### 涡轮增压发动机
- **涡轮冷却**：长途高速行驶后，建议怠速运转 1-2 分钟再熄火，让涡轮充分冷却（若配备电子水泵延时冷却则无需刻意等待）。
- **机油品质**：务必使用厂家推荐标号的全合成机油（通常为 SN/SP级5W-30）。
### ESP车身稳定系统
- **默认开启**：ESP 默认开启，日常驾驶请勿关闭。
- **脱困时关闭**：在泥地、沙地或雪地陷车时，可短时关闭 ESP 以帮助脱困。
## 三、特殊路况驾驶
| 路况 | 操作建议 |
|------|----------|
| **雨天/湿滑路面** | 降低车速，避免急刹急转；开启雾灯或近光灯 |
| **山路/长下坡** | 使用手动模式低挡位控制车速，避免长时间踩刹车导致刹车热衰减 |
| **高速公路** | ACC自适应巡航保持安全跟车距离（建议3秒法则），注意车道保持辅助提示 |
| **拥堵路段** | IACC集成式自适应巡航可辅助跟车启停；注意防止加塞 |
## 四、智能驾驶辅助系统使用
> CS55PLUS搭载L2级智能驾驶辅助系统（视具体配置而定）
1. **ACC自适应巡航**
   - SET键设定速度后车辆自动跟车
   - +/-键调节车速（每次±5km/h）
   - CANCEL取消后需重新设定
   - ⚠️ ACC不能完全替代人工驾驶，弯道过急或雨雾天气可能失效
2. **车道偏离预警**
   - >60km/h自动激活
   - LDW仅报警不干预转向
   - LKA会轻微修正方向盘（需双手握持）
3. **360°全景影像**
   - R挡自动激活
   - D挡低速接近障碍物时自动弹出
   - ⚠️摄像头有盲区且易脏污遮挡
## 五、停车与熄火
1. **正确停车顺序**
   
   踩刹车 → N挡 →拉手刹 →松脚刹(释放应力) → P挡 →熄火
   
   直接挂P挡熄火可能导致下次挂挡困难。
2. **坡道停车**
   先拉手刹再挂P挡，防止变速箱锁止机构受力过大。
3. **熄火前检查**
   关闭空调、大灯、雨刮等用电设备后再熄火。
## ⚠️六、重要安全提醒
1. A柱较宽大转弯时注意左右观察盲区。
2. SUV重心较高高速过弯注意侧倾。
3. D柱视野受限倒车时多依赖360影像和后视镜结合判断。
4. TJA交通拥堵辅助在车道线模糊路段可能退出需及时接管。
5. U盘音乐支持MP3/WMA格式视频仅P挡可播放画面。

# 小老虎 内饰参数介绍
  
更新时间：2026年07月17日11时13分37秒   

## 一、座舱布局与设计理念
*   **设计风格**：环抱式运动座舱，强调驾驶者中心，融合科技感与年轻活力元素。
*   **内饰色调**：提供 **酷黑动感** 与 **黑红双拼** 两种主题，满足不同审美需求。
## 二、座椅系统
*   **座椅材质**：高级仿皮与织物混搭（部分高配车型为全仿皮包裹）。
*   **前排座椅**：
    *   主驾驶座椅：6向手动调节（前后/靠背/高低）。
    *   副驾驶座椅：4向手动调节（前后/靠背）。
    *   功能：前排座椅加热（选装/高配标配）。
*   **后排座椅**：
    *   比例放倒：支持 **4/6 比例放倒**，扩展后备箱空间。
    *   中央扶手：带杯架的后排中央扶手（高配车型）。
    *   头枕：三个独立高度可调头枕。
## 三、方向盘与仪表盘
*   **方向盘**：
    *   类型：三辐式多功能运动方向盘。
    *   材质：真皮包裹（高配）/ 发泡材质（低配）。
    *   调节：上下两向手动调节。
    *   控制：集成定速巡航、多媒体控制、语音唤醒按键。
*   **仪表盘**：
    *   类型：**12.3 英寸全液晶仪表盘**（高配）/ 机械指针+3.5英寸行车电脑显示屏（低配）。
    *   显示内容：车速、转速、油耗、胎压监测、导航信息投射（高配）。
## 四、中控台与信息娱乐系统
*   **中控屏**：
    *   尺寸：**10.25 英寸悬浮式触控液晶屏**。
    *   分辨率：1920 x 720。
    *   系统：小老虎智能互联系统。
*   **功能配置**：
    *   手机互联：支持 **Apple CarPlay** 与 **百度 CarLife**。
    *   语音控制：自然语音识别，可控制导航、电话、空调、多媒体。
    *   车联网：4G 网络、在线音乐、在线导航、OTA 远程升级。
    *   蓝牙：蓝牙电话与音乐播放。
    *   USB接口：前排 **2个USB接口**（Type-A + Type-C），后排 **1个USB接口**。
## 五、空调系统
*   **类型**：电动空调 / 自动恒温空调（高配）。
*   **控制方式**：物理按键+旋钮（低配）/ 触控面板+物理快捷键（高配）。
*   **功能**：
    *   花粉过滤系统。
    *   后排出风口（全系标配）。
    *   空气净化系统（高配，PM2.5过滤）。
## 六、储物空间
*   **前排**：
    *   门板储物格：可容纳 2 瓶 500ml 矿泉水。
    *   中央扶手箱：深度约 15cm，支持 USB 走线口。
    *   手套箱：带阻尼，容量约 8L。
    *   杯架：中控台前方 2 个杯架。
    *   手机储物格：中控台下方镂空储物空间（带无线充电板，高配）。
*   **后排**：
    *   门板储物格：可容纳 1 瓶 500ml 矿泉水。
    *   前排座椅背袋：地图袋。
    *   中央扶手杯架：2 个杯架。
## 七、座椅舒适性参数
| 项目 | 参数 |
| :--- | :--- |
| 前排座椅坐垫长度 | 490 mm |
| 前排座椅宽度 | 520 mm |
| 后排座椅坐垫长度 | 450 mm |
| 后排座椅宽度 | 1350 mm |
| 后排腿部空间（正常坐姿） | 约 870 mm |
| 后排头部空间（正常坐姿） | 约 950 mm |
## 八、内饰灯光
*   **阅读灯**：前后排独立 LED 阅读灯。
*   **氛围灯**：**64 色可调氛围灯**（高配），支持随音乐律动。
*   **迎宾灯**：前排车门下方迎宾灯（高配）。
## 九、其他细节配置
*   **遮阳板**：主副驾带化妆镜（副驾带照明灯，高配）。
*   **眼镜盒**：驾驶位头顶眼镜盒。
*   **后排中央头枕**：高度可调。
*   **内后视镜**：手动防眩目 / 自动防眩目（高配）。
*   **车窗一键升降**：主驾车窗一键升降带防夹（全系）/ 四门一键升降（高配）。

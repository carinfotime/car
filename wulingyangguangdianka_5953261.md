# 五菱扬光电卡 内饰参数介绍
  
更新时间：2026年07月16日08时13分19秒   

## 一、内饰整体风格
五菱扬光电卡内饰采用**实用至上**的设计理念，整体布局简洁明了，以功能性为主导，注重驾驶操作便利性与日常使用耐久性。
## 二、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 织物面料（部分高配版本可选配仿皮材质） |
| 座椅布局 | 2座（单排座舱） |
| 座椅调节 | 主驾座椅支持前后滑动 + 靠背角度调节 |
| 座椅填充 | 高密度发泡海绵，支撑性良好 |
| 头枕 | 一体式或可调式头枕（视具体批次而定） |
## 三、仪表台与中控
| 项目 | 参数 |
|------|------|
| 仪表盘 | 机械指针式仪表 + 单色液晶行车信息显示屏 |
| 显示内容 | 车速、电量、续航里程、档位、故障提示等 |
| 中控台 | 简约式面板，空调控制区域集中布局 |
| 空调系统 | 手动空调（冷暖风），旋钮式控制 |
| 储物空间 | 副驾前方开放式储物槽、车门内侧储物格、中控台下储物区 |
| USB接口 | 前排配备 1 个 USB 充电接口 |
## 四、方向盘
| 项目 | 参数 |
|------|------|
| 方向盘材质 | 普通塑料（部分版本可选装方向盘皮革包裹） |
| 方向盘调节 | 不支持调节（固定式） |
| 多功能按键 | 基础版无，高配版可选装（如音量控制、蓝牙接听） |
## 五、娱乐与音响
| 项目 | 参数 |
|------|------|
| 音响系统 | 2扬声器（左右车门各一） |
| 娱乐配置 | 基础收音机 + AUX音频输入（部分版本可选装蓝牙模块或简易多媒体屏） |
| 屏幕（选装） | 可选装 7 英寸触控屏，支持蓝牙音乐、USB播放、倒车影像显示 |
## 六、空间尺寸
| 项目 | 参数 |
|------|------|
| 车内宽度 | 约 1400 mm |
| 座垫长度 | 约 480 mm |
| 头部空间（坐姿） | 约 950 mm |
| 腿部空间 | 约 650 mm（以正常驾驶坐姿计） |
| 后备箱/货厢 | 后方为开放式货厢，货厢内部尺寸约为 2000×1200×300 mm（视具体版本略有差异） |
## 七、安全与辅助配置
| 项目 | 参数 |
|------|------|
| 安全带 | 前排三点式预紧限力安全带 |
| 安全气囊 | 主驾安全气囊（部分版本配备） |
| 倒车辅助 | 选装倒车雷达 / 倒车影像 |
| 儿童锁 | 配备后排儿童安全锁 |
| 车内灯光 | 顶棚阅读灯（手动开关） |
## 八、其他细节
| 项目 | 参数 |
|------|------|
| 内门把手 | 机械式内扣手 |
| 车窗控制 | 手动摇窗（部分版本可选装电动车窗） |
| 遮阳板 | 主驾侧配备遮阳板（带化妆镜为选装） |
| 地毯 | 耐磨PVC材质地毯 |
| 内饰颜色 | 深灰色/黑色为主（耐脏实用） |
---
> **总结：** 五菱扬光电卡的内饰以经济实用为核心，配置以满足基本驾驶需求为主，没有过多豪华装饰，但在储物便利性、操作直观性和日常耐用性方面表现良好，适合物流配送、个体经营等商用场景。

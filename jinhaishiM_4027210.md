# 金海狮M 驾驶注意事项
  
更新时间：2026年07月12日09时46分40秒   

## 一、启动与预热
1. **启动前检查**：绕车一周，检查轮胎气压及外观、车身周围是否有障碍物、灯光是否正常。
2. **钥匙启动**：将钥匙拧到“ON”档，等待仪表盘自检灯熄灭（约2-3秒），再启动发动机。
3. **冷车预热**：冬季或长时间停放后，建议怠速热车 **30秒至1分钟**，待转速回落至正常怠速（约700-800转）后再起步。切勿长时间原地热车或猛踩油门。
## 二、行驶中注意事项
### 1. 挡位与换挡
- **手动挡车型**：
  - 换挡时机建议：1挡起步后，车速达15km/h左右换2挡；25km/h换3挡；40km/h换4挡；60km/h以上换5挡。
  - 避免低挡高速或高挡低速行驶，以免增加油耗和磨损。
- **离合器操作**：踩离合器要踩到底，挂挡轻柔，避免打齿。
### 2. 速度控制
- **经济时速**：金海狮M的经济时速通常在 **60-90km/h**，此时燃油经济性最佳。
- **高速行驶**：请勿超过车辆标定的最高安全速度（通常为120km/h），且需注意横风影响。
### 3. 刹车技巧
- **预判刹车**：提前松油门，利用发动机制动辅助减速。
- **避免急刹**：该车型重心相对较高（MPV/轻客类型），急刹易导致车身晃动或货物移位。
- **ABS系统**：配备ABS的车型在紧急刹车时感觉踏板弹脚是正常现象，请继续踩住不松。
### 4. 转弯与变道
- **盲区较大**：A柱较粗且车身较长，转弯和变道时务必多观察后视镜并转头确认侧方盲区。
- **控制车速过弯**：入弯前提前减速至安全速度（建议30km/h以下），避免高速过弯导致侧倾。
## 三、载货与载人
1. **切勿超载**：严格遵守车辆核定载质量或载人数。超载会严重影响制动性能和悬挂寿命。
2. **货物固定**：若装载货物，务必使用绳索或网兜固定好货物，防止行驶中滑动撞击乘客或车门。
3. **重心变化**：
   - 满载时刹车距离会明显变长，需增大跟车距离。
   - 装载不均匀会导致车辆跑偏或侧倾加剧。
## 四、特殊路况驾驶
| 路况 |  注意事项 |
|------|-----------|
| **雨天/湿滑路面** |  降低车速（建议比干燥路面慢20%），增大跟车距离至少4秒以上。 |
| |  避免猛打方向或急加速/急刹车。 |
| |  开启近光灯及前后雾灯以提高辨识度。 |
| |  若发生轻微侧滑，松开油门并向侧滑方向轻打方向修正。 |
| |  涉水深度不要超过排气管口（约30cm），涉水后轻踩刹车排水。 |
| |  若车辆被困雪地/泥地，可关闭ESP（如有）尝试脱困。 |
## 五、停车与熄火
1. **停车步骤**
   - （手动挡）踩离合 → （自动挡）挂N/P挡 →拉紧手刹 →熄火 →松开踏板 →拔钥匙。
2. **坡道停车**
   - （手动挡）挂入1档或倒档（上坡挂1档/下坡挂倒档）+拉紧手刹+轮胎下垫石块/砖头以防溜车。
3. **熄火前**
   - （涡轮增压车型）长途高速后应怠速运转1分钟再熄火以保护涡轮。（自然吸气车型可直接熄火）
## ⚠️特别提醒
> ✅定期检查机油尺液位及冷却液液位是日常维护的关键。  
> ✅轮胎气压建议保持在标准值范围（参考驾驶座门框上的标签）。  
> ❌不要在发动机运转时离开车辆且不拉手刹。  
> ❌不要单手操控方向盘或在弯道中空档滑行。

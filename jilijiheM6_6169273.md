# 吉利几何M6 驾驶注意事项
  
更新时间：2026年07月13日18时30分00秒   

## 一、启动与驾驶前检查
1.  **钥匙与启动**：确保智能钥匙在车内，踩下制动踏板，按下“启动/停止”键即可上电启动。若仪表盘显示“未检测到钥匙”，请将钥匙靠近中央扶手箱的感应区域。
2.  **充电状态**：出发前确认续航里程是否满足当日行程需求。若电量低于20%，建议提前规划充电。
3.  **轮胎检查**：检查胎压是否正常（建议冷车胎压2.5 Bar左右），观察轮胎表面有无破损或鼓包。
4.  **仪表盘警示灯**：上电后观察仪表盘，确认无红色或黄色故障灯常亮（如动力电池故障、电机故障、制动系统故障等）。
## 二、行驶中的操作规范
1.  **驾驶模式选择**：
    -   **经济模式**：适合城市拥堵路况，动力输出平缓，能量回收最强，可最大化续航。
    -   **舒适模式**：适合日常综合路况，动力响应适中，驾乘体验最均衡。
    -   **运动模式**：适合超车或需要强劲动力时，电门响应灵敏，但能耗较高。
2.  **能量回收（动能回收）**：
    -   可通过中控屏或换挡拨片（如有配备）调节回收强度（L1/L2/L3）。
    -   L3模式下松电门减速感明显，类似单踏板逻辑；L1模式下滑行更顺畅。建议根据个人习惯和副驾/后排乘客的舒适度调整。
3.  **加速与减速**：
    -   电动车扭矩释放快，起步时轻踩电门，避免急加速导致轮胎打滑或电量快速消耗。
    -   预判路况，提前松电门利用动能回收减速，减少刹车片磨损并回收电能。
4.  **高速行驶**：
    -   时速超过100km/h后电量消耗会显著增加。如需长途高速行驶，建议预留更多电量余量。
    -   M6重心较低且电池位于底盘，车身稳定性较好，但雨天仍需注意积水路段。
## 三、充电注意事项
1.  **充电接口**：
    -   **快充口（直流）**：位于车辆左前翼子板处（慢充口通常也在同侧或右前翼子板），用于高速公路服务区或快充站。
    -   **慢充口（交流）**：用于家用充电桩或公共交流桩。
2.  **充电安全**：
    -   **雷雨天气禁止充电**：避免在露天雷雨环境下进行充电操作。
    -   **检查接口干燥**：插入充电枪前确保充电口内无积水、无异物。若遇雨雪天插拔枪时注意遮挡防溅水。
3.  **电池保养建议**：
    -   **最佳电量区间**：日常使用建议保持电量在20%-80%之间。长期停放时建议保持50%-60%电量。
    -   **避免过放/过充**：尽量不要将电量耗尽至0%再充；充满后若长时间不开车也应及时拔枪（部分车型有预约充电功能）。
4.  **冬季低温充电提示**：
    -   冬季气温低时电池活性下降。若车辆支持电池预加热功能（通常在导航至快充站时会自动开启），可提前预热以缩短快充时间。
## 四、智能辅助驾驶系统使用须知
*本车配备L2级辅助驾驶功能（如ICC智能领航、ACC自适应巡航等），请注意以下事项：*
1.  **驾驶员始终是责任主体**：辅助驾驶仅为辅助功能，双手不得长时间脱离方向盘。系统可能无法识别静止障碍物、锥桶、行人横穿等复杂场景。
2.  **恶劣天气禁用/慎用**：大雨、大雾、大雪或强光逆光条件下可能导致摄像头/雷达失效或误判。
3.  **车道保持介入感强吗？**
    几何M6的车道居中保持在弯道较大时可能需人工干预修正；当系统发出接管提示音且仪表盘闪烁时请立即接管方向盘。
## 五、特殊场景应对
|场景|操作建议|
|---|---|
|涉水路段|电动车涉水能力优于燃油车但仍有风险。水深超过轮毂一半高度时请绕行；通过后轻踩刹车排出水分并检查底盘是否有异响|
|长下坡路段|利用强动能回收模式减少刹车热衰减风险；无需挂N挡滑行|
|极端高温暴晒后|先开窗通风散热再开启空调；避免立即使用快充以免电池温度过高限制功率|
## ⚠️六、核心安全警告
-   ❌严禁私自改装电路或加装非原厂电器设备
-   ❌严禁在车内放置易燃易爆品
-   ❌严禁用高压水枪直接冲洗机舱内部高压部件
-   ✅定期回授权服务站进行三电系统检测

# 阿斯顿·马丁 DBX 技术参数介绍
  
更新时间：2026年07月14日11时45分14秒   

## 一、车型概述
阿斯顿·马丁 DBX 是品牌首款豪华高性能 SUV，于2019年正式发布，标志着这家英国超豪华汽车制造商正式进入 SUV 细分市场。DBX 在保留阿斯顿·马丁经典 GT 跑车基因的同时，兼具全地形通过能力与日常实用性。
---
## 二、动力系统
### DBX V8（标准版）
| 项目 | 参数 |
|------|------|
| **发动机类型** | 4.0L V8 双涡轮增压（梅赛德斯-AMG M177） |
| **排量** | 3,982 cc |
| **最大功率** | 550 hp（405 kW）@ 6,500 rpm |
| **最大扭矩** | 700 N·m @ 2,200–5,000 rpm |
| **变速箱** | 9速自动变速箱（带液力变矩器） |
| **驱动方式** | 全时四驱（主动式中央差速器 + e-Differential后桥电控差速锁） |
| **0–100 km/h加速** | 4.5秒 |
| **最高时速** | 291 km/h |
### DBX707（高性能版）
| 项目 | 参数 |

| **最高时速** | **310 km/h** |

## 三、底盘与悬架
### 悬架系统
- **前悬架**:  双叉臂独立悬架
- **后悬架**:  多连杆独立悬架
-  标配自适应三腔空气弹簧 + CDC连续可变阻尼减震器
-  车身高度可调范围：+45 mm（越野模式）/ –50 mm（运动模式）
-  48V主动防侧倾系统（eARC），有效抑制弯道侧倾
### 转向系统
-  电动助力转向，可变转向比
-  最小转弯直径：12.1 m
### 制动系统
-  前制动：410 mm x38 mm通风碳陶瓷刹车盘 +六活塞卡钳
-  后制动：390 mm x32 mm通风碳陶瓷刹车盘 +单活塞浮动卡钳
- *DBX707升级为碳陶瓷制动系统*

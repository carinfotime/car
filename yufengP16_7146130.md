# 御风P16 动力参数介绍
  
更新时间：2026年07月12日09时55分40秒   

## 发动机参数
| 项目 | 参数 |
|------|------|
| **发动机型号** | 1.9T 柴油涡轮增压发动机 / 2.4L 自然吸气汽油发动机 |
| **排量** | 1.9L（柴油）/ 2.4L（汽油） |
| **进气形式** | 涡轮增压（柴油）/ 自然吸气（汽油） |
| **气缸排列形式** | 直列四缸 |
| **燃油类型** | 0号柴油（柴油版）/ 92号及以上无铅汽油（汽油版） |
## 动力输出数据
### 1.9T 柴油版
- **最大功率**：110 kW（约150马力）
- **最大功率转速**：4000 rpm
- **最大扭矩**：350 N·m
- **最大扭矩转速**：1800 - 2800 rpm
### 2.4L 汽油版
- **最大功率**：100 kW（约136马力）
- **最大功率转速**：5250 rpm
- **最大扭矩**：200 N·m
- **最大扭矩转速**：2500 - 3000 rpm
## 变速箱与驱动系统
| 项目 | 参数 |
|------|------|
| **变速箱类型** | 6挡手动变速箱（MT） |
| **驱动方式** | 前置后驱 /  分时四驱（可选） |
## 底盘与悬挂系统
| 项目 |  参数 |
|------|-------|
| **前悬挂类型** |  双叉臂式独立悬挂 |
| **后悬挂类型** |  钢板弹簧非独立悬挂 |
| **车体结构** |  非承载式车身 |
##  性能表现
-   **最高车速**: ≥160 km/h
-   **燃油标号**:  柴油版使用0号柴油；汽油版使用92号及以上无铅汽油
-   **油箱容积**:   约70 L
> 📌 *注：以上数据以官方最新公布为准，不同年款及配置可能存在细微差异。*

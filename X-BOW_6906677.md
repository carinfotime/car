# KTM X-BOW 技术参数介绍
  
更新时间：2026年07月15日09时43分17秒   

## 车型概览
KTM X-BOW 是奥地利摩托车制造商 KTM 与德国赛车工程公司 Dallara 合作开发的极致轻量化开放式座舱跑车，定位为赛道日用车及公路合法超级跑车。
---
## 动力系统
| 参数 | 规格 |
|------|------|
| **发动机类型** | 2.0L TFSI 直列四缸涡轮增压（奥迪技术） |
| **排量** | 1,984 cc |
| **最大功率** | X-BOW GT：285 hp / 6,400 rpm |
| | X-BOW GT-XR：493 hp / 6,400 rpm |
| **最大扭矩** | GT：420 N·m / 3,200 rpm |
| | GT-XR：581 N·m / 5,500 rpm |
| **燃油供给** | 缸内直喷 + MPI（双喷射系统） |
| **涡轮增压器** | BorgWarner EFR（GT-XR） |
---
## 传动系统
- **变速箱**：6速手动变速箱（GT）/ 7速双离合DSG（GT-XR）
- **驱动方式**：中置后驱
- **离合器**：赛车级多片式离合器
- **限滑差速器**：机械式限滑差速器
---
## 底盘与车身结构
- **车架形式**：碳纤维单体壳（monocoque）
- **车身材料**：碳纤维增强复合材料（CFRP）
- **整备质量**：
  - X-BOW GT：约847 kg
  - X-BOW GT-XR：约1,130 kg
- **轴距**：2,430 mm
- **长度/宽度/高度**：
  - GT：3,738 mm / 1,915 mm / 1,202 mm
- **离地间隙**：约120 mm
---
## 悬挂系统
- **前悬挂**：双叉臂独立悬挂，推杆式减震器
- **后悬挂**：双叉臂独立悬挂，推杆式减震器
- **减震器品牌/类型**：
   - GT: KW Motorsport可调避震
   - GT-XR: TTX系列赛车避震（选装）

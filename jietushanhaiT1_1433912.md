# 捷途山海T1 动力参数介绍
  
更新时间：2026年07月13日16时35分15秒   

## 动力系统概述
捷途山海T1搭载了一套 **插电式混合动力系统（PHEV）**，结合了燃油发动机与电动机的优势，兼顾强劲动力输出与出色燃油经济性。
---
## 核心动力参数
| 项目 | 参数 |
|------|------|
| **发动机类型** | 1.5T 涡轮增压四缸发动机 |
| **发动机排量** | 1.5L（1498ml） |
| **发动机最大功率** | **115 kW（156 Ps）** |
| **发动机最大扭矩** | **230 N·m** |
| **驱动电机类型** | 永磁同步电机 |
| **电机最大功率** | **150 kW（204 Ps）** |
| **电机最大扭矩** | **310 N·m** |
| **系统综合功率** | **≥240 kW（约326 Ps）** |
| **系统综合扭矩** | **≥500 N·m** |
---
## 传动系统
- **变速箱类型**: 3挡DHT混合动力专用变速箱
- **驱动方式**: 前置前驱 / 电四驱（高配版）

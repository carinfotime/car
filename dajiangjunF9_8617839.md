## 三、各系统保养详解
### 1. 发动机系统
- **机油选择**：推荐使用 **5W-30 或 10W-40**（柴油版推荐 CI-4 或以上级别）
- **机油容量**：约 5.5~6.0 升（视具体发动机型号）
- **更换步骤**：
  1. 热车后熄火，静置5分钟
  2. 拧开放油螺栓，排放旧机油
  3. 更换机油滤清器
  4. 安装放油螺栓（更换垫片）
  5. 加注新机油至标准刻度
  6. 启动发动机运行2分钟，熄火检查液位
### 2. 变速箱系统
- **手动变速箱油**：GL-4 75W-90
- **自动变速箱油**：ATF 6速专用油（建议使用原厂油品）
- **检查方法**：着车后挂入各档位，检查油位是否在热区标记内
### 3. 制动系统
- **刹车片磨损极限**：摩擦材料厚度＜3mm 必须更换
- **刹车盘**：表面沟槽深度＞1.5mm 或跳动量超标需更换
- **刹车油更换**：含水量＞3% 必须更换
### 4. 四驱系统（四驱版）
- **前/后桥油**：GL-5 80W-90
- **分动箱油**：专用分动箱油（建议原厂）
- **注意**：涉水或重度越野后需提前检查
### 5. 轮胎系统
- **标准胎压**（参考值）：
  - 前轮：2.3~2.5 bar（空载）
  - 后轮：2.5~2.8 bar（空载），2.8~3.2 bar（满载）
- **轮胎磨损标记**：到达磨损指示线立即更换
- **换位方式**：交叉换位法（前驱车型）

## 四、季节性保养重点
### 春季
- 检查空调系统，清洁或更换空调滤芯
- 检查雨刮器及喷水嘴
- 清理底盘泥沙及腐蚀物
### 夏季
- 检查冷却系统（水箱、风扇、水泵）
- 检查空调制冷效果
- 注意发动机高温报警
- 检查轮胎胎压（夏季适当降低0.1~0.2 bar）
### 秋季
- 检查蓄电池性能及接线柱
- 检查暖风系统
- 检查防冻液冰点（建议降至 -35℃ 以下）
- 检查灯光系统（雾天增多）
### 冬季
- 更换冬季玻璃水（冰点 -25℃ 以下）
- 检查轮胎花纹深度（＞4mm 为安全）
- 柴油版注意更换负号柴油（根据当地气温选择 -10#、-20#、-35#）
- 检查预热系统（柴油版）
- 检查手刹及制动性能（冰雪路面）

## 五、常见故障与预防
| 问题 | 可能原因 | 预防措施 |
|-----|---------|---------|
| 启动困难（柴油版） | 预热系统故障 / 柴油标号不合适 | 定期检查预热塞，冬季加注合适标号柴油 |
| 发动机抖动 | 油路堵塞 / 喷油嘴积碳 | 定期清洗喷油嘴，使用正规燃油 |
| 刹车异响 | 刹车片磨损 / 异物卡入 | 定期检查清洁制动系统 |
| 变速箱顿挫 | 变速箱油老化 / 油量不足 | 按时更换变速箱油 |
| 四驱挂不上 | 分动箱电机故障 / 油压不足 | 定期使用四驱功能，避免长期不用 |
| 尿素系统报警（柴油版） | 尿素液位低 / 喷嘴堵塞 | 使用正规尿素液，定期清理喷嘴 |

## 六、重要提醒
1. **首保**：新车行驶 **3,000~5,000公里** 或 **3~6个月**（先到为准）务必进行首次保养，免费更换机油机滤。
2. **质保要求**：建议在授权服务站进行保养，保留保养记录，避免影响质保权益。
3. **油品选择**：务必使用符合厂家规格的正品油液，劣质油液会加速机件磨损。
4. **柴油版特别提示**：
   - 务必加注正规加油站柴油
   - 定期添加柴油添加剂（除水、润滑）
   - 注意DPF（颗粒捕集器）再生提示，必要时进行主动再生
5. **皮卡载货**：切勿超载，超载会严重损害悬挂、刹车及传动系统。
6. **涉水后**：检查前后桥及变速箱是否进水，及时更换油液。

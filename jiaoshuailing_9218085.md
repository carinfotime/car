# 吉奥帅凌 保养指南
  
更新时间：2026年07月17日14时11分44秒   

## 一、保养周期总览
| 保养项目 | 周期 |
|---------|------|
| 首次保养 | 购车后 **3个月** 或 **行驶5000公里**（以先到为准） |
| 常规保养 | 每 **6个月** 或 **每行驶10000公里**（以先到为准） |
| 大保养 | 每 **2年** 或 **行驶40000公里**（以先到为准） |
---
## 二、定期保养项目明细
### 1. 每5000公里 / 3个月（首次保养）
- [ ] 更换发动机机油及机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查全车灯光及信号系统
- [ ] 检查雨刮器及清洗液
- [ ] 紧固底盘及悬挂螺栓
- [ ] 检查蓄电池状态及接线柱清洁度
### 2. 每10000公里 / 6个月（常规保养）
- [ ] 更换发动机机油及机油滤清器
- [ ] 更换空气滤清器滤芯
- [ ] 检查并调整火花塞间隙
- [ ] 检查制动片厚度（前/后）
- [ ] 检查制动盘磨损情况
- [ ] 检查传动轴及万向节
- [ ] 检查底盘各橡胶衬套
- [ ] 检查排气管及三元催化器
- [ ] 润滑车门铰链及锁扣
### 3. 每20000公里 / 12个月
- [ ] 以上常规保养全部项目
- [ ] 更换空调滤清器滤芯
- [ ] 检查并清洁节气门
- [ ] 检查喷油嘴工作状态
- [ ] 检查正时皮带/链条状态
- [ ] 四轮换位及动平衡
- [ ] 检查并调整前束及四轮定位
### 4. 每40000公里 / 2年（大保养）
- [ ] 以上所有保养项目
- [ ] 更换制动液
- [ ] 更换变速箱油（手动/自动）
- [ ] 更换冷却液
- [ ] 更换火花塞
- [ ] 更换燃油滤清器
- [ ] 检查并清洁EGR阀
- [ ] 检查发电机及起动机
- [ ] 检查空调压缩机及制冷剂压力
- [ ] 更换正时皮带（如为皮带式）
- [ ] 清洗发动机积碳
### 5. 每60000公里 / 3年
- [ ] 完成大保养全部项目
- [ ] 更换轮胎（视磨损情况）
- [ ] 更换蓄电池（视状态）
- [ ] 检查并更换刹车盘（视磨损情况）
- [ ] 检查悬挂减震器是否漏油
- [ ] 检查转向拉杆球头间隙

## 三、日常检查项目（建议每周/每次用车前）
| 检查项目 | 要求 |
|---------|------|
| 机油液位 | 冷车状态下，油位在上下刻度之间 |
| 冷却液液位 | 在MAX与MIN之间 |
| 制动液液位 | 在MAX与MIN之间 |
| 轮胎气压 | 前轮2.3-2.5 bar，后轮2.5-2.8 bar |
| 轮胎外观 | 无鼓包、裂纹、异常磨损 |
| 灯光系统 | 大灯、转向灯、刹车灯、雾灯正常 |
| 仪表盘警示灯 | 无异常报警灯亮起 |
| 雨刮器 | 刮拭清晰，无异响 |
| 玻璃清洗液 | 液位充足 |

## 四、油液规格及加注量
| 油液类型 | 推荐规格 | 参考加注量 |
|---------|---------|-----------|
| 发动机机油 | SAE 5W-30 / 5W-40（API SN级以上） | 约4.0-4.5升 |
| 变速箱油（手动） | GL-4 75W-90 | 约2.0-2.5升 |
| 变速箱油（自动） | ATF Dexron VI | 约6.0-7.0升 |
| 冷却液 | 乙二醇基长效冷却液 | 约6.0-7.0升 |
| 制动液 | DOT4 | 约1.0升 |
| 转向助力油 | ATF Dexron III | 约1.0升 |
| 空调制冷剂 | R134a | 约500-600g |
> **注意：** 具体加注量请以随车用户手册为准，不同年款及动力配置可能存在差异。

## 五、保养注意事项
### 🔧 保养前准备
1. 提前预约正规售后服务站或具备资质的维修厂
2. 携带车辆行驶证及保养手册
3. 记录近期用车过程中发现的异常情况（异响、抖动、油耗变化等）
### ⚠️ 重要提醒
- **质保期内** 建议在授权服务站进行保养，以免影响质保权益
- 使用 **原厂或同等级别** 的配件及油液
- 每次保养后索要 **保养单据** 并存档
- 如车辆长期停放（超过1个月），建议每周启动运行15-20分钟
- 恶劣路况（多尘、多雨、频繁短途行驶）需 **缩短保养周期**
- 冬季气温低于-20℃时，建议使用 **低温专用机油**
### 🛑 需立即进厂检查的异常情况
- 发动机故障灯亮起
- 水温过高报警
- 制动异常（刹车变软、异响、跑偏）
- 转向沉重或异响
- 底盘有金属撞击声
- 仪表盘出现红色报警灯
- 车辆行驶中抖动严重
- 排气管冒蓝烟或黑烟

## 七、季节性保养建议
### 🌸 春季
- 检查空调系统，清洗或更换空调滤芯
- 检查雨刮器胶条，必要时更换
- 检查并补充玻璃清洗液
### ☀️ 夏季
- 检查冷却系统，确保散热风扇正常工作
- 检查空调制冷效果
- 适当降低轮胎气压（约0.1 bar）防止高温爆胎
- 检查蓄电池电解液液位
### 🍂 秋季
- 检查暖风系统
- 检查除霜/除雾功能
- 检查轮胎花纹深度，为冬季做准备
- 检查防冻液冰点
### ❄️ 冬季
- 更换冬季专用玻璃清洗液（防冻型）
- 检查蓄电池启动电流
- 检查防冻液冰点（建议低于当地最低气温10℃）
- 检查轮胎胎压，可适当调高0.1-0.2 bar
- 冰雪地区建议更换雪地胎

## 车内空间
- **座位布局**：2+2 四座布局
- **前排空间**：头部空间充裕，腿部空间表现良好，适合身高175cm以下驾乘者
- **后排空间**：可容纳两名成年人短途乘坐，更适合儿童或身材较小的乘客
- **后排座椅**：支持整体放倒，提升储物灵活性
## 储物能力
### 后备箱容积
- **常规状态**：约 **50L**（可放置少量背包、购物袋等）
- **后排座椅放倒后**：扩展至约 **400L**，可容纳行李箱、折叠婴儿车等大件物品
### 车内储物细节
- **车门储物格**：前后门板均设有储物槽，可放置水瓶、杂物
- **中控区域**：杯架、手机槽、开放式储物格
- **副驾前方**：开放式储物平台，方便随手放置包包或小件物品
- **座椅后方**：设有储物袋（部分配置）
## 乘坐体验
- **座椅材质**：织物/仿皮（视配置而定）
- **前排座椅调节**：手动前后 + 靠背角度调节
- **后排进出便利性**：前排座椅支持一键前移折叠，方便后排上下车
## 适用场景总结
QQ冰淇淋作为一款微型纯电动车，空间设计以城市短途通勤和日常代步为核心：
| ✅适合场景 | ❌不适合场景 |
|------------|--------------|
| - 单人/双人城市通勤 | - 长途满载出行 |
| - 接送孩子上学 | - ≥4名成年人同时乘坐 |
| - 买菜、购物代步 | - 需要大量行李空间的旅行 |
| - “最后一公里”接驳 | - |
> 💡 *提示：QQ冰淇淋的空间优势在于小巧灵活、停车方便；若对载物或后排乘坐有较高要求，建议实际到店体验后再做决定。*

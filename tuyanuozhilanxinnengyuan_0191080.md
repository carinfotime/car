# 图雅诺智蓝新能源 - 空间参数介绍
  
更新时间：2026年07月19日18时25分18秒   

图雅诺智蓝新能源作为一款纯电动轻客，其核心优势之一在于宽敞、灵活且高效的车内空间设计。无论是城市物流配送还是商务通勤，其空间布局都能满足多样化的需求。
## 一、整车尺寸与轴距
-   **车身长度：** 5495mm / 5990mm (长轴版)
-   **车身宽度：** 2000mm
-   **车身高度：** 2445mm / 2700mm (高顶版)
-   **轴距：** 3350mm
> **解读：** 长轴距设计不仅提升了行驶稳定性，更为车厢内部提供了充裕的纵向空间。高顶版本则保证了人员在车内可以直立行走，装卸货物更加便捷。
## 二、货箱/载货空间
该车型主要面向物流用途，其后部货箱设计方正，空间利用率极高。
-   **货箱内部尺寸 (长 x 宽 x 高)：**
    -   标准版：约 **2580mm x 1780mm x 1600mm**
    -   长轴高顶版：约 **3400mm x 1780mm x 1900mm**
-   **货箱容积：**
    -   标准版：约 **7.5 m³**
    -   长轴高顶版：约 **10.5 m³** - **12.2 m³** (根据具体配置不同略有差异)
> **解读：** 方正的内厢设计使得空间利用率接近100%，可轻松容纳标准托盘货物。10m³以上的容积在同级纯电轻客中处于领先水平。
## 三、乘员舱与座椅布局
除了货运版本，图雅诺智蓝新能源也提供客货两用或客运版本。
-   **座位数选择：**
    -   **3座（厢式货车）：** 仅保留前排主副驾座椅，后方为全封闭货仓。
    -   **6座/7座（客货两用）：** “2+2+2”或“3+2+2”布局，后排座椅可折叠侧挂以扩展储物空间。
    -   **9座（商务通勤）：** “3+3+3”布局，提供更多载客能力。
-   **座椅特点：**
    -   前排座椅支持多向调节。
    -   后排座椅头枕高度可调。
    -   侧挂式座椅在折叠后可将车厢中部通道宽度扩展至近1米。
## 四、内部通过性与细节
-   **车内净高 (高顶版)：** ≥1900mm
-   **侧滑门开口宽度：** ≥700mm
-   **尾门开口高度 (对开门)：** ≥1300mm
-   **尾门开口宽度 (对开门)：** ≥1500mm
-   **离地间隙 (空载)：** ≥160mm
> **解读：**
> *   ****站立自由****: >1.9米的车内高度让大多数成年人无需弯腰作业。
> *   ****装卸便利****: >70cm的侧滑门开口和宽大的对开尾门设计，配合低地板门槛（离地间隙适中），使用叉车或手动搬运均十分方便。
> *   ****尾门形式****:通常采用180°对开式尾门，便于靠墙装卸；部分车型可选装上掀式尾门。

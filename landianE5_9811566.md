# 蓝电E5 用户操作指南
  
更新时间：2026年07月17日08时42分02秒   

## 一、车辆启动与熄火
### 1.1 启动车辆
1. **携带钥匙**：确保智能钥匙在车内。
2. **踩下制动踏板**：用力踩下制动踏板。
3. **一键启动**：按下位于方向盘右侧的“START/STOP”按钮。
4. **系统自检**：仪表盘指示灯亮起，系统完成自检后，READY指示灯常亮，表示车辆已进入可行驶状态。
### 1.2 熄火车辆
1. **停稳车辆**：确保车辆完全停止，档位挂入P档。
2. **按下熄火按钮**：再次按下“START/STOP”按钮，发动机/电机停止工作。
3. **断电下车**：确认仪表盘熄灭后，解开安全带，携带钥匙下车并锁车。
> **注意**：若未挂入P档直接熄火，系统会发出提示音并自动挂入P档。
## 二、驾驶模式选择
蓝电E5提供多种驾驶模式以适应不同路况和能耗需求。通过中控台或中控屏幕的驾驶模式旋钮/按键进行切换。
| 模式 | 适用场景 | 特点 |
| :--- | :--- | :--- |
| **EV（纯电）** | 城市通勤、短途行驶 | 仅由电机驱动，零油耗、安静平顺。电量充足时优先使用。 |
| **HEV（混动）** | 长途出行、综合路况 | 发动机与电机协同工作，系统自动优化能耗与动力平衡。 |
| **运动（SPORT）** | 超车、爬坡、追求动力 | 动力响应更积极，油门更灵敏，能耗相对较高。 |
| **经济（ECO）** | 节能驾驶、电量较低时 | 限制空调功率和动力输出，延长续航里程。 |
## 三、档位操作
采用电子换挡机构（旋钮式或怀挡式），请根据您车型的实际配置操作。
-   **P档（驻车档）**：停车时使用。按下P键或旋钮旋转至P位置。
-   **R档（倒车档）**：倒车时使用。需在车辆完全停止后切换。
-   **N档（空挡）**：短暂停留或洗车时使用。
-   **D档（前进挡）**：正常行驶时使用。
> *换挡时必须踩住制动踏板。*
## 四、充电操作
### 4.1 交流慢充（家用充电桩/随车充）
1.  车辆熄火并解锁。
2.  打开位于车身侧面的充电口盖板。
3.  取下充电枪插入充电口，确认锁止到位。
4.  充电桩端开始充电后，仪表盘会显示充电状态和预计时间。
5.  充满后解锁车辆并拔下充电枪。
### 4.2 直流快充（公共快充桩）
1.  同样先熄火解锁。
2.  打开直流快充口盖板（通常与慢充口分开或共用）。
3.  将快充枪插入对应接口。
4.  按照充电桩屏幕提示启动充电（扫码/刷卡）。
5.  电量达到目标值后停止充电并拔枪。
> -   建议在电量剩余20%-30%左右进行充电以保护电池寿命。
> -   长期停放建议保持电量在50%-80%之间。
## 五、智能互联与多媒体
### HUAWEI HiCar / CarLink
1.  确保手机蓝牙已开启并与车辆配对成功。
2.  首次连接需通过中控屏上的“HiCar”或“CarLink”应用扫码连接。
3.  连接成功后可在中控屏上使用导航、音乐、电话等应用。
### OTA升级
-   当系统推送新版本时，中控屏会弹出更新提示。
-   建议在停车状态下连接Wi-Fi进行下载和安装更新包以节省流量并确保安全。
## 六、常见功能操作
### L2级辅助驾驶
-   开启方式：
    1.  按下方向盘左侧的巡航按键图标进入准备状态；
    2.
    按“SET/-”键设定当前速度为巡航速度；
    3.
    可通过“RES/+”加速，“SET/-”减速；
    4.
    按CANCEL取消巡航；再次按下主开关彻底关闭功能；
-   车道保持辅助默认开启；可通过方向盘左侧按键调整跟车距离等级；
### A/C空调控制面板
-   温度调节：
    -   实体旋钮旋转调节主驾温度；
    -   副驾可通过触摸面板滑动调节；
-   风量调节：
    -   点击风扇图标增大风量；长按可快速调至最大风量；
-   除雾功能：
    -   点击前风挡除雾图标快速清除雾气；
## ⚠️七、安全注意事项
1.
儿童安全锁：
后排车门侧面设有儿童安全锁拨杆；拨至锁定位置后无法从车内开门；
2.
紧急呼叫SOS：
位于顶灯控制区域附近；紧急情况下按压即可联系救援中心；
3.
低速行人警示音EVAS：
纯电模式下低速行驶时会自动发出模拟声音提醒行人注意安全；此为法规要求不可关闭；
4.
高压安全警告：
严禁私自拆卸维修高压部件如橙色线束电池包等；发生碰撞事故后请立即远离车辆并联系专业人员处理漏液冒烟等情况；

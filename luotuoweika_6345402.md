# 骆驼微卡驾驶注意事项
  
更新时间：2026年07月20日11时05分20秒   

## 一、出车前检查
1.  **轮胎检查**：检查轮胎气压是否正常，胎面有无破损、鼓包或异常磨损。确保备胎状态良好。
2.  **油液检查**：
    -   **机油**：拔出机油尺，检查油位是否在上下刻度线之间。
    -   **冷却液**：查看膨胀水箱液位是否在标准范围内。
    -   **制动液**：检查制动液储液罐液位是否正常。
    -   **玻璃水**：确保玻璃水充足。
3.  **灯光与信号**：检查大灯（远近光）、转向灯、刹车灯、雾灯及危险报警闪光灯是否正常工作。
4.  **车身与货箱**：检查货箱门锁扣是否扣紧，栏板是否牢固。确认货物装载均匀、固定牢靠，无超高、超长、超载情况。
5.  **底盘与传动**：目视检查底盘有无漏油、漏水现象，传动轴、半轴螺栓有无松动。
6.  **随车工具**：确认千斤顶、扳手、三角警示牌等随车工具齐全。
## 二、启动与起步
1.  **钥匙门操作**：插入钥匙后，先拧至“ON”档（通电自检），等待仪表盘上的故障指示灯（如发动机故障灯、电瓶灯等）熄灭后再启动发动机。切勿直接拧至“START”档强行启动。
2.  **预热（柴油版）**：若为柴油动力车型，冬季气温较低时，需等待预热指示灯熄灭后再启动。
3.  **起步操作**：
    -   系好安全带（主副驾均需佩戴）。
    -   踩下离合器踏板，挂入一档或倒挡（手动挡车型）。
    -   松开手刹，缓慢抬起离合器同时轻踩油门，平稳起步。避免猛抬离合导致熄火或闯动。
## 三、行驶中注意事项
1.  **控制车速**：
    -   严格遵守道路限速规定。微卡空载时重心较高，满载时惯性大，转弯或湿滑路面应提前减速。
    -   **经济时速**：通常保持在60-80km/h区间较为省油且安全。
2.  **保持车距**：
    -   由于车辆自重较大且载货后制动距离会显著增加，应与前车保持比普通轿车更长的安全距离（至少4秒以上反应时间）。
3.  **转弯与变道**：
    -   **提前减速打灯**：转弯前必须提前减速至安全速度（建议30km/h以下），并开启转向灯。
    -   **注意内轮差**：微卡车身较长，转弯时后轮轨迹会向内偏移（内轮差），需注意避让右侧的非机动车和行人。
4.  **坡道行驶**：
    -   **上坡**：根据坡度提前降档（如从4档降至3档或2档），保持发动机转速在扭矩输出区间。避免高档位硬撑导致拖挡熄火或爬坡无力。
    -   **下坡**：严禁空挡滑行！应挂入低速档利用发动机制动辅助减速。长时间下坡需注意刹车鼓/盘温度，防止刹车热衰退失效。必要时停车冷却刹车系统。
5.  **夜间行车**：
    -    会车时提前切换近光灯。注意观察路面坑洼及障碍物。
## 四、特殊路况驾驶
1.  **雨天/湿滑路面**
        ：降低车速至干燥路面的70%左右；避免急刹车和急转向；开启雾灯或近光灯提高辨识度；涉水深度不得超过排气管口高度；涉水后应轻踩刹车数次以清除刹车片水分恢复制动力。
2.  **山路**
        ：连续弯道多鸣笛示警；靠右行驶不占道；使用低档位上下坡；留意长下坡路段设置的紧急避险车道指示牌。
3.  **高速公路**
        ：严格按照分道行驶规定走慢车道或货车专用道；最高时速不超过100km/h（具体视当地法规）；连续驾驶不超过4小时必须休息20分钟以上防疲劳。
## ⚠️五、严禁事项
-   ❌严禁酒后驾车及疲劳驾驶
-   ❌严禁超载超限运输
-   ❌严禁客货混装
-   ❌严禁空挡滑行
-   ❌严禁在未停稳时挂倒挡
-   ❌严禁单手操作方向盘接打电话
## ⚙️六、停车与收车
1.     选择平坦坚实地面停放；拉紧手刹并挂入一档或倒挡防止溜车；
2.
     熄火前怠速运转30秒左右让涡轮增压器降温再关闭发动机；
3.
     锁好车门车窗并拔下钥匙；
4.
     冬季若气温极低建议加注对应标号防冻液以防冻裂缸体。

# 创维D11 操作指南
  
更新时间：2026年07月19日16时48分04秒   

## 一、车辆基本参数
| 项目 | 参数 |
|------|------|
| 车型定位 | 纯电动中型SUV |
| 驱动方式 | 前置前驱 |
| 续航里程 | 约520km（NEDC） |
| 电池类型 | 三元锂电池 |
| 充电方式 | 快充/慢充 |
---
## 二、启动与熄火
### 启动车辆
1. 携带智能钥匙进入车辆
2. 踩下制动踏板
3. 按下 **启动/停止按钮**（位于方向盘右侧中控台）
4. 仪表盘亮起，车辆进入 **READY** 状态，即可挂挡行驶
### 熄火车辆
1. 将挡位置于 **P挡**
2. 按下 **启动/停止按钮**
3. 确认仪表盘熄灭后，方可下车锁门
---
## 三、挡位操作
| 挡位 | 操作方式 | 说明 |
|------|----------|------|
| **P（驻车挡）** | 按下挡杆顶部按钮 | 停车时使用 |
| **R（倒车挡）** | 按住挡杆侧面按钮，向前推 | 倒车时使用 |
| **N（空挡）** | 短促向前或向后推 | 临时停车或洗车时使用 |
| **D（前进挡）** | 向后拉 | 正常行驶 |
| **B（能量回收挡）** | D挡状态下再向后拉一次 | 增强能量回收力度 |
> 提示：挂挡时必须踩住制动踏板。
---
## 四、驾驶模式选择
通过中控屏或挡杆附近的 **驾驶模式旋钮** 切换：
| 模式 | 适用场景 |
|------|----------|
| **经济模式（ECO）** | 城市拥堵、追求续航 |
| **舒适模式（COMFORT）** | 日常通勤，平衡动力与能耗 |
| **运动模式（SPORT）** | 需要强劲动力、超车或高速行驶 |
| **雪地模式** | 雨雪湿滑路面 |
---
## 五、方向盘按键功能
### 左侧按键区
- **音量+ / 音量-**：调节多媒体音量
- **上一曲 / 下一曲**：切换歌曲
- **语音唤醒**：唤醒智能语音助手
- **接听电话**：蓝牙电话接听
### 右侧按键区
- **定速巡航开启/关闭**
- **SET/-**：设定当前速度为巡航速度 / 减速
- **RES/+**：恢复巡航 / 加速
- **取消巡航**：退出定速巡航
- **菜单切换**：切换仪表盘显示信息
---
## 六、中控屏操作
### 主界面功能
- **导航**：高德/百度地图导航
- **音乐**：在线音乐、USB音乐、蓝牙音乐
- **车辆设置**：灯光、门锁、驾驶辅助等
- **空调控制**：温度、风量、模式调节
- **应用商店**：下载第三方应用
### 快捷操作
- 从屏幕顶部向下滑动：打开快捷设置面板
- 从屏幕底部向上滑动：返回主界面
- 长按桌面图标：调整图标位置
---
## 七、空调系统
### 操作方式
1. 中控屏点击 **空调图标** 进入控制界面
2. 或使用方向盘左侧 **实体空调按键** 快速调节
### 功能说明
- **自动模式（AUTO）**：自动调节温度与风量
- **A/C**：制冷开关（电动车A/C由电池供电）
- **内外循环**：市区拥堵用内循环，高速用外循环
- **前/后除霜**：雾天或冬季使用
- **座椅加热**：冬季舒适功能
---
## 八、充电指南
### 快充（直流快充桩）
1. 打开车辆 **快充口**（通常位于车辆右后侧）
2. 将快充枪插入充电口
3. 扫码启动充电
4. 充电至80%约需 **30-40分钟**
5. 充电完成后拔枪，关闭充电口盖
### 慢充（家用交流桩/随车充）
1. 打开车辆 **慢充口**（通常位于车辆左前侧）
2. 连接充电枪
3. 充满时间约 **8-10小时**
4. 建议设置 **预约充电**，利用谷电时段
### 充电注意事项
- 建议电量保持在 **20%-80%** 之间
- 长期停放时保持电量在 **50%左右**
- 冬季充电前建议先预热电池（可通过APP操作）

## 九、智能互联功能
### SkyLink APP
1. 下载 **SkyLink** 手机APP并注册
2. 绑定车辆（扫描车内二维码）
3. 功能包括：
   - 远程开关空调
   - 查看车辆状态（电量、续航）
   - 远程解锁/锁车
   - 寻车功能
   - 预约充电
### 语音控制
- 唤醒词：**"你好，小维"** 或按方向盘语音键
- 常用指令：
  - "导航到XXX"
  - "打开空调，温度26度"
  - "播放音乐"
  - "打开车窗"
  - "今天天气怎么样"

## 十、安全与辅助驾驶
### 主动安全
- **自动紧急制动（AEB）**：检测前方碰撞风险，自动刹车
- **车道偏离预警**：车辆偏离车道时发出警报
- **盲区监测**：侧后方有车辆时后视镜指示灯亮起
### 辅助驾驶
- **定速巡航**：设定固定速度行驶
- **自适应巡航（ACC）**：自动跟车，保持安全距离
- **360°全景影像**：倒车或低速时自动显示，辅助泊车
### 泊车辅助
- 挂入R挡自动激活全景影像
- 可通过中控屏切换不同视角
- 前后雷达实时监测障碍物距离

## 十二、应急处理
### 车辆无法启动
1. 检查智能钥匙是否在车内
2. 检查制动踏板是否踩到底
3. 尝试用钥匙贴近启动按钮感应区
4. 联系创维客服或道路救援
### 行驶中电量耗尽
1. 打开双闪警示灯
2. 将车辆停至安全区域
3. 拨打救援电话或联系保险公司
4. 切勿推车行驶
### 轮胎漏气
1. 缓慢减速，靠边停车
2. 打开双闪，放置三角警示牌
3. 使用随车补胎工具或呼叫救援
---
## 十三、重要提示
1. 首次使用前请仔细阅读随车 **用户手册**
2. 定期通过APP检查车辆 **OTA升级** 提示
3. 长期停放超过15天，建议断开12V小电瓶负极
4. 洗车时避免高压水枪直接冲洗充电口
5. 儿童安全锁位于后车门侧面，防止儿童误开车门
6. 如遇到任何问题，可拨打创维汽车客服热线或联系当地授权服务中心

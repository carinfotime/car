# 别克君越 动力参数介绍
  
更新时间：2026年07月13日08时15分21秒   

## 动力总成概述
别克君越作为品牌旗舰级轿车，提供多种动力组合，兼顾平顺性、经济性与性能表现。
---
## 2.0T 涡轮增压发动机（主力版本）
| 参数项 | 数值 |
|--------|------|
| 发动机类型 | 2.0L 直列四缸涡轮增压 |
| 排量 | 1998 mL |
| 最大功率 | **174 kW（237 Ps）** |
| 最大功率转速 | 5000 rpm |
| 最大扭矩 | **350 N·m** |
| 最大扭矩转速 | 1500-4000 rpm |
| 燃油标号 | 92# 及以上无铅汽油 |
| 供油方式 | 缸内直喷 |
| 排放标准 | 国VI b |
### 变速箱
- **9速HYDRA-MATIC智能变速箱**
- 齿比绵密，换挡平顺
- 支持手动模式及换挡拨片
### 性能表现
- 0-100 km/h 加速时间：**约7.2秒**
- 最高车速：**235 km/h**
- WLTC综合油耗：**约7.0-7.5 L/100km**

## 48V轻混系统（部分版本搭载）
- **BSG电机功率**：约 10 kW
- **主要作用**：
  - 优化启停平顺性
  - 辅助起步加速
  - 降低油耗与排放
  - 提升低速行驶质感
---
## 底盘与驱动系统
| 项目 | 规格 |
|------|------|
| 驱动方式 | 前置前驱 |
| 前悬架 | 麦弗逊式独立悬架 |
| 后悬架 | 多连杆式独立悬架 |
| 转向系统 | EPS电子助力转向 |
| 制动系统 | 前通风盘 / 后实心盘 |
---
## 驾驶模式
君越提供多种驾驶模式选择，适应不同驾驶需求：
- **舒适模式**：平顺省油，适合日常通勤
- **运动模式**：动力响应更积极，换挡时机延后
- **手动模式**：通过换挡拨片自主控制挡位
---
## 一句话总结
> **2.0T + 9AT** 是君越的黄金动力组合，237马力、350N·m的输出带来从容的加速体验，配合优秀的NVH表现，兼顾了**动力性**与**舒适性**，在同级中属于中上水准。

# 仰望U9 动力参数介绍
  
更新时间：2026年07月17日08时24分37秒   

## 一、动力系统概述
仰望U9是比亚迪旗下高端品牌仰望推出的纯电动超级跑车，搭载**易四方**四电机独立驱动技术平台，实现了前所未有的动力输出与操控性能。
---
## 二、核心动力参数
| 参数项 | 数值 |
|--------|------|
| **驱动形式** | 四电机四驱（易四方） |
| **电机数量** | 4台（前后轴各2台） |
| **综合最大功率** | **960 kW（约1,306马力）** |
| **综合最大扭矩** | **1,680 N·m** |
| **0-100 km/h加速时间** | **2.36秒** |
| **0-400米加速时间** | 约9.78秒（尾速约230 km/h） |
| **最高车速** | **309.19 km/h**（实测数据） |
---
## 三、电机详细参数
### 单电机性能
- **单台电机最大功率**：240 kW
- **单台电机最大扭矩**：420 N·m
- **电机类型**：永磁同步电机
- **冷却方式**：高效油冷系统
### 四电机协同控制
- 每个车轮由独立电机驱动，可实现：
  - 瞬时独立扭矩分配
  - ±360°差动转向能力（原地掉头）
  - 爆胎后稳定控制
  - ICE（冰雪路面）防滑模式

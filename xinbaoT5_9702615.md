# 新豹T5 操作指南
  
更新时间：2026年07月20日18时22分02秒   

## 一、车辆启动与熄火
### 1.1 启动前检查
- **绕车检查**：检查轮胎气压、外观有无损伤、车底有无漏油漏水。
- **调整座椅与后视镜**：确保驾驶姿势舒适，视野良好。
- **系好安全带**：驾驶员及所有乘员必须佩戴。
- **检查挡位**：确保变速杆处于空挡（手动挡）或 P 挡（如配备自动挡）。
### 1.2 启动发动机
1. 将钥匙插入点火开关，或确保智能钥匙在车内。
2. 踩下离合器踏板（手动挡）或制动踏板（自动挡）。
3. 转动钥匙至 “START” 位置，或按下 “一键启动” 按钮。
4. 发动机启动后，立即松开钥匙（手动回位）或按钮。
5. 怠速运转 30 秒至 1 分钟，观察仪表盘有无异常报警灯。
> **注意**：冬季冷启动时，建议怠速热车 1-2 分钟，待发动机转速稳定后再起步。
### 1.3 熄火
1. 踩下制动踏板，将车辆完全停稳。
2. 拉紧手刹（驻车制动）。
3. 手动挡挂入空挡；自动挡挂入 P 挡。
4. 关闭所有用电设备（空调、大灯、音响等）。
5. 将钥匙转回 “LOCK” 位置并拔出，或按下 “一键启动” 按钮熄火。

## 二、驾驶操作
### 2.1 手动挡车型 (MT)
| 步骤 | 操作 |
|------|------|
| 1 | 踩下离合器踏板到底 |
| 2 | 挂入 1 挡 |
| 3 | 松开手刹 |
| 4 | 缓慢抬起离合器，同时轻踩加速踏板（油门） |
| 5 | 车辆开始移动后，完全松开离合器，平稳加油 |
**换挡时机参考：**
- 1 挡 → 2 挡：15 km/h 左右
- 2 挡 → 3 挡：25 km/h 左右
- 3 挡 → 4 挡：40 km/h 左右
- 4 挡 → 5 挡：55 km/h 左右
> **提示**：换挡时务必踩下离合器到底，避免打齿。
### 2.2 倒车
1. 车辆完全停稳。
2. 踩下离合器（手动）或制动（自动）。
3. 将换挡杆挂入 R 挡（倒挡）。
4. 观察后方及两侧后视镜，缓慢松开离合器或制动，轻踩油门控制速度。
### 2.3 坡道起步（手动挡）
1. 拉起手刹。
2. 踩下离合器，挂入 1 挡。
3. 轻踩油门，使发动机转速升至 1500-2000 转。
4. 缓慢抬起离合器至半联动点（发动机声音变沉，车身轻微抖动）。
5. 松开手刹，同时继续加油并完全抬起离合器。

## 三、仪表盘与指示灯
### 3.1 主要仪表
- **车速表**：显示当前行驶速度（km/h）。
- **转速表**：显示发动机转速（×1000 rpm）。
- **燃油表**：显示油箱剩余油量，指针接近 E 时需加油。
- **水温表**：显示发动机冷却液温度，正常范围在 C（冷）与 H（热）之间。
### 3.2 重要报警灯
| 图标 | 名称 | 含义与处理 |
|------|------|------------|
| 🛑 | 制动系统报警灯 | 手刹未松或制动液不足，检查手刹或补充制动液 |
| 🔧 | 发动机故障灯 | 发动机电控系统故障，尽快检修 |
| 🔋 | 充电系统报警灯 | 发电机或蓄电池故障，检查充电系统 |
| 🛢️ | 机油压力报警灯 | 机油压力过低，立即停车检查机油量 |
| 🌡️ | 水温过高报警灯 | 冷却液温度过高，停车怠速冷却，检查冷却液 |
| 🔺 | ABS 报警灯 | 防抱死制动系统故障，制动时无 ABS 辅助，谨慎驾驶 |
> **提示**：启动时所有报警灯会自检点亮，几秒后应全部熄灭，若行驶中亮起需及时处理。

## 四、安全操作规范
### 4.1 行驶中
- **严禁空挡滑行**：空挡滑行时发动机失去制动辅助，增加刹车距离。
- **合理使用挡位**：避免低挡高速或高挡低速，防止发动机积碳或损伤。
- **保持安全车距**：载货后制动距离会增加，应比空车时留出更多空间。
- **转弯减速**：货车重心较高，转弯时提前减速，避免侧翻。
### 4.2 载货注意事项
- **严禁超载**：严格遵守车辆最大总质量（GVWR）限制。
- **货物固定**：使用绑带或绳索固定货物，防止行驶中移位。
- **均匀分布**：货物重心尽量居中且靠前，避免尾部过重导致转向不稳。
### 4.3 恶劣天气
- **雨天**：降低车速，避免急刹，保持更大跟车距离。
- **雾天**：开启雾灯和危险报警灯，必要时靠边停车。
- **冰雪路面**：使用防滑链，轻踩油门和刹车，避免急打方向。

## 五、日常维护与保养
### 5.1 每日检查
- [ ] 机油液位（冷车时检查）
- [ ] 冷却液液位
- [ ] 制动液液位
- [ ] 轮胎气压及外观
- [ ] 灯光（大灯、转向灯、刹车灯、雾灯）
- [ ] 雨刮器及清洗液
### 5.2 定期保养周期（建议）
| 项目 | 周期 |
|------|------|
| 更换机油及机油滤清器 | 每 5000 km 或 6 个月 |
| 更换空气滤清器 | 每 10000 km |
| 更换燃油滤清器 | 每 20000 km |
| 检查/更换火花塞 | 每 30000 km |
| 更换变速箱油 | 每 40000 km |
| 更换制动液 | 每 2 年 |
| 更换冷却液 | 每 2 年 |
| 检查轮胎磨损/换位 | 每 10000 km |
### 5.3 轮胎维护
- **标准胎压**（参考值，具体以油箱盖或车门标签为准）：
  - 前轮：3.5 - 4.0 bar
  - 后轮：4.0 - 4.5 bar（载货时取上限）
- **轮胎换位**：每 10000 km 进行交叉换位。
- **磨损标记**：胎面花纹深度低于 1.6 mm 时必须更换。
### 5.4 长期停放
- 停放在干燥通风处，拉紧手刹。
- 断开蓄电池负极，防止亏电。
- 每 2 周启动发动机运行 15 分钟，为蓄电池充电并润滑发动机部件。
- 轮胎气压适当调高，或定期移动车辆防止轮胎变形。

## 六、应急处理
### 6.1 发动机无法启动
1. 检查燃油量。
2. 检查蓄电池接线柱是否松动或腐蚀。
3. 尝试再次启动，每次启动不超过 5 秒，间隔 10 秒。
4. 若仍无法启动，联系售后服务。
### 6.2 行驶中爆胎
1. 紧握方向盘，保持直线行驶。
2. 缓慢松开油门，切勿急刹。
3. 让车辆自然减速，轻踩刹车，靠边停车。
4. 打开危险报警灯，放置三角警示牌（车后 50-100 米）。
### 6.3 制动失灵
1. 立即松开油门。
2. 快速连续踩踏制动踏板（尝试恢复制动压力）。
3. 逐级降挡，利用发动机制动减速。
4. 拉手刹（注意不要拉死，防止抱死甩尾）。
5. 寻找避险车道或利用路边障碍物摩擦减速。
### 6.4 车辆涉水
1. 确认水深不超过排气管高度。
2. 挂低挡（1 挡或 2 挡），匀速通过，中途不停车、不换挡。
3. 涉水后轻踩刹车，排出刹车鼓内的水分，恢复制动性能。

4. **禁止改装**：未经厂家许可，不得擅自改装电路、底盘、悬挂等系统。

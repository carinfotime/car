# 奥迪Q2L 技术参数介绍
  
更新时间：2026年07月16日11时12分41秒   

## 一、车型概述
奥迪Q2L是一汽-大众奥迪旗下的一款小型豪华SUV，基于MQB平台打造，专为中国市场进行了轴距加长（L代表长轴距），兼顾了时尚外观与实用空间。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **发动机类型** | 1.4T涡轮增压直列四缸发动机 |
| **发动机型号** | EA211（DJS） |
| **排量** | 1395 mL |
| **进气形式** | 涡轮增压 |
| **最大功率** | 110 kW（150 Ps）@5000-6000 rpm |
| **最大扭矩** | 250 N·m @1750-3000 rpm |
| **燃油标号** | 95号及以上无铅汽油 |
| **供油方式** | 缸内直喷 |
| **排放标准** | 国VI |

## ⚙️六、性能参数
> *注：以下数据为官方公布值，实际表现受路况、驾驶习惯等因素影响。*
-   **最高车速：200 km/h**
-   **官方0–100 km/h加速时间：8.7 s**
-   NEDC综合油耗：
    -   ≈6.1 L/100km
- WLTC综合油耗：
    - ≈6.86 L/100km
-  油箱容积：
    - ≈50 L
-后备厢容积：
    标准状态约347 L；
    后排座椅放倒后可扩展至约1000 L以上。

## 📦七、主要配置亮点
### 🛡️安全配置
- ABS防抱死制动系统 + EBD电子制动力分配
- ESP车身电子稳定系统 + ASR牵引力控制
- HHC坡道起步辅助
- TPMS胎压监测报警系统
-前排正面双安全气囊 +前排侧面安全气囊 +前后贯穿式头部气帘
主动安全辅助功能：
定速巡航 / ACC自适应巡航（选装）
主动刹车/主动安全系统（部分车型标配或选装）
车道偏离预警及保持辅助（选装）
### 🎨外观与灯光
全LED大灯及LED日间行车灯；
大灯清洗装置；
电动调节外后视镜带加热功能；
全景天窗；
铝合金轮毂。
### 🎵内饰与科技配置
12.3英寸奥迪虚拟座舱全液晶仪表盘；
8.3英寸MMI多媒体中控触摸屏；
支持Apple CarPlay / Baidu CarLife手机互联映射；
语音识别控制系统；
真皮多功能方向盘带换挡拨片；
自动空调带PM2.5过滤功能。
### 💺座椅与舒适性配置
织物座椅 /皮质+织物混搭座椅 /真皮座椅 （视配置等级而定）；
主副驾手动调节或电动调节+腰部支撑 （高配）；
后排座椅支持比例放倒 (4:6)。
---
## 📝八、总结评价要点 (参考)
✅优点：
1.品牌溢价较高，豪华感突出。
2.加长轴距后后排腿部空间优于海外版Q2。
3.NEDC油耗较低，日常使用经济性较好。
4.外观设计年轻动感，配色丰富。
⚠️需注意：
1.WLTC实际油耗略高于NEDC标称值。
2.部分舒适性及智能驾驶辅助功能需选装。
3.后备箱标准容积在同级中不算突出。
4.DSG双离合在低速蠕行时偶有顿挫感。

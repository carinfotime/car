# 本田皓影（BREEZE）技术参数介绍
  
更新时间：2026年07月19日13时25分48秒   

## 一、车型概览
皓影（BREEZE）是广汽本田旗下的一款紧凑型SUV，与东风本田CR-V为姊妹车型，于2019年正式上市。提供燃油版和混动版两种动力选择。
---
## 二、动力系统
### 1. 燃油版（240TURBO）
| 项目 | 参数 |
|------|------|
| 发动机型号 | L15BT |
| 发动机类型 | 1.5T 直列四缸涡轮增压发动机 |
| 排量 | 1498 mL |
| 进气形式 | 涡轮增压 |
| 最大功率 | **142 kW（193 PS）** / 5600 rpm |
| 最大扭矩 | **243 N·m** / 2000-5000 rpm |
| 供油方式 | 缸内直喷 |
| 压缩比 | 10.3:1 |
| 燃油标号 | 92号及以上无铅汽油 |
### 2. 混动版（e:PHEV / e:HEV）
#### e:HEV（油电混动）
| 项目 | 参数 |
|------|------|
| **发动机** | LFB11 / LFB16（2.0L阿特金森循环） |
| ——排量 | **1993 mL** |
| ——最大功率 | **107 kW（145 PS）** / 6200 rpm |
| ——最大扭矩 | **175 N·m** / -3500 rpm |
| **驱动电机** | ——— |
| ——电机类型 | AC同步电动机 |
| ——最大功率 | **135 kW（184 PS）** |
| ——最大扭矩 | **315 N·m** |
#### e:PHEV（插电混动）
- **纯电续航里程（WLTC）：约73 km**
- **综合油耗：约1.3-1.6 L/100km**

## 四、底盘与悬架
- **驱动方式：前置前驱 / Real-Time AWD智能四驱**
- **前悬架类型：麦弗逊式独立悬架**
- **后悬架类型：多连杆式独立悬架**
- **转向系统：电动助力转向系统（EPS）**
- **制动系统：前通风盘式 /  后盘式**

## 六、性能数据
### 🏎️加速性能 (0→100 km/h)
> *官方数据仅供参考*
*  燃油两驱 CVT ：约9.3秒  
*  燃油四驱 CVT ：约9.9秒  
*  混动 e:HEV ：约7.8秒  
*  插电 e:PHEV ：约8秒左右  
### 🛞最高车速 (km/h)
*  燃油两驱 CVT ：188 km/h  
*  燃油四驱 CVT ：188 km/h  
*  混动 e:HEV ：165 km/h  
### ⛽油耗表现 (WLTC综合)
*  燃油两驱 CVT ：7.31 L/100km  
*  燃油四驱 CVT ：7.74 L/100km  
*  混动 e:HEV ：5.2~5.8 L/100km  
*  插电 e:PHEV （馈电状态）：~5L/100km

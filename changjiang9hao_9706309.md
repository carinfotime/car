# 长江9号 保养指南
  
更新时间：2026年07月12日14时10分33秒   

## 一、概述
长江9号作为一款经典车型，定期保养是确保其性能稳定、延长使用寿命的关键。本指南涵盖常规保养项目、周期及注意事项，适用于日常使用与长途出行前的检查。
---
## 二、保养周期总表
| 保养项目 | 建议周期 |
|---------|---------|
| 机油更换 | 每5,000公里或6个月 |
| 机油滤清器更换 | 每5,000公里（与机油同步） |
| 空气滤清器检查/更换 | 每10,000公里检查，每20,000公里更换 |
| 空调滤清器更换 | 每10,000公里或每年 |
| 燃油滤清器更换 | 每20,000公里 |
| 火花塞检查/更换 | 每20,000公里检查，每40,000公里更换 |
| 制动液更换 | 每40,000公里或2年 |
| 防冻液更换 | 每40,000公里或2年 |
| 变速箱油更换 | 每60,000公里（手动/自动均需） |
| 正时皮带/链条检查 | 皮带：每60,000-80,000公里；链条：免维护（需定期检查） |
| 轮胎换位与动平衡 | 每10,000公里 |
| **四轮定位** | **出现跑偏或轮胎偏磨时** |
---
## 三、日常检查项目（建议每月/每次加油时）
- [ ] **轮胎气压**：冷车状态下检查，标准值参见驾驶侧门框标签
- [ ] **轮胎外观**：有无鼓包、裂纹、异常磨损
- [ ] **机油液位**：冷车平地拔出机油尺，油位应在上下刻度之间
- [ ] **冷却液液位**：观察膨胀罐，应在MIN与MAX之间
- [ ] **制动液液位**：在MAX线附近为正常
- [ ] **玻璃清洗液**：不足时及时补充
- [ ] **灯光系统**：大灯、转向灯、刹车灯、雾灯是否正常
- [ ] **雨刮片**：刮拭是否清晰，有无异响或老化

## 四、各系统详细保养说明
### 🔧  发动机系统
1. **机油**
   - **推荐标号**：5W-30（四季通用）或根据当地气温选择0W-40/10W-40
   - **用量**：约4.0~4.5升（以实际油尺为准）
   - **注意事项**：务必使用正品机油，劣质机油会加速发动机磨损
2. **空气滤清器**
   - *拆卸后轻拍除尘*可延长寿命，但不可水洗（干式滤芯）
   - *安装时确保密封严实*，防止未过滤空气进入发动机
3. **火花塞**
   - *推荐型号*：NGK BKR6E / DENSO K20PR-U（具体以原厂规格为准）
   - *安装扭矩*：约25~30 N·m，切勿过紧
### 🛞  底盘与悬挂系统
1. **制动系统**
   - *刹车片厚度*低于3mm时必须更换
   - *刹车盘磨损*超过1.5mm台阶或表面严重沟槽需光盘或更换
   - *制动液含水量*超过3%必须立即更换（可用检测笔测试）
2. **轮胎**
   - *胎压标准*：
     - 🚗前轮：2.3 bar (230 kPa)
     - 🚗后轮：2.5 bar (250 kPa)
     （满载时可适当增加0.1~0.2 bar）
   - *花纹深度*低于1.6mm必须更换
   - *建议品牌*：朝阳 / Michelin / Bridgestone（视预算选择）
3. **悬挂部件**
   - *减震器漏油*是常见问题，发现渗漏及时检修
   - *下摆臂胶套老化开裂*会导致底盘松散异响
### ⚡️  电气系统
1. **蓄电池**
   - *电压标准*：
     ✅静态电压 ≥12.4V  
     ✅启动瞬间电压 ≥9.6V  
     低于上述值建议充电或更换
2. **发电机皮带**
   目视检查有无龟裂、毛边，张紧度以按压下沉约10~15mm为宜。
### ❄️🚗💨   空调系统
1. *制冷剂压力检测*
   低压侧约0.15~0.25 MPa  
   高压侧约1.3~1.7 MPa  
   压力异常需专业检修。
2. *冷凝器散热片*
   定期用低压水枪冲洗表面脏污杂物。

## ⚠️⚠️⚠️   五、常见故障预警信号 ⚠️⚠️⚠️  
> ⚡仪表盘亮起以下指示灯请立即停车检修：
🚨发动机故障灯 → OBD电脑读取故障码  
🚨ABS故障灯 → ABS泵/轮速传感器可能异常  
🚨安全气囊灯 → SRS系统故障  
🚨水温过高报警 → 🔥立即熄火！等待冷却后补充防冻液

- ✅换空气滤芯和空调滤芯（无需工具）
- ✅添加玻璃水 / 💧冷却液补充 / 🔋电瓶桩头清洁除锈  
❌建议去专业维修店的项目：
- ❌变速箱油循环机换油  
- ❌正时皮带拆装  
- ❌四轮定位调校  
- ❌空调制冷剂加注

## 📌📌📌   八、特别提醒 📌📌📌  
🔹新车首保通常在3个月或3000~5000公里内完成。  
🔹若长期短途行驶或多尘环境行驶→适当缩短空滤和机油的更换周期。  
🔹雨季过后注意检查天窗排水孔是否堵塞。  
🔹冬季来临前提前测试暖风系统和电瓶健康状况。

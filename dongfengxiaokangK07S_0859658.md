# 东风小康K07S 驾驶注意事项
  
更新时间：2026年07月19日17时46分03秒   

## 一、启动前检查
1.  **环绕检查**：绕车一周，检查轮胎气压、外观有无破损，车身周围有无障碍物或儿童。
2.  **油液检查**：检查机油液位、冷却液液位、制动液液位是否在正常刻度范围内。
3.  **灯光与信号**：检查大灯、转向灯、刹车灯、雾灯及双闪灯是否工作正常。
4.  **仪表盘自检**：插入钥匙拧至ON档，等待仪表盘所有警示灯自检完毕后（除手刹灯、安全带灯外）熄灭再启动。
## 二、驾驶操作要点
1.  **离合与换挡**：
    -   该车离合行程较长，结合点较低，起步时需缓抬离合，适当给油以防熄火。
    -   换挡时务必**将离合踩到底**，避免打齿损伤变速箱。
    -   行驶中保持转速在2000-2500转左右换挡较为平顺且省油。
2.  **动力特性**：
    -   发动机排量较小（多为1.0L-1.4L），满载或开空调时动力会明显下降。
    -   超车或爬坡时建议**提前降挡**，保持充足扭矩。
3.  **转向手感**：
    -   该车多配备液压助力（部分低配无助力），低速时方向盘偏重，掉头或入库时需用更大力度。
    -   切记**禁止原地打死方向盘**，以免损坏助力泵。
4.  **刹车注意**：
    -   前盘后鼓刹车，刹车距离相对较长，且连续下坡时鼓刹容易热衰减。
    -   长下坡时请**利用发动机制动**（挂低挡位），避免长时间踩刹车导致刹车失效。
## 三、载货与载人安全
1.  **严禁超载**：严格按照行驶证核定人数/载质量装载，超载会严重影响制动和操控稳定性。
2.  **货物固定**：若拆除后排座椅载货，货物必须**均匀分布**并牢固固定，防止急刹车时货物前冲伤及驾驶员。
3.  **重心较高**：车身高、轮距窄，过弯时**务必减速**（建议时速不超过40km/h），防止侧翻。
## 四、特殊路况驾驶
1.  **雨天/湿滑路面**：
    -   保持更长的跟车距离。
    -   避免急加速、急刹车，防止后轮抱死侧滑。
2.  **山路/坡道**：
    -   坡道起步善用手刹配合，防止溜车。
    -   下山挂1挡或2挡，用发动机牵引力控制车速。
## 五、日常维护与保养
1.  **定期保养**：严格按照保养手册更换机油、机滤、空滤等。建议每5000公里或半年保养一次。
2.  **轮胎维护**：定期检查胎压（建议前轮3.0-3.5bar，后轮3.5-4.0bar，具体参考油箱盖标签），注意轮胎磨损标记。
3.  **皮带检查**：发电机皮带若出现裂纹或异响，需及时更换，否则抛锚风险高。
4.  **底盘防锈**：该车底盘防锈处理一般，涉水后或冬季雪后建议冲洗底盘。
## 六、其他提醒
-   **限速**：车辆最高设计时速有限，高速行驶建议控制在90-100km/h以内，超过110km/h车辆发飘明显，油耗剧增。
-   **空调使用**：开空调会增加发动机负荷，起步时可先关闭AC，车辆动起来后再开启。
-   **熄火后**：停车熄火前，建议先关闭空调、大灯等用电设备，减轻电瓶负担。

# 恺达EX6 保养指南
  
更新时间：2026年07月11日04时13分52秒   

## 一、概述
恺达EX6 作为一款纯电动微型汽车，其保养与传统燃油车有较大区别，主要集中在**三电系统（电池、电机、电控）**以及底盘、制动、空调等常规部件。正确的保养不仅能延长车辆寿命，还能保障行车安全。

## 三、核心部件保养详解
### 1. 高压动力电池
- 避免长时间将电池电量耗尽至0%，建议电量剩余20%-30%时及时充电
- 长期停放（超过7天），建议保持电量在50%-80%之间
- 尽量使用**慢充**进行日常充电，减少快充频率
- 每月至少进行一次完整充放电循环（充至100%再使用至20%左右），有助于电池SOC校准
- 注意充电环境温度，避免在0℃以下或45℃以上极端温度下充电
- 如发现续航异常下降、充电时间异常变长，请及时到服务站检测
### 2. 驱动电机及控制器
- 保持电机及控制器表面清洁，避免积尘影响散热
- 检查高压线束接头是否松动、氧化或破损
- 检查电机运行是否有异响、异味
- 控制器散热风扇是否正常工作
### 3. 减速器（变速箱）
- 每10,000公里（首次）检查齿轮油油位及状态
- 之后每40,000公里或24个月更换一次
- 使用专用齿轮油，请勿混用其他型号
### 4. 制动系统
- 由于电动车带有能量回收功能，机械刹车片磨损相对较慢，但仍需定期检查
- 每10,000公里检查刹车片厚度，低于3mm需更换
- 每2年或40,000公里更换制动液
- 检查制动管路有无渗漏、老化

## 四、日常自检项目（建议每月一次）
| 检查项目 | 检查内容 |
|---------|---------|
| 轮胎 | 胎压（建议2.3-2.5bar）、胎面磨损、有无鼓包或裂纹 |
| 灯光 | 大灯、转向灯、刹车灯、雾灯是否正常 |
| 雨刮 | 雨刮片是否老化、刮水效果是否良好 |
| 玻璃水 | 液位是否充足，冬季需使用防冻型 |
| 充电口 | 充电口是否清洁、无异物、无锈蚀 |
| 仪表盘 | 是否有故障灯亮起 |
| 空调 | 制冷/制热效果是否正常 |
| 车门密封条 | 是否老化、漏水 |
---
## 五、各季节保养注意事项
### 春季
- 检查空调系统，为夏季制冷做准备
- 清理底盘泥沙，检查底盘防锈
- 更换空调滤芯
### 夏季
- 注意电池散热，避免暴晒后立即快充
- 检查轮胎胎压，夏季可适当降低0.1bar
- 检查冷却液液位及状态
### 秋季
- 检查雨刮片及玻璃水，准备应对多雨天气
- 检查灯光系统，应对能见度低的季节
- 检查轮胎花纹深度
### 冬季
- 电池续航会有所下降，属正常现象
- 尽量将车辆停放在室内或地下车库
- 使用前可提前预热电池（如有此功能）
- 检查防冻液冰点
- 冬季充电尽量在行驶后立即进行，利用电池余温
---
## 六、保养注意事项
1. **必须到授权服务站**进行三电系统相关保养，非授权拆修可能导致保修失效
2. 保养时请携带**保养手册**，记录每次保养内容
3. 如车辆出现以下情况，请立即送修：
   - 仪表盘出现红色故障灯
   - 充电异常（无法充电、充电中断）
   - 行驶中突然动力下降或消失
   - 车辆底部有液体泄漏
   - 闻到焦糊味或异常气味
4. 非官方改装（加装电器、改装电路等）可能影响高压系统安全
5. 建议使用原厂配件及专用油液

## 八、常见问题 FAQ
**Q：电动车需要保养什么？没有发动机是不是就不用保养了？**
A：虽然不需要换机油机滤，但三电系统、制动、底盘、空调等部件仍需要定期检查和维护，不可忽视。
**Q：续航下降是不是电池坏了？**
A：不一定。续航受温度、驾驶习惯、充电方式等影响。冬季续航下降30%左右属正常现象。如持续异常下降，建议进站检测。
**Q：可以自己更换空调滤芯吗？**
A：可以。空调滤芯通常位于手套箱后方，可参考用户手册自行更换，难度较低。
**Q：长期不开需要注意什么？**
A：保持电量50%-80%，停放于干燥阴凉处，每两周检查一次电量并适当补充。

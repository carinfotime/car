# 长安Lumin 儿童乘车安全注意事项
  
更新时间：2026年07月10日18时03分30秒   

## 一、核心安全原则
1.  **儿童严禁坐在前排**：无论任何情况，12岁以下的儿童均不得坐在副驾驶位置。前排的安全气囊在弹出时会对儿童造成严重伤害。
2.  **必须使用儿童安全座椅**：
    -   4周岁以下或体重低于18kg的儿童，必须使用符合国家标准的儿童安全座椅。
    -   建议反向安装（尤其是1岁以内或体重低于13kg的婴儿），以更好地保护头颈。
3.  **开启儿童安全锁**：长安Lumin的后车门配有儿童安全锁。每次搭载儿童前，请务必手动将后车门侧面的锁止机构拨至“LOCK”位置，防止行车中儿童误开车门。
## 二、行车途中注意事项
-   **车窗管理**：
    -   请勿让儿童将头、手或身体任何部位伸出窗外。
    -   建议开启主驾驶位的“车窗锁止”功能（通常位于主驾门板），防止后排儿童随意升降车窗造成夹伤。
-   **车内物品摆放**：
    -   中控台、后窗台等处不要放置硬物、尖锐物品或易碎品（如香水瓶、摆件），急刹车时可能飞溅砸伤儿童。
-   **安全带使用**：
    -   对于身高达到可以使用成人安全带标准的较大儿童（通常身高超过140cm），请确保安全带斜跨肩膀和胸部，而非勒住脖子或腹部。
## 三、停车与上下车
-   **驻车确认**：停车后务必挂入P挡并拉起手刹（电子手刹需确认指示灯亮起），确保车辆完全静止后再让儿童上下车。
-   **开门观察**：教导并监督儿童在开门前先通过车窗或后视镜观察后方是否有来车或行人（警惕“开门杀”）。
-   **禁止单独滞留车内**：切勿将儿童单独留在长安Lumin车内，哪怕只是短暂离开。夏季车内温度会迅速升高至致命水平；冬季也存在缺氧风险。
## 四、车辆特性相关提醒
-   **低速提示音 (AVAS)**：长安Lumin作为电动车，在低速行驶时会发出模拟提示音。请提前告知年龄较小的乘客不要因此感到惊慌。
-   **充电安全**：充电时，请勿让儿童触碰充电枪接口及车辆充电口金属端子。尽量在充电时将车辆停放在远离孩子玩耍的区域。
## ✅ 家长自查清单
| 项目 | 检查结果 |
| :--- | :--- |
| ✅ 后排两侧是否已开启“儿童安全锁”？ | ☐ |
| ✅ 副驾驶位是否已关闭前方气囊？ (若反向安装座椅在前排) | ☐ (不推荐) |
| ✅ “车窗锁止”按钮是否已按下？ | ☐ |
| ✅ 车内是否有易滚动/尖锐的危险物品？ | ☐ |
| ✅ 所有乘客（包括后排）是否已系好安全带/固定好座椅？ | ☐ |
> **总结：**
> “反向座椅 + 后排位置 + 开启童锁 + 关好车窗”是保护孩子在长安Lumin内乘车安全的黄金法则。

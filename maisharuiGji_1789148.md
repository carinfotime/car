# 迈莎锐G级（Mansory G-Class）保养指南
  
更新时间：2026年07月20日08时23分42秒   

## 一、概述
迈莎锐G级是基于梅赛德斯-奔驰G-Class进行重度改装的高性能豪华越野车。因其独特的碳纤维车身套件、升级的动力系统及定制化内饰，保养要求高于普通G级。本指南适用于搭载 **Mansory 升级套件** 的 G500 / G63 / G65 等车型。

##  三、Mansory专属部件保养要点
### 🔹1.碳纤维车身套件
-  使用中性pH值洗车液，避免强酸强碱腐蚀清漆层。
-  不可使用高压水枪直射碳纤维接缝处，防止脱层。
-  每月使用专用碳纤维护理蜡（如 Swissvax Carbon Wax）。
-  轻微划痕可用P21S抛光剂手工处理；深度损伤需返厂修复。
### 🔹2.轮毂与轮胎
- Mansory锻造轮毂：定期检查螺栓扭矩（建议220 N·m），防止松动。
-  高性能轮胎（如 Pirelli P Zero / Michelin Pilot Sport）：胎压建议前2.8 bar /后3.0 bar（冷态）。
-  轮胎磨损不均匀时立即做四轮定位及动平衡。
### 🔹3.排气系统
- Mansory不锈钢运动排气：每年检查吊耳橡胶老化情况。
-  阀门机构：每月手动开关几次防止卡滞。
-  尾喉积碳：使用金属抛光剂清洁。
### 🔹4.动力系统
- ECU程序升级后必须使用≥98号汽油（RON），国内推荐98#或100#赛用燃油。
-  涡轮增压车型：熄火前怠速1~2分钟冷却涡轮，避免机油结焦。
-  中冷器散热片：每半年用低压水冲洗灰尘昆虫尸体。
### 🔹5.制动系统
- Mansory大尺寸刹车盘+多活塞卡钳：
    -   刹车片厚度低于3mm必须立即更换。
    -   刹车盘表面出现深沟槽或裂纹需光盘或更换。
    -   推荐使用原厂Mansory刹车片或 Brembo HP系列替代品。

##  四、内饰保养
### 🔹真皮/Alcantara材质
> - Alcantara：专用泡沫清洁剂 +软毛刷顺向刷洗→吸尘器吸干→晾干24小时。
> - Nappa真皮：每月一次 pH中性皮革清洁剂 +皮革护理霜（不可含硅油）。
> -   避免阳光直射暴晒，建议贴隔热膜 +遮阳挡。
### 🔹碳纤维饰板
> -   微湿超细纤维布擦拭即可；顽固污渍用异丙醇稀释液(1:10)。
> -   禁止使用含氨清洁剂(如玻璃水)，会腐蚀清漆层。

## ⚠️五、常见故障预警与处理
1. **发动机故障灯亮**
    可能原因：进气温度过高 /燃油品质不达标 /氧传感器中毒。  
    处理方式：立即停车→读取故障码→联系Mansory授权服务中心。
2. **底盘异响**
    可能原因：改装悬挂衬套磨损 /防倾杆连杆松动。  
    处理方式：升起车辆检查所有底盘紧固件扭矩值。
3. **空调异味**
    可能原因：蒸发箱霉菌滋生(Mansory内饰隔音材料吸潮)。  
    处理方式：专业蒸发箱清洗 +紫外线杀菌 +空调滤芯更换为活性炭型。
4. **电瓶亏电**
    可能原因：Mansory加装电子设备(氛围灯/音响/电控排气)静态电流过大。  
    处理方式：长期停放建议连接智能充电维护器(CTEK MXS5.0)。

## 📅七、季节性保养提示
### ❄️冬季准备(November)
> - [ ]测试电池健康度(CCA≥700A)
> - [ ]更换冬季胎(推荐倍耐力ScorpionWinter)
> - [ ]检查防冻液冰点(-35℃以下)
> - [ ]润滑门锁铰链防止冻结
### ☀️夏季准备(May)
> - [ ]清洗中冷器散热片
> - [ ]检查空调制冷剂压力(R134a/R1234yf)
> - [ ]轮胎气压调整至夏季标准(-0.2bar)
> - [ ]全车漆面打蜡保护紫外线伤害

---

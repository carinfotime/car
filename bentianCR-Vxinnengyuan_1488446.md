# 本田CR-V新能源 保养指南

## 一、保养周期总览
| 保养项目 | 推荐周期 |
|---------|---------|
| 机油及机滤更换 | 每10,000公里或12个月（先到为准） |
| 空调滤芯更换 | 每10,000公里或12个月 |
| 空气滤芯更换 | 每20,000公里或24个月 |
| 制动液更换 | 每40,000公里或36个月 |
| 冷却液更换 | 每100,000公里或60个月（首次），之后每50,000公里 |
| 火花塞更换 | 每80,000公里 |
| 变速箱油更换（E-CVT） | 每40,000公里检查，视情况更换 |
| 轮胎换位及动平衡 | 每10,000公里 |
| 制动片检查 | 每10,000公里 |

## 二、动力系统保养
### （一）发动机部分（2.0L阿特金森循环发动机）
1. **机油选择**
   - **推荐标号**：0W-20（全合成机油）
   - **容量**：约3.5~4.0升
   - **标准**：API SP / ILSAC GF-6及以上级别
   - **品牌建议**：本田原厂机油、嘉实多、美孚、壳牌等大品牌全合成机油
2. **机滤**
   - **每次换油必须同时更换机滤**
   - **建议使用原厂或曼牌、马勒等高品质机滤**
3. **注意事项**
   - CR-V新能源的发动机并非全程工作，频繁启停对机油品质要求更高
   - EV模式行驶较多时，发动机运行时间短，但依然需按时间周期换油
   - **切勿超期使用机油**，防止乳化或劣化
### （二）混合动力系统（i-MMD）
1. **高压电池散热系统**
   - **后座下方电池散热进风口需定期清理**（建议每次保养检查）
   - **禁止堵塞后排座椅下方的散热口**
   - *注意*：散热口堵塞会导致电池过热，影响寿命和性能
2. **逆变器/电机冷却液**
   - *与发动机冷却液分开*，有独立的冷却回路
   - **检查周期**：每20,000公里
   - *注意*：不可混用不同颜色的冷却液
3. **能量回收系统**
   - *无需特别维护*
   - *注意*：刹车踏板行程感觉变化时及时进店检查
---
## 三、制动系统保养
1. **制动液**
   - DOT4标准制动液
   - *含水量检测*：每次保养检测，超过2%必须立即更换
2. **制动片与制动盘**
   - CR-V新能源因能量回收机制，机械刹车磨损相对传统车型更慢
   - *但仍需定期检查厚度*
3. **电子驻车系统**
   - *无需手动调整*
4. *特别注意*
   由于混合动力车型的再生制动特性：
      > ⚠️ CR-V新能源长期低强度使用机械刹车时，制动盘表面可能生锈。建议每月至少进行几次急加速后的重刹车操作以清除锈迹。
---
## 四、轮胎与底盘保养
1. **轮胎气压标准**
| |前轮（kPa） |后轮（kPa）|
|---|---|---|
|标准负载 |230~240 |230~240|
|满载/高速长途 |250~260 |250~260|
2. **轮胎换位方式**：交叉换位法
3. *四轮定位*：
   遇到以下情况需做四轮定位：
   车辆跑偏 /方向盘不正 /轮胎偏磨 /更换悬挂部件后
4. *底盘关键点检查*
   高压线束防护罩是否完好 → 
   排气管吊耳是否老化 → 
   悬挂胶套是否开裂 →

## 五、空调与空气过滤系统
1. **空调滤芯**
   位置：手套箱后方
   类型：活性炭/普通纸质可选
   建议每年至少更换一次，雾霾/沙尘地区缩短至6个月
2. **空气滤芯（发动机进气）**
   位置：发动机舱内黑色塑料盒中
   类型：干式纸质滤芯
3. *蒸发箱清洗*
   出现异味时进行专业清洗，
   不建议自行使用泡沫清洗剂。

## ⚠️ CR-V新能源专属注意事项
### 🔋高压电池维护建议：
- ✅长期停放时保持电量在40%~60%
- ✅每月至少启动车辆一次并运行HEV模式30分钟以上
- ❌避免长期满电存放（>90%超过两周）
- ❌避免长期亏电存放（<20%超过一周）
- ✅夏季高温停车尽量选择阴凉处或地下车库
### 🛞EV模式与HV模式合理切换：
- EV模式适合市区短途通勤（<50km）
- HV模式适合高速长途行驶以保持最佳燃油经济性
- B档位在下坡路段可增强能量回收效果并减少刹车磨损
### 🧊冬季使用提示：
- ❄️低温下纯电续航会明显下降属正常现象
- ❄️建议预热车辆后再出发以降低冷启动损耗
- ❄️冰雪路面慎用强能量回收档位以防打滑失控风险。

## 🛠️常见问题解答Q&A：
Q: CR-V新能源需要热车吗？  
A:不需要原地热车。启动后低速行驶即可让发动机和电机进入工作状态。
Q:可以用普通机油吗？  
A:不可以。必须使用0W-20全合成机油以满足频繁启停需求。
Q:多久需要去4S店做一次大保养？  
A:通常40,000km为一次大保养节点包含制动液变速箱油等项目费用约1500~2500元视地区而定。
Q:纯电续航衰减怎么办？  
A:正常衰减属于保修范围具体参考购车时的电池质保政策通常为8年或15万公里。

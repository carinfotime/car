## 二、A/B类保养详解
### A保（每10,000公里/1年）
1. **更换机油及机滤**
   - 推荐机油规格：MB 229.51 / 229.52 认证的5W-30或0W-40全合成机油
   - 用量：约5.5L（依具体发动机型号略有差异）
2. **检查项目**
   - 检查全车灯光、喇叭、雨刮器
   - 检查轮胎气压及磨损情况
   - 检查制动液含水量及液位
   - 检查冷却液液位及浓度
   - 检查电瓶状态
   - 检查底盘及悬挂系统
   - 读取故障码，复位保养提示
3. **更换空调滤芯**（每2次A保执行一次）
### B保（每20,000公里/2年）
在A保基础上增加以下项目：
1. **更换空气滤芯**
2. **更换空调滤芯**
3. **更换制动液**
4. **检查火花塞状态**
5. **检查皮带及张紧器**
6. **检查发动机燃烧室积碳情况**
7. **四轮换位**

## 三、核心保养项目操作要点
### 1. 机油更换
- **准备工作：** 车辆热车后熄火，举升车辆或使用千斤顶
- **放油：** 拧下油底壳放油螺栓，彻底放尽旧机油
- **更换机滤：** 拆下旧机滤，密封圈涂抹新机油后安装新机滤
- **加注新油：** 按规定量加注，启动发动机运转1-2分钟后检查油位
- **注意：** 务必使用符合MB认证的机油，非认证机油可能导致发动机损坏
### 2. 火花塞更换
- 奔驰A级涡轮增压发动机建议使用**铱金火花塞**
- 拆卸时需使用专用火花塞套筒，扭矩控制在25-30N·m
- 建议同步检查点火线圈状况
### 3. 变速箱油更换
- **推荐方法：** 循环机更换（更彻底）
- **油品规格：** MB 236.21 认证的7速/8速双离合变速箱油
- **操作流程：**
  1. 举升车辆，拆下护板
  2. 放油螺栓放油，更换滤芯（如有）
  3. 加注新油至溢流口
  4. 启动发动机，各档位切换后再次检查油位
  5. 调整至标准油位
### 4. 制动液更换
- **规格：** DOT 4+ 或 DOT 5.1（推荐使用原厂制动液）
- **方法：** 使用专用换油机或手动排气法
- **标准：** 制动液含水量不得超过2%
- **注意：** 操作时避免制动液接触车漆（具有腐蚀性）

## 四、日常检查清单（建议每月一次）
- [ ] 机油液位（冷车或热车熄火5分钟后检查）
- [ ] 冷却液液位（应在MIN与MAX之间）
- [ ] 制动液液位
- [ ] 玻璃水液位
- [ ] 轮胎气压（含备胎）
- [ ] 轮胎外观（鼓包、裂纹、异常磨损）
- [ ] 全车灯光功能
- [ ] 雨刮片刮水效果
- [ ] 仪表盘警示灯

## 五、常见问题及注意事项
### 常见问题
| 问题 | 可能原因 | 解决方案 |
|------|---------|---------|
| 发动机抖动 | 火花塞老化/积碳 | 更换火花塞，清洗节气门 |
| 加速无力 | 空气滤芯堵塞/燃油系统积碳 | 更换空滤，添加燃油添加剂 |
| 变速箱顿挫 | 变速箱油老化 | 更换变速箱油 |
| 空调异味 | 空调滤芯过脏/蒸发箱滋生细菌 | 更换滤芯，清洗空调系统 |
| 仪表盘亮发动机灯 | 氧传感器/节气门/排放系统问题 | 读取故障码针对性维修 |
### 重要注意事项
1. **保养复位：** 更换机油后需通过仪表盘菜单或诊断仪复位保养提示
2. **原厂配件：** 建议优先使用原厂配件或知名品牌（曼牌、博世、马勒等）
3. **保养记录：** 保留完整保养记录，有助于二手车保值
4. **保修期内：** 保修期内建议在4S店保养，避免影响质保权益
5. **电脑匹配：** 部分保养项目（如更换电瓶）需进行电脑匹配，建议由专业人员操作

## 八、季节性保养建议
### 春季/夏季
- 检查空调制冷效果，必要时补充制冷剂
- 更换空调滤芯
- 检查冷却液浓度（防冻/防沸）
- 检查雨刮片（雨季前更换）
### 秋季/冬季
- 检查蓄电池状态（低温下启动性能下降）
- 更换冬季玻璃水（-25℃或更低标号）
- 检查轮胎胎纹深度，考虑更换冬季胎
- 检查暖风系统

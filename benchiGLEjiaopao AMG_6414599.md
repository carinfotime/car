# 奔驰GLE轿跑 AMG 技术参数介绍
  
更新时间：2026年07月16日12时37分17秒   

## 车型概览
梅赛德斯-奔驰GLE轿跑 AMG 是一款融合了轿跑车的优雅线条与SUV的高通过性，并由AMG部门进行高性能调校的豪华运动SUV。它不仅拥有极具攻击性的外观设计，更在动力性能、操控精准度和科技配置上达到了顶级水准。
## 动力系统
| 参数项 | 具体规格 |
| :--- | :--- |
| **发动机型号** | AMG 4.0升 V8 双涡轮增压发动机（以主流型号为例） |
| **排量** | 3982 cc |
| **进气形式** | 双涡轮增压 |
| **最大功率** | 450 kW (612 马力) |
| **最大扭矩** | 850 N·m |
| **变速箱** | AMG SPEEDSHIFT TCT 9速自动变速箱 |
| **驱动方式** | AMG 高性能四轮驱动系统 (4MATIC+) |
## 性能表现
| 参数项 | 具体规格 |
| :--- | :--- |
| **0-100 km/h 加速时间** | 3.8 秒 |
| **最高时速** | 250 km/h (电子限速) / 可选解除限速至 280 km/h |
| **燃油类型** | 高标号无铅汽油 (推荐 98#) |
| **油箱容积** | 约 85 L |

## 底盘与操控
- **悬挂系统**: AMG 主动式行驶控制 (ACTIVE RIDE CONTROL) 悬挂，带自适应可调减震系统，可有效抑制侧倾。
- **转向系统**: AMG 车速感应式运动转向系统，带可变转向比。
- **制动系统**: 前轮：六活塞固定卡钳 + 通风打孔制动盘；后轮：单活塞浮动卡钳 + 通风打孔制动盘（可选装AMG陶瓷复合制动系统）。
- **轮胎规格**: 前：275/40 R22；后：315/35 R22（具体视配置而定）。

| **轴距** | 2935 mm |
| **整备质量** | 约 2445 kg |
| **后备箱容积** | 490 - 1490 L |
## 核心技术与配置
- **AMG 4MATIC+**: 带完全可变扭矩分配功能的高性能四驱系统，可在前后轴之间按需分配动力。
- **AMG TRACTION CONTROL**: 牵引力控制系统，提供湿滑、运动、赛道等多种模式，允许车轮可控滑动。
- **AMG DYNAMIC SELECT**: 驾驶模式选择器，提供“防滑”、“舒适”、“运动”、“运动增强”、“赛道”及“个性化”模式。
- **MBUX 智能人机交互系统**: 带AMG专属显示风格和性能菜单，可实时显示功率、扭矩、G值、圈速等数据。
- **AMG 高性能排气系统**: 带可变阀门，可通过按键控制声浪，从低沉到高亢。
- **主动式车道保持辅助系统**: 标配智能驾驶辅助组件。
## 总结
奔驰GLE轿跑 AMG 是一款将极致性能、豪华轿跑美学与日常实用性高度结合的车型。它适合追求顶级动力体验、独特个性外观且不牺牲SUV空间与通过性的消费者。

# 宝马iX 空间参数介绍
  
更新时间：2026年07月11日01时23分58秒

| 参数 | 数值 |
|------|------|
| 车长 | 4,953 mm |
| 车宽（含后视镜） | 2,067 mm |
| 车宽（不含后视镜） | 1,967 mm |
| 车高 | 1,695 mm |
| 轴距 | 3,000 mm |
| 前轮距 | 1,700 mm |
| 后轮距 | 1,707 mm |
| 最小离地间隙 |约200 mm |
## 车内空间
### 乘坐空间
- **前排头部空间**：充裕，得益于SUV高车身设计，身高180cm乘客仍有余量
- **后排腿部空间**：轴距达3米，后排腿部空间宽敞，可轻松翘腿
- **后排头部空间**：车顶线条平直，后排头部空间表现优秀
- **座椅布局**：5座布局，后排座椅支持40:20:40比例放倒
### 储物空间
- **中控台储物**：悬浮式中控台下方设有大型储物格
- **门板储物格**：前后门板均可容纳多瓶饮料
- **中央扶手箱**：深度适中，可放置随身物品
- **杯架**：前排2个、后排2个杯架
## 后备箱容积
| 状态 | 容积 |
|------|------|
|标准状态 |约500升|
|后排座椅放倒后 |约1,750升|
|后备箱开口宽度 |宽敞，便于大件物品装卸|
### 后备箱特点
- **电动尾门**：标配电动开合，支持脚踢感应开启
- **地板平整**：放倒后排座椅后可形成近乎纯平的地板
- **隐藏储物格**：地板下方设有额外储物暗格
- **挂钩与锚点**：配备多个挂钩和固定锚点
## 其他空间亮点
1. **全景玻璃天幕**：可选装固定式全景天幕（不可开启），提升车内通透感
2. **“Shy Tech”隐藏科技内饰**：中控区域采用极简设计，视觉上更显开阔
3. **无边框车门设计**：视觉上增加车内横向宽度感
4. **空气悬架调节高度（选装）** ：可调节车身高度以适应不同路况和装载需求

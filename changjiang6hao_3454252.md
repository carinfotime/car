# 长江6号 用户操作指南
  
更新时间：2026年07月14日11时57分17秒   

欢迎使用长江6号。本指南将帮助您快速熟悉车辆的各项功能与操作规范，确保行车安全与舒适。
## 一、车辆启动与熄火
### 1.1 启动车辆
1.  确保智能钥匙在车内。
2.  踩下制动踏板。
3.  按下“一键启动”按钮（位于方向盘右侧）。
4.  仪表盘指示灯自检完成后，车辆进入可行驶状态。
### 1.2 熄火车辆
1.  将车辆完全停稳，挂入 **P档**（驻车档）。
2.  拉起电子手刹。
3.  再次按下“一键启动”按钮，发动机停止工作。
> **注意**：长时间停车请务必挂入P档并拉起手刹。
## 二、驾驶操作
### 2.1 换挡操作
长江6号采用电子旋钮式换挡机构：
- **R档（倒车）**：旋钮向左旋转至R位置。
- **N档（空挡）**：旋钮居中位置。
- **D档（前进）**：旋钮向右旋转至D位置。
- **P档（驻车）**：按下旋钮顶部的“P”按键。
> **提示**：换挡时必须踩住制动踏板。
### 2.2 转向灯与雨刮器
- **左转向灯**：拨杆向下推。
- **右转向灯**：拨杆向上推。
- **雨刮器控制**：
    - `MIST`：单次刮刷
    - `OFF`：关闭
    - `INT` / `AUTO`：间歇/自动模式
    - `LO`：低速连续刮刷
    - `HI`：高速连续刮刷
### 2.3 灯光控制
灯光控制拨杆位于方向盘左侧：
- `OFF` / `AUTO`：关闭/自动大灯
- `示宽灯`
- `近光灯`
- **远光灯**：向前推动拨杆；向后拉动为闪灯一次。
## 三、中控屏与娱乐系统
### 3.1 主界面功能
中控屏主界面包含以下快捷功能：
| 图标 | 功能 |
| :--- | :--- |
| 🗺️ | GPS导航 |
| 📻 | FM/AM收音机 |
| 🎵 | USB/蓝牙音乐 |
| ⚙️ | 车辆设置 |
### 3.2 Apple CarPlay / Android Auto (如有配备)
1.  使用原装USB数据线连接手机至前排USB接口。
2.  手机弹出提示时，选择“允许”。
3.  中控屏自动识别并投射手机应用界面。
## 四、空调系统
### 4.1 A/C开关与温度调节
- **A/C按钮**：按下开启压缩机（制冷）；再次按下关闭。
- **温度调节旋钮**：
    - 🔵蓝色区域 → ❄️冷风 (降低温度)
    - 🔴红色区域 → ☀️暖风 (升高温度)
- **风量调节旋钮**：
    - “0”位为关闭风扇；数字越大风力越强。
### 4.2 AUTO模式 (自动空调)
按下“AUTO”按钮，系统将根据设定温度自动调节风量与出风模式。您只需设定目标温度即可。
## 五、安全辅助系统
### 5.1 ESP车身稳定系统 (默认开启)
ESP在每次启动时默认开启。如需手动关闭（如脱困时），请按以下步骤：
1.  点击中控屏【⚙️设置】→【驾驶辅助】→【ESP开关】→选择【关闭】。
2.  仪表盘会亮起 ESP OFF ⚠️指示灯。
3.  脱困完成后建议立即重新开启ESP以确保行车稳定。
### 5.2 TPMS胎压监测系统
当仪表盘亮起胎压报警图标时：
1.  立即安全靠边停车检查轮胎外观是否明显漏气或扎钉。
2.  如外观无异常，可通过中控屏查看各轮胎实时气压值：
    【⚙️设置】→【车辆信息】→【胎压监测】
3.  补充气压至标准值后，需在中控屏选择【胎压复位】以消除报警图标。

# 纵横G700 动力参数介绍
  
更新时间：2026年07月12日18时29分50秒   

## 一、动力总成概述
纵横G700搭载了品牌最新的 **“鲲鹏”系列混合动力系统**，旨在平衡强劲性能与燃油经济性。该系统由一台高效内燃机与前后双电机组成，提供多种驾驶模式以适应不同路况。
## 二、核心动力参数
| 项目 | 参数 |
| :--- | :--- |
| **发动机类型** | 2.0T 涡轮增压直喷发动机 |
| **发动机最大功率** | 200 kW (272 Ps) / 5500 rpm |
| **发动机峰值扭矩** | 400 N·m / 2000 - 4000 rpm |
| **前电机最大功率** | 150 kW (204 Ps) |
| **前电机峰值扭矩** | 320 N·m |
| **后电机最大功率** | 200 kW (272 Ps) |
| **后电机峰值扭矩** | 410 N·m |
| **系统综合最大功率** | **550 kW (748 Ps)** |
| **系统综合峰值扭矩** | **950 N·m** |
## 三、传动与驱动系统
-   **变速箱类型**: E-CVT无级变速系统（集成双电机控制器）
-   **驱动形式**: **智能电四驱**
    -   支持前后轴扭矩独立分配
    -   可实现从纯前驱到纯后驱的多种驱动模式切换
-   **百公里加速时间**: **3.9秒**
## 四、电池与续航
-   **电池类型**: 三元锂电池（高能量密度）
-   **电池容量**: 45 kWh
-   **纯电续航里程 (CLTC)**: ≥200 km
-   **综合续航里程 (满油满电)**: >1200 km
-   **快充时间**: SOC从20%充至80%约需25分钟
## 五、能耗表现
-   **最低荷电状态油耗 (WLTC)**: <7.5 L/100km
-   **综合能耗表现**: NEDC工况下综合油耗低至1.5 L/100km（有充电条件时）
## 六、驾驶模式与特色功能
1.  **[EV]纯电模式**: 
    -   适合城市通勤，零油耗，安静平顺。
2.  **[HEV]混动模式**:
    -   系统自动协调发动机与电机工作，兼顾性能与经济性。
3.  **[SPORT]运动模式**:
    -   发动机持续介入，电机全功率输出，油门响应极为灵敏。
4.  **[TERRAIN]越野模式**:
    -   针对非铺装路面优化扭矩分配，提升脱困能力。
5.  支持外放电功能（V2L），可满足户外用电需求。

# 山海炮 Hi4-T 操作指南
  
更新时间：2026年07月10日21时22分45秒   

## 一、车辆概述
山海炮 Hi4-T 是长城汽车旗下高端皮卡系列，搭载 **Hi4-T 越野超级混动架构**，结合2.0T涡轮增压发动机与P2电机，匹配9HAT混动专用变速箱，实现油电协同驱动。
### 核心参数
- **动力系统**：2.0T发动机 + P2电机
- **变速箱**：9HAT混动专用自动变速箱
- **电池类型**：三元锂电池（支持外放电）
- **驱动形式**：智能电四驱（可切换两驱/四驱）
- **悬挂系统**：前双叉臂独立悬架 + 后多连杆非独立悬架

## 二、驾驶模式选择
### 1. 模式切换旋钮
位于中控台中央，提供以下模式：
| 模式 | 适用场景 | 说明 |
|------|----------|------|
| **纯电 EV** | 城市通勤、短途行驶 | 仅电机驱动，零油耗、安静 |
| **混动 HEV** | 日常综合路况 | 系统自动分配油电动力 |
| **智能模式** | 全场景自适应 | AI学习驾驶习惯，自动切换策略 |
| **运动 SPORT** | 超车、山路、性能需求 | 发动机常启，动力响应最快 |
| **雪地 SNOW** | 冰雪湿滑路面 | ESP介入更积极，扭矩输出平缓 |
| **沙地 SAND** | 沙漠、沙滩行驶 | TCS放宽限制，保持轮端扭矩 |
| **泥地 MUD** | 泥泞沼泽路段 | ESC部分关闭，允许适度打滑脱困 |
| **岩石 ROCK**（4L） | 攀爬、大坡度越野 |需挂入N挡并按下4L按键 |
### ⚠️ EV/HEV切换提示
- EV模式下电量低于15%时自动切入HEV
- HEV模式下深踩油门会自动启动发动机补充动力

## 三、启动与熄火
### ✅正确启动流程
1.  携带智能钥匙进入车内（或手机蓝牙钥匙已连接）
2.  踩住刹车踏板
3.  按下方向盘右侧的“START/STOP”按键
4.  仪表盘显示“READY”即启动成功（纯电/混动均可）
### ❌熄火流程
1.  车辆停稳后挂入 P挡
2.  再次按下“START/STOP”按键熄火
3.  若需长时间停放，建议关闭所有用电设备后再熄火

## 四、换挡操作（怀挡式电子档杆）
位于方向盘右后方拨杆：
| 操作方式         |    挡位    |
| ---------------- | :--------: |
|   向上拨动       |     R (倒挡)    |
|   向下拨动       |     D (前进挡)    |
|   侧面按钮 +向下拨动      |     M (手动模式) / L(低速爬行)    |
|   顶端按钮按下       |     P (驻车挡)    |
> 💡 D/R切换时无需按任何按钮直接拨动即可；P挡需按压顶端按钮。

## ⚡五、能量管理与充电
### 🔌充电方式
1. **交流慢充（家用桩）**
   -  使用随车赠送的7kW家用充电桩或便携式充电枪
   -  充满约需6~8小时（视电池容量）
2. **直流快充（公共桩）**
   -  支持国标直流快充接口
   -  30%~80%约需30~40分钟
3. **对外放电 V2L/V2V**
   - V2L：通过放电枪为电器供电（最大3.3kW）
   - V2V：可为其他新能源车应急补电
### 🔋能量回收调节
-   通过方向盘左侧拨片调节回收强度：
- `+`拨片：减弱回收力度（滑行更顺滑）
- `-`拨片：增强回收力度（单踏板感更强）

## 🛣️六、越野辅助功能
### 📌差速锁控制
位于中控台物理按键：
- `前桥差速锁` —极端脱困时使用  
- `后桥差速锁` —中度以上越野推荐先锁后桥  
> ⚠️必须在低速四驱(4L)模式下且车轮几乎静止时操作。
### 📌蠕行模式 /坦克转弯 /透明底盘 
通过中控屏 →「越野信息」菜单开启：
- `蠕行模式`：设定速度后车辆自动控制油门刹车通过复杂地形  
- `坦克转弯`：制动内侧后轮大幅减小转弯半径  
- `透明底盘`：显示车底影像辅助避障

## 🛡️九、安全注意事项 
1️⃣ 【高压安全】严禁私自拆卸橙色高压线束；发生碰撞后请勿触摸损坏的高压部件。  
2️⃣ 【涉水注意】最大涉水深度800mm；涉水后及时清理底盘并检查电池包密封状态。  
3️⃣ 【拖车规范】必须使用平板拖车或抬起驱动轮拖运；禁止直接牵引以免损坏电机。  
4️⃣ 【紧急断电】如遇异常情况可长按 START/STOP键3秒强制切断高压系统。  
5️⃣ 【儿童安全】后排配备ISOFIX接口；行驶中确保儿童锁处于正确位置。  
6️⃣ 【露营用电】使用V2L外放电时确保放电枪完全插入且远离水源。

# 五菱荣光小卡 操作指南
  
更新时间：2026年07月20日12时01分54秒   

## 一、车辆基本介绍
五菱荣光小卡是一款微型货车，主要分为单排座（2座）和双排座（5座）两种版本。本指南适用于搭载汽油发动机的常规型号。
---
## 二、启动与熄火
### 1. 启动前检查
- **绕车一周**：检查轮胎气压、外观有无异常、车底有无漏油漏水。
- **调整座椅与后视镜**：确保能舒适地踩下离合/刹车/油门，并看清后方路况。
- **系好安全带**：驾驶员和副驾驶必须佩戴。
### 2. 启动步骤
1.  确认挡位处于 **空挡**（手动挡车型）。
2.  插入钥匙，拧至 **“ON”** 位置，等待仪表盘自检灯熄灭（约2-3秒）。
3.  踩下 **离合器踏板**（安全习惯）。
4.  将钥匙继续拧至 **“START”** 位置，发动机启动后立即松手。
5.  怠速热车30秒至1分钟（冬季可适当延长），待转速稳定后起步。
### 3. 熄火步骤
1.  车辆停稳后，拉紧手刹。
2.  将挡位挂回 **空挡**。
3.  关闭所有用电设备（空调、大灯等）。
4.  将钥匙拧回 **“LOCK”** 位置并拔出。
> ⚠️ **注意**：长时间高速行驶或重载后，建议怠速运转1-2分钟再熄火，以保护涡轮增压器（如配备）或让发动机均匀散热。
---
## 三、驾驶操作
### 1. 起步
1.  踩死离合器。
2.  挂入 **1挡**。
3.  松开手刹。
4.  缓慢抬起离合器至半联动点（车身轻微抖动），同时轻踩油门。
5.  完全松开离合器，平稳起步。
### 2. 换挡时机 (参考)
| 挡位 | 推荐车速 | 用途 |
| :--- | :--- | :--- |
| **1挡** | 0 -15 km/h | 起步、爬陡坡 |
| **2挡** |15 -30 km/h |低速行驶、市区跟车 |
| **3挡** |30 -50 km/h |城市道路 |
| **4挡** |50 -70 km/h |郊区、快速路 |
| **5挡** |70 km/h以上 |高速公路巡航 |
> ⚠️ **注意**:荣光小卡通常没有转速表或仅有简易指示。换挡应以车速和发动机声音为准：当发动机声音变得沉闷时降档；变得高亢嘶吼时升档。载重时应适当提高换挡转速。
### 3.倒车
-   必须在车辆完全停稳后挂入倒挡。
-   部分车型倒挡需要按下换挡杆上的锁止环或向下按压换挡杆才能挂入。
-   倒车时控制车速，多利用后视镜观察后方障碍物。
### 4.刹车技巧
-   提前预判，避免急刹。重载时刹车距离会明显增加。
-   下长坡时使用发动机制动（挂低档位），避免长时间踩刹车导致刹车过热失效。

## 四、装载与货物运输
### 1 .核定载质量
-   严格遵守车门上铭牌标注的核定载质量。超载会严重影响制动性能和悬挂寿命,且违法。
### 2 .货物固定
-   使用绳索或紧固带将货物牢固捆绑在货箱上,防止行驶中移位或掉落。
-   对于较高或较轻的货物,需特别注意防风防侧倾。
### 3 .重心控制
-   重的货物放在货箱靠前(靠近驾驶室)的位置,有助于保持前轮抓地力,改善操控稳定性。
-   避免偏载(一侧过重),会导致车辆跑偏甚至侧翻风险增加。

## 五 、仪表盘指示灯解读
|指示灯图标含义|颜色|建议措施|
|---|---|---|
|发动机故障灯(像水龙头)|黄色/橙色|尽快检修,可能影响动力和排放|
|机油压力灯(像油壶)|红色|立即停车熄火!检查机油量|
|充电指示灯(像电池)|红色|发电机不发电,尽快维修|
|ABS故障灯(圆圈+ABS)|黄色|防抱死系统故障,可正常刹车但无防抱死功能|
|安全气囊指示灯(坐着的人+球)|红色/黄色||气囊系统故障|
>⚠️重要:任何红色指示灯亮起都应引起高度重视;黄色指示灯表示需要尽快检查维修;绿色/蓝色为功能指示(如远光灯)。

## 六 、日常维护与保养建议
### 定期检查项目(每周)
-**轮胎气压**:标准值通常在前轮约3 .0Bar ,后轮约4 .0Bar (具体看油箱盖内侧标签)。
-**机油液位**:冷车状态下拔出机油尺擦净再插回查看液面应在上下刻度之间 。
-**冷却液液位**:观察副水壶液面应在MAX-MIN之间 。
-**灯光**:所有外部灯光是否正常工作 。
### 定期更换项目(参考里程)
-**机油及机滤**:每5000公里或6个月 。
-**空气滤清器**:每10000公里清洁 ,20000公里更换 。
-**变速箱油 /差速器油**:每40000公里左右 。
-**火花塞**:每30000公里左右 。
-**制动液**:每两年或40000公里 。

## 七 、常见问题与应急处理
### 问题:方向盘沉重 /转向异响
可能原因:转向助力油不足 (液压助力车型) ;轮胎气压过低 。检查并补充助力油或充气 。
### 问题:离合器踏板变软 /挂挡困难
可能原因:离合器液压系统漏油 ;离合器片磨损 。需进厂检修 。
### 问题:水温过高 (仪表盘红灯亮)
应急措施:
1 .立即靠边停车 ,不要立即熄火 !保持怠速运转 。
2 .打开引擎盖帮助散热 (注意防烫 ) 。
3 .检查冷却液是否泄漏 ,若不足待降温后补充 。切勿在高温时打开水箱盖 !
---
## 八 、安全驾驶提示
1.**盲区大**:货箱较高较长 ,右后方及正后方盲区较大 ,变道转弯需格外小心并多看后视镜 。
2.**雨天慢行**:空载时后轮抓地力差 ,容易打滑甩尾 ;重载时刹车距离长 。
3.**夜间行车**:原厂大灯亮度一般 ,对向有车时及时切换近光灯 。注意观察路面坑洼以防颠簸损伤货物或悬挂 。
4.**切勿客货混装**:双排座版本后排严禁超员且不得堆放影响驾驶视线及安全的物品 。

# 比亚迪 唐新能源 保养指南
  
更新时间：2026年07月11日07时30分09秒   

## 一、保养周期总览
| 保养项目 | 周期 |
|---------|------|
| 首次保养 | **购车后6个月或HEV里程3,500公里**（以先到为准） |
| 常规保养 | **每12个月或HEV里程7,500公里**（以先到为准） |
| 整车质保 | 6年或15万公里 |
| 三电系统质保 | **终身保修**（首任车主，需按手册定期保养） |
> ⚠️ **注意**：唐新能源包含 DM-i、DM-p、EV 三种动力形式，以下内容主要针对 DM-i/DM-p（混动车型），EV纯电版部分项目有所不同。
---
## 二、各阶段保养项目明细
### （一）每次常规保养（每12个月/HEV 7,500km）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查并补充冷却液
- [ ] 检查制动液液位及品质
- [ ] 检查空调滤芯（视情况更换）
- [ ] 检查轮胎胎压及磨损情况
- [ ] 检查底盘及悬挂系统
- [ ] OBD车载诊断系统检测
- [ ] BMS电池管理系统检测
- [ ] **动力电池均衡检测**
### （二）每2年或HEV行驶20,000km
包含常规保养全部项目，另增加：
- [ ] **更换空调滤芯**
- [ ] **更换空气滤清器**
- [ ] **更换燃油滤清器**（DM-i/DM-p车型）
- [ ] **检查高压线束及接插件**
- [ ] **检查驱动电机及控制器**
- [ ] **轮胎换位**
### （三）每4年或HEV行驶40,000km
包含以上全部项目，另增加：
- [ ] **更换制动液**
- [ ] **更换火花塞**（DM-i/DM-p车型）
- [ ] **更换变速箱油/减速器油**
- [ ] **清洗节气门**（视情况）
- [ ] **检查并清洁散热器**
### （四）每6年或HEV行驶60,000km
包含以上全部项目，另增加：
- [ ] **更换冷却液**
- [ ] **更换正时皮带/附件皮带**（如有）
- [ ] **全面检查高压系统绝缘性**
- [ ] **建议更换轮胎**

# MINI 汽车操作指南
  
更新时间：2026年07月12日17时10分01秒   

## 一、启动与熄火
### 启动车辆
1. **智能钥匙**：确保智能钥匙在车内。
2. **踩下刹车**：右脚踩住刹车踏板。
3. **按下启动按钮**：按下中控台或方向盘旁的“START/STOP”按钮。
4. **检查仪表盘**：确认所有指示灯正常亮起后熄灭，发动机运转平稳。
### 熄火车辆
1. **挂入 P 挡**：将挡位挂入“P”（驻车挡）。
2. **拉起电子手刹**（如有）。
3. **按下 START/STOP 按钮**：发动机熄火。
4. **关闭所有用电设备**（空调、灯光、音响等）。
> **注意**：MINI 部分车型支持无钥匙启动，请确保钥匙始终随身携带。

## 二、挡位操作
| 挡位 | 名称 | 使用场景 |
|------|------|----------|
| P | 驻车挡 | 长时间停车时使用 |
| R | 倒车挡 | 倒车时使用 |
| N | 空挡 | 短暂停车（如等红灯）或拖车时使用 |
| D | 前进挡 | 正常行驶时使用 |
| S / Sport | 运动模式 | 需要更强动力响应时使用 |
- **换挡拨片**（部分车型）：在 D/S 挡下可通过方向盘后的拨片手动升降挡位。
- **自动启停**：默认开启，深踩刹车时发动机自动熄火，松开刹车自动重启。可按下中控台 `A OFF` 按钮关闭此功能。

## 三、灯光与雨刮
### 灯光控制
- **自动大灯**：旋钮调至 `AUTO`，车辆根据环境光线自动开关近光灯。
- **近光灯 / 示宽灯**：手动旋转至对应图标。
- **远光灯**：向前推动左侧拨杆；向后拉为闪灯。
- **转向灯**：向上/向下拨动左侧拨杆。
- **雾灯**：在大灯开启状态下，旋转或按压雾灯开关。
### 雨刮控制
- **前雨刮**：
    - `OFF`：关闭
    - `INT` / `AUTO`：间歇/自动感应模式
    - `LO` / `HI`：低速/高速连续刮刷
    - `MIST` / `单次`：向上推一次单次刮刷
- **喷水清洗**：向驾驶员方向拉动雨刮拨杆。
- **后雨刮与喷水**（部分车型）：旋转雨刮拨杆末端旋钮控制。

## 四、空调与多媒体
### 空调系统
- **温度调节**：旋转左右两侧温度旋钮（双区独立空调）。
- **风量调节**：旋转风量旋钮或按 +/- 。
- **风向选择**：按对应按键选择吹面、吹脚、除雾等模式。
- **A/C**:  开启压缩机进行制冷或除湿。
- **MAX A/C**:  快速最大制冷。
- **前后窗除雾/加热**:  按对应图标按键。
### 多媒体系统（MINI Connected）
1.  通过中央圆形屏幕操作导航、音乐、电话等功能。
2.  支持 Apple CarPlay / Android Auto (部分车型需选装)。
3.  可通过方向盘右侧多功能按键或语音控制进行操作。

## 五、驾驶辅助功能
|功能名称|操作方法|
|---|---|
|定速巡航 (CC)|方向盘左侧按键开启 SET +/-调节速度 CANCEL取消|
|限速器 (LIM)|按 LIM键设定最高速度限制|
|车道偏离警告 (LDW)|通过中控菜单开启/关闭 (部分车型)|
|倒车影像 +雷达|挂入 R档自动激活|
> ⚠️驾驶辅助仅为辅助功能，驾驶员始终对车辆安全负全责。

## 六、加油与保养提示
### 加油步骤
1.  车辆解锁后按压油箱盖右侧即可打开外盖。
2.  逆时针拧开内盖并挂在盖上挂钩处。
3.  加注推荐标号汽油 (通常为95号及以上)。
4.  加满后拧紧内盖至“咔哒”一声，关好外盖。
### 保养提示复位 (示例)
1.  接通电源(不启动发动机)。
2.  连续按压仪表盘左下角复位按钮直到出现保养菜单。
3.  选择需要复位的项目并长按确认。
>具体复位方式因年款不同略有差异,请参考随车手册。

## 七、常见问题处理
### 仪表盘警示灯亮起怎么办？
1.  红色警示:立即安全停车并联系救援或维修站检查。例如:制动系统故障,机油压力过低,冷却液温度过高,安全气囊故障等。切勿继续行驶!
2.  黄色警示:尽快安排检修。例如:轮胎气压不足,发动机管理系统故障,灯光故障等。可谨慎行驶至最近维修站检查处理;若伴随性能明显下降建议拖车前往维修站以免造成更大损坏;切勿忽视任何黄色警示灯!
3绿色/蓝色指示灯为正常功能指示无需担心例如远光灯开启状态指示等;白色指示灯通常为辅助功能状态指示如定速巡航待命状态等;请放心使用相关功能即可无需特别处理!若不确定具体含义请查阅随车用户手册获取详细说明及应对措施建议!
### 轮胎失压怎么办？
1 MINI配备胎压监测系统(TPMS)当胎压异常时会点亮黄色感叹号图标并显示具体哪个轮胎气压偏低建议尽快检查并补充气压至标准值(标准值标注在驾驶侧门框标签上);若补气后警示依然存在需检查轮胎是否有扎钉破损等情况必要时更换备胎(如有配备)或联系道路救援服务进行进一步处理以确保行车安全避免高速行驶导致爆胎风险增加!若补气后恢复正常则无需担心继续正常行驶即可但建议定期检查胎压保持标准值以延长轮胎寿命并提升燃油经济性及操控稳定性!
### 电瓶亏电无法启动怎么办？
1如果按下启动按钮仪表盘变暗或无反应且无法听到启动机转动声音则很可能是12V蓄电池电量耗尽导致无法正常启动车辆此时可尝试搭电启动:准备一根搭电线将救援车电瓶正极(+)与本车电瓶正极(+)连接再将救援车电瓶负极(-)与本车车身金属搭铁点(如发动机吊耳减震塔顶螺栓等非电瓶负极直接连接以避免产生火花引发危险!)连接好后进入本车尝试正常启动流程即可成功着车后先拆除负极线再拆除正极线注意顺序不可颠倒以免短路损坏电器设备!若无搭电线或不确定操作方法请直接拨打道路救援电话寻求专业帮助避免因操作不当造成额外损坏风险!

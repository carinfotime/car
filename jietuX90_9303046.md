# 捷途X90 操作指南
  
更新时间：2026年07月13日12时36分19秒   

## 一、车辆启动与熄火
### 启动发动机
1. **智能钥匙启动**
   - 确保智能钥匙在车内
   - 踩下制动踏板（自动挡）或离合器踏板（手动挡）
   - 按下一键启动按钮，发动机启动
2. **遥控钥匙应急启动**
   - 如智能钥匙电量不足，将钥匙贴近一键启动按钮
   - 踩下制动踏板，按下一键启动按钮
### 熄火
- 将车辆完全停稳，挂入P挡（自动挡）或空挡（手动挡）
- 拉紧电子手刹
- 按下一键启动按钮关闭发动机
---
## 二、挡位操作（自动挡）
| 挡位 | 说明 |
|------|------|
| **P** | 驻车挡，停车时使用 |
| **R** | 倒车挡，车辆完全停稳后挂入 |
| **N** | 空挡，短暂停车或拖车时使用 |
| **D** | 前进挡，正常行驶使用 |
| **S** | 运动模式，动力响应更积极 |
| **M** | 手动模式，可通过换挡拨片或挡杆手动加减挡 |
> **注意**：行驶中请勿挂入R挡或P挡，以免损坏变速箱。
---
## 三、方向盘与仪表盘
### 方向盘调节
- 支持上下、前后四向调节
- 调节手柄位于方向盘下方，解锁后调整至合适位置，锁紧即可
### 仪表盘信息
- **左侧**：转速表 / 能量回收指示
- **右侧**：车速表
- **中央显示屏**：可切换显示油耗、胎压、导航、行驶里程等信息
- **指示灯说明**：
  - 🟢 绿色：功能正常开启（如近光灯、定速巡航）
  - 🟡 黄色：警告提示（如胎压低、发动机故障）
  - 🔴 红色：紧急故障（如制动系统、安全气囊）
---
## 四、灯光操作
| 操作 | 功能 |
|------|------|
| 旋钮至 **OFF** | 关闭灯光 |
| 旋钮至 **AUTO** | 自动大灯（推荐） |
| 旋钮至 **示宽灯** | 示宽灯/尾灯亮起 |
| 旋钮至 **近光灯** | 近光灯亮起 |
| 向前推拨杆 | 远光灯常亮 |
| 向后拉拨杆 | 远光灯闪烁（超车示意） |
| 拨杆向上/下 | 左/右转向灯 |
### 雾灯
- 近光灯开启状态下，旋转雾灯开关开启前/后雾灯
---
## 五、雨刮器操作
| 操作 | 功能 |
|------|------|
| 向上推一挡 | 单次刮刷 |
| 向下推一挡 | 间歇刮刷（可调节速度） |
| 向下推两挡 | 低速连续刮刷 |
| 向下推三挡 | 高速连续刮刷 |
| 向驾驶员方向拉 | 前挡风玻璃喷水+刮刷 |
| 旋钮调节 | 后雨刮开关及喷水 |
---
## 六、空调与多媒体系统
### 空调控制
- **温度调节**：旋转温度旋钮（蓝→冷，红→热）
- **风量调节**：旋转风量旋钮
- **模式选择**：吹面/吹脚/除霜/吹面+吹脚
- **A/C按钮**：开启空调压缩机（制冷/除湿）
- **内外循环**：内循环（快速降温/防尘） / 外循环（换新鲜空气）
- **自动空调**：按下 AUTO 键，自动调节温度与风量
### 多媒体系统（中控大屏）
- **主页**：导航、音乐、车辆设置、应用
- **导航**：支持在线/离线导航，可通过语音控制
- **蓝牙连接**：设置 → 蓝牙 → 配对手机
- **CarPlay / 百度CarLife**：通过USB连接手机投屏
- **语音控制**：说出“你好，捷途”或按下方向盘语音键唤醒
---
## 七、驾驶辅助功能
### 定速巡航
1. 车速达到30km/h以上
2. 按下 **CRUISE** 键
3. 按 **SET/-** 设定当前速度为巡航速度
4. 按 **RES/+** 恢复/加速，按 **SET/-** 减速
5. 踩刹车或按 **CANCEL** 取消巡航
### 电子手刹（EPB）与自动驻车（Auto Hold）
- **电子手刹**：向上拉起启用，向下按压解除
- **Auto Hold**：开启后，车辆停稳自动驻车，踩油门自动解除
### 360°全景影像
- 挂入R挡自动激活
- 也可通过中控屏手动开启
- 可切换视角：前/后/左/右/鸟瞰
---
## 八、座椅与空间
### 座椅调节
- **主驾座椅**：电动调节（前后、高低、靠背角度）
- **副驾座椅**：手动/电动调节（视配置而定）
- **后排座椅**：支持前后滑动及靠背角度调节
- **第三排座椅**（7座版）：可折叠放平
### 座椅加热/通风
- 通过中控屏或物理按键开启
- 通常分为3挡可调
---
## 九、天窗与车窗
### 天窗
- **上掀**：按天窗开关前端
- **滑动开启**：向后拉动开关
- **关闭**：向前推动开关
- **遮阳帘**：手动拉合或电动控制
### 车窗
- 主驾侧：一键升降（四门）
- 其他车门：手动升降
- 锁车后长按锁车键：自动关闭所有车窗
---
## 十、加油与充电
### 燃油版
- **燃油标号**：建议使用92号及以上无铅汽油
- **油箱盖**：车辆解锁状态下按压油箱盖即可打开
- **油箱容积**：约55L
### 混动/插电混动版
- **充电口**：位于车身左/右后侧（视车型）
- **支持**：家用220V交流充电 / 公共充电桩
- **充电时间**：视电池容量及充电功率而定

## 十二、应急处理
### 车辆无法启动
- 检查智能钥匙是否在车内
- 检查是否踩下制动踏板
- 检查蓄电池电量是否充足
- 尝试应急启动模式（钥匙贴近启动按钮）
### 轮胎气压报警
- 中控屏查看具体轮胎气压值
- 及时补充至标准气压（通常2.3~2.5 bar）
- 如持续报警，检查轮胎是否破损
### 拖车注意事项
- 自动挡：挂N挡，车速不超过50km/h，拖车距离不超过50km
- 建议使用平板拖车运输

## 十三、智能钥匙功能
| 操作 | 功能 |
|------|------|
| 短按锁车键 | 锁车 |
| 短按解锁键 | 解锁 |
| 长按锁车键 | 关闭所有车窗及天窗 |
| 长按解锁键 | 打开所有车窗 |
| 长按后备箱键 | 开启后备箱 |
| 遥控启动 | 先锁车，再长按启动键 |

## 十四、安全注意事项
1. **安全带**：所有乘员必须系好安全带
2. **儿童安全**：12岁以下儿童应使用儿童安全座椅，并坐在后排
3. **驾驶分心**：行驶中请勿操作中控屏或手机
4. **载重限制**：请勿超过车辆最大允许总质量
5. **改装**：未经授权改装可能影响车辆安全及保修
6. **冬季驾驶**：冰雪路面请使用雪地模式（如有）或谨慎驾驶

# 理想MEGA 用户操作指南
  
更新时间：2026年07月15日11时44分54秒   

## 一、车辆基本操作
### 1. 启动与熄火
- **启动**：携带钥匙进入车辆，踩下制动踏板，车辆即可上电 Ready。
- **熄火**：车辆停稳后，按下挡杆顶端的“P”挡按键，或直接下车锁车，车辆将自动下电。
### 2. 挡位操作
- **前进 (D)**：踩住刹车，将右侧拨杆向下拨到底。
- **倒车 (R)**：踩住刹车，将右侧拨杆向上拨到底。
- **空挡 (N)**：踩住刹车，短按右侧拨杆顶端按钮。
- **驻车 (P)**：按下右侧拨杆顶端按钮。
### 3. 车门与侧滑门
- **前门**：常规拉手开启/关闭。
- **侧滑门（电动）**：
    - B柱、C柱上有物理按键。
    - 可通过中控屏控制开启/关闭。
    - 支持语音控制：“打开/关闭侧滑门”。
    - *注意*：侧滑门开启时需确保侧面无障碍物。
### 4. 后备箱
- **电动尾门**：
    - 可通过尾门上的物理按键开启/关闭。
    - 可通过中控屏或语音控制。
    - *脚踢感应*：携带钥匙在车尾下方做踢腿动作（位置通常在车牌下方偏左），可感应开启/关闭。
## 二、驾驶与辅助系统
### 1. AD Max（理想智能驾驶）
- **开启**：
    1.  在方向盘左侧功能区按下“智能驾驶”启动键（方向盘图标）。
    2.  仪表盘显示灰色车道线变为蓝色即激活。
    3.  再次按下该键或轻踩刹车可退出。
- **功能说明**：
    - LCC（车道居中辅助）：保持车道内行驶并跟随前车。
    - NOA（导航辅助驾驶）：设定导航后，车辆可自动变道、进出匝道。需在高精地图覆盖路段使用。
- *重要提示*：
    - L2级辅助驾驶，驾驶员需时刻手握方向盘并观察路况。
    - MEGA的智能驾驶系统会通过方向盘上的电容感应判断是否脱手。若长时间未握持，系统会发出警告并逐步降级。
### 2. ACC（自适应巡航）
- **激活**：向下拨动右侧换挡拨杆一次（D挡状态下）。
- **调节车速**：上下滚动方向盘左侧的滚轮。
- **调节跟车距离**：按下方向盘左侧的“跟车距离”调节键（通常为车辆+距离图标）。
### 3. HUD（抬头显示）
- MEGA配备大尺寸HUD，显示车速、导航信息、限速、智能驾驶状态等关键信息。
- *调整*：可在中控屏“设置 > HUD”中调节亮度、高度和倾斜角度。
## 三、座舱与娱乐系统
### 1. 中控屏与副驾屏
- **主界面**：显示地图导航、音乐播放等核心卡片。支持分屏操作。
- **快捷控制栏**（屏幕底部）：快速调节空调温度、风量、座椅加热/通风、除雾等。向左滑动可展开更多选项。
- **副驾屏**：主要为娱乐功能（视频、音乐），可通过蓝牙耳机独立连接使用。
### 2. “理想同学”语音助手
唤醒词：“理想同学”。
常用指令示例：
|  功能类别 |  指令示例 |
| :--- | :--- |
|  空调 | “打开空调”、“温度调到24度”、“风量大一点” |
|  座椅 | “打开座椅按摩”、“副驾座椅加热” |
|  车窗 | “打开所有车窗”、“关闭遮阳帘” |
|  导航 | “导航到朝阳公园”、“去最近的充电站” |
|  媒体 | “播放周杰伦的歌”、“下一集” |
|  车辆控制 | “打开侧滑门”、“打开后备箱” |
*注*：支持连续对话和可见即可说功能。
### 3. Magic Carpet™座椅
前排及第二排座椅支持多向电动调节：
- *通风/加热*：全系标配前排和第二排座椅通风加热。第三排仅加热。可在屏幕或语音控制。
- *按摩*：前排和第二排配备十点按摩功能。可在屏幕选择模式及强度。
## 四、充电与补能
### MEGA专属特性——5C超快充
理想MEGA基于800V高压平台打造，支持最高520kW的快充功率。
#### A. MEGA超充站充电流程
1.  将车辆驶入超充车位（建议倒车入库以方便插枪）。
2.  挂入P挡并熄火下电（部分站点支持即插即充）。
3.  从充电桩取下液冷超充枪头。枪头较重且带有冷却液管，请正确握持手柄插入快充口（位于左后翼子板处）。
4.  听到“咔哒”一声锁止声即连接成功。充电桩屏幕和车内仪表盘会显示充电状态及功率信息。（峰值功率可达520kW以上）。
5.  充电完成后按压枪头按钮解锁拔出并放回原位。
#### B.家用交流慢充
使用随车赠送的家用充电桩或国标交流充电枪连接右后翼子板的慢充口即可。充满时间约8~10小时左右。
#### C.V2L外放电功能
MEGA支持最大3kW的外放电功率：
1.  通过转接器连接至慢充口或车内12V电源接口处的专用外放电插座。（具体请参考随车附件说明）。
2.  在中控屏设置中启用“对外放电”。
3.  即可为电磁炉、投影仪等设备供电。（露营场景非常实用）。
## 五、日常维护与提示
### A.Tire Care轮胎养护建议
由于MEGA整备质量较大且采用低滚阻轮胎以优化续航：
1.**胎压监测**:定期检查胎压(标准值通常在车门B柱标签上)。冬季建议适当调高0~0,避免因低温导致胎压过低报警；夏季长途行驶前确保胎压在标准范围内以避免过热风险；
2.**磨损检查**:每行驶约一万公里建议进行轮胎换位(前后交叉)以保证均匀磨损延长使用寿命；
3.**更换周期**:若发现异常偏磨鼓包或扎钉严重应及时更换原厂规格轮胎以确保安全性能及续航表现不受影响；
> ⚠️注意:非原厂规格轮胎可能导致续航里程显著下降甚至影响底盘悬挂几何参数引发安全隐患！
### B.Wiper雨刮器维护技巧:
由于MPV车型前风挡面积巨大且迎风面大容易沾染虫胶鸟粪等污渍:
1.**清洁玻璃**:每月至少一次使用专用玻璃油膜去除剂彻底清洁前档玻璃表面油膜层；
2.**雨刮片寿命**:原厂雨刮片通常使用寿命约6个月至一年之间；当发现刮拭出现条纹异响或模糊不清时应立即更换否则夜间行车遇雨时眩光极其危险！
3.**冬季注意**:北方寒冷地区过夜停车务必竖起雨刮臂防止冻结粘连导致电机烧毁！
> 💡小贴士:若长期停放户外建议购买前档遮阳挡保护巨大仪表台延缓老化开裂风险同时降低夏季暴晒后车内温度！

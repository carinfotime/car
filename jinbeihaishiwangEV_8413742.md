# 金杯海狮王EV 技术参数介绍
  
更新时间：2026年07月19日15时21分07秒   

## 一、车型概述
金杯海狮王EV是华晨雷诺金杯旗下的一款纯电动轻型客车，基于经典的海狮平台打造，采用纯电驱动系统，主要面向城市物流、客运及商用出行场景。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| 驱动电机类型 | 永磁同步电机 |
| 最大功率 | **100 kW**（约136马力） |
| 最大扭矩 | **300 N·m** |
| 最高车速 | ≥ **100 km/h** |
| 0-50km/h加速时间 | ≤ **10秒** |

## 三、电池与续航
| 项目 | 参数 |

## 六、充电接口
交流充电接口（慢充）：国标 GB/T
直流充电接口（快充）：国标 GB/T
支持家用220V充电及公共充电桩充电。

# 图雅诺智蓝新能源 · 儿童乘车安全注意事项
  
更新时间：2026年07月12日15时31分52秒   

为了保障您在驾驶图雅诺智蓝新能源车型时，车内儿童的安全，请务必遵守以下事项：
## 一、上车前检查
1.  **确认儿童锁状态**：在关闭车门之前，请检查后排车门的**儿童安全锁**是否已拨至锁定位置。开启后，车门无法从车内打开，防止行驶中儿童误开车门。
2.  **清理周边障碍物**：确保车辆周围（尤其是盲区）没有儿童逗留或玩耍。
3.  **避免高温烫伤**：夏季车辆暴晒后，车内座椅、安全带金属扣及车窗玻璃温度较高。请先开窗通风降温，再让儿童上车。
## 二、行驶中安全
1.  **必须使用安全座椅/增高垫**：
    -   **12岁以下或身高不足1.5米**的儿童，严禁直接使用成人安全带。必须使用符合国家标准的**儿童安全座椅**或**增高垫**。
    -   **反向安装**：建议4岁以下婴幼儿使用反向安装的儿童安全座椅。
2.  **禁止坐副驾驶位**：
    -   **12岁以下儿童严禁坐在副驾驶位置。**
    -   即使副驾驶有安全气囊，弹出时的冲击力也会对儿童造成致命伤害（如颈椎骨折或窒息）。
3.  **全程系好安全带**：
    -   确保乘坐安全座椅的儿童已正确扣紧五点式安全带。
    -   使用增高垫的儿童，需确保肩带跨过肩膀中间而非颈部，腰带紧贴大腿根部而非腹部。
4.  **勿将头手伸出窗外**：
    -   **绝对禁止**将头、手或身体任何部位伸出天窗或侧窗。图雅诺智蓝新能源车型车窗面积较大，尤其要警惕此风险。
5.  **电动车窗防夹功能验证**：
    -   虽然本车配备车窗防夹功能，但仍需教育孩子不要随意触碰车窗开关。家长应开启“车窗锁止键”，防止后排儿童误操作。
## 三、停车与离车
1.  **熄火拔钥匙/携带钥匙离车时**
    -   **切勿将儿童单独留在车内！**
        -   *夏天*：车内温度可在10分钟内升至60°C以上，导致热射病死亡。
        -   *冬天*：熄火后空调停止供暖，车内会迅速降温导致失温危险。
        -   *意外启动风险*：好奇的儿童可能误触换挡旋钮或加速踏板（即使未上电），造成溜车碰撞事故。
2.  **离开前清点人数与物品**
    -   养成下车前回头检查后排座位的习惯（“伸手还要钱”口诀：身份证、手机、钥匙、孩子）。
3.  **后备箱禁止载人**
    -   **严禁任何人员（包括成人及儿童）进入后备箱区域。**
        -   *原因*：后备箱无通风设施且无逃生拉手（部分车型），一旦意外锁闭极易导致窒息死亡；同时追尾碰撞时毫无保护。
## 四、针对本车型的特殊提醒
| 项目 | 注意事项 |
| :--- | :--- |
| **纯电静音特性 (AVAS)** | EV模式低速行驶时声音极小。请牵好孩子的手通过停车场或小区道路；倒车时注意后方是否有蹲下玩耍的幼儿。 |
| **高压电池组 (底盘下方)** | 教育孩子不要敲击底盘或在车辆下方钻爬玩耍；如发生托底事故需立即检修电池组以防漏电起火风险。 |
| **充电口盖板 &充电枪线缆 (充电时)** | <ul><li>充电期间不要让小孩触碰充电枪插头金属端子(高压危险)。</li><li>不要让小孩拉扯拖拽充电电缆以防绊倒摔伤。</li></ul> |
## ⚠️紧急情况预案
-   【如果被锁在车内】教会大龄孩子按喇叭（断电状态下部分车型喇叭仍可工作）或拍打车窗呼救；但最根本措施仍是——家长永远不要将孩子遗忘在车里！

# 菱势电卡 驾驶注意事项
  
更新时间：2026年07月13日13时44分59秒   

## 一、启动与起步
1. **上电检查**：启动前确认仪表盘各项指示灯正常，尤其注意动力电池电量、故障灯及系统自检状态。
2. **钥匙操作**：将钥匙拧至“ON”档，等待仪表自检完成后再挂挡行驶，切勿直接快速启动。
3. **挡位确认**：确认挡位处于N挡（空挡）后再旋转启动开关，挂挡时务必踩住制动踏板。
4. **低速起步**：电卡电机扭矩大，起步时轻踩加速踏板，避免急加速导致轮胎打滑或货物移位。
## 二、行驶中注意事项
1. **能量回收**：菱势电卡具备能量回收功能，松开加速踏板即进入回收模式，可提前预判减速，减少刹车片磨损并回收电能。
2. **匀速行驶**：建议保持40-60km/h的经济时速，避免频繁急加速、急减速，有助于延长续航里程。
3. **爬坡与重载**：满载爬坡时保持匀速，避免中途停车再起步；下长坡时利用能量回收辅助制动，防止刹车过热。
4. **涉水行驶**：电动车电池及高压部件防水等级有限，遇积水路面应减速慢行，积水深度不得超过轮胎中心，切勿强行涉水。
5. **噪音注意**：纯电行驶噪音极低，行人和非机动车不易察觉，经过路口、人行道时请鸣笛或减速提醒。
## 三、充电注意事项
1. **充电环境**：选择干燥、通风、无易燃物的环境充电，避免在暴晒或雨雪天气下露天充电。
2. **充电枪操作**：先插充电枪再启动充电，充电结束后先停止充电再拔枪；插拔时保持枪头干燥清洁。
3. **电量管理**：建议电量剩余20%-30%时及时充电，避免过度放电（低于10%）损伤电池寿命；日常充电充至90%-95%即可，不必每次充满。
4. **快慢充选择**：日常优先使用慢充，延长电池寿命；紧急情况可使用快充，但连续快充后建议让电池冷却一段时间再行驶。
5. **充电异常**：充电过程中如发现异味、异响或充电桩报警，立即停止充电并联系专业人员检查。
## 四、电池与续航维护
1. **温度影响**：高温或低温环境会影响电池活性，夏季避免暴晒后立即快充，冬季建议热车2-3分钟后再行驶。
2. **长期停放**：若车辆长期不用，保持电量在50%-60%之间，每两周检查一次电量并及时补电。
3. **空调使用**：空调制冷/制热会显著消耗电量，建议合理使用，夏季可先开窗通风再开空调，冬季适当减少暖风档位。
4. **胎压检查**：定期检查轮胎气压，胎压不足会增加行驶阻力，导致电耗上升。
## 五、安全与应急
1. **高压安全**：严禁私自拆卸、改装高压部件；发生碰撞后如闻到焦味或看到烟雾，立即断电撤离并报警。
2. **紧急断电**：熟悉车辆紧急断电开关位置（通常在驾驶员座椅下方或仪表台附近），遇到紧急情况可快速切断高压电源。
3. **拖车注意**：电动车不建议随意拖车，如需救援请使用平板拖车，避免前轮或后轮着地拖动造成电机损坏。
4. **定期保养**：按保养手册要求定期到授权服务站检查高压系统、冷却液、减速器油等，切勿超期保养。
## 六、日常检查清单
| 检查项目 | 频率 | 要点 |
|---------|------|------|
| 轮胎气压与磨损 | 每周 | 胎压参照驾驶位门框标签值 |
| 灯光与信号 | 每日 | 前后大灯、转向灯、刹车灯 |
| 制动系统 | 每日 | 刹车踏板行程及制动效果 |
| 电量与续航 | 每日 | 规划好当日行驶里程 |
| 冷却液液位 | 每月 | 检查副水箱液位是否在刻度线内 |
| 高压线束 | 每季 | 检查是否有破损、松动 |

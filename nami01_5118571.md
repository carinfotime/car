# 东风纳米01 空间参数介绍
  
更新时间：2026年07月11日04时28分22秒   

东风纳米01作为一款纯电动小型车，在有限的车身尺寸内，通过精巧的设计实现了较为出色的空间利用率。以下是其核心空间参数与特点：
## 基础尺寸数据
| 项目 | 参数 |
| :--- | :--- |
| 车身长度 (mm) | 3715 |
| 车身宽度 (mm) | 1815 |
| 车身高度 (mm) | 1578 (含行李架) / 1530 (不含行李架) |
| **轴距 (mm)** | **2660** |
> **亮点说明**：2660mm的轴距在同级车型中属于领先水平，为车内纵向乘坐空间提供了良好基础。
## 乘坐空间
-   **前排空间**：头部空间充裕，得益于较高的车身设计（跨界风格），身高175cm的乘客头部余量约一拳以上。
-   **后排空间**：
    -   腿部空间：得益于长轴距，后排腿部空间约有一拳半至两拳（针对175cm乘客），不输于部分A级轿车。
    -   头部空间：后排头部空间约为一拳左右。
-   **座椅布局**：标准5座布局。后排座椅支持整体放倒。
## 储物与后备箱
-   **后备箱容积**：
    -   常规状态：约 **300升**（可容纳多个20寸登机箱及背包）。
    -   放倒后排后：可扩展至约 **1100升**（形成近乎纯平的空间，便于搬运大件物品或临时休息）。
-   **车内储物细节**：
    -   中控台下方设有镂空储物格。
    -   门板储物槽可放置水瓶及杂物。
    -   副驾手套箱、中央扶手箱等常规储物位置齐全。
## 综合评价
东风纳米01在“小车大空间”的理念下表现突出：
1.  **越级轴距**：2660mm的轴距是保证后排腿部空间的根本。
2.  **灵活多变**：后排座椅放倒后形成的平整大空间提升了实用性。
3.  **跨界优势**：较高的车身带来了优于传统轿车的头部空间表现。
对于城市通勤和家庭日常使用而言，纳米01的空间配置能够较好地满足4人舒适出行或2人带较多行李的需求。

# 钧天格瑞维亚 空间参数介绍
  
更新时间：2026年07月20日18时44分11秒   

## 车型概述
钧天格瑞维亚是一款基于丰田格瑞维亚（GRANVIA）平台打造的高端定制商务MPV，在保留原车优秀空间布局的基础上，进行了豪华升级与个性化改装。
## 整车尺寸参数
| 项目 | 参数 |
| :--- | :--- |
| 车长（mm） | **5175** |
| 车宽（mm） | **1995** |
| 车高（mm） | **1785**（含天线）/ **1765**（不含天线） |
| 轴距（mm） | **3060** |
| 前轮距（mm） | **1725** |
| 后轮距（mm） | **1726** |
| 整备质量（kg） |约 **2160 - 2190** （根据配置不同略有差异） |
## 内部乘坐空间
- **座椅布局**：通常提供 **2+2+3** 七座布局，部分定制版本可提供四座或六座方案。
- **第一排（驾驶舱）**
    -  头部空间充裕，驾驶视野开阔。
    -  中央通道宽敞，便于前排与后排沟通。
- **第二排（贵宾舱 / “老板位”）**
    -  采用独立航空座椅，支持电动调节、加热、通风、按摩及腿托功能。
    -  前后滑动距离大，可形成超大腿部空间。
    -  座椅宽度与包裹性极佳，长途乘坐舒适度极高。
- **第三排**
    -  得益于3060mm的超长轴距，第三排腿部空间依然可观，可满足成年人正常乘坐。
    -  头部空间无压抑感，侧窗面积大，通透性好。
    -  支持靠背角度调节及前后移动。
## 储物与行李厢空间
- **常规状态 (7座满员)**：行李厢纵深适中，可放置3-4个登机箱或若干背包。
- **第三排折叠/放倒后**：行李厢容积大幅扩展至约 **1000升以上** ，可容纳高尔夫球包、大型行李箱等大件物品。
- **车内储物细节**
    -   门板储物格：可放置水瓶、雨伞等杂物。
    -   中央扶手箱：深度可观，内置USB接口及无线充电板预留位。
    -   第二排座椅扶手内：隐藏式杯架与储物槽。
    -   第三排两侧：杯架与小型储物槽。
## 改装带来的空间变化
作为定制车型，钧天格瑞维亚的内部空间会根据选装配置有所调整：
1.  **隔断设计**：部分高配版本会在第二排前方加装液晶升降隔断墙或吧台。这会略微压缩第二排的腿部伸展极限长度，但提升了私密性与豪华感。
2.  **地板升级**：原车绒面地毯常被更换为游艇级柚木地板或大理石纹铝地板。地板厚度增加极小 (约2-3mm)，不影响头部空间净高；但脚感更平整易清洁。
3.  **顶棚造型**：星空顶或氛围灯顶棚会占用少量头部高度 (约10-20mm)，但由于原车高度充裕 (1785mm)，身高185cm以下的乘客仍不会感到压抑。
## “移动会客厅”体验总结
钧天格瑞维亚的核心优势在于：
> ✅ **3060mm超长轴距 + ≈1995mm宽体车身 = MPV中的“宽适标杆”**
> ✅ “2+2+3”布局下每一排都有尊严的乘坐体验
> ✅ 定制化内饰让“商务接待”与“家庭出行”无缝切换

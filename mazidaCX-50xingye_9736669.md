## 一、整车尺寸参数
| 项目 | 参数 |
|------|------|
| 车长（mm） | 4,785 |
| 车宽（mm） | 1,920 |
| 车高（mm） | 1,638 |
| 轴距（mm） | **2,815** |
| 前轮距（mm） | 1,659 |
| 后轮距（mm） | 1,662 |
| 最小离地间隙（mm） | **210** |
> **亮点解读：** CX-50行也拥有同级领先的轴距（2,815 mm），为宽敞的纵向乘坐空间奠定了基础。同时210 mm的高离地间隙赋予了它出色的通过性，兼顾城市与轻度越野场景。

## 二、乘坐空间
### （一）前排空间
- **头部空间：**  
  前排头部空间充裕，身高175 cm的乘客调整至合适坐姿后，头部距离车顶仍有约一拳以上的余量。
- **腿部及横向空间：**  
  前排座椅支持多向调节，横向宽度达到1,500 mm以上，主副驾之间互不干扰，肘部活动范围宽裕。
### （二）后排空间
- **腿部空间：**  
  得益于2,815 mm的长轴距，后排腿部空间表现突出。身高180 cm的乘客落座后，膝部距离前排椅背仍有约两拳以上的距离。
- **头部空间：**  
  后排顶部采用内凹设计优化头部余量，175 cm乘客头部距车顶约一拳左右。
- **横向宽度：**  
  后排可轻松容纳三名成年乘客，肩部横向宽度充足，长途乘坐不显拥挤。
---
## 三、储物与后备箱
### （一）后备箱容积
- **标准状态容积：**  
  后备箱常规容积约为 **435 L**（官方数据），可轻松放置多个28寸行李箱及随身行李。
- **扩展能力：**  
  后排座椅支持 **4/6比例放倒**，放倒后可形成近乎纯平的地台，最大装载深度超过1.8米，容积可扩展至约 **1,000 L以上**。
- **开口设计：**  
  后备箱开口宽大且门槛较低，便于搬运大件物品或重物。
### （二）车内储物细节
| 位置 | 特点 |
|------|------|
| **中央扶手箱** | 深度适中，可放置饮料瓶、证件等杂物 |
| **门板储物格** | 前后门板均可容纳500 ml水瓶及随身小物 |
| **手套箱** | 容积规整，可存放文件、手册等物品 |
| **杯架** | 前后排均配备杯架（前排中央通道2个+后排中央扶手2个） |
| **其他储物位** | 中控台下方的开放式储物槽、座椅背袋等 |

## 四、总结
马自达CX-50行也在紧凑型SUV中凭借超长轴距和合理的车内布局提供了越级的乘坐体验。无论是日常通勤还是家庭出游，其前后排腿部和横向空间都能满足需求；后备箱在标准状态下已足够实用，放倒座椅后更能应对搬家或户外装备运输等大件场景。整体而言，“人马一体”的设计理念在CX-50行也上不仅体现在操控层面，同样延伸到了对驾乘者空间的尊重与考量。

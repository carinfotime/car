# 睿立达V5 驾驶注意事项
  
更新时间：2026年07月12日10时08分06秒   

## 一、启动与预热
1.  **钥匙/启动按钮操作**：请确保智能钥匙在车内，踩下制动踏板后，按下一键启动按钮。若车辆无法启动，请检查钥匙电池或是否处于强电磁干扰区域。
2.  **仪表盘自检**：启动后，观察仪表盘指示灯是否正常熄灭。重点关注**发动机故障灯**、**电池灯**、**机油压力灯**及**胎压监测灯**。
3.  **冷车预热（冬季）**：冬季气温较低时，建议原地热车 **30秒至1分钟**，待转速表指针回落至正常怠速（约700-800转）后再缓慢行驶。避免长时间原地热车（超过3分钟），以免产生积碳。
## 二、行驶中关键操作
### 1. 变速箱使用（自动挡）
-   **D挡行驶**：正常行驶挂D挡。睿立达V5的变速箱换挡逻辑偏向经济性，如需超车或爬陡坡，可深踩油门触发降档或切换至S挡（运动模式）。
-   **N/P挡切换**：等红灯时间短（<30秒）建议D挡+刹车；时间长建议挂N挡拉手刹。停车熄火前务必挂入P挡。
-   **禁止空挡滑行**：自动变速箱严禁N挡滑行，否则会严重损坏变速箱。
### 2. 制动系统
-   **刹车距离预判**：睿立达V5车身较重（视具体配置），满载时刹车距离会延长。请保持比轿车更长的跟车距离。
-   **紧急制动辅助（EBA）**：紧急情况下请一脚将刹车踩到底，系统会自动加大制动力。切勿点刹。
-   **坡道起步辅助（HSA）**：在半坡起步时，松开刹车后有约2-3秒的驻车时间防止溜车，请在此时间内迅速踩下油门。
### 3. 转向与底盘
-   **方向盘打死限制**：尽量避免长时间将方向盘打死到底（如掉头、倒库），此举会加剧转向助力泵负荷和轮胎磨损。建议打死方向后回正少许角度。
-   **过减速带/烂路减速慢行**：该车型底盘调校偏舒适或偏硬朗（视具体版本），通过减速带或坑洼路面时务必低速通过（时速<20km/h），以保护悬挂系统和减震器。
## 三、特殊路况驾驶
| 路况类型 | 注意事项 |
| :--- | :--- |
| **雨天/湿滑路面** | -   降低车速，避免急加速、急刹车和急转弯。<br>-   开启雾灯或近光灯。<br>-   涉水深度不超过排气管高度（约半个车轮高度），匀速通过，熄火后切勿二次启动。 |
| **高速公路行驶** | -   保持安全车距至少100米。<br>-   避免长时间占用超车道。<br>-   注意横风影响，握紧方向盘。<br>-   胎压建议略高于标准值0.1-0.2bar以减少爆胎风险。 |
| **山路/连续弯道** | -   入弯前提前减速鸣笛。<br>-   长下坡路段使用低档位配合发动机制动（手动模式或L档），避免长时间踩刹车导致刹车过热失效。<br>-   注意对向车辆占道行驶。 |
## 四、载货与载人规范
1.  **核定载重限制**
    -   *总质量*：请查看车辆铭牌上的最大允许总质量 (GVWR)。
    -   *装载原则*：货物均匀分布并固定牢固；人员不得超过座位数且必须系安全带。
2.  **行李架使用**
    -   若睿立达V5配备行李架：
        -   *最大承载重量*通常为50kg左右。
        -   *高度限制*：装载后整车高度不得超过4米及当地法规限高。
        -   *风阻影响*：高速行驶时行李架会增加油耗和风噪。
## 五、日常检查与维护
### （一）出车前检查
1.  **轮胎外观与胎压**
2.  **灯光系统**
3.  **机油液位**
4.  **冷却液液位**
5.  **玻璃水余量**

## ⚠️六、安全警告与禁止事项
1. ❌严禁酒后驾车及疲劳驾驶
2. ❌严禁超速及非法改装电路/动力系统
3. ❌儿童乘车必须使用符合规定的儿童安全座椅并放置于后排位置
4. ❌发生事故后立即开启双闪警示灯并在后方规定距离放置三角警示牌撤离至护栏外报警求助

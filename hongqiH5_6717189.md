# 红旗H5 操作指南
  
更新时间：2026年07月15日14时09分01秒   

## 一、车辆启动与熄火
### 1.1 启动发动机
1.  确保智能钥匙在车内。
2.  踩下制动踏板（自动挡车型必须踩住刹车）。
3.  按下位于方向盘右侧或中控台上的 **“一键启动”** 按钮。
4.  仪表盘亮起，发动机启动。此时可松开制动踏板。
> **注意**：在寒冷天气下，建议先通电自检（不踩刹车按一次启动键），待仪表盘指示灯熄灭后再踩刹车启动。
### 1.2 熄火
1.  将车辆完全停稳，挂入 **P挡**（驻车挡）。
2.  拉起电子手刹（若有自动驻车功能，部分车型会自动拉起）。
3.  再次按下 **“一键启动”** 按钮，发动机熄火。
---
## 二、驾驶模式选择
红旗H5通常提供多种驾驶模式，可通过中控台或屏幕上的模式切换旋钮/按键进行切换：
- **经济模式 (ECO)**：适合城市拥堵路况，油门响应变缓，更省油。
- **舒适模式 (Comfort)**：默认模式，兼顾动力与平顺性。
- **运动模式 (Sport)**：油门响应更灵敏，换挡时机延后，动力更强。
- **雪地模式 (Snow)**：湿滑路面起步更平稳，防止打滑。
---
## 三、电子手刹与自动驻车
### 3.1 电子手刹
- **启用**：向上拨动位于挡杆附近的电子手刹按钮（P图标）。
- **解除**：踩下刹车后向下按压按钮；或直接挂D挡轻踩油门自动解除。
### 3.2 Auto Hold（自动驻车）
- **开启**：按下中控台上的 `Auto Hold` 按钮（指示灯亮起）。
- **使用**：等红灯时踩停车辆后松开刹车，车辆会自动保持静止；绿灯时轻点油门即可解除制动。
---
## 四、灯光操作
灯光控制拨杆位于方向盘左侧：
| 操作 | 说明 |
|------|------|
| AUTO | 自动大灯模式（推荐日常使用） |
| OFF | 关闭灯光 |
| -/-- | （示宽灯）位置灯 |
| （大灯图标） | 近光灯 |
| （向前推） | **远光灯常亮**（仪表盘显示蓝色远光图标） |
| （向后拉） | **闪远光灯**（超车或提醒前车） |
> - **转向灯**：拨杆上下拨动。
> - **雾灯**：通常在灯光旋钮上旋转至雾灯位置。
---
## 五、雨刮器操作
雨刮控制拨杆位于方向盘右侧：
- `MIST` / `单次刮刷`：向上推一下。
- `OFF`：关闭。
- `INT` / `AUTO`：间歇刮刷 / 自动感应雨量（根据雨量大小自动调节速度）。
- `LO` / `HI`：低速连续刮刷 /高速连续刮刷。
- `喷水清洗`：向驾驶员方向拉动拨杆。
---
## 六、空调与多媒体系统
### 6.1空调控制
可通过中控实体按键或触摸屏调节：
- `A/C` ：制冷开关。
- `AUTO` ：自动恒温空调。
- `温度调节` ：左右两侧旋钮分别控制主副驾温度。
- `风量调节` ：风扇图标旋钮或屏幕滑动条。
- `内外循环切换`：
    - *内循环* ：快速降温/隔绝外部空气污染；
    - *外循环* ：引入新鲜空气。
### 6.2多媒体系统
红旗H5搭载旗偲系统：
1.  主页界面可查看导航、音乐、天气等卡片信息。
2.  支持语音控制：“你好红旗” + “打开导航”、“播放音乐”、“调低温度”等指令。
3.  支持手机互联（CarPlay / CarLife），通过USB线连接扶手箱内的接口即可投屏使用导航和娱乐功能。

## 七、智能辅助驾驶功能
部分高配车型配备L2级辅助驾驶系统：
### 7.1自适应巡航 (ACC)
1.  按下方向盘左侧的巡航开关按钮进入待机状态；
2.  设定车速 (`SET/+`, `SET/-`)；
3.  调整跟车距离 (`距离调节`)；
4.  取消巡航 (`CANCEL`)；恢复巡航 (`RES`)；
> *注意*:驾驶员需始终手握方向盘并观察路况。
### 7,2车道保持辅助 (LKA)
默认开启。当车辆偏离车道且未打转向灯时,系统会轻微修正方向并发出警示音。可在设置菜单中关闭此功能(建议新手保留开启状态)。
---
## 八、加油与充电(混动版)
### 8,1燃油版
油箱盖位于车身左侧。按压油箱盖板即可打开(需解锁车门)。推荐加注92号及以上无铅汽油(具体参见油箱盖内侧标注)。
### 8,2混动版(H5 PHEV)
除加油外还需充电:
打开左侧翼子板处的充电口盖;
使用随车附带的交流充电枪连接家用插座或公共交流桩;
充满电约需4小时(视剩余电量而定);
充电期间仪表盘会显示进度;完成后拔出充电枪并关闭盖子即可出发行驶;

## 九、常见问题提示(Q&A)
Q:为什么锁不上车门?
A:检查是否所有车门都已关好;检查是否在车内留有另一把钥匙;检查是否处于N挡而非P挡;
Q:仪表盘出现黄色扳手图标?
A:提示需要保养了;请尽快联系授权经销商进行维护;
Q:如何重置胎压监测?
A:在中控屏设置中找到轮胎监测选项;确认胎压正常后点击复位即可消除报警;

# 金琥EV70 内饰参数介绍
  
更新时间：2026年07月19日13时59分27秒   

## 一、整体设计风格
金琥EV70采用简约科技风内饰设计，以实用性与舒适性为核心，兼顾商乘两用需求。车内空间布局合理，提供多种座椅组合方式。
## 二、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 高级织物 / 仿皮材质（视配置版本而定） |
| 座椅布局 | 5座（2+3）/ 7座（2+2+3）可选 |
| 前排座椅功能 | 手动调节（前后/靠背角度），部分高配版支持主驾电动调节 |
| 后排座椅 | 后排可整体折叠或按比例放倒，扩展储物空间 |
| 头枕 | 前排独立头枕，后排三头枕（5座版） |
## 三、仪表与中控
| 项目 | 参数 |
|------|------|
| 仪表盘 | LCD液晶组合仪表，显示电量、续航、车速等信息 |
| 中控屏幕 | **低配版**：无中控屏或基础收音机；**高配版**：8英寸/10英寸触控屏 |
| 多媒体系统 | USB接口 + AUX音频输入；高配支持蓝牙电话/音乐、手机互联（CarLife） |
| GPS导航 | **选装**或高配标配 |
## 四、空调系统
- **空调类型**：手动空调
- **后排出风口**：部分车型配备
- **PM2.5过滤**：基础过滤功能
## 五、方向盘与操控
- **方向盘材质**：PU塑料 / PVC包裹
- **方向盘调节**：手动上下调节
- **多功能按键**：高配版集成音量控制、蓝牙接听等快捷键
## 六、储物空间
- **手套箱**：副驾前方标准储物箱
- **门板储物格**：四门均有杯架及储物槽
- **中央通道**：杯架 + USB接口区域 + （部分车型）扶手箱
- **后备箱容积**：
    -  5座版常态容积约400L，放倒后排可扩展至约800L+
    -  7座版常态容积较小，放倒第三排后可大幅扩展
## 七、其他内饰配置
| 项目 | 说明 |
|------|------|
| USB充电口 |  前排1–2个USB接口；部分车型后排增设USB口 |
|  12V电源接口 |  前排点烟器接口（可转换供电） |
|  阅读灯 |  前排顶灯；部分车型后排增设阅读灯 |
|  遮阳板化妆镜 |  主副驾遮阳板均带化妆镜（副驾通常标配） |
|  内后视镜 |  手动防眩目内后视镜 |
## ⚠️注意事项
> -以上参数基于金琥EV70常见配置版本整理，具体配置可能因生产批次、选装包及地区差异有所不同。
> -建议以实车或官方最新配置表为准。

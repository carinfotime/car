## 凯翼炫界 - 空间参数介绍
  
更新时间：2026年07月17日17时49分19秒   

### 车身尺寸与轴距
-   **车长 (mm)：** 4400
-   **车宽 (mm)：** 1831
-   **车高 (mm)：** 1653
-   **轴距 (mm)：** 2632
### 内部乘坐空间
-   **前排头部空间：** 充裕，身高175cm的乘客头部距车顶约有一拳余量。
-   **后排腿部空间：** 得益于2632mm的轴距，后排腿部空间表现良好，约有一拳半至两拳的距离。
-   **后排头部空间：** 经过优化设计，后排头部空间同样宽裕，无明显压抑感。
-   **后排中央地台：** 近乎纯平设计，提升了中间乘客的乘坐舒适性。
### 储物能力
#### 后备箱容积
-   **标准容积：** 约400升（常规状态下可满足日常出行及超市购物需求）。
-   **最大扩展容积：** 后座椅支持4/6比例放倒，放倒后可形成较为平整的空间，最大容积可扩展至约1100升以上（具体视车型配置略有差异），便于搬运大件物品。
#### 车内储物格
车辆内部设计了多处实用的储物空间：
| 位置 | 用途 |
| :--- | :--- |
| **车门板储物槽** | 前后门板均设有杯架及储物槽，可放置水瓶、雨伞等杂物。 |
| **中央扶手箱** | 深度适中，可存放票据、零钱或小件电子产品。 |
| **手套箱** | 常规尺寸，适合存放文件、手册等。 |
| **中控台下储物区** | （部分高配车型）设有无线充电板或开放式储物格。 |
| **前后排杯架** | 中央通道及后排扶手（如配备）均提供杯架。 |

# 传祺E8 儿童乘车安全与注意事项
  
更新时间：2026年07月18日09时32分23秒   

为了保障您孩子的乘车安全，请仔细阅读并遵守以下事项：
## 一、 安全座椅使用规范
1.  **强制使用**：12周岁以下或身高低于1.5米的儿童乘车时，**必须**使用符合国家标准的儿童安全座椅，严禁直接使用成人安全带。
2.  **反向安装**：建议 **0-4岁**（尤其是体重低于18kg）的婴幼儿使用反向安装的儿童安全座椅，以更好地保护头颈部。
3.  **安装位置**：儿童安全座椅最安全的位置是 **第二排** 或 **第三排** 的ISOFIX接口位置，请勿安装在副驾驶座。
4.  **正确固定**：使用车辆ISOFIX接口和顶部锚点（或支撑腿）牢固固定座椅，安装后用力摇晃，确保晃动幅度不超过2.5厘米。
## 二、 车内乘坐禁忌
1.  **严禁怀抱**：切勿在车辆行驶中怀抱儿童，碰撞时产生的冲击力远超成人手臂力量，无法保护孩子安全。
2.  **禁止副驾**：**绝对禁止** 12岁以下儿童坐在副驾驶位。安全气囊弹出时的巨大冲击力可能对儿童造成致命伤害。
3.  **身体限制**：行车途中，提醒孩子 **不要将头、手、身体任何部位伸出天窗或车窗外**，防止被夹伤或与外部物体碰撞。
4.  **车门安全**：开启儿童安全锁（位于后车门侧面），防止行车中儿童误开车门。停车后由成人协助开门，注意观察后方来车。
## 三、 车内环境与物品
1.  **温度控制**：夏季暴晒后，先开窗散热再让孩子上车。车内温度高，切勿将孩子单独留在车内，哪怕几分钟。
2.  **小物件收纳**：收好车内所有零散物品（如硬币、瓶盖、硬质玩具），防止急刹车时飞溅伤人。中控台、后窗台勿放置坚硬或尖锐物品。
3.  **零食与饮水**：避免在行车中给孩子吃果冻、坚果、棒棒糖等易噎食物。使用吸管杯喝水，防止洒漏烫伤或滑倒。
4.  **遮阳防护**：使用侧窗遮阳帘，避免阳光直射孩子眼睛。可调节空调出风口风向，避免冷风直吹儿童面部。
## 四、 行车途中监护
1.  **专人看护**：最好有成人坐在第二排或第三排陪伴儿童，帮助其保持正确坐姿，并安抚情绪。
2.  **分散注意力**：长途行车时，可播放舒缓音乐或儿童故事，但避免孩子过度兴奋或哭闹影响驾驶员注意力。
3.  **停车再处理**：若孩子哭闹或需要捡拾掉落物品，请 **靠边安全停车** 后再处理，切勿边开车边回头或单手操作。
## 五、 上下车安全
1.  **右侧上下**：尽量在车辆右侧（靠路沿一侧）上下车，避开左侧车流。
2.  **观察后方**：开门前先观察后方是否有来车或行人，采用“荷式开门法”（用离车门较远的手开门）。
3.  **踏板使用**：传祺E8离地间隙较高，儿童上下车时可使用原车配备的侧踏板，成人需在旁扶稳。
4.  **远离车尾**：教育孩子不要在车尾玩耍或逗留，车辆倒车时存在视觉盲区。
## 六、 特别提醒
-   **安全气囊**：副驾驶安全气囊无法关闭时，绝对禁止放置反向儿童座椅。
-   **空调滤芯**：定期更换空调滤芯，保持车内空气清新，减少儿童过敏风险。
-   **应急准备**：车内常备儿童专用急救包、湿巾、垃圾袋等物品。
> **安全口诀**：座椅固定好，门窗都锁牢；头手不外伸，停车再照料。

| 项目 | 参数 |
|------|------|
| **驱动形式** | 前置前驱 |
| **电动机类型** | 永磁同步电机 |

| 项目 | 参数 |
|------|------|
| **电池类型** | 磷酸铁锂电池 |
| **电池容量** | **40.55 kWh** |

| **充电方式** | 支持快充/慢充 |
## 充电性能
- **快充时间：** 约1.5小时（电量从20%充至80%）
- **慢充时间：** 约8-10小时（使用家用交流充电桩）
## 动力特点
1. **起步加速快：** 电动机起步即可输出最大扭矩，0-50km/h加速表现优于同级别燃油车型。
2. **运行平顺：** 无传统变速箱，动力输出线性流畅，无换挡顿挫感。
3. **经济节能：** 百公里电耗约15-18 kWh，使用成本远低于燃油车。
4. **低噪音：** 电机运行安静，驾乘体验更舒适。
## 适用场景
优劲EV搭载的这套纯电动力系统主要面向城市物流配送、短途货运等场景，60kW的功率和200N·m的扭矩足以应对日常满载起步、爬坡等工况，238km的续航里程可满足大多数城市内及城郊运输需求。

# LUMMA G770 驾驶注意事项
  
更新时间：2026年07月13日15时32分46秒   

## 一、启动与预热
1.  **智能钥匙确认**：确保智能钥匙在车内有效范围内。
2.  **一键启动**：踩下制动踏板，按下“ENGINE START/STOP”按钮启动发动机。
3.  **冷车预热**：启动后，建议怠速运转 **30-60秒**，待机油充分润滑涡轮增压器后再行驶。避免立即大脚油门。
4.  **检查仪表盘**：确认无故障灯亮起（如发动机、机油压力、胎压监测等）。
## 二、行驶中操作规范
### 1. 动力与变速箱
-   **换挡逻辑**：LUMMA G770 基于高性能调校，变速箱响应灵敏。日常驾驶建议使用 **舒适（Comfort）模式**；激烈驾驶时切换至 **运动（Sport）或运动+（Sport+）模式**。
-   **避免空挡滑行**：自动变速箱严禁 N 挡滑行，会损坏变速箱。
-   **停车顺序**：车辆完全停稳后 -> 挂 N 挡 -> 拉电子手刹 -> 松脚刹 -> 挂 P 挡 ->熄火。
### 2. 宽体车身与视野
-   **注意轮距加宽**：该车拥有宽体套件，车身宽度大于普通车型。在窄路会车、通过限宽墩或机械车位时需格外小心，建议开启360°全景影像辅助。
-   **盲区管理**：A柱较粗壮，转弯时请多观察。变道时务必使用并线辅助提示。
### 3. 底盘与悬挂
-   **低底盘风险**：LUMMA G770配备运动悬挂，离地间隙较低。遇到陡坡、减速带或坑洼路面时，务必斜向通过（单轮依次过坎），防止前唇或底盘刮擦。
-   **空气悬挂（如配备）**：若选装空气悬挂，在复杂路况前可手动升高底盘。
### 4. 制动系统
-   **高性能刹车**：配备大尺寸打孔刹车盘和高性能卡钳。冷车时刹车效果略有延迟；激烈驾驶后刹车盘温度极高，停车后勿立即洗车。
-   **刹车磨合期**：新更换的刹车片/盘需经过约200公里的轻中度制动磨合才能达到最佳效能。
## 三、特殊场景应对
|场景|操作要点|
|---|---|
|雨天/湿滑路面|关闭“运动排气”阀门减少分心；降低车速；避免急加速导致大扭矩甩尾。|
|高速行驶|超过120km/h时，尾翼可能自动升起以增加下压力；切勿超速行驶。|
|长途行驶|每行驶2小时建议休息20分钟；注意检查轮胎温度和胎压变化。|
## 四、停车与熄火
1.  **涡轮延时冷却（如配备）**：若刚进行过激烈驾驶或长途高速行驶，熄火前请怠速运转1-2分钟再熄火以保护涡轮增压器。
2.  **方向回正**：停车务必回正方向盘并关闭所有灯光及空调系统再熄火。
3.  **防盗锁止**：离车后检查车门是否锁好。
## ⚠️重要警示
> -   本车为改装高性能车型，保险条款可能对改装件有特殊规定（如包围、轮毂、排气），请确认保单覆盖范围。
> -   严禁使用非标号燃油或劣质燃油添加剂。
> -   未经授权私自刷写ECU程序可能导致动力总成质保失效及安全隐患。

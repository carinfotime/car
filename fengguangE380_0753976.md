# 东风风光E380 技术参数介绍
  
更新时间：2026年07月13日16时19分19秒   

## 一、车型概览
风光E380是东风小康旗下的一款纯电动紧凑型MPV，主打多功能、大空间和低运营成本，适用于家庭出行、网约车及城市物流等多种场景。该车基于燃油版风光380打造，在保持实用性的基础上实现了电动化升级。
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **驱动电机类型** | 永磁同步电机 |
| **电机最大功率** | 60 kW (约82马力) |
| **电机最大扭矩** | 220 N·m |
| **最高车速** | 100 km/h |
| **电池类型** | 磷酸铁锂电池 |
| **电池容量** | 42.3 kWh (部分版本) / 另有其他容量版本可选 |
| **NEDC综合续航里程** | 约310 km (标准版) |
| **充电方式** | 支持交流慢充与直流快充 |
| **快充时间** | SOC 30%-80% ≤ 80分钟 (典型值) |
> *注：具体续航及充电数据可能因电池版本、驾驶习惯和环境温度有所差异。*
## 三、车身尺寸与结构
| 项目 | 参数 |
|------|------|
| **车身形式** | 5门MPV（侧滑门设计） |
| **长 × 宽 × 高 (mm)** | 4610 × 1750 × 1860 |
| **轴距 (mm)** |  2850 |
| **座位数** |  7座（2+2+3布局）/可选5座版 |
| **整备质量 (kg)** |  约1560 -1620（视配置而定） |
## 四、底盘与操控
-   **驱动方式：**前置前驱
-   **前悬架类型：**麦弗逊式独立悬架
-   **后悬架类型：**钢板弹簧非独立悬架（注重承载能力）
-   **转向系统：**电动助力转向（EPS）
-   **制动系统：**前盘后鼓式制动器
-   **轮胎规格：**185/65 R15
## 五、内部配置与空间
-   **内饰风格：**简约实用设计，配备液晶仪表盘
-   **中控屏：**高配版搭载悬浮式触控屏，支持蓝牙/收音机/手机互联
-   **座椅材质：**织物或仿皮（视配置）
-   **空调系统：**手动空调，后排独立出风口
-   **储物空间：**
    -   常规后备箱容积可观
    -   第三排座椅可折叠翻转，扩展载货空间
    -   车内多处储物格设计
## 六、安全配置
-   ABS防抱死制动系统 + EBD制动力分配
-   主驾驶安全气囊（高配增加副驾驶气囊）
-   倒车雷达（高配带倒车影像）
-   胎压监测系统
-   儿童座椅接口 (ISOFIX)
-   遥控钥匙 /中控锁
## 七、主要竞品与定位
风光E380的主要竞争对手包括五菱EV50、长安之星9 EV等同级纯电微面/MPV车型。其核心优势在于：
1.  侧滑门设计方便上下客及装卸货物。
2.  长轴距带来宽敞的乘坐和载物空间。
3.  磷酸铁锂电池安全性高，循环寿命长。
4.  较低的购置和使用成本适合运营用途。

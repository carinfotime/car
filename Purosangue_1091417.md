# Ferrari Purosangue 驾驶注意事项
  
更新时间：2026年07月20日09时28分46秒   

## 一、启动与预热
1.  **钥匙识别**：确保智能钥匙在车内有效范围内。
2.  **启动程序**：踩下制动踏板，按下方向盘上的 `ENGINE START` 按钮。
3.  **冷车预热**：
    -   启动后，请勿立即大脚油门。
    -   等待发动机转速回落至稳定的怠速状态（约 30-60 秒）。
    -   在机油温度达到工作温度（约 70°C）前，建议将发动机转速保持在 **4,000 rpm** 以下。
## 二、驾驶模式选择 (manettino)
Purosangue 提供多种模式以适应不同路况：
| 模式 | 适用场景 | 注意事项 |
| :--- | :--- | :--- |
| **ICE** (冰雪/湿地) | 雪地、结冰路面、低附着力路面 | ESP/牵引力控制全开，动力输出平缓。 |
| **WET** (湿滑) | 雨天、潮湿路面 | 稳定性优先，油门响应变柔和。 |
| **GT** (舒适) | 日常城市/高速巡航 | **推荐日常使用**。悬挂最软，油耗经济。 |
| **SPORT** (运动) | 山路、干地激烈驾驶 | -   悬挂变硬，换挡更快。<br>-   ESC部分介入允许轻微滑动。 |
| **SPORT+ ESC OFF** | (赛道/封闭场地) | -   **仅在赛道或封闭道路使用**。<br>-   ESC关闭，车辆完全由驾驶者控制。<br>-   **严禁在公共道路使用此模式。** |
> ⚠️ **注意**：切换模式时请确保车速和路况安全。
## 三、行驶中的关键操作
### V12自然吸气发动机特性
-   **高转特性**：这是一台高转速 V12（红线区约8,250 rpm），最大马力在较高转速释放。
-   **低扭充沛**：虽然高转强劲，但日常行驶时低扭表现依然从容。
-   **换挡建议**：
    -   日常 GT/WET模式：变速箱会在2,500-3,500 rpm自动升挡。
    -   SPORT模式：建议手动换挡或允许变速箱在高转区域保持档位。
### PDK双离合变速箱
-   R倒挡：通过方向盘左侧的拨片（拉）或中控按钮挂入。
-   N空挡：同时拉动左右换挡拨片挂入空挡。
-   Auto/Manual切换：按下中控台上的 `AUTO` / `MANUAL`按钮切换自动/手动模式。
### AWD四驱系统
-   Purosangue采用独特的四驱布局（前桥取力器位于变速箱前端）。
-   **正常行驶时主要为后轮驱动**；需要时前桥介入提供牵引力。
-   在冰雪或湿滑路面起步时，四驱系统会自动优化分配扭矩。
## 四、停车与熄火
1.  踩住制动踏板直至车辆完全静止。
2.  挂入空挡 (N)。
3.  拉起电子手刹（位于中控台左侧）。
4.  按下方向盘上的 `ENGINE STOP`按钮熄火。
5.  离开车辆前确认所有车窗关闭并锁好车门。
## ⚠️五、重要安全警告
1.  **底盘高度与通过性**
    -   虽然具备一定越野能力（可升降悬挂），但Purosangue本质上是一台高性能跑车化的SUV。
    -   避免驶入深坑、尖锐石块区域或超过20cm的涉水路段。
2.  **轮胎压力**
    -   出厂配备高性能轮胎（夏季胎或全季胎）。
    -    冬季必须更换专用冬季轮胎或雪地胎。
3.  **刹车系统**
    -    碳陶瓷刹车盘在冷车状态下制动力略有下降；激烈驾驶前需先轻踩刹车进行预热。
4.  **燃油要求**
    -    必须使用98号及以上无铅汽油（RON）。加注错误燃油将严重损坏发动机。
## ✅六、日常维护提示
1.     定期检查机油液位（电子机油尺可通过仪表盘查看）。
2.     每两年更换一次制动液及冷却液。
3.     保持空气滤清器清洁——V12发动机对进气质量敏感。

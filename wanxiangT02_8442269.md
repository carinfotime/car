## 万象T02 空间参数介绍
  
更新时间：2026年07月16日15时14分28秒   

### 车身尺寸与轴距
| 项目 | 参数 |
| :--- | :--- |
| 车长（mm） | **4,750** |
| 车宽（mm） | **1,910** |
| 车高（mm） | **1,720** |
| 轴距（mm） | **2,850** |
> **亮点：** 长轴距设计为车内纵向空间提供了良好基础，后排腿部空间充裕。
### 乘坐空间
- **前排头部空间：** **1,020 mm**
- **后排头部空间：** **990 mm**
- **后排腿部空间：** **920 mm**
- **后排肩部空间：** **1,480 mm**
> - 前排座椅支持多向电动调节，可适应不同身高驾驶者。
> - 后排座椅靠背角度可调，长途乘坐更舒适。
### 储物能力
#### 🧳 后备箱容积
| 状态 | 容积 |
| :--- | :--- |
| 标准状态（5座满员） | **520 L** |
| 后排放倒后（最大扩展） | **1,450 L** |
> - 后备箱开口宽度达 **1,120 mm**，便于搬运大件物品。
> - 内部平整度高，放倒后排可形成近乎纯平的地台。
#### 🧩 车内储物格
- **中央扶手箱：** 容积约 **8.5 L**，可放置饮料、纸巾等。
- **手套箱：**标准大小，带阻尼开启。
- **门板储物槽：**
    - *前门：*可容纳3瓶500ml矿泉水。
    - *后门：*可容纳2瓶500ml矿泉水。
- **中控台下方储物区：**支持手机无线充电及小物件存放。
- **杯架数量：**
    - *前排：*2个
    - *后排中央扶手：*2个
    - *后门板各1个*
### 📐其他关键数据
- **最小离地间隙（空载）：180 mm**
    >兼顾城市与轻度非铺装路面通过性。
- **接近角 / 离去角 / 通过角 ：18° /  22° /  16°**

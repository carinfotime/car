# 搏胜汽车儿童安全注意事项
  
更新时间：2026年07月17日16时50分47秒   

## 1. 乘车前检查
- **确认儿童锁已开启**：上车后，请确保两侧后门的儿童安全锁已拨至“LOCK”位置，防止孩子在行驶中意外打开车门。
- **调整座椅与安全带**：确保孩子乘坐的儿童安全座椅或增高垫安装牢固。若孩子身高达到可使用成人安全带的标准（通常为身高≥145cm），请调整座椅位置，使安全带能正确跨过孩子的肩部和髋部，而非颈部或腹部。
- **清理车内小物件**：收起中控台、座椅缝隙及脚垫上的硬币、弹珠、尖锐玩具等小物件，避免急刹车时飞溅伤人。
## 2. 行驶中注意事项
- **全程使用安全约束系统**：
  - **12岁以下或身高<145cm的儿童**：必须使用符合国家标准的儿童安全座椅或增高垫。
  - **切勿怀抱儿童**：即使坐在后排，成人怀抱也无法在碰撞中保护孩子，反而可能因惯性挤压孩子。
- **禁止探头或伸手出窗外**：提醒孩子不要将头、手或身体任何部位伸出天窗或侧窗，避免被路边物体刮伤或被其他车辆碰撞。
- **避免在车内打闹**：行驶中不要让孩子在座位上站立、跳跃或前后排爬行，防止车辆颠簸或急刹时摔倒受伤。
- **车窗与天窗操作**：
  - 电动车窗具备防夹功能，但仍需由成人操作。切勿让孩子独自操控车窗开关。
  - 天窗开启时，务必确保孩子头部未伸出车顶。
## 3. 停车与上下车
- **停车后再解开安全带**：务必在车辆完全停稳并熄火后，再帮孩子解开安全带。
- **右侧下车原则**：尽量让孩子从右侧（靠路肩一侧）上下车。下车前先观察后方是否有来车或行人。
- **勿将孩子单独留在车内**：
  - 无论天气冷暖，车内温度会迅速升高或降低。夏季阳光直射下，车内温度可在10分钟内上升至60°C以上，极易导致热射病或缺氧窒息。
  - 离开车辆时务必检查所有座位，确认孩子已全部带离。
## 4. “搏胜”车型特别提示
| 项目 | 说明 |
|------|------|
| ISOFIX接口 | “搏胜”后排两侧座位配备ISOFIX儿童座椅固定接口（通常位于坐垫与靠背缝隙处），请优先使用该接口安装硬连接式儿童座椅。 |
|  后排USB充电口 | “搏胜”后排可能提供USB充电口。请使用原装线缆为电子设备充电；避免让孩子拉扯线缆或将金属物品插入接口。 |
|  后备箱区域 | “搏胜”作为皮卡车型（如适用），其后备箱（货斗）严禁载人。任何情况下不得让儿童在后备箱内乘坐或玩耍。 |
## ⚠️紧急情况应对
1. **若孩子在车内被反锁**：
   -  保持冷静并安抚孩子情绪。
   -  立即拨打救援电话（如119、道路救援热线）。
   -  若情况紧急且条件允许（如烈日下高温），可破窗救人——选择离孩子最远的侧窗进行破拆。
2. **发生碰撞后**：
   -  优先检查孩子的意识与呼吸。
   -  非必要不移动受伤的孩子（尤其是怀疑颈椎损伤时），等待专业医护人员到场。

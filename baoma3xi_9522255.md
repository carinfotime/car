# 宝马3系驾驶注意事项
  
更新时间：2026年07月18日18时12分55秒   

## 一、启动与预热
1.  **启动流程**：踩住刹车，一键启动车辆。无需长时间热车。
2.  **冷车起步**：启动后等待约30秒至1分钟，待转速表指针回落至正常怠速（约700-800转）即可缓慢行驶。
3.  **水温达标前**：在水温达到正常工作温度前，避免发动机转速超过3000转或急加速。
## 二、行驶中注意事项
### 变速箱使用
-   **自动模式**：D挡行驶时，变速箱会自动匹配最佳换挡时机。
-   **手动模式**：在需要激烈驾驶或长下坡控制车速时，可将挡杆左拨至M/S挡位，使用换挡拨片或前后推拉挡杆进行手动换挡。
-   **停车等红灯**：短时间（30秒内）可D挡踩刹车；长时间建议挂N挡拉手刹。不建议长时间D挡踩刹车等待。
-   **倒车注意**：车辆完全停稳后再挂入R挡。
### 转向与底盘
-   **方向盘**：宝马转向系统较为精准且偏重，避免原地大幅度打死方向盘（易损伤转向助力系统）。
-   **底盘保护**：3系离地间隙较低，通过减速带、坑洼路面或陡坡时务必减速慢行，防止托底损伤油底壳或悬挂。
### 制动系统
-   **磨合期**：新车前500公里避免紧急制动，以利于刹车盘片磨合。
-   **涉水后**：涉水后轻踩几次刹车，利用摩擦热量蒸发刹车盘上的水分，恢复制动效能。
## 三、特殊路况驾驶
### 雨天/湿滑路面
1.  关闭自动启停功能（防止涉水熄火后自动重启导致发动机进水）。
2.  降低车速，避免急加速、急刹车和急转弯。
3.  开启雾灯或近光灯，确保视野清晰。
### 高速公路
1.  保持安全车距（至少保持2秒反应时间距离）。
2.  如需超车或变道，提前打转向灯并观察盲区。
3.  合理使用定速巡航功能以减轻疲劳。
### 山路/弯道
1.  入弯前提前减速，弯中保持匀速。
2.  出弯时可平顺加油。利用后驱特性享受操控乐趣的同时注意控制油门深度。
## 四、停车与熄火
1.  **停车步骤**：
    -   车辆停稳 → 挂N挡 → 拉电子手刹 → 松脚刹 → 挂P挡 → 熄火。
    -   直接挂P挡熄火也可行（电子系统会自动处理），但上述步骤可减少变速箱锁止机构受力。
2.  **坡道停车**：先拉手刹再挂P挡，防止P挡锁止齿轮承受过大压力导致下次摘挡困难。
3.  **熄火前检查**：关闭空调、音响等用电设备后再熄火，有助于保护蓄电池寿命。
## 五、日常维护提醒
| 项目 | 建议周期 | 备注 |
| :--- | :--- | :--- |
| **机油检查** |每月一次 |冷车状态下检查机油尺（电子机油尺通过iDrive查看） |
| **胎压监测** |每月一次 /长途前 |标准胎压见驾驶员侧门框标签 |
| **轮胎换位/动平衡** |每10000公里 /出现抖动时 |后驱车前后轮胎磨损不同 |
| **空调滤芯更换** |每20000公里 /每年一次 |视空气质量缩短周期 |
| **制动液更换** |每2年一次 |确保制动安全 |
## ⚠️特别警示
1.  切勿在发动机运转时离开车辆且不拉手刹（尤其是坡道）。
2.  仪表盘亮起黄色/红色故障灯时请及时检修（如发动机故障灯、胎压报警等）。
3.  改装需谨慎——随意改动排气、ECU程序或悬挂可能影响质保及行车安全。
4.  使用正规渠道燃油和机油（推荐95号及以上汽油）。
> *以上内容基于通用驾驶规范及宝马3系常见特性整理。具体操作请以您的车辆用户手册为准。*

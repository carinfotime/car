# 雷斯特 (Rexton) 操作指南
  
更新时间：2026年07月18日12时18分30秒   

本指南涵盖雷斯特车型的常见操作、功能说明及安全提示。具体功能可能因年款、配置及市场版本不同而有所差异，请以实车为准。
## 一、启动与熄火
### 1. 启动发动机
- **智能钥匙系统**：
    1.  确保智能钥匙在车内。
    2.  踩下制动踏板（自动挡）或离合器踏板（手动挡）。
    3.  按下“ENGINE START/STOP”按钮。
- **传统钥匙系统**：
    1.  将钥匙插入点火开关。
    2.  拧至“ON”位置，等待仪表盘自检灯熄灭（尤其是预热指示灯，柴油车型）。
    3.  拧至“START”位置启动。
### 2. 熄火
- **自动挡**：挂入P挡，拉起电子手刹或手刹，按下“ENGINE START/STOP”按钮熄火。
- **手动挡**：挂入空挡，拉起手刹，关闭钥匙或按下按钮。
> **注意**：涡轮增压车型（尤其柴油版），长途或高负荷行驶后，建议怠速运转1-2分钟再熄火，以保护涡轮。
## 二、驾驶操作
### 1. 变速箱模式
- **自动变速箱 (奔驰技术)**：
    - **D档**：正常行驶。
    - **S档**：运动模式，延迟换挡时机，动力响应更积极。
    - **W档 (冬季模式)**：雪地或湿滑路面起步时使用，以2档起步减少打滑。
    - **手动模式 (+/-)**：将档杆拨至手动槽位，向前推为升档(+)，向后拉为降档(-)。
### 2. 四驱系统 (分时四驱)
雷斯特通常配备分时四驱系统（部分新款为适时四驱），通过旋钮切换：
| 模式 | 适用场景 | 操作说明 |
| :--- | :--- | :--- |
| **2H** | 普通公路、高速行驶 | 后轮驱动，省油。 |
| **4H** | 雨雪、砂石、泥泞等附着力低的路面 | 高速四驱。可在时速80km/h以下直接切换。 |
| **4L** | 陡坡攀爬、深坑脱困、极端越野 | 低速四驱。需停车挂N挡后切换。扭矩放大。 |
> ⚠️ **严禁在干燥铺装路面使用4H或4L模式！**
### 3. ESP / TCS (电子稳定/牵引力控制)
- **默认开启**：提供最佳稳定性。
- **关闭方法**：长按ESP OFF按钮约3秒。
- **何时关闭**：
    -  陷入沙地、泥地或雪地需要车轮空转脱困时。
    -  安装防滑链行驶时。
## 三、仪表盘与信息显示
### Trip Computer (行车电脑)
通过方向盘左侧的 `TRIP` / `MODE` / `SET` / `RESET`按键控制：
-   `TRIP` / `MODE`：切换显示内容（续航里程、平均油耗、瞬时油耗、小计里程A/B等）。
-   `SET` / `RESET`：重置小计里程或平均油耗数据。
### 指示灯含义
|图标|名称|含义|
|---|---|---|
|黄色发动机灯|检查发动机灯(Check Engine)|发动机电控系统故障。尽快检修。|
|红色电池灯|充电指示灯|发电机不充电或电瓶故障。|
|黄色预热塞灯(柴油)|Glow Plug Indicator|等待预热完成后再启动车辆。|
|绿色前后雾灯灯||对应灯光已开启。|
## 四、灯光与雨刷
### 灯光控制杆
-   `OFF` -> `AUTO` (如有) -> `示宽灯(Parking)` -> `近光灯(Low Beam)`。
-   向前推远光灯闪；向后拉远光灯常亮。
### 雨刷控制杆
-   `MIST` (向上)：单次刮擦。
-   `INT/AUTO` (向下第一档)：间歇刮擦/自动感应刮擦(根据雨量)。
    -   旋转中间的旋钮可调节间歇频率/灵敏度。
-   `LO/HI` (向下第二/三档)：低速/高速连续刮擦。
> 💡 *喷水清洗*：将雨刷杆向驾驶员方向拉动即可喷水并刮洗前风挡。
## 五、空调与气候控制
### 双区自动空调
1.温度调节：
    -旋转左右两侧的温度旋钮分别设定驾驶员和副驾驶侧温度(`LO`:最低冷,`HI`:最高热)。
2.A/C键:压缩机开关(制冷除湿)。
3.MODE键:出风模式(吹面/吹脚/除霜)。
4.RECIRC键:内循环(快速降温/阻隔外部异味)。建议定期切换外循环换气。
5.FRONT键:前风挡除雾除霜(会自动开启A/C和最大风量)。
6.REAR键:后窗加热除雾除霜(以及外后视镜加热)。
## 六、座椅与便利功能
### 座椅调节
-**电动座椅**:通过门板上的按键调节前后高低靠背角度以及腰部支撑(部分高配)。
-**加热通风**:中控台或门板上的座椅图标按键,分为三档可调。(*通风功能仅高配车型提供*)
### 电动折叠后视镜
-**折叠**:旋转门板上的后视镜调节旋钮至折叠图标位置即可电动收折两侧外后视镜。(*狭窄路段会车时使用*)
## 七、维护保养提示
-**机油**:推荐使用符合ACEA C3标准的高品质全合成机油,粘度参考用户手册(常见5W30或5W40)。
-**柴油滤清器**:柴油版需定期更换,建议每15,000公里更换一次以防止高压共轨系统损坏。(*务必加注正规加油站的高品质柴油*)
-**正时皮带**:部分老款发动机为正时皮带设计,务必按手册要求每60,000~80,000公里更换正时皮带套件(*否则断裂会导致发动机顶缸报废*)；新款多为正时链条免维护但需留意异响检查张紧器状态.
-**轮胎气压**:建议每月检查一次胎压包括备胎；标准值通常在前轮32PSI(~220kPa)后轮36PSI(~250kPa),满载时可适当增加；具体数值请参考驾驶座B柱标签.

# 麒领汽车 驾驶注意事项
## 一、启动前检查
- **绕车一周检查**：查看轮胎气压、外观有无刮擦或异物，车底有无漏油、漏水痕迹。
- **调整座椅与后视镜**：确保坐姿舒适，能清晰观察到后方及侧方路况。
- **系好安全带**：驾驶员及所有乘员必须全程系好安全带。
- **检查仪表盘**：启动后观察是否有故障灯亮起（如发动机故障灯、胎压报警灯等）。
## 二、行驶中注意事项
### 1. 控制车速
- 严格遵守道路限速标志。
- 雨雪、雾霾等恶劣天气应降低车速，保持安全车距（至少为平时两倍）。
### 2. 合理使用灯光
- 夜间行车开启近光灯，会车时切换近光避免眩目。
- 雾天开启雾灯和危险报警闪光灯。
### 3. 避免疲劳驾驶
- 连续驾驶不超过4小时，建议每2小时休息15分钟。
- 感到困倦时立即停车休息，不要强撑。
### 4. 正确使用挡位与刹车
- **自动挡车型**：等红灯时挂N挡拉手刹；长下坡利用发动机制动（切换手动模式或L挡）。
- **手动挡车型**：换挡时离合踩到底；下坡勿空挡滑行。
- **刹车技巧**：提前预判路况，轻踩长磨为主，避免急刹。
### 5. 特殊路况应对
|  路况 |  操作建议 |
|-------|-----------|
|  湿滑路面 |  缓加速、缓转向、缓刹车 |
|  颠簸路段 |  低速通过，保护悬挂系统 |
|  涉水路段 |  确认水深不超过排气管高度 |
## 三、停车与熄火
1. **停车位置选择**：停在平整地面或指定车位；坡道停车需拉紧手刹并挂入P挡（自动挡）或挂一挡/倒挡（手动挡）。
2. **熄火前操作**：关闭空调、大灯等用电设备后再熄火。
3. **锁车确认**：锁车后拉一下门把手确认已锁好。
## 四、日常保养提醒
|  项目 |  周期/标准 |
|-------|------------|
|  机油更换 |  每5000~10000公里或半年 |
|  轮胎胎压 |  每月检查一次 |
|  刹车片厚度 |  低于3mm需更换 |
|  冷却液/玻璃水 |  定期检查液位 |

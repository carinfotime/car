# 蔚来ES9 保养指南
  
更新时间：2026年07月17日14时25分07秒   

## 一、保养总览
蔚来ES9作为蔚来汽车旗下的旗舰级SUV，采用纯电驱动系统，相比传统燃油车，保养项目大幅简化，但仍需定期维护以确保车辆性能与安全。
| 保养周期 | 建议时间 |
|---------|---------|
| 常规检查 | 每12个月或20,000公里（以先到为准） |
| 空调滤芯更换 | 每12个月或20,000公里 |
| 制动液更换 | 每24个月或40,000公里 |
| 减速器齿轮油更换 | 每48个月或80,000公里 |
| 冷却液更换 | 每48个月或80,000公里 |

## 二、各阶段保养项目明细
### ✅ A类保养（12个月 / 20,000公里）
**基础检查项目：**
- [ ] 高压电池系统状态检测
- [ ] 电机及电控系统检查
- [ ] OTA软件版本检查与升级
- [ ] 空调滤芯清洁/更换
- [ ] 轮胎胎压检测与调整（建议2.5~2.8 bar）
- [ ] 轮胎磨损情况检查及换位
- [ ] 制动片厚度检查
- [ ] 制动盘磨损情况检查
- [ ] 底盘及悬挂系统目视检查
- [ ] 转向系统功能测试
- [ ] 灯光系统全面检查
- [ ]雨刮片状态检查与清洁
- [ ]玻璃水补充
### ✅ B类保养（24个月 / 40,000公里）
包含A类全部项目，另增加：
- [x] A类全部项目已完成
- [ ] **制动液更换**（含水量超标时必须更换）
- [ ] **空调系统杀菌消毒**
- [ ] **车门铰链、天窗滑轨润滑**
- [ ] **12V蓄电池健康度检测**
- [ ] **充电口及充电线束检查**
### ✅ C类保养（48个月 / 80,000公里）
包含A+B类全部项目，另增加：
- [x] A+B类全部项目已完成
- [ ] **减速器齿轮油更换**
- [ ] **电机冷却液更换**
- [ ] **驱动电机绝缘检测**
- [ ] **高压线束老化程度检查**
- [ ] **全车橡胶密封件检查**

## 三、日常自检清单（建议每月执行）
> 🔧 **以下项目车主可自行完成**
1. **胎压检测** — NIO App或仪表盘查看，标准值见驾驶侧B柱标签。
2. **玻璃水液位** —不足时及时添加。
3. **外观灯光** —绕车一周确认所有灯光正常。
4. **充电口清洁** —用干布擦拭充电接口，防止异物影响充电。
5. **雨刮效果** —刮不干净时及时更换雨刮片。
6. **车内空气** —开启空调闻有无异味，必要时提前更换滤芯。

## 四、重要注意事项
### ⚠️ NIO Service专属提醒
1. **首保免费**：提车后12个月内或行驶20,000公里内首次保养免费。
2. **服务包用户**：购买蔚来「服务无忧」或「保险无忧」套餐的用户，按套餐约定周期享受免费保养服务。
3. **异地加电不影响质保**：使用第三方充电桩不影响整车质保，但建议优先使用蔚来超充站或换电站。

## 五、延长电池寿命建议
1. ✅日常电量保持在 `20%~90%`之间，避免长期满充或亏电存放。
2. ✅长期停放时电量保持在 `50%~60%`。
3. ✅尽量避免频繁使用直流快充至100%，日常以交流慢充为主。
4. ✅高温天气下尽量选择阴凉处停放充电。
5. ✅冬季用车前可通过NIO App远程开启电池预热功能。

## 六、常见问题 FAQ
**Q1：ES9需要换机油吗？**
A：不需要。纯电动车无发动机和变速箱，无需机油和变速箱油。
**Q2：刹车片多久换一次？**
A：取决于驾驶习惯。ES9配备动能回收系统，刹车片磨损远低于燃油车。通常60,000~100,000公里才需考虑更换。
**Q3：不去服务中心自己找修理厂行吗？**
A：可以但需注意——非授权渠道维修可能导致三电系统质保失效。建议优先选择蔚来服务中心。

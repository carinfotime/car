# 宝马M4 操作指南
  
更新时间：2026年07月17日08时31分07秒   

## 一、启动与熄火
### 启动发动机
1. **智能钥匙感应**：确保遥控钥匙在车内。
2. **踩下制动踏板**：必须踩下制动踏板。
3. **按下启动按钮**：按下位于方向盘右侧的“START/STOP”按钮。
4. **发动机自动启停**：默认开启，如需关闭可按下中控台上的 `A OFF` 按钮。
### 熄火
1. **挂入P挡**：将车辆完全停稳后，挂入驻车挡 (P)。
2. **拉起电子手刹**：向上拉动位于换挡杆左侧的电子手刹按钮。
3. **再次按下启动按钮**：关闭发动机。

## 二、驾驶模式选择
M4提供多种驾驶模式，通过中控台上的红色 `SETUP` 按钮和方向盘上的 `M1` / `M2` 按钮进行快速切换。
### 核心模式（通过换挡杆左侧的拨杆调节）
- **EFFICIENT (高效)**：适合日常巡航，油门响应平缓，悬挂偏软，油耗最低。
- **COMFORT (舒适)**：默认模式，兼顾日常驾驶与运动感。
- **SPORT (运动)**：油门响应更灵敏，转向变重，悬挂变硬，排气阀门打开。
- **SPORT+ (运动+)**：最激进的模式。所有系统设定为最高性能状态。**注意**：此模式下部分电子稳定系统（DSC）会部分关闭或完全关闭（取决于其他设置），仅建议在赛道或封闭道路使用。
### M1 / M2 自定义模式
您可以通过 iDrive 系统预设两套完整的车辆配置（包含发动机、悬挂、转向、DSC等）。
- **方向盘上的 M1 / M2 按键**：
    - **短按**：直接激活预设模式。
    - **长按（约2秒）**：激活预设模式并同时关闭 DSC（动态稳定控制系统），进入完全手动控制状态。此操作需谨慎。

## 三、变速箱操作
M4配备8速M Steptronic自动变速箱或6速手动变速箱（视具体车型而定）。
### 自动变速箱
- **D挡 (Drive)** ：标准自动模式。
- **S挡 (Sport)** ：换挡逻辑更激进，延迟升档时机。可通过将换挡杆向左拨入S/M区域激活。
- **手动模式 (M/S)**：
    1.  将换挡杆向左推入S/M槽位。
    2.  使用换挡拨片 (`+` / `-`) 进行手动加减档。
    3.  仪表盘会显示当前档位数字。
### Drivelogic 调节
在S/M模式下：
-  按压换挡杆上的 `Drivelogic` 按键可调整换挡速度与性格：
    -  1级（最舒适）：平顺换挡
    -  2级（运动）：快速换挡
    -  3级（赛道）：最快速度、冲击感最强的换挡

## 四、重要功能与设置
### M Mode （方向盘右侧 MODE 按钮）
用于调整仪表盘和HUD抬头显示的信息密度以及辅助系统的干预程度：
-   ROAD （公路）：标准显示与全功能辅助系统。
-   SPORT （运动）：简化显示内容，减少辅助系统干预。
-   TRACK （赛道）：极简显示（仅显示转速、时速、水温油温等关键信息），关闭大部分辅助系统。仅限赛道使用。
### DSC （动态稳定控制系统）
位于中控台的 `DSC OFF` / `DSC ON` 按钮：
-  短按一次：开启M动态模式 (MDM)，允许一定程度的车轮打滑后再介入控制。适合激烈山路驾驶或赛道入门练习。
-  长按约5秒直至仪表盘显示“DSC OFF”：完全关闭稳定控制。需要极高的驾驶技术才能安全操控。
### HUD抬头显示
可投影车速、导航指示、转速及当前档位等信息。亮度与高度可在 iDrive > “设置” > “显示器” > “平视显示器”中调节。

## 五、日常使用注意事项
|项目|说明|
|---|---|
|燃油标号|必须使用95号及以上无铅汽油；推荐98号以获得最佳性能。|
|冷车启动|启动后请等待机油温度达到约70°C以上再激烈驾驶。机油温度表位于仪表盘中央下方。|
|轮胎压力|请参照驾驶员侧门框上的胎压标签进行充气；激烈驾驶前建议适当提高胎压0.2~0.3 bar。|
|刹车预热|赛道或激烈驾驶前，需在直道上多次中等力度刹车以预热刹车盘片至工作温度。|
|冷却系统|激烈驾驶后不要立即熄火；让发动机怠速运转30秒至1分钟以帮助涡轮和冷却液循环散热。|

## ⚠️安全警告
1.  在任何公共道路上行驶时，请遵守当地交通法规及限速规定。
2.  仅在封闭的专业赛道内尝试 SPORT+ / TRACK / DSC OFF等极限设定组合。
3.  切勿在湿滑路面或低附着力路面上完全关闭DSC行驶——极易失控导致严重事故！
4.  长时间激烈驾驶后请注意检查机油液位和冷却液温度；若出现红色警告灯应立即停车检查并呼叫救援。

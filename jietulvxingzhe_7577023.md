# 捷途旅行者 操作指南
  
更新时间：2026年07月13日17时22分46秒   

## 一、车辆启动与熄火
### 1.1 智能钥匙启动
- 携带智能钥匙进入车辆，踩下刹车踏板。
- 按下“一键启动”按钮，仪表盘亮起即表示车辆已通电/启动。
- **注意**：若仪表盘提示“未检测到钥匙”，请将钥匙靠近杯托内的感应区域。
### 1.2 紧急启动（钥匙亏电）
- 将智能钥匙贴近方向盘右侧的感应标识处。
- 踩下刹车，按下“一键启动”按钮。
### 1.3 熄火
- 车辆停稳后，挂入 P 挡（电子挡杆按顶部 P 键）。
- 再次按下“一键启动”按钮，发动机熄火。

## 二、驾驶模式选择
捷途旅行者提供多种地形模式，通过中控台旋钮切换：
| 模式 | 适用场景 |
|------|----------|
| **经济模式** | 城市道路、高速巡航，节能优先 |
| **标准模式** | 日常综合路况 |
| **运动模式** | 需要快速超车、爬坡或激烈驾驶 |
| **雪地模式** | 冰雪、湿滑路面 |
| **泥泞模式** | 泥地、沼泽路段 |
| **沙地模式** |沙漠、沙滩等松软路面 |
| **岩石模式** |崎岖山路、大石块路段（需低速四驱） |
> **提示**：在越野模式下（泥地/沙地/岩石），建议关闭车身稳定系统（ESP）以获得更好通过性。

## 三、四驱系统操作
### XWD（博格华纳智能四驱）
- **自动模式**：默认状态，系统根据路况自动分配前后轴扭矩。
- **低速四驱（4L）**：在 N/P挡停车状态下，按下中控“4L”按键启用。适用于极端越野或脱困场景。
- **后桥差速锁**：在低速四驱模式下按下差速锁按键。注意：
    -  仅限非铺装路面使用。
    -  车速超过30km/h时自动解除。

## 四、电子换挡操作
捷途旅行者采用电子怀挡设计：
- **D挡（前进）**：向下拨动挡杆一次。
- **R挡（倒车）**：向上拨动挡杆一次。
- **N挡（空挡）**：向上或向下轻拨并保持1秒。
- **P挡（驻车）**：按下挡杆顶部的 P键。
-  停车时挂入P挡会自动拉起电子手刹。

##  六、空调与舒适系统
###  快捷操作
-   前风窗除雾 /   后风窗加热 +外后视镜加热
    -   通过中控屏下方触控面板或语音指令控制。
    -   语音指令示例：“你好捷途，打开空调”“温度调到24度”“吹脚模式”。
###  座椅功能
-   主驾座椅支持电动调节（前后/高低/靠背）。
-   部分配置带座椅加热 /通风 /记忆功能，可在中控屏设置界面调整。

##  七、灯光与雨刮
###  灯光控制拨杆
位置：方向盘左侧拨杆
旋转旋钮：
: OFF关闭  
: AUTO自动大灯  
:示宽灯  
:近光灯  
前后推拉：
向前推 →远光灯常亮  
向后拉 →闪远光一次  
雾灯开关：
旋转拨杆中间旋钮至雾灯图标位置即可开启前后雾灯。
###  雨刮控制拨杆
位置：方向盘右侧拨杆
上下拨动调节雨刮速度：
MIST单次刮刷  
INT间歇刮刷  
LO低速连续刮刷  
HI高速连续刮刷  
向驾驶员方向拉动 →喷水+刮刷清洁前风窗。

##  八、后备箱及外设使用
###  侧开式尾门
从外部拉动把手即可打开。注意后方空间是否足够开门角度。关闭时用力推至锁止声响起即可。
备胎架 /小书包储物盒需先解锁卡扣再向外拉开取物。取用备胎前请确认车辆停稳且处于P档状态。
拖车钩安装位置在前保险杠下方预留孔位处拧入拖车环即可使用牵引功能。（注意仅限救援牵引用途）
涉水喉为选装件无需日常操作但需定期检查进气口是否有异物堵塞以确保涉水能力正常发挥！

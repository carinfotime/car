# 瑞虎7 PLUS 驾驶注意事项
  
更新时间：2026年07月12日13时20分03秒   

## 一、新车磨合期注意事项 (前1500公里)
1.  **转速控制**：发动机转速不要超过 **4000转/分钟**。
2.  **车速控制**：建议最高车速不超过 **100公里/小时**。
3.  **避免恒定速度**：不要长时间以同一速度行驶（无论是高速还是低速），应适当变化车速，使各部件充分磨合。
4.  **避免重负荷**：不要满载或超载行驶，避免拖拽其他车辆或挂车。
5.  **温和驾驶**：避免急加速、急刹车和强制降档。
## 二、启动与熄火
1.  **启动前检查**：
    -   确认轮胎气压正常。
    -   检查机油、冷却液、制动液液位是否在标准范围内。
    -   调整好座椅、后视镜，系好安全带。
2.  **一键启动操作**：
    -   **自动挡 (DCT)**：踩住刹车踏板，按下一键启动按钮。挂入D档或R档前，确保车辆已完全启动且电子手刹已解除。
    -   **手动挡 (MT)**：踩下离合器踏板并踩住刹车，按下一键启动按钮。
3.  **冷车启动**：
    -   冷车启动后，建议原地怠速热车30秒至1分钟（冬季可适当延长），待转速回落至正常怠速后再缓慢行驶。无需长时间原地热车。
4.  **熄火操作**：
    -   **涡轮增压车型特别提醒**：在高速行驶或长途爬坡后，建议怠速运转1-2分钟再熄火，以保护涡轮增压器轴承。日常市区短途行驶无需此操作。
## 三、行驶中注意事项
### （一）变速箱使用 (重点)
-   **双离合变速箱 (DCT)**
    1.  等红灯时（30秒以上）：建议挂入N档并拉起电子手刹。不建议长时间挂D档踩刹车等待，以免变速箱过热和增加油耗。
    2.  拥堵路况/蠕行时：在极度拥堵走走停停的路段，可以尝试切换至手动模式（M档）并锁定在1档或2档，减少变速箱频繁换挡导致的离合器片磨损和过热风险。
    3.  禁止空挡滑行！严禁N档滑行！这会导致变速箱润滑不足而严重损坏。
-   **手动变速箱 (MT)**
    1.  换挡时务必踩死离合器踏板到底。
    2.  根据车速及时换挡（参考换挡提示灯），避免高档低速导致发动机积碳或低档高速导致油耗过高。
### （二）底盘与悬挂
-   SUV重心较高，过弯时请提前减速，切勿高速入弯以防侧倾过大失控。
-   通过减速带、坑洼路面时务必减速慢行（建议20km/h以下），保护悬挂系统和轮胎轮毂。
### （三）制动系统
-   新车刹车片需要磨合（约300公里），初期制动效果可能偏软，请保持更大跟车距离。
-   涉水后轻踩几次刹车以蒸发水分恢复制动效能。
## 四、特殊路况驾驶
### （一）雨天/湿滑路面
-   开启ESP车身稳定系统（默认开启）。
-   降低车速，增大跟车距离（至少为干燥路面的两倍）。
-   关闭自动启停功能（若有配备），防止涉水时发动机二次启动损坏。
### （二）山路/长下坡
-   DCT车型：使用手动模式将档位控制在低档位（如M2/M3），利用发动机制动辅助减速。严禁长时间连续踩刹车以防刹车热衰减失效！
-   MT车型：挂入低速档利用发动机制动。
### （三）高速公路
-   保持安全车距（至少保持“两秒法则”）。
-   注意胎压监测报警灯。夏季长途行车胎压不宜过高。
## 五、智能科技配置使用提醒
1.  **自适应巡航 (ACC)** / L2级辅助驾驶：
    -   ACC是辅助功能不是自动驾驶！双手必须始终握住方向盘并保持注意力观察路况。
    -   雨雪天气、大雾天气及弯道密集路段不建议使用ACC功能。
2.  **360度全景影像**：
    -   AVM影像存在盲区和畸变。倒车入库时仍需结合后视镜观察实际环境；夜间光线不足时清晰度会下降需谨慎判断距离。
3.  **自动驻车 (Auto Hold)**：
    倒车入库时建议关闭此功能以免影响精细操控油门响应节奏感不佳导致窜动剐蹭风险增加！
## ⚠️六、核心安全警告！
| ❌禁止事项 | ✅正确做法 |
| :--- | :--- |
| N挡滑行 | D/M/R正常行驶 |
| D挡+脚刹长时间等红灯 | N/P+电子手刹 |
| DCT激烈弹射起步 |平稳加速 |
| SUV当赛车开高速过弯 |提前减速慢过弯 |
|无视仪表盘故障灯亮起继续开 |立即靠边停车检查 |

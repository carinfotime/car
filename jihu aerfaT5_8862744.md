# 极狐 阿尔法T5 保养指南
  
更新时间：2026年07月17日10时43分57秒   

## 一、保养周期总览
| 保养项目 | 周期 |
|---------|------|
| 常规检查（三电系统、底盘、灯光等） | 每 **12个月** 或 **20,000公里**（以先到为准） |
| 空调滤芯更换 | 每 **12个月** 或 **20,000公里** |
| 减速器油更换 | 每 **48个月** 或 **80,000公里** |
| 冷却液更换 | 每 **48个月** 或 **80,000公里** |
| 制动液更换 | 每 **24个月** 或 **40,000公里** |
| 空调系统检查/制冷剂补充 | 按需或每 **24个月** |
> ⚠️ *纯电动车无发动机，无需更换机油、机滤、火花塞等传统燃油车项目。*

## 二、各阶段保养内容详解
### 🔹 A类保养（12个月 / 20,000公里）
-  高压三电系统检查（电池、电机、电控）
-  充电系统功能检测
-  底盘及悬挂系统检查
-  制动系统检查（刹车片厚度、制动盘磨损）
-  轮胎磨损及胎压检测
-  灯光及信号装置检查
-  雨刮器及洗涤液检查
-  空调滤芯清洁/更换
-  车载系统软件升级与故障码读取
### 🔹 B类保养（24个月 / 40,000公里）
包含A类全部项目，并增加：
-  制动液更换
-  空调滤芯更换
-  四轮换位及动平衡检测
-  转向系统全面检查
### 🔹 C类保养（48个月 / 80,000公里）
包含B类全部项目，并增加：
-  减速器油更换
-  冷却液更换
-  高压线束及连接器全面检查
-  电池健康度（SOH）专业检测

## 三、日常自查项目（建议每月一次）
| 项目 | 操作建议 |
|------|---------|
| 🛞 **轮胎** | -   检查胎压（标准值见驾驶侧门框标签）<br>-   观察胎面磨损是否均匀<br>-   清理胎纹中的石子杂物 |
| 🪟 **玻璃水** | -   及时补充专用玻璃清洗液<br>-   冬季使用防冻型玻璃水 |
| 💡 **灯光** | -   开启所有灯光，确认前后灯组正常<br>-   转向灯、刹车灯尤其重要 |
| 🧹 **车身清洁** | -   定期清洗车身，避免腐蚀性物质长期附着<br>-   充电口保持干燥清洁 |

## 四、关键部件养护说明
### 🔋动力电池养护
1. **充电建议**
   -   日常使用建议电量保持在 **20% ~ 90%**
   -   长途出行前可充至100%
   -   尽量避免频繁快充，慢充更有利于电池寿命
2. **温度管理**
   -   长期停放时尽量选择室内或阴凉处
   -   冬季用车前可提前通过APP预热电池（若有此功能）
3. **长期停放**
   -   若超过15天不使用，建议保持电量在50%~60%
   -   每月至少启动车辆并行驶一段距离，激活电池管理系统
### 🛞轮胎养护
-   标准胎压：通常为2.5 bar左右（具体以门框标签为准）
-   每10,000~20,000公里进行四轮换位，延长轮胎寿命
-   发现偏磨应及时做四轮定位
### ❄️空调系统养护
1. PM2.5过滤芯建议每年更换一次
2. EV车型空调压缩机为电动驱动，需定期检查制冷效果
3.    夏季使用前可提前预约开启空调降温

## ⚠️五、常见注意事项与警示
1. ❌ **禁止自行拆卸高压部件**
    动力电池及高压线束涉及安全风险，必须由授权服务站专业人员操作。
2. ✅ **务必使用原厂配件**
    尤其是减速器油、冷却液等需符合极狐原厂规格。
3. 📱 APP提醒功能利用好：
    极狐APP通常会推送保养提醒和车辆状态信息，请保持APP通知开启。
4. 📒保留保养记录：
    每次进店保养后保留工单和记录，有助于二手车保值和质保索赔。
5. 🚨仪表盘报警灯处理：
    若出现红色故障灯应立即停车联系售后；黄色警告灯可预约进店检查。
6. 🌧️涉水注意事项：
    虽然电动车涉水能力优于燃油车，但仍不建议长时间涉水行驶；水深超过轮毂一半时应绕行。
7. ❄️冬季用车提示：
    低温会导致续航下降属正常现象；提前预热车辆有助于恢复部分续航。

# 瑞虎5x 儿童乘车安全注意事项
  
更新时间：2026年07月14日16时56分28秒   

为了确保您孩子的乘车安全，请务必遵守以下事项：
## 1. 儿童安全座椅使用规范
- **强制使用**：12周岁以下或身高低于1.4米的儿童，必须使用与其年龄、体重相匹配的儿童安全座椅。
- **安装位置**：**最安全的座位是后排中间位置**。切勿将儿童安全座椅安装在副驾驶位。
- **安装检查**：请严格按照瑞虎5x车辆说明书及安全座椅说明书进行安装。安装后应用力摇晃座椅，确保其牢固不松动。
- **ISOFIX接口**：瑞虎5x后排座椅通常配备ISOFIX儿童座椅接口，请优先使用该接口固定，比仅用安全带固定更稳固。
## 2. 绝对禁止的行为
- **禁止抱坐**：严禁家长怀抱儿童坐在副驾驶或后排。发生碰撞时，巨大的冲击力会使成人无法抱住孩子。
- **禁止副驾驶位**：绝对不能让12岁以下儿童坐在副驾驶位。一旦发生碰撞，弹出的安全气囊会对儿童头部和颈部造成致命伤害。
- **禁止站立或探头**：严禁儿童在车辆行驶过程中站立在座位上或将头、手伸出天窗、车窗。
## 3. 车内环境与设备设置
- **车门儿童锁**：瑞虎5x后门通常配备有儿童安全锁。建议在搭载儿童时开启此功能（通常位于车门侧面），防止孩子在行驶中误开车门。
- **车窗控制**：建议开启驾驶员侧的车窗锁止功能（一键锁窗），防止孩子在后排随意升降车窗导致夹伤或探头危险。
- **天窗操作**：切勿让儿童自行操作天窗开关，避免夹伤风险。
## 4. 行车途中注意事项
- **勿留车内**：无论任何情况（如短暂停车、下车办事），都不要将孩子单独留在车内。夏季高温下，车内温度会迅速升至致命水平；冬季也存在缺氧风险。
- **避免零食与玩具**：车辆行驶中尽量避免给孩子喂食果冻、棒棒糖等易噎住的食物；避免携带尖锐或硬质玩具。
- **系好安全带**：对于已不使用安全座椅的大龄儿童（身高>1.4米），需确保其正确佩戴后排安全带（肩带跨过肩膀中间，腰带贴紧髋部）。
## 5. 应急准备
- **随车物品**：车内常备儿童的应急联系卡、常用药品及少量饮用水和零食。
- **告知逃生方法**：对于年龄较大的孩子，可以简单告知如何解锁车门或在紧急情况下如何呼救。

# 智己L6 儿童乘车安全注意事项
  
更新时间：2026年07月13日18时13分54秒   

## 🚗 基本安全须知
1. **12岁以下儿童** 建议使用符合国家标准的儿童安全座椅，并安装在后排。
2. **4岁以下儿童** 必须使用反向安装的儿童安全座椅。
3. **严禁将儿童单独留在车内**，尤其是在夏季高温或冬季低温环境下，以免发生中暑或冻伤风险。
## 🪟 车窗与车门安全
- **开启后排车窗童锁功能**：防止儿童在行驶中误开车门或车窗。
- **切勿让儿童将头、手或身体任何部位伸出天窗或侧窗**。
- 使用 **智己L6 的智能车窗防夹功能**，但仍需家长注意监督。
## 🎮 车内娱乐与交互系统
- **智己L6配备大屏交互系统**，请勿让儿童在行驶中操作中控屏幕或方向盘上的按键。
- 建议开启 **“儿童模式”**（如有），限制屏幕内容和操作权限。
- 避免让儿童长时间注视屏幕，以防晕车或视力疲劳。
## 🌡️ 空调与空气质量
- **智己L6配备智能空气净化系统**，建议开启内循环及PM2.5过滤功能，保障车内空气质量。
- 空调温度不宜过低（建议22–26℃），避免直吹儿童。
## 🔋 电动车特有注意事项
- **智己L6为纯电动车型**，起步加速较快。请平稳驾驶，避免急加速、急刹车导致儿童不适或受伤。
- **充电时请勿让儿童靠近充电口及充电桩**，以防触电风险。
## 📋 家长自查清单
| ✅ | 检查项目 |
|---|---------|
| ☐ | 后排车门童锁已开启 |
| ☐ | 车窗防夹功能正常 |
| ☐ | 安全座椅正确安装并固定 |
| ☐ | 车内无小零件、尖锐物品可被误吞 |
| ☐ | “儿童模式”已启用（如适用） |

## 三、详细保养项目说明
### （一）发动机机油与机滤
-   *推荐机油*：Castrol EDGE Professional C3 / Mobil1 ESP Formula
-   *粘度等级*：0W-40 / SAE C3
-   *容量*：约8.5L（含机滤）
-   *操作要点*：
    1.  热车后放油更彻底。
    2.  务必使用原厂或同等级高品质机滤。
    3.  加注后启动发动机怠速运行30秒，熄火等待5分钟后检查油位。
### （二）空气滤清器
-   使用原厂高流量干式空滤。
-   安装时确保密封圈完好，壳体卡扣锁紧到位。
-   不建议使用湿式油性空滤（可能污染MAF传感器）。
### （三）火花塞
-   原厂推荐 NGK ILKAR7B11 / Denso IKH24TT
-   间隙：0.9mm ~1.1mm
-   扭矩：25 N·m
-   拆卸时注意清理缸盖螺纹积碳。
### （四）制动系统
-   *制动液*：DOT4+ / DOT5.1（推荐 Castrol React SRF Racing）
-   *标准沸点*：≥260°C（干沸点）
-   *排空方法*：从最远轮开始依次排气（右后→左后→右前→左前）。
-   刹车片最小厚度不低于3mm。
### （五）冷却系统
-   使用 OAT (Organic Acid Technology)长效防冻液。
-   混合比例：50%防冻液 +50%去离子水。
-   排气需专用工具或通过排气螺栓手动排空空气。
### （六）变速箱油与差速器油
#### ZF8HP75变速箱
-   *油品*：ZF Lifeguard Fluid8 / Shell ATF134FE
-   *容量*：约9L（重力换油）/12L（循环机换油）
-   *注意*：
    1.  必须使用专用加油工具从侧边加油口加注至溢流孔出油为止。
    2.  换油时需同时更换变速箱油底壳及内置滤芯。
#### E-Diff电子限滑差速器
-   *油品*：Castrol Syntrax Long Life75W140 / Mobilube SHC75W90-LS
-   *容量*：约1.2L
-   必须添加限滑差速器添加剂（如已包含在配方中则无需额外添加）。

## ⚠️六、重要注意事项
1️⃣ DB12采用干式油底壳设计，日常行驶无需频繁补加机油，但赛道驾驶后务必检查。  
2️⃣ 车辆举升时必须使用指定顶升点——位于门槛下方四个专用橡胶垫块位置。严禁顶升底盘中部铝制结构件。  
3️⃣ 电瓶位于后备箱右侧盖板下。长时间停放建议连接智能充电器 CTEK MXS5.0(兼容AGM电瓶)。  
4️⃣ 所有保养操作完成后必须通过中控屏进入“Service Mode”复位保养提示灯。  
5️⃣ 保修期内强烈建议全程在授权经销商处进行保养并保留完整记录以维持质保有效性。  
6️⃣ 改装排气/ECU程序将影响排放合规性及发动机保修条款，请谨慎评估后再行操作。

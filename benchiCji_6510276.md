# 奔驰C级 驾驶注意事项
  
更新时间：2026年07月12日09时12分06秒   

## 一、启动与预热
1.  **启动步骤**：踩住刹车，按下一键启动按钮。对于配备48V轻混系统的车型，启动更为平顺安静。
2.  **冷车预热**：冷车启动后，建议怠速运转30秒至1分钟，待转速表指针回落至正常怠速区间（约700-800转）后再起步。无需长时间原地热车。
3.  **起步初期**：在水温达到正常工作温度前，避免发动机转速超过3000转或急加速行驶。
## 二、行驶中操作
### 变速箱使用
-   **换挡逻辑**：奔驰C级通常配备9速自动变速箱（9G-TRONIC）。D挡下变速箱会积极升挡以保持燃油经济性。
-   **手动模式**：需要激烈驾驶或长下坡控制车速时，可将档位切换至M模式或使用换挡拨片。
-   **等红灯**：短时间等待（30秒内）可挂D挡踩刹车；长时间等待建议挂N挡或开启自动驻车功能（HOLD）。
    -   *激活HOLD功能*：车辆停稳后，快速深踩一脚刹车，仪表盘会显示“HOLD”标识。
### 驾驶模式选择
-   **Eco（经济模式）**：适合城市拥堵路况或巡航。油门响应变缓，变速箱升挡积极，滑行时可进入空档滑行状态以省油。
-   **Comfort（舒适模式）**：日常驾驶推荐。兼顾动力与平顺性，悬挂调校偏软（如配备空气悬挂）。
-   **Sport/Sport+（运动/运动增强模式）**：适合高速超车或山路。油门响应灵敏，变速箱降挡迅速，悬挂变硬以提供更好支撑。
-   **Individual（个性化模式）**：可根据个人喜好自定义动力、转向、悬挂等参数。
### 自动启停系统
-   系统默认开启。在频繁启停的严重拥堵路段或在涉水行驶前，建议手动关闭此功能。
## 三、安全与辅助系统
1.  **主动制动辅助系统**：
    -   系统会在检测到碰撞风险时发出警告并主动制动。切勿完全依赖此系统，驾驶员需时刻保持专注。
2.  **盲点辅助系统**：
    -   当侧后方有车辆接近时，后视镜上的红色三角警示灯会亮起。若此时打转向灯变道，系统会发出声音警告并可能进行单侧制动干预。
3.  **车道保持辅助系统**：
    -   当车辆无意识偏离车道时，方向盘会震动提醒并轻微修正方向。打转向灯变道时不会触发干预。
4.  **自适应巡航控制（ACC）**：
    -   设定好速度和跟车距离后车辆可自动跟走跟停。使用时仍需手握方向盘并观察路况。
## 四、停车与熄火
1.  **停车步骤**：
    -   踩刹车将车辆完全停稳 -> 挂入N挡 -> （可选）拉电子手刹 -> （可选）松开脚刹释放应力 -> （可选）再次踩脚刹 -> 挂入P挡 -> （可选）再次拉电子手刹 -> 熄火。
2.  *简化步骤*：直接P挡熄火即可（电子手刹通常会自动拉起）。
3.  **方向盘归正**：熄火前确保方向盘回正到位。
4.  **关闭用电设备**：熄火前关闭空调、音响、灯光等用电设备。
## ⚠️特别注意事项
1.  **轮胎气压监测**
    定期检查胎压。C级通常配有实时胎压监测系统；若仪表盘出现胎压报警提示请及时检查轮胎状况并校准胎压值。
2.  **雨刮器维护**
    冬季停车时请将雨刮器竖起；夏季高温暴晒后勿直接用雨刮器干刮玻璃以免损伤胶条及玻璃表面涂层。
3.  **燃油标号**
    推荐使用95号及以上无铅汽油；加注劣质燃油可能导致发动机故障灯亮起及性能下降。

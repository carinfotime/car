# 五菱缤果 技术参数介绍
  
更新时间：2026年07月17日15时09分18秒   

## 一、车型概览
五菱缤果是上汽通用五菱推出的一款纯电动小型车，主打城市通勤与代步市场。凭借圆润复古的外观设计、灵活的车内空间以及较低的用车成本，自上市以来获得了较高的市场关注度。
## 二、动力系统
| 项目 | 参数 |
| :--- | :--- |
| **驱动电机类型** | 永磁同步电机 |
| **电机布局** | 前置前驱 |
| **最大功率** | 30kW（标准版） / 50kW（长续航/灵犀版） |
| **最大扭矩** | 110N·m（标准版） / 150N·m（长续航/灵犀版） |
| **最高车速** | 100 km/h |
| **0-50km/h加速时间** | 约5.5秒（50kW版本） |
## 三、电池与续航
| 项目 | 参数 |
| :--- | :--- |
| **电池类型** | 磷酸铁锂电池 |
| **电池容量** | 17.3 kWh（标准版）/ 31.0 kWh（长续航版）|
| **CLTC纯电续航里程** | 203km（标准版）/333km（长续航版）|
| **百公里耗电量** | 约10.2 -11.0 kWh/100km |
| **充电方式** |  交流慢充 + （部分高配支持直流快充）|
| **快充时间 (30%-80%)** |  约35分钟 (仅支持快充的版本) |
> *注：具体续航里程受驾驶习惯、环境温度及路况影响。*
## 四、车身尺寸与空间
### 🔹外观尺寸
-   **长 ×宽 ×高 (mm)**：3950 ×1708 ×1580
-   **轴距 (mm)**：2560
-   **整备质量 (kg)**：约990 -1125 kg (因配置而异)
-   **最小离地间隙 (mm)**：约150
### 🔹内部空间
-   **座位数**：5座
-   **后备箱容积 (L)**：
    -   常规状态：约110 L
    -   后排座椅放倒后：约790 L
## 五、底盘与操控
-   **驱动方式**：前置前驱
-   **前悬挂类型**：麦弗逊式独立悬挂
-   **后悬挂类型**：扭力梁式非独立悬挂
-   **转向系统**：电动助力转向(EPS)
-   **制动系统 (前/后)**：前通风盘式 /后鼓式刹车
## ⚙️六、主要配置亮点
### 🎨外观与内饰
-   提供多种双拼色车身配色方案。
-   全系标配LED大灯及日间行车灯。
-   内饰采用双10.25英寸联屏设计(部分高配)。
### 🛡️安全配置
-   主/副驾驶座安全气囊。
-   胎压监测系统。
-   倒车雷达 +倒车影像。
-   部分高配车型配备ESC车身电子稳定系统。
### 📱智能互联(部分高配)
-   支持手机App远程控制(车门解锁/闭锁、空调开关)。
-   语音识别控制系统。
-   OTA远程升级。

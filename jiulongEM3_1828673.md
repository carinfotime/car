# 九龙EM3 驾驶注意事项
  
更新时间：2026年07月17日08时11分39秒   

## 一、启动与行驶前检查
1.  **钥匙与档位**：确保钥匙处于“ON”档，仪表盘自检正常。挂挡前必须确认车辆已完全静止，踩住刹车。
2.  **仪表盘检查**：启动后观察仪表盘是否有故障灯亮起（如电池、电机、系统故障灯）。
3.  **轮胎检查**：定期检查胎压（建议冷车胎压2.3-2.5 Bar），观察轮胎磨损及表面有无异物。
4.  **灯光与信号**：检查大灯、转向灯、刹车灯、倒车灯及雾灯是否正常工作。
5.  **后视镜调整**：调整内外后视镜至最佳视野，消除盲区。
## 二、行驶中的操作规范
1.  **起步与加速**：
    -   缓慢踩下加速踏板，避免急加速导致电能消耗过快或电机过载。
    -   起步时注意周围行人及非机动车，因电动车低速行驶噪音极低。
2.  **车速控制**：
    -   遵守道路限速标志。九龙EM3最高设计时速通常较低（约80-100km/h），请勿超速行驶。
    -   高速行驶时（>60km/h）能耗会显著增加，续航里程会明显缩短。
3.  **制动技巧**：
    -   **预判性驾驶**：提前松加速踏板，利用能量回收系统减速（松开踏板即有制动感）。
    -   **紧急制动**：遇到突发情况果断踩死刹车踏板（配备ABS的车型可放心重刹）。
    -   **避免空挡滑行**：电动车空挡滑行不仅危险，且无法进行能量回收。
4.  **转向与变道**：
    -   EM3车身较高且窄，过弯时注意控制车速（建议30km/h以下），防止侧倾。
    -   变道前务必打转向灯并观察后视镜盲区。
## 三、特殊路况驾驶
1.  **雨天/湿滑路面**：
    -   EM3轮胎较窄，抓地力有限。减速慢行，避免急刹或猛打方向。
    -  积水路段谨慎通过，水深不超过轮毂中心线；涉水后轻踩刹车排除水分。
2.  **坡道行驶**：
    -   EM3可能未配备坡道辅助功能。坡道起步时需快速从刹车切换至油门，配合手刹防止溜车。
3.  **夜间行车**：
    -   EM3大灯亮度可能有限，适当降低车速。注意对向车辆灯光干扰。
## 四、充电与电池维护
1.  **充电时机**：建议电量剩余20%-30%时进行充电，避免过度放电损伤电池寿命。
2.  **充电规范**：
    -  使用原厂或符合国标的充电桩/线缆。严禁使用不合格插座或飞线充电。
    -  充电前检查充电枪和接口是否干燥清洁。雷雨天气尽量避免户外充电。
3.  **冬季保养**：
    -  低温会导致续航大幅缩水（约30%-50%）。冬季尽量在室内或温度较高的环境下充电。
4.  **长期停放**：若长时间不使用车辆，保持电量在50%-70%之间停放；每隔一个月补充一次电。
## 五、安全与应急处理
1.  **安全带提醒**：所有乘员必须系好安全带。EM3为微型车结构吸能空间有限，安全带至关重要。
2.  **儿童安全锁**：如有儿童乘坐后排（若存在后排），务必启用两侧车门儿童安全锁。
3.  **故障应急处理**
|情况|处理方式|
|---|---|
|仪表盘亮红灯|立即靠边停车熄火并联系售后|
|动力突然中断|打开双闪警示灯→利用惯性靠边→放置三角警示牌于车后50米外|
|底盘磕碰/进水|立即停车断电并呼叫救援|
## ⚠️特别提示
-   [ ] EM3属于低速/微型电动车范畴？请确认当地法规对上路要求及驾照类型的规定
-   [ ] A柱较粗壮转弯时注意左右摆头观察
-   [ ]空调制热耗电极大冬季非必要不开暖风以保续航

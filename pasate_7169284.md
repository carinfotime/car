# 帕萨特驾驶注意事项
  
更新时间：2026年07月10日19时54分20秒   

## 一、启动与预热
1. **钥匙操作**：插入钥匙后，先拧至“ON”档（通电自检），等待仪表盘警示灯熄灭（约3-5秒）后再启动发动机。
2. **冷车启动**：冬季或长时间停放后，建议怠速热车 **30-60秒**，待转速回落至800转左右再缓慢行驶。无需长时间原地热车。
3. **一键启动**：不踩刹车按一下为通电，踩刹车按一下为启动。确保智能钥匙在车内。
## 二、行驶中操作
### 1. 变速箱使用（DSG双离合/自动挡）
- **D档**：日常行驶使用。
- **S档**：超车或爬坡时使用，动力响应更快，油耗会升高。
- **手动模式**：长下坡或山路建议切换手动模式，利用发动机制动，减少刹车过热风险。
- **等红灯**：短时间（30秒内）可D档踩刹车；长时间建议挂N档拉手刹（或开启Auto Hold）。
- **禁止空挡滑行**：N档滑行不仅不省油，还会损坏变速箱。
### 2. Auto Hold（自动驻车）
- 开启后，车辆停稳会自动拉手刹，轻点油门即可解除。
- 倒车入库时建议关闭此功能，避免油门控制不当导致窜车。
### 3. 启停系统
- 拥堵路段或涉水行驶时，建议手动关闭自动启停功能。
- 启停工作时空调制冷效果会下降。
### 4. ESP车身稳定系统
- 默认开启。遇到雨雪天气切勿关闭。
- 仅在陷入泥地、沙地需要脱困时短暂关闭。
## 三、特殊路况驾驶
| 路况 | 注意事项 |
|------|----------|
| **雨天/积水** | - 水深超过半个轮胎高度请勿强行通过<br>- 涉水后轻踩刹车数次，蒸发刹车盘水分 |
| **高速行驶** | - 胎压按标准值调整（通常前2.3bar/后2.5bar）<br>- 避免长时间定速巡航导致疲劳 |
| **山路/长下坡** | - S档或手动模式低档位下坡<br>- L挡（如有）用于陡坡缓降 |
| **冰雪路面** | - ESP保持开启<br>-起步用2挡或雪地模式<br>-保持更大跟车距离 |
## 四、日常检查与维护
1. **机油检查**：冷车状态下拔出机油尺擦净再插入查看液位应在上下刻度之间。每1000公里检查一次。
2. **冷却液**：粉色液体储液罐液位在MIN-MAX之间。不足时添加同品牌同颜色冷却液。
3. **玻璃水**：冬季需更换防冻型玻璃水（标号低于当地最低气温）。
4. **轮胎磨损**：
   - TWI磨损标记露出即需更换
   - DSG车型前轮磨损较快
   -每20000公里前后轮交叉换位
5. **刹车片报警线磨损提示**:仪表出现刹车报警需立即检查更换。
## ⚠️五、常见故障预警
|现象|可能原因及处理|
|---|---|
|EPC灯亮 +加速无力|节气门脏污/传感器故障 →清洗节气门并匹配|
|变速箱顿挫/异响(低速)|DSG离合器片磨损 →回4S店升级程序或检修|
|烧机油(排气管冒蓝烟)|活塞环卡滞 →使用高粘度机油并定期清理积碳|
|天窗漏水(顶棚湿)|排水管堵塞 →定期疏通天窗排水孔|
## 📋六、其他重要提醒
1. 🚫严禁加注劣质燃油——帕萨特对油品敏感,易引发发动机故障灯亮起,推荐95号及以上汽油.
2. 🔧保养周期:涡轮增压车型建议每7500公里或半年保养一次(全合成机油).
3. 🅿️停车熄火前:涡轮增压车型长途高速后应怠速运转1~2分钟再熄火,保护涡轮轴承.
4. 🔑遥控钥匙电量不足:靠近方向盘感应区仍可应急启动.
5 .📱手机支架位置:不要遮挡中控出风口——影响空调效率且易导致手机过热.

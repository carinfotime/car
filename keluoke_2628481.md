# 斯柯达柯珞克（Skoda Karoq）操作指南
  
更新时间：2026年07月20日15时48分29秒   

## 一、启动与熄火
### 1.1 启动车辆
- **钥匙启动车型**：插入钥匙 → 踩下刹车 → 转动钥匙至 START 位置，发动机启动后松开。
- **一键启动车型**：确保智能钥匙在车内 → 踩下刹车 → 按下“ENGINE START/STOP”按钮。
> **注意**：冷车启动后建议怠速运转约30秒再行驶，让机油充分润滑。
### 1.2 熄火
- **钥匙启动车型**：将钥匙转回 OFF 位置并拔出。
- **一键启动车型**：挂入 P 挡 → 拉起电子手刹 → 按下“ENGINE START/STOP”按钮熄火。

## 二、驾驶模式选择
柯珞克通常提供多种驾驶模式，通过中控附近的“MODE”按钮或驾驶模式选择旋钮切换：
| 模式 | 适用场景 | 特点 |
|------|----------|------|
| **经济（Eco）** | 城市拥堵、长途巡航 | 油门响应变缓，空调功率降低，更省油 |
| **标准（Normal）** | 日常通勤 | 均衡的油门与转向力度 |
| **运动（Sport）** | 山路、超车、激烈驾驶 | 油门更灵敏，转向加重，换挡时机延后 |
| **个性化（Individual）** | — | 可自定义发动机、转向、空调等参数 |
---
## 三、挡位操作（7速DSG双离合变速箱）
- **P（驻车挡）**：停车熄火时使用。必须在车辆完全停止后才能挂入。
- **R（倒车挡）**：车辆停稳后挂入。
- **N（空挡）**：短暂停留或拖车时使用。
- **D（前进挡）**：日常行驶使用。变速箱自动换挡。
- **S（运动挡）**：在 D 挡位置向后拉一下即可切换。延迟升挡以获得更强动力。
> **手动模式**：将挡杆向右推入“+/-”槽位，向前推升挡，向后拉降挡。仪表盘会显示当前推荐挡位。

## 四、方向盘与仪表盘
### 4.1多功能方向盘
- **左侧区域**：
    - `+ / -`：调节音量
    - `滚轮`：切换电台/曲目
    - `语音键`：激活语音控制
    - `电话键`：接听/挂断电话
- **右侧区域**：
    - `左右箭头` / `滚轮`：切换仪表盘显示信息
    - `VIEW` / `菜单键`：切换仪表盘视图风格
###  4.2仪表盘信息查看
通过方向盘右侧滚轮可循环查看：
-  当前车速 / RPM
-  平均油耗 /瞬时油耗 /续航里程
-  行驶时间 /平均速度
-  导航提示 /交通标志识别

##  五、灯光控制
灯光旋钮位于方向盘左侧：
|  旋钮位置     |  功能             |
|---------------|-------------------|
| OFF           |关闭所有灯光       |
| AUTO          |自动大灯           |
| （示宽灯图标）|示宽灯             |
| （近光灯图标）|近光灯             |
>  雾灯开关：将旋钮向外拉出即可开启前后雾灯。

## ⚙️六、空调与座椅加热
###  6.1自动空调操作
- `AUTO` ：设定目标温度后自动调节风量及风向。
- `AC` ：开启压缩机用于制冷或除湿。
- `SYNC` ：同步主副驾驶温度设置。
- `前风挡除雾键` ：快速清除前风窗雾气。
### ⚡6.2座椅加热
中控台下方或空调面板上有座椅加热按键：

按一次 → Level3 (最高)
按两次 → Level2 (中等)
按三次 → Level1 (低热)
按四次 → OFF (关闭)

## 🅿️七、泊车辅助系统
### 📡7.1倒车雷达 &影像
挂入 R 档时自动激活：
✅中控屏显示后方影像及雷达警示线  
✅前后雷达通过蜂鸣声频率提示障碍物距离  
### 🅿️7.2自动泊车辅助  
1️⃣按下中控台附近的“P 🅿️”按钮  
2️⃣打转向灯选择停车侧  
3️⃣缓慢前行直至系统找到合适车位  
4️⃣按照提示挂入 R/D 档并控制刹车

## 🔐八、电子手刹与 Auto Hold  
### 🧲8.1电子手刹  
向上拨起 = 制动锁止  
向下按压 = 解除制动  
> 💡起步时系好安全带 + 关好车门 + 轻踩油门会自动解除手刹  
### ⏸️8.2 Auto Hold (自动驻车)  
按下 Auto Hold 按钮(指示灯亮起)即可激活。  
等红灯时深踩刹车→仪表出现绿色 HOLD 图标→松开脚刹车辆保持静止；轻点油门即可继续行驶。

## 🛞十、轮胎与胎压监测  
⚠️胎压报警时请立即检查轮胎状况。确认无扎破后可通过中控设置重置胎压监测系统(需在冷胎状态下进行)。  
标准参考胎压(以油箱盖内侧标签为准)：  

前轮: 2.3 bar ~ 2.5 bar  
后轮: 2.3 bar ~ 2.5 bar (满载时可适当提高至2.7~2.9 bar)  

---
## 🧰十一、应急操作  
🔑遥控钥匙没电时可用机械钥匙片手动解锁车门(按压遥控器侧面释放按钮取出)。  
🛞更换备胎步骤简记:  
①取出备胎及千斤顶工具包(位于后备箱地板下)   
②用千斤顶顶住底盘指定支撑点   
③拧松螺栓→取下破损轮胎→安装备胎→对角顺序拧紧螺栓   
④降下千斤顶后再做最终紧固   
🚨发动机无法启动时可尝试搭电:正极接正极,负极接车身金属搭铁点(不要直接接亏电电池负极)。

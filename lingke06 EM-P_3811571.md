## 领克06 EM-P 动力参数介绍
  
更新时间：2026年07月19日09时43分46秒   

### 动力系统总览
领克06 EM-P 搭载了 Lynk E-Motive 智能电混系统，由一台混动专用发动机、一台前置驱动电机以及一个3挡DHT Pro混动专用变速箱组成。这套系统旨在平衡强劲的动力输出与出色的燃油经济性。
### 核心动力参数
| 项目 | 参数 |
| :--- | :--- |
| **动力类型** | 插电式混合动力 (PHEV) |

| **发动机最大功率** | 120 kW (约163马力) / 5500 rpm |
| **发动机峰值扭矩** | 255 N·m / 2500-4000 rpm |
| **驱动电机** | 前置永磁同步电机 |
| **电机最大功率** | 160 kW (约218马力) |
| **电机峰值扭矩** | 350 N·m |
| **系统综合功率** | **220 kW (约299马力)** |
| **系统综合扭矩** | **578 N·m** |

### DHT Pro变速箱特点
3挡DHT Pro是这套系统的核心之一，它能够实现：
1.  **智能切换模式**: 
    *   EV模式（纯电）
    *   串联模式（发动机发电，电机驱动）
    *   并联模式（发动机和电机共同驱动）
2.  **全速域并联**: 
    在时速20km/h以上即可进入并联模式，让发动机在高效区间参与直驱或与电机共同发力。
3.  **3个物理挡位**: 
    使发动机能够更灵活地匹配不同车速和负载工况，提升全速域的动力响应和燃油经济性。
### E-Motive工作模式简介
领克06 EM-P提供多种驾驶模式以适应不同场景：
*   **[纯电]**：适合城市通勤，零油耗。
*   **[性能]**：发动机与电机全力输出，获得最佳加速性能。
*   **[超级增程]**：主要依靠电机驱动，发动机在高效区间作为增程器发电。
*   **[智能]**：系统根据行驶状态自动切换最合适的动力输出模式。

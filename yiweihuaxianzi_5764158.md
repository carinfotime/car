# 钇为花仙子 保养指南
  
更新时间：2026年07月18日08时18分16秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 首次保养 | 购车后 **3个月或5000公里**（以先到为准） |
| 常规保养 | 每 **6个月或10000公里**（以先到为准） |
| 制动液更换 | 每 **2年或40000公里** |
| 空调滤芯更换 | 每 **6个月或10000公里** |
| 减速器油更换 | 每 **2年或40000公里** |
| 冷却液更换 | 每 **3年或60000公里** |
> ⚠️ **注意**：纯电动车无发动机，无需更换机油、机滤、火花塞等传统燃油车项目。
---
## 二、日常自检项目（建议每周一次）
### 🔋 高压电池系统
- **电量检查**：保持电量在20%-80%之间，避免长期亏电或满电存放
- **充电状态**：观察充电是否正常，充电接口有无松动、过热现象
- **续航表现**：如发现续航明显下降，及时进店检查电池健康度
### 🛞 轮胎检查
- **胎压标准**：建议冷态胎压保持在 **2.3-2.5 bar**
- **胎面磨损**：花纹深度不低于1.6mm，注意有无偏磨、鼓包
- **轮胎换位**：每行驶10000公里进行前后轮交叉换位
### 💡灯光与信号
- 检查大灯、转向灯、刹车灯、雾灯是否正常工作
- 检查喇叭功能是否正常
### 🧹雨刮与玻璃水
- 雨刮片如有异响或刮不干净，及时更换（建议6个月一换）
- 玻璃水定期加注，冬季使用防冻型玻璃水
---
## 三、定期进店保养项目明细
### 🔹 A类保养（每6个月/10000公里）
| 检查项目 | 说明 |
|---------|------|
| ✅ OBD系统诊断 | 读取故障码，检查整车电控系统 |
| ✅高压电池检测 | BMS系统数据读取，电池均衡性检查 |
| ✅驱动电机检测 |电机运行状态、温度、噪音检查 |
| ✅减速器油位/状态 |油量及油质检查（视情况添加）|
| ✅制动系统检查 |刹车片厚度（前≥3mm，后≥2mm）、刹车盘磨损 |
| ✅底盘及悬挂系统 |螺栓紧固情况、减震器有无漏油 |
| ✅转向系统 |转向拉杆球头间隙、方向机工作状态 |
| ✅空调滤芯清洁/更换 |视脏污程度决定清洁或更换 |
| ✅冷却液液位及冰点检测 |不足时补充同型号冷却液 |
### 🔸 B类保养（每2年/40000公里）
在A类基础上增加：
- 🔧 **制动液更换**
- 🔧 **减速器油更换**
- 🔧 **空调滤芯强制更换**
- 🔧 **全车线束及插接件紧固与清洁**

## 四、关键部件维护要点
### ⚡高压电池（最核心部件）
1. **充电规范**
   - ⚡优先使用家用慢充桩充电
   - ⚡快充仅用于应急，不宜频繁使用
   - ⚡避免电量低于10%仍继续行驶
   - ⚡夏季高温时避免刚停车立即快充
2. **存放注意事项**
   - 📌长期停放（超过15天）：保持电量在50%-60%
   - 📌停放环境：干燥通风，避免阳光直射和高温暴晒
3. ❗若仪表盘出现“电池故障”报警灯，请立即停止驾驶并联系售后。
### ❄️空调系统
- 🌀定期开启空调制冷/制热各运行10分钟以上，防止压缩机卡滞
- 🌀夏季使用前建议进行空调杀菌消毒处理
- 🌀异味明显时需清洗蒸发箱
### 🛞制动系统
纯电动车因能量回收机制的存在，机械刹车使用频率较低：
- ⚠️建议每月在安全路段进行一次急刹操作以清除刹车盘表面锈迹
- ⚠️定期检查刹车分泵活动是否灵活
---
## 五、季节性保养提醒
### 🌸春季（雨季）
✅天窗排水孔清理  
✅车门密封条检查  
✅雨刮片提前更换  
✅底盘防锈处理  
### ☀️夏季（高温）
✅胎压适当降低至2.2bar  
✅冷却液冰点检测  
✅空调制冷效果测试  
✅避免暴晒后立即充电  
### 🍂秋季（换季）
✅轮胎花纹深度确认  
✅灯光全面检修  
✅玻璃水换成防冻型  
### ❄️冬季（低温）
❄️续航会下降约30%-40%，属正常现象  
❄️提前预热车辆再行驶可减少能耗  
❄️冰雪路面注意胎压不宜过低  
❄️充电尽量选择室内或有遮蔽的场所  
---
## ✨六、车主日常养护小贴士
1. 📖随车携带《用户手册》，按手册要求执行保养计划。
2. 💳首次保养免费且必须按时完成，逾期可能影响质保权益。
3. 📱建议通过官方APP预约维保服务。
4. 💰三电系统（电池+电机+电控）通常享有8年/15万公里质保。
5. 🛑非官方改装电路可能导致失去整车质保。
6. 🧼车身清洗时避开高压水枪直射充电口区域。
7. 📝每次进店保养后保留好工单和记录。

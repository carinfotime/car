# 轩度EV 内饰参数介绍
  
更新时间：2026年07月20日17时47分39秒   

## 一、整体设计风格
轩度EV内饰采用简约科技风设计，整体布局以驾驶者为中心，强调实用性与舒适性的结合。
## 二、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 仿皮/织物混搭（根据配置不同） |
| 主驾驶座椅调节 | 手动6向调节（部分高配支持电动调节） |
| 副驾驶座椅调节 | 手动4向调节 |
| 前排座椅功能 | 部分车型配备加热功能 |
| 后排座椅 | 整体放倒（比例放倒视配置而定） |
| 座椅填充物 | 高密度发泡海绵，支撑性良好 |
## 三、仪表盘与中控
- **仪表盘**：全液晶仪表盘，尺寸为 **7英寸**（部分低配为机械指针+小屏组合）
- **中控屏幕**：悬浮式触控屏，尺寸为 **10.25英寸**
- **车机系统**：支持蓝牙连接、USB多媒体播放、导航（部分车型）、语音控制
- **手机互联**：支持 CarPlay / CarLife（视具体配置）
## 四、方向盘
- **材质**：多功能方向盘，塑料/皮质包裹（高配）
- **调节方式**：手动上下调节
- **功能按键**：音量控制、蓝牙电话、菜单切换等
## 五、空调系统
- **空调类型**：手动空调（标配）/自动空调（高配）
- **后排出风口**：部分车型配备
- **PM2.5过滤装置**：标配
- **空气净化系统**：高配车型可选装
## 六、储物空间
| 位置 | 说明 |
|------|------|
| 中央扶手箱 | ✅，深度适中 |
| 手套箱 | ✅，带阻尼开启 |
| 门板储物格 | ✅，可放置水瓶及杂物 |
| 杯架 | ✅，前排2个+后排中央扶手杯架（部分车型） |
| USB接口数量 | ⚡前排1~2个 +后排1个 |
## 七、内饰颜色选项
- **深色内饰**（黑色为主）
- **浅色内饰**（米色/灰色搭配）
> *注：具体颜色搭配以实际在售车型为准。*
## ⚠️说明
以上参数基于轩度EV常见配置整理，实际内饰配置可能因车型年款、版本及选装包不同而有所差异。建议以实车或官方最新资料为准。

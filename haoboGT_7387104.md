# 昊铂GT 内饰参数介绍
  
更新时间：2026年07月12日13时24分00秒   

## 一、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | Nappa真皮（部分版本）/ 仿皮材质 |
| 前排座椅功能 | 电动调节（主驾12向/副驾8向）、加热、通风、记忆功能 |
| 后排座椅 | 支持比例放倒（4/6分割） |
| 座椅布局 | 2+3五座布局 |
| 头枕调节 | 前后排均支持高度调节 |
## 二、仪表与中控
| 项目 | 参数 |
|------|------|
| 仪表盘 | **8.8英寸**全液晶仪表盘 |
| 中控屏 | **14.6英寸**悬浮式中控触摸屏 |
| HUD抬头显示 | **AR-HUD**增强现实抬头显示系统（部分配置） |
| 车机系统 | ADiGO智驾互联系统，支持OTA远程升级 |
## 三、方向盘
- **类型**：D型平底多功能方向盘
- **材质**：真皮包裹
- **调节方式**：手动上下+前后调节
- **功能按键**：多媒体控制、语音控制、定速巡航/辅助驾驶控制
## 四、空调系统
- **空调类型**：自动双区恒温空调
- **后排出风口**：标配
- **PM2.5过滤**：配备空气净化系统及PM2.5过滤功能
- **香氛系统**：部分版本配备智能香氛系统
## 五、音响系统
- **扬声器数量**：23扬声器（高配）/基础版标配数量视配置而定
- **音响品牌**：Premium音响系统（部分版本）
- **环绕音效**：支持7.1.4全景声布局
## 六、内饰颜色与风格
### 🔹内饰配色方案：
1. **深色系**
   - ➤全黑内饰 +银色饰条点缀
   - ➤黑灰双拼色搭配
2. **浅色系**
   - ➤米白色/象牙白内饰 +木纹饰板（部分版本）
3. **运动风格**
   - ➤黑红撞色搭配 +碳纤维纹理饰板（运动版）
### 🔹饰板材质：
- ➤金属拉丝饰板
- ➤钢琴烤漆面板
- ➤仿碳纤维纹理面板（运动版）
- ➤木纹饰板（豪华版）

# 思皓E50A 空间参数介绍
  
更新时间：2026年07月11日18时23分27秒   

## 车身尺寸
| 项目 | 参数 |
|------|------|
| 车长（mm） | 4770 |
| 车宽（mm） | 1820 |
| 车高（mm） | 1510 |
| 轴距（mm） | **2760** |
| 前轮距（mm） | 1555 |
| 后轮距（mm） | 1550 |
## 车内乘坐空间
- **前排头部空间**：约980mm，身高175cm的乘客头部距车顶约一拳余量
- **后排头部空间**：约920mm，得益于溜背造型优化，后排头部空间表现良好
- **后排腿部空间**：约850~900mm，轴距2760mm带来充裕的后排腿部空间，膝部距离前排座椅约两拳以上
- **后排横向空间**：可轻松容纳三名成年人，肩部空间不显局促
## 座椅布局与功能
- **座椅布局**：标准5座布局（2+3）
- **前排座椅**：支持多向手动/电动调节，驾驶席带腰托调节
- **后排座椅**：支持4/6比例放倒，扩展储物能力
- **后排中央扶手**：带杯架设计
## 储物空间
### 后备箱
| 项目 | 参数 |
|------|------|
| 后备箱容积（标准状态） | **540L** |
| 后备箱容积（座椅放倒后） | **约1650L** |
| 后备箱开口宽度 | ~1050mm |
| 后备箱进深 | ~1000mm（放倒后可达~1850mm） |
### 车内储物细节
- **中控台杯架**：前排2个，后排中央扶手2个
- **门板储物格**：四门均配备，可放置水瓶及杂物
- **手套箱**：常规容积，可存放文件、小物件
- **中央扶手箱**：深度适中，可放置随身物品
- **手机储物区**：中控下方设有手机专用放置位
## EV专属空间优势
- **纯电平台优化**：电池包平铺于底盘下方，不侵占车内乘坐与后备箱空间
- **平整后排地台**：后排中央地台几乎纯平，中间乘客脚部舒适度高
- **前备箱（部分配置）**：机舱盖下设有小型储物区，可用于存放充电线等物品
---
> **总结**：思皓E50A凭借2760mm的长轴距和纯电平台优势，在紧凑型轿车级别中提供了越级的乘坐与储物空间表现。540L的大容量后备箱配合可放倒后排座椅，日常家用或长途出行均能从容应对。

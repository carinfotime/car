# 瑞驰C3 动力参数介绍
  
更新时间：2026年07月18日11时04分10秒   

瑞驰C3作为一款纯电动微型车，其动力系统主要针对城市短途通勤和日常代步场景设计，核心特点是**能耗低、起步轻快、维护简单**。
## 一、驱动电机
| 项目 | 参数 |
| :--- | :--- |
| **电机类型** | 永磁同步电机 |
| **最大功率** | **30 kW (约40.8马力)** |
| **最大扭矩** | **150 N·m** |
> **解读：**
> *   **功率方面：** 30kW的最大功率虽然数值不大，但对于整备质量较轻的微型车而言，足以满足城市道路的行驶需求。
> *   **扭矩方面：** 150N·m的峰值扭矩是电动车的优势所在。得益于电机“零转速即可输出最大扭矩”的特性，车辆在起步和低速加速时表现轻快、直接，红绿灯起步响应迅速。
## 二、电池与续航
| 项目 | 参数 |
| :--- | :--- |
| **电池类型** | 磷酸铁锂电池 / 三元锂电池 (根据具体配置版本) |
| **电池容量** | 约 **26.8 kWh - 35 kWh** (不同批次/配置有差异) |
| **NEDC续航里程** | **200 km - 300 km** (视具体电池容量版本而定) |
> **解读：**
> *   **续航定位：** NEDC续航里程覆盖200-300公里区间。在夏季或春秋季节，实际城市续航可达标称的80%-90%左右；冬季低温环境下续航会有一定衰减。
> *   **充电方式：**
>     *   支持家用220V交流慢充（随车配备充电枪）。
>     *   部分高配版本支持直流快充（需使用符合国标的快充桩），从20%充至80%电量约需1-1.5小时。
## 三、底盘与传动
| 项目 | 参数 |
| :--- | :--- |
| **驱动方式** | **后置后驱 (RR)** |
| **变速箱类型** | **电动车单速变速箱 (固定齿比)** |
| **最高车速** | ≥100 km/h |
> **解读：**
> *   “后置后驱”布局使得车辆转向更加灵活，同时电机直接布置在后轴附近，传动效率高。
> *   单速变速箱结构简单可靠，没有换挡顿挫感，驾驶体验平顺。
## 四、能耗表现
*   **百公里耗电量：** 约10-12 kWh/100km
*   **使用成本估算：**
    *   以家用电0.6元/kWh计算，每公里电费成本仅约6-7分钱。
    *   相比传统燃油车（每公里0.5-0.8元），日常通勤费用大幅降低。
---
### ⚠️注意事项
1.  以上数据基于公开资料及常见配置版本整理，具体参数请以您所购车辆的官方《整车出厂合格证》及《用户手册》为准。
2.  实际续航受驾驶习惯（急加速/急刹车）、空调使用情况、载重及环境温度影响较大。
3.  瑞驰C3定位为低速城市代步工具（部分区域可能归类为“低速电动车”或“新能源乘用车”），不适合高速公路长途行驶。

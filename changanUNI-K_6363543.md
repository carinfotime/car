# 长安UNI-K 内饰参数介绍
  
更新时间：2026年07月16日09时49分03秒   

## 一、内饰整体设计风格
长安UNI-K采用“未来科技座舱”设计理念，整体风格偏向简约、科技与豪华感兼具，环抱式座舱布局，强调驾驶者为中心的沉浸式体验。
---
## 二、座椅系统
| 项目 | 参数 |
|------|------|
| 座椅材质 | NAPPA真皮（高配）/ 仿皮（中低配） |
| 前排座椅功能 | 电动调节（主驾8向/副驾4向）、座椅加热、座椅通风（高配）、座椅记忆（主驾） |
| 后排座椅 | 支持靠背角度调节（多级可调） |
| 座椅布局 | 2+3 五座布局 |
| 后排中央扶手 | 带杯架、储物盒 |
| 座椅颜色选择 | 黑色、米色、棕色（因配置而异） |
---
## 三、中控台与仪表盘
| 项目 | 参数 |
|------|------|
| 仪表盘 | 10.25英寸全液晶仪表盘（悬浮式） |
| 中控屏 | 12.3英寸高清触摸屏 |
| 中控屏分辨率 | 1920×720 |
| 中控屏操作系统 | 梧桐车联系统（基于Android） |
| 中控台材质 | 软质搪塑 + 金属拉丝饰板 + 钢琴烤漆面板 |
| 空调控制区 | 触控式面板（带震动反馈） |
---
## 四、方向盘
| 项目 | 参数 |
|------|------|
| 方向盘材质 | 真皮包裹 |
| 方向盘形状 | D形（平底运动风格） |
| 方向盘功能 | 多功能按键（多媒体、巡航、语音控制） |
| 方向盘调节 | 手动四向调节（上下+前后） |
| 换挡拨片 | 标配 |
---
## 五、智能科技配置
| 项目 | 参数 |
|------|------|
| 语音控制系统 | 智能语音交互（支持自然语言识别） |
| 手机互联 | CarPlay、CarLife、华为HiCar |
| 车载导航 | 高德地图车机版 |
| 车联网 | 4G网络、OTA远程升级 |
| 蓝牙 | 蓝牙5.0 |
| USB接口 | 前排2个Type-A + 1个Type-C / 后排1个Type-A + 1个Type-C |
| 无线充电 | 前排手机无线充电（15W） |
---
## 六、音响系统
| 项目 | 参数 |
|------|------|
| 扬声器数量 | 14扬声器（索尼品牌，高配） |
| 音响品牌 | SONY（索尼） |
| 音效技术 | 7.1声道环绕声、DSP数字信号处理 |
| 低音炮 | 后备箱隐藏式低音炮 |
---
## 七、空调系统
| 项目 | 参数 |
|------|------|
| 空调类型 | 自动双区独立空调（高配为三区） |
| 后排空调出风口 | 标配 |
| 空气净化 | 车载空气净化器、PM2.5过滤、负离子发生器 |
| 香氛系统 | 高配可选装 |
---
## 八、车内灯光
| 项目 | 参数 |
|------|------|
| 氛围灯 | 64色LED氛围灯 |
| 氛围灯分布 | 中控台、门板、脚窝 |
| 阅读灯 | 前后排LED阅读灯 |
| 化妆镜灯 | 主副驾遮阳板带照明灯 |
---
## 九、储物空间
| 位置 | 容量/说明 |
|------|-----------|
| 中央扶手箱 | 大容量储物盒（约8L） |
| 手套箱 | 带阻尼，支持冷藏功能 |
| 门板储物格 | 前门可放3瓶水，后门可放2瓶水 |
| 杯架 | 前排2个 + 后排中央扶手2个 |
| 卡片/手机储物 | 中控台下方镂空储物区（带USB接口） |
---
## 十、其他内饰细节
| 项目 | 参数 |
|------|------|
| 天窗 | 全景天窗（可开启，带电动遮阳帘） |
| 遮阳帘 | 电动遮阳帘 |
| 内后视镜 | 自动防眩目（高配） |
| 门内饰板 | 软质包裹 + 缝线装饰 |
| 门槛饰条 | 金属迎宾门槛饰条（带UNI标识） |
| 脚垫 | 原厂绒面脚垫 |

| 项目 | 数值 |
|------|------|
| 轴距 | 2890 mm |
| 前排头部空间（含天窗） | 约990 mm |
| 后排腿部空间 | 约950 mm |
| 后排头部空间 | 约960 mm |
| 后排座椅宽度 | 约1380 mm |
| 后备箱容积 | 518 L（标准）/ 最大可扩展至约1400 L（后排放倒） |

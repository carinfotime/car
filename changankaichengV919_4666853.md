# 长安凯程V919 儿童乘车安全注意事项
  
更新时间：2026年07月18日12时38分16秒   

## 一、儿童安全座椅使用规范
- **必须使用儿童安全座椅**：12周岁以下或身高低于1.4米的儿童乘车时，必须使用与其年龄、体重相匹配的儿童安全座椅。
- **安装位置**：建议将儿童安全座椅安装在后排座位上，切勿安装在副驾驶位置。
- **ISOFIX接口**：长安凯程V919后排配备ISOFIX儿童座椅固定接口，请按照说明书正确安装并锁定。
- **反向安装**：对于体重小于9公斤（约0-9个月）的婴儿，安全座椅必须反向安装。
## 二、禁止行为
- **严禁怀抱儿童乘车**：车辆碰撞时，成人无法抱住孩子，极易造成严重伤害。
- **严禁儿童坐副驾驶**：副驾驶位置的安全气囊弹出时会对儿童造成致命冲击。
- **严禁儿童使用成人安全带**：成人安全带可能勒住儿童的颈部或腹部，造成伤害。
- **严禁将头手伸出窗外**：行驶中或停车时均不得将身体任何部位伸出车窗外。
## 三、车内环境与物品管理
| 注意事项 | 说明 |
|---------|------|
| 车窗锁止 | 启用后门儿童安全锁及电动车窗锁止功能，防止儿童误操作 |
| 小物件收纳 | 硬币、弹珠、尖锐物品等应放置于儿童无法触及的位置 |
| 热饮与食物 | 车内禁止放置热饮、热汤等高温物品，防止烫伤 |
| 遮阳帘/遮阳膜 | V919后窗建议加装遮阳帘或贴膜，避免阳光直射伤害儿童眼睛 |
## 四、上下车与行车过程
1. **停车熄火后再上下车**：必须在车辆完全停稳并熄火后，由成人协助儿童上下车。
2. **右侧上下车**：尽量从右侧（靠路侧）车门上下车，避免左侧开门发生危险。
3. **确认周围环境**：开车门前先观察后方是否有来车或行人。
4. **行车中勿玩耍**：提醒孩子在行车过程中保持安静坐好，不要在后排站立或打闹。
## 五、下车后的重要提醒
- ⚠️ **切勿将孩子单独留在车内！**
  长安凯程V919车厢空间较大且密封性较好。夏季车内温度可在短时间内升至60℃以上；冬季也存在缺氧风险。无论时间长短、车窗是否留缝，都绝不可将孩子独自留在车内。
## 六、随车应急物品建议
建议在车内常备以下物品以备不时之需：

□  急救包（含创可贴、消毒液、纱布等）
□  便携式反光三角牌
□  应急手电筒
□  备用饮用水
□  小毯子（保暖/安抚）

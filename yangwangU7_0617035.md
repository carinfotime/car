# 仰望U7 用户操作指南
  
更新时间：2026年07月14日10时22分42秒   

## 一、车辆启动与关闭
### 1.1 启动车辆
1. **携带钥匙**：确保智能钥匙或已绑定的手机（通过“仰望”App）在车内。
2. **系好安全带**：主驾位安全带系好后，车辆方可进入可行驶状态。
3. **踩下制动踏板**：用力踩下制动踏板。
4. **启动操作**：
   - **怀挡式换挡**：向下拨动右侧怀挡至“D挡”（仪表盘显示D）。
   - **按键启动（如有）**：按下中控台或方向盘后的“启动/停止”按钮。
5. **确认READY状态**：仪表盘显示“READY”或动力系统指示灯亮起，表示车辆已上电。
### 1.2 关闭车辆
1. **挂入P挡**：按下怀挡顶部的“P”按钮。
2. **熄火操作**：
   - 直接下车锁车（携带钥匙离开，车辆自动断电）。
   - 或长按中控台的“启动/停止”按钮手动熄火。

## 二、驾驶模式选择
仰望U7提供多种驾驶模式，适应不同路况和驾驶偏好。
| 模式名称 | 适用场景 | 特点 |
| :--- | :--- | :--- |
| **经济模式** | 城市通勤、追求续航 | 动力输出平缓，能量回收最强 |
| **舒适模式** | 日常驾驶、长途巡航 | 悬架柔软，加速线性 |
| **运动模式** | 激烈驾驶、山路超车 | 动力响应迅猛，悬架变硬 |
| **雪地/湿滑模式** |雨雪天气、低附着力路面 |限制扭矩输出，ESP介入更早 |
| **个性化模式** |自定义设置 |可单独调节动力、转向、悬架等 |
- **切换方式**：通过中控屏的“驾驶模式”菜单或方向盘右侧的物理滚轮/按键进行切换。

## 三、易四方技术平台操作说明
仰望U7搭载易四方技术平台，具备四电机独立矢量控制能力。
### 3.1原地掉头
1. **激活条件**：车辆静止，挂P挡或N挡。
2. **操作方法**：
   - 在中控屏打开“原地掉头”功能。
   - 根据屏幕提示选择旋转方向（顺时针/逆时针）。
   - 松开刹车后，使用方向盘左侧的滚轮控制旋转速度。
3. **退出方式**：轻踩制动踏板或再次点击屏幕关闭功能。
### 3.2应急浮水脱困
> ⚠️ *此功能为极端情况下的被动安全功能，非日常使用。*
- **触发条件**：当系统检测到车辆进入深水区并失去抓地力时自动激活。
- **注意事项**：
  1.保持冷静，勿急打方向或急加速。
  2.系统会自动调整车轮转速产生推力使车辆漂浮前进。
  3.脱困后请立即联系仰望服务中心进行全车检查。

## 四、云辇-X智能车身控制系统
云辇-X系统可主动调节车身高度和阻尼。
### 4.1高度调节
- **手动调节范围**:可在标准高度基础上升高50mm或降低25mm。
- **自动调节场景**:高速行驶时自动降低以减小风阻；越野路况自动升高增加通过性。
### 4.2便捷上下车
-开启方式:在中控屏设置中开启"迎宾模式"或在停车后点击屏幕上的"便捷下车"按钮。
-效果:车身自动降低40mm,方便乘客进出。

## 五、充电与补能
### 5.1交流慢充(家用充电桩)
1.**连接充电枪**:将充电枪插入车辆的交流充电口(位于左后翼子板)。
2.**刷卡/App控制**:使用随车附带的充电卡或在App上点击开始充电。
3.**结束充电**:在App上停止充电,拔下充电枪并盖好盖子。
### 5.2直流快充(公共快充桩)
1.**连接快充枪**:将快充枪插入直流快充口(位于右后翼子板)。
2.**扫码启动**:按照充电桩屏幕提示扫码支付并启动充电。
3.**电池预热**(冬季):在导航至快充站时,系统会自动对电池进行预热以缩短充电时间。
>建议:日常使用建议将电量保持在20%-80%之间,有助于延长电池寿命。每月至少进行一次满充满放校准电池SOC。

## 六、智能座舱与娱乐系统
### 6.1中控屏操作
-**主界面**:显示导航、音乐和常用快捷卡片。三指下滑进入快捷设置面板(如座椅加热通风开关)。
-**分屏显示**:支持左右分屏,左侧导航右侧娱乐互不干扰。
### 6.2语音助手("仰望你好")
-**唤醒词**:说出"你好,仰望"即可唤醒;也可按方向盘右侧的语音按键唤醒。
-**常用指令示例**
    - "打开所有车窗"
    - "导航到最近的充电站"
    - "空调设为24度"
    - "播放周杰伦的歌"
### 6.3后排娱乐屏(选装)
-通过后排中央扶手上的触控面板控制前排座椅靠背上的屏幕。支持投屏和独立蓝牙耳机连接(不影响驾驶员)。

## 七、辅助驾驶系统(DiPilot)
本车配备L2+级辅助驾驶功能,需驾驶员全程监控路况并随时准备接管。
### 7 .1自适应巡航(ACC)
-**开启方法**
    挂入D挡后向下拨动两次怀挡杆进入巡航待命状态;再按SET键设定当前速度为巡航速度;按RES/+键恢复上次设定速度;向上拨动怀挡杆取消巡航但保留记忆;踩刹车直接退出巡航状态 。
-**跟车距离调节**
    通过方向盘左侧的距离调节旋钮调整与前车的跟车间距(共5档)。
### 7 .2车道居中保持(LCC)
需在ACC激活状态下同时启用LCC 。仪表盘会显示两侧车道线变为蓝色 。若驾驶员长时间脱手 ,系统会发出警告并最终退出辅助驾驶 。
>⚠️ *重要安全提示*:即使开启辅助驾驶 ,双手仍需轻握方向盘 ,时刻关注前方路况 。辅助驾驶不能替代人类驾驶员的责任 。

## 八 、紧急情况处理
|情况类型|应对措施|
|---|---|
|制动失灵|持续用力踩死刹车踏板 ,同时按下电子手刹按钮不放 。|
|爆胎|紧握方向盘保持直线行驶 ,切勿急刹车 ;缓慢减速并打开双闪灯靠边停车 。|
|全车断电 (黑屏)|尝试长按中控电源键重启系统 ;若无效则利用机械门把手开门逃生 ;联系道路救援 。|

## 九 、保养与维护建议
-**首保时间**:购车后6个月或行驶里程达到5000公里 (以先到为准)。
-**常规保养周期**:每12个月或每行驶20000公里进行一次常规保养 。
-**轮胎换位**:每10000公里进行一次前后轮胎交叉换位 。
-**空调滤芯更换**:每12个月更换一次空调滤芯 。
>请务必使用原厂配件及指定规格的油液 (如冷却液 、制动液等 ),否则可能影响质保权益 。

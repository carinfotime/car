# 福特锐界 (Ford Edge) 操作指南
  
更新时间：2026年07月19日08时07分18秒   

## 一、启动与熄火
### 1.1 智能钥匙启动
- **启动**：携带智能钥匙进入车内，踩下制动踏板，按下“ENGINE START/STOP”按钮。
- **熄火**：将车辆停稳并挂入 P 挡，按下“ENGINE START/STOP”按钮。
### 1.2 遥控启动（如配备）
- 按一下遥控钥匙上的锁车键，然后快速连按两次远程启动键。
- 车辆将运行预设时间（通常为10-15分钟），之后自动熄火。
## 二、挡位操作
锐界采用旋钮式电子换挡器（Rotary Gear Shift Dial）。
| 挡位 | 说明 |
|------|------|
| **P (驻车挡)** | 停车时使用。熄火前必须挂入此挡。 |
| **R (倒车挡)** | 倒车时使用。需在车辆完全停止后切换。 |
| **N (空挡)** | 短暂停车或洗车时使用。 |
| **D (前进挡)** | 正常行驶时使用。 |
| **S (运动模式)** | D挡状态下向下拨动换挡旋钮即可进入，提供更灵敏的油门响应和延迟换挡时机。 |
> **注意**：换挡时必须踩住制动踏板。
## 三、仪表盘与信息显示
### 3.1 LCD仪表盘
通过方向盘右侧的十字方向键和 OK/确认按钮进行控制：
- **上下滚动**：切换主菜单（行程信息、燃油经济性、导航等）。
- **左右切换**：进入子菜单或调整设置。
- **OK按钮**：确认选择或重置数据。
### 3.2 HUD抬头显示（如配备）
在车辆设置菜单中可开启/关闭HUD，并可调节亮度和显示内容（车速、导航指示、限速标志等）。
## 四、灯光控制
灯光控制杆位于方向盘左侧：
| 操作 | 功能 |
|------|------|
| AUTO |自动大灯模式（推荐日常使用）|
|   旋转至近光灯图标   |手动开启近光灯|
|   向外推控制杆   |开启远光灯（仪表盘显示蓝色远光指示灯）|
|   向内拉控制杆   |闪远光灯（超车或提醒时使用）|
|   控制杆上下拨动   |开启左/右转向灯|
## 五、雨刮器与清洗
雨刮器控制杆位于方向盘右侧：
- **向上拨动**：单次刮刷。
- **向下拨动**：间歇/低速/高速刮刷。
- **向内拉动控制杆**：前风窗喷水清洗。
- **向外推动控制杆**：后风窗喷水清洗（如配备）。
## 六、空调与座椅功能
### SYNC®系统空调控制
通过中控触摸屏或物理按键调节：
- **温度调节**：左右双区独立温控。
- **AUTO键**：自动恒温模式。
- **座椅加热/通风**：在中控屏空调界面内开启，分多档可调。
> *部分高配车型前排座椅支持多级加热与通风功能。*
## 七、驾驶辅助系统
### LKA车道保持辅助
通过中控屏“设置 → Driver Assistance → Lane Keeping System”开启。
当车辆无意识偏离车道时，系统会轻微修正方向盘并发出警示。
### BLIS盲区监测
外后视镜上的指示灯会在侧后方有车辆接近时亮起提醒。
### ACC自适应巡航
1.  按下方向盘右侧的巡航开关按钮。
2.  达到目标速度后按下 SET/-键设定当前速度为巡航速度。
3.  按 RES/+加速，按 SET/-减速。
4.  按下 CANCEL取消当前巡航设定。
> *ACC可自动跟随前车加减速直至完全停止。*
## 八、泊车辅助
### Rear View Camera倒车影像
挂入 R挡后中控屏自动显示后方影像及引导线。
### Active Park Assist主动泊车辅助（如配备）
1.  挂入 D或R挡以低于30km/h行驶寻找车位。
2.  中控屏提示发现车位后，按提示选择泊车方式。
3.  按照提示换挡并控制刹车，系统自动转动方向盘完成泊入。
## 九、后备箱操作
### 电动尾门
- **车内开关**：驾驶员侧门板上的尾门开关按钮。
- **遥控钥匙**：长按尾门解锁键约2秒。
- **外部开关**：尾门把手处的感应开关（需携带智能钥匙靠近）。
> *可通过中控屏设置尾门开启高度。*
## 十、紧急情况处理
### 轮胎气压监测
仪表盘出现胎压报警图标时：
1.  尽快安全停车检查轮胎外观及气压值（可在仪表信息界面查看各轮胎压力）。
2.  补充气压至标准值后系统会自动复位报警。
### 应急三角牌与千斤顶位置
三角牌和千斤顶存放于后备箱地板下方储物格内。

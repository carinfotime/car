# 博速 V 级 儿童乘车安全注意事项
  
更新时间：2026年07月20日13时50分12秒   

为了确保您孩子的乘车安全，请在使用博速 V 级（Brabus V-Class）搭载儿童出行时，严格遵守以下注意事项：
## 1. 儿童安全座椅使用规范
- **强制使用**：所有身高低于 **150厘米** 或年龄小于 **12岁** 的儿童，必须使用与其体重和身高相匹配的儿童安全座椅。
- **反向安装**：建议 **4岁以下**（或体重低于18公斤）的儿童使用反向安装的儿童安全座椅，以更好地保护头颈部位。
- **安装位置**：安全座椅应优先安装在后排座位。如果安装在副驾驶位，必须 **关闭副驾驶正面安全气囊**。
- **固定检查**：每次出行前请确认安全座椅已通过 ISOFIX 接口或安全带牢固固定，无松动现象。
## 2. 车窗与天窗安全
- **防止夹伤**：博速 V 级配备电动侧滑门及车窗。请务必开启 **车窗防夹功能**（如配备），并教育孩子不要将手、头或身体其他部位伸出窗外。
- **天窗管理**：车辆行驶中严禁让孩子站立或将身体探出天窗外。停车后如需开启天窗通风，请全程看护。
- **童锁启用**：建议启用后排车窗及侧滑门的电子童锁（Child Lock），防止孩子在车内误操作。
## 3. 车内活动限制
- **禁止站立或爬行**：车辆行驶过程中，严禁孩子在车厢内站立、奔跑或在座位上爬行。突发刹车或转弯极易导致跌落受伤。
- **后排就座原则**：无论路程长短，孩子都应始终坐在自己的座位上并系好安全带/约束系统。
- **避免干扰驾驶**：教育孩子不要触碰方向盘、换挡杆、中控屏幕等驾驶相关部件。
## 4. 温度与环境管理
- **切勿独留车内**：任何时候都不要将孩子单独留在车内。夏季车内温度可在几分钟内升至致命水平；冬季也存在缺氧和低温风险。
- **空调风向调节**：避免空调出风口直吹儿童面部或腹部，建议调节风向至上方或避开座位区域。
## 5. 上下车安全
- **观察周围环境**：开门前务必观察后方是否有来车、自行车或行人。博速 V级的侧滑门开口较大，需特别注意后方盲区。
- **防止夹手夹脚**：电动侧滑门在关闭时具有较大力度。家长应亲自操作车门开关，并确保孩子的手脚远离门缝轨道区域。
- **踏板使用指导**：上下车时请使用原厂配备的侧踏板（如有），并提醒孩子踩稳扶好。
## ⚠️紧急情况处理提示
| 情况 |  应对措施 |
| :--- | :--- |
|  误锁车内 |  教导大龄孩子如何按喇叭或在紧急情况下解锁车门；低龄幼儿需家长随身携带钥匙。 |
|  晕车不适 |  提前准备呕吐袋；适当开启外循环通风；避免空腹或过饱乘车。 |
|  意外受伤 |  随车常备儿童急救包；第一时间靠边停车处理伤口并及时就医。 |

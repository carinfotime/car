## 二、各阶段保养项目详解
### 1. 首保（5,000公里 / 6个月）
- 检查高压系统及电池状态
- 检查电机及减速器状态
- 检查制动系统（刹车片、刹车盘、制动液）
- 检查轮胎磨损及胎压
- 检查灯光、雨刮器、喇叭等电气设备
- 检查空调滤芯
- 检查冷却液液位
- 检查底盘及悬挂系统
- OTA系统检查与升级
- 紧固全车关键螺栓
### 2. 常规保养（每10,000公里 / 12个月）
**包含首保所有项目，另增加：**
- 更换空调滤芯
- 更换雨刮片（视磨损情况）
- 检查并补充玻璃水
- 检查并补充冷却液
- 检查12V蓄电池状态
- 检查充电接口及充电系统
- 检查驱动电机冷却系统
- 检查高压线束及连接器
- 检查电子驻车系统
- 检查转向系统
### 3. 大保养（每40,000公里 / 24个月）
**包含常规保养所有项目，另增加：**
- 更换减速器油
- 更换制动液
- 更换冷却液
- 更换空调滤芯（如未在常规保养中更换）
- 检查并清洁电池散热系统
- 检查并清洁电机散热系统
- 检查并更换（如需要）驱动电机冷却液
- 检查高压继电器及接触器
- 检查并清洁充电机
- 检查全车密封胶条
- 检查底盘防锈涂层
- 检查悬挂系统橡胶衬套

## 三、关键保养项目说明
### 1. 动力电池保养
| 项目 | 建议 |
|-----|------|
| 日常充电 | 建议使用慢充为主，快充为辅 |
| 充电区间 | 建议保持电量在20%-80%之间 |
| 长期停放 | 建议保持电量在40%-60%，每30天充电一次 |
| 电池均衡 | 建议每月至少进行一次满充（充至100%） |
| 温度管理 | 避免在极端高温或低温下长时间停放 |
### 2. 轮胎保养
- **胎压标准**：建议2.3-2.5 bar（具体以驾驶位车门标签为准）
- **轮胎换位**：每10,000公里进行前后轮交叉换位
- **轮胎更换**：建议每60,000公里或磨损至安全线时更换
- **动平衡**：每次换胎或补胎后需做动平衡
- **四轮定位**：出现跑偏或轮胎偏磨时需做四轮定位
### 3. 制动系统保养
- **制动液更换**：每40,000公里或2年更换一次（DOT4标准）
- **刹车片检查**：每10,000公里检查，厚度低于3mm需更换
- **刹车盘检查**：每20,000公里检查，出现沟槽或变形需更换
- **电子驻车**：定期检查电子驻车功能是否正常
### 4. 空调系统保养
- **空调滤芯**：每10,000公里或12个月更换一次
- **蒸发器清洗**：每20,000公里或出现异味时清洗
- **制冷剂检查**：每2年检查一次制冷剂压力
- **PM2.5滤芯**：如配备，建议每6个月更换一次

## 四、日常自检项目
| 项目 | 频率 | 检查内容 |
|-----|------|---------|
| 胎压 | 每月 | 检查四个轮胎胎压是否正常 |
| 灯光 | 每周 | 检查大灯、尾灯、转向灯、刹车灯 |
| 雨刮器 | 每月 | 检查雨刮片是否老化、刮水效果 |
| 玻璃水 | 每月 | 检查液位，及时补充 |
| 充电接口 | 每次充电前 | 检查接口有无异物、氧化或损坏 |
| 仪表盘 | 每次启动 | 观察有无故障灯亮起 |
| 轮胎外观 | 每周 | 检查有无鼓包、裂纹、异物扎入 |
| 刹车 | 每月 | 检查刹车是否灵敏，有无异响 |

## 六、保养注意事项
1. **建议到极狐授权服务中心进行保养**，使用原厂配件及专用设备。
2. **保留所有保养记录**，以便二手车交易时提供完整维保历史。
3. **OTA升级**：关注官方推送的系统更新，及时进行升级。
4. **三电质保**：整车及三电系统有质保期，请务必按保养手册要求进行保养，避免影响质保权益。
5. **冬季保养**：寒冷地区建议增加电池保温措施，检查冷却液冰点。
6. **雨季保养**：检查天窗排水孔、车门密封条，防止进水。
7. **充电桩保养**：如使用家用充电桩，建议每半年检查一次充电桩状态。

## 七、常见问题解答
**Q1：纯电动车需要换机油吗？**
A：不需要。极狐阿尔法S6为纯电动车型，无发动机，无需更换机油。
**Q2：可以不去4S店保养吗？**
A：建议在质保期内前往授权服务中心保养，以免影响三电系统质保权益。
**Q3：减速器油多久换一次？**
A：建议每40,000公里或2年更换一次。
**Q4：电池衰减有保修吗？**
A：极狐阿尔法S6三电系统有较长质保期，具体衰减标准请参阅保修手册。
**Q5：长期不开需要注意什么？**
A：建议保持电量40%-60%，每30天充电并启动一次，检查轮胎气压。

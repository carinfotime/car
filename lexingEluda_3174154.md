# 乐行E路达 内饰参数介绍
  
更新时间：2026年07月15日11时04分58秒   

## 一、座舱布局与空间
| 项目 | 参数 |
|------|------|
| 座椅布局 | 2+3 五座布局（可选2+2+2六座） |
| 轴距 | 2850mm |
| 车内高度 | ≥1250mm |
| 后排腿部空间 | ≥900mm |
| 后备箱容积 | 标准状态：480L / 放倒后排：1200L |
## 二、座椅配置
- **座椅材质**：高级织物 / 仿皮（高配）
- **主驾座椅调节**：6向手动调节（前后/靠背/高低）
- **副驾座椅调节**：4向手动调节
- **后排座椅**：支持4/6比例放倒
- **前排头枕**：可调式头枕
- **后排中央扶手**：带杯架（高配）
## 三、仪表与中控
- **仪表盘**：7英寸彩色液晶组合仪表
- **中控屏**：10.25英寸悬浮式触控屏
- **车机系统**：智能网联系统，支持OTA升级
- **手机互联**：支持CarPlay / HiCar / Carlife
- **语音控制**：多媒体/导航/空调语音识别控制
## 四、空调系统
- **空调类型**：自动恒温空调（高配）/手动空调（低配）
- **后排出风口**：标配
- **PM2.5过滤装置**：标配
- **负离子空气净化器**（高配选装）
## 五、方向盘与操控
| 项目 | 参数 |
|------|------|
| 方向盘材质 | PU皮质包裹 |
| 方向盘调节 |上下两向手动调节 |
|多功能按键 |音响控制/蓝牙电话/定速巡航 |
## 六、储物空间
- **前排杯架** ×2
- **中央扶手箱储物盒**
- **手套箱**
- **门板储物格**（四门）
- **前排座椅背袋**
- **眼镜盒**
## 七、灯光与氛围
 -阅读灯 ：前后排LED阅读灯 
 -化妆镜灯 ：主副驾遮阳板带镜带灯 
 -氛围灯 ：单色氛围灯（高配）
## 八、便利配置 
 -一键启动 
 -无钥匙进入 
 -电子手刹 + Auto Hold 
 -USB接口 ：前排×2 +后排×1 （Type-A） 
 -12V电源接口 ：中控台下方×1 
 -无线充电板 （高配）
## 九、安全内饰 
 -主/副驾安全气囊 
 -前排侧气囊 （高配） 
 -安全带未系提醒 
 -ISOFIX儿童座椅接口 （后排两侧） 
---
>以上参数以官方最终配置为准，不同版本可能存在差异。

# 示界06 空间参数介绍
  
更新时间：2026年07月16日14时24分12秒   

## 车身尺寸
| 参数 | 数值 |
|------|------|
| 车长（mm） | 4,765 |
| 车宽（mm） | 1,890 |
| 车高（mm） | 1,680 |
| 轴距（mm） | 2,850 |
## 乘坐空间
- **前排头部空间**：1,020 mm
- **后排头部空间**：980 mm
- **后排腿部空间**：920 mm
- **后排肩部空间**：1,480 mm
- **前排座椅调节范围**：前后240 mm / 上下60 mm
## 座椅布局
- **座位数**：5座（2+3布局）
- **前排座椅**：支持电动调节、加热、通风功能
- **后排座椅**：支持4/6比例放倒，带中央扶手和杯架
## 储物空间
### 车内储物格
- **中控台储物盒**：容积约3.5 L
- **门板储物格**：前门可容纳1.5L水瓶×2，后门可容纳500ml水瓶×2
- **中央扶手箱**：容积约8 L，带空调出风口（可制冷/保温）
- **手套箱**：容积约12 L
- **杯架数量**：前排2个 + 后排2个 + 门板4个 = 共8个杯架位
### 后备箱空间
| 状态 | 容积 |
|------|------|
| 标准状态（5座） | 520 L |
| 后排座椅放倒（4/6比例） | — |
|   -  单侧放倒 | ~800 L |
|   -  完全放倒 | ~1,380 L |
|   -  放倒后最大进深 | ~1,850 mm |
### 后备箱细节
- **开口宽度**：1,120 mm
- **开口高度**：780 mm
- **离地高度（尾门门槛）**：720 mm（便于装卸重物）
- **地板下方隐藏储物格**：容积约35 L，可存放工具、急救包等小件物品
- **挂钩数量**：4个（每个承重3 kg）
- **锚点数量**：4个
- **照明灯**：LED照明灯×1
## EV版本前备箱（如有）
> *注：若为纯电版车型，取消发动机后提供前备箱。*
| EV版前备箱容积 | ~60 L |
|---|---|
适合存放充电线缆、便携工具或小型行李包。
---
> *以上数据基于示界06官方公布信息整理，实际参数以最终量产车型为准。*

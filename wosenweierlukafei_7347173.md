# 沃森威尔路咖啡 保养指南
  
更新时间：2026年07月13日15时43分11秒   

## 一、日常检查（每次用车前或每周）
- **轮胎**：检查胎压是否正常（参考驾驶座门框内侧标签），观察胎面有无异物、鼓包或异常磨损。
- **灯光**：确认大灯、转向灯、刹车灯、雾灯等全部正常工作。
- **油液**：打开发动机舱，检查机油尺刻度、刹车油壶液位、冷却液副水箱液位是否在上下限之间。
- **雨刮与玻璃水**：确保雨刮片无老化开裂，玻璃水充足。
## 二、定期保养项目
### 1. 每5,000公里或6个月（以先到为准）
| 项目 | 说明 |
|------|------|
| 更换机油 & 机滤 | 使用符合 ACEA C3 / API SP 标准的全合成机油（推荐5W-30） |
| 检查空气滤芯 | 视脏污程度清洁或更换 |
| 检查空调滤芯 | 视脏污程度清洁或更换 |
| 底盘与悬挂检查 | 紧固螺栓，检查减震器有无漏油 |
### 2. 每20,000公里或2年
| 项目 | 说明 |
|------|------|
| **更换空气滤芯** | — |
| **更换空调滤芯** | — |
| **更换火花塞** | — |
| **清洗节气门** | — |
| **制动系统检查** | - |
###  3.  每40,000公里或4年
-   **更换变速箱油**
-   **更换刹车油**
-   **更换防冻液**
-   **检查正时皮带/链条状态**
###  4.  每60,000公里或6年
-   **全面大保养**：包含以上所有项目
-   **更换燃油滤清器**
-   **更换轮胎**
    -   即使花纹深度尚可，超过6年的轮胎也建议强制换新
    -   做四轮定位 & 动平衡
##  三、季节性保养要点
### 🌞夏季前
-   检查空调制冷效果及冷媒压力
-   清理散热器表面杂物（柳絮/昆虫尸体）
-   注意冷却液冰点及沸点
### ❄️冬季前
-   测试电瓶健康度（低温下启动电流下降明显）
-   若所在地区气温低于 -10°C，考虑使用冬季专用玻璃水（防冻型）
-   轮胎花纹深度低于3mm时建议提前换雪地胎

##  五、常见故障提示与应对
1. **[发动机故障灯亮]**  
    →优先检查油箱盖是否拧紧；若未解决，尽快进店读取故障码。
2. **[启动困难 /怠速抖动]**  
    →常见原因：节气门积碳 /火花塞老化 /燃油品质不佳。
3. **[行驶中水温报警]**  
    →立即安全停车熄火，等待15分钟后检查冷却液液位。切勿在高温时打开水箱盖！
4. **[刹车异响 /踏板变软]**  
    →立即停止激烈驾驶，尽快到店检查刹车片厚度及刹车油含水量。
##  六、特别提醒
1. **[质保期内]**  
      务必在授权服务站进行保养并保留记录，否则可能影响质保权益。
2. **[长期停放]**  
      若停放超过一个月：
        -断开电瓶负极
        -胎压充至标准值上限 +0.3 bar
        -每两周原地启动运行15分钟（让机油循环）
3. **[加油建议]**  
      推荐使用95号及以上无铅汽油。避免频繁加注劣质燃油添加剂。

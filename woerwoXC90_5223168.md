# 沃尔沃XC90 操作指南
  
更新时间：2026年07月11日04时02分54秒   

## 一、车辆基本操作
### 1. 启动与熄火
- **启动**：智能钥匙在车内，踩下刹车，按下中控台启动旋钮（向右旋转）。
- **熄火**：车辆停稳，挂入P挡，按下启动旋钮（向左旋转）。
- **紧急熄火**：长按启动旋钮（超过3秒）可强制熄火。
### 2. 挡位操作（电子换挡杆）
- **D挡（前进挡）**：向后拉动换挡杆。
- **R挡（倒车挡）**：向前推动换挡杆。
- **N挡（空挡）**：短促向前或向后推动至N挡位置。
- **P挡（驻车挡）**：按下换挡杆侧面的P按钮。
- **手动模式**：将换挡杆向左拨至“±”位置，向上推为升挡，向下拉为降挡。
> **注意**：换挡需踩住刹车，车辆静止时方可挂入R挡或P挡。
### 3. 方向盘调节
- 方向盘下方左侧有手动调节杆。
- 向下扳动解锁，调整方向盘高度和前后距离。
- 调整完毕后，向上推回锁定。

## 二、中控屏幕（Sensus系统）操作
### 1. 主界面
- 主页显示四个功能区域：导航、媒体、电话、应用。
- 从屏幕顶部向下滑动可进入快捷设置（如蓝牙、Wi-Fi、屏幕亮度等）。
### 2. 空调控制
- 点击屏幕底部的温度区域，进入空调控制界面。
- 可调节双区/四区温度、风量、风向、座椅加热/通风、方向盘加热。
- 点击“AUTO”启用自动空调。
- 点击“AC”开启/关闭制冷。
### 3. 导航系统
- 点击主页“导航”图标，或按方向盘上的语音控制按钮说出目的地。
- 支持输入地址、搜索兴趣点（加油站、餐厅等）。
- 导航设置中可避开收费站、高速路等。
### 4. 媒体与连接
- **蓝牙连接**：设置 → 蓝牙 → 添加设备，在手机上搜索并配对。
- **Apple CarPlay**：使用USB线连接iPhone（仅限左侧USB接口），自动激活。
- **Android Auto**：使用数据线连接安卓手机，需预先安装Android Auto应用。
### 5. 车辆设置
- 点击“设置”图标（齿轮状）。
- 可调整驾驶辅助、车灯、门锁、座椅、驾驶员配置文件等。

## 三、驾驶辅助与安全功能
### 1. City Safety 城市安全系统
- **自动开启**，无法完全关闭。
- 检测车辆、行人、骑行者及大型动物。
- 碰撞风险时，先发出警告音，若驾驶员未反应，系统自动刹车。
### 2. Pilot Assist 领航辅助系统
- **开启**：方向盘左侧按键，按下中央的“▶”按钮。
- **设置速度**：按“+/-”键调节巡航速度（30-140 km/h）。
- **设置跟车距离**：按左侧“←/→”键调节与前车距离（5档）。
- **取消**：按“X”键或轻踩刹车。
- **功能**：自动保持车道居中、保持设定速度与跟车距离。
> **重要**：驾驶员必须始终手握方向盘，随时准备接管控制。
### 3. BLIS 盲点信息系统
- **默认开启**。
- 侧后方有车辆接近时，对应侧后视镜内侧指示灯亮起。
- 若此时打转向灯，指示灯会闪烁并发出警示音。
### 4. 倒车辅助
- 挂入R挡自动激活倒车影像。
- 360°全景影像（如配备）可查看车辆周围情况。
- 倒车时若检测到后方障碍物或移动物体，系统会发出警报并自动刹车。

## 四、灯光与雨刷
### 1. 灯光控制
- **左侧拨杆**：旋转旋钮控制灯光模式。
  - `0`：关闭（日间行车灯自动亮起）
  - `A`：自动大灯（推荐日常使用）
  - 近光灯图标：强制近光灯
  - 雾灯图标：前后雾灯
- **远光灯**：向驾驶员方向拉动拨杆。
- **自适应远光灯**：在自动大灯模式下，远光灯会根据对向车辆自动切换远近光。
### 2. 雨刷控制
- **右侧拨杆**：
  - 向上拨：单次刮刷
  - 向下拨：自动雨刷（可调节灵敏度，旋钮在拨杆顶端）
  - 向下拨到底：连续刮刷（低速/高速）
- **喷水**：向驾驶员方向拉动拨杆（前挡风）；向后推（后挡风）。
- **自动雨刷**：默认开启，下雨时自动启动。

## 五、座椅与舒适性
### 1. 座椅调节
- **电动调节**：座椅侧面按键控制前后、高低、坐垫角度、靠背角度。
- **腰部支撑**：圆形按键调节支撑位置与强度。
- **座椅记忆**：调整好位置后，按“M”键，再按数字键（1/2/3）保存。下次长按对应数字键恢复。
### 2. 座椅加热/通风
- 中控屏空调界面或快捷设置中开启。
- 分为三档：高、中、低。
### 3. 方向盘加热
- 中控屏空调界面或方向盘左侧按键（带方向盘图标）开启。
### 4. 全景天窗
- **开启/关闭**：车顶控制区域滑动按键。
- **遮阳帘**：手动拉合或电动控制（视配置）。

## 六、其他实用功能
### 1. 电动尾门
- 尾门下方踢脚感应（需携带钥匙，在尾门下方做踢腿动作）。
- 尾门内外均有关闭按钮。
- 可设置尾门开启高度（中控屏设置 → 车辆 → 尾门开启高度）。
### 2. 加油
- 油箱盖位于车辆右侧（副驾驶侧）。
- 车辆解锁后，按压油箱盖即可打开。
- 推荐使用95号及以上无铅汽油。
### 3. 儿童安全锁
- 后门侧面有机械式儿童锁，用钥匙或机械钥匙旋转开关。
- 启用后，后门无法从车内打开。
### 4. 紧急呼叫（On Call）
- 车顶控制区域有“SOS”按钮。
- 长按可连接沃尔沃紧急呼叫中心。
- 车辆发生碰撞时，系统会自动呼叫。
---
## 七、保养与提示
### 1. 保养提醒
- 中控屏会按里程或时间显示保养提醒。
- 常规保养周期：每 12 个月 或 每 10,000 公里（以先到为准）。
### 2. 胎压监测
- 系统自动监测，异常时仪表盘显示警告。
- 可在中控屏查看具体胎压数值（需配备胎压传感器）。
### 3. 机油液位检查
- 中控屏 → 设置 → 车辆状态 → 机油液位。
- 需在车辆水平、发动机达到工作温度并熄火后测量。
### 4. 添加清洗液
- 前引擎盖内蓝色盖子的容器，加注专用玻璃清洗液。

## 八、常见问题与提示
| 问题 | 解决方法 |
|------|----------|
| 钥匙没电怎么办？ | 将钥匙贴近方向盘下方的钥匙感应区（杯架内也有感应区），仍可启动车辆。 |
| 如何重置胎压监测？ | 中控屏 → 设置 → 车辆 → 胎压监测 → 校准。 |
| 如何关闭发动机自动启停？ | 中控屏快捷设置或空调界面找到“Start/Stop”按钮，点击关闭。 |
| 导航无声音？ | 检查导航音量设置，或旋转中控音量旋钮（导航播报时调节）。 |
| 如何关闭车内监控报警？ | 车顶控制区域，按下“车内监控”按钮（带人形图标）。 |
---

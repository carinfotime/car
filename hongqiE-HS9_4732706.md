# 红旗E-HS9 操作指南
  
更新时间：2026年07月11日00时11分13秒   

## 一、车辆基本介绍
红旗E-HS9是一款全尺寸纯电动智能SUV，提供6座/7座/4座三种座椅布局。车辆采用双电机/三电机四驱系统，配备空气悬架、L2+级智能驾驶辅助系统及智能座舱。
---
## 二、启动与关闭
### 2.1 启动车辆
1. 携带智能钥匙进入车辆，系统自动识别钥匙。
2. 踩下制动踏板，仪表盘显示“READY”指示灯。
3. 将换挡旋钮旋转至D挡或R挡，松开电子手刹即可行驶。
4. 若使用手机蓝牙钥匙或NFC卡片钥匙，将设备放置于感应区域（中央扶手箱前方）完成认证。
### 2.2 关闭车辆
1. 将车辆停稳，挂入P挡。
2. 按下启动按键（或直接锁车离开，系统自动下电）。
3. 确认所有车窗、天窗关闭，携带钥匙离开车辆。
### 2.3 紧急断电
长按启动按键3秒以上，车辆将强制切断高压电源。

## 三、驾驶操作
### 3.1 换挡操作
- **D挡（前进）**：旋转旋钮至D位置。
- **R挡（倒车）**：旋转旋钮至R位置。
- **N挡（空挡）**：旋转旋钮至N位置。
- **P挡（驻车）**：按下旋钮中间P键，或车辆熄火后自动挂入。
> 注意：换挡时需踩住制动踏板。
### 3.2 驾驶模式选择
通过中控屏或物理按键选择驾驶模式：
| 模式 | 适用场景 | 特点 |
|------|----------|------|
| 经济模式 | 城市通勤 | 降低电耗，加速平缓 |
| 舒适模式 | 日常驾驶 | 均衡性能与舒适性 |
| 运动模式 | 高速超车 | 动力响应迅速 |
| 雪地模式 | 冰雪路面 | 降低扭矩，防止打滑 |
| 越野模式 | 非铺装路面 | 空气悬挂升高，四驱锁止 |
### 3.3 能量回收
通过方向盘拨片或中控屏调节能量回收强度：
- **Lv.1（弱）**：滑行阻力小，接近燃油车感受。
- **Lv.2（中）**：适当回收，适合日常使用。
- **Lv.3（强）**：单踏板模式，松开加速踏板即明显减速。
### 3.4 电子手刹
- **启用**：挂入P挡后自动启用，或手动拉起P键。
- **解除**：挂入D/R挡后踩加速踏板自动解除，或手动按下P键。

## 四、充电操作
### 4.1 充电接口位置
- **左侧（慢充）**：交流充电口，适用于家用充电桩。
- **右侧（快充）**：直流充电口，适用于公共快充桩。
### 4.2 充电步骤
1. 车辆熄火并挂入P挡。
2. 按压充电口盖板，自动弹开。
3. 取下防尘塞，插入充电枪直至锁止。
4. 充电桩启动充电（刷卡/APP扫码/即插即充）。
5. 充电完成后，解锁车辆，按下充电枪按钮拔出。
6. 盖好防尘塞和盖板。
### 4.3 充电提示
- 快充（20%-80%）约需45分钟。
- 慢充（0-100%）约需8-10小时（视充电桩功率）。
- 建议日常使用保持电量在20%-90%之间。
- 低温环境下充电速度会下降，建议使用预约充电功能。

## 五、智能座舱操作
### 5.1 中控屏系统
- 主界面：导航、音乐、车辆设置、空调等。
- 下滑菜单：快捷开关（驾驶模式、座椅加热、能量回收等）。
- 车辆设置：灯光、门锁、驾驶辅助、充电设置等。
### 5.2 语音控制
唤醒方式：说出“**你好，红旗**”或按下方向盘语音键。
常用指令示例：
- “导航到XXX”
- “打开空调，温度26度”
- “播放音乐”
- “打开座椅加热”
- “车窗开启一半”
### 5.3 空调控制
- 前排：通过中控屏或语音调节双区自动空调。
- 后排：可通过后排触控屏独立调节。
- 快捷操作：中控台保留物理按键（温度、风量、除雾）。
### 5.4 座椅功能
- 前排座椅：电动调节、通风、加热、按摩（部分配置）。
- 第二排座椅：电动调节、加热、通风（高配）。
- 第三排座椅：电动折叠/展开（6座/7座版）。
### 5.5 仪表盘与HUD
- 仪表盘显示：车速、电量、续航、驾驶模式、导航信息。
- HUD抬头显示：车速、导航箭头、限速标志、ADAS警示。

## 六、智能驾驶辅助
### 6.1 自适应巡航（ACC）
1. 按下方向盘左侧“巡航”按键。
2. 设定速度（SET +/-）。
3. 调节跟车距离（远/中/近三档）。
4. 取消：轻踩刹车或按取消键。
### 6.2 车道保持辅助（LKA）
- 默认开启，车速超过60km/h自动激活。
- 车辆偏离车道时，方向盘施加纠正力矩。
- 可通过中控屏关闭或调整灵敏度。
### 6.3 自动紧急制动（AEB）
- 默认开启，检测到碰撞风险时自动刹车。
- 可在中控屏调整预警灵敏度。
### 6.4 自动泊车
1. 中控屏点击“自动泊车”或语音唤醒。
2. 低速行驶搜索车位。
3. 选定车位后，按提示挂挡，系统自动操作。
4. 随时可接管方向盘。
### 6.5 全景影像
- 挂入R挡自动激活。
- 可手动点击中控屏开启。
- 支持2D/3D视图切换，透明底盘显示。
---
## 七、灯光与雨刷
### 7.1 灯光控制
- **自动大灯**：旋钮调至AUTO，根据光线自动开关。
- **远光灯**：拨杆向前推。
- **自适应远近光**：在AUTO模式下，拨杆向前推一次激活。
- **雾灯**：旋钮旋转至雾灯位置。
- **日间行车灯**：车辆启动后自动点亮。
### 7.2 雨刷控制
- **间歇/自动**：拨杆向上推一档，旋转旋钮调节频率。
- **低速连续**：向上推两档。
- **高速连续**：向上推三档。
- **单次刮刷**：向下推一次。
- **喷水清洗**：拨杆向驾驶员方向拉。

## 八、应急处理
### 8.1 车辆无法启动
1. 检查智能钥匙是否在车内。
2. 检查12V小电瓶是否亏电（搭电位置在前机舱右侧）。
3. 尝试使用NFC卡片或手机蓝牙钥匙。
4. 联系红旗道路救援：**400-817-0000**。
### 8.2 充电故障
1. 检查充电枪是否完全插入。
2. 检查充电桩是否正常通电。
3. 尝试重新插拔充电枪。
4. 尝试更换充电桩。
### 8.3 轮胎失压
- 仪表盘显示胎压报警。
- 使用车载充气泵（后备箱储物格内）临时补气。
- 就近寻找轮胎维修点。
- 备胎位置：部分车型配备备胎，位于后备箱底板下方。
### 8.4 拖车注意事项
- 必须使用平板拖车，严禁两轮着地拖车（可能损坏电机）。
- 拖车钩位于前保险杠下方盖板内。

## 九、保养与维护
| 项目 | 周期/建议 |
|------|-----------|
| 空调滤芯更换 | 每1万公里或1年 |
| 减速器油更换 | 每4万公里 |
| 冷却液检查 | 每2万公里 |
| 制动液更换 | 每2年或4万公里 |
| 轮胎换位 | 每1万公里 |
| 高压系统检查 | 每2万公里（需在授权服务中心） |
### 日常检查
- 每月检查一次轮胎气压（建议2.5-2.8 bar）。
- 定期清理充电口，保持干燥清洁。
- 长期停放时保持电量在50%-70%。

## 十、常见问题与提示
### 10.1 续航里程突然下降
- 低温、激烈驾驶、空调高功率使用均会影响续航。
- 建议使用经济模式+弱回收可优化续航。
- 使用预约充电功能可预热电池，提升冬季续航。
### 10.2 车机卡顿
- 尝试重启车机：长按中控屏电源键10秒。
- 检查系统更新：设置 → 系统 → 系统更新。
### 10.3 钥匙无法识别
- 将钥匙放置于中央扶手箱前方的感应区域。
- 更换钥匙电池（CR2032纽扣电池）。
### 10.4 OTA升级
- 系统推送更新时，中控屏会弹出提示。
- 建议在充电时进行升级，确保电量充足。
- 升级期间车辆无法使用，预计耗时30-60分钟。
---
## 十一、安全提示
1. 涉水深度不超过30cm，避免浸泡电池包。
2. 充电时严禁使用高压水枪冲洗充电口。
3. 车辆发生碰撞后，请及时联系服务中心检查高压系统。
4. 儿童安全锁位于后门侧面，防止儿童误开门。
5. 离开车辆前确认已挂入P挡并关闭电源。
6. 严禁私自改装高压电路系统。

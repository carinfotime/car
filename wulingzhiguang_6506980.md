## 二、日常检查（每周/每次出车前）
- [ ] **机油液位**：冷车状态下拔出机油尺，液位应在上下刻度之间
- [ ] **冷却液液位**：观察副水箱，应在MIN与MAX之间
- [ ] **制动液液位**：不低于MIN线
- [ ] **轮胎气压**：前轮约2.3-2.5 bar，后轮约2.8-3.0 bar（视载重调整）
- [ ] **轮胎外观**：检查是否有鼓包、裂纹、异常磨损
- [ ] **灯光系统**：大灯、转向灯、刹车灯、倒车灯是否正常
- [ ] **雨刮器 & 玻璃水**：雨刮片是否老化，玻璃水是否充足
- [ ] **仪表盘警示灯**：启动后所有警示灯应熄灭

## 三、机油与机滤更换（5,000km/6个月）
### ✅推荐机油规格
- SAE黏度等级：
    - **夏季/高温地区**：15W-40 / 20W-50
    - **冬季/寒冷地区**：10W-30 /  5W-30
    - **四季通用**：10W-40
- API质量等级：不低于 **SL级别**
- SAE黏度等级：
### 🔧更换步骤
1.  热车后熄火，等待5分钟让机油回流油底壳
2.  用千斤顶顶起车辆前部并放置安全支架（或开上坡道）
3.  拧下放油螺栓，放出旧机油（注意防烫）
4.  拆下旧机油滤清器
5.  在新机滤密封圈上涂抹少量新机油，用手拧紧（不要用工具过紧）
6.  装回放油螺栓并更换垫片（扭矩约25-35 N·m）
7.  加注新机油至合适液位（五菱之光约3.0L左右）
8.  启动发动机运转1分钟，检查底部有无漏油

## 四、空气滤清器更换（10,000km/1年）
1.  打开副驾驶座椅下方或发动机舱内的空滤盒卡扣
2.  取出旧空滤芯
3.  清理空滤盒内部的灰尘杂物
4.  装入新空滤芯（注意安装方向）
5.  扣紧卡扣，确保密封严实

## ⚠️五、常见故障与预防措施
### 🔥发动机高温
常见原因：
    - ❌冷却液不足 → ✅定期检查副水箱液位
    - ❌散热风扇不转 → ✅检查风扇电机及保险丝
    - ❌节温器卡死 → ✅按周期更换防冻液时一并检查节温器
### 🔊底盘异响
常见原因：
    - ❌减震器漏油失效 → ✅按压车身检查回弹是否正常
    - ❌球头/衬套磨损 → ✅举升车辆检查各胶套有无开裂松旷
### 🚗刹车偏软/刹不住
常见原因：
    - ❌刹车片磨损到极限 → ✅厚度低于3mm必须更换  
    - ❌制动管路内有空气 → ✅定期更换制动液并排气  
    - ❌刹车鼓失圆 → ✅拆检刹车鼓必要时光鼓或更换  
---
## ⚡六、电气系统保养要点
### 🔋蓄电池维护
1.  定期清洁桩头氧化物（可用开水冲洗或用砂纸打磨）
2.  紧固接线柱螺母防止松动打火  
3.  长期停放超过两周建议断开负极  
4.  冬季注意保温——低温会大幅降低电池容量  
### 💡灯光系统易损点  
五菱之光前大灯灯泡型号多为 H4 ，尾灯灯泡型号多为 P21W 。建议随车备一套备用灯泡。

## 🛞七、轮胎专项保养  
1️⃣ **胎压标准**
|位置 |空载(bar) |满载(bar)|
|---|---|---|
前轮 |2\.3–2\.5 |2\.5–2\.8|
后轮 |2\.8–3\.0 |3\.0–3\.5|
> *具体数值以驾驶座门框上的标签为准*  
2️⃣ **换位规则**
每行驶10,000公里进行轮胎换位：

前左 ←→前右 (交叉)
后左 ←→后右 (交叉)

若备胎为全尺寸且参与轮换则按“五轮换位法”。
3️⃣ **报废标准**
出现以下任一情况必须立即更换：
*   花纹深度低于1\.6mm  
*   胎侧鼓包  
*   帘布层外露  
*   使用超过6年(即使花纹尚可)

## 📋八、季节性保养提醒  
### 🌸春季(回暖多雨)
*   检查雨刮片及喷水嘴  
*   清理空调蒸发箱避免霉味  
*   检查天窗排水孔(如有)
### ☀️夏季(高温多雨)
*   重点检查冷却系统——水箱盖密封圈是否老化  
*   空调制冷剂压力检测  
*   防止爆胎——适当降低胎压0\.1–0\.2bar  
### 🍂秋季(入冬准备)
*   更换冬季玻璃水(-25℃以下规格)  
*   检测蓄电池健康度——CCA值不低于标称值70%   
*    预热塞功能检测(柴油版车型)
### ❄️冬季(严寒冰雪)
*    冷启动时先通电自检再点火   
*    冰雪路面建议加装防滑链   
*    停车尽量选择室内或有遮挡处

## ⚖️十一、重要注意事项  
✅务必做到：
✔️每次加油前看一眼仪表盘警示灯   
✔️跑长途前绕车一周做目视检查   
✔️严格按照手册规定里程保养   
✔️使用正品配件和符合规格的油液   
❌切勿做：
✘用普通水代替冷却液——易结垢腐蚀水道   
✘不同品牌混加防冻液——可能产生化学反应   
✘超期使用正时皮带——断裂将顶坏气门导致大修   
✘用铁丝代替保险丝——可能引发线路起火

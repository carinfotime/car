# 跃进大拿T1 操作指南
  
更新时间：2026年07月15日17时49分27秒   

## 一、车辆概述
跃进大拿T1是一款面向城市物流、短途运输的轻型卡车。本指南旨在帮助驾驶员快速熟悉车辆的基本操作、安全规范及日常维护要点。
## 二、驾驶前检查
每次出车前，请务必完成以下检查项目：
1.  **外观检查**：绕车一周，检查车身有无明显损伤、轮胎气压是否正常、有无漏油漏水现象。
2.  **油液检查**：
    -   **机油**：拔出机油尺，检查油位应在上下刻度线之间。
    -   **冷却液**：观察副水箱液位，应在“MAX”与“MIN”之间。
    -   **制动液**：检查制动液储液罐液位是否正常。
    -   **尿素液（如适用）**：检查尿素液位，不足时及时添加。
3.  **灯光与信号**：开启所有灯光（近光、远光、转向灯、刹车灯、雾灯），确认工作正常。
4.  **轮胎与螺栓**：目视检查轮胎有无鼓包、裂纹，并用工具（如扭矩扳手）抽查轮胎螺母是否紧固。
## 三、启动与驾驶
### 1. 启动步骤
1.  插入钥匙，将钥匙旋转至“ON”档（通电自检），等待仪表盘所有指示灯熄灭或仅保留少数常亮指示灯（如充电指示灯）。
2.  确认挡位置于空挡（N挡）。
3.  踩下离合器踏板到底。
4.  将钥匙继续旋转至“START”档启动发动机。发动机启动后，立即松开钥匙。
5.  怠速热车约30秒至1分钟（冬季可适当延长），观察仪表盘各指示正常。
### 2. 起步与行驶
-   **手动挡车型**：
    1.  踩下离合器踏板。
    2.  挂入1挡或倒挡（R挡）。
    3.  松开手刹。
    4.  缓慢抬起离合器踏板，同时轻踩油门踏板，平稳起步。
    5.  根据车速适时换挡。推荐换挡转速区间为1500-2000转/分钟。
-   **自动挡/AMT车型**：
    1.  踩下制动踏板。
    2.  将换挡旋钮或手柄旋至“D”（前进）或“R”（倒车）位置。
    3.  松开手刹和制动踏板，轻踩油门即可行驶。
### 3. 驾驶注意事项
-   车辆满载时制动距离会显著增加，请保持安全车距。
-   转弯时注意内轮差（大货车转弯半径大），避免剐蹭行人或障碍物。
-   避免长时间高转速低挡位行驶或低转速高挡位行驶（拖挡），以免损伤发动机和变速箱。

## 五、载货与卸货安全
1.  **装载原则**：
    -   货物均匀分布，避免偏载导致侧翻风险。
    -   总重量不得超过车辆铭牌标注的额定载质量。
    -   货物高度不得超过栏板规定高度；封闭厢式货车严禁人货混装。
2.  **固定货物**：
    -   使用绑带、绳索等工具将货物牢固固定于车厢锚点处。运输途中应定期停车检查绑扎情况。
3.  **卸货操作**：
    -   打开栏板前确认周围无人站立在危险区域。对于液压举升自卸车型，举升前确保车厢上方无高压线等障碍物；举升过程中严禁人员靠近车厢下方。
## ⚠️六、紧急情况处理
-   **制动失灵**：立即松开油门踏板；迅速抢挂低挡利用发动机制动；配合使用手刹减速（切勿一次性拉死）；寻找避险车道或利用路边障碍物摩擦减速直至停车。
-   **爆胎处理**：紧握方向盘保持直线行驶；切勿猛踩刹车；缓慢减速并靠边停车；打开双闪警示灯并在后方放置三角警示牌。
-   **车辆起火初期处置方法参考如下表所示。**
| 🔥起火类型分类表 🔥 |
| :---: |
| *电气火灾* → *切断电源总开关* → *使用干粉灭火器扑救* → *严禁用水！* |
| *燃油火灾* → *关闭燃油开关* → *使用干粉灭火器扑救* → *严禁用水！* |
> 💡如果火势无法控制且危及人身安全时请立即弃车撤离至安全地带并拨打119报警电话求助！

# 奥迪Q3 操作指南
  
更新时间：2026年07月12日14时06分26秒   

## 一、启动与熄火
### 1. 启动发动机
- **钥匙启动**：将遥控钥匙插入点火开关（位于方向盘右侧），踩下制动踏板，按下 `START ENGINE STOP` 按钮。
- **无钥匙启动 (Advanced Key)**：确保遥控钥匙在车内，踩下制动踏板，直接按下仪表台上的 `START ENGINE STOP` 按钮。
### 2. 熄火
- 车辆完全停稳后，挂入 `P` (驻车挡)，按下 `START ENGINE STOP` 按钮即可熄火。
> **注意**：如果车辆带有启停系统（Start-Stop），临时停车时发动机会自动熄火，松开制动踏板或转动方向盘会自动重启。

## 二、驾驶模式选择
奥迪Q3通常提供多种驾驶模式，通过中控台上的 `drive select` 按钮切换：
| 模式 | 适用场景 | 特点 |
|------|----------|------|
| **舒适 (Comfort)** | 日常城市通勤、长途 | 悬挂偏软，转向轻盈，油耗经济 |
| **自动 (Auto)** | 默认模式 | ECU自动平衡动力与油耗 |
| **动态 (Dynamic)** | 山路、超车、运动驾驶 | 油门响应更快，转向变重，换挡时机延后 |
| **越野 (Offroad)** | （四驱车型）非铺装路面 | ESP介入更晚，陡坡缓降辅助激活 |
| **个性化 (Individual)** | - | 可自定义发动机、转向、悬挂等参数 |

## 三、挡位操作（7速双离合/8速手自一体）
### P R N D S / M
- **P**：驻车挡。熄火前必须挂入。
- **R**：倒车挡。车辆完全停止后再挂入。
- **N**：空挡。短暂停车或拖车时使用。
- **D**：前进挡。日常行驶最常用。
- **S**：运动模式。延迟升挡以获得更强动力。
- **M** / ± ：手动模式。通过换挡拨片（方向盘后方）或排挡杆手动加减挡。
> *换挡拨片使用方法*：在D/S/M模式下均可使用；拉动右侧拨片升档(+)，左侧拨片降档(-)。
---
## 四、灯光与雨刮
### MMI灯光控制旋钮（位于仪表台左侧）
- `0` / `AUTO` ：关闭 /  自动大灯
- `近光灯`
- `示宽灯`
- `雾灯`：向外拉出旋钮第一段为前雾灯，第二段为后雾灯
### MMI雨刮控制杆（位于方向盘右侧）
- `间歇刮刷`：向上推一档；可通过旋钮调节间歇频率
- `低速连续刮刷`：向上推两档
- `高速连续刮刷`：向上推三档
- `单次刮刷`：向下轻推
- `喷水+刮刷`：向驾驶员方向拉动控制杆
---
##  五、空调与座椅加热
### MMI空调面板操作
1. **温度调节**：
   - LEFT/RIGHT旋钮分别调节主/副驾温度。
   - SYNC键可同步两侧温度设置。
2. **风量调节**：
   - AUTO键开启自动恒温；手动旋转风量旋钮调整风速。
3. **风向选择**：
   - ⬆️ =  吹面；⬇️ =  吹脚；⬆️⬇️ =  吹面+吹脚；除霜图标 = ⛰️前风挡除雾/除霜。
4. **座椅加热**：
   - （如配备）按下座椅图标旁的加热按键（通常有三档强度）。

## 🛞六、驻车辅助系统
### MMI倒车影像 + OPS模拟显示
1. R挂入倒车挡时自动激活后视摄像头和OPS雷达模拟图像。
2. OPS显示车身周围障碍物距离（颜色变化提示距离远近）。
3. P按键可手动激活前后雷达辅助。
### MMI泊车辅助系统（选装）
1. P按键激活泊车辅助 → MMI屏幕选择“泊车” → 
2. D/R根据提示切换 → 
3. Q3会自动打方向完成平行或垂直泊入/泊出。

## 🛞七、MMI多媒体系统操作
### MMI中央旋钮+触摸板
1. MMI主菜单通过旋转/按压中央旋钮进入各项功能：
   - 📻收音机 / 🎵媒体源 / 📞电话 / 🗺️导航 / ⚙️车辆设置等。
2. “触摸板”支持手写输入地址或联系人姓名。
### MMI语音控制
1. “语音按键”（方向盘右侧人像图标）→ 
2. “导航到 [目的地]” / “拨打 [联系人]”等自然语言指令。

## 🛞八、安全与辅助功能
### ESP电子稳定程序开关位置：
ESP OFF按键位于中控台下方储物格内；长按可关闭ESP用于脱困场景。
### ACC自适应巡航控制杆位置：
ACC控制杆位于方向盘左侧后方；
1. SET键设定当前车速；
2. +/-键调整跟车间距；
3. CANCEL取消巡航；
4 RES恢复上次设定速度。
> *注意*：ACC无法识别静止物体或行人；请始终保持对路况的注意力。
---
## 🛞九、保养提示复位方法
当仪表盘显示“保养到期”时：
1️⃣按住仪表盘上的“0,0/SET”按钮不松手；
2️⃣同时打开点火开关至通电状态；
3️⃣等待保养提示闪烁消失后松开按钮；
4️⃣再次按压“0,0/SET”确认复位完成即可清除保养提醒信息。

## 🛞十、常见故障应急处理建议
⚠️以下情况建议立即联系授权经销商进行专业检修：
1️⃣发动机故障灯亮起且伴随抖动/加速无力 → ❗尽快进厂检查；
2️⃣变速箱报警且无法正常换挡 → ❗靠边停车呼叫救援；
3️⃣胎压监测报警且轮胎明显亏气 → ❗更换备胎或补气后重置胎压监测系统；
4️⃣安全气囊故障灯常亮 → ❗禁止继续行驶！立即进厂检修！
5️⃣MMI黑屏死机 → ✅长按音量旋钮约10秒强制重启MMI系统尝试恢复功能正常使用！

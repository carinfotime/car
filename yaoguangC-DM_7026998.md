# 星途瑶光C-DM 动力参数介绍
  
更新时间：2026年07月15日08时40分09秒   

## 动力系统概述
瑶光C-DM搭载 **鲲鹏超性能电混C-DM系统**，是奇瑞集团旗下的插电式混合动力SUV，采用“3擎3挡9模11速”技术。
---
## 核心动力总成
| 项目 | 参数 |
|------|------|
| 发动机 | 1.5T 四缸涡轮增压发动机（第五代ACTECO 1.5TGDI混动专用发动机） |
| 发动机最大功率 | 115 kW（156 马力） |
| 发动机最大扭矩 | 220 N·m |
| 驱动电机 | 永磁同步电机（前置/后置，视版本而定） |
| 电机最大功率 | 165 kW（两驱）/ 175 kW（四驱） |
| 电机最大扭矩 | 390 N·m（两驱）/ 390 N·m + 后桥电机（四驱） |
| 系统综合功率 | 270 kW（约 367 马力） |
| 系统综合扭矩 | ≥ 600 N·m |
---
## 变速箱系统
- **3挡超级电混DHT变速箱**
- 支持：纯电、串联、并联、发动机直驱等多种模式
- 3个物理挡位，覆盖全速域高效区间
---
## 驱动形式
| 版本 | 驱动方式 |
|------|----------|
| 两驱版 | 前置前驱 |
| 四驱版 | 电动四驱（前电机 + 后桥独立电机） |

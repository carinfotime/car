# 奔驰礼骊汽车 V 级 技术参数介绍
  
更新时间：2026年07月19日16时39分48秒   

## 车型概览
奔驰V级是一款集商务、家用与豪华旅行于一体的多功能MPV。礼骊版作为其高端定制或特殊配置版本，在保留奔驰V级核心优势的基础上，进一步提升了内饰的豪华感与乘坐舒适性。

| :--- | :--- |
| **发动机类型** | 2.0T 直列四缸涡轮增压汽油发动机 (M274系列) |
| **排量** | 1991 ml |
| **最大功率** | 155 kW (211 Ps) / 5500 rpm |
| **最大扭矩** | 350 N·m / 1500 - 4000 rpm |

| **驱动方式** | 前置后驱 (FR) |
| **最高车速** |约185 km/h (电子限速)|
| **0-100 km/h加速时间** |约9.1秒 |
## 底盘与悬挂
-   **前悬挂：** AGILITY CONTROL敏捷操控悬挂系统（麦弗逊式独立悬挂）
-   **后悬挂：** AGILITY CONTROL敏捷操控悬挂系统（半拖曳臂式独立悬挂）
-   **制动系统：** 前后通风盘式制动器
-   **转向系统：** ELECTRIC POWER STEERING (EPS)电动助力转向

##  内部配置与空间
###  座椅布局
-   **标准布局:**2+2+3七座布局
-   **礼骊版特色:**通常采用2+2或2+2+3布局，第二排配备航空级独立座椅。
###  第二排航空座椅功能（礼骊版核心亮点）
-   电动调节（前后、靠背、腿托、头枕）
-   加热、通风及按摩功能
-   旋转功能（部分版本支持180°旋转，形成对坐模式）
-   中央扶手箱集成控制面板及无线充电板
###  储物空间
-   后备箱容积:1030 L(七座状态)
-   车内多处储物格、杯架及挂衣钩
##  科技与多媒体
### MBUX智能人机交互系统(部分高配)
-   10.25英寸悬浮式中控触摸屏
-   支持语音控制、导航、蓝牙连接及手机映射(CarPlay/Android Auto)
### Burmester®柏林之声环绕立体声音响(选装/高配)
-   15个扬声器，总功率640W
-   营造沉浸式听觉体验
### COMAND Online车载互联(部分版本)
-   实时路况信息、在线音乐等互联服务
##  安全与辅助驾驶系统
### 主动安全
-**主动式制动辅助系统:**可识别前方车辆及行人并主动刹车。
-**注意力辅助系统:**监测驾驶员疲劳状态并发出提醒。
-**车道保持辅助系统:**当车辆无意偏离车道时进行修正。
-**盲点辅助系统:**监测侧后方盲区车辆并预警。
### 被动安全
-**高强度车身结构**
-**正副驾驶座安全气囊**
-**前后排侧气囊**
-**贯穿式侧气帘**
-**ISOFIX儿童座椅接口**
## 总结:奔驰礼骊汽车V级的技术亮点
1.****动力平顺高效**:M274发动机配合9速变速箱，兼顾动力输出与燃油经济性。**
2.****豪华乘坐体验**:第二排航空座椅是核心卖点，提供超越同级的舒适性与私密性。**
3.****空间灵活性**:超长轴距带来宽敞的车内空间，满足商务接待或家庭出游需求。**
4.****全面安全保障**:丰富的主动和被动安全配置，为每一次出行保驾护航。**

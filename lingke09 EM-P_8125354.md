# 领克09 EM-P 保养指南
  
更新时间：2026年07月20日13时41分37秒   

## 一、保养周期总览
| 保养项目 | 周期 |
|---------|------|
| 首次保养 | 行驶里程达到 **10,000公里** 或 **12个月**（以先到者为准） |
| 常规保养 | 每 **10,000公里** 或 **12个月**（以先到者为准） |
> ⚠️ **注意**：EM-P为插电混动车型，发动机使用频率低于纯燃油车，但保养仍需按时间/里程执行，不可仅凭发动机运行时长判断。

## 三、核心部件专项说明
### 🔋 动力电池系统
| 项目 | 建议 |
|-----|------|
| SOC校准 |  每3个月进行一次满充满放循环 |
|  散热风口清洁 |  每6个月检查后排座椅下方进风口是否堵塞 |
|  高压线束检查 |  每次进店保养时由专业人员检测绝缘电阻 |
|  电池健康度(SOH) |  每年检测一次，低于70%可考虑保修申请 |
### ⚡ EM-P混动系统特殊注意事项
1. **长期停放建议**：若超过15天不使用，建议保持电量在40%~60%之间。
2. *EV模式与HEV模式交替使用*：建议每周至少运行HEV模式30分钟以上，确保发动机得到充分润滑。
3. *能量回收系统*：定期检查制动能量回收功能是否正常。
4. *充电接口维护*：每月清理充电口灰尘异物，防止接触不良。

## ⚠️六、重要提醒与避坑指南
1. ❗首保必须按时完成：领克首保免费且必须在10,000km或12个月内完成，超期视为放弃免费权益。
2. ❗质保政策：领克09 EM-P提供整车5年或10万公里质保；动力电池提供8年或20万公里质保。
3. ❗必须使用原厂认证机油：推荐壳牌0W20全合成机油，符合领克ACEA C5标准。
4. ❗不要在非授权渠道进行高压系统维修——涉及高压电安全风险且可能失去质保。
5. ❗避免长期亏电行驶：即使不充电也建议保持一定电量储备以保护电池寿命。

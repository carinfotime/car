# 启腾N60 驾驶注意事项
  
更新时间：2026年07月12日18时49分22秒   

## 一、启动前检查
1. **轮胎检查**：确保胎压正常（建议冷车状态下检查），轮胎表面无破损、鼓包或异常磨损。
2. **液体检查**：检查机油、冷却液、制动液、转向助力液及玻璃清洗液是否在正常刻度范围内。
3. **灯光与信号**：确认大灯、转向灯、刹车灯、雾灯及危险报警灯功能正常。
4. **后视镜与座椅**：调整好内外后视镜及座椅位置，确保视野良好且驾驶姿势舒适。
5. **安全带**：所有乘员必须系好安全带。
## 二、启动与起步
1. **点火启动**：将钥匙拧至“ON”档，等待仪表盘自检完成后再启动发动机。
2. **预热**：冷车启动后建议怠速预热30秒至1分钟，待发动机转速稳定后再起步。
3. **离合与档位**（手动挡）：踩下离合器踏板至底，挂入1档，缓慢松开离合的同时轻踩油门，避免熄火或窜车。
4. **手刹解除**：确认手刹已完全释放，观察仪表盘手刹指示灯熄灭。
## 三、行驶中注意事项
1. **转速控制**：
   - 换挡建议转速：2000-2500 rpm
   - 避免长时间低转速高负荷行驶或高转速空转
2. **换挡操作**：
   - 手动挡车型换挡时离合必须踩到底
   - 倒车时需完全停稳后再挂入倒挡
3. **车速控制**：
   - 遵守道路限速规定
   - 下长坡时利用发动机制动，避免长时间踩刹车导致制动过热
4. **载重限制**：
   - 不超过车辆额定载重（具体数值请参照用户手册）
   - 货物均匀分布，避免偏载影响操控稳定性
5. **空调使用**：合理使用空调，避免长时间最大风量运行增加油耗和发动机负荷。
## 四、特殊路况驾驶
1. **雨天/湿滑路面**：
   - 降低车速，增大跟车距离
   - 避免急刹车、急转向
   - 涉水时减速慢行，水深不超过排气管高度
2. **山路/坡道**：
   - 上坡提前降档，保持动力充足
   - 下坡使用低档位辅助制动
3. **拥堵路况**：
   - 频繁启停时注意离合与油门的配合
   - 避免长时间半联动行驶，防止离合器过热
## 五、停车与熄火
1. **停车操作**：
   - 踩下离合和刹车，车辆停稳后挂入空挡（手动挡）或P挡（自动挡）
   - 拉紧手刹，确认车辆不会溜动
2. **熄火前**：
   - 关闭空调、音响等用电设备
   - 若长时间高速行驶后，建议怠速运转1-2分钟再熄火，保护涡轮增压器（如配备）
3. **坡道停车**：
   - 手动挡车型可将档位挂入1档或倒挡，配合手刹双重保险
   - 前轮朝向路沿方向
## 六、日常维护建议
| 项目 | 检查/更换周期 |
|------|--------------|
| 机油与机滤 | 每5000公里或6个月 |
| 空气滤清器 | 每10000公里检查，必要时更换 |
| 制动液 | 每2年或40000公里更换 |
| 轮胎换位 | 每10000公里 |
| 火花塞 | 每30000公里检查 |
| 正时皮带 | 参照用户手册建议里程 |
## 七、安全提醒
1. **严禁酒驾、疲劳驾驶**：连续驾驶不超过4小时，中途适当休息。
2. **保持安全车距**：车速每增加10km/h，跟车距离应相应增加。
3. **仪表盘警示灯**：任何异常指示灯亮起时，应及时停车检查或联系维修。
4. **随车工具**：随车携带灭火器、三角警示牌、千斤顶及备胎（如有）。

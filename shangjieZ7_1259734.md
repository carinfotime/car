## 二、日常检查项目（建议每周/每月）
### ✅ **每周检查**
- **胎压**：标准胎压参考驾驶侧门框标签（通常前轮2.3bar，后轮2.5bar）
- **灯光系统**：大灯、转向灯、刹车灯、雾灯是否正常
- **仪表盘警示灯**：是否有异常亮起
### ✅ **每月检查**
- **机油液位**：冷车状态下拔出机油尺，液位应在上下刻度之间
- **冷却液液位**：观察膨胀罐，应在MIN与MAX之间
- **玻璃水**：不足时及时补充
- **刹车片厚度**：目测是否低于3mm
- **雨刮片**：刮拭是否清晰，有无异响或老化

## 三、各阶段深度保养内容
### 🔧 **首保（5,000km / 3个月）**
- ✅更换机油及机滤
- ✅全车螺栓紧固检查
- ✅底盘及悬挂系统检查
- ✅制动系统检查
- ✅轮胎磨损及气压检查
- ✅各种油液液位检查补充
### 🔧 **小保养（每5,000km）**
- ✅更换机油 &amp;机滤
- ✅全车灯光检查
- ✅轮胎气压及磨损检查
- ✅底盘螺丝紧固
### 🔧 **大保养（每20,000km）**
包含小保养全部项目 +：
- ✅更换空气滤清器
- ✅更换空调滤清器
- ✅更换燃油滤清器（视情况）
- ✅四轮定位检测调整
- ✅制动系统深度清洁与检测
### 🔧 **全车大保养（每60,000km）**
包含大保养全部项目 +：
- ✅更换变速箱油
- ✅更换火花塞（如未提前更换）
- ✅更换制动液（如未提前更换）
- ✅更换冷却液（如未提前更换）
- ✅正时皮带/链条检查（视发动机类型而定）
- ✅发电机、空调压缩机皮带检查与调整

##  四、易损件更换建议
|  易损件 |  建议寿命/里程 |
|--------:|:--------------|
| 🛞轮胎 |  50,000–60,000km / ≤4年 |
| 🛑刹车片 |  30,000–50,000km（视驾驶习惯） |
| 🛑刹车盘 |  60,000–80,000km |
| 💡灯泡/LED模组 |  按需更换，亮度下降即换 |
| 🌧️雨刮片 |  6–12个月 /出现异响或刮不净时换 |

## ⚠️五、重要注意事项
1. **使用原厂配件与推荐标号机油**
   - Z7推荐使用 SAE 5W‑30 全合成机油，符合 API SN/SP 等级。
2. **不要超期超里程保养**
   - “再跑几百再保”可能造成发动机磨损加剧。
3. **注意仪表盘提示**
   - Z7配备智能保养提醒系统，请勿忽视“立即保养”提示。
4. **冬季特殊养护**
   - ❄️入冬前应检查防冻液冰点、蓄电池健康度。
5. **长期停放注意事项**
   - 🅿️超过15天不使用应断开电瓶负极，并每周启动一次运行15分钟。

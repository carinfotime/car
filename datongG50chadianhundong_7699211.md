# 上汽大通MAXUS G50 插电混动版 技术参数介绍
  
更新时间：2026年07月19日14时56分43秒   

## 一、车型概览
上汽大通MAXUS G50插电混动版是一款紧凑型家用MPV，搭载了由1.3T发动机与电动机组成的插电式混合动力系统，兼顾燃油经济性与纯电续航能力，适合城市通勤与家庭出行。
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **发动机类型** | 1.3T 涡轮增压发动机 |
| **发动机排量** | 1.3L |
| **发动机最大功率** | 120 kW（约163马力） |
| **发动机最大扭矩** | 230 N·m |
| **电动机类型** | 永磁同步电机 |
| **电动机最大功率** | 60 kW |
| **电动机最大扭矩** | 160 N·m |
| **系统综合功率** | ≈180 kW（约245马力） |
| **系统综合扭矩** | ≈385 N·m |
| **变速箱类型** | AMT（机械式自动变速箱）或 IAMT（智能自动手动变速箱） |
## 三、电池与续航
| 项目 | 参数 |
|------|------|
| **电池类型** | 三元锂电池 |
| **电池容量** | ≈13 kWh / ≈52.5 kWh（视具体版本而定） |
| **纯电续航里程（NEDC）** | ≥64 km（部分版本可达80km以上） |
| **综合续航里程（满油满电）** | ≥900 km |
> *注：不同年款及配置版本电池容量与续航存在差异，以上为常见参考值。*
## 四、充电性能
- **交流慢充时间（家用220V/16A）：** ≈3-4小时充满
- **直流快充时间：** （部分高配版本支持，约30分钟充至80%）
## 五、车身尺寸与空间
| 项目 | 参数 |
|------|------|
| **长 × 宽 × 高 (mm)** | ≈4825 × 1825 × 1778 / (视配置略有差异) |
| **轴距 (mm)** | ≈2800 mm |
| **座位数** | 6座 /7座可选 （2+2+2 /2+2+3布局）|
| **油箱容积 (L)** | ≈38 L |
##  六、底盘与操控
-  驱动方式：前置前驱
-  前悬挂类型：麦弗逊式独立悬挂
-  后悬挂类型：扭力梁式非独立悬挂
-  转向系统：电动助力转向
-  制动系统：前通风盘式 /后盘式
## ⚙️七、能耗表现
-  最低荷电状态油耗（馈电油耗）：≈5.0 -6.0 L/100km
-  综合油耗（满油满电）：≈1.4 -1.8 L/100km （NEDC工况）
## 🛡️八、安全配置
-  主/副驾驶安全气囊
-  前排侧气囊
-  前后排头部气帘 （部分高配）
-  车身稳定控制系统 (ESC)
-  牵引力控制系统 (TCS)
-  坡道辅助起步 (HHC)
-   360°全景影像 （高配）
-   前后驻车雷达
## 📱九、智能科技配置
-   12.3英寸中控触摸屏 （部分版本）
-   7英寸液晶仪表盘 （部分版本）
-   斑马智行互联系统 （支持语音控制、导航、远程控制等）
-   手机App远程控制 （空调开关、车门解锁等）
-   无钥匙进入 +一键启动
-   定速巡航 /自适应巡航 （高配）

# 金海狮M 保养指南
  
更新时间：2026年07月20日08时29分35秒   

## 一、概述
本指南适用于 **金海狮M** 车型。定期保养不仅能延长车辆使用寿命，还能确保行车安全与燃油经济性。建议严格按照以下周期进行维护。
---
## 二、保养周期总览
| 项目 | 周期（公里或时间，以先到为准） |
| :--- | :--- |
| **首次保养** | **2,500公里 或 3个月** |
| **常规保养** | **每5,000公里 或 6个月** |
| **中期保养** | **每20,000公里 或 12个月** |
| **大保养** | **每40,000公里 或 24个月** |
> *注：若经常在恶劣路况（多尘、高温、短途频繁启停）下行驶，建议适当缩短保养间隔。*
---
## 三、各阶段详细项目
### （一）首次保养（2,500km / 3个月）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查并调整发动机怠速
- [ ] 检查冷却液液位及管路
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查变速箱油液位
- [ ] 检查轮胎气压及磨损情况
- [ ] 紧固底盘及车身螺栓
- [ ] 检查灯光及信号系统
### （二）常规小保养（每5,000km / 6个月）
- [ ] **更换发动机机油**
- [ ] **更换机油滤清器**
- [ ] *清洁空气滤清器滤芯*（视情况）
- [ ] *清洁空调滤清器滤芯*（视情况）
- [ ] *补充玻璃清洗液*
- [ ] *检查并调整皮带张紧度*
- [ ] *检查蓄电池状态*
### （三）中期保养（每20,000km / 12个月）
包含小保养所有项目，并增加：
#### 🔧必换项目：
- [x] **更换空气滤清器滤芯**
- [x] **更换空调滤清器滤芯**
#### 🔍检查项目：
- [x] *火花塞状况*（建议30,000km左右考虑更换）
- [x] *制动片厚度*（前/后）
- [x] *制动盘磨损*
- [x] *手刹行程*
#### 🔄调整/润滑：
- [x] *润滑车门铰链及锁扣*
#### 🛞轮胎：
> -  四轮换位（交叉换位法）
> -  动平衡检测（如方向盘抖动）
### （四）大保养（每40,000km / 24个月）
包含中期所有项目，并增加：
#### 🔧必换项目：
| # |  部件名称 |  备注 |
|---|:----------:|:------|
| ✅ |  变速箱油 (MT) |  手动挡专用GL‑4级 |
| ✅ |  后桥齿轮油 (GL‑5) |  后驱车型必需 |
| ✅ |  防冻液 (冷却液) |  建议使用原厂长效型 |
| ✅ |  火花塞 (4支) |  铱金型寿命更长 |
| ✅ |  燃油滤清器 (汽油格) |  外置式需同步更换 |
#### 🔍深度检查：
> -   正时皮带/链条状态 (1.5L/1.6L发动机通常为链条免维护，但需确认)
> -   发电机、空调压缩机皮带裂纹
> -   各橡胶衬套老化程度 (下摆臂、平衡杆)
> -   减震器是否漏油
---
## ⚠️四、关键部件推荐规格
### 🛢️机油规格
> SAE等级：**5W‑30** (四季通用)  
> API等级：不低于 **SL级**  
> SN级及以上更佳  
> ※容量约：3.0~3.5升 (以油尺为准)
### ❄️冷却液规格
> -   颜色：通常为绿色或红色  
> -   冰点要求：北方地区 ≤ ‑35℃；南方地区 ≤ ‑25℃  
> -   推荐使用乙二醇基长效防冻液  
### 🚗变速箱油规格
|    类型     |     推荐型号      |
|:-----------:|:----------------:|
| MT手动挡    | GL‑4 / SAE 75W‑90 |
### ⚙️后桥齿轮油规格
> GL‑5 / SAE 80W‑90

## 🛠️五、日常自查清单（每周/每月）
1. □ **胎压检测**
2. □ *机油尺刻度* —冷车时在上下限之间
3. □ *冷却液副水壶液面*
4. □ *制动踏板行程与脚感*
5. □ *灯光全检* —大灯/转向灯/刹车灯/倒车灯
6. □ *雨刮喷水效果*
7. □ *仪表盘故障灯* —启动自检后应全部熄灭

## 📌七、重要提示
1. 【质保期内】请务必前往授权服务站进行维保，以免影响保修权益。
2. 【配件选择】尽量使用原厂配件或符合原厂标准的品牌件。
3. 【记录留存】每次保养后保留工单与发票，便于二手车交易时证明车况。
4. 【特殊工况】如长期重载或跑工地等恶劣环境，建议将换油周期缩短至3,500~4,500公里。
5. 【冬季预热】寒冷地区启动后怠速运转约1分钟再行驶。

# 福特烈马（Ford Bronco）驾驶注意事项
  
更新时间：2026年07月18日08时32分27秒   

## 一、提车与初期检查
1.  **胎压调整**：新车出厂胎压通常较高（约3.5 bar），提车后请立即将胎压调整至标准值（约2.4-2.6 bar，具体见驾驶位B柱标签），否则会影响舒适性和抓地力。
2.  **熟悉可拆卸部件**：烈马的车顶、车门多为可拆卸设计。首次操作前，请仔细阅读说明书，掌握正确的拆装步骤和存放方法，避免损坏卡扣或密封条。
3.  **检查油液**：确认机油、冷却液、玻璃水等液位正常。
## 二、日常公路驾驶
1.  **转向特性**：烈马采用非承载式车身和整体桥悬挂（部分车型），转向存在一定虚位，高速行驶时需微调方向，避免猛打方向盘。
2.  **风噪与共振**：可拆卸硬顶或软顶的隔音性能不如传统SUV。时速超过80km/h时风噪明显增大；部分车型在特定转速区间可能存在共振，属正常现象。
3.  **油耗预期**：烈马自重较大且风阻高，综合油耗通常在12-16L/100km甚至更高。建议平稳加速，提前预判路况以减少刹车。
4.  **高速行驶**：
    -   请勿超速行驶。高重心车型在急变道或强侧风下稳定性下降。
    -   若车顶或车门已拆卸，严禁在高速公路上行驶（法规及安全要求）。
5.  **停车注意**：
    -   由于车身较高且盲区较大，停车时请善用后视镜和倒车影像。
    -   确认车位上方净空高度是否足够（尤其是加装行李架或天线后）。
## 三、越野驾驶核心要点
1.  **四驱系统使用**
    -   **铺装路面严禁使用4H/4L模式**：只能在湿滑、沙地、泥地或非铺装路面使用四驱模式。干燥柏油路上使用会损坏传动系统并导致转向困难。
    -   **切换至4L（低速四驱）前必须停车挂N挡**：操作需在车辆静止、变速箱处于N挡时进行。听到机械啮合声后再挂挡行驶。
    -   **差速锁使用**：仅在低速越野且车轮打滑时开启差速锁。脱困后应立即关闭，否则转弯时会损坏半轴。
2.  **地形模式选择**
    -   G.O.A.T.（Goes Over Any Type of Terrain）模式管理系统：
        -   Normal / Eco / Sport / Slippery / Mud & Ruts / Sand / Rock Crawl
        -   *泥地/沙地*：保持高转速以甩掉泥土或防止陷车。
        -   *岩石攀爬*：启用4L + 差速锁 + Rock Crawl模式，控制车速缓慢通过。
3.  **涉水注意事项**
    -   Bronco原厂涉水深度约为800mm（具体视配置而定）。
    -   **涉水前**：检查并确认发动机进气口位置未被淹没；关闭自动启停功能。
    -   **涉水中**：保持匀速低档位通过（约5-10 km/h），切勿收油或换挡以免排气管倒吸进水。
    -   **涉水后**：轻踩刹车以蒸发刹车盘水分恢复制动性能；检查差速器及变速箱是否进水乳化。
4.  **底盘保护**
    -   Bronco原厂带有钢制底盘护板。但高强度越野仍建议加装更全面的护板保护油箱、分动箱等脆弱部位。
## 四、特殊结构与功能操作
1.  **可拆卸车顶与车门**
    -   *拆卸步骤*：严格按照手册顺序操作——先拆前部面板 → B柱连接件 → 后部面板/硬顶总成。
    -   *存放*：拆下的部件应平放于专用支架上或用软布包裹存放于阴凉处以防变形划伤。
    -   *重新安装*确保所有卡扣到位且密封条贴合紧密否则雨天会漏水。
2.  ***防倾杆断开功能***
    部分高配车型配备可断开的前防倾杆用于极限攀爬提升悬挂行程：
    只能在极低速越野时启用且必须在平坦地面操作启用前后注意观察是否有异物卡滞脱困后务必及时重新连接防倾杆再上路行驶否则公路过弯有侧翻风险！
3.**后备胎架**
尾门承载备胎重量较大开关尾门需用力到位确保锁止机构完全闭合长期颠簸路段建议定期检查备胎架螺栓是否松动异响
## 五安全警示清单
|场景|关键提醒|
|---|---|
|公路变道|盲区监测系统有效但不可完全依赖扭头确认侧方盲区|
|儿童上下车|车身离地间隙高建议使用踏板辅助防止摔伤|
|拖拽牵引|Bronco具备拖拽能力但必须正确计算拖车重量并安装制动控制器|
|燃油加注|建议使用92号及以上无铅汽油越野归来及时检查油箱是否有泥沙渗入|
## 六定期维护提示
1.**轮胎换位**:每5000-8000公里进行交叉换位因四驱系统前后轮胎磨损差异大
2.**润滑保养**:定期给车门铰链限位器以及可拆卸车顶的锁扣涂抹润滑脂防止异响卡滞
3.**排水孔清理**:敞篷版车型需经常清理A柱及地板排水孔防止积水发霉
4.**电瓶维护**:若长期停放不使用建议连接智能充电器维持电瓶状态因车辆电子模块较多静态耗电量较高

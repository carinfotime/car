## 三、核心保养项目详解
### 1. 驱动电机润滑油
- **更换周期：** 每 60,000km 或 3年
- **作用：** 润滑电机齿轮及轴承，散热
- **建议：** 使用丰田原厂纯电动专用电机油
### 2. 减速器油
- **更换周期：** 每 60,000km 或 3年
- **用量：** 约 1.2L
- **注意事项：** 必须使用 EV 专用减速器油，不可混用普通齿轮油
### 3. 制动液
- **更换周期：** 每 40,000km 或 2年
- **规格：** DOT4 或更高标准
- **注意：** bZ3 配备再生制动系统，制动片磨损较燃油车慢，但制动液仍需按时更换
### 4. 空调滤芯
- **更换周期：** 每 20,000km 或 1年（建议每年换季前更换）
- **建议：** 使用带活性炭的PM2.5过滤滤芯
### 5. 冷却液
- **类型：** 电池冷却液 + 电机冷却液（通常共用循环）
- **首次更换：** 100,000km 或 5年
- **之后：** 每 50,000km 或 2.5年
- **注意：** 必须使用丰田 EV 专用长效冷却液
### 6. 12V 蓄电池
- **类型：** 免维护铅酸电池
- **检查周期：** 每次保养
- **寿命：** 一般 3-5 年
- **提示：** bZ3 的 12V 蓄电池由动力电池充电，如长期停放需注意亏电

## 四、日常自查项目（建议每月一次）
| 检查项目 | 检查内容 |
|----------|----------|
| 轮胎气压 | 标准值：2.4-2.6 bar（根据车门标签） |
| 轮胎磨损 | 花纹深度 ≥ 1.6mm，无偏磨、鼓包 |
| 玻璃水 | 加注专用玻璃水（冬季注意防冻） |
| 灯光系统 | 大灯、转向灯、刹车灯、雾灯 |
| 雨刮片 | 刮拭是否清晰，有无异响 |
| 充电口 | 检查充电口有无异物、腐蚀 |
| 仪表盘报警 | 是否有故障灯亮起 |

## 五、电池保养建议
### 1. 充电习惯
- **日常使用：** 建议电量保持在 **20%-80%** 之间
- **长途出行：** 可充至 100%，但到达后尽快使用，避免长时间满电停放
- **快充使用：** 控制快充频次，日常建议以慢充为主
- **冬季充电：** 建议刚行驶结束后立即充电（电池温度较高，充电效率更好）
### 2. 长期停放
- 建议保持电量在 **50%-60%**
- 每 **30天** 至少启动一次并充电
- 停放地点避免高温、暴晒、潮湿
### 3. 电池健康度
- 丰田 bZ3 提供 **电池质保：8年或20万公里（以先到为准）**
- 建议每年到 4S 店做一次 **电池健康度检测**
- 如发现续航衰减明显（超过 30%），及时到店检测

## 六、季节性保养
### 春季
- 更换空调滤芯
- 检查空调系统制冷效果
- 清理充电口及机舱灰尘
### 夏季
- 检查冷却液液位
- 注意高温下充电散热（避免暴晒后立即快充）
- 检查轮胎气压（夏季适当调低 0.1-0.2 bar）
### 秋季
- 检查雨刮片及玻璃水
- 检查制动系统
- 检查灯光系统
### 冬季
- 更换冬季玻璃水（-25°C 以下）
- 检查 12V 蓄电池电量
- 注意低温对续航的影响（提前规划充电）
- 冰雪路面建议更换冬季轮胎

# 沃尔沃 S60 操作指南
  
更新时间：2026年07月15日14时24分07秒   

本指南旨在帮助您快速熟悉并安全操作您的沃尔沃 S60。请在驾驶前仔细阅读车辆用户手册，并始终遵守当地交通法规。
## 一、启动与驾驶
### 1. 钥匙与解锁
- **无钥匙进入**：钥匙在身上时，直接拉动车门把手即可解锁。
- **钥匙遥控**：
    - **解锁**：短按钥匙上的解锁按钮。
    - **锁车**：短按锁车按钮。长按可同时关闭所有车窗和天窗。
- **手套箱锁**：手套箱可通过中控锁控制，也可使用物理钥匙锁止。
### 2. 启动发动机
S60 配备无钥匙启动系统。
1.  确保钥匙位于车内。
2.  踩住刹车踏板。
3.  按一下中控台上的 **“START/STOP ENGINE”** 按钮。
4.  发动机启动后，仪表盘会显示“READY”。
### 3. 选择档位
- **D (前进挡)**：向后拉动换挡杆。
- **R (倒车挡)**：向前推动换挡杆。
- **N (空挡)**：在 D 或 R 位置时，短时间前后推动换挡杆至 N 位置。
- **P (驻车挡)**：按下换挡杆顶端的 **P** 按钮。
- **手动模式**：将换挡杆从 D 位置向左或向右拨动，进入手动模式。向上（+）或向下（-）推动换挡杆进行加减档。
### 4. 电子手刹
- **启用**：按下中控台上的 **P** 按钮（带刹车图标）。
- **解除**：踩住刹车，按下 **P** 按钮。挂入 D 档并轻踩油门，手刹会自动解除。
### 5. 关闭发动机
1.  挂入 P 档。
2.  按下 **“START/STOP ENGINE”** 按钮。
3.  车辆熄火。
## 二、驾驶辅助系统
### 1. 自适应巡航控制 (ACC)
- **开启**：按下方向盘左侧的 **ACC 按钮** (带车速表图标)。
- **设定速度**：达到目标速度后，按下 **SET/-** 按钮。
- **调整速度**：按 **+** 或 **-** 按钮微调（通常以 5 km/h 为步进）。
- **调整跟车距离**：按 **距离调节按钮**（车辆图标带横线）。
- **暂停/恢复**：按下 **CANCEL** 暂停，按下 **RESUME** 恢复。
- **关闭**：再次按下 **ACC 按钮**。
### 2. Pilot Assist (领航辅助系统)
- **激活**：在 ACC 激活状态下，按下方向盘左侧的 **Pilot Assist 按钮** (方向盘图标)。
- **功能**：系统会保持设定速度、跟车距离，并主动将车辆保持在车道中央。
- **注意**：驾驶员需始终手握方向盘，并观察路况。
### 3. 盲点信息系统 (BLIS)
- 当侧后方有车辆接近时，对应侧的后视镜内侧会亮起黄色指示灯。
- 若此时打转向灯，指示灯会闪烁并发出警示音。
### 4. 车道保持辅助
- 当车辆无意中偏离车道时，系统会轻微修正方向盘，将其引导回车道。
- 可通过中控屏幕关闭此功能（设置 > 驾驶辅助）。
## 三、信息娱乐系统 (Sensus)
### 1. 中控触摸屏
- **主页**：点击屏幕顶部的 **主页按钮** (房子图标)。
- **应用程序**：滑动屏幕查看所有应用。
- **空调控制**：屏幕底部常驻空调控制条，点击可展开详细设置。
- **音量**：使用物理旋钮或方向盘上的滚轮调节。
### 2. Apple CarPlay / Android Auto
- 使用原装 USB 数据线连接手机。
- 手机上的应用程序会自动投射到中控屏幕。
- 可通过 Siri 或 Google Assistant 进行语音控制。
### 3. 导航
- **内置导航**：在“导航”应用中输入目的地。
- **在线导航**：通过 CarPlay/Android Auto 使用手机地图。
### 4. 语音控制
- 按下方向盘右侧的 **语音控制按钮**。
- 说出指令，例如：“导航到最近的加油站”、“打电话给张三”、“调高空调温度”。
## 四、灯光与视野
### 1. 灯光控制杆
- **位置**：方向盘左侧。
- **AUTO**：自动大灯模式（推荐）。
- **近光灯**：旋转至近光灯位置。
- **远光灯**：向前推动控制杆。
- **自适应远光灯**：在 AUTO 模式下，远光灯会根据对向车辆自动切换远近光。
- **雾灯**：旋转控制杆上的环。
- **转向灯**：向上或向下拨动控制杆。
### 2. 雨刮器控制杆
- **位置**：方向盘右侧。
- **单次刮刷**：向下拨动。
- **间歇/自动刮刷**：向上拨动至第一档（可调节灵敏度）。
- **连续刮刷**：向上拨动至第二档（慢速）或第三档（快速）。
- **喷水清洗**：向驾驶员方向拉动控制杆。
- **后雨刮**：旋转控制杆末端旋钮。
### 3. 电动后视镜
- 调节按钮位于驾驶员侧车门上。
- 选择左/右后视镜，然后用方向键调节角度。
- 按下 **折叠按钮** 可折叠后视镜。
## 五、舒适性与便利性
### 1. 座椅调节
- **电动座椅**：座椅侧面的按钮可调节前后、高低、坐垫角度和靠背角度。
- **腰部支撑**：座椅侧面的圆形按钮可调节腰部支撑的硬度和位置。
- **座椅记忆**：设置好座椅位置后，按下 **M** 按钮，再按数字键（1 或 2）保存。下次上车时按数字键即可恢复。
### 2. 空调系统
- **自动模式**：点击中控屏幕上的 **AUTO** 按钮。
- **温度调节**：滑动屏幕上的温度条。
- **座椅加热/通风**：点击屏幕上的座椅图标。
- **方向盘加热**：点击屏幕上的方向盘图标。
- **除霜**：点击屏幕上的 **前/后除霜** 按钮。
### 3. 天窗
- **开启**：向后滑动天窗控制开关。
- **倾斜**：向上按开关前端。
- **关闭**：向前滑动开关。
### 4. 后备箱
- **开启**：按下车尾沃尔沃标志下方的按钮。
- **关闭**：按下后备箱内部的关闭按钮。
- **脚踢感应**：钥匙在身上时，在后保险杠下方做踢腿动作，可开启或关闭后备箱（需配备此功能）。
## 六、维护与保养
### 1. 加油
- **燃油类型**：推荐使用 **95号及以上无铅汽油**。
- **油箱盖**：位于车辆右侧。车辆解锁后，按压油箱盖即可打开。
- **燃油加注口**：无盖加油系统，直接插入加油枪即可。
### 2. 轮胎压力
- **标准胎压**：可在驾驶员侧车门门框上的标签上找到。
- **胎压监测**：当胎压异常时，仪表盘会显示警告。可在中控屏幕中查看和重置胎压监测系统。
### 3. 保养提示
- 车辆会根据行驶里程和时间自动计算保养周期。
- 当仪表盘显示保养提示时，请尽快联系沃尔沃授权经销商进行保养。
### 4. 发动机舱检查
- **打开引擎盖**：拉动驾驶员侧脚部空间的拉手，然后按压引擎盖前端的锁扣。
- **检查机油**：定期检查机油尺，确保机油量在正常范围内。
- **检查冷却液**：检查膨胀壶内的冷却液液位。
- **检查玻璃水**：定期添加玻璃水。
## 七、安全功能
### 1. 安全带
- 所有乘客必须始终系好安全带。
- 未系安全带时，车辆会发出警告音。
### 2. 安全气囊
- 车辆配备多个安全气囊，包括前气囊、侧气囊和侧气帘。
- 请勿在安全气囊展开区域放置任何物品。
### 3. 儿童安全
- **儿童座椅**：后排座椅可安装 ISOFIX 接口的儿童座椅。
- **儿童安全锁**：后车门上的儿童安全锁可防止儿童从车内打开车门。
### 4. City Safety (城市安全系统)
- 系统会自动检测前方车辆、行人、骑行者及大型动物。
- 当存在碰撞风险且驾驶员未采取行动时，系统会自动刹车。
## 八、紧急情况
### 1. 应急警示灯
- 按下中控台上的 **红色三角按钮**，所有转向灯会闪烁。
### 2. 拖车
- 车辆前部有一个可拆卸的拖车钩盖板。
- 拖车时需将档位置于 N 档，并松开电子手刹。
### 3. 备用轮胎
- 部分车型配备补胎工具包，部分车型配备备胎。请查阅用户手册了解具体位置和使用方法。
### 4. 紧急呼叫
- 按下车顶控制面板上的 **SOS 按钮**，可连接沃尔沃紧急服务中心。
---
**重要提示**：
- 本指南仅为快速参考，详细操作请以车辆用户手册为准。
- 驾驶辅助系统不能替代驾驶员的注意力和判断力。
- 定期保养是确保车辆安全和性能的关键。

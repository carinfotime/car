# 奥迪A4（进口）技术参数介绍
  
更新时间：2026年07月11日09时23分44秒   

## 一、车型概述
奥迪A4（进口）是奥迪品牌旗下的一款中型豪华轿车，以原装进口形式引入国内市场。相比国产版奥迪A4L，进口版A4在轴距、底盘调校和配置上有所不同，更偏向原汁原味的欧式驾驶体验。
---

| 参数项 | 数值 |
|--------|------|
| 发动机类型 | 直列四缸涡轮增压 |
| 排量 | 1984 mL |
| 最大功率 | 150 kW（204 PS）/5000-6000 rpm |
| 最大扭矩 | 320 N·m /1450-4200 rpm |
| 燃油标号 | 95号及以上无铅汽油 |
| 排放标准 | 国VI |
### quattro® ultra智能四驱系统
- **结构**：多片离合器式中央差速器
- **特点**：可根据路况自动切换两驱/四驱模式，兼顾燃油经济性与操控稳定性

## 六、性能表现
### Avant / Sedan （2.0T + quattro）
- **0–100 km/h加速时间**：约6.9秒
- **最高车速**：250 km/h（电子限速）
- **综合油耗（WLTC）**：约7.3–8.1 L/100km
- **油箱容积**：58 L

# 宝马 i5 M60 空间参数介绍
  
更新时间：2026年07月16日18时15分18秒   

## 车身尺寸
| 参数 | 数值 |
|------|------|
| 车长 | 5,060 mm |
| 车宽（含后视镜） | 2,168 mm |
| 车宽（不含后视镜） | 1,900 mm |
| 车高 | 1,515 mm |
| 轴距 | 2,995 mm |
| 前轮距 | 1,624 mm |
| 后轮距 | 1,638 mm |
## 车内空间
### 前排空间
- **头部空间**：约 **990–1,010 mm**（视座椅高度调节）
- **腿部空间**：充裕，支持多向电动调节座椅
- **肩部空间**：约 **1,450 mm**
- **座椅调节范围**：电动前后调节 + 高低调节 + 靠背角度调节
### 后排空间
- **头部空间**：约 **950–970 mm**
- **腿部空间**：约 **890–920 mm**（前排调至适中位置）
- **肩部空间**：约 **1,420 mm**
- **后排座椅特点**：40:20:40比例放倒，中央扶手带杯架
## 储物空间
### 后备箱
| 项目 | 数据 |
|------|------|
| **标准容积** | **490 L** |
| **最大容积（后排放倒）** | **约1,670 L** |
| **后备箱开口宽度** | ~1,080 mm |
| **后备箱深度（至后排）** | ~1,050 mm |
| **后备箱深度（放倒后排）** | ~2,000 mm |
### 车内储物格
- **前门储物格**：可容纳2瓶500ml水瓶 +杂物
- **后门储物格**：可容纳1瓶500ml水瓶 +杂物
- **中央扶手箱**：深度约150mm，带USB-C接口
- **手套箱**：容积约8L，带照明和锁止功能
- **杯架数量**：
    -前排中央杯架 ×2
    -后排中央扶手杯架 ×2
    -前门板杯架 ×2（每侧各一）
    -后门板杯架 ×2（每侧各一）
## 💡亮点说明
> - BMW i5 M60采用长轴距设计，轴距接近3米，为后排提供了宽敞的腿部空间。
> - M60版本相比普通i5在底盘调校上更运动化，但车内乘坐空间保持一致。
> -后备箱标准容积490L在同级纯电轿车中表现优秀，且支持4/2/4比例放倒扩展实用性。
> -由于底盘布置电池组，后排地台有轻微隆起（约50mm），但整体平整度控制良好。
---
*以上数据基于官方发布及实测参考，实际体验可能因配置、座椅材质及个人坐姿略有差异。*

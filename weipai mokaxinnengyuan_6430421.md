# 魏牌 摩卡新能源 动力参数介绍
  
更新时间：2026年07月18日13时12分11秒   

## 动力系统概述
魏牌摩卡新能源搭载 **长城汽车柠檬智能混动DHT系统**，采用 **1.5T涡轮增压发动机 + 前置单电机（两驱）/前后双电机（四驱）** 的插电式混合动力架构，匹配 **2挡DHT专用混动变速箱**。
---
## 核心动力参数
### 1.5T涡轮增压发动机
| 项目 | 参数 |
|------|------|
| 发动机型号 | E15BD |
| 排量 | 1.5L |
| 进气形式 | 涡轮增压 |
| 最大功率 | **115 kW（156 Ps）** |
| 最大扭矩 | **235 N·m** |
| 燃油标号 | 92号及以上无铅汽油 |
### 驱动电机
#### 🚗两驱版（前置单电机）
| 项目 | 参数 |
|------|------|
| 电机类型 | 永磁同步电机 |
| 最大功率 | **130 kW** |
| 最大扭矩 | **300 N·m** |
#### 🚙四驱版（前后双电机）
| - | - |
|---|---|
| **前电机**：最大功率130 kW /  最大扭矩300 N·m |
| **后电机**：最大功率135 kW /  最大扭矩232 N·m |

### 🔋电池组

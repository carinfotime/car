# 图雅诺大麦 动力参数介绍
  
更新时间：2026年07月16日14时43分58秒   

## 核心动力总成
图雅诺大麦搭载了福田汽车自主研发的 **欧康（Ounk）2.0T 柴油发动机**，并匹配了行业领先的传动系统，旨在提供强劲、高效且平顺的驾驶体验。
### 发动机参数
| 项目 | 参数 |
| :--- | :--- |
| **发动机型号** | 欧康 4F20TC |
| **排量** | 2.0L (1968ml) |
| **进气形式** | 涡轮增压 |
| **燃料类型** | 柴油 |
| **最大马力** | **150 Ps** (约110kW) |
| **最大功率转速** | 3400 rpm |
| **最大扭矩** | **330 N·m** |
| **最大扭矩转速区间** | 1600 - 2400 rpm |
### 技术亮点
-   **高效燃烧技术：** 采用博世（Bosch）高压共轨燃油喷射系统，喷射压力高，雾化效果更好，实现充分燃烧。
-   **轻量化设计：** 缸体及关键部件采用铝合金等轻量化材料，有效降低发动机自重，提升燃油经济性。
-   **低噪音优化：** 通过正时静音链、双平衡轴等技术，大幅降低发动机运转噪音和振动，提升驾乘舒适性。
## 传动系统
-   **变速箱类型：**
    -   **6挡手动变速箱 (6MT)：** （主力车型）齿比绵密，换挡清晰顺畅，兼顾动力输出与燃油经济性。
    -   *部分高配或特定版本可选配自动变速箱。*
-   **驱动方式：** **前置后驱 (FR)**。这种布局有利于载重时获得更好的抓地力和爬坡能力。
## 性能表现与油耗
-   **最高车速：** ≥120 km/h
-   **综合油耗 (NEDC/WLTC)：**
    -   得益于2.0T柴油机的高效特性及轻量化车身设计，百公里综合油耗控制在较低水平（具体数值因配置和工况略有差异）。
    -   官方宣称其燃油经济性在同级车型中具有明显优势。
-   **排放标准：** 国六b
## NVH与驾驶感受
-   NVH（噪声、振动与声振粗糙度）经过专项优化。怠速及中低速行驶时车内静谧性良好。
-   330N·m的大扭矩在1600转即可爆发，使得车辆在起步、加速超车以及满载爬坡时动力响应迅速，“低扭强劲”是其核心驾驶感受。
---
### *总结*
图雅诺大麦的动力系统以“高效、可靠、低扭强”为特点。2.0T柴油机提供了充足的扭矩储备（330N·m），配合手动变速箱和前驱布局（*注：应为后驱*），能够很好地满足物流配送、客运通勤等场景对动力和承载的需求。

# 鑫源E3 驾驶注意事项
  
更新时间：2026年07月10日22时24分32秒   

## 一、启动与预热
1. **启动前检查**：确认挡位处于 P 挡或 N 挡，踩下制动踏板后再启动车辆。
2. **预热**：冬季低温环境下，建议启动后怠速运转 1-2 分钟，待电池温度上升后再行驶。
3. **仪表盘确认**：启动后检查仪表盘是否有故障灯亮起，确认电量充足。
## 二、行驶中注意事项
1. **平稳驾驶**：
   - 避免急加速、急刹车，有助于延长续航里程和电池寿命。
   - 建议保持匀速行驶，经济时速约为 40-60 km/h。
2. **能量回收**：
   - 合理利用动能回收系统（E挡或能量回收模式），在下坡或减速时可回收电能。
   - 能量回收强度可根据路况调节，湿滑路面建议使用低回收强度。
3. **续航管理**：
   - 剩余电量低于 20% 时建议及时充电，避免过度放电损伤电池。
   - 长途出行前规划好充电站点，预留余量。
4. **载重控制**：
   - E3为物流/商用车型，注意不超过核定载重质量。
   - 货物均匀分布，避免重心偏移影响操控稳定性。
## 三、充电注意事项
1. **充电设备**：使用符合国标的交流或直流充电桩，避免使用劣质充电设备。
2. **充电时机**：
   - 建议电量在20%-80%之间进行日常充电，有助于延长电池寿命。
   - 尽量避免将电池完全耗尽再充电。
3. **充电环境**：
   - 避免在高温暴晒或极寒环境下立即充电。
   - 雨天充电时确保充电接口干燥无积水。
4. **快充与慢充**：
   - E3支持快充和慢充两种方式。日常使用优先选择慢充（交流），对电池更友好。
   - 应急时可使用快充（直流），但不宜频繁使用。
## 四、安全驾驶要点
1. **视野盲区**：E3车身较高且方正，注意A柱盲区和后方视野盲区，变道时多观察后视镜。
2. **转弯与倒车**：
   - E3轴距较长（约2700mm以上），转弯半径较大，窄路掉头需注意空间。
   -倒车时建议配合倒车影像和雷达提示音。
3. **制动距离**：电动车自重较大（含电池组），满载时制动距离会延长，保持安全跟车距离。
4. **涉水与恶劣天气**：
   - E3电池组位于底盘下方，涉水深度不宜超过轮毂中心线（约30cm）。
   -雨雪天气路面湿滑时减速慢行，关闭能量强回收模式以防打滑。
## 五、日常维护与保养
1. **轮胎检查**：定期检查胎压（建议冷态胎压2.5-2.8bar），注意轮胎磨损情况。
2. **冷却液与玻璃水**：定期检查电机冷却液液位和玻璃水余量。
3. **空调系统**：定期清洁空调滤芯（每6个月或10000公里更换一次）。
4. **高压系统警示**：非专业人员严禁私自拆卸高压部件（橙色线束区域）。

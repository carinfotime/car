# 瑞虎3x 保养指南
  
更新时间：2026年07月10日22时01分04秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 首次保养 | 5000公里或6个月（以先到为准） |
| 常规保养 | 每5000公里或6个月 |
| 中期保养 | 每20000公里 |
| 大保养 | 每40000公里 |
---
## 二、各阶段详细保养内容
### 🔧 首次保养（5000km / 6个月）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查灯光系统是否正常
- [ ] 检查雨刮器及喷水功能
- [ ] **全车免费检测**
### 🔧 常规小保养（每5000km / 6个月）
- [ ] **更换机油**（推荐5W-30或10W-40，视季节而定）
- [ ] **更换机油滤清器**
- [ ] **更换空气滤清器**（每10000km或1年）
- [ ] **更换空调滤芯**（每10000km或1年，建议入夏前更换）
- [ ] **检查/补充**：
    -  冷却液
    -  制动液
    -  玻璃清洗液
    -  轮胎气压及胎面磨损
### 🔧 ⚠️中期保养（20000km / 2年）
在常规小保养基础上增加：
#### 🚗燃油与进气系统
- [x] **更换汽油滤清器**
- [x] **清洗节气门**
- [x] **清洗喷油嘴**（视车况而定）
#### 🛞底盘与传动系统  
- [x] **检查刹车片厚度**（前刹车片通常3~5万公里更换）
- [x] **检查刹车盘磨损**
- [x] **轮胎换位**（交叉换位，延长轮胎寿命）
#### ⚡电气系统  
- [x] **检查火花塞**（建议30000km左右考虑更换）
- [x] **检查蓄电池状态**
### 🔧 ⚠️⚠️大保养（40000km / 4年）
在中期保养基础上增加：
#### 🛢️全车油液更换  
|项目 |说明 |
|---|---|
| ✅ **变速箱油** | MT：约2L / CVT：约4~5L |
| ✅ **制动液** | DOT4级别，建议2年/40000km换一次 |
| ✅ **转向助力油** | EPS车型无需更换，液压助力需换 |
| ✅ **防冻液** | -35℃规格，建议2~3年/40000km换一次 |
#### 🔩关键部件  
> ⚠️以下项目需重点检查：
> - ✅正时皮带/链条：瑞虎3x为链条式，一般免维护，但需定期检查张紧度  
> - ✅发电机皮带：查看有无裂纹、老化  
> - ✅水泵：结合正时系统一并检查  
---
## ❄️季节专项维护
### 🌞夏季前准备（每年5月前后）  
1. ☑️清洗冷凝器散热片  
2. ☑️补充/更换空调冷媒  
3. ☑️测试空调制冷效果  
4. ☑️雨刮片老化则立即更换  
### ❄️冬季前准备（每年11月前后）  
1. ☑️检测防冻液冰点是否达标  
2. ☑️蓄电池电量检测与充电  
3. ☑️玻璃水换成冬季专用型(-25℃以下)

## ⚠️常见注意事项
1. ❗首保必须按时进行，超期将失去免费首保资格。
2. ❗瑞虎3x CVT版本严禁使用手动变速箱油。
3. ❗不要混用不同品牌/型号的防冻液。
4. ❗切勿用水代替冷却液或玻璃水。
5. ❗轮胎磨损至标记线时必须立即更换。
6. ❗每次加注机油量约为4L左右，以机油尺为准。

# 丰田卡罗拉锐放 操作指南
  
更新时间：2026年07月15日08时16分11秒   

---
## 一、启动与熄火
### 智能钥匙启动
1. 携带钥匙进入车内，踩下制动踏板
2. 按下 **ENGINE START STOP**（一键启动按钮）
3. 仪表盘指示灯自检完毕后，挂挡行驶
### 机械钥匙启动（如配备）
1. 将钥匙插入点火开关
2. 拧至 **ON** 位置，等待仪表盘自检
3. 拧至 **START** 位置启动发动机
### 熄火步骤
1. 车辆完全停稳，挂入 **P 挡**
2. 拉起电子手刹（如有）或机械手刹
3. 按下 **ENGINE START STOP** 熄火
---
## 二、挡位操作
| 挡位 | 名称 | 用途 |
|------|------|------|
| P | 驻车挡 | 停车时使用，需车辆完全停稳 |
| R | 倒车挡 | 倒车时使用 |
| N | 空挡 | 短暂停车、拖车时使用 |
| D | 前进挡 | 正常行驶 |
| M / S | 手动模式 | 手动换挡，适合上下坡、超车 |
> **注意**：行驶中请勿挂入 P 或 R 挡；挂挡前务必踩死制动踏板。
---
## 三、方向盘与仪表盘
### 方向盘调节
- 拉下方向盘下方的锁止杆
- 调整方向盘高度和前后距离
- 推回锁止杆固定
### 仪表盘指示灯
| 图标 | 含义 |
|------|------|
| 红色电池灯 | 充电系统故障 |
| 红色机油灯 | 机油压力低 |
| 黄色发动机灯 | 发动机系统异常 |
| 黄色轮胎灯 | 胎压异常 |
| 绿色/蓝色大灯 | 远光灯开启 |
---
## 四、灯光操作
### 灯光拨杆（方向盘左侧）
- **OFF**：关闭
- **AUTO**：自动大灯（推荐日常使用）
- **示宽灯**：位置灯
- **近光灯**：手动开启近光
- **远光灯**：向前推拨杆开启，向后拉闪灯
### 前后雾灯
- 旋转拨杆中间环或按压按钮开启雾灯（需先开启近光灯）
---
## 五、雨刮器操作
### 右侧拨杆
- **MIST**：单次刮刷
- **INT / AUTO**：间歇/自动雨刮（可调节灵敏度）
- **LO**：低速连续刮
- **HI**：高速连续刮
- **向后拉**：喷水清洗前挡风玻璃
---
## 六、空调与除雾
### 自动空调
1. 按 **AUTO** 键，设定目标温度
2. 调节风量（风扇图标）
3. **A/C**：制冷/除湿开关
4. **FRONT**：前挡风玻璃除雾（自动开启空调和最大风量）
5. **REAR**：后窗/外后视镜加热除雾
### 手动空调
- 旋转温度旋钮调节冷热
- 旋转风量旋钮调节风速
- 按下 **A/C** 制冷
- 旋转出风模式选择吹面/吹脚/除雾
---
## 七、多媒体与手机连接
### 蓝牙连接
1. 在中控屏点击 **设置 → 蓝牙**
2. 开启手机蓝牙，搜索设备
3. 选择车辆名称，配对确认
4. 连接成功后可使用电话、音乐功能
### Apple CarPlay / Android Auto
- 使用原装 USB 数据线连接手机
- 中控屏自动识别并投射手机界面
- 支持导航、音乐、电话等应用
---
## 八、驾驶辅助功能
### Toyota Safety Sense 智行安全
| 功能 | 说明 |
|------|------|
| PCS 预碰撞安全 | 检测前方车辆/行人，主动预警或制动 |
| LDA 车道偏离警示 | 车辆偏离车道时发出警报 |
| LTA 车道循迹辅助 | 主动辅助保持车道中央行驶 |
| AHB 自动远光灯 | 根据对向车辆自动切换远近光 |
| DRCC 动态雷达巡航 | 设定车速与前车保持距离自动跟车 |
### 巡航控制操作
1. 按下 **ON/OFF** 或 **CRUISE** 按钮
2. 达到目标车速后按 **SET/-** 设定
3. 按 **RES/+** 恢复或加速，**SET/-** 减速
4. 踩刹车或按 **CANCEL** 取消巡航
---
## 九、驻车与泊车辅助
### 倒车影像
- 挂入 R 挡自动显示后方影像
- 参考屏幕上的引导线辅助泊车
### 前后雷达（如配备）
- 接近障碍物时发出蜂鸣声
- 蜂鸣频率越快，距离越近
### 电子手刹 / Auto Hold
- **电子手刹**：拉起驻车，按下解除
- **Auto Hold**：开启后，停车时自动保持制动，踩油门自动解除
---
## 十、日常检查与保养提示
### 定期检查项目
| 项目 | 频率 |
|------|------|
| 机油液位 | 每月或长途前 |
| 冷却液液位 | 每月 |
| 制动液液位 | 每月 |
| 轮胎气压 | 每月 |
| 轮胎磨损 | 每 5000 km |
| 雨刮片 | 每 6 个月 |
### 保养提醒
- 仪表盘出现 **MAINT REQD** 或扳手图标，表示需进行保养
- 建议每 10000 km 或 6 个月（以先到为准）进店保养
- 每次保养后由服务人员重置保养提醒
---
## 十一、应急处理
### 轮胎漏气 / 爆胎
1. 稳住方向盘，缓慢减速，避免急刹
2. 停至安全平坦区域，开启双闪
3. 放置三角警示牌（车后 50-100 米）
4. 使用备胎或补胎工具更换/修补
### 发动机无法启动
- 检查是否挂入 P 挡并踩下制动踏板
- 检查智能钥匙电池是否耗尽（将钥匙贴近启动按钮尝试）
- 检查蓄电池是否亏电（搭电启动）
### 仪表盘报警灯亮起
- 红色报警灯：立即停车，联系救援
- 黄色报警灯：尽快前往维修店检查
---
## 十二、省油驾驶技巧
1. 平稳加速，避免急踩油门
2. 提前预判路况，减少急刹车
3. 高速行驶时使用巡航控制
4. 减少不必要的怠速
5. 保持合理胎压（参考驾驶侧门框标签）
6. 定期清理车内不必要的重物

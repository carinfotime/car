# 五菱宏光PLUS 内饰参数介绍
  
更新时间：2026年07月11日04时10分54秒   

## 一、内饰整体风格
五菱宏光PLUS内饰采用**简约实用**的设计风格，整体布局规整，注重功能性与空间利用。提供深色内饰配色方案，兼顾家用与商用需求。
## 二、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 织物座椅（高配车型可选皮/织物混搭） |
| 座椅布局 | 2+2+3七座布局 / 2+3+3八座布局（可选） |
| 前排座椅调节 | 主驾驶：前后+靠背手动调节 |
| 第二排座椅 | 独立座椅，支持前后滑动及靠背角度调节 |
| 第三排座椅 | 整体式座椅，支持折叠翻转 |
## 三、仪表台与中控
- **仪表盘**：双圆筒机械仪表 + 中央单色行车电脑显示屏
- **中控屏**：8英寸触控液晶屏（高配车型）
- **中控台材质**：硬质塑料，部分区域采用仿木纹或银色饰板装饰
- **方向盘**：三幅式多功能方向盘（高配），支持多媒体控制
## 四、多媒体系统
| 功能 | 说明 |
|------|------|
| 中控屏尺寸 | 8英寸（高配） |
| 手机互联 | 支持手机映射/互联 |
| 蓝牙 | 蓝牙电话/音乐 |
| USB接口 | 前排1个USB接口 |
| 扬声器数量 | 2-4个 |
## 五、空调系统
- **空调类型**：手动空调
- **后排空调**：独立后排空调出风口（第二排顶部出风）
- **空调控制**：旋钮式物理按键控制
## 六、储物空间
| 位置 | 储物设计 |
|------|----------|
| 前排 | 手套箱、门板储物格、中控台杯架、中央通道储物槽 |
| 第二排 | 门板储物格、座椅旁储物袋 |
| 第三排 | 侧壁杯架、储物槽 |
| 后备箱 | 常规纵深约440mm，第三排折叠后可扩展至约1200mm |
## 七、其他内饰配置
| 配置项 | 参数 |
|--------|------|
| 车窗 | 前门电动窗（低配）/ 四门电动窗（高配） |
| 内后视镜 | 手动防眩目 |
| 遮阳板 | 主副驾遮阳板（副驾带化妆镜） |
| 顶棚材质 | 织物顶棚 |
| 地板 | 平直地板设计（便于货物装载） |
| 车内照明 | 前排阅读灯、后排顶灯 |
## 八、尺寸参数
| 项目 | 数值 |
|------|------|
| 轴距 | 2800mm |
| 车内高度 | 约1250mm |
| 后备箱常规容积 | 约410L |
| 最大扩展容积 | 约2000L+（第三排折叠后） |

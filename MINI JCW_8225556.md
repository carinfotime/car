# MINI JCW 内饰参数介绍
  
更新时间：2026年07月12日10时16分22秒   

## 座椅
| 项目 | 参数 |
|------|------|
| 座椅材质 | Dinamica/真皮组合（Nappa皮革可选） |
| 前排座椅 | 运动型座椅（John Cooper Works专属设计） |
| 座椅调节 | 电动12向调节（含腰托） |
| 座椅加热 | 前排座椅加热功能（标配） |
| 座椅记忆 | 驾驶位座椅记忆功能 |
| 后排座椅 | 比例放倒（50:50） |
## 仪表盘与显示屏
| 项目 | 参数 |
|------|------|
| 仪表盘 | 数字式仪表盘（5英寸LCD显示屏） |
| 中控屏 | 圆形OLED触控屏（8.8英寸或9.4英寸，视配置） |
| 抬头显示 | MINI平视显示系统（可选） |
| 车机系统 | MINI Operating System 9 |
## 方向盘
| 项目 | 参数 |
|------|------|
| 方向盘材质 | Nappa真皮运动方向盘 |
| 方向盘功能 | 多功能按键、换挡拨片 |
| 方向盘调节 | 手动上下+前后调节 |
| 专属标识 | JCW专属徽标、红色缝线 |
## 内饰氛围与装饰
| 项目 | 参数 |
|------|------|
| 内饰氛围灯 | 多色LED氛围灯（可调色） |
| 内饰面板 | JCW专属碳纤维/钢琴黑内饰条 |
| 踏板组 | 不锈钢运动踏板（JCW标识） |
| 迎宾踏板 | 发光迎宾踏板（John Cooper Works字样） |
| 顶棚 | 黑色顶棚（运动风格） |
| 缝线 | 红色/黑色对比缝线 |
## 空调与舒适配置
| 项目 | 参数 |
|------|------|
| 空调系统 | 双区自动恒温空调 |
| 空气净化 | 微尘过滤器+空气质量传感器 |
| 后排空调出风口 | 标配 |
| 储物空间 | 前排中央扶手箱、门板储物格、杯架 |
## 音响系统
| 项目 | 参数 |
|------|------|
| 基础音响 | 6扬声器高保真音响系统 |
| 选装音响 | Harman Kardon高端音响（12扬声器） |
| 无线连接 | Apple CarPlay / Android Auto |
## 安全与辅助配置
| 项目 | 参数 |
|------|------|
| 安全气囊 | 前排双气囊+侧气囊+头部气帘 |
| 驾驶辅助 | 主动巡航控制、碰撞预警、自动刹车 |
| 泊车辅助 | 后倒车雷达+倒车影像（选装360°全景） |
## 其他内饰参数
| 项目 | 参数 |
|------|------|
| 车内高度 | 约 1260 mm |
| 后排腿部空间 | 约 650 mm |
| 后备厢容积 | 211 L（后排放倒后 731 L） |
| 内饰配色方案 | 黑色为主，可选红/黑双色内饰 |
---
> **注：** 以上参数基于最新款 MINI JCW（含三门版/五门版/敞篷版），具体配置因市场与选装包不同可能有所差异。

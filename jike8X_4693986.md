# 极氪8X 驾驶注意事项
  
更新时间：2026年07月19日08时30分57秒   

## 一、提车与初始设置
1.  **仔细验车**：提车时务必检查外观漆面、内饰细节、玻璃生产日期、轮胎磨损及胎压。核对车辆识别代号（VIN码）与文件是否一致。
2.  **激活账户**：完成车主账户激活，绑定手机App，熟悉远程控制功能（如提前开启空调、查看充电状态）。
3.  **熟悉界面**：花时间熟悉中控大屏的菜单逻辑，特别是驾驶模式选择、能量回收调节以及辅助驾驶功能的开关位置。
4.  **设置驾驶偏好**：根据个人习惯调整方向盘手感（舒适/标准/运动）、动能回收强度（低/标准/强）以及踏板响应。
## 二、日常驾驶核心要点
1.  **动能回收适应**：
    -   初次驾驶时建议将动能回收设置为“低”或“标准”，待完全适应后再尝试“强”模式。
    -   强回收模式下松开电门会有明显减速感，类似单踏板逻辑，需提前预判路况，减少刹车踏板使用频率。
2.  **电门与刹车**：
    -   电动车扭矩输出直接且迅猛，起步和超车时轻踩电门即可获得强劲加速，避免深踩导致窜动。
    -   保持平稳的加速和刹车习惯，不仅提升乘坐舒适性，也能有效降低能耗。
3.  **能量管理**：
    -   **高速行驶**：时速超过100km/h后能耗会显著增加。建议高速巡航使用自适应巡航或智能领航辅助，保持匀速更省电。
    -   **空调使用**：夏季高温时建议先远程通风再开启空调；冬季可多用座椅加热和方向盘加热代替部分暖风以节省电量。
4.  **底盘与通过性**：
    -   （若配备空气悬架）通过颠簸路面或减速带前可适当升高底盘；高速行驶时自动降低以提升稳定性并降低风阻。
    -   SUV重心较高，过弯时注意控制车速，避免急打方向。
## 三、辅助驾驶使用规范
1.  **切勿过度依赖**：LCC/ACC等辅助功能仅为辅助工具，驾驶员必须始终保持双手握住方向盘并时刻关注路况。
2.  **适用场景**：在车道线清晰的高速公路或城市快速路上使用效果最佳。避免在雨雪天气、施工路段或弯道过急的道路上开启。
3.  **接管时机**：系统发出接管请求（如警示音或方向盘振动）时请立即接管车辆。注意识别静止障碍物（如故障车辆、锥桶），系统可能无法及时识别。
## 四、充电与电池保养
1.  **最佳充电区间**：日常使用建议将电量保持在20%-80%之间。长途出行前可充至90%-100%，但到达后尽快消耗至80%以下。
2.  **快慢充结合**：
    -   （若支持800V架构）极充站补能速度极快（10%-80%约15-30分钟），适合长途应急。
    -   （若为400V架构或非极充站）日常通勤建议优先使用家充桩慢充（7kW/11kW），有利于延长电池寿命。
3.  **低温环境注意**：冬季用车后建议立即充电（电池温度较高时充电效率更高）。长期停放时应保持电量在50%-60%左右。
## 五、安全与应急处理
1.  **涉水行驶**：电动车涉水能力优于燃油车但仍有风险。遇到积水路段应减速慢行确认水深不超过半个车轮高度；通过后轻踩刹车排出水分恢复制动性能；切勿强行冲入深水区以防电池包受损引发短路甚至火灾风险——一旦发现车辆进水应立即断电撤离并联系专业救援处理严禁自行启动车辆
2.  **紧急断电流程**:若发生碰撞或电路异常请立即按下位于主驾位附近的紧急断电按钮(通常有红色标识)切断高压系统确保救援安全
3.  随车工具检查:定期确认随车配备的三角警示牌反光背心千斤顶拖车钩及便携式灭火器是否齐全有效
4.  爆胎应对:紧握方向盘稳住方向缓慢减速切勿急刹急转向平稳停靠至安全区域后开启双闪放置警示牌联系道路救援更换备胎(如有)或等待拖车
## ⚠️特别警示:涉水安全红线
-   【绝对禁止】严禁在积水深度超过车轮中心(约半个车轮高度)的区域行驶
-   【绝对禁止】一旦发现车内进水仪表盘报警或闻到焦糊味必须立即停车熄火(下电)并迅速离开车辆至安全地带
-   【绝对禁止】浸水后的电动车严禁自行通电启动必须由专业技术人员检查高压系统绝缘值合格后方可使用否则存在致命触电及火灾风险

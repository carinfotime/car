# 腾势Z9 保养指南
  
更新时间：2026年07月20日13时39分36秒   

## 一、概述
腾势Z9作为一款高端新能源车型（插电式混合动力/纯电动），其保养核心在于**三电系统（电池、电机、电控）**、增程器（如有）以及常规底盘与车身部件的维护。正确的保养不仅能延长车辆寿命，还能保障行车安全与性能。
---
## 二、保养周期总览
| 项目 | 周期 | 备注 |
| :--- | :--- | :--- |
| **小保养** | 每 **12个月** 或 **20,000公里** | 以先到为准 |
| **大保养** | 每 **24个月** 或 **40,000公里** | 包含小保养项目 |
| **空调滤芯** | 每 **12个月** / **20,000公里** | 空气质量差地区可缩短至6个月 |
| **制动液** | 每 **24个月** / **40,000公里** | - |
| **减速器油（齿轮油）** | 每 **48个月** / **80,000公里** | - |
| **冷却液（三电系统）** | 首次更换：60,000公里；之后每40,000公里/2年 | - |
| **增程器机油及机滤（DM版）** | 每12个月或总里程达到HEV模式10,000公里（以先到为准） |
> *注：以上为官方建议标准周期。若长期在恶劣环境（高温、多尘、潮湿）或频繁激烈驾驶，建议适当缩短保养间隔。*

## 三、核心保养项目详解
### （一）三电系统检查（每次必查）
-   **高压电池包**
    -   检查电池外观有无磕碰、变形或漏液。
    -   使用诊断仪读取电池健康度（SOH）、单体电压均衡性。
    -   检查电池散热风道/液冷管路是否通畅。
-   **驱动电机**
    -   检查电机运行无异响，绝缘性能正常。
-   **电控单元**
    -   检查高压线束连接是否牢固，无氧化腐蚀。
    -   升级最新BMS（电池管理系统）及VCU（整车控制器）软件。
### （二）增程器系统保养 (DM-i/DM-p版本)
-   **更换机油与机滤**
    -   使用符合ACEA C5或API SP等级的低粘度全合成机油（推荐0W-20）。
-   **火花塞**
    -   建议每40,000公里更换一次。
-   **燃油系统**
    -   定期添加原厂燃油添加剂，防止喷油嘴积碳。
### （三）底盘与制动系统
-   **制动片与制动盘**
    -   Z9车重较大，前刹车片磨损较快。建议每次小保养时测量厚度。
-   **制动液**
    -   DOT4级别。含水量超过3%时必须立即更换。
-   **轮胎**
    -   Z9标配高性能轮胎，胎压建议保持在2.5~2.8 bar。
        > *注：具体胎压值请以驾驶位B柱标签为准。*
        > *注：Z9配备后轮转向功能，四轮定位参数较特殊，换胎后必须去具备后轮转向校准能力的专业店做定位。*
-   **悬架与转向**
    -   Z9搭载云辇-A智能空气车身控制系统：
        > *注意：云辇系统属于精密液压+电气部件，非授权店严禁拆卸维修！*
        > *注意：定期检查空气减震器气包有无裂纹漏气；检查高度传感器连杆是否松动。*
### （四）空调与冷却系统
-   **空调滤芯**
    1. Z9的空调滤芯位于副驾驶手套箱下方或前舱防火墙处。
    2. DIY更换步骤：
        
        1.打开手套箱并拆下阻尼杆；
        2.按压两侧卡扣放下手套箱；
        3.取出旧滤芯；
        4.按箭头方向装入新滤芯；
        5.复位手套箱及阻尼杆。
        
    3. *提示：建议选购带活性炭的PM2.5高效滤芯。*
-   **冷却液**
    1. Z9拥有两套独立的冷却循环：
        
        a) 【电机/电控冷却】使用专用电子水泵冷却液；
        b) 【电池包冷却】通过冷媒直冷或水冷板循环；
        c) 【增程器冷却】与传统发动机相同的水冷循环(DM版)。
        
    2. *警告：严禁混用不同颜色的冷却液！必须使用腾势原厂指定的专用冷却液！*

## 五、常见问题解答 (FAQ)
### Q1: Z9纯电版需要换变速箱油吗？
A: Z9纯电版采用单速减速器结构而非传统变速箱。但减速器内部仍有齿轮和轴承需要润滑——通常称为“减速器油”或“齿轮油”，需按周期更换。
### Q2: Z9的刹车为什么感觉偏软？
A: Z9配备博世IPB智能集成制动系统——CRBS协调再生制动策略优先使用电机反拖回收能量。当动能回收介入时机械刹车介入较少是正常现象；但在急刹情况下机械刹车会全力介入。
### Q3: OTA升级算不算保养？
A: OTA升级属于软件维护范畴——虽不强制但强烈建议及时更新以获得更好的续航逻辑和智驾体验。

## ⚠️重要安全警告
1️⃣ ⚡️高压安全——非专业人员严禁触碰橙色高压线束！维修前必须断电并等待15分钟以上待电容放电完毕！
2️⃣ 🔥火灾风险——严禁私自改装电池包结构或在非授权店加装底盘护板以免影响电池散热！
3️⃣ 🛞举升安全——Z9底盘有大量高压线束和云辇管路分布——普通举升机垫块位置不当可能压坏管路！务必使用腾势专用举升垫块！
4️⃣ 📜质保条款——未在授权服务店进行定期保养可能导致三电终身质保失效！

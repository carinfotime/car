# 九龙A4 空间参数介绍
  
更新时间：2026年07月17日12时27分27秒   

## 车身尺寸
| 参数项 | 数据 |
|--------|------|
| 车长（mm） | 5380 |
| 车宽（mm） | 1880 |
| 车高（mm） | 2285 |
| 轴距（mm） | 3110 |
| 车身结构 | 轻客（轻型客车） |
| 座位数 | 10-15座 |
## 内部空间
- **乘坐空间**：采用高顶设计，车内净高充裕，乘客可在车内站立，头部空间表现优秀
- **座椅布局**：提供多种座椅组合方案，包括 10座、11座、14座、15座 等布局形式
- **座椅间距**：前后排座椅间距合理，腿部空间充足，适合中长途出行
- **通道宽度**：中央通道宽敞，方便乘客在车内走动
## 载物空间
| 项目 | 说明 |
|------|------|
| 后备厢容积 | 标准状态下约 0.8-1.2 m³ |
| 后排座椅折叠后 | 可扩展至约 3.5 m³ 以上 |
| 车内储物格 | 前排中控台、门板等多处储物设计 |
| 行李架 | 可选配车顶行李架，进一步扩展储物能力 |
## 车门与上下车
- **乘客门**：右侧配备手动或电动侧滑门，开口宽度约 700mm，上下车方便
- **尾门**：对开式尾门设计，开口角度大，便于装卸行李
- **踏步高度**：一级踏步高度适中，配合扶手设计，老人儿童上下车较为便利
## 空间优势总结
> 九龙A4 作为一款轻型客车，在空间方面兼具 **载人** 与 **载物** 的双重实用性。长轴距 + 高车顶的设计确保了宽敞的乘坐空间，灵活的座椅布局使其适合通勤、商务接待、旅游包车等多种使用场景。

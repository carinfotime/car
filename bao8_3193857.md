# 豹8 保养指南
  
更新时间：2026年07月12日15时29分23秒   

## 一、保养周期总览
| 保养项目 | 周期 |
|---------|------|
| 首次保养 | 购车后 **3个月或5,000公里**（以先到为准） |
| 常规保养 | 每 **12个月或10,000公里**（以先到为准） |
| 大保养 | 每 **24个月或20,000公里**（以先到为准） |
---
## 二、各阶段保养内容
### 🔹 首次保养（3个月 / 5,000公里）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查灯光系统功能
- [ ] 检查雨刮器及喷水嘴
- [ ] 全车电脑系统检测与复位
### 🔹 A类常规保养（12个月 / 10,000公里）
包含首次保养全部项目，另加：
- [ ] 更换空气滤清器滤芯
- [ ] 更换空调滤清器滤芯
- [ ] 检查火花塞状况
- [ ] 检查蓄电池状态及接线柱清洁度
- [ ] 检查底盘螺栓紧固情况
- [ ] **轮胎换位**（前后交叉对调）
- [ ] **四轮定位检测**（如发现跑偏则需调整）
### 🔹 B类大保养（24个月 /20,000公里）
包含A类全部项目，另加：
- [x] **更换火花塞**
- [x] **更换制动液**
- [x] **更换冷却液**
- [x] **更换变速箱油**（自动变速箱）
- [x] **更换分动箱油**（四驱车型）
- [x] **更换前后桥齿轮油**
- [x] **清洗节气门**
- [x] **清洗喷油嘴**
- [x] **检查并清洁EGR阀**
- [x] **燃油滤清器更换**

##  三、易损件更换建议
|  部件 |  建议更换周期 |
|--------|--------------|
| 🛞轮胎 |  6万~8万公里或磨损至极限标记 |
| 🧽刹车片 |  前轮约3~5万公里，后轮约5~7万公里 |
| 🪞雨刮片 |  每6~12个月或刮不干净时 |
| 🔋蓄电池 |  3~4年或出现启动困难时 |
| 💡灯泡/LED模组 |  损坏即换 |
---
## ⚠️四、日常自检清单（建议每月一次）
1. ✅ **机油液位** —冷车状态下查看标尺，应在上下刻度之间。
2. ✅ **冷却液** —副水箱液面在MIN与MAX之间。
3. ✅ **制动液** —储液罐刻度不低于MIN线。
4. ✅ **轮胎气压** —冷态标准值参见驾驶侧B柱标签。
5. ✅ **灯光系统** —大灯、转向灯、刹车灯、雾灯是否正常。
6. ✅ **仪表盘警示灯** —启动自检后无异常常亮。
7. ✅ **雨刮水壶** —及时补充专用玻璃水。
---
## 🛠️五、注意事项
1. 🚫禁止混用不同品牌/型号的机油和冷却液。
2. 🚫切勿用水代替冷却液，冬季需使用防冻型冷却液。
3. ⚡豹8为混动/新能源车型，高压系统维护必须由授权服务站操作，严禁私自拆卸高压部件。
4. 📌每次保养后保留好工单和记录，质保期内须在授权服务站完成保养以免影响质保权益。
5. 🌡️极端气候地区（高温/严寒/多尘）应适当缩短保养间隔。

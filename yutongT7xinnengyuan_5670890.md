# 宇通T7新能源 驾驶注意事项
  
更新时间：2026年07月17日09时58分38秒   

## 一、启动与上电
1. **上电前检查**：确认高压线束、充电接口无破损、松动或异物。
2. **钥匙操作**：将钥匙拧至“ON”档，等待仪表盘自检完成（约3-5秒），确认无故障报警后再挂挡行驶。
3. **低压上电**：先接通低压蓄电池电源，再启动高压系统，严禁直接操作高压开关。
4. **紧急断电**：熟悉紧急断电开关位置（通常位于驾驶员左侧或仪表台下方），遇异常情况立即切断。
## 二、行驶中注意事项
1. **平稳驾驶**：避免急加速、急减速，建议匀速行驶以延长续航里程。
2. **能量回收**：合理利用制动能量回收系统（E-REC），下坡或减速时可提前松油门，提高电能利用率。
3. **车速控制**：
   - 城市道路建议时速 ≤ 60 km/h
   - 高速公路建议时速 ≤ 100 km/h（具体以车辆标定为准）
4. **涉水限制**：
   - 积水深度不得超过电池包底部高度（通常 ≤ 30 cm）
   - 涉水后应低速驶出并检查仪表是否有绝缘报警
5. **空调使用**：空调系统由动力电池供电，长时间使用会显著降低续航，建议合理设定温度（24-26℃为宜）。
## 三、充电操作
1. **充电前检查**：
   - 确认充电枪及线缆无破损、进水
   - 确认车辆处于熄火状态且高压已断开
2. **充电流程**：
   - 先插充电枪 → 再启动充电桩
   - 结束充电时：先停止充电桩 → 再拔充电枪
3. **禁止事项**：
   - ⚠️严禁在车辆行驶中或上高压状态下插拔充电枪
   - ⚠️严禁使用破损、老化或非标充电设备
4. **环境要求**：
   - 避免在雷雨天气露天充电
   - 避免在高温暴晒后立即快充（建议冷却30分钟以上）
## 四、电池与热管理
1. **温度监控**：关注仪表盘电池温度指示，正常工作温度范围为 -20℃ ~ +55℃。
2. **高温预警**：
   - 电池温度 ≥ 55℃时，车辆会自动限功率运行
   - ≥60℃时可能触发强制断电保护
3. **低温注意**：
   - ≤0℃环境下续航会明显下降（约减少20%-40%）
   - ≤-20℃时应先预热电池再行驶（如有预热功能）
4. **SOC管理建议**：
   - SOC保持在20%-90%之间使用最佳
   - SOC低于10%时应尽快就近充电，避免亏电损伤电池
## 五、安全与应急处理
1. **碰撞事故处理流程**：

发生碰撞 → 
立即熄火并断开高压开关 → 
人员迅速撤离车辆 → 
联系专业人员处理高压系统 → 
切勿自行触碰橙色高压线束

2. **火灾应急措施**：
   ⚠️锂电池起火不可用水扑灭！应使用干粉灭火器或沙土覆盖。
3. **绝缘故障报警处理步骤**：

①立即靠边停车熄火 →
②断开高压开关 →
③联系售后维修 →
④等待专业人员检修前不得再次启动

4. **拖车注意事项**
   必须采用平板拖车方式运输，严禁前轮着地拖行（否则可能损坏电机/减速器）。

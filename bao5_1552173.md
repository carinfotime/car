# 豹5 保养指南
  
更新时间：2026年07月12日09时14分38秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 首次保养 | **购车后3个月或行驶3,000公里**（以先到为准） |
| 常规保养 | **每12个月或每行驶10,000公里**（以先到为准） |
| 恶劣工况下 | **建议缩短至6个月或5,000公里** |
> ⚠️ **注意**：豹5为插电混动车型，保养需同时关注发动机与三电系统。
---
## 二、各阶段保养内容
### 🔹 首次保养（3个月 / 3,000公里）
- [ ] 更换发动机机油及机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查动力电池高压线束连接情况
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查灯光、喇叭、雨刮系统
- [ ] OBD系统诊断检测
- [ ] 底盘及悬挂系统螺栓紧固检查
### 🔹 A类常规保养（每10,000公里 / 12个月）
- [x] **必须更换**：机油、机滤
- [x] **必须检查**：
    - 空气滤清器（视情况清洁/更换）
    - 空调滤清器（视情况清洁/更换）
    - 制动片厚度及制动盘磨损
    - 轮胎胎压、磨损及换位（建议每10,000公里四轮换位）
    - 底盘螺栓紧固力矩
    - 高压电池冷却系统管路密封性
    - DC-DC转换器工作状态
    - OBD系统故障码读取
### 🔹 B类常规保养（每20,000公里 / 24个月）
在A类基础上增加：
- [x] **更换空气滤清器**
- [x] **更换空调滤清器**
- [x] **检查并补充冷却液**
- [x] **检查驱动电机减速器油液**
- [x] **检查制动液含水量**（视情况更换，建议每2年更换一次）
- [x] **检查高压电池绝缘电阻**
- [x] **全车灯光功能检测**
### 🔹 C类大保养（每40,000公里 / 48个月）
在B类基础上增加：
- [x] **更换火花塞**
- [x] **更换制动液**
- [x] **更换驱动电机减速器油**
- [x] **更换冷却液**
- [x] **清洗节气门**（视积碳情况）
- [x] **清洗喷油嘴**（视积碳情况）
- [x] **四轮定位检测与调整**

## 三、三电系统专项维护
| 项目 | 周期 |  说明 |
|-----|------|------|
| 🟢动力电池均衡充电 |  每月至少1次 |  使用慢充充满至100%，保持涓流充电2小时以上 |
| 🟢高压线束绝缘检测 |  每年1次 |  4S店专用设备检测，防止漏电风险 |
| 🟢电池冷却系统清洗 |  每2年或40,000公里 |  清理散热通道，确保热管理效率 |
| 🟢驱动电机状态监测 |  每次常规保养时 |  读取电机控制器数据流，确认无异常 |
---
## 四、日常自检清单（建议每周执行）

□  外观：车身有无明显损伤、划痕  
□  轮胎：胎压是否正常，表面有无异物扎入  
□  灯光：前后大灯、转向灯、刹车灯是否正常  
□  仪表盘：有无故障报警灯亮起  
□  电量/续航：SOC显示是否与实际一致  
□  刹车：踩下刹车踏板有无异响或行程异常  
□  雨刮水：玻璃水是否充足  
□  充电口：充电接口有无异物或损坏  

---
## ⚡充电注意事项
1. ✅优先使用厂家认证的交流慢充桩进行日常充电。
2. ✅每月至少完成一次满充均衡操作。
3. ❌避免长期将电量保持在20%以下或90%以上存放。
4. ❌避免在极端高温或低温环境下立即快充。
5. ✅长时间停放时，建议保持电量在50%~60%之间。

|    项目       |       标准值/要求       |
|--------------|------------------------|
|    标准胎压     |    前轮2.5 bar /后轮2.6 bar |
|    备胎胎压     |    4.2 bar             |
|    换位周期     |    每10,000公里         |
|    最小花纹深度    |    不低于1.6 mm         |

## ⚠️常见警示信号与应对
|     现象          |     可能原因        |     应对措施           |
|------------------|-------------------|----------------------|
| "EV功能受限"提示灯亮起 |     高压系统故障      |     立即停车，联系售后救援    |
| "请检查发动机"提示灯亮起｜     混动系统异常       ｜尽快前往4S店检修      ｜
｜加速无力且油耗偏高｜节气门/火花塞积碳｜清洗节气门并更换火花塞｜
｜纯电续航明显下降｜电池SOC未校准或老化｜做满充均衡；如无效则检测电池健康度｜

## 📋实用小贴士
1. 📱通过比亚迪官方App可查看车辆健康报告和预约保养。
2. 🧾保留所有保养记录和发票，二手车交易时保值率更高。
3. ❄️冬季用车前建议先HEV模式热车1~2分钟再切换EV模式。
4. 🌧️雨季涉水后应尽快检查高压部件密封性。
5. 🔧非官方改装电气线路可能导致三电质保失效。

# 东风风神皓极 动力参数介绍
  
更新时间：2026年07月14日11时11分46秒   

## 一、动力总成概览
皓极提供 **燃油版** 与 **马赫双擎MHD混动版** 两套动力系统，均基于东风集团自主研发的 **马赫动力** 技术平台打造。
---
## 二、燃油版（1.5T 涡轮增压发动机）
| 项目 | 参数 |
| :--- | :--- |
| **发动机型号** | DFMC15TP1（马赫动力1.5T） |
| **发动机类型** | 直列四缸、涡轮增压、缸内直喷 |
| **排量** | 1.5 L |
| **最大马力** | **204 Ps**（150 kW） |
| **最大功率转速** | 5200 rpm |
| **最大扭矩** | **305 N·m** |
| **最大扭矩转速区间** | 2000 - 4000 rpm |
| **变速箱** | 7速湿式双离合变速箱（7DCT） |
| **最高车速** | ≥190 km/h |
| **WLTC综合油耗** | ≈6.99 L/100km |
> 🔑 *亮点：1.5T排量压榨出204匹马力，扭矩达到305N·m，在同排量发动机中属于第一梯队水平。*
---
## 三、混动版（马赫双擎 MHD）
### 🔹系统构成
-   **发动机：** 马赫动力1.5T四缸混动专用发动机
-   **电机：** P1+P3双电机结构
-   **变速箱：** E-CVT无级变速机构
### 🔹详细参数
#### 📌 发动机部分
| 项目 | 参数 |
| :--- | :--- |
| **发动机最大功率** | ≤130 kW（约177 Ps）*（注：系统综合功率更高）* |
#### 📌 电机部分
| 项目 | P1电机（发电/启动一体机） | P3电机（驱动电机） |
| :--- | :---: | :---: |
| **峰值功率** | — *（主要辅助发电与启停）* | ≥130 kW / ≥177 Ps *（实际驱动主力）* |
#### 📌 系统综合输出
> ⚡ *注：不同调校版本数据略有差异，以下为常见版本数据。*
-   **系统综合最大功率：约180 - >200 kW**
-   **系统综合最大扭矩：约540 - >600 N·m**
    - *实际轮端扭矩表现强劲，起步加速体感优于燃油版。*
#### 📌 E-CVT & EV模式
-   E-CVT支持多种工作模式：
    -   EV纯电模式（低速/拥堵）
    -   串联模式（发动机发电+电机驱动）
    -   并联模式（发动机+电机共同驱动）
    -   能量回收模式
-   WLTC综合油耗：约5.8 L/100km
> 🔑 *亮点：低速用电、高速用油或并联发力，兼顾燃油经济性与动力响应。*

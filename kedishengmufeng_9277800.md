# 克蒂圣姆峰 驾驶注意事项
  
更新时间：2026年07月21日09时47分07秒   

## 一、车辆基础信息确认
- **车身尺寸**：该车型为大型商务MPV，车长通常超过5.3米，宽约2米，高约2米，需充分注意车辆通过性。
- **整备质量**：车辆自重较大，满载时可达3吨以上，制动距离较长。
- **燃油要求**：请使用规定标号（通常为95号及以上）无铅汽油，避免加注劣质燃油。
## 二、出车前检查
1. **轮胎检查**
   - 检查胎压是否处于标准范围（建议冷车胎压2.4-2.6 bar）
   - 检查轮胎磨损情况及有无鼓包、裂纹
2. **油液检查**
   - 机油液位是否在正常刻度内
   - 冷却液、制动液、转向助力液是否充足
   - 玻璃清洗液加满
3. **灯光与信号**
   - 检查前后大灯、转向灯、刹车灯、雾灯是否正常工作
4. **刹车系统**
   - 启动后轻踩刹车，感受踏板行程及制动力是否正常
## 三、驾驶操作要点
### 1. 起步与加速
- 启动后怠速热车30秒至1分钟，待转速回落至正常怠速后再行驶
- 起步轻踩油门，避免急加速，防止后部乘客不适
- 涡轮增压发动机建议避免冷车高转速行驶
### 2. 转向操控
- 转弯半径较大，掉头或窄路转弯时提前预留足够空间
- 注意车身侧倾，过弯速度不宜过快
- 变道时注意后视镜盲区，建议配合盲区监测系统使用
### 3. 制动技巧
- 提前预判路况，采用“早踩、轻踩、长踩”的制动方式
- 避免急刹车，防止车内物品移位及乘客前倾
- 下长坡时建议使用发动机辅助制动（手动模式降挡）
### 4. 倒车与泊车
- 充分利用360°全景影像及前后雷达
- 尽量选择宽敞车位，避免狭窄车位
- 倒车时安排人员下车指挥更安全
## 四、特殊路况驾驶
| 路况类型 | 注意事项 |
|---------|---------|
| **高速行驶** | 建议巡航速度控制在100-110km/h，避免超速；注意横风影响 |
| **山路弯道** | 入弯前减速，弯中保持匀速，出弯缓慢加速 |
| **雨雪天气** | 开启雾灯，增大跟车距离（建议保持正常2倍以上），避免急转向 |
| **颠簸路面** | 减速慢行，保护空气悬挂系统及车内电器设备 |
| **地下车库** | 注意限高标识（标准版约2米，改装版可能更高，需确认实际高度） |
## 五、乘客与载物安全
- 全车乘客务必系好安全带，包括后排及第三排
- 行李箱载物均匀分布，避免重心偏移
- 车顶行李架载物不得超过规定载荷（如有配备）
- 车内勿放置未固定的硬物或重物，急刹时易造成伤害
## 六、电子辅助系统使用
- **自适应巡航（ACC）**：高速路段推荐使用，但需保持手握方向盘
- **车道保持辅助**：开启后变道需提前打转向灯
- **自动驻车**：坡道起步时建议开启，防止溜车
- **驾驶模式选择**：经济模式适合城市代步，运动模式适合超车需求，根据路况合理切换
## 七、停车与熄火
- 熄火前关闭空调、音响等用电设备，保护电瓶
- 涡轮增压车型建议怠速运转30秒后再熄火（长途行驶后）
- 长时间停放建议使用车衣或停入车库，避免内饰老化
- 电子手刹确认拉起后再离开车辆
## 八、定期维护提醒
| 项目 | 建议周期 |
|------|---------|
| 机油更换 | 每5000-10000公里或按保养提示 |
| 空气滤芯 | 每10000公里检查/更换 |
| 刹车片检查 | 每20000公里 |
| 轮胎换位 | 每10000-15000公里 |
| 空调滤芯 | 每10000公里或换季时 |
| 全车检测 | 每5000公里或半年 |

# 仰望U9 技术参数介绍
  
更新时间：2026年07月11日03时50分10秒   

## 一、核心动力系统
| 项目 | 参数 |
| :--- | :--- |
| **动力类型** | 纯电动超级跑车 |
| **电机布局** | 四轮独立电机（易四方技术平台） |
| **电机数量** | 4台（前后轴各2台） |
| **综合最大功率** | 约 **960 kW（1306匹马力）** |
| **综合最大扭矩** | 约 **1680 N·m** |
| **0-100 km/h加速** | **2.36秒**（实测数据，含起步时间） |
| **最高时速** | 实测可达 **309.19 km/h** |
## 二、电池与续航
| 项目 | 参数 |
| :--- | :--- |
| **电池类型** | 磷酸铁锂刀片电池（CTB电池车身一体化技术） |
| **电池容量** | **80 kWh** |
| **CLTC纯电续航里程** | **450 km** |
## 三、底盘与悬架
-   **悬架系统：**
    -   前后均为双横臂式独立悬架
    -   配备云辇-X智能全主动车身控制系统
-   **特色功能：**
    -   *原地掉头*（四电机差动控制）
    -   *应急浮水脱困*（部分极端工况下短时涉水行驶）
    -   *跳舞模式 / 三轮行驶*（云辇-X主动调节实现）
## 四、车身尺寸与重量
-   **长 × 宽 × 高：** `4966 mm × 2029 mm × 1295 mm`
-   **轴距：** `2900 mm`
-   **整备质量：** `2475 kg`
## 五、制动与轮胎
-   **制动系统：**
    -   前轮：碳陶瓷制动盘 + Brembo多活塞卡钳
    -   后轮：碳陶瓷制动盘 + Brembo多活塞卡钳
-   *100 km/h -0制动距离：* < `33米`
-   *轮胎规格：*
    -   前轮：`275/35 R21`
    -   后轮：`325/30 R21`
## 六、核心技术亮点
1.  **[易四方]** ——四个独立电机可实现对每个车轮扭矩的毫秒级独立控制。
2.  **[云辇-X]** ——全主动悬架，通过摄像头预瞄路面，实时调整悬架软硬和高度。
3.  **[CTB]** ——电池车身一体化技术，提升整车刚度和空间利用率。
4.  整车风阻系数优化至超跑水平，配合主动式尾翼提供下压力。

# Lotus Emira 保养指南
  
更新时间：2026年07月16日08时02分01秒   

## 1. 概述
本指南旨在帮助 Lotus Emira 车主了解车辆的常规保养项目、周期及注意事项。遵循正确的保养计划，可以确保车辆性能、延长使用寿命并保持高保值率。
**重要提示：** 本指南仅供参考，具体操作请以随车附带的《用户手册》为准。对于复杂或涉及安全系统的维修，建议前往授权服务中心处理。

## 2. 保养周期总览
Lotus Emira 的保养间隔通常为 **每行驶 12,000 英里（约19,000公里）或每12个月**，以先到者为准。以下是主要项目的建议周期：
| 项目 | 周期 | 备注 |
| :--- | :--- | :--- |
| **发动机机油 & 机滤** | 每12个月 / 12,000英里 | AMG M139发动机需使用指定全合成机油 |
| **空气滤清器** | 每24个月 / 24,000英里 | — |
| **空调滤清器** | 每12个月 / 12,000英里 | — |
| **火花塞** | 每48个月 / 48,000英里 | — |
| **制动液** | 每24个月更换一次 | DOT4等级，赛道驾驶后建议检查含水量 |
| **冷却液** | 每60个月 / 60,000英里首次更换，之后每36个月更换一次 | — |
| **变速箱油 (手动)** | — | *免维护* (视驾驶方式而定) |
| **变速箱油 (自动)** | — | *免维护* (视驾驶方式而定) |
| **燃油滤清器** | — | *集成于油箱内，免维护* |

## 3. DIY（自己动手）基础保养项目
### A.检查机油液位
1.  将车辆停在水平地面。
2.  启动发动机并怠速运行至正常工作温度。
3.  熄火后等待约5分钟。
4.  拔出机油尺并用干净布擦拭干净。
5.  重新插入并再次拔出，检查油位应在上下刻度线之间。
### B.检查轮胎气压与磨损
-   **标准胎压：**
    -   前轮：2.2 bar (32 psi)
    -   后轮：2.4 bar (35 psi)
-   *注意：赛道驾驶时需适当降低胎压（参考用户手册）。*
-   定期检查轮胎花纹深度（法定最低1.6mm），注意内侧磨损情况。
### C.清洁与润滑
-   **车门铰链/锁扣：**每年涂抹少量锂基润滑脂。
-   **敞篷机构（如配备）：**保持导轨清洁并使用专用硅基润滑剂。
-   **中控屏幕/内饰：**使用微纤维布和中性清洁剂擦拭。
---
## 4. AMG M139发动机特殊注意事项
Emira搭载的2.0T涡轮增压发动机源自AMG平台，具有以下特殊要求：
1.  **机油规格：**
    -   必须使用符合 MB229.51 / MB229.52 / MB229.71认证的0W-40或5W-40全合成机油。
    -   推荐容量约6升（含机滤）。
2.  **冷却系统：**
    -   该发动机热负荷较高，务必使用原厂指定长效冷却液（通常为G48或同等品）。
3.  **涡轮增压器养护：**
    -   激烈驾驶后不要立即熄火，怠速运行30秒至1分钟让涡轮降温后再熄火。
4.  使用高品质燃油（建议95号及以上无铅汽油）。

## BONUS: V6版本额外说明
若您拥有搭载丰田2GR-FE V6自然吸气发动机的版本：
-   **机油规格：**5W-30全合成机油（符合API SN/SP标准）。
-   **火花塞更换周期：**更短——建议每36,000英里更换一次。
-   V6版本对点火线圈老化较为敏感，如出现怠速抖动请优先检查点火系统。

## C:赛道日前后检查清单
### ✅出发前
- [ ]确认刹车片厚度 >3mm
- [ ]制动液含水量 <2%
- [ ]轮胎无鼓包、裂纹
- [ ]所有液体液位正常
- [ ]备好扭矩扳手和胎压计
### ✅结束后
1️⃣让车辆怠速冷却10分钟再熄火  
2️⃣检查刹车盘是否有裂纹  
3️⃣测量胎压是否恢复正常值  
4️⃣清洗刹车粉尘及轮胎表面异物  
---
## D:常见故障预警信号
以下情况出现时请尽快进厂检修：
⚠️仪表盘亮起黄色引擎灯 →可能为氧传感器或排放系统问题  
⚠️冷启动异响持续超过3秒 →关注VVT执行器或链条张紧器  
⚠️变速箱换挡顿挫明显 →可能是变速箱油温异常或软件需要更新  
⚠️底盘传来金属敲击声 →检查防倾杆连杆及悬挂衬套

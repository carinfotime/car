## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 机油及机滤更换 | 每5,000公里或6个月 |
| 空气滤清器 | 每10,000公里或12个月 |
| 空调滤清器 | 每10,000公里或12个月 |
| 燃油滤清器 | 每20,000公里 |
| 火花塞 | 每30,000公里 |

| 防冻液 | 每40,000公里或2年 |

| 正时皮带/链条 | 每80,000-100,000公里（视具体发动机型号） |

## 二、日常检查项目（建议每周/每次用车前）
- [ ] **轮胎**：检查胎压（标准值见驾驶侧门框标签），观察胎面磨损及有无异物
- [ ] **灯光**：大灯、转向灯、刹车灯、雾灯是否正常工作
- [ ] **仪表盘**：是否有故障灯亮起
- [ ] **刹车**：踩下刹车踏板感受力度是否正常，有无异响
- [ ] **雨刮器与玻璃水**：雨刮片是否老化，玻璃水液位是否充足
- [ ] **车身外观**：检查车漆、车玻璃有无损伤

## 三、各阶段详细保养内容
### 🔹 A类保养（5,000km / 6个月）
- ✅更换发动机机油及机油滤清器
- ✅检查并补充各油液（制动液、防冻液、转向助力液）
- ✅检查轮胎气压及磨损情况
- ✅检查底盘及悬挂系统紧固情况
- ✅检查灯光系统功能
- ✅清洁空气滤清器（视情况决定是否更换）
### 🔹 B类保养（10,000km / 12个月）
包含A类全部项目，另增加：
- ✅更换空气滤清器
- ✅更换空调滤清器
- ✅检查火花塞状况
- ✅检查制动片厚度
- ✅检查蓄电池状态及接线柱清洁度
- ✅润滑车门铰链及天窗导轨
### 🔹 C类保养（20,000km / 24个月）
包含B类全部项目，另增加：
- ✅更换燃油滤清器
- ✅清洗节气门及喷油嘴（建议）
- ✅四轮换位及动平衡检测
- ✅全面检测制动系统（刹车盘、刹车片）
- ✅检测转向系统间隙
### 🔹 D类大保养（40,000km / 48个月）
包含C类全部项目，另增加：
- ✅更换制动液
- ✅更换防冻液
- ✅更换火花塞（如未提前更换）
- ✅清洗发动机积碳（建议）
- ✅检测空调系统制冷效果及压力
### 🔹 E类深度保养（60,000km / 72个月）
包含D类全部项目，另增加：
- ✅更换变速箱油（自动挡/手动挡均需注意）
- ✅更换正时皮带及相关张紧轮（如为皮带结构）
- ✅全面检测发电机、起动机工作状态
- ✅清洗三元催化器及氧传感器（建议）
- ⚠️建议进行全车线路老化排查

## 四、关键部件维护说明
### 🛞轮胎维护
|事项|说明|
|---|---|
|标准胎压范围|前轮：2.3–2.5 bar；后轮：2.3–2.8 bar|
|轮胎换位周期|每10,000–15,000公里|
|磨损极限标记|<1.6mm时必须更换|
|使用年限建议|超过5年需定期专业检查；超过8年建议强制更换|
### 🛢️机油选择建议
> - **推荐标号**：5W‑30 / SN级及以上全合成机油  
> - **适用环境**：
>   - -30°C~40°C → SAE 5W‑30  
>   - -25°C~50°C → SAE 10W‑40  
>   - -15°C~50°C → SAE 15W‑40  
> - **用量参考**：约3.5–4.0升（以实际放油量为准）
### 🧊冷却系统注意事项
> - ❗严禁在热车状态下打开水箱盖  
> - ❗不同品牌/颜色的防冻液不可混加  
> - ❗必须使用符合原厂规格的防冻液

## ⚠️五、常见故障预警与处理建议
1. **发动机故障灯亮**
   →  尽快到维修站读取故障码，切勿长时间行驶。
2. **刹车异响或制动力下降**
   →  立即停车检查刹车片厚度，必要时拖车送修。
3. **水温过高报警**
   →  靠边停车熄火，待冷却后检查冷却液位及管路泄漏。
4. **方向盘抖动**
   →  多为轮胎动平衡问题或转向拉杆松动，及时检修。
5. **启动困难**
   →  优先排查蓄电池电压与启动电机接线。

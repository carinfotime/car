# 🚗 冠道 · 儿童乘车安全注意事项
  
更新时间：2026年07月20日08时56分43秒   

## 👶 儿童安全座椅使用规范
- **12 岁以下**或身高不足 **150cm** 的儿童，必须使用符合国家标准的儿童安全座椅。
- **反向安装**：建议 **0-4 岁**（尤其是 2 岁以下）婴幼儿使用反向安装的儿童座椅，以更好保护头颈部。
- **ISOFIX 接口**：冠道后排座椅配备 ISOFIX + Top Tether（上拉带）固定点，请务必正确连接并锁紧。
- **切勿将儿童座椅安装在副驾驶位**（如确需安装，必须关闭副驾驶正面气囊）。
## 🚪 车门与车窗安全
- **开启儿童安全锁**：上车后请将后排两侧车门的儿童安全锁拨至锁定状态，防止行车中儿童误开车门。
- **车窗防夹功能**：冠道配备四门一键升降防夹车窗，但仍需教育孩子不要将头、手伸出窗外。
- **勿让孩子单独操作车窗开关**，避免夹伤手指或颈部。
## 🔐 行车途中注意事项
- **全程系好安全带**：即使使用安全座椅，也需确保约束带松紧适度、不扭曲。
- **车内勿放置尖锐/硬质物品**：急刹车或碰撞时可能飞溅伤及儿童。
- **勿让孩子在车内站立、打闹或爬行**，尤其在后排地板或后备箱区域。
- **停车后切勿将孩子单独留在车内**（无论是否开空调），夏季车内温度可在短时间内升至致命水平。
## 🧸 其他细节提醒
| 项目 | 说明 |
|------|------|
| 🧴 遮阳帘 | 冠道后排原厂配备手动遮阳帘，建议拉起以减少阳光直射及紫外线伤害。 |
| 💨 空调出风口 | 避免冷风直吹儿童头部/腹部；可调节出风口方向或关闭对应风口。 |
| 🎮 HONDA CONNECT | 家长可设置中控屏童锁模式（如适用），限制孩子操作娱乐系统。 |
| 🧳 后备箱 | ⚠️ **严禁让儿童在后备箱内玩耍或乘坐**——该区域非乘员舱，无安全带且碰撞时极度危险。 |

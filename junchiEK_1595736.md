# 骏驰EK 技术参数介绍
  
更新时间：2026年07月17日09时46分05秒   

## 一、车型概览
骏驰EK是一款面向城市通勤与家庭出行的紧凑型轿车，兼顾燃油经济性与驾驶舒适性。
## 二、动力系统
| 项目 | 参数 |
|------|------|
| 发动机类型 | 1.5L 自然吸气 / 1.5T 涡轮增压 |
| 最大功率 | **88 kW** (120 Ps) / **110 kW** (150 Ps) |
| 最大扭矩 | **145 N·m** / **210 N·m** |
| 变速箱 | CVT无级变速 / 6速手动 |
| 驱动方式 | 前置前驱 (FF) |
| 最高车速 | ≥180 km/h |
| 0-100 km/h加速时间 | ≤12.5 s (自吸) / ≤9.8 s (涡轮) |
## 三、车身尺寸
| 项目 | 数值 |
|------|------|
| 车长 (mm) | **4650** |
| 车宽 (mm) | **1820** |
| 车高 (mm) | **1480** |
| 轴距 (mm) | **2700** |
| 整备质量 (kg) | ≈1350 –1420 |
## 四、底盘与悬挂
- **前悬挂**:麦弗逊式独立悬挂
- **后悬挂**:扭力梁式非独立悬挂
- **制动系统**:前通风盘式 /  后盘式
- **转向系统**:电动助力转向(EPS)
##  五、燃油经济性
-  油箱容积:50 L
-  推荐燃油:92#及以上无铅汽油
-  综合油耗(L/100km):
    -   1.5L自吸 + CVT:**6.2**
    -   1.5T + CVT:**6.8**
##  六、安全配置
-  主/副驾安全气囊
-  前排侧气囊(高配)
-  车身稳定控制系统(ESC)
-  牵引力控制(TCS)
-  坡道起步辅助(HSA)
-  倒车雷达+倒车影像(高配)
## ⚙️七、舒适与科技配置
-   多功能方向盘
-   10.25英寸中控触摸屏
-   手机互联(CarPlay/Android Auto)
-   自动空调(高配)
-   无钥匙进入+一键启动(高配)
> *注:以上参数因具体年款及配置版本可能略有差异,请以实车为准。*

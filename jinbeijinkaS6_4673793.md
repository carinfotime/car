# 金杯金卡S6 动力参数介绍
  
更新时间：2026年07月19日18时48分06秒   

金杯金卡S6 作为一款定位商用的微卡车型，其动力系统主打 **可靠、经济、实用**。目前该车型主要搭载了来自华晨鑫源的 **1.6L自然吸气发动机**。
## 核心动力总成
### 发动机型号：华晨鑫源 SWD16M
这是一款技术成熟、广泛搭载于商用车上的自然吸气发动机，注重低扭输出和燃油经济性。
-   **排量**：1.6L
-   **进气形式**：自然吸气
-   **最大功率**：91kW（约 **124马力**）
-   **最大功率转速**：6000rpm
-   **最大扭矩**：162 N·m
-   **最大扭矩转速**：4000-4400rpm
-   **供油方式**：多点电喷（MFI）
-   **燃油标号**：92号及以上无铅汽油
-   **排放标准**：国VI b
### 变速箱匹配
全系标配 **5速手动变速箱 (5MT)**。
-   齿比经过优化，兼顾了低速爬坡的扭矩需求和高速巡航的燃油经济性。
-   换挡手感清晰，吸入感较强，符合工具车的耐用特性。
## 性能表现与特点
1.  **低扭充沛**
    得益于针对商用场景的调校，发动机在2000rpm左右即可输出大部分扭矩。这使得车辆在满载起步、城市拥堵路况或乡村坡道行驶时，无需频繁降挡即可获得足够的牵引力。
2.  **燃油经济性**
    采用多点电喷技术，配合轻量化车身设计（整车质量约1.3吨），官方综合油耗约为 **7.5L/100km**。实际使用中根据载重和路况会有所浮动，但整体属于同级别中较为省油的水平。
3.  **可靠性高**
    该发动机结构简单、技术成熟，没有复杂的涡轮增压系统或高压直喷系统。这对于需要长期高强度使用的商用车来说，意味着更低的故障率和更简便的后期维护成本。
4.  **最高车速**
    官方最高车速可达 **120km/h**以上（空载状态），能够满足高速路段的基本行驶需求。满载状态下建议保持在经济时速区间（60-90km/h）行驶。

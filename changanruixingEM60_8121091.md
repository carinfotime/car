# 长安睿行EM60 内饰参数介绍
  
更新时间：2026年07月20日11时18分23秒   

## 一、整体内饰风格
长安睿行EM60采用简洁实用的内饰设计风格，以深色为主色调，兼顾商务与家用需求，整体布局清晰，功能分区明确。
## 二、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 织物面料（部分高配可选仿皮） |
| 座椅布局 | 5座/7座可选（2+3或2+2+3布局） |
| 前排座椅调节 | 手动4向调节（前后+靠背角度） |
| 后排座椅 | 可整体折叠放倒，扩展储物空间 |
| 头枕 | 前排独立头枕，后排一体式头枕 |
## 三、仪表盘与中控
- **仪表盘**：传统机械指针式仪表 + 中央单色液晶显示屏
- **显示信息**：车速、电量、续航里程、电机转速、故障提示等
- **中控台**：对称式设计，功能按键布局合理
- **多媒体系统**：
  - USB接口（支持MP3播放）
  - AM/FM收音机
  - Bluetooth蓝牙连接（部分车型）
- **空调系统**：
  - 手动空调
  - PM2.5过滤功能（部分配置）
## 四、方向盘
- **材质**：PVC塑料材质
- **调节方式**：手动上下调节
- **多功能按键**：部分车型配备音量控制/蓝牙电话快捷键
## 五、储物空间
| 位置 | 说明 |
|------|------|
| 手套箱 |  副驾驶前方开放式储物格 |
|  门板储物槽 |  前后门板均设有杯架及储物槽 |
|  中央通道 |  换挡杆前方小型储物格 +  杯架 |
|  后排 |  后排侧壁储物袋 |
##  六、其他内饰配置
- **电动车窗**：前排一键升降（带防夹功能）
- **外后视镜**：手动调节
- **内后视镜**：手动防眩目
- **顶棚照明灯**：前排阅读灯 +  后排顶灯
- **遮阳板**：主副驾均配备，副驾带化妆镜
- **安全配置内饰相关**：
    - ABS+EBD
    -  主驾驶安全气囊（部分车型）
    - ISOFIX儿童座椅接口（第二排）
##  七、尺寸与空间参数
|     项目     |      参数      |
|-------------|---------------|
|   车身长度    |    4495 mm    |
|   车身宽度    |    1680 mm    |
|   车身高度    |    1990 mm    |
|     轴距     |    2925 mm    |
|  后备箱容积   |     约2.6 m³ （后排放倒后） |

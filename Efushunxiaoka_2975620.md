# 江铃福特 E福顺小卡 技术参数介绍
  
更新时间：2026年07月15日08时26分09秒   

## 一、车型概述
**E福顺小卡**是江铃福特基于福顺平台打造的一款纯电动微卡车型，定位于城市物流配送、商超供货、个体运输等场景，具备零排放、低运营成本、高承载能力等特点。
---
## 二、整车基本参数
| 项目 | 参数 |
|------|------|
| 车型名称 | E福顺小卡（纯电动） |
| 车身形式 | 单排座微卡 / 双排座微卡 |
| 驱动形式 | 后轮驱动 |
| 轴距 | **3360 mm** |
| 整车尺寸（长×宽×高） | **5995 × 1970 × 2200 mm**（货箱版） |
| 货箱尺寸（长×宽×高） | **3700 × 1890 × 380 mm**（单排栏板式） |
| **总质量** | **4495 kg** |
| **额定载质量** | **约1500–1800 kg**（视具体公告配置） |
| **整备质量** | **约2600–2800 kg** |

## 三、三电系统
### （一）动力电池
| 项目 | 参数 |
|------|------|
| 电池类型 | **磷酸铁锂（LFP）电池** |
| **电池容量** | **50.23 kWh /  63.75 kWh /  77.28 kWh （可选配不同容量版本）** |
| CLTC工况续航里程 | **约200–300 km（视电池容量及载重情况而定）** |
| BMS系统 | 智能电池管理系统，具备过温保护、过放保护、均衡管理等功能 |
### （二）驱动电机
| 项目 | 参数 |
|------|------|
| **电机类型** | **永磁同步电机（PMSM）** |
| **额定功率/峰值功率** | **40 kW /  90 kW （部分版本峰值可达105 kW）** |
| **额定扭矩/峰值扭矩** | **约120 N·m /  300 N·m以上** |
### （三）电控系统
- VCU整车控制器：集成能量回收策略，支持多模式驾驶
- MCU电机控制器：高效矢量控制，响应迅速
- OBC车载充电机：支持交流慢充与直流快充
- DCDC转换器：高效稳定供电至低压系统
---
## 四、充电性能
- **快充接口：国标直流快充**
- SOC从20%充至80%时间：约40–60分钟（视电池容量及充电桩功率）
- AC慢充时间：约8–10小时充满
- V2L外放电功能：部分版本支持对外放电，满足野外作业或应急供电需求
---
## 五、底盘与悬挂系统
### （一）悬架结构
- **前悬架：麦弗逊式独立悬架**
- **后悬架：钢板弹簧非独立悬架**
### （二）制动系统
-  前盘后鼓式制动器
-  标配ABS防抱死制动系统 + EBD电子制动力分配
-  再生制动能量回收系统，提升续航表现
### （三）转向系统
- EPS电动助力转向，低速轻便、高速沉稳
-  最小转弯直径约12 m左右，城市窄路调头灵活
---
## ⚙️六、轮胎与承载能力
> -   轮胎规格：185R15LT /  195R15LT（承载型）
> -   后双胎设计可选，提升载重稳定性与安全性
> -   车架采用高强度钢纵梁结构，抗扭性能优异
> -   最大爬坡度 ≥20%

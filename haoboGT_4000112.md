# 昊铂GT 动力参数介绍
  
更新时间：2026年07月17日11时18分40秒   

## 一、动力系统概览
昊铂GT（Hyper GT）是广汽埃安旗下高端品牌昊铂推出的纯电动轿跑车型，基于全新的 **AEP 3.0** 纯电专属平台打造，采用后驱布局。
---
## 二、动力性能参数
| 参数项 | 数值 |
|--------|------|
| **驱动方式** | 后置后驱 |
| **电机类型** | 永磁同步电机 |
| **最大功率** | 250 kW（约 340 马力） |
| **最大扭矩** | 430 N·m |
| **0-100 km/h 加速时间** | 4.9 秒 |
| **最高车速** | 180 km/h |
> 注：以上为单电机后驱版本数据。
---
## 三、电池与续航
| 参数项 | 数值 |
|--------|------|
| **电池类型** | 三元锂电池 / 磷酸铁锂电池 |
| **电池容量** | 59.9 kWh / 70.1 kWh / 80.1 kWh（因版本不同） |
| **CLTC 纯电续航** | 560 km / 600 km / 710 km（因版本不同） |
| **快充时间** | 15 分钟（30% → 80%） |
| **快充功率** | 最高支持 480 kW 超快充 |
---
## 四、核心技术亮点
### 1. **AEP 3.0 纯电平台**
- 高度集成化电驱系统，体积更小、效率更高
- 后驱布局带来更优秀的操控表现
### 2. **高压快充系统**
- 支持 800V 高压平台
- 充电 15 分钟可补充续航约 450 km
- 适配广汽自研的超级充电桩
### 3. **能量回收系统**
- 多级可调能量回收强度
- 单踏板模式可选，提升续航表现
### 4. **热管理系统**
- 智能热泵空调，低温环境下续航衰减更少
- 电池液冷温控系统，保障充放电性能稳定
---
## 五、驾驶模式与动态表现
| 驾驶模式 | 特点 |
|----------|------|
| **舒适模式** | 动力输出平顺，适合日常通勤 |
| **运动模式** | 动力响应更积极，加速迅猛 |
| **经济模式** | 优化能耗，延长续航里程 |
| **个性化模式** | 可自定义动力响应、转向力度等 |
- **底盘结构**：前双叉臂 + 后五连杆独立悬架
- **制动系统**：前通风盘 + 后盘式刹车，搭配博世 iBooster 电控刹车系统
- **风阻系数**：Cd 值低至 **0.197**，有效降低高速行驶能耗
---
## 六、小结
昊铂GT 作为一款高性能纯电轿跑，凭借 **250 kW 大功率电机**、**4.9 秒零百加速**、**最高 710 km 续航**以及 **800V 超快充技术**，在动力性能和充电体验上均展现出较强的竞争力，兼顾了日常通勤与驾驶乐趣的双重需求。

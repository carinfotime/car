# 江铃·顺达 技术参数介绍
  
更新时间：2026年07月12日15时03分46秒   

## 一、车型概览
江铃顺达是江铃汽车旗下经典轻卡系列，以经济耐用、高承载和低油耗著称，广泛应用于城市物流、商超配送及短途运输场景。
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **发动机型号** | JX493ZLQ5 / JX4D30B6H（国六） |
| **发动机类型** | 直列四缸、增压中冷、高压共轨柴油机 |
| **排量** | 2.771 L / 2.892 L |
| **最大功率** | 85 kW（115 Ps）@ 3200 rpm |
| **最大扭矩** | 285 N·m @ 1800-2200 rpm |
| **排放标准** | 国五 / 国六 b |
| **变速箱** | 5挡手动（格特拉克或江齿） |
## 三、底盘与悬挂
| 项目 | 参数 |
|------|------|
| **驱动形式** | 4×2（后轮驱动） |
| **前桥载荷** | ≥1.5 t |
| **后桥载荷** | ≥3.0 t（部分加强型可达3.5 t） |
| **前悬挂** | 独立悬挂 + 钢板弹簧（3片或4片） |
| **后悬挂** | 钢板弹簧（5+4片或7+6片，依轴距而定） |
## 四、制动与转向
- **制动系统：**  
  前盘式制动 + ABS防抱死系统  
  后鼓式制动  
- **转向助力：**  
  液压助力转向  
## 五、车身尺寸与货箱规格
### （一）单排标准轴距版
- **总长 × 总宽 × 总高：** ≈5995 × ≈2020 × ≈2350 mm  
- **轴距：** ≈3360 mm  
- **货箱内尺寸（长×宽×高）：** ≈4200 × ≈1940 × ≈400 mm  
### （二）排半/双排版本
- **货箱长度：** ≈3800 mm / ≈3200 mm  
- *其余宽度与高度同单排版*
> *注：具体尺寸因公告型号及改装需求略有差异。*
## 六、质量参数
- **整备质量：** ≈2.65 - 2.8 t  
- **额定载质量：** ≥1.495 t（蓝牌合规上限约1.8 t视公告而定）  
- **总质量：** ≤4.495 t  
## 七、油耗与续航
- **综合工况油耗：** ~8 -10 L/100 km（视载重及路况浮动）  
- **油箱容积：** ≥80 L  
## 八、驾驶室配置
-  座椅材质 ：织物 / PVC仿皮可选
-  空调 ：手动冷暖空调
-  电动车窗 ：主副驾一键升降
-  中控锁 +遥控钥匙 
-  仪表盘 ：机械指针+液晶行车电脑 
-  收音机/USB/蓝牙 （部分高配带大屏导航）
## 九、安全配置
1. ABS防抱死制动系统 
2. EBD电子制动力分配 
3.倒车雷达 （选装）
4.高位刹车灯 
5.驾驶室碰撞吸能结构

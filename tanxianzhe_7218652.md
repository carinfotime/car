# 福特探险者（Ford Explorer）操作指南
  
更新时间：2026年07月12日18时33分51秒   

## 一、启动与熄火
### 1.1 智能钥匙启动
- **进入车辆**：携带智能钥匙靠近车辆，拉动门把手即可解锁。
- **启动发动机**：
  1. 确保档位处于 **P（驻车档）**。
  2. 踩下制动踏板。
  3. 按下 **ENGINE START/STOP** 按钮。
- **熄火**：
  1. 将车辆完全停稳。
  2. 挂入 **P（驻车档）**。
  3. 按下 **ENGINE START/STOP** 按钮。
### 1.2 远程启动（如配备）
- 按一下遥控钥匙上的 **锁车键**，然后长按 **远程启动键（2X）**。
- 车辆将自动启动并运行预设时间（通常为10-15分钟）。
- 进入车内后，需按正常步骤踩刹车并按启动键方可行驶。

## 二、档位操作
探险者通常配备旋钮式或电子换挡器。
| 档位 | 名称 | 用途 |
|------|------|------|
| P | 驻车档 | 停车时使用 |
| R | 倒车档 | 倒车时使用 |
| N | 空档 | 短暂停车或拖车时使用 |
| D | 前进档 | 正常行驶 |
| S / L / M | 运动/低速/手动模式 | - |
> **注意**：换挡时必须踩住制动踏板。旋钮式换挡器需旋转至对应位置。
---
## 三、驾驶模式选择
探险者提供多种地形管理模式，通过中控台旋钮切换：
- **标准模式（Normal）**：日常城市道路驾驶。
- **运动模式（Sport）**：更积极的油门响应和换挡时机。
- **经济模式（Eco）**：优化燃油经济性，降低空调功率等。
- **越野/沙地模式（Sand / Trail）**：适合非铺装路面或沙地。
- **雪地/湿滑模式（Slippery）**：低附着力路面下提供更平稳的动力输出。

## 四、信息娱乐系统
### SYNC® / SYNC®3 / SYNC®4
- **触摸屏控制**：主页可访问导航、音频、电话、设置等。
- **语音控制**：
  1.  按下方向盘上的语音按钮。
  2.  说出指令，例如：
     - “导航到 [目的地]”
     - “拨打 [联系人]”
     - “播放 [歌曲/电台]”
     - “调高温度”
- **Apple CarPlay / Android Auto**
  1.  使用原厂 USB数据线连接手机至指定接口。
  2.  中控屏自动识别并投射手机界面。

## 五、安全与辅助驾驶功能
### Co-Pilot360™ (部分功能为选配)
#### ACC自适应巡航
1.  按下方向盘上的巡航开启键 `ON` 。
2.  设定速度 (`SET+` / `SET–`) 。
3.  调节跟车距离 (`GAP`) 。
4.  取消巡航 (`CANCEL`) 。
#### LKA车道保持辅助
-   默认开启。当车辆偏离车道且未打转向灯时，系统会轻微修正方向盘并发出警示。
#### BLIS盲区监测
-   外后视镜上的指示灯亮起表示盲区内有车辆。若此时打转向灯准备变道，指示灯会闪烁并伴有警示音。
#### AEB自动紧急制动
-   系统检测到前方碰撞风险且驾驶员未采取行动时，会自动施加制动以减轻或避免碰撞。

## ⚠️六、仪表盘指示灯与警告
常见警告灯含义：
|图标颜色|名称|含义|
|---|---|---|
|红色|制动系统警告灯||
||制动液不足或手刹未完全释放||
||发动机机油压力低||
||立即停车检查机油||
||电池充电故障||
||充电系统异常||
||安全气囊故障灯||
||安全气囊系统存在故障|
|---|---|---|
黄色/琥珀色|发动机故障灯 (Check Engine)|
||排放或发动机控制系统异常|
||胎压监测 (TPMS)|
||某个轮胎气压过低|
||牵引力控制关闭|
> ⚠️出现红色警告灯时应尽快安全停车检查；黄色警告灯建议尽快前往维修站检修。

## ⚠️七、应急操作
### ❗拖车与牵引
-   建议使用平板拖车。若需四轮着地牵引：
    -   必须将分动箱置于空挡 (N)，或将传动轴断开。具体请参考车主手册关于“ Recreational Towing”的说明。错误牵引可能导致变速箱严重损坏！
### ❗更换备胎 (如配备)
1.  将车辆停于平坦坚硬地面并拉紧手刹。
2.  取出备胎及千斤顶工具组 (通常位于后备箱地板下方)。
3.  用千斤顶支撑在指定举升点 (门槛下方标记处)。
4.  拧松并取下螺栓 →更换备胎→按对角线顺序拧紧螺栓→降下千斤顶→最终紧固螺栓至规定扭矩(约135 N·m)。
### ❗紧急呼叫 (如配备)
部分车型配备 SOS按钮 ，位于头顶控制区。按下即可连接紧急服务中心并提供位置信息。

## ⚠️八、日常保养建议
项目周期备注机油更换每10,000公里或12个月建议使用符合 WSS-M2C961-A1标准的全合成机油轮胎换位每10,000公里交叉换位以均匀磨损空气滤芯每30,000公里检查必要时更换刹车油每40,000公里检查含水量雨刮片每年更换根据实际磨损情况电池寿命一般3–5年寒冷地区寿命较短注意低温性能
> 💡定期查看仪表盘上的“机油寿命”百分比提示作为参考依据之一。

## ⚠️九、常见问题解答 (FAQ)
Q:如何重置胎压监测系统？
A:补气至标准值后以超过32 km/h的速度行驶几分钟即可自动复位；部分车型需通过中控菜单手动校准(设置→车辆→TPMS校准)。
Q:油箱盖在哪里打开？
A:驾驶员侧车门解锁状态下直接按压油箱盖即可弹开；内部无机械拉手(除极个别早期型号外)。
Q:如何开启四驱锁定？
A:在越野模式下系统会自动启用；也可通过中控台旋钮选择特定地形模式来激活前后轴动力分配锁定功能(具体视配置而定)。
Q:儿童安全锁在哪里？
A:位于两个后门的门锁扣下方拨杆位置——拨至锁止图标一侧即从车内无法开门只能从外部打开。
Q:如何使用陡坡缓降？
A:挂入D挡或L挡 →按下中控台陡坡缓降按钮(图标为下坡箭头)→松开刹车让系统自动控制车速下坡(速度约5–8 km/h可调)。

# 跨越星V5 EV 竞品汽车分析报告
  
更新时间：2026年07月12日12时34分05秒   

## 一、车型定位概述
**跨越星V5 EV** 是一款由长安跨越推出的纯电动微面（微型面包车），主要面向城市物流配送、短途货运、个体商户等商用场景。其核心卖点在于大空间、低运营成本以及新能源路权优势。
## 二、主要竞品车型
在纯电动微面/轻客市场，跨越星V5 EV的主要竞争对手包括：
| 竞品品牌 | 代表车型 | 厂商 |
|----------|----------|------|
| 五菱 | 五菱EV50 / 五菱荣光EV | 上汽通用五菱 |
| 远程 | 远程星享V6E / V5E | 吉利商用车 |
| 开瑞 | 开瑞海豚EV / 优优EV | 奇瑞商用车 |
| 福田 | 福田风景智蓝G7 / G5 EV | 北汽福田 |
| 东风 | 东风御风EM26 / EM19 | 东风汽车 |

# 风光MINIEV 操作指南
  
更新时间：2026年07月13日11时54分35秒   

## 一、车辆启动与关闭
### 1. 启动车辆
1.  **携带钥匙**：确保智能钥匙在车内。
2.  **踩下制动踏板**：右脚踩住刹车。
3.  **启动电源**：
    -   如果车辆处于“OFF”或“ACC”档，请按一下“启动/停止”按键。
    -   仪表盘上“READY”（或“OK”）指示灯点亮，表示车辆已进入可行驶状态。
4.  **松开电子手刹**（如有）：部分车型需手动按下电子手刹开关解除制动。
### 2. 关闭车辆
1.  **踩下制动踏板**，将车辆停稳。
2.  **挂入P档**（驻车档）。
3.  再次按下 **“启动/停止”按键**，仪表盘熄灭，车辆断电。
> **提示**：若未挂入P档直接熄火，系统会发出提示音并拒绝断电。

## 二、档位操作
风光MINIEV 采用旋钮式换挡机构。
| 档位 | 操作说明 |
| :--- | :--- |
| **R** | 倒车档。车辆完全停止后，踩刹车旋转至R档。 |
| **N** | 空档。临时停车（如等红灯）时使用。 |
| **D** | 前进档。日常行驶使用。 |
| **S** | 运动档（如有）。动力响应更积极，但能耗较高。 |
**操作注意**：
-   换挡时必须**踩住刹车**。
-   档位切换后，仪表盘会显示当前档位。
---
## 三、驾驶模式选择
通过中控台或仪表盘上的 **“ECO/SPORT”** 按键切换：
-   **ECO（经济模式）**：动力输出平缓，能量回收较强，续航更长。
-   **SPORT（运动模式）**：加速响应更快，适合超车或爬坡，能耗较高。
> 建议：城市拥堵路况使用ECO，快速路或需要动力时切换SPORT。
---
## 四、仪表盘指示灯解读
| 图标 | 含义 | 建议 |
| :--- | :--- | :--- |
| **READY** | 车辆已就绪，可行驶 | 正常状态 |
| **电池图标** | 电量显示 | 电量低于20%时请及时充电 |
| **红色电池+感叹号** | 动力电池故障 | 立即停车，联系售后 |
| **黄色扳手** | 系统故障或需保养 | 尽快进店检查 |
| **P** | 驻车档/电子手刹拉起 | 起步前确认解除 |

## 五、充电操作
### 1. 交流慢充（家用）
1.  车辆熄火，挂P档。
2.  打开车身充电口盖（通常位于车头或车侧）。
3.  将充电枪插入充电口，听到“咔哒”声即锁止。
4.  充电桩或随车充电器通电后，仪表盘充电指示灯闪烁表示正在充电。
5.  充满后，解锁车辆，拔下充电枪并盖好充电口盖。
### 2. 充电注意事项
-   **禁止**在充电时启动车辆。
-   **避免**在极端高温或低温下立即充电。
-   建议使用**有接地保护的插座**（随车充）。
-   充电时长（0-100%）：约6-8小时（具体视电池容量和充电功率而定）。

## 六、日常检查与维护
### 1. 每次出行前
-   检查轮胎外观及胎压（建议2.3-2.5 bar）。
-   检查电量是否足够当天行程。
-   检查灯光、喇叭是否正常。
### 2. 定期保养
-   **每1年或1万公里**：进店检查三电系统、刹车、底盘等。
-   **冷却液**：检查液位，不足时添加专用冷却液。
-   **玻璃水**：定期加注，冬季使用防冻型。
### 3. 长期停放
-   建议保持电量在 **50%-80%** 之间。
-   每两周启动一次，检查车辆状态。
-   断开12V小电瓶负极（如需停放超过一个月）。
---
## 七、安全驾驶提示
1.  **起步缓加速**：电动车扭矩大，轻踩电门即可。
2.  **能量回收**：松开电门时车辆会自动减速并回收电能，可减少刹车使用。
3.  **涉水**：避免通过积水超过轮毂中心的路段。
4.  **儿童锁**：后门设有儿童安全锁，防止儿童从车内开门。
5.  **紧急情况**：若车辆异常断电，可尝试长按启动键3秒以上强制重启系统。
---
## 八、常见问题解答
**Q：车辆无法启动？**
A：检查智能钥匙是否在车内，刹车是否踩到底，档位是否在P档。
**Q：充电时指示灯不亮？**
A：检查充电枪是否插紧、电源插座是否通电、充电桩是否故障。
**Q：续航里程下降明显？**
A：冬季低温或激烈驾驶会缩短续航，属正常现象。可尝试切换ECO模式并平稳驾驶。

# 黄海威龙 保养指南
  
更新时间：2026年07月17日15时53分15秒   

## 一、定期保养周期表
| 保养项目 | 建议周期 |
|---------|---------|
| 更换机油及机油滤清器 | 每5,000公里或6个月 |
| 更换空气滤清器 | 每10,000公里或12个月 |
| 更换燃油滤清器 | 每20,000公里或24个月 |
| 更换空调滤清器 | 每10,000公里或12个月 |
| 检查/更换火花塞 | 每30,000公里 |
| 检查/更换正时皮带/链条 | 每60,000-80,000公里 |
| 更换变速箱油（手动） | 每40,000公里 |
| 更换变速箱油（自动） | 每60,000公里 |
| 更换刹车油 | 每40,000公里或2年 |
| 更换防冻液 | 每40,000公里或2年 |
| 检查/更换转向助力油 | 每40,000公里 |
| **轮胎换位** | **每10,000公里** |
> **注：**以上为常规建议，具体请以车辆使用手册为准。恶劣路况或重载使用应适当缩短保养周期。
---
## 二、日常检查项目（建议每周或每次出车前）
- [ ] **机油液位**：冷车状态下检查，应在标尺上下刻度之间
- [ ] **冷却液液位**：观察膨胀水箱，应在MIN与MAX之间
- [ ] **刹车油液位**：确保在正常范围内
- [ ] **轮胎气压及外观**：检查胎压是否正常，有无鼓包、裂纹、异常磨损
- [ ] **灯光系统**：大灯、转向灯、刹车灯、雾灯是否正常工作
- [ ] **雨刮器及玻璃水**：雨刮片是否老化，玻璃水是否充足
- [ ] **仪表盘警示灯**：启动后所有警示灯应正常熄灭
---
## 三、发动机保养要点
1. **机油选择**
   - SAE粘度等级推荐：5W-30 /10W-40（根据当地气温选择）
   - API质量等级不低于 SL/SM/SN级别
   - **柴油版车型需使用柴油专用机油（CF-4/CH-4级别以上）**
2. **冷却系统**
   - 使用长效防冻液，冰点低于当地最低气温10°C以上
   - **严禁混用不同品牌防冻液**
   -定期检查水泵皮带张紧度及有无裂纹
3. **进气系统**
   -空气滤清器不可用高压气枪从内向外吹洗（会破坏滤纸结构）
   -灰尘大的环境下应缩短更换周期至5,000公里
---
## 四、底盘与传动系统保养
### （一）变速箱保养
#### 手动变速箱：
-使用GL-4 / GL-5级别齿轮油
-换挡生涩时优先检查油量及油质
#### 自动变速箱：
-**必须使用厂家指定型号ATF油**
-**严禁用普通机油替代**
-**建议循环机换油更彻底**
### （二）制动系统
1.刹车片厚度低于3mm时必须立即更换
2.刹车盘出现深沟槽或跳动时需光碟或更换
3.**定期检查制动管路有无渗漏**
### （三）悬挂与转向系统
1.定期检查减震器有无漏油现象
2.转向拉杆球头间隙过大时应及时更换
3.**四轮定位建议每年做一次**
---
## 五、电气系统保养
1.蓄电池维护：
   -保持极柱清洁无氧化物（可用开水冲洗后涂抹凡士林）
   -电解液不足时添加蒸馏水（非免维护型）
   -长期停放超过15天建议断开负极线
2.发电机皮带：
   -每次保养时检查张紧度及老化裂纹情况
   -一般60,000公里左右需更换
---
## 六、轮胎保养要点
1.**胎压标准参考值（冷态）：**
   -前轮：2.3~2.5 bar
   -后轮：2.5~2.8 bar（空载）/根据载重适当增加
2.**轮胎磨损极限标记**位于花纹沟槽内，磨至该标记时必须换胎
3.**轮胎使用寿命不超过6年**（从生产日期算起），即使花纹尚可也应考虑更换
4.**备胎同样需要定期检查气压**
---
## 七、季节性特别养护
### 春季/雨季：
✔️清理排水槽及天窗排水孔  
✔️检查雨刮片并及时更换  
✔️空调系统杀菌除臭  
### 夏季：
✔️重点检查冷却系统  
✔️空调制冷效果检测  
✔️防止高温爆胎，适当降低胎压0.1~0.2 bar  
### 秋季：
✔️全面底盘检查  
✔️准备冬季防冻液  
✔️蓄电池性能检测  
### 冬季：
✔️使用冬季专用玻璃水（冰点低于当地最低温）  
✔️柴油版车型注意加注低凝点柴油  
✔️冷启动前预热后再行驶  
---
## 八、常见问题提示与注意事项
⚠️黄海威龙部分车型为柴油动力，请特别注意：
1.**柴油版必须加注符合国标的车用柴油**
2.**切勿加注劣质柴油以免损坏高压共轨系统**
3.**DPF颗粒捕捉器需定期再生清理**
4.**尿素液位需及时补充**
⚠️其他注意事项：
1.**首次保养尤为重要**——新车磨合期结束后务必按时进站首保  
2.**保修期内尽量在授权服务站进行保养**以免影响质保权益  
3.**所有保养记录请妥善保存**以便二手车交易时提供完整维保记录  
4.**改装电路可能引发车辆自燃风险，请谨慎对待**

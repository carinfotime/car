# 保时捷 Cayenne 空间参数介绍
  
更新时间：2026年07月20日16时33分01秒   

## 车身尺寸
| 参数 | 数据 |
|------|------|
| 车长 | 4,918 mm |
| 车宽（不含后视镜） | 1,983 mm |
| 车高 | 1,696 mm |
| 轴距 | 2,895 mm |
| 前轮距 | 1,675 mm |
| 后轮距 | 1,695 mm |
| 最小离地间隙（标准悬架） | 约200 mm（空气悬架可调范围更大） |
## 内部空间
### 乘坐空间
- **前排头部空间**：约980 mm
- **后排头部空间**：约950 mm
- **后排腿部空间**：约900 mm（前排座椅调至适中位置）
- **后排座椅宽度**：可舒适容纳三名成年人
- **座椅布局**：5座布局，后排座椅支持40:20:40比例放倒
### 后备箱容积
| 状态 | 容积 |
|------|------|
| 标准状态（后排座椅直立） | **772 L** |
| 后排座椅放倒后 | **1,708 L**（最大装载容积） |
> *注：Cayenne Coupé车型后备箱容积略有不同，标准状态下约为625L，放倒后约为1,540L。*
## 储物空间细节
### 前排储物
- **中央扶手箱**：容积约8L，内置USB接口和无线充电板选装位
- **手套箱**：带制冷功能，可容纳约6瓶500ml矿泉水
- **门板储物格**：每侧可放置2瓶500ml水瓶及小件物品
- **杯架**：前排中央通道配备2个杯架
### 后排储物
- **中央扶手杯架**：2个杯架（隐藏式设计）
- **门板储物格**：每侧可放置1瓶水瓶及杂物
- **座椅背袋**：前排座椅背部配备储物袋
## Towing & Roof Load
| 参数 | Cargo Capacity |
|---|---|
|  最大拖拽质量（带制动器） |  3,500 kg |
|  最大车顶负载 |  100 kg |
## Key Features
*   电动尾门 (标配)
*   行李箱分隔网/行李固定系统 (选装)
*   后排座椅一键放倒功能 (通过后备箱内拉手)
*   空气悬架系统可在装载时降低车身高度，方便装卸行李

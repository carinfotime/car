# 上汽大通MAXUS G90 技术参数介绍
  
更新时间：2026年07月12日12时51分52秒   

## 一、车型定位
上汽大通MAXUS G90是一款定位于**中大型高端MPV**的车型，主打商务接待与家庭出行市场。其竞争对手主要瞄准别克GL8、丰田赛那、传祺M8等车型。
## 二、动力系统
| 项目 | 参数 |
| :--- | :--- |
| **发动机类型** | 2.0T涡轮增压汽油发动机 + 48V轻混系统 |
| **发动机型号** | 20A4E |
| **排量** | 1986 mL |
| **进气形式** | 涡轮增压 |
| **最大马力** | 234 Ps (172 kW) |
| **最大功率转速** | 5000 - 5500 rpm |
| **最大扭矩** | 390 N·m |
| **最大扭矩转速** | 1500 - 4000 rpm |
| **燃油标号** | 92号及以上无铅汽油 |
| **供油方式** | 缸内直喷 |
| **排放标准** | 国VI b |
## 三、传动与底盘
-   **变速箱类型**: ZF（采埃孚）8速自动变速箱（8AT）
-   **驱动方式**: 前置前驱
-   **前悬挂类型**: 麦弗逊式独立悬挂
-   **后悬挂类型**: 五连杆式独立悬挂
-   **转向系统**: EPS电动助力转向
-   **制动系统（前/后）**: 通风盘式 / 盘式
-   **驻车制动类型**: EPB电子手刹 + Auto Hold自动驻车

### （二）座椅布局
-   **七座版**: `2 + 2 +3` （第二排为独立航空座椅）
-   **九座版**: `2 +2 +2 +3` （第四排为3座布局）
### （三）核心空间数据
-   第二排座椅前后滑动距离：最长可达近1米。
-   第三排空间：在保证第二排舒适的前提下，第三排腿部空间依然充裕。
-   后备箱容积：满员状态下，七座版仍可放置多个登机箱；九座版后备箱空间相对紧凑。
## ⚙️五、核心配置亮点
### （一）智能驾驶辅助（L2+级别）
G90全系标配了较为丰富的驾驶辅助功能，部分高配车型达到L2+级：
-   全速自适应巡航（ACC）
-   车道保持辅助（LKA）
-   车道偏离预警（LDW）
-   主动刹车 / AEB自动紧急制动
-   交通标志识别（TSR）
-   360°全景影像（部分高配带透明底盘功能）
-   自动泊车辅助（APA）
### （二）智能座舱与互联
1.  三联屏设计：
    -   仪表屏：10.25英寸
    -   中控屏：12.3英寸
    -   副驾娱乐屏：12.3英寸
2.  流媒体后视镜。
3.  搭载蜘蛛智联系统，支持：
    -   OTA远程升级
    -   语音交互控制
    -    车载微信/导航/音乐等生态应用。
4.  后排独立娱乐屏（高配可选）。
### （三）舒适性配置
1.  第二排VIP航空座椅：
    -    电动调节（靠背、腿托、前后移动）。
    -    加热、通风、按摩功能。
        -    小桌板。
        -    独立USB充电口及杯架。
2.  双侧电动侧滑门。
3.    电动尾门（带位置记忆）。
4.    分段式全景天窗或双天窗设计。
5.    后排独立空调及负离子空气净化器。
## ⚡六、其他关键参数
### （一）油耗与续航
*注：实际油耗受驾驶习惯及路况影响*
-   WLTC综合油耗：约 `8.5 ~9.2 L/100km` （根据配置不同略有差异）。
-     油箱容积：约 `70 L`。
### （二）轮胎规格
根据配置不同提供以下规格：
1.     225/60 R17
2.     225/55 R18
3.     235/50 R19
备胎类型：非全尺寸备胎。

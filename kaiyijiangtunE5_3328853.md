# 凯翼江豚E5 内饰参数介绍
  
更新时间：2026年07月18日10时16分04秒   

## 一、整体设计风格
凯翼江豚E5内饰采用**简约科技风**设计理念，整体布局以驾驶员为中心，强调实用性与舒适性的平衡。车内以深色为主色调，搭配银色镀铬装饰条，营造出年轻、时尚的座舱氛围。
## 二、座椅系统
| 项目 | 参数 |
|------|------|
| 座椅材质 | 仿皮/织物混搭（高配为全仿皮） |
| 前排座椅调节 | 主驾6向手动调节 / 副驾4向手动调节 |
| 后排座椅 | 整体式座椅，支持整体放倒 |
| 座椅功能 | — |
## 三、仪表盘与中控屏
- **仪表盘**：7英寸液晶组合仪表（部分配置为传统指针+小屏）
- **中控屏**：10.25英寸悬浮式触控液晶屏
- **车机系统**：智能互联系统，支持蓝牙、USB连接
- **手机互联**：支持CarLife手机映射
- **语音控制**：基础语音识别（导航、电话等）
## 四、方向盘
- **材质**：塑料多功能方向盘（高配可选皮质包裹）
- **调节方式**：手动上下调节
- **功能按键**：集成多媒体控制、蓝牙电话接听键
## 五、空调系统
- **空调类型**：手动空调
- **后排出风口**：部分配置配备
- **空气净化**：PM2.5过滤装置（选装）
## 六、储物空间
| 位置 | 说明 |
|------|------|
| 前排杯架 | 2个（中控台前方） |
| 中央扶手箱 | 小型储物盒 |
| 手套箱 | 常规尺寸 |
| 门板储物格 | 前后门均有，可放置水瓶及杂物 |
| 后排杯架 | — |
## 七、其他内饰配置
- **换挡机构**：旋钮式电子换挡
- **车窗控制**：四门电动车窗（主驾一键升降）
- **内后视镜**：手动防眩目
- **遮阳板**：主副驾均带化妆镜
- **阅读灯**：前排顶灯
- **USB接口**：前排1个USB + Type-C接口 / （后排1个USB接口）

> -以上参数基于凯翼江豚E5主流配置版本整理，具体配置可能因车型年款及选装包不同有所差异。
> -建议购车前以当地经销商提供的实车配置清单为准。

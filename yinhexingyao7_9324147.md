# 银河星耀7 动力参数介绍
  
更新时间：2026年07月11日17时59分50秒   

## 动力总成
| 项目 | 参数 |
|------|------|
| 发动机类型 | 1.5T 四缸涡轮增压发动机 |
| 排量 | 1499 mL |
| 最大马力 | 181 Ps（133 kW） |
| 最大功率转速 | 5500 rpm |
| 最大扭矩 | 290 N·m |
| 最大扭矩转速 | 2000 - 4000 rpm |
| 燃油标号 | 92# 及以上无铅汽油 |
| 供油方式 | 缸内直喷 |
| 环保标准 | 国VI b |
## 传动系统
| 项目 | 参数 |
|------|------|
| 变速箱类型 | 7速湿式双离合变速箱（7DCT） |
| 驱动方式 | 前置前驱 |

| 项目 | 参数 |
|------|------|
| 0-100 km/h 加速时间 | 7.9 秒 |
| 最高车速 | 210 km/h |
| WLTC 综合油耗 | 6.4 L/100 km |
| 油箱容积 | 55 L |
## 底盘与操控
| 项目 | 参数 |
|------|------|
| 前悬挂类型 | 麦弗逊式独立悬架 |
| 后悬挂类型 | 多连杆式独立悬架 |
| 转向系统 | 电动助力转向（EPS） |
| 制动系统（前/后） | 通风盘式 / 盘式 |
| 轮胎规格 | 225/55 R18 |
## 驾驶模式
提供 **经济（ECO）**、**舒适（Comfort）**、**运动（Sport）** 三种驾驶模式，可根据不同路况和驾驶需求灵活切换，兼顾燃油经济性与动力响应。

# 东风多利卡 D6 操作指南
  
更新时间：2026年07月21日09时50分25秒   

## 一、车辆启动与预热
1.  **上车前检查**：绕车一周，检查轮胎气压、外观有无异常，确认车底无漏油、漏水。
2.  **进入驾驶室**：调整座椅、后视镜至合适位置，系好安全带。
3.  **钥匙启动**：
    -   将钥匙插入点火开关。
    -   拧至“ON”档（通电自检），等待仪表盘所有指示灯熄灭（约3-5秒）。
    -   拧至“START”档启动发动机。启动后立即松手，钥匙自动回弹。
4.  **冷车预热**：
    -   **冬季或寒冷地区**：怠速运转3-5分钟，待水温表指针开始上升后再起步。
    -   **夏季**：怠速运转30秒即可。
> **注意**：严禁长时间连续启动（每次不超过15秒），两次启动间隔至少30秒。
## 二、起步与行驶
### 1. 离合器与挂挡
-   D6通常为手动变速箱（常见5挡或6挡）。
-   **操作要领**：踩下离合器到底 -> 挂入1挡 -> 松开手刹 -> 慢抬离合器至半联动点 -> 轻踩油门 -> 完全松开离合器。
### 2. 换挡时机（参考）
| 档位 | 推荐车速范围 |
|------|-------------|
| 1→2 | ≤15 km/h |
| 2→3 | ≤30 km/h |
| 3→4 | ≤45 km/h |
| 4→5 | ≤60 km/h |
| (6挡) | ≥70 km/h |
> **原则**：根据发动机转速换挡，柴油机建议在1500-2000转/分时换挡。避免低转速高档位行驶（拖挡）。
### 3.  行驶注意事项
-   **载重控制**：严格遵循车辆铭牌上的额定载质量，切勿超载。
-   **转弯减速**：空车或轻载时转弯速度不超过30km/h；重载时不超过20km/h。
-   **长下坡**：使用发动机制动（降档），避免长时间踩刹车导致刹车过热失效。
-   **雨天/湿滑路面**：减速慢行，避免急刹车和急转向。
## 三、停车与熄火
1.  选择平坦安全地点停车。
2.  踩下离合器和刹车 -> 挂入空挡 ->拉紧手刹 ->松开脚刹。
3.  关闭所有用电设备（空调、大灯等）。
4.  怠速运转1-2分钟（特别是涡轮增压车型），让涡轮降温润滑。
5.  将钥匙拧回“OFF”档拔出钥匙。
> **坡道停车**：熄火后挂入1档或倒档作为辅助制动。
## 四、仪表盘指示灯解读
|    指示灯图标名称     |        含义        |            处理方式            |
| --------------------- | ------------------- | ------------------------------ |
|     发动机故障灯      |      电控系统故障      |      尽快进服务站检修        |
|       机油压力灯        |      机油压力过低      |    立即熄火检查机油量及泄漏     |
|       充电指示灯        |      发电机不发电      |      检查皮带及发电机线路      |
|         水温灯          |        冷却液温度过高        |    停车怠速降温检查冷却液液位     |
|          ABS灯          |      ABS系统故障      |        可行驶但需谨慎制动         |
## ⚠️五、安全须知
1.  严禁酒后驾车、疲劳驾驶。连续驾驶4小时必须休息20分钟以上。
2.  起步前确认货厢门已锁好；行驶中严禁站人。
3.  定期检查轮胎磨损及气压（建议每月一次）。
4.  添加尿素液（国五/国六车型）时使用正规品牌产品，防止堵塞SCR系统。

# 荣威D6 保养指南
  
更新时间：2026年07月11日09时05分54秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 首次保养 | 购车后3个月或行驶3000公里（以先到者为准） |
| 常规保养 | 每6个月或每行驶10000公里（以先到者为准） |
| 大保养 | 每2年或每行驶40000公里（以先到者为准） |
---
## 二、各阶段保养项目明细
### 1. 首次保养（3个月/3000公里）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查转向助力液
- [ ] 检查蓄电池状态
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查灯光系统
- [ ] 检查雨刮器及喷水系统
- [ ] 全车紧固件检查
- [ ] 读取车辆故障码
### 2. 常规保养（每6个月/10000公里）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查并清洁空气滤清器
- [ ] 检查火花塞状态
- [ ] 检查制动片厚度
- [ ] 检查制动盘磨损情况
- [ ] 检查轮胎磨损及胎压（含备胎）
- [ ] 轮胎换位（建议每10000公里）
- [ ] 检查底盘及悬挂系统
- [ ] 检查传动轴及球笼防尘套
- [ ] 检查排气管及三元催化器
- [ ] 检查空调滤清器
- [ ] 检查各油液液位（冷却液、制动液、转向液、玻璃水）
- [ ] 检查皮带张紧度及老化情况
- [ ] 检查电瓶电压及接线柱
### 3. 大保养（每2年/40000公里）
包含常规保养全部项目，并增加以下内容：
- [ ] 更换空气滤清器
- [ ] 更换空调滤清器
- [ ] 更换火花塞
- [ ] 更换制动液
- [ ] 更换冷却液
- [ ] 更换燃油滤清器
- [ ] 清洗节气门
- [ ] 清洗喷油嘴
- [ ] 检查并清洁发动机积碳
- [ ] 检查正时皮带/链条状态
- [ ] 检查发电机及启动电机
- [ ] 四轮定位检查及调整
- [ ] 动平衡检查

## 三、关键油液规格
| 油液类型 | 推荐规格 | 更换量参考 |
|---------|---------|-----------|
| 发动机机油 | SAE 5W-30（全合成，API SN或以上） | 约4.0-4.5升 |
| 冷却液 | 长效型乙二醇基冷却液（-35℃） | 约6.0-7.0升 |
| 制动液 | DOT 4 | 约1.0升 |
| 转向助力液 | ATF Dexron VI | 约1.0升 |
| 玻璃水 | 根据季节选择防冻型 | 约2.5升 |

| 部件名称 | 建议更换周期 | 备注 |

| 机油滤清器 | 每次换油时 | 必须同步更换 |
| 空气滤清器 | 每20000公里或2年 | 多尘环境缩短至10000公里 |
| 空调滤清器 | 每20000公里或1年 | 建议每年入夏前更换 |
| 燃油滤清器 | 每40000公里 | — |
| 火花塞 | 每40000公里 | 铱金火花塞可延长至60000公里 |

| 制动片（后） | 每40000-60000公里 | 视驾驶习惯而定 |

| 轮胎 | 每50000-60000公里或5年 | 以先到者为准 |

## 五、日常自检项目（建议每月一次）
- [ ] 检查机油液位（冷车状态下）
- [ ] 检查冷却液液位
- [ ] 检查制动液液位
- [ ] 检查玻璃水余量
- [ ] 检查轮胎气压及外观
- [ ] 检查所有灯光是否正常工作
- [ ] 检查仪表盘是否有故障灯亮起
- [ ] 检查雨刮器刮水效果
- [ ] 检查车身有无异常抖动或异响
---
## 六、各季节保养重点
### 春季
- 更换空调滤清器
- 检查空调制冷效果
- 清理散热器杂物
- 检查雨刮器
### 夏季
- 检查冷却系统
- 检查空调系统
- 注意轮胎气压（夏季可略低）
- 防止暴晒，使用遮阳挡
### 秋季
- 检查暖风系统
- 更换防冻型玻璃水
- 检查蓄电池
- 检查轮胎花纹深度
### 冬季
- 更换冬季轮胎（如需）
- 检查防冻液冰点
- 检查蓄电池性能
- 预热后再行驶
---
## 七、保养注意事项
1. **质保期内**：建议在荣威授权4S店进行保养，以免影响质保权益
2. **保养记录**：保留每次保养的工单和发票，作为后续维保凭证
3. **使用正品配件**：机油、滤清器等关键部件建议使用原厂或知名品牌产品
4. **不要超期保养**：即使行驶里程未到，时间到了也应及时保养
5. **留意保养提醒灯**：仪表盘保养提醒灯亮起后请尽快安排保养
6. **驾驶环境恶劣时**：如长期短途行驶、多尘环境、拥堵路况，建议适当缩短保养间隔
---
> **温馨提示**：以上保养周期和项目为通用建议，具体请以荣威D6随车《用户手册》和4S店实际要求为准。不同年款、配置可能存在差异，请结合实际情况灵活调整。

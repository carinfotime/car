# 星迈罗儿童乘车注意事项
  
更新时间：2026年07月19日18时39分07秒   

## 🚗 儿童乘车安全须知
### 1. 儿童安全座椅使用
- **必须使用**：12岁以下或身高不足150cm的儿童必须使用符合国家标准的儿童安全座椅
- **反向安装**：建议0-4岁（体重≤18kg）的儿童使用反向安装的安全座椅
- **正确固定**：确保安全座椅通过ISOFIX接口或安全带牢固固定在后排座位
### 2. 乘坐位置要求
- **后排乘坐**：儿童必须坐在后排座位，严禁坐在副驾驶位置
- **禁止怀抱**：切勿将儿童抱在怀中乘车，急刹车或碰撞时无法保护
- **车窗安全**：启用后窗儿童锁功能，防止儿童误开车窗或探出身体
### 3. 行车中注意事项
- **车门锁止**：启动车辆前确保所有车门已锁好，启用后门儿童安全锁
- **勿玩车窗**：教育孩子不要将头、手伸出窗外，避免夹伤或意外
- **勿扰驾驶**：提醒孩子不要干扰驾驶员操作（如触碰方向盘、挡位等）
- **车内物品**：避免放置尖锐、硬质物品在座椅周围，防止急刹时飞溅伤人
### 4. 上下车安全
- **右侧上下车**：尽量从右侧（靠路边一侧）上下车，避开行车道
- **确认后方来车**：下车前先观察后方是否有车辆或行人经过
- **家长陪同**：年幼儿童上下车需由成人协助开门和引导
### 5. 停车后注意事项
- **切勿独留车内**：任何时候都不要将儿童单独留在车内，尤其是夏季高温天气（车内温度可在短时间内升至致命水平）
- **检查后排**：熄火离车前务必检查后排座位，防止遗忘儿童
### ⚠️ 特别提醒
| 项目 | 说明 |
|------|------|
| 🔒 | 启用后门机械式儿童锁（位于车门侧面） |
| 🌡️ | 夏季车内温度可达60℃以上，切勿滞留 |
| 🧸 | 避免放置易滚动的小玩具在脚下空间 |
| 🚫 | 严禁让儿童站立或跪在座位上 |
> 💡 *星迈罗车主可参考车辆用户手册中关于“儿童安全”章节的详细说明。*

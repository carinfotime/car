## 领克09 空间参数介绍
  
更新时间：2026年07月16日10时11分29秒   

领克09作为一款中大型豪华SUV，基于SPA架构打造，在空间设计上兼顾了乘坐舒适性与储物实用性。以下是其核心空间参数详解：
### 一、车身尺寸
| 项目 | 参数 |
| :--- | :--- |
| 车长 | **5042 mm** |
| 车宽（不含后视镜） | **1977 mm** |
| 车高 | **1780 mm** |
| 轴距 | **2984 mm** |
> **解读：** 超过5米的车长与近3米的轴距，为车内纵向空间提供了充裕的基础，属于标准的中大型SUV尺寸范畴。
### 二、座椅布局
-   **座椅形式：** 提供 **6座（2+2+2）** 与 **7座（2+3+2）** 两种布局可选。
-   **座椅材质：** Nappa真皮包裹，前排座椅支持加热、通风及按摩功能。
### 三、乘坐空间表现
#### 1. 第一排（驾驶位/副驾驶）
-   **头部空间：** 约 **1拳至1拳半**（身高180cm体验者），视野开阔。
-   **腿部/肩部：** 横向宽度充裕，中央通道宽敞，无局促感。
#### 2. 第二排
-   **6座版：**
    -   独立座椅设计，配备扶手及电动调节。
    -   腿部空间：可调范围大，最宽可达约 **3拳以上**。
    -   头部空间：约 **1拳左右**。
    -   中央通道宽度适中，方便进入第三排。
-   **7座版：**
    -   三连座设计，支持前后滑动及靠背角度调节。
    -   腿部空间：常规状态下约 **1拳半至2拳**。
#### 3. 第三排
-   **定位：** “应急/短途乘坐”级别，适合儿童或身高170cm以下的成人短途出行。
-   **头部/腿部：**
    -   头部空间约 **2指至3指**（身高175cm体验者）。
    -   腿部需将第二排前移后获得约 **1拳左右余量**。
-   *注：第三排坐垫高度较低，对大腿支撑有限。*
### 四、储物与后备箱
#### （一）后备箱容积
| 座椅状态 | 容积 |
| :--- | :--- |
| （6座/7座）第三排立起 | **269 L / ~391 L（实测数据略有差异）** |
| （6座/7座）第三排放倒 | **862 L / ~862 L（形成平整大平台）** |
| （6座/7座）第二、三排放倒 | ≈1520 L |
> *注：具体数值因测量标准不同略有浮动。*
#### （二）车内储物细节
-   **门板储物格：** 可容纳500ml水瓶及杂物。
-   **中央扶手箱：**
    -   *MHEV车型*：深度适中，内置USB-C接口及无线充电板。
    -   *PHEV车型*：因电池布局影响，扶手箱深度略浅于MHEV版本。
-   **杯架数量：**
    -   前排杯架 ×2
    -   后排中央扶手杯架 ×2
        *（6座版第二排独立座椅各配一个杯架位；7座版中央扶手放下后含杯架×2）*
-   ***其他亮点***：
    1.  副驾手套箱带锁止功能且内部植绒处理。
    2.  后备箱地板下方设有隐藏式储物格（非全尺寸备胎车型）。
### ⚠️ PHEV版本特别说明
由于插电混动版搭载了动力电池组：
1.  后排地板中央隆起高度略高于MHEV版本（约增加20mm）。
2.  油箱容积从71L缩减至50L左右以容纳电池包。

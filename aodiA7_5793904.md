# 奥迪A7 空间参数介绍
  
更新时间：2026年07月15日09时08分32秒   

## 车身尺寸
| 参数 | 数值 |
|------|------|
| 车长 | 4,969 mm |
| 车宽（含后视镜） | 2,108 mm |
| 车宽（不含后视镜） | 1,908 mm |
| 车高 | 1,422 mm |
| 轴距 | 2,926 mm |
## 乘坐空间
- **前排头部空间**：约 980 mm
- **后排头部空间**：约 930 mm（因溜背造型，后排头部空间略有压缩）
- **前排腿部空间**：充裕，支持多向电动调节
- **后排腿部空间**：约 720 mm，中大型轿跑级别表现
- **后排肩部空间**：约 1,400 mm
- **座椅布局**：5座布局（后排中间位置适合短途乘坐）
## 行李厢空间
| 项目 | 数据 |
|------|------|
| **标准容积** | **535 L** |
| **最大扩展容积**（放倒后排座椅） | **1,390 L** |
| **行李厢开口宽度** | ~1,050 mm |
| **行李厢深度** | ~1,100 mm（标准状态） |
> ⚠️ A7采用掀背式尾门设计，开口较大，便于放置大件物品。
## Sportback（溜背）造型对空间的影响
- ✅ **优势**：掀背式尾门带来超大开口，装载便利性优于传统三厢轿车
- ❌ **劣势**：
   - C柱下压明显，后排头部空间相比A6L等传统轿车略小
   - 后排中间位置坐姿较局促，头顶易触顶
## 📐内部储物细节
- **前门板储物格**：可容纳500ml水瓶 +杂物
- **后门板储物格**：可容纳500ml水瓶
- **中央扶手箱容积**：约3.5L，带USB接口
- **手套箱容积**：约8L，带照明和锁止功能
- **杯架数量**：前排2个 +后排中央扶手2个
- **眼镜盒/卡片槽等小型储物位**: ✅配备

### 💡总结建议
奥迪A7作为一款四门轿跑（Sportback），在保持优雅溜背线条的同时提供了相对实用的日常空间表现：
- ✅后备箱容积在同级轿跑中表现优秀（535L）
- ✅掀背尾门极大提升装载便利性
- ⚠️后排头部受溜背影响较大，身高超过180cm的乘客建议试乘体验后再决定

更新时间：2026年07月17日13时15分46秒   

## 一、概述
理想 i8 作为一款增程式电动 SUV，其动力系统由 **1.5T 四缸增程器 + 前后双电机** 组成。保养策略需兼顾 **纯电驱动系统** 与 **燃油增程器** 两部分。建议严格按照官方《用户手册》及车机系统提示进行保养。

## 三、各系统详细保养说明
### 🔧 A. 增程器系统
-  使用正规加油站92号及以上无铅汽油。
-  长期不启动增程器时，建议每月主动运行一次（电量低于70%时启动“强制发电”模式），防止机油乳化及密封件老化。
-  机油更换时必须使用符合规格的全合成低粘度机油。
### 🔋 B. 高压电池与电驱
-  日常充电：建议保持 SOC（荷电状态）在20%~90%之间。
-  长期停放：电量保持在50%~60%，断开低压蓄电池负极。
-  避免频繁深度放电至10%以下。
-  定期检查高压线束及插接件是否松动、破损。
### 🛞 C. 底盘与轮胎
-  每月检查胎压：标准值约2.5~2.7 bar（视负载调整）。
-  每10,000 km进行轮胎换位（前后交叉对调）。
-  每年做一次四轮定位检测。
### 🧊 D. 冷却系统
-  增程器和电池共用一套电子水泵+低温散热回路。
-  冷却液为长效型，一般无需更换；若发现液位下降过快或颜色浑浊，需进店检修。
### 🛑 E. 制动系统
-  由于能量回收占主要制动力，机械刹车片磨损较慢。
-  每年检查刹车片厚度及刹车盘状态。
-  制动液含水量超过3%时必须更换。

## ⚠️五、常见注意事项
1. ❌ 不要在亏电状态下长期行驶——会严重增加油耗并影响 NVH。  
2. ❌ 不要自行拆卸高压部件——高压电高达400V以上，存在致命风险。  
3. ✅ 冬季低温时提前用 App 开启“电池预热”，提升续航和充电效率。  
4. ✅ 每次洗车后检查充电口是否有水渍残留。

## 📅六、推荐年度维保流程
text
第1年 /  10,000 km：
├─ ✔️更换机油 +机滤
├─ ✔️更换空调滤芯
├─ ✔️全车灯光/喇叭/雨刮检查
├─ ✔️底盘螺栓紧固 &悬挂检查
└─ ✔️胎压校准 +轮胎换位
第2年 /  20,000 km：
├─ ✔️重复第1年项目
├─ ✔️更换空气滤芯
├─ ✔️检测制动液含水量
└─ ✔️四轮定位检测
第3年 /  30,000 km：
├─ ✔️重复第1年项目
├─ ✔️更换火花塞(如适用)
├─ ✔️清洗节气门(如适用)
└─ ✔️全面高压系统绝缘检测
第4年 /  40,000 km：
├─ ✔️重复第1~2年项目
├─ ✔️更换减速器油(如适用)
└─ ✔️更换制动液(强制)

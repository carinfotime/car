# 御风EM26 驾驶注意事项
  
更新时间：2026年07月15日09时00分17秒   

## 一、启动与起步
1.  **钥匙上电**：插入钥匙后，先拧至“ON”档（仪表盘亮起），等待车辆完成自检（约2-3秒），待仪表盘上的“READY”指示灯亮起后，方可挂挡行驶。
2.  **挡位确认**：确保挡位处于 **N挡（空挡）** 再启动。起步时踩住刹车，将旋钮或挡杆切换至 **D挡（前进挡）**。
3.  **电子手刹释放**：系好安全带后，轻踩加速踏板或手动按下电子手刹开关解除制动。若未系安全带，部分车型可能无法自动解除手刹。
4.  **缓慢起步**：电机的瞬时扭矩较大，起步时请轻踩加速踏板，避免因动力过猛导致车辆窜动或失控。
## 二、行驶中注意事项
1.  **动能回收调节**：
    -   车辆通常配备多级动能回收（如强/弱/关闭）。在市区拥堵路况下建议使用强回收模式以增加续航；在高速或湿滑路面建议使用弱回收模式，避免突然的拖拽感影响操控。
2.  **车速控制**：
    -   该车型重心较高（载货状态），过弯时请提前减速，时速建议控制在30km/h以下，防止侧倾。
    -   高速行驶时（如80km/h以上），注意横风影响，握紧方向盘。
3.  **电量管理**：
    -   当电量低于20%时，车辆可能会限制功率输出（限速或加速无力），请提前规划充电。
    -   避免将电量完全耗尽再充电（即“亏电”状态），这会损伤电池寿命。
4.  **噪音提示**：
    -   EM26在低速行驶时会发出行人警示音（AVAS），此为正常现象。若该声音异常消失或过大，请及时检查。
## 三、充电与停放
1.  **充电规范**：
    -   **快充/慢充接口区分**：确认好直流快充口与交流慢充口位置。快充用于应急补电，慢充更有利于电池均衡保养。
    -   **充电环境**：避免在暴晒、暴雨或极寒环境下长时间充电。推荐在阴凉通风处进行。
2.  **长期停放**：
    -   若车辆需停放超过7天以上，建议保持电量在40%-60%之间。
    -   断开12V小电瓶负极线束（若长时间不用），防止小电瓶亏电导致无法启动高压系统。
## 四、装载与载重
1.  **严禁超载**：严格遵守车辆的额定载重质量。超载不仅会导致刹车距离变长、轮胎爆胎风险增加，还会严重损害电机和悬架系统。
2.  **货物固定**：货箱内的货物必须均匀摆放并固定牢固。急刹车时货物前冲可能撞击驾驶室隔板造成危险。
## 五、特殊路况应对
-   **雨天涉水**：纯电动车涉水深度一般不超过30cm（约半个车轮高度）。通过积水路段应低速匀速通过（时速5-10km/h），切勿冲水。涉水后应轻踩刹车排出刹车鼓内水分。
-   **冬季用车**：
    1.  冬季续航会明显衰减（约30%-50%），这是锂电池特性所致。
    2.  建议随用随充，利用刚停车电池尚有余温时立即充电效果更佳。
    3.  出发前可提前通过手机APP预约开启空调预热座舱（若支持）。
## ⚠️六、紧急情况处理
| 情况 | 操作 |
|------|------|
| “READY”灯不亮 | 检查是否挂入N挡；尝试重新上电；若仍无效请联系售后 |
| “乌龟灯”（功率限制）亮起 | 多为电池温度过高或电量过低引起。靠边停车休息10分钟 |
| “动力电池故障”报警 |立即靠边停车熄火并呼叫救援；严禁自行拆卸高压部件 |
| “绝缘故障”报警 |表明高压系统漏电风险；立即停车并远离车辆直至专业人员到达 |
> 📌 *提示：以上内容基于通用新能源物流车及御风系列常见特性整理。具体操作请以您所持车辆的《用户手册》为准。*

# 启辰星 操作指南
  
更新时间：2026年07月17日17时00分33秒   

## 一、车辆基本介绍
启辰星是东风启辰旗下的一款紧凑型SUV，搭载1.5T涡轮增压发动机，匹配7速双离合变速箱或6速手动变速箱，部分车型配备48V轻混系统。
---
## 二、启动与熄火
### 启动车辆
1. 携带智能钥匙进入车内
2. 确保挡位处于 **P挡**（驻车挡）或 **N挡**（空挡）
3. 踩下制动踏板
4. 按下 **一键启动按钮**（位于方向盘右侧）
5. 仪表盘指示灯自检完成后，发动机启动
### 熄火停车
1. 踩下制动踏板，将车辆完全停稳
2. 将挡位置于 **P挡**
3. 拉起 **电子手刹**
4. 再次按下 **一键启动按钮** 熄火
> **注意**：熄火前请确认所有车窗、天窗已关闭，大灯已关闭。
---
## 三、挡位操作
启辰星采用电子挡杆，挡位布局如下：
| 挡位 | 名称 | 用途 |
|------|------|------|
| P | 驻车挡 | 停车时使用，需车辆完全停稳后挂入 |
| R | 倒车挡 | 倒车时使用 |
| N | 空挡 | 临时停车或拖车时使用 |
| D | 前进挡 | 正常行驶时使用 |
| S | 运动模式 | 需要更强动力输出时使用 |
### 换挡操作
- 按住挡杆侧面的 **解锁按钮**，同时向前或向后推动挡杆
- 从D挡切换至S挡：在D挡位置向后拉动挡杆一次
---
## 四、仪表盘与信息显示
### 仪表盘主要显示内容
- **车速表**：当前行驶速度
- **转速表**：发动机转速
- **油量表**：燃油剩余量
- **水温表**：发动机冷却液温度
- **里程信息**：总里程、小计里程
- **挡位显示**：当前挡位
- **驾驶辅助信息**：车道偏离、盲区监测等
### 信息切换
- 通过方向盘右侧的 **多功能按键** 切换仪表盘显示信息
- 可查看：瞬时油耗、平均油耗、续航里程、胎压监测等
---
## 五、灯光操作
灯光控制拨杆位于方向盘左侧：
| 操作 | 功能 |
|------|------|
| 旋钮旋转至 **AUTO** | 自动大灯（根据光线自动开关） |
| 旋钮旋转至 **近光灯** | 手动开启近光灯 |
| 旋钮旋转至 **示宽灯** | 开启示宽灯 |
| 拨杆向前推 | 远光灯常亮 |
| 拨杆向后拉 | 远光灯闪烁（超车灯） |
| 拨杆上下拨动 | 左右转向灯 |
### 雾灯
- 在大灯开启状态下，旋转雾灯开关开启前/后雾灯
---
## 六、雨刮器操作
雨刮器控制拨杆位于方向盘右侧：
| 操作 | 功能 |
|------|------|
| 拨杆向上推 | 单次刮刷 |
| 拨杆向下推一挡 | 间歇刮刷（可调节速度） |
| 拨杆向下推二挡 | 低速连续刮刷 |
| 拨杆向下推三挡 | 高速连续刮刷 |
| 拨杆向身体方向拉 | 喷水+刮刷 |
---
## 七、空调与多媒体系统
启辰星配备中控大屏，空调和多媒体主要通过触控屏幕操作。
### 空调控制
- **温度调节**：点击屏幕上的温度加减按钮或滑动调节
- **风速调节**：调节风扇转速
- **模式选择**：吹面、吹脚、除雾等模式切换
- **A/C按钮**：开启/关闭空调压缩机（制冷）
- **内外循环**：切换车内/车外空气循环
- **自动模式（AUTO）**：设定目标温度后自动调节
### 多媒体系统
- **收音机**：通过中控屏选择电台
- **蓝牙连接**：设置 → 蓝牙 → 搜索设备 → 配对连接
- **USB/车载互联**：支持CarPlay、CarLife等手机互联功能
- **导航**：使用车载导航或手机投屏导航
---
## 八、驾驶辅助功能
### 定速巡航（如配备）
1. 车速达到40km/h以上
2. 按下方向盘上的 **定速巡航开启按钮**
3. 按下 **SET/-** 键设定当前速度为巡航速度
4. 使用 **RES/+** 和 **SET/-** 调节巡航速度
5. 踩刹车或按下 **CANCEL** 取消定速巡航
### 360°全景影像（如配备）
- 挂入 **R挡** 自动激活
- 也可通过中控屏幕手动开启
- 显示车辆周围环境，辅助泊车
### 其他辅助功能
- **胎压监测**：仪表盘查看各轮胎气压
- **车身稳定系统（ESP）**：默认开启，建议保持开启状态
- **上坡辅助**：坡道起步时自动保持制动2-3秒
---
## 九、加油指南
1. 车辆熄火，解锁车门
2. 油箱盖位于车辆 **左侧**
3. 按压油箱盖外侧，打开油箱盖
4. 逆时针旋转内盖，取下加油
5. 推荐使用 **92号及以上无铅汽油**
6. 加油完成后，顺时针拧紧内盖，关闭外盖
> **油箱容量**：约50升
---
## 十、日常维护与保养
### 定期保养周期
| 项目 | 周期 |
|------|------|
| 更换机油、机滤 | 每5000公里或6个月 |
| 更换空气滤芯 | 每10000公里 |
| 更换空调滤芯 | 每10000公里 |
| 更换火花塞 | 每30000公里 |
| 更换制动液 | 每40000公里 |
| 更换变速箱油 | 每60000公里 |
| 检查轮胎 | 每月检查胎压和磨损 |
### 日常检查
- **机油液位**：冷车状态下拔出机油尺检查
- **冷却液**：观察膨胀水箱液位在MIN-MAX之间
- **玻璃水**：液位不足时及时添加
- **轮胎气压**：建议2.3-2.5 bar（标准值以车门标签为准）
---
## 十一、应急处理
### 轮胎漏气/爆胎
1. 握紧方向盘，保持车辆直线行驶
2. 缓慢减速，不要急刹车
3. 将车辆停至安全平坦地带
4. 开启双闪警示灯
5. 放置三角警示牌（车后50-100米）
6. 使用备胎或呼叫道路救援
### 车辆无法启动
- 检查是否在P挡或N挡
- 检查刹车踏板是否踩下
- 检查智能钥匙是否在车内
- 检查蓄电池是否亏电（搭电启动）
### 发动机过热
1. 立即靠边停车，不熄火
2. 开启暖风至最大挡位辅助散热
3. 检查冷却液是否泄漏
4. 待温度下降后，谨慎打开膨胀水箱盖检查液位
---
## 十二、安全提示
1. **系好安全带**：所有乘员必须系好安全带
2. **儿童安全**：12岁以下儿童应使用儿童安全座椅
3. **禁止酒驾**：饮酒后严禁驾驶
4. **手机使用**：行驶中禁止手持手机
5. **车速控制**：遵守道路限速规定
6. **定期保养**：按时保养，确保车辆处于良好状态
7. **冬季预热**：寒冷天气启动后建议怠速预热30秒-1分钟再行驶

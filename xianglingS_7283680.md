# 祥菱S 技术参数介绍
  
更新时间：2026年07月14日16时11分14秒   

## 一、基本概况
祥菱S是福田汽车旗下的一款高端微卡车型，定位城市物流配送及短途运输市场，以高承载、高舒适性、低油耗为特点。
---
## 二、整车参数
| 项目 | 参数 |
|------|------|
| 车型类别 | 微卡（微型货车） |
| 驱动形式 | 4×2（后轮驱动） |
| 驾驶室形式 | 单排 / 排半 / 双排 |
| 额定载质量 | 约 **1.0 - 1.5吨**（具体视公告版本） |
| 总质量 | **2.5 - 3.5吨** |
| 整车尺寸（长×宽×高） | **5.0 - 6.0m × 1.7 - 1.85m × 2.0 - 2.2m**（因货箱形式而异） |
| 轴距 | **2750mm / 3000mm / 3200mm**（多种可选） |

## 三、发动机参数
祥菱S提供多种动力选择，主流配置如下：
### （1）东安DAM15KR / DAM16KR
| 项目 | DAM15KR | DAM16KR |
|------|---------|---------|
| **排量** | **1.5L** | **1.6L** |
| **最大功率** | **85kW（116马力）** | **90kW（122马力）** |
| **最大扭矩** | **150 N·m** | **158 N·m** |
| **燃油类型** | **汽油** | **汽油** |
| **排放标准** | **国六b** | **国六b** |
### （2）可选柴油动力
部分版本搭载福田环保动力4A2-82C50柴油发动机：
-  排量：2.3L
-  最大功率：60kW（82马力）
-  最大扭矩：220 N·m
-  排放标准：国五/国六
---
## 四、变速箱系统
-  变速箱型号：DAT18R / ML515
-  类型：手动机械式，5挡前进挡 +1倒挡
-  换挡方式：地排换挡
-  离合器形式：单片干式膜片弹簧离合器，液压操纵
---
## 五、底盘与悬挂系统
### （1）车架
-  车架形式：直通梁车架
-  车架宽度：约700mm
-  纵梁断面尺寸：140mm ×50mm ×4mm（加强型）
### （2）前桥
-  前桥载荷：约1.5T级
-  转向形式：齿轮齿条助力转向 / EPS电子助力转向（部分高配）
### （3）后桥
-  后桥载荷：约2.5T级
-  主减速比：4.875 /5.375（根据动力匹配）
-  后桥形式：冲压焊接桥壳
### （4）悬挂系统
-  前悬挂：钢板弹簧悬架，3片/4片不等长渐变刚度板簧
-  后悬挂：钢板弹簧悬架，主副簧结构，常见为3+2片或5+3片板簧组合，承载能力强

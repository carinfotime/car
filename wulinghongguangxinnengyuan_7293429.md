# 五菱宏光新能源（纯电版）保养指南
  
更新时间：2026年07月16日13时48分44秒   

## 一、概述
五菱宏光新能源车型（如宏光MINI EV、宏光PLUS EV等）采用纯电驱动系统，与传统燃油车相比，**无需更换机油、机滤**，保养项目主要集中在三电系统（电池、电机、电控）、底盘及易损件检查上。定期保养不仅能延长车辆寿命，更能保障行车安全。
---
## 二、保养周期总表
| 项目 | 周期 | 说明 |
|------|------|------|
| **首次保养** | **行驶5000公里或6个月**（以先到为准） | 免费检查 |
| **常规保养** | **每10000公里或12个月**（以先到为准） | 基础检查与更换 |
| **空调滤芯** | **每10000公里或12个月** | 视使用环境可缩短 |
| **减速器油（齿轮油）** | **首次：5000公里；后续：每20000公里或24个月** | 相当于燃油车的变速箱油 |
| **冷却液** | **每40000公里或48个月** | 检查液位及冰点 |
| **制动液** | **每20000公里或24个月** | 检测含水量 |
| **轮胎换位/动平衡** | **每10000公里** | 延长轮胎寿命 |
> ⚠️ *以上为厂家建议标准周期，若车辆长期在高温、多尘、潮湿或多雨地区使用，建议适当缩短部分项目的检查间隔。*

## 三、核心保养项目详解
### 1. 三电系统检查（每次保养必做）
-   **动力电池**
    -   外观检查：壳体有无变形、磕碰、漏液。
    -   连接线束：插头是否松动、氧化。
    -   SOC显示准确性：仪表电量显示是否与实际一致。
    -   均衡状态：必要时由服务站进行电池均衡维护。
-   **驱动电机**
    -   运行无异响、无异常振动。
    -   电机冷却系统管路无渗漏。
-   **电控单元（MCU/VCU）**
    -   OBD诊断仪读取故障码，清除历史偶发故障。
    -   软件版本检查，如有必要进行升级。
### 2. 减速器油更换
-   *注意：部分早期宏光MINI EV后桥为整体式，需更换专用齿轮油。*
-   **操作要点：**
    1.  举升车辆，找到减速器放油螺栓和加油螺栓。
    2.  放尽旧油，观察是否有金属碎屑（如有大量铁屑需进一步检修）。
    3.  按规定扭矩拧紧放油螺栓。
    4.  加注指定型号的电动车专用齿轮油至溢流孔流出为止。
### 3. 空调滤芯更换
-   *位置：副驾驶手套箱后方。*
-   *步骤：*
    1.  打开手套箱，按压两侧限位卡扣拆下手套箱。
    2.  取出旧滤芯（注意方向箭头）。
    3.  清洁滤芯槽内灰尘及杂物。
    4.  装入新滤芯（箭头朝上/朝车头方向），装回手套箱。
### 4. 制动系统检查
-   **制动片厚度：**前盘式刹车片厚度低于3mm时需更换；后鼓式刹车蹄片磨损至极限标记时更换。
-   **制动盘状态：**表面无深沟槽、无裂纹。
-   **制动液含水量：**使用检测笔测试，含水量大于3%时必须更换。
### 5. 轮胎与底盘
-   **胎压标准：**
    -   前轮：2.3 bar
    -   后轮：2.5 bar
        *（具体以驾驶侧B柱标签为准）*
-   **胎纹深度：低于1.6mm必须更换。**
-   **底盘螺栓扭矩抽查：悬架连接件、副车架螺栓。**

## 四、用户日常自检清单
建议每周或每次充电时花5分钟完成以下自检：
| ✅ | 项目 |
|---|---|
| ☐ | 【仪表】查看有无故障灯亮起 |
| ☐ | 【电量】确认续航里程与实际使用是否匹配正常 |
| ☐ | 【轮胎】目测胎压是否明显不足，有无扎钉鼓包 |
| ☐ | 【灯光】全车灯光功能正常 |
| ☐ | 【雨刮】刮水效果良好无异响 |
| ☐ | 【充电口】充电口盖开闭顺畅，内部无积水异物 |

## 五、常见问题与注意事项
1.  ❗️禁止私自改装电路
     加装大功率电器可能导致低压蓄电池亏电甚至损坏BMS系统。
2. ❗️长时间停放注意事项
     若超过15天不使用：
     -   保持电量在50%-80%之间
     -   断开12V低压蓄电池负极
     -   每月至少补充充电一次至满电再放电至50%
3. ❗️涉水行驶限制
     最大涉水深度不超过轮毂中心线。通过积水路段后应轻踩刹车排出水分。
4. ❗️充电注意事项
     -   优先使用家用慢充桩充电
     -   快充仅用于应急补充电量
     -   避免在电量低于10%时仍继续行驶再充电
5. ❗️质保政策提醒
     三电系统通常享有8年或12万公里的质保期。请务必在官方授权服务站进行定期保养并保留记录单。

## 🛠️ DIY小技巧——如何自己换空调滤芯？
> *适用于大多数五菱宏光MINI EV车型*
1️⃣准备工具：
新空调滤芯一个 + T20梅花扳手一把 + （可选）吸尘器
2️⃣操作流程：

①打开副驾手套箱 → 
②拆下手套箱两侧卡扣 →
③取下旧滤芯 →
④用吸尘器清理风道入口 →
⑤按箭头方向装入新滤芯 →
⑥装回手套箱 → ✅完成！

全程约10分钟即可搞定！

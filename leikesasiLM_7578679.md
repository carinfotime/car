# 雷克萨斯LM 驾驶注意事项
  
更新时间：2026年07月10日17时41分49秒   

## 一、启动与预热
1.  **智能钥匙**：确保智能钥匙在车内有效范围内。
2.  **启动步骤**：踩下制动踏板，按下“ENGINE START STOP”按钮启动发动机。
3.  **冷车预热**：建议在冷车启动后怠速运转30秒至1分钟，待发动机转速稳定后再行驶。避免立即大脚油门。
4.  **混动系统检查**：启动后注意观察仪表盘上的“READY”指示灯是否亮起，确认混合动力系统正常工作。
## 二、行驶中注意事项
### 1. 动力与操控
-   **平顺加速**：作为豪华MPV，应保持平顺的加速和减速，避免急加速和急刹车，以提升乘坐舒适性并降低油耗。
-   **能量回收**：熟悉油门踏板的“能量回收”感觉。松开油门时会有轻微的拖拽感，这是混动系统在回收能量为电池充电。
-   **EV模式**：在短途低速行驶（如小区、地库）时，可手动切换至EV（纯电）模式，实现零排放、低噪音行驶。注意电池电量不足时会自动退出。
### 2. 视野与盲区
-   **车身尺寸**：LM车身较长且宽（车长约5.1米），转弯半径较大。在狭窄道路、停车场或掉头时需预留充足空间。
-   **盲区监测**：充分利用车辆的盲点监测系统（BSM），变道前务必观察后视镜及BSM指示灯。
-   **全景影像**：倒车或通过狭窄区域时，务必开启360度全景监视系统，辅助判断车辆周围障碍物。
### 3. 特殊路况
-   **湿滑路面**：雷克萨斯LM整备质量较重（约2.3吨），湿滑路面制动距离会相应增加。请降低车速，避免急转向。
-   **颠簸路段**：虽然配备AVS自适应可变悬架，但在连续颠簸路面仍建议低速通过，以保护悬挂系统和车内乘客舒适度。
## 三、停车与驻车
1.  **电子手刹**：停车后确保拉起电子手刹（EPB）。挂入P挡后仪表盘会有相应提示。
2.  **自动驻车（Auto Hold）**：等红灯时可开启Auto Hold功能，解放右脚。轻踩油门即可解除。
3.  **侧滑门使用**：
    -   电动侧滑门可通过车内按键、外部门把手或遥控钥匙操作。
    -   开门前务必确认车门周围没有行人或障碍物。
    -   不要在车辆行驶中操作侧滑门开关。
4.  **后备箱开启**：电动尾门支持脚踢感应开启。请确保钥匙在身上，用脚在保险杠下方扫过即可。
## 四、安全辅助系统使用
1.  **自适应巡航（ACC）**：
    -   设定好速度和跟车距离后，车辆会自动跟随前车加减速甚至刹停。
    -   驾驶员仍需时刻关注路况并准备随时接管控制权。
2.  **车道保持辅助（LTA）**：
    -   功能启用时车辆会轻微修正方向盘以保持在车道中央行驶。
    -   这是辅助功能而非自动驾驶。双手请勿离开方向盘太久。
3.  **预碰撞安全系统（PCS）**:当系统判断有碰撞风险时会发出警报并辅助制动。切勿故意测试此功能。
## 五、日常维护与检查
|检查项目|频率|备注|
|---|---|---|
|轮胎气压|每月一次|标准胎压通常标注在驾驶员侧B柱标签上|
|机油液位|每月一次/长途前|冷车状态下检查|
|冷却液液位|每月一次|需在发动机冷却后打开水箱盖|
|玻璃水液位|按需添加|-|
|混动电池散热口|每季度清理一次|位于后排座椅下方两侧通风口处|
## ⚠️特别提醒
1.  雷克萨斯LM的离地间隙较低（约150mm），上下马路牙子或通过较高减速带时要格外小心底盘刮擦。
2.  后排“老板座”（Ottoman座椅）的按摩/通风/加热功能建议在发动机运转状态下使用以避免蓄电池亏电。
3.  如果长时间停放超过一个月：
    -   建议将12V蓄电池负极断开或连接充电器维护
    -   混动动力电池应保持约50%-70%电量存放
4.  加油时请使用92号及以上无铅汽油即可满足要求；但若追求最佳性能推荐95号汽油。
> *以上内容基于通用车型特性整理而成；具体操作请以您所购车辆的《用户手册》为准。*

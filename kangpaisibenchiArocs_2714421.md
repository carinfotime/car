# 康派斯奔驰Arocs 空间参数介绍
  
更新时间：2026年07月14日08时06分09秒   

## 整车尺寸参数
| 项目 | 参数 |
|------|------|
| 整车长度 | 约 5995 mm |
| 整车宽度 | 约 2500 mm |
| 整车高度 | 约 3500 mm（含空调） |
| 轴距 | 约 3650 mm |
## 驾驶室空间
- **驾驶室类型**：奔驰Arocs原厂双排座驾驶室
- **前排空间**：主副驾独立空气悬挂座椅，头部空间充裕，腿部空间宽敞
- **后排卧铺**：设有单层或双层卧铺，宽度约700mm，可供休息使用
- **储物空间**：驾驶室顶部及门板多处储物格，方便存放随车物品
## 生活舱（厢体）空间
### 内部尺寸
| 项目 | 参数 |
|------|------|
| 厢体长度 | 约4500 mm |
| 厢体宽度 | 约2300 mm |
| 厢体高度 | 约2100 mm（内部净高） |
### 功能分区布局
#### 🛏️ 卧室区
- **主卧床铺**：横置双人床，尺寸约 **2000×1500 mm**
- **额头床**（如有）：位于驾驶室上方，尺寸约 **2000×1200 mm**
- **上下铺/卡座变床**：部分车型配备对坐卡座可拼床，尺寸约 **1900×1100 mm**
#### 🍳 厨房区
- **操作台面长度**：约1200~1500 mm
- **洗菜池**：不锈钢嵌入式水槽
- **炉灶**：电磁炉或燃气灶（2~3眼）
- **储物柜**：上下组合柜，容积充裕，可存放餐具及食材
- **冰箱**：12V/220V车载冰箱，容积约80~140L
#### 🚿 卫生间区域
- **卫生间形式**：一体式干湿分离卫生间（部分车型）
- **淋浴间尺寸**：约800×800 mm
- **马桶**：旋转式或固定式盒式马桶
- **洗手台盆**：陶瓷或不锈钢材质
- **换气扇/天窗**：顶部换气扇保证通风
#### 🛋️ 会客/休闲区
- **卡座沙发**：对坐式卡座（4人位），座椅宽度适中
- **升降桌**：可升降调节，降下后可拼成临时床位
- **电视挂架位置**：预留电视安装位及电源接口
### 📦 储物空间总览
| 位置 |  说明 |
|------|-------|
|  外部储物舱 |  车侧及尾部设有贯通式或独立储物舱，容积可达1000L以上 |
|  吊柜系统 |  生活区顶部环绕吊柜，适合放置衣物、日用品 |
|  底置储物箱 |  底盘下方设置工具箱或行李舱 |
|  衣柜/鞋柜 |  部分车型配备独立衣柜或鞋柜 |
## 📐 有效载荷与空间利用率
- **整备质量**: ~4500 kg  
- **最大总质量**: ~6500 kg（依底盘配置而定）  
- **有效载荷**: ~2000 kg  
> *注:实际载荷受选装配置、水箱水量、油箱油量等因素影响。*

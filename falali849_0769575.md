# 法拉利849 动力参数介绍
  
更新时间：2026年07月11日18时36分33秒   

## 概述
法拉利849（Ferrari 849）是法拉利品牌旗下搭载**4.0L V8双涡轮增压发动机**的高性能跑车，融合了赛道技术与日常驾驶舒适性，代表了法拉利在中置引擎跑车领域的最新成果。
---
## 发动机参数
| 项目 | 参数 |
|------|------|
| 发动机类型 | V8 双涡轮增压 |
| 排量 | 3,990 cc |
| 气缸布局 | 90° V8 |
| 进气形式 | 双涡轮增压（并联） |
| 压缩比 | 9.5 : 1 |
| 最大功率 | **720 CV**（约 530 kW）@ 7,500 rpm |
| 最大扭矩 | **850 N·m** @ 3,000 – 5,500 rpm |
| 红线转速 | 8,000 rpm |
| 燃油系统 | 缸内直喷 |
---
## 传动系统
| 项目 | 参数 |

| 驱动方式 | 中置后驱（MR） |

## 技术亮点
- **双涡管涡轮技术**：有效降低涡轮迟滞，提升低扭响应
- **可变增压控制**：根据驾驶模式动态调整涡轮压力
- **主动式排气阀**：可切换运动与静音模式，兼顾声浪与法规
- **冷却系统优化**：三回路散热设计，确保赛道持久高负载运行
- **ECU 调校**：支持 Race / Sport / Wet / CT Off 四种动力输出模式
---
## 总结
法拉利 849 凭借 **4.0L V8 双涡轮增压发动机** 输出 **720 CV / 850 N·m** 的强悍动力，配合 **8速双离合变速箱** 与轻量化车身，实现了 **0–100 km/h 加速 2.9 秒**、极速 **340 km/h+** 的顶级性能表现，是一款兼具赛道锋芒与公路巡航能力的超级跑车。

# 长城炮保养指南
  
更新时间：2026年07月17日17时07分23秒   

> 适用车型：长城炮（乘用版、商用版、越野版）
---
## 一、保养周期总览
| 项目 | 周期 | 备注 |
|------|------|------|
| 机油 & 机滤 | 每 5,000 km 或 6 个月 | 优先以先到为准 |
| 空气滤芯 | 每 10,000 km | 恶劣路况可缩短至 5,000 km |
| 空调滤芯 | 每 10,000 km | 建议每年换季时更换 |
| 燃油滤芯 | 每 20,000 km | 柴油版需更注意 |
| 火花塞 | 每 30,000 km | 汽油版 |
| 变速箱油 | 每 60,000 km | 自动/手动均需检查 |
| 前后桥油 | 每 40,000 km | 四驱版需检查前桥 |
| 分动箱油 | 每 60,000 km | 四驱版 |
| 刹车油 | 每 40,000 km 或 2 年 | 检测含水量 |
| 防冻液 | 每 60,000 km 或 3 年 | 检查冰点与液位 |
| 转向助力油 | 每 60,000 km | 电子助力版无需更换 |
| 正时皮带/链条 | 每 100,000 km 检查 | 链条版免维护，皮带版需更换 |
| 轮胎换位 | 每 10,000 km | 交叉换位 |
| 刹车片检查 | 每 10,000 km | 厚度低于 3mm 需更换 |
---
## 二、机油选择
| 发动机类型 | 推荐粘度 | 标准 |
|-----------|---------|------|
| 2.0T 汽油（GW4C20B） | 5W-30 / 5W-40 | API SN 及以上 / ACEA C3 |
| 2.0T 柴油（GW4D20M） | 5W-30 / 5W-40 | API CK-4 / ACEA E7 |
- **容量**：汽油版约 5.5L，柴油版约 6L
- **建议品牌**：壳牌、美孚、嘉实多、长城润滑油
---
## 三、轮胎保养
| 项目 | 标准 |
|------|------|
| 标准胎压（空载） | 前 2.3 bar / 后 2.5 bar |
| 标准胎压（满载） | 前 2.5 bar / 后 2.8 bar |
| 轮胎规格（乘用版） | 265/65 R18 |
| 轮胎规格（商用版） | 245/70 R17 |
| 备胎胎压 | 4.2 bar |
- 每月检查一次胎压，包括备胎
- 每 10,000 km 做轮胎动平衡与四轮定位
- 越野后检查胎面是否有划伤、鼓包
---
## 四、柴油版（炮商用/越野柴油）特别保养
> 柴油版车主需额外关注以下项目：
| 项目 | 周期 | 说明 |
|------|------|------|
| 柴油滤芯（粗滤+精滤） | 每 10,000 km | 必须使用合格柴油 |
| 尿素液位检查 | 每 5,000 km | 使用专用车用尿素 |
| DPF 再生 | 自动进行 | 避免长时间怠速 |
| 油水分离器放水 | 每 2,000 km 或加油后 | 防止水入高压油泵 |
| 高压共轨油管检查 | 每 40,000 km | 检查渗漏 |
---
## 五、四驱系统保养（四驱版）
| 项目 | 周期 | 说明 |
|------|------|------|
| 前桥差速器油 | 每 40,000 km | 使用 GL-5 80W-90 |
| 后桥差速器油 | 每 40,000 km | 使用 GL-5 80W-90 |
| 分动箱油 | 每 60,000 km | 使用 ATF 或专用油 |
| 前轴半轴防尘套 | 每 10,000 km 检查 | 破损需立即更换 |
| 电子差速锁功能测试 | 每 10,000 km | 越野版 |
---
## 六、刹车系统
| 项目 | 标准 |
|------|------|
| 前刹车片厚度 | 新件约 12mm，极限 3mm |
| 后刹车片厚度 | 新件约 10mm，极限 3mm |
| 刹车盘磨损极限 | 单面磨损 ≤ 2mm |
| 刹车油含水量 | 超过 3% 必须更换 |
- 刹车油建议使用 DOT4 级别
- 每 10,000 km 检查刹车管路有无渗漏
---
## 七、常见问题与注意事项
1. **DPF 再生提醒**：若仪表提示"颗粒捕集器再生中"，请保持中高速行驶（车速 > 40 km/h）约 20 分钟，不要熄火
2. **尿素结晶**：冬季注意使用低温尿素（-11°C 以下用低温型）
3. **后桥异响**：检查后桥油位，四驱版注意差速锁是否完全分离
4. **涡轮增压器**：熄火前建议怠速 1-2 分钟再熄火（尤其长途或越野后）
5. **货箱保养**：定期清洁货箱，防止锈蚀，建议加装货箱宝
6. **涉水后**：检查前后桥、分动箱、变速箱是否进水
---
## 八、保养建议总结
| 里程 (km) | 必做项目 | 建议检查项目 |
|-----------|---------|-------------|
| 5,000 | 机油、机滤 | 全车油液液位、轮胎胎压 |
| 10,000 | 机油、机滤、空滤、空调滤 | 刹车片厚度、轮胎换位 |
| 20,000 | 机油、机滤、燃油滤 | 四轮定位、底盘检查 |
| 40,000 | 机油、机滤、空滤、空调滤、刹车油、前后桥油 | 火花塞、正时皮带 |
| 60,000 | 机油、机滤、变速箱油、分动箱油、防冻液 | 全车皮带、张紧轮 |
| 100,000 | 全车油液更换、正时皮带（如适用） | 发动机机脚、减震器 |
---
> **提示**：以上数据以长城炮官方说明书为基准，实际保养请结合车辆使用环境（如重载、越野、高寒、多尘等）适当缩短周期。建议使用长城官方配件或同等品质品牌件。

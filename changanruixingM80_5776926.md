## 二、各阶段详细保养内容
### 1. 首保（2,500km 或 2个月）
首保免费，必须按时到4S店进行，否则可能失去保修资格。
- 更换发动机机油及机油滤清器
- 检查并紧固底盘螺栓、螺母
- 检查轮胎气压及磨损情况
- 检查制动系统工作状况
- 检查各油液液位（冷却液、制动液、转向助力液）
- 检查灯光、喇叭、雨刮器
- 检查电瓶连接及状态
- 清洁空气滤芯
### 2. 常规保养（每5,000km 或 6个月）
- 更换机油、机油滤清器
- 检查并补充冷却液
- 检查制动液液位及品质
- 检查转向助力液
- 检查轮胎磨损及胎压（含备胎）
- 检查全车灯光及信号系统
- 检查底盘及悬挂系统
- 检查传动轴及万向节
- 润滑车门铰链、锁扣
### 3. 中保养（每10,000km 或 12个月）
在常规保养基础上增加：
- 更换空气滤清器滤芯
- 更换空调滤清器滤芯（如有）
- 检查火花塞状况
- 检查刹车片厚度
- 检查刹车盘磨损
- 检查发动机皮带张紧度及磨损
- 检查排气管及三元催化器
- 检查转向系统间隙
### 4. 大保养（每20,000km 或 24个月）
在中保养基础上增加：
- 更换燃油滤清器
- 更换变速箱油（手动变速箱）
- 更换后桥齿轮油
- 清洗节气门
- 检查并调整气门间隙（如有）
- 检查发动机正时系统
- 检查空调系统制冷剂压力
### 5. 全面保养（每40,000km 或 48个月）
在大保养基础上增加：
- 更换制动液
- 更换发动机冷却液
- 更换正时皮带及张紧轮（如为皮带正时）
- 更换火花塞
- 检查或更换电瓶
- 检查全车橡胶管路老化情况
- 检查燃油管路及接头
- 检查并清洁EGR阀（如有）

## 三、油液规格及用量
| 油液类型 | 推荐规格 | 用量参考 |
|---------|---------|---------|
| 发动机机油 | SAE 5W-30 / 10W-40（API SL以上） | 约 3.5~4.0 L |
| 变速箱油 | GL-4 75W-90 | 约 2.0~2.5 L |
| 后桥齿轮油 | GL-5 80W-90 | 约 1.5~2.0 L |
| 制动液 | DOT3 或 DOT4 | 约 1.0 L |
| 冷却液 | 乙二醇型长效防冻液 | 约 6.0~7.0 L |
| 转向助力液 | ATF DEXRON III 或专用助力油 | 约 0.8~1.0 L |
| 空调制冷剂 | R134a | 约 500~600g |

## 四、日常检查项目（车主自检，建议每周一次）
- [ ] 检查机油液位（冷车时，在上下刻度之间）
- [ ] 检查冷却液液位（在MIN~MAX之间）
- [ ] 检查制动液液位
- [ ] 检查轮胎胎压（含备胎），标准值：前轮 2.2~2.4 bar，后轮 2.4~2.8 bar（视载重调整）
- [ ] 检查轮胎外观有无鼓包、裂纹、异常磨损
- [ ] 检查全车灯光（大灯、转向灯、刹车灯、雾灯、倒车灯）
- [ ] 检查雨刮器及喷水嘴
- [ ] 检查仪表盘是否有故障灯亮起
- [ ] 听发动机怠速及行驶时有无异响
- [ ] 检查刹车踏板行程及制动效果

## 六、保养注意事项
### 1. 机油选择
- 建议使用 **5W-30** 四季通用机油
- 高里程车辆（8万公里以上）可改用 **10W-40**
- 务必使用正品机油，劣质机油会加速发动机磨损
### 2. 防冻液
- 不同颜色/品牌的防冻液不可混加
- 更换时需彻底清洗冷却系统
- 建议使用长安原厂防冻液或同规格产品
### 3. 制动系统
- 刹车片厚度低于3mm时必须更换
- 更换刹车片后需进行磨合（200km内避免急刹）
- 制动液吸水性极强，建议每2年更换一次
### 4. 变速箱油
- 手动变速箱油建议每20,000km更换
- 若挂挡生涩，应提前检查更换
- 使用错误规格的油液会导致换挡困难
### 5. 轮胎
- 花纹深度低于1.6mm必须更换
- 建议每10,000km进行轮胎换位（前后交叉）
- 载重时后轮胎压适当调高至2.6~2.8 bar
### 6. 正时皮带
- 如发动机为正时皮带（非链条），务必在40,000~60,000km更换
- 正时皮带断裂会导致气门顶坏，维修费用高昂
- 更换时建议同时更换张紧轮和水泵

## 七、常见故障及预防
| 故障现象 | 可能原因 | 预防/处理方法 |
|---------|---------|-------------|
| 发动机启动困难 | 电瓶亏电/火花塞老化/燃油系统问题 | 定期检查电瓶、按时更换火花塞和燃油滤清器 |
| 发动机抖动 | 火花塞积碳/节气门脏污/油品差 | 定期清洗节气门、使用正规加油站燃油 |
| 油耗升高 | 空气滤芯堵塞/轮胎气压不足/机油粘度过高 | 按时保养、保持标准胎压 |
| 刹车偏软 | 制动液老化/刹车片磨损/制动系统有空气 | 及时更换制动液、检查刹车片 |
| 空调制冷差 | 制冷剂泄漏/冷凝器脏污/空调滤芯堵塞 | 定期清洗冷凝器、更换空调滤芯 |
| 挂挡困难 | 变速箱油老化/离合器故障 | 按时更换变速箱油、检查离合器 |
| 底盘异响 | 悬挂胶套老化/球头磨损/减震器漏油 | 定期检查底盘、及时更换老化部件 |

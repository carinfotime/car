## 二、各阶段保养项目明细
### 🔹 首次保养（3个月 / 5000公里）
- [x] 更换发动机机油
- [x] 更换机油滤清器
- [x] 检查冷却液液位及状态
- [x] 检查制动液液位
- [x] 检查玻璃清洗液液位
- [x] 检查轮胎气压及磨损情况
- [x] 检查灯光系统功能
- [x] 检查雨刮器工作状态
- [x] OBD系统检测与故障码读取
- [x] **动力电池系统健康度检测**
- [x] **高压线束及接口检查**
### 🔹 A类常规保养（每6个月 / 10000公里）
包含首次保养全部项目，另增加：
- [ ] **更换空调滤清器**（建议每年至少更换1次）
- [ ] **检查并清洁动力电池散热通道**
- [ ] **检查充电口及充电线束**
- [ ] **检查驱动电机减速器油液**
- [ ] **底盘螺栓扭矩复检**
- [ ] **制动片厚度测量**
### 🔹 B类大保养（每24个月 / 40000公里）
包含A类全部项目，另增加：
- [ ] **更换制动液**（建议2年/40000公里）
- [ ] **更换冷却液**（建议2年/40000公里）
- [ ] **更换驱动电机减速器油**
- [ ] **更换火花塞**（如为混动发动机配置）
- [ ] **清洗节气门及进气道**
- [ ] **四轮换位及动平衡**
- [ ] **动力电池均衡维护检测**
- [ ] **高压系统绝缘检测**

## 三、关键部件专项保养说明
### ⚡️ 动力电池系统
| 项目 | 建议周期 |
|------|---------|
| SOC校准（满充满放） |  每3个月至少1次 |
| BMS系统升级检查 |  每次进店时查询 |
| PACK气密性检测 |  每年1次 |
| DC-DC转换器状态检测 |  每次常规保养 |
> 💡 *长期停放超过15天，建议保持电量在40%~60%之间；长期停放超过30天，建议进行一次慢充满电循环。*

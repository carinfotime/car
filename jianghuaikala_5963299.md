# 江淮卡拉（JAC Kala）操作指南
  
更新时间：2026年07月14日15时32分31秒   

## 一、车辆基本介绍
江淮卡拉是一款轻型商用车/微卡，主要用于城市物流、短途运输等场景。本指南涵盖日常驾驶、基础操作及维护要点。
## 二、驾驶前检查
1. **外观检查**：绕车一周，检查轮胎气压、外观有无损伤、灯光是否正常。
2. **油液检查**：
   - 机油：拔出机油尺，油位应在上下刻度之间。
   - 冷却液：观察副水箱液位，应在MIN与MAX之间。
   - 制动液：液位应处于正常范围。
   - 玻璃水：确保充足。
3. **轮胎检查**：
   - 胎压建议值（参考）：前轮3.5-4.0 bar，后轮4.0-5.0 bar（视载重调整）。
   - 轮胎花纹深度不低于1.6mm。
4. **灯光与信号**：开启大灯、转向灯、刹车灯，确认工作正常。
## 三、启动与熄火
### 启动步骤
1. 插入钥匙，拧至ON档，等待仪表盘自检完成（约2-3秒）。
2. 确认挡位在空挡（手动挡车型）。
3. 踩下离合器踏板（手动挡）或刹车踏板（部分自动挡/AMT车型）。
4. 将钥匙拧至START档启动发动机。
5. 启动后松开钥匙，发动机怠速运转30秒左右再起步。
### 熄火步骤
1. 车辆停稳后，拉紧手刹。
2. 将挡位置于空挡。
3. 关闭所有用电设备（空调、大灯等）。
4. 将钥匙拧回OFF档并拔出。
## 四、起步与驾驶
### 手动挡车型
1. **起步**：
   - 踩下离合器踏板，挂入1挡。
   - 缓慢松开离合器同时轻踩油门，车辆平稳起步。
2. **换挡时机参考**：
   - 1→2挡：15-20 km/h
   - 2→3挡：25-35 km/h
   - 3→4挡：40-50 km/h
   - 4→5挡：55-65 km/h
3. **降档**：根据车速和发动机转速适时降档，避免拖挡行驶。
### AMT/自动挡车型
1. **D挡模式**：挂入D挡后松开刹车即可行驶。
2. **手动模式**：部分车型支持手动加减档，用于爬坡或重载工况。
## ⚠️五、重要注意事项
| ⚠️ | **冷车启动后不要立即高速行驶** |
|----|---------------------------|
|     | **重载时适当降低车速，避免急刹车** |
|     | **下长坡时利用发动机制动（低档位），避免长时间踩刹车导致制动过热失效** |
|     | **雨天/湿滑路面保持更大跟车距离** |
## ⚠️六、特殊路况操作建议
### （一）重载行驶
- **胎压适当提高至推荐上限。**
- **起步用1档，避免高转速半离合。**
- **转弯提前减速。**
- **保持安全车距。**
### （二）坡道起步
#### （a）有坡道辅助功能车型：
松开刹车后有约2秒驻车时间，快速踩油门即可。
#### （b）无辅助功能车型：
拉紧手刹 →挂入1档 →慢抬离合至半联动点 →感觉车身抖动时松手刹并加油门 →完全松开离合。
### （三）涉水路段
水深不超过排气管高度时方可通行。通过后应轻踩几次刹车以恢复制动性能。

## ⚠️九、常见应急处理
### （一）轮胎爆裂
紧握方向盘 →不要急刹车 →缓慢减速 →靠边停车 →放置三角警示牌(车后50米以上) →更换备胎或呼叫救援。
### （二）发动机过热
靠边停车但不熄火 →打开引擎盖散热(注意防烫) →检查冷却液是否泄漏 →待温度下降后再决定是否继续行驶。
### （三）制动失灵
连续多次踩刹车尝试恢复压力 →降档利用发动机制动(逐级降档) →拉手刹(注意不能一次拉死) →寻找避险车道或障碍物减速。

# 皇冠陆放（Crown Kluger）保养指南
  
更新时间：2026年07月18日08时02分21秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 更换机油及机滤 | 每10,000公里或12个月（先到为准） |
| 空气滤清器 | 每20,000公里检查，每40,000公里更换 |
| 空调滤清器 | 每10,000公里检查，每20,000公里更换 |
| 制动液 | 每40,000公里或24个月更换 |
| 火花塞 | 每100,000公里更换 |
| 变速箱油（自动） | 每60,000-80,000公里检查，必要时更换 |
| 冷却液 | 首次160,000公里或96个月，之后每80,000公里或48个月 |
| 刹车片/刹车盘 | 每次保养检查，视磨损情况更换 |
| 轮胎换位/动平衡 | 每10,000公里 |
| 燃油滤清器 | 每80,000公里 |

## 二、常规保养项目详解
### 🔧 A类保养（10,000km / 12个月）
- ✅ **更换机油**：推荐使用丰田纯牌0W-16或0W-20全合成机油
- ✅ **更换机油滤清器**
- ✅ **检查并补充各油液**：制动液、冷却液、玻璃水
- ✅ **检查轮胎气压及磨损情况**
- ✅ **检查灯光系统**：大灯、转向灯、刹车灯等
- ✅ **检查底盘及悬挂系统**
- ✅ **清洁或更换空调滤芯**（视情况）
- ✅ **电脑检测**：读取故障码，确认系统正常
### 🔧 B类保养（20,000km / 24个月）
在A类基础上增加：
- ✅ **更换空气滤清器**
- ✅ **更换空调滤清器**
- ✅ **轮胎换位及动平衡**
- ✅ **检查刹车片厚度**
- ✅ **检查驱动皮带状态**
- ✅ **润滑车门铰链及天窗导轨**
### 🔧 C类保养（40,000km / 48个月）
在B类基础上增加：
- ✅ **更换制动液**
- ✅ **清洗节气门**（视积碳情况）
- ✅ **深度检查底盘螺栓紧固度**
- ✅ **四轮定位检测**（如有跑偏现象）
### 🔧 D类保养（80,000km / 96个月）
在C类基础上增加：
- ✅ **更换燃油滤清器**
- ✅ **检查/更换变速箱油**（视使用工况）
- ✅ **检查正时链条状态**（通常免维护，但需确认张紧度）
- ✅ **全面清洗发动机积碳**

## 三、油液规格与用量
### 🛢️发动机机油
|项目|规格|
|---|---|
|推荐粘度等级|0W-16 / SAE OW‑20|
|标准容量（含机滤）|约4.5~5.0升|
|质量标准|丰田纯牌 / API SP / ILSAC GF‑6|
### 🧊冷却液
> Toyota Super Long Life Coolant (SLLC) — Pink色
> -  标准容量：约7.5~8.0升
> -  严禁混用不同颜色冷却液
### 🛑制动液
> DOT3或DOT4 — Toyota Brake Fluid or equivalent
> -  建议使用丰田原厂制动液
> -  标准容量：约0.5~0.7升

## ⚠️四、关键注意事项
1. **[混动车型特别提醒]**  
   皇冠陆放搭载丰田THS混动系统：
   - ❗高压电池散热通风口位于后排座椅侧方，务必定期清理灰尘堵塞。
   - ❗切勿自行触碰橙色高压线束。
   - ❗逆变器冷却系统需专用冷却液，不可与发动机冷却液混用。
2. **[机油选择]**  
   强烈建议使用低粘度全合成机油(0W‑16/0W‑20)，高粘度机油会增加油耗并影响动力响应。
3. **[燃油标号]**  
   推荐使用92号及以上无铅汽油。若长期加注低标号燃油可能导致爆震。
4. **[GPF颗粒捕捉器]**  
   部分国六B车型配备GPF：
   - ❗必须使用低灰分机油(API SP)。
   - ❗避免长期短途行驶，定期高速行驶有助于GPF再生。
5. **[四驱系统]**  
   若为E-Four电子四驱版本：
   - ❗后桥差速器油建议每40,000km检查。
   - ❗轮胎磨损差异过大会损坏耦合器。

## 📋五、日常自查清单（车主可自行操作）
每天/每次加油前：
1. [ ] Check Engine灯是否异常点亮
2. [ ]仪表盘是否有警告提示信息
3. [ ]轮胎外观是否正常、胎压是否过低
4. [ ]车身周围有无漏油痕迹
每月一次：
1. [ ]打开引擎盖——目测各油壶刻度是否在MIN-MAX之间
2. [ ]玻璃水是否充足——及时补充专用玻璃水(冬季注意防冻)
3. [ ]雨刮片刮拭效果——如有异响/条纹及时更换
4. [ ]车内空调出风是否正常——异味需清洗蒸发箱
---
## 📌六、常见故障提示解读
仪表盘出现以下提示时请及时进店检修：
⚠️ "Check Engine" → OBD检测 →可能涉及氧传感器/点火线圈/混合气过稀等  
⚠️ "HV Battery Cooling" →高压电池散热堵塞→清理通风口  
⚠️ "Tire Pressure Warning" →胎压监测报警→补气至标准值(前230kPa /后230kPa)  
⚠️ "Maintenance Required Soon" →接近保养里程→预约进店

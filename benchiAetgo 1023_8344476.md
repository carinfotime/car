# 奔驰 Actros 1023 内饰参数介绍
## 一、驾驶室整体布局
- **驾驶室类型**：StreamSpace / BigSpace / GigaSpace（根据配置可选）
- **驾驶室宽度**：2.5 米（宽体设计）
- **内部高度**：1.92 米（BigSpace 版本可达 2.13 米）
- **平地板设计**：驾驶室中部完全平坦，无发动机鼓包，方便车内移动
## 二、驾驶员工作区
### 座椅
- **座椅类型**：空气悬浮式驾驶员座椅
- **调节功能**：
  - 前后滑动调节
  - 高度调节
  - 倾斜角度调节
  - 腰部支撑调节（气动）
  - 座椅加热（选装）
- **座椅材质**：高级织物或仿皮（选装真皮）
### 方向盘
- **方向盘类型**：多功能真皮方向盘
- **调节方式**：上下 + 前后角度可调
- **控制按键**：集成定速巡航、多媒体控制、电话接听、仪表盘信息切换
### 仪表盘
- **仪表盘类型**：12.3 英寸全液晶仪表盘（可选双屏布局）
- **显示内容**：
  - 车速、转速
  - 油量、水温
  - 瞬时油耗、平均油耗
  - 胎压监测信息
  - 驾驶辅助系统状态
## 三、中控与多媒体系统
- **中控屏幕**：10.3 英寸触控液晶屏（可选 12.3 英寸）
- **系统名称**：MBUX 多媒体交互系统（专为卡车优化）
- **支持功能**：
  - 蓝牙电话/音乐
  - Apple CarPlay / Android Auto
  - 语音控制（“Hey Mercedes”）
  - 导航系统（带实时交通信息）
  - 车辆设置与诊断
## 四、储物空间
| 位置 | 容量/说明 |
|------|-----------|
| 仪表台储物格 | 开放式储物槽，可放手机、票据 |
| 中央储物箱 | 大容量带盖储物箱，约 30L |
| 车门储物格 | 双侧车门，可放 1.5L 水瓶 + 杂物 |
| 顶部储物柜 | 驾驶室上方多个封闭式储物柜 |
| 卧铺下方抽屉 | 可拉出式储物抽屉，适合衣物 |
| 杯架 | 中控区域 2 个杯架 |
## 五、卧铺区域
- **下铺尺寸**：2000mm × 750mm（宽体版本可达 800mm）
- **上铺尺寸**：1900mm × 700mm（可选装）
- **床垫材质**：高密度海绵床垫
- **卧铺配置**：
  - 独立阅读灯
  - USB 充电接口
  - 窗帘（全遮光）
  - 储物网兜
## 六、空调与气候控制
- **空调类型**：自动恒温空调
- **分区控制**：驾驶员/副驾驶分区调节
- **辅助功能**：
  - 驻车空调（可选装，无需发动机运行）
  - 驻车加热器（冬季选装）
  - 空气循环与过滤系统
## 七、照明系统
- **驾驶区照明**：LED 顶灯 + 氛围灯（多色可调）
- **阅读灯**：驾驶员侧 + 卧铺区独立 LED 阅读灯
- **储物区照明**：储物柜内部照明
- **夜间模式**：仪表盘/中控可切换夜间暗光模式
## 八、安全与辅助配置（内饰相关）
- **注意力辅助系统**（疲劳监测）
- **倒车影像**（中控屏显示）
- **盲区监测**（A柱指示灯）
- **电子后视镜**（MirrorCam，选装，屏幕位于A柱内侧）
## 九、其他舒适配置
- **电源接口**：
  - 12V 点烟器接口 × 2
  - USB-A / USB-C 充电口 × 3
  - 220V 逆变器插座（选装）
- **遮阳板**：驾驶员侧 + 副驾驶侧
- **室内后视镜**：手动防眩目
- **地板材质**：防滑耐磨橡胶地板（易清洁）

> **注**：以上内饰参数基于奔驰 Actros 1023 常见配置整理，实际配置因生产年份、版本及选装包不同可能存在差异，请以实车为准。

# 五菱之光新能源 操作指南
  
更新时间：2026年07月20日08时47分15秒   

## 一、车辆启动与熄火
### 启动车辆
1. **智能钥匙确认**：确保智能钥匙在车内。
2. **踩下制动踏板**：右脚踩下制动踏板。
3. **启动操作**：
   - 旋钮式换挡：将旋钮旋转至“N”或“D”挡，仪表盘显示“READY”即表示车辆已上电。
   - 一键启动（如配备）：按下“START/STOP”按钮，仪表盘亮起并显示“READY”。
4. **挂挡行驶**：踩住刹车，将挡位旋钮旋至“D”（前进）或“R”（倒车），松开电子手刹（如有），缓慢松开刹车即可行驶。
### 熄火车辆
1. **停稳车辆**：踩住刹车，确保车辆完全静止。
2. **挂入P挡或N挡**：将旋钮旋转至“N”或按下“P”挡按钮（如配备）。
3. **拉起手刹**：拉起机械手刹或按下电子手刹按钮。
4. **关闭电源**：
   - 一键启动车型：再次按下“START/STOP”按钮，仪表盘熄灭。
   - 非一键启动车型：将钥匙拧回OFF位置并拔出。
> **注意**：下车前请确认所有车窗、灯光已关闭，携带好智能钥匙。

## 二、驾驶模式与换挡操作
### 换挡旋钮说明
- **P (驻车)**：停车时使用，需在车辆完全停止后挂入。
- **R (倒车)**：倒车时使用，需在车辆静止时挂入。
- **N (空挡)**：临时停车（如等红灯）或拖车时使用。
- **D (前进)**：正常行驶模式。
### 驾驶模式切换（如配备）
部分车型提供经济/运动模式切换：
- **经济模式 (ECO)**：降低电机输出功率，延长续航里程。适合城市拥堵路况。
- **运动模式 (SPORT)**：提升加速响应和动力输出。适合超车或爬坡。
> *切换方式*：按下中控台上的 “MODE” 或 “ECO/SPORT” 按键进行切换。

## 三、充电操作指南
### 充电接口位置
- 快充口 / 慢充口位于车身侧面（通常为左后翼子板处）。
### 交流慢充（家用充电桩 / 随车充）
1. **熄火并拔出钥匙**。
2. **打开充电口盖板**。
3. **插入充电枪**直至听到“咔哒”锁止声。
4. **开始充电**：
   - 家用桩刷卡/APP启动；
   - 随车充直接连接220V插座即可自动开始。
5. **结束充电**：
   - APP/刷卡停止；
   - 解锁车辆后按住充电枪按钮拔出；
   - 盖好充电口盖板。
### 直流快充（公共快充桩）
1. **熄火并拔出钥匙**。
2. **打开快充口盖板**（通常与慢充口分开）。
3. **连接快充枪并按照充电桩屏幕提示操作支付、启动充电。**
4. *建议电量低于80%时停止快充*，以保护电池寿命。
> ⚠️ **安全提示**
> - *雷雨天气禁止室外充电。*
> - *请勿使用破损、潮湿的充电设备。*
> - *长期停放建议保持电量在40%-60%。*

## 四、仪表盘指示灯解读
|指示灯图标|名称|含义|
|---|---|---|
|⚡ READY|准备就绪|表示高压系统已接通，可正常行驶|
|🔋 |电池电量低|剩余电量不足20%，请及时充电|
|⚠️ |系统故障灯亮起红色|高压系统异常；请立即停车联系售后|
|🛞 |胎压报警灯亮起黄色|轮胎气压异常；检查胎压并调整|
|🅿️ P |驻车指示灯亮起绿色/红色|已挂入P挡|

- [ ] *灯光是否正常工作*

- [ ] *刹车油液位*（如有）

- [ ] *底盘有无刮擦异响*

## ⚠️紧急情况处理
### 🆘动力中断 / “趴窝”
1. *开启双闪警示灯*；
2. *尝试重新上电*（踩刹车+按START）；
3. *若无法恢复*：
   - N挡状态下推至安全区域；
   -拨打救援电话等待拖车。
### 🆘涉水行驶注意事项
1.*水深不得超过轮毂中心线*；
2.*匀速通过避免激起大浪*；
3.*涉水后轻踩刹车排出水分恢复制动性能。*

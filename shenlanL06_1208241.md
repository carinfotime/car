## 深蓝L06 动力参数介绍
  
更新时间：2026年07月14日08时43分49秒   

深蓝L06作为深蓝汽车旗下的重磅车型，提供了**纯电版**与**增程版**两种动力形式，以满足不同用户的需求。以下是详细的动力参数信息。
### 一、增程版（1.5L 四缸增程器 + 后置单电机）
增程版车型搭载一台1.5L自然吸气四缸发动机作为增程器，主要负责发电，不直接驱动车轮。驱动任务由后置的永磁同步电机完成。
| 项目 | 参数 |
| :--- | :--- |
| **增程器类型** | 1.5L 自然吸气四缸发动机 |
| **增程器最大功率** | 70 kW (95 Ps) |
| **驱动电机类型** | 永磁同步电机 |
| **驱动电机布局** | 后置 |
| **驱动电机最大功率** | 160 kW (218 Ps) |
| **驱动电机最大扭矩** | 320 N·m |
| **系统综合功率** | 160 kW (218 Ps) |
| **0-100km/h加速时间** | 约7.5秒 - 8.0秒（视具体配置） |
| **最高车速** | ≥170 km/h |
#### CLTC纯电续航与综合续航
-   **纯电续航里程**: 
    -   标准版：约135 km
    -   长续航版：约200 km
-   **综合续航里程**: 
    -   标准版：约1140 km
    -   长续航版：约1200 km
---
### 二、纯电版（后置单电机）
纯电版车型采用后置单电机布局，提供两种不同功率的调校版本。
#### 标准功率版
| 项目 | 参数 |
| :--- | :--- |
| **驱动电机类型** | 永磁同步电机 |
| **驱动电机布局** | 后置 |
| **最大功率** | 160 kW (218 Ps) |
| **最大扭矩** | 320 N·m |
| **0-100km/h加速时间** | ~6.9秒 |
| **最高车速** | ≥170 km/h |
#### CLTC纯电续航
-   **电池容量**: ~58.9 kWh (磷酸铁锂)
-   **CLTC纯电续航**: ~530 km
#### 高功率版
| 项目 | 参数 |
| :--- | :--- |
| **驱动电机类型** | 永磁同步电机 |
| **驱动电机布局** | 后置 |
| **最大功率** | ≥190 kW (258 Ps) *（具体数值以官方最终发布为准）*|
| **最大扭矩** | ≥320 N·m *（具体数值以官方最终发布为准）*|
| **0-100km/h加速时间** | ~6.2秒 *（预估）*|
| **最高车速** | ≥180 km/h |
#### CLTC纯电续航
-   **电池容量**: ~79.97 kWh (三元锂)
-   **CLTC纯电续航**: ~705 km

### 📝核心亮点总结
1.  全系标配后驱布局，提供更好的操控乐趣和加速体验。
2.  增程版在馈电状态下油耗表现优秀（CLTC馈电油耗约4.9L/100km）。
3.  高配纯电版本支持800V高压快充技术，充电10分钟可增加约200公里续航。
4.  全系采用微核高频脉冲加热技术，有效提升冬季低温环境下的充电速度和动力性能。

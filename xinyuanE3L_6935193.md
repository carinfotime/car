## 四、充电注意事项
| 要点 | 说明 |
|-----|------|
| 充电环境 | 避免在高温暴晒或雷雨天气下充电 |
| 充电枪 | 保持干燥清洁，插拔时避免暴力操作 |
| 充电电量 | 建议日常充至80%-90%，长途可充至100% |
| 放电深度 | 避免电量低于10%再充电 |
| 快充使用 | 建议快充与慢充交替使用，减少电池损耗 |
| 充电时状态 | 充电时车内尽量熄火断电 |
| 长期停放 | 电量保持在40%-60%，每半个月补电一次 |

## 五、季节性保养重点
### 春季
- 清洁空调系统，更换空调滤芯
- 检查雨刮片，更换老化件
- 检查底盘是否有冬季融雪剂腐蚀
### 夏季
- 检查冷却液液位及散热器清洁
- 检查空调制冷效果
- 避免高温时段长时间快充
- 检查轮胎气压（夏季适当降低）
### 秋季
- 检查制动系统，为冬季做准备
- 检查蓄电池状态
- 更换冬季玻璃水
### 冬季
- 检查冷却液冰点
- 电池预热后再充电（低温环境）
- 检查轮胎花纹深度，确保抓地力
- 减少快充频率，优先慢充

## 六、常见故障及处理建议
| 故障现象 | 可能原因 | 建议处理 |
|---------|---------|---------|
| 续航明显下降 | 电池老化/温度过低/驾驶习惯 | 检查BMS数据，调整驾驶模式 |
| 充电速度变慢 | 充电桩功率/电池温度/BMS限制 | 更换充电桩，等待电池降温 |
| 仪表报动力电池故障 | 绝缘故障/连接松动 | 立即停车，联系服务站 |
| 电机异响 | 轴承磨损/减速器缺油 | 进站检查，避免继续行驶 |
| 刹车偏软 | 制动液不足/刹车片磨损 | 检查制动液，更换刹车片 |

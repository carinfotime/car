# 艾瑞泽8 车主操作指南
  
更新时间：2026年07月15日11时36分57秒   

## 一、启动与熄火
### 启动发动机
1. 确保智能钥匙在车内。
2. 踩下 **制动踏板**（自动挡车型必须踩住）。
3. 按下 **一键启动按钮**，发动机启动。
4. 仪表盘自检完成后，挂挡即可行驶。
> 注意：若仅需通电（不启动发动机），在不踩刹车的情况下按下一键启动按钮即可。
### 熄火
1. 车辆完全停稳后，挂入 **P挡**。
2. 拉起 **电子手刹**（EPB）。
3. 按下 **一键启动按钮** 熄火。
---
## 二、挡位操作（自动挡）
| 挡位 | 说明 |
|------|------|
| **P** | 驻车挡，停车熄火时使用 |
| **R** | 倒车挡 |
| **N** | 空挡，短暂停车或拖车时使用 |
| **D** | 前进挡，日常行驶使用 |
| **M** | 手动模式，可通过换挡拨片或挡杆手动加减挡 |
> 切换挡位需踩住刹车并按下挡杆侧面的解锁按钮。
---
## 三、电子手刹与自动驻车
- **电子手刹（EPB）**：向上拨起拉起手刹，向下按压释放手刹。
- **自动驻车（Auto Hold）**：按下按钮开启（指示灯亮起）。车辆刹停后自动保持制动，轻踩油门即可解除。
---
## 四、灯光控制
| 操作 | 功能 |
|------|------|
| 旋钮至 **OFF** | 关闭灯光 |
| 旋钮至 **示宽灯** | 位置灯亮起 |
| 旋钮至 **近光灯** | 近光灯开启 |
| 向外推拨杆 | 远光灯常亮 |
| 向内拉拨杆 | 远光灯闪动（超车提示） |
| 旋钮至 **AUTO** | 自动大灯（根据环境光线自动开关） |
> 雾灯：在大灯开启状态下，旋转雾灯开关即可。
---
## 五、雨刮器操作
- **上推**：单次刮刷（点动）。
- **下拉**：低速连续刮刷。
- **再下拉**：高速连续刮刷。
- **向内拉**：喷水并刮刷。
---
## 六、空调与多媒体
### 空调控制
- **温度调节**：旋转温度旋钮（或通过中控屏调节）。
- **风量调节**：旋转风量旋钮。
- **模式切换**：吹脸、吹脚、除霜等模式，通过按键或屏幕选择。
- **A/C**：空调压缩机开关（制冷用）。
- **内外循环**：市区拥堵或隧道建议内循环，高速建议外循环。
### 多媒体系统
- 支持 **CarPlay / HiCar / 百度CarLife** 手机互联。
- 通过中控屏操作导航、音乐、车辆设置等。
- 方向盘右侧按键可控制音量、切歌、语音助手。
---
## 七、定速巡航 / 自适应巡航（ACC）
1. 按下 **巡航开关按钮**（方向盘左侧）。
2. 车速达到所需速度后，按下 **SET/-** 键设定当前速度。
3. 按 **RES/+** 加速，按 **SET/-** 减速。
4. 轻踩刹车或按 **CANCEL** 取消巡航。
5. 按 **巡航开关按钮** 关闭巡航系统。
> 高配车型支持自适应巡航，可自动跟随前车加减速。
---
## 八、智能钥匙与无钥匙进入
- **锁车**：触摸门把手外侧感应区域（或按钥匙锁车键）。
- **解锁**：手伸入门把手内侧自动解锁（或按钥匙解锁键）。
- **后备箱**：长按钥匙后备箱键，或车尾感应开启（高配）。
---
## 九、仪表盘指示灯简识
| 图标 | 含义 |
|------|------|
| 红色电池 | 充电系统故障 |
| 红色机油壶 | 机油压力过低 |
| 黄色发动机 | 发动机系统故障 |
| 黄色ESP | 车身稳定系统关闭或故障 |
| 绿色P | 电子手刹已拉起 |
| 绿色刹车盘 | 自动驻车工作 |
> 出现红色指示灯请立即停车检查；黄色指示灯请尽快检修。
---
## 十、日常维护提示
1. **机油**：建议每 5000-7500 公里或 6 个月更换一次。
2. **胎压**：标准胎压 2.3-2.5 bar，每月检查一次。
3. **冷却液**：冷车时检查液位，介于 MIN 与 MAX 之间。
4. **玻璃水**：不足时及时添加，冬季使用防冻型。
5. **轮胎换位**：每 10000 公里前后轮交叉换位一次。
---
## 十一、紧急情况处理
- **轮胎漏气**：使用后备箱下方的补胎液或备胎（视配置），按照说明书操作。
- **电瓶亏电**：使用搭电线连接正极（+）和负极（-），注意先接正极，后接负极。
- **拖车**：挂入 N 挡，松开电子手刹，拖车速度不超过 30 km/h，距离不超过 50 公里。

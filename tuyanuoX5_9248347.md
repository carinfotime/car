# 图雅诺X5 空间参数介绍
  
更新时间：2026年07月15日16时36分55秒   

## 车身尺寸
| 项目 | 参数 |
| :--- | :--- |
| 车长（mm） | 4923 / 5015 / 5065 (根据不同版本) |
| 车宽（mm） | 2000 |
| 车高（mm） | 2198 / 2430 (根据不同版本) |
| **轴距（mm）** | **2933** |
> **注：** 短轴版车长约为4923mm，长轴版可达5065mm；低顶版车高约2198mm，中高顶版约2430mm。
## 内部空间
- **内部最大宽度：** ~1800 mm
- **内部最大高度：** ~1600 mm (低顶) / ~1800 mm (中高顶)
- **载货容积：** **约6.8 m³ - 9.1 m³** (视具体车型与顶高而定)
## 座椅布局与载客能力
| 座位数 | 布局形式 |
| :--- | :--- |
| **5座** | 2+3 (前2后3) |
| **6座** | 2+2+2 / 3+3 |
| **7座** | 2+2+3 / 3+2+2 |
| **9座** | (部分加长或定制版本可选) |
> *注：具体座位数及布局以官方配置表为准。*
## 货箱尺寸（厢式货车版）
- **货箱长度：** ~2500 mm - ~2900 mm
- **货箱宽度：** ~1750 mm
- **货箱高度：** ~1500 mm - ~1700 mm
- **侧滑门开口宽度：** ≥700 mm
- **尾门开口宽度：** ≥1200 mm
## 
其他亮点
1. **侧滑门设计：**
   -  右侧单/双侧滑门可选。
   -  开口宽敞，方便人员上下及货物装卸。
2. **尾门开启方式：**
   -  对开式尾门（180°开启），便于叉车作业。
3. **内部平整度：**
   -  地板平整无凸起，利于货物堆放。
4. **轮罩间距：**
   -  轮罩内宽约1200mm，可轻松放置标准托盘。
---
### 
总结
图雅诺X5在紧凑型轻客/物流车市场中提供了较为灵活的空间选择：
-   短轴低顶版适合城市穿梭、商超配送；
-   长轴中高顶版则更适合需要站立作业或装载大件货物的场景。
其方正的车身轮廓、平整的地板以及多种座椅/货箱组合，能够较好地兼顾客运舒适性与货运实用性。

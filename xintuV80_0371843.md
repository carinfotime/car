## 一、车型定位与概述
**新途 V80** 是上汽大通 MAXUS 旗下的主力轻客车型，主打 **城市物流、客货两用、改装专用车** 市场。其核心卖点在于：宽体轻客空间、前驱/后驱可选、AMT/MT变速箱选择以及较高的性价比。
**主要竞品车型：**
1. **江铃福特 全顺（经典全顺 / 新世代全顺 Pro）**
2. **南京依维柯 得意 / 欧胜**
3. **福田图雅诺（风景 G5 / 图雅诺）**
4. **江淮星锐**

## <a name="三、分维度深度对比"></a>三、分维度深度对比
### <a name="1.-动力与驾驶感受"></a>1️⃣ 动力与驾驶感受
-   **[新途V80]**:搭载上汽π系列2️⃣0T柴油机，动力输出平顺，油耗表现优秀。AMT版本在市区拥堵路况下能解放左脚，但换挡逻辑偏保守，偶有顿挫感。前驱版操控更轻便，后驱版爬坡和重载更有力。
-   **[江铃全顺]**:福特PUMA发动机技术成熟，噪音控制较好。9AT变速箱换挡平顺性优于V80的AMT。整体驾驶质感更偏向乘用车化。
-   **[依维柯欧胜]**:F1C发动机低扭强劲，重载起步优势明显。非承载式车身+后双胎版本承载能力极强，但舒适性较差，方向盘虚位较大。
-   **[福田图雅诺]**:动力够用但调校偏粗犷，噪音和震动控制不如前两者。价格是最大优势。
### <a name="2.-空间与装载能力"></a>2️⃣ 空间与装载能力
-   **[新途V80]**:方正的车厢设计使得空间利用率较高。尾门对开+侧滑门为标准配置。地板离地高度适中，装卸货物较为便利。
-   **[江铃全顺]**:内部高度和宽度表现均衡，座椅折叠灵活性好。部分车型可选装推拉窗。
-   **[依维柯欧胜]**:车厢内宽较窄是短板（受限于非承载式大梁），但长度和高度优势明显。底盘承重能力最强。
-   **[福田图雅诺]**:空间数据上与V80相当甚至更大一些，但做工细节和密封性一般。
### <a name="3.-燃油经济性"></a>3️⃣ 燃油经济性
-   在同等工况下：
    1️⃣ **[新途V80]** (前驱版)：由于传动效率较高且车重较轻(约1️⃣8吨)，综合油耗通常在7️⃣5️⃣L~8️⃣5️⃣L/100km之间。
    2️⃣ **[江铃全顺]**：约8️⃣0L~9️⃣0L/100km。
    3️⃣ **[依维柯欧胜]**：由于自重大且后驱传动损耗较大(约2吨以上)，综合油耗通常在9️⃣0L~11L左右。
    4️⃣ **[福田图雅诺]**：约8️⃣5L~10L左右。

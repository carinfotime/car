# 金杯金卡S2 保养指南
  
更新时间：2026年07月15日10时21分26秒   

## 一、保养周期总表
| 保养项目 | 首保（建议） | 常规周期 |
|---------|------------|---------|
| 首次保养 | **行驶2000-2500公里或购车后3个月**（以先到为准） | - |
| 常规保养 | - | **每5000公里或6个月**（以先到为准） |
---
## 二、各阶段保养项目明细
### 🔧 A级保养（每5000公里 / 6个月）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查玻璃清洗液液位
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查灯光系统（大灯、转向灯、刹车灯等）
- [ ] 检查底盘及悬挂系统有无松动或异响
- [ ] 检查传动皮带张紧度及老化情况
- [ ] 清洁空气滤清器滤芯（视情况吹扫或更换）
### 🔧 B级保养（每10000公里 / 12个月）
在A级保养基础上增加：
- [ ] **更换空气滤清器滤芯**
- [ ] **更换燃油滤清器**
- [ ] **更换火花塞**（建议每2万公里）
- [ ] **检查并调整气门间隙**（如适用）
- [ ] **检查蓄电池状态**（电压、接线柱腐蚀情况）
- [ ] **润滑车门铰链及锁扣**
- [ ] **检查并紧固底盘螺栓**
### 🔧 C级保养（每20000公里 / 24个月）
在B级保养基础上增加：
- [ ] **更换变速箱油**（手动变速箱）
- [ ] **更换后桥齿轮油**
- [ ] **更换制动液**（建议每2年一次）
- [ ] **更换冷却液**
- [ ] **清洗节气门**
- [ ] **清洗喷油嘴**
- [ ] **四轮换位及动平衡**
- [x] *如为自动挡车型，需同时更换自动变速箱油*

## 三、关键油液规格推荐
| 项目 | 推荐规格 |
|------|---------|
| **发动机机油** | SAE **10W-40** / **15W-40** （API SL及以上级别，夏季可选20W-50） |
| *冬季低温地区* | SAE **5W-30** / **5W-40** |
| *高温/重载工况* | SAE **20W‑50** |
| **变速箱油** | GL‑4 **75W‑90** （手动变速箱） |
| *自动变速箱* | ATF Dexron III / VI （视具体型号而定） |
| *后桥齿轮油* | GL‑5 **80W‑90** |
| *制动液* | DOT3 / DOT4 |
| *冷却液* | -35℃乙二醇型长效防冻液 |
> ⚠️ *具体标号请以车辆使用手册为准，不同批次发动机可能略有差异。*

## 四、日常自检清单（建议每周一次）
1. ✅  冷车状态下检查机油尺刻度，应在上下限之间。
2. ✅  观察冷却液储液罐水位是否在“MAX”与“MIN”之间。
3. ✅  启动后观察仪表盘有无故障灯亮起。
4. ✅  原地转动方向盘，感受转向是否沉重或有异响。
5. ✅  踩下刹车踏板，感受行程是否正常、有无发软或下沉现象。
6. ✅  绕车一周查看轮胎外观有无鼓包、裂纹或异常磨损。
7. ✅  测试所有灯光是否正常工作。

## ⏱️五、易损件参考更换周期
> ⚠️ *以下为参考值，实际寿命受使用环境与驾驶习惯影响。*
### 🛞轮胎类
> -   轮胎花纹深度低于1.6mm时强制更换。
> -   正常使用下建议每4年或6万公里左右换新。
### 🪟雨刮片
> -   出现刮不净、异响或胶条开裂时立即更换。
> -   一般寿命约6~12个月。
### 🔋蓄电池
> -   免维护蓄电池寿命通常为2~4年。
> -   冬季启动困难时优先检测蓄电池电压。
### 🧊空调滤芯
> -   建议每年至少更换一次；多尘地区缩短至半年。

## 📌六、重要注意事项
1. 📍  首保必须按时进行，否则可能影响质保权益。
2. 📍  每次进店保养务必保留好维修工单和记录凭证。
3. 📍  切勿混用不同品牌/标号的机油和冷却液。
4. 📍  长时间重载行驶后应适当缩短保养间隔里程。
5. 📍  若发现仪表盘上“发动机故障灯”、“ABS灯”、“气囊灯”常亮，请尽快进站检修。
---
## 🛠️七、常见故障自查提示
### ❄️冬季启动困难？
> -   优先检测蓄电池电压与启动机状态；
> -   确认机油粘度是否适合当地冬季气温；
> -   预热系统工作是否正常。
### 🚗行驶中水温偏高？
> -   停车怠速降温后检查冷却液是否缺失；
> -   散热风扇能否正常运转；
> -   节温器是否存在卡滞故障。
### 🔊底盘异响？
> -   低速转弯异响 →　排查外球笼；
> -   过减速带“咯吱”声 →　排查平衡杆胶套；
> -   高速抖动 →　做轮胎动平衡。
---
## 💡八、省心小贴士
✅  养成每次加油前看一眼仪表盘的习惯。  
✅  每月至少测量一次胎压。  
✅  长期停放超过两周时断开蓄电池负极。  
✅  涉水行驶后及时检查制动效果和底盘锈蚀情况。  
✅  尽量选择正规加油站加注符合标号的燃油。

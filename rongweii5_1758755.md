# 荣威i5 驾驶注意事项
  
更新时间：2026年07月15日16时43分35秒   

## 一、启动与预热
1. **启动前检查**：确认挡位处于 P 挡或 N 挡，踩下制动踏板后再一键启动。
2. **冷车预热**：冬季或长时间停放后，建议怠速热车 30-60 秒，待转速回落至正常怠速（约 800rpm）后再起步。
3. **避免急加速**：水温未达到正常工作温度前（约 90°C），不要猛踩油门。
## 二、行驶中注意事项
### 1. 变速箱使用（CVT / DCT）
- **CVT车型**：避免长时间大脚油门急加速，防止钢带打滑；上下长坡可切换至手动模式或 L 挡。
- **DCT车型**：低速蠕行（如堵车）时建议切换至手动模式或运动模式，减少离合器频繁半联动磨损。
- **停车等待**：等红灯超过30秒建议挂 N 挡拉手刹，减少变速箱油温升高。
### 2. 刹车系统
- **新车磨合期**（前500公里）：避免紧急刹车，逐步磨合刹车盘片。
- **涉水后**：轻踩几次刹车，排出刹车盘上的水膜，恢复制动性能。
- **刹车异响**：若出现尖锐异响，可能是刹车片磨损到极限或夹杂异物，及时检查。
### 3. 转向与底盘
- **方向盘打死**：尽量避免长时间打死方向（尤其原地打轮），防止转向助力泵过热。
- **过减速带/坑洼**：提前减速通过，保护悬挂系统和轮胎。
### 4. ESP车身稳定系统
- **默认开启**：日常行驶保持 ESP ON，不要随意关闭。
- **脱困时**：陷车或雪地起步困难时可短暂关闭 ESP，脱困后立即重新开启。
## 三、仪表盘与报警灯
| 指示灯 | 含义 |  处理方式 |
|--------|------|----------|
| 🟡发动机故障灯 |  排放或传感器问题 |  尽快进店检修 |
| 🔴机油压力灯 |  机油不足或压力低 |  立即停车检查机油 |
| 🔴电瓶灯 |  充电系统故障 |  检查发电机/电瓶 |
| 🟡胎压报警 |  胎压异常 |  及时补气或检查轮胎 |
## 四、燃油与保养
1. **燃油标号**：推荐使用92号及以上无铅汽油。
2. **油箱盖提示**：加油时注意油箱盖标识方向（通常在车辆左侧）。
3. **保养周期**：
   - 首保：5000公里或6个月（以先到为准）
   - 常规保养：每10000公里或12个月
4. **机油选择**：
   - CVT车型推荐5W-30全合成机油
   - DCT车型推荐0W-20全合成机油
## 五、特殊路况驾驶
### 🚗高速公路
-   保持安全车距（至少2秒法则）
-   避免长时间定速巡航在单一速度区间内行驶（CVT车型）
-   超车提前打转向灯并观察盲区
### 🌧️雨天/湿滑路面
-   降低车速20%-30%
-   关闭自动启停功能（涉水路段）
-   积水深度超过半个轮胎高度时请绕行
### ❄️冬季驾驶
-   提前检查防冻液冰点、玻璃水冰点
-   冰雪路面起步用2挡缓抬离合/轻踩油门
-   切勿空挡滑行下坡
## ⚠️六、常见禁忌行为
1. ❌油箱见底才加油 ——容易损坏燃油泵
2. ❌长期短途行驶不热车 ——加剧积碳和磨损
3. ❌停车后方向盘不回正 ——影响转向拉杆和轮胎寿命
4. ❌N挡滑行 ——CVT/DCT变速箱严重损坏风险
5. ❌熄火后长时间使用电器 ——亏电缩短电瓶寿命

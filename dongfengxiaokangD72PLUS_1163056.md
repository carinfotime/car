| 项目 | 参数 |
|------|------|
| **发动机型号** | SFG18A（1.8L自然吸气） |
| **排量** | 1.8L（1798ml） |
| **进气形式** | 自然吸气 |
| **气缸排列形式** | 直列四缸 |
| **每缸气门数** | 4气门 |

## 变速箱参数
- **变速箱类型**: 5挡手动变速箱（MT）
- **驱动方式**: 前置后驱（FR）
## 底盘转向系统
- **前悬架类型**: 麦弗逊式独立悬架
- **后悬架类型**: 钢板弹簧非独立悬架
- **转向助力类型**: EPS电动助力转向
- **车体结构**: 非承载式车身
## 制动系统
- **前制动器类型**: 盘式制动器
- **后制动器类型**: 鼓式制动器
- **驻车制动类型**: 手刹（机械拉线式）
## 轮胎与轮毂
- **轮胎规格**: 
   - （前）185 R14LT
   - （后）185 R14LT
- **轮毂材质**: 钢制轮毂
## ●动力性能特点总结
1. ✅ **1.8L大排量自然吸气发动机**，动力输出平顺线性，适合载货工况。
2. ✅ **低扭表现突出**：155N·m峰值扭矩在3600rpm即可爆发，中低速爬坡、起步有力。
3. ✅ **前置后驱布局**：后轮驱动提供更好的载重爬坡抓地力，适合山区、工地等复杂路况。
4. ✅ EPS电动助力转向：低速轻盈、高速沉稳，减轻驾驶疲劳。
5. ✅钢板弹簧后悬架：承载能力强，适合重载运输场景。

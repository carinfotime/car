## 捷途X70L 空间参数介绍
  
更新时间：2026年07月19日11时34分00秒   

### 车身尺寸
*   **车长：** 4749mm
*   **车宽：** 1900mm
*   **车高：** 1720mm
*   **轴距：** 2745mm
### 座椅布局
*   **座椅形式：** 提供5座、6座（2+2+2）及7座（2+3+2）三种布局可选。
*   **第二排：**
    *   支持前后滑动及靠背角度调节。
    *   6座版为独立座椅，带独立扶手，中央通道方便进入第三排。
*   **第三排：**
    *   两个独立座椅，支持靠背角度调节。
    *   可完全放倒，与后备箱形成纯平空间。
### 内部乘坐空间
| 项目 | 数据/描述 |
| :--- | :--- |
| **前排头部空间** | ≥1拳（身高175cm体验者） |
| **第二排腿部空间** | 最大可达2拳以上（滑至最后时） |
| **第二排头部空间** | ≥1拳 |
| **第三排腿部空间** | 约1拳（第二排调至适中位置时） |
| **第三排头部空间** | ≤1拳（适合短途或儿童乘坐） |
### 储物与后备箱
#### 常规储物
*   **全车储物格数量：** ≥30处
*   **特色设计：**
    *   门板杯架可容纳1.5L大瓶饮料。
    *   中控台下方的镂空储物区。
    *   第三排两侧杯架及储物槽。
#### 后备箱容积
| 座椅状态 | 容积 (L) | 说明 |
| :--- | :--- | :--- |
| **5座模式 (三排放倒)** | ≥1680 L | 纵深可观，可放置大件物品。 |
| **7座模式 (三排立起)** | ≈89 L ~ ≤200 L | 仅能放置背包或小行李箱。 |
### ✨亮点设计
1.  **纯平放倒功能：** 第二、三排座椅支持比例放倒，放倒后可形成一个近乎纯平的装载平面，适合搬运长物件或作为临时休息空间。
2.  **灵活的空间组合：**
    *   “2+3”五座版拥有超大后备箱。
    *   “2+2+2”六座版中央通道宽敞，进出便利性最佳。
3.  **全景天窗加持：** （部分配置车型）大尺寸全景天窗有效提升车内开扬感，减少后排尤其是第三排的压抑感。
> ⚠️ *注：以上数据基于官方公布信息及实测参考值，实际体验可能因配置版本、坐姿习惯略有差异。*

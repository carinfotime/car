# 智己LS9 技术参数介绍
  
更新时间：2026年07月11日15时31分13秒   

## 一、车型定位
- **品牌**：智己汽车 (IM Motors)
- **车型**：LS9
- **级别**：全尺寸豪华智能纯电SUV
- **平台**：iO“原点”高端纯电架构
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **驱动方式** | 双电机四驱 / 三电机四驱 |
| **前电机最大功率** | 约175 kW |
| **后电机最大功率** | 约250 kW（双电机版） / 双后电机（三电机版） |
| **系统综合功率** | 双电机版约425 kW（约578匹）；三电机版超700 kW（约950匹） |
| **系统综合扭矩** | >1000 N·m（三电机版） |
| **0-100 km/h加速时间** | <3秒（三电机版） |
| **最高车速** | >200 km/h |
## 三、电池与续航
| 项目 | 参数 |
|------|------|
| **电池类型** | 三元锂电池 / 半固态电池（可选） |
| **电池容量** | 118 kWh / 130 kWh+（半固态版本） |
| **CLTC综合续航里程** | >800 km（半固态版本）；标准版约660 km+ |
| **充电技术** | 800V高压快充平台 |
| **快充时间 (10%-80%)** | <20分钟（峰值充电功率>350 kW） |
## 四、车身尺寸与空间
| 项目 | 参数 |
|------|------|
| **车长 (mm)** | ~5300 mm |
| **车宽 (mm)** | ~2000 mm |
| **车高 (mm)** | ~1730 mm |
| **轴距 (mm)** | ~3200 mm |
| **座位数** | 6座 / 7座可选 |
## 五、底盘与悬架
- **前悬架类型**:  双叉臂式独立悬架
- **后悬架类型**:  多连杆式独立悬架
- **车身结构**:  全铝混合材料车身
- **空气悬架**:  标配智能空气悬架 + CDC电磁减震系统
- **后轮转向**:  支持双向共12°后轮转向，最小转弯直径 <11 m
## 六、智能驾驶辅助系统
- **芯片方案**:  英伟达Orin X芯片 *2颗（算力>500 TOPS）
- **(可选升级)**:  英伟达Thor芯片 *1颗（算力>2000 TOPS）
- **(可选升级)**:  高通Snapdragon Ride Flex SoC *1颗
- **(可选升级)**:  地平线征程6旗舰芯片 *1颗
- **(可选升级)**:  华为MDC计算平台 *1套
- **(可选升级)**:  黑芝麻智能华山A2000芯片 *1颗
**(注：以上芯片方案为不同配置或选装包选项，非同时搭载)**

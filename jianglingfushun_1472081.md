## 江铃福顺 动力参数介绍
  
更新时间：2026年07月11日03时46分40秒   

### 核心动力总成
江铃福顺搭载的是由**江铃与福特**合作开发的 **JX4D20A6H** 型号 **2.0T 柴油涡轮增压发动机**。这款发动机在技术上与福特PUMA发动机同源，具备高效、可靠、低噪音的特点。
### 主要动力参数
| 参数项目 | 具体数值 |
| :--- | :--- |
| **发动机型号** | JX4D20A6H |
| **排量** | 2.0T (1998ml) |
| **燃料形式** | 柴油 |
| **进气形式** | 涡轮增压 |
| **最大马力** | **146 匹** (Ps) |
| **最大功率** | **107 kW** |
| **最大功率转速** | 3200 rpm |
| **最大扭矩** | **355 N·m** |
| **最大扭矩转速** | 1400 - 2600 rpm |
| **排放标准** | 国六 b |
### 变速箱匹配
与发动机匹配的是 **6挡手动变速箱 (6MT)**，部分高配或自动挡版本可选配 **8挡自动变速箱 (8AT)**。
-   **6MT**：齿比绵密，换挡手感清晰，燃油经济性好。
-   **8AT**：换挡平顺，能更好地发挥发动机的扭矩输出，降低驾驶疲劳感，尤其适合城市拥堵路况和长途运输。
### 动力特性解读
1.  **低扭强劲**：最大扭矩在 **1400 转** 即可爆发，并持续到 2600 转。这意味着车辆在起步、爬坡、满载时动力响应迅速，无需频繁换挡即可获得充足的牵引力，非常符合轻客和物流车的使用场景。
2.  **高效经济**：采用博世高压共轨燃油喷射系统，配合涡轮增压技术，实现了较高的燃烧效率。在保证动力输出的同时，有效控制油耗，降低了运营成本。
3.  **静谧舒适**：相较于传统柴油机，该发动机在NVH（噪音、振动与声振粗糙度）方面进行了优化，怠速和行驶过程中的噪音与振动控制表现较好，提升了驾乘舒适性。
### 应用场景总结
-   **城市物流**：低扭充沛，频繁启停和红绿灯起步轻松。
-   **城际运输**：高速巡航时，8AT变速箱能将发动机转速维持在较低水平，兼顾动力与油耗。
-   **改装与多功能用途**：充足的动力储备能够满足房车、工程车等特殊车辆的改装需求。
**一句话总结：** 江铃福顺的 **2.0T + 355N·m** 动力组合，核心优势在于 **低扭强劲、高效省油**，完美契合了轻客领域对“拉得多、跑得快、用得省”的核心需求。

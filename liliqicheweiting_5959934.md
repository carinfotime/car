# 梅赛德斯-奔驰 威霆 (Vito) 技术参数介绍
  
更新时间：2026年07月11日08时25分48秒   

## 车型概览
梅赛德斯-奔驰威霆是一款定位高端商务与多功能用途的轻型客车。凭借其宽敞的内部空间、坚固的非承载式车身结构以及源自奔驰的先进动力总成，威霆在商务接待、通勤班车及专用改装车领域拥有广泛的应用。
---
## 动力系统
| 项目 | 参数 |
| :--- | :--- |
| **发动机型号** | M274 / M260 (根据年款不同) |

| **最大扭矩** | 350 N·m @ 1,250 - 4,000 rpm |

| **排放标准** | 国 VI |

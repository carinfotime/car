# 奔驰EQA 空间参数介绍
  
更新时间：2026年07月11日04时47分56秒   

## 车身尺寸
| 参数 | 数值 |
|------|------|
| 车长 | 4,463 mm |
| 车宽（不含后视镜） | 1,834 mm |
| 车高 | 1,624 mm |
| 轴距 | 2,729 mm |
| 前轮距 | 1,585 mm |
| 后轮距 | 1,587 mm |
| 最小离地间隙 | 约 140 mm |
| 整备质量 | 约 1,940 kg（因配置而异） |
## 车内空间
### 前排空间
- **头部空间**：约 1,010 mm
- **腿部空间**：宽敞，座椅支持多向电动调节
- **肩部空间**：约 1,410 mm
### 后排空间
- **头部空间**：约 970 mm
- **腿部空间**：约 880 mm（中等身材乘客可舒适乘坐）
- **肩部空间**：约 1,380 mm
- **后排座椅**：支持 40:20:40 比例放倒
## 储物空间
| 位置 | 容量/说明 |
|------|-----------|
| 后备箱标准容积 | 340 升（后排座椅正常状态下） |
| 后排座椅放倒后容积 | 1,320 升 |
| 前备箱（前行李箱） | 无（因电机前置布局） |
| 车门储物格 | 前后车门均有，可容纳 1.5L 水瓶 |
| 中央扶手箱 | 深度适中，可放置小件物品 |
| 手套箱 | 标准尺寸，带照明 |
| 杯架 | 前排 2 个，后排中央扶手 2 个 |
## 后备箱细节
- 后备箱开口宽度：约 1,050 mm
- 后备箱开口高度：约 700 mm
- 后备箱纵深（座椅正常）：约 820 mm
- 后备箱纵深（座椅放倒）：约 1,660 mm
- 后备箱高度：约 720 mm
- 后备箱地板平整，下方有隐藏储物格（可存放充电线等物品）
- 电动尾门（部分配置标配）
## 乘坐体验说明
- **座椅布局**：5座，2+3 标准布局
- **后排中间座位**：地台略有隆起（约 80 mm），中间座位适合短途乘坐
- **座椅材质**：ARTICO 人造皮革 / 真皮（视配置而定）
- **空间感受**：作为紧凑型纯电 SUV，EQA 的空间表现处于同级主流水平，后排满足日常家用需求，后备箱在放倒座椅后可应对大件物品运输。
## 与燃油版 GLA 空间对比
- EQA 基于 GLA 平台打造，轴距相同（2,729 mm）
- 由于电池组位于底盘下方，车内高度与 GLA 基本一致
- 后备箱容积相比 GLA（435 升 / 放倒 1,410 升）略有减小，主要因电机和电控系统占用部分空间

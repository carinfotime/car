# 豹5 操作指南
  
更新时间：2026年07月19日10时54分13秒   

## 一、车辆基本操作
### 1. 启动与熄火
- **启动**：携带智能钥匙进入车辆，踩下制动踏板，按下“启动/停止”按键。
- **熄火**：车辆停稳后，再次按下“启动/停止”按键。
### 2. 挡位操作
- **P挡（驻车挡）**：停车时按下挡杆顶部的 P 键。
- **R挡（倒车挡）**：按住挡杆侧面的解锁键，向前推入 R 挡。
- **N挡（空挡）**：短时间停车或洗车时使用，向前或向后轻推至 N 挡。
- **D挡（前进挡）**：向后拉入 D 挡。
- **M挡（手动模式）**：在 D 挡位置向左拨动进入手动模式，通过拨片或前后推拉升降挡位。
### 3. 电子手刹与自动驻车
- **电子手刹**：向上拉起为锁止，向下按压为释放。
- **自动驻车（Auto Hold）**：系好安全带后按下 Auto Hold 键激活，车辆停稳后自动保持制动，轻踩油门即可解除。

## 二、驾驶模式选择
豹5提供多种驾驶模式以适应不同路况：
| 模式 | 适用场景 |
|------|----------|
| **经济模式** | 城市道路、节能行驶 |
| **舒适模式** | 日常通勤、铺装路面 |
| **运动模式** | 需要快速响应、超车或激烈驾驶 |
| **雪地模式** |冰雪、湿滑路面 |
| **沙地模式** |沙漠、沙滩等松软路面 |
| **泥地模式** |泥泞、沼泽路段 |
| **岩石模式** |攀爬岩石、大坡度越野 |
| **涉水模式** |涉水行驶时使用 |
> 💡切换方式：通过中控屏或物理旋钮选择对应驾驶模式。

## 三、四驱系统操作
### 1. 差速锁控制
- **前桥差速锁 / 后桥差速锁**
- *注意*：差速锁应在低速或静止状态下开启；脱困后及时关闭，避免损坏传动系统。
### 2. 低速四驱（4L）
- *用途*：极端越野、大扭矩爬坡场景。
- *切换条件*：车辆挂 N 挡并静止状态下切换。

## 四、能量回收与充电
### EV / HEV / SERIES / PARALLEL
豹5为插电混动车型：
- **EV（纯电优先）**：电量充足时优先用电。
- **HEV（混动模式）**：系统自动分配油电输出。
- *能量回收等级*可在中控屏调节：
    - *标准回馈* —滑行平顺
    - *较大回馈* —回收效率更高但拖拽感明显
###  充电口
位于车身右后侧：
1.  打开充电口盖板
2.  插入充电枪（交流慢充 /直流快充）
3.  充电完成后拔出并关闭盖板

##  五、智能座舱与中控屏操作
###   三屏交互
豹5配备12.3英寸仪表 +15.6英寸中控屏 +12.3英寸副驾屏：
- *仪表盘* —显示车速、电量/油量、导航信息等
- *中控屏* —控制空调、导航、音乐、车辆设置等
    -  支持语音唤醒：“你好，小迪”
    -  支持手势控制及分屏显示
- *副驾屏* —娱乐功能为主，可独立连接蓝牙耳机

## 🧰八、应急操作指南
### 🔋亏电无法启动怎么办？
1️⃣使用机械钥匙打开车门(隐藏在主驾门把手内)  
2️⃣联系道路救援或使用搭电宝(正极接正极,负极接车身搭铁点)  
### 🛞更换轮胎步骤:
1️⃣将车辆停在平坦安全路面,拉起电子手刹  
2️⃣取出后备箱下方工具包及千斤顶  
3️⃣在指定支撑点升起千斤顶  
4️⃣拧松螺栓→取下旧胎→安装备胎→对角顺序拧紧螺栓

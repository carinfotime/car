# 途逸T3 儿童乘车安全注意事项
  
更新时间：2026年07月13日14时30分36秒   

## ⚠️ 核心原则
- **12岁以下儿童** 或 **身高低于1.5米** 的儿童，严禁乘坐副驾驶位。
- **必须使用** 符合国家标准的儿童安全座椅，并安装在后排。
## 🚗 安全座椅使用规范
1. **按年龄/体重选择座椅**
   - **0-15个月（约0-13kg）**：使用反向安装的婴儿提篮式座椅。
   - **1-4岁（约9-18kg）**：使用正向安装的儿童安全座椅。
   - **4-12岁（约15-36kg）**：使用增高垫 + 成人安全带。
2. **正确安装**
   - 途逸T3后排配备 ISOFIX 接口，请优先使用 ISOFIX + 上拉带固定。
   - 安装后用力摇晃座椅，移动范围应小于2.5厘米。
## 🔒 行车中注意事项
| 项目 | 要求 |
|------|------|
| **车门锁** | 务必开启 **儿童安全锁**（后门侧面拨杆），防止行车中误开车门。 |
| **车窗** | 启用主驾位车窗锁止功能，防止孩子将头手伸出窗外。 |
| **安全带** | 确保安全带绕过孩子的肩部中间位置，不可勒到脖子或滑落手臂。 |
| **车内物品** | 收起中控台、座椅上的硬物/尖锐物品，避免急刹车时飞溅伤人。 |
## 🌡️ 停车与上下车
- **切勿将孩子单独留在车内**：夏季车内温度可在10分钟内升至60°C以上，极易导致热射病。
- **上下车引导**：让孩子从右侧（靠路侧）上下车，避开左侧来往车辆。
- **检查盲区**：发车前绕车一周，确认没有孩子在车头、车尾或车轮旁玩耍。
## 🧸 其他提醒
- **避免进食果冻、坚果等易噎食物**：车辆颠簸时容易发生呛咳窒息。
- **不要让孩子玩方向盘、挡把、手刹等操控部件**。
- *建议*：在后排车窗加装遮阳帘，减少阳光直射对孩子眼睛的刺激。
---
> 📌 *以上内容适用于东风途逸T3车型。如有改装或加装设备，请确认不影响原车安全结构。*

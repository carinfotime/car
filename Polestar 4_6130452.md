# Polestar 4 操作指南
  
更新时间：2026年07月14日15时04分54秒   

## 一、车辆启动与关闭
### 启动车辆
1. **携带钥匙**：确保智能钥匙（或已绑定的手机蓝牙钥匙）在车内。
2. **踩下制动踏板**：无需按任何启动按钮，直接踩下制动踏板，车辆即进入“就绪”状态。
3. **确认仪表盘**：仪表盘显示“READY”字样，表示车辆已可行驶。
### 关闭车辆
1. **挂入P挡**：按下中控台或方向盘后的P挡按钮。
2. **离开车辆**：携带钥匙离开座椅并关上车门，车辆将自动断电并锁车（需在设置中开启“离车自动锁车”功能）。
3. **手动关机**：在中控屏幕顶部下拉菜单中点击“关闭电源”。
> **注意**：若仅需短暂停留（如等红灯），保持D挡并踩住刹车即可；深踩刹车可激活Auto Hold功能。

## 二、驾驶模式选择
通过中控屏幕进入“驾驶”设置界面，可选择以下模式：
| 模式 | 适用场景 | 特点 |
|------|----------|------|
| **标准模式 (Standard)** | 日常通勤 | 平衡性能与能耗 |
| **运动模式 (Sport)** | 激烈驾驶/山路 | 动力响应更快，转向更沉 |
| **续航模式 (Range)** | 长途/低电量时 | 限制功率输出，关闭空调部分功能以延长续航 |
| **雪地/湿滑模式 (Snow/Wet)** | 雨雪天气 | ESP介入更早，动力输出平缓 |
切换方式：
- **方向盘右侧滚轮**：长按滚轮可快速切换预设的驾驶模式。
- **中控屏**：点击“设置 > 驾驶 > 驾驶模式”。

## 三、充电操作
### AC慢充（家用充电桩）
1.  打开车辆左侧后方的充电盖板。
2.  插入Type2充电枪。
3.  充电口指示灯变为蓝色闪烁表示连接成功，绿色常亮表示正在充电。
4.  可通过Polestar App或中控屏设置定时充电、限充百分比。
### DC快充（公共充电桩）
1.  打开左侧后方快充盖板（通常与慢充口共用）。
2.  插入CCS快充枪（较重的枪头）。
3.  指示灯变为蓝色闪烁后开始充电。
4.  建议电量至80%时结束快充以保护电池。
### 应急解锁
若充电枪无法拔出：
-   打开后备箱右侧的盖板，拉动内部的机械拉线即可解锁。

## 四、智能互联与导航
### Polestar Connect
-   通过手机App可远程查看电量、续航、位置及预约空调预热/预冷。
-   支持蓝牙钥匙分享给家人或朋友。
### Google原生导航
1.  点击主屏幕的Google地图图标。
2.  语音输入目的地：“Hey Google, navigate to...”。
3.  系统会结合当前电量和路况规划路线，并自动建议沿途充电站。
> **提示**：导航时会实时显示到达目的地的剩余电量百分比。

## 五、安全辅助系统
Polestar Pilot包含以下功能：
| 功能名称 | 开启方式 |说明 |
|----------|----------|------|
| Pilot Assist (领航辅助) |方向盘左侧按键启动 |自适应巡航+车道居中保持 |
| Adaptive Cruise Control (自适应巡航) |方向盘左侧按键切换 |仅控制车速与前车距离 |
| Lane Keeping Aid (车道保持) |默认开启 /可在设置中调整强度 |- |
| Blind Spot Warning (盲点监测) ||后视镜内侧指示灯亮起提醒 |
使用Pilot Assist：
1.  按下方向盘左侧中间的按钮激活ACC/Pilot Assist。
2.  用 +/-键设定速度（5km/h步进）。
3.  用右侧滚轮调节跟车距离（共5档）。
> ⚠️ Pilot Assist是辅助功能，驾驶员必须始终手握方向盘并观察路况。

## ⚠️六、重要注意事项
1. **无传统后风挡玻璃**
    Polestar4取消了后风挡玻璃。请依赖流媒体内后视镜和两侧外后视镜观察后方情况。流媒体后视镜亮度可在镜面底部调节。
2. **开门预警**
    下车前注意观察后方来车/来人。系统会通过声音和灯光提示侧方风险。
3. **紧急断电**
    若发生碰撞等紧急情况，高压系统会自动切断。如需手动断开12V蓄电池负极位于后备箱地板下方盖板内。
4. **轮胎维护**
    原厂配备静音轮胎或自修补轮胎。补胎时请告知维修人员使用专用设备以免损坏内部静音棉层。胎压标准值位于驾驶员侧B柱标签上。
5. OTA升级
    定期检查系统更新：“设置 > System > Software Update”。建议连接Wi-Fi进行下载以节省流量。升级期间车辆不可行驶请停在安全区域完成更新约需30分钟至1小时。

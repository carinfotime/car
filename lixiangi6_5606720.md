# 理想 i6 保养指南
  
更新时间：2026年07月12日17时14分03秒   

## 一、保养周期总览
理想 i6 作为增程式电动 SUV，其保养逻辑与传统燃油车有所不同。核心原则是：**以时间或里程先到者为准**。
| 项目 | 周期（公里/月） | 备注 |
| :--- | :--- | :--- |
| **常规保养（小保）** | 每 **10,000 km** 或 **12个月** | 更换机油、机滤，检查三电系统 |
| **大保养** | 每 **40,000 km** 或 **48个月** | 在小保基础上增加空调滤芯、火花塞等 |
| **制动液更换** | 每 **2年**（不限里程） | DOT4 / DOT5.1 |
| **防冻液更换** | 每 **4年 / 80,000 km** | - |
| **减速器油更换** | 每 **40,000 km** | - |
> ⚠️ *注：以上为官方推荐标准周期。若长期在恶劣环境（高温、多尘、频繁短途）下使用，建议适当缩短部分项目的周期。*

## 二、常规保养项目详解
### ✅ A类保养（10,000 km / 12个月）
- [x] **更换发动机机油及机滤**
  增程器（1.5T四缸发动机）需定期更换机油。建议使用标号：`0W-20`（全合成），容量约 `4.5L`。
- [x] **检查高压电池系统**
  包括电池冷却管路密封性、电池健康度（SOH）、绝缘电阻。
- [x] **检查驱动电机及减速器**
  有无异响、漏油。
- [x] **检查底盘及悬挂**
  紧固螺栓扭矩，检查减震器是否漏油。
- [x] **轮胎换位与胎压校准**
  建议前后交叉换位；标准胎压：`2.5 bar`（冷态）。
- [x] **更新车载系统 OTA**
  确认车机版本为最新。
### ✅ B类保养（40,000 km / 48个月）
在 A类基础上增加：
- [x] **更换空调滤芯**
  含 PM2.5过滤层，建议使用原厂件。
- [x] **更换火花塞**
  增程器火花塞寿命约 `40,000 km`。
- [x] **清洗节气门 & EGR阀**
  防止积碳导致怠速不稳或油耗偏高。
- [x] **更换减速器油**
  约 `1L`，标号 `75W-90` GL‑5。
- [x] **深度检测高压线束及连接器**
  排查氧化或松动风险。

## ✨四 、日常自查清单 （每月一次）
1️⃣胎压检测 ：冷态下保持前后轮均为2 .5 bar左右 ，若长期满载可适当调高至2 .7 bar 。
2️⃣灯光检查 ：日行灯 、远近光 、转向灯 、刹车灯均正常点亮 。
3️⃣玻璃水补充 ：冬季使用 -25℃防冻型 。
4️⃣充电口清洁 ：用干布擦拭充电口内部 ，防止异物导致充电故障 。
5️⃣增程器启动测试 ：如果长期纯电行驶 ，建议每月强制启动增程器运行15分钟 ，润滑发动机部件 。

## 🛠五 、常见问题与注意事项
🔹关于“终身质保”条件：
必须严格按照官方App内记录的保养计划进行 ，且每次保养需在授权服务中心完成 。自行在外维修可能导致质保失效 。
🔹关于“强制纯电”模式：
当电量低于SOC阈值时增程器会强制介入 。此时若继续行驶 ，注意观察机油温度是否正常 。
🔹关于OTA升级：
部分升级包会包含电池管理策略优化 ，升级后可能出现续航显示变化 ，属正常现象 。
🔹冬季用车提示：
低温下电池活性下降属物理特性 。建议出发前使用“远程电池预热”功能 ，可改善充放电效率并减少能耗损失 。

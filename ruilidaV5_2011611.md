# 睿立达V5 驾驶注意事项
  
更新时间：2026年07月15日11时47分26秒   

## 一、启动前检查
### 1. 车辆外部检查
- **轮胎状态**：检查胎压是否正常（建议冷车胎压2.3-2.5bar），轮胎表面无鼓包、裂纹或异常磨损
- **灯光系统**：确认前大灯、尾灯、转向灯、刹车灯及雾灯工作正常
- **车身外观**：检查车身有无明显损伤，后视镜及摄像头清洁无遮挡
- **充电接口**：如为新能源车型，确认充电盖已关闭且密封良好
### 2. 车内检查
- **座椅与后视镜**：调整至舒适驾驶姿势，确保内外后视镜视野清晰
- **安全带**：全车乘员必须系好安全带，儿童需使用安全座椅
- **仪表盘警示灯**：通电自检后确认无异常警示灯常亮（如电池故障、制动系统警告等）
- **电量/油量**：确认续航里程足够本次行程，提前规划补能站点
## 二、驾驶操作规范
### 1. 起步与行驶
| 项目 | 注意事项 |
|------|----------|
| **起步** | 松开电子手刹，轻踩加速踏板，避免急加速 |
| **车速控制** | 根据路况合理控制车速，避免超速行驶 |
| **跟车距离** | 保持至少2秒安全车距，雨天/雾天适当增加 |
| **变道超车** | 提前3秒打转向灯，观察盲区后平稳变道 |
### 2. 制动与停车
- **预见性制动**：提前松加速踏板，利用动能回收（如配备）减速
- **紧急制动**：双手握稳方向盘，全力踩下制动踏板，切勿猛打方向避让
- **坡道停车**：拉紧电子手刹或机械手刹，挂P挡（自动挡）或挂挡别轮（手动挡）
### 3. 特殊路况驾驶
- **雨天行驶**
  - 开启雨刮器及雾灯/近光灯
  - 降低车速（建议低于正常时速20%）
  - 避免急刹车和急转向，防止侧滑
- **夜间行驶**
  - -合理切换远近光灯（会车时提前150米切换近光）
- **山路/弯道**
##  三、智能系统使用须知
### ADAS辅助驾驶功能（如配备）
1. LKA车道保持辅助：
   - LKA仅为辅助功能，
   -  驾驶员双手不得长时间离开方向盘
2. ACC自适应巡航：
   - ACC在弯道、
   雨雪天气下性能受限，
   需随时准备接管车辆
3. AEB自动紧急制动：
   AEB对静止车辆、
   行人及横穿物体的识别存在局限性，
   不可完全依赖
## ⚠️四、安全警告⚠️
1.
  严禁酒后驾车、
   毒驾或疲劳驾驶（连续驾驶不超过4小时）
2.
  行车过程中禁止使用手机，
   如需导航请使用车载系统
3.
  儿童不得坐在副驾驶位置，
   12岁以下儿童必须使用安全座椅
4.
  车内不要放置易燃易爆物品（打火机、
   香水、
   充电宝等）
5.
  发生事故时立即开启双闪灯，
   放置三角警示牌（城市道路50米外，
   高速150米外），人员撤离至护栏外报警

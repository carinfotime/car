## 一、保养周期总览
| 项目 | 周期（公里/时间） | 说明 |
| :--- | :--- | :--- |
| **首保** | **5000公里 或 6个月**（以先到者为准） | 免费更换机油、机滤，并进行全车检查 |
| **常规保养** | **每5000公里 或 6个月**（以先到者为准） | 更换机油、机滤，检查各项油液及部件 |
| **中期保养** | **每20000公里** | 在常规保养基础上增加空气滤芯、空调滤芯更换 |

> **注意：** “时间”和“里程”以先到者为准。即使行驶里程较少，也建议按时进行保养，防止油液变质或橡胶件老化。

## 四、日常自检清单（建议每月一次）
1. ✅   冷车状态下拔出机油尺，检查机油量是否在上下刻度之间。
2. ✅   观察冷却液储液罐液面是否在“MAX”与“MIN”之间。
3. ✅   启动车辆后观察仪表盘是否有故障灯亮起。
4. ✅   原地转动方向盘，听是否有异响。
5. ✅   按压车身四个角，看减震回弹是否正常。
6. ✅   绕车一周观察轮胎有无鼓包、裂纹或异常磨损。

## ⚠️五、重要注意事项
1. **[质保期内]** ：务必前往风行授权服务站进行保养，否则可能影响整车质保权益。
2. **[机油选择]** ：推荐使用符合 ACEA A3/B4 (C3)标准或 API SN/SM级别以上的全合成机油。粘度等级通常为：
   - `5W‑30` （四季通用）
   - `10W‑40` （南方地区夏季使用）
3. **[正时皮带]** ：若您的车型搭载的是正时皮带而非链条，请务必在厂家规定的里程数前进行更换。通常为8~10万公里。一旦断裂将导致发动机严重损坏！
4. **[涡轮增压车型]** ：停车熄火前建议怠速运转1~2分钟再熄火，有助于保护涡轮增压器。

| 项目 | 参数 |
|------|------|
| 车长 | 4730 mm |
| 车宽 | 1820 mm |
| 车高 | 1505 mm |
| 轴距 | 2700 mm |
| 前轮距 | 1555 mm |
| 后轮距 | 1566 mm |

## 二、乘坐空间
### 前排空间
- **头部空间**：约980mm（座椅调至最低时可达1000mm以上）
- **腿部空间**：充裕，驾驶位支持6向调节，可满足1.85m身高驾驶者舒适驾驶
- **肩部空间**：约1420mm，横向宽敞，主副驾互不干扰
### 后排空间
- **腿部空间**：约890mm（以身高1.75m乘客为标准），膝部距离前排座椅约两拳余量
- **头部空间**：约920mm，得益于车顶内凹设计，后排头部表现良好
- **肩部空间**：约1380mm，后排坐满三位成年人不会过于拥挤
- **后排中央位置**：地板凸起较低（约50mm），中间乘客舒适性尚可
## 三、储物空间
### 车内储物格
| 位置 | 容量/特点 |
|------|-----------|
| **手套箱** | ≈8L，可放置A4文件及日常杂物 |
| **中央扶手箱** | ≈3L，深度适中，可放小物件 |
| **前排杯架** | ×2，可放置标准水杯/饮料瓶 |
| **后排杯架** | ×2（中央扶手带杯架） |
| **车门储物槽** | ×4（前后门），前门可放500ml水瓶+杂物，后门略小 |
| **中控台储物格** | ≈1L，适合放手机、钥匙等小件物品 |
### 💡其他便利储物设计
- **眼镜盒**：驾驶位头顶左侧
- **卡片槽**：中控台下方或挡把前方区域
- **前排座椅背袋**：后排乘客使用
## 四、后备箱
### 📦后备箱参数
- **标准容积**：500L
- **最大纵深**：约1050mm
- **最大宽度**：约1350mm（轮拱处最窄约1020mm）
- **高度（开口至底板）**：约560mm
- **开口宽度**：约1100mm
### 🧳装载能力参考
> - ✅可同时放入28寸+24寸+20寸行李箱各一个  
> - ✅支持后排座椅4/6比例放倒（放倒后纵深扩展至约1800mm）  
> - ❌放倒后与后备箱地板存在一定台阶差  
> - ⚠️备胎位于底板下方（非全尺寸备胎）

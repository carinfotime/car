# 瑞风M3 空间参数介绍
  
更新时间：2026年07月19日08时03分46秒   

## 一、整车尺寸参数
| 项目 | 参数 |
| :--- | :--- |
| **车身长度** | 4715 mm / 5145 mm（长轴版） |
| **车身宽度** | 1765 mm |
| **车身高度** | 1900 mm / 1935 mm（带行李架） |
| **轴距** | 2810 mm / 3110 mm（长轴版） |
## 二、内部乘坐空间
### 1. 座椅布局
- **标准版：** **2+2+3**（7座）
- **可选布局：** **2+3+3**（8座）、**2+2+2+3**（9座）
### 2. 头部与腿部空间
- **第一排：**
    -  头部空间：约 **1050 mm**
    -  腿部空间：充裕，驾驶姿态舒展
- **第二排：**
    -  独立座椅设计，支持前后滑动
    -  最大腿部空间可达 **850-950 mm**
- **第三排：**
    -  头部空间约 **1000 mm**
    -  腿部空间约 **700-800 mm**（支持靠背角度调节）
## 三、载货与储物能力
### 1. 后备箱容积
- **7座状态下：**
    -  纵深约 **450 mm**
    -  可放置多个登机箱或日常杂物
- **第三排折叠/侧挂后：**
    -  最大纵深扩展至约 **1500 mm**
    -  容积提升至约 **1700 L**
### 2. 内部最大装载尺寸
> *以下为拆除/折叠第三排后的数据*
| 项目 | 数值 |
| :--- | :--- |
| ****最大进深**** | ≈1850 mm |
| ****最大宽度**** | ≈1550 mm |
| ****内部高度**** | ≈1280 mm |
### ✅装载能力参考：
-  可轻松容纳标准托盘货物
-  可直立放置小家电、工具箱等物品
-  长轴版可容纳长度超过2米的管材或板材
## ⚙️四、其他实用设计
### 🚪车门与开口
- ****侧滑门设计****：
    -  开口宽度约 **700mm**
    -  方便狭窄车位上下车及装卸货物
- ****尾门开口****：
    -  下沿离地高度较低，便于搬运重物
### 💺座椅灵活性
- ****第三排座椅****：
    - ✅支持整体折叠前翻或向两侧挂起（视具体配置）
    - ✅折叠后地板平整度高，利于码放货物
- ****第二排座椅****：
    - ✅独立座椅可前后滑动及调节靠背角度

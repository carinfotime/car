# 龙耀5 内饰参数介绍
  
更新时间：2026年07月13日09时29分10秒   

## 一、座椅系统
| 项目 | 参数 |
|------|------|
| 座椅材质 | Nappa真皮 + 翻毛皮混搭 |
| 前排座椅功能 | 电动12向调节、加热、通风、记忆功能 |
| 后排座椅 | 4/6比例放倒，中央扶手带杯架 |
| 座椅布局 | 2+3五座布局 |
| 头枕调节 | 前后/上下四向调节 |
## 二、仪表台与中控
| 项目 | 参数 |
|------|------|
| 仪表盘 | 12.3英寸全液晶仪表盘 |
| HUD抬头显示 | W-HUD彩色投影显示 |
| 中控屏尺寸 | 14.6英寸悬浮式触控屏 |
| 车机系统 | DragonOS智能座舱系统 |
| CPU芯片 | Qualcomm骁龙8155 |
## 三、方向盘
- **材质**：真皮包裹
- **调节方式**：手动四向调节（上下+前后）
- **多功能按键**：音量控制、语音唤醒、定速巡航控制
- **换挡拨片**：标配金属换挡拨片
## 四、空调系统
- **类型**：双区自动恒温空调
- **后排出风口**：标配
- **PM2.5过滤**：CN95级高效滤芯
- **负离子发生器**：标配
- **香氛系统**：三种香型可调（选装）
## 五、音响系统
- **扬声器数量**：8个（标配）/12个（高配BOSE音响）
- **功放功率**：320W / BOSE版710W
- **主动降噪**：ANC主动降噪技术（高配）
## 六、灯光与氛围
- **氛围灯颜色**：64色可调
- **氛围灯区域**：中控台、门板饰条、脚窝灯
- **阅读灯类型**：LED触摸式阅读灯
## 七、储物空间
| 位置 |容积/说明|
|---|---|
|手套箱容积 |约8L，带阻尼开启|
|中央扶手箱容积 |约12L，内置USB-C接口×2|
|门板储物格容量 |前门可放1.5L水瓶×2，后门可放1L水瓶×2|
|杯架数量 |前排2个 +后排中央扶手2个|
## 八、其他内饰配置
-**天窗类型**:全景天幕（不可开启）/可选电动天窗
-**遮阳帘**:电动遮阳帘（全景天幕版）
-**内后视镜**:自动防眩目内后视镜
-**手机无线充电**:前排15W无线充电板
-**USB接口数量**:前排Type-C×2 + Type-A×1；后排Type-C×1 + Type-A×1

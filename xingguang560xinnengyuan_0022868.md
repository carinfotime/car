# 星光560新能源 - 儿童乘车安全注意事项
  
更新时间：2026年07月20日10时31分10秒   

为了保障您孩子的乘车安全，请务必遵守以下事项：
## 一、儿童安全座椅
- **强制使用**：12周岁以下或身高低于1.4米的儿童，必须使用符合国家标准的儿童安全座椅。
- **反向安装**：建议0-3岁（体重18公斤以下）的婴幼儿使用反向安装的儿童安全座椅，以更好地保护头颈。
- **正确固定**：确保安全座椅与车辆ISOFIX接口或安全带牢固连接，无松动。
## 二、车内乘坐规范
- **禁止坐副驾**：12岁以下儿童严禁坐在副驾驶位置。副驾驶的安全气囊在弹出时可能对儿童造成严重伤害。
- **后排落座**：所有儿童必须坐在后排座位。
- **系好安全带**：使用安全座椅的儿童按座椅说明固定；身高足够直接使用安全带时，确保肩带跨过肩膀和胸部中央，腰带紧贴大腿根部。
## 三、车窗与天窗
- **防止夹伤**：电动车窗具有防夹功能，但仍需看管好孩子的手和头，避免伸出窗外。
- **禁止探头**：严禁将头、手或身体任何部位伸出天窗或侧窗外。
- **启用童锁**：建议开启后排车窗的“玻璃锁止”功能（主驾位控制），防止孩子误操作。
## 四、车门安全
- **开启童锁**：务必启用后门的儿童安全锁。开启后，车门无法从车内打开，只能从车外开启。
- **下车观察**：教育孩子下车前先通过后视镜观察后方来车和行人，确认安全后再开门。
## 五、停车与驻车
- **切勿独留车内**：无论时间长短、天气如何（尤其夏季高温），切勿将孩子单独留在车内。车内温度会迅速升高至危险水平。
- **熄火拔钥匙**：停车熄火后请随身携带钥匙/智能卡，防止孩子在车内误触启动按钮或挂挡。
## 六、充电安全（新能源专属）
- **远离充电口**：充电过程中及充电结束后一段时间内，请勿让儿童触摸充电枪插头及车辆充电口金属端子。
- **禁止玩耍线缆**：不要让孩子拉扯、踩踏充电线缆或在充电桩附近奔跑打闹。
- **高压警示**：告知孩子车辆底部电池及高压部件不可触碰。如发生碰撞或底盘刮擦后出现异味/烟雾/异响，立即远离车辆并通知成人处理。
## 七、其他注意事项
| 项目 | 说明 |
|------|------|
| 车内物品 | 避免放置尖锐玩具或重物在座位上（急刹车时可能飞溅伤人） |
| 中控屏 | 行驶中勿让孩子操作中控大屏或方向盘上的按键 |
| 零食饮水 | 行驶中尽量避免进食果冻、棒棒糖等易噎食的食物 |
| 应急演练 | 教会年龄稍大的孩子如何解锁车门（机械拉手位置）以及如何鸣笛求助 |

# 🚗 大道新能源 · 儿童乘车安全注意事项
  
更新时间：2026年07月14日17时59分01秒   

为了保障您和孩子的出行安全，请务必遵守以下事项：
## ⚠️ 行车前检查
- **确认儿童锁已开启**：上车后请确保两侧后门的儿童安全锁已拨至“锁定”位置，防止孩子在行驶中意外开门。
- **调整座椅与安全带**：根据孩子身高调整后排座椅，确保安全带肩带跨过肩膀中部（而非颈部），腰带紧贴大腿根部。
- **清理车内小物件**：收起中控台、座椅缝隙中的硬币、纽扣等小物品，避免孩子误吞。
## 👶 儿童座椅使用规范
| 年龄 / 身高 | 必须使用 | 安装方向 |
| :--- | :--- | :--- |
| **0-15个月** | 反向式儿童安全座椅 | 反向安装（面朝车尾） |
| **1-4岁 / 9-18kg** | 正向式儿童安全座椅 | 正向安装（面朝车头） |
| **4-12岁 / 15-36kg** | 增高垫 + 成人安全带 | — |
> **注意**：请严格按照产品说明书安装，并确认ISOFIX接口（大道新能源车型均配备）已锁紧。
## 🚫 行驶中禁止行为
1. **严禁将头、手伸出天窗或车窗**——即使开启“通风模式”也不行。
2. **禁止在车内站立或跪在座位上**——急刹车时极易造成颈椎损伤。
3. **不要让孩子操作中控屏幕或换挡旋钮**——建议开启“后排娱乐屏童锁”（如有配备）。
4. **切勿将孩子单独留在车内**——夏季车内温度可在10分钟内升至60℃以上。
## 🌡️ 空调与空气质量
- **空调温度建议设定在22-26℃**，避免直吹孩子头部。
- **定期更换空调滤芯**（建议每6个月或10,000公里），保持车内空气清新。
- **如车辆配备PM2.5过滤功能**，建议长途出行前开启“座舱净化模式”。
## 🔋 充电时的特别提醒
- **充电过程中请勿让孩子靠近充电口**——高压电缆存在触电风险。
- **不要在车内玩水或湿手触摸充电接口。**
- **充电枪未使用时请收纳至后备箱专用位置。**
## 📋 应急准备

| ✅  安抚玩具（软质无尖锐角） |  分散注意力，减少哭闹 |

# 远程锋锐 操作指南
  
更新时间：2026年07月17日18时50分47秒   

## 一、车辆基本介绍
远程锋锐是一款面向城市物流与短途运输的纯电动微卡。本指南涵盖日常驾驶、充电操作、仪表盘解读及安全注意事项。
---
## 二、启动与关闭车辆
### 2.1 启动车辆
1. 确保车辆处于 **P挡（驻车挡）**。
2. 将智能钥匙靠近驾驶室感应区或插入钥匙槽。
3. 踩下制动踏板，按下 **“启动/停止”按钮**。
4. 仪表盘点亮，READY指示灯亮起，表示车辆已进入可行驶状态。
### 2.2 关闭车辆
1. 将挡位置于 **P挡**。
2. 按下 **“启动/停止”按钮**，仪表盘熄灭。
3. 确认所有用电设备已关闭后，锁好车门离开。
> ⚠️ **注意**：停车熄火前请确保电子手刹已拉起。
---
## 三、驾驶操作
### 3.1 挡位说明
| 挡位 | 功能 |
|------|------|
| P | 驻车挡 |
| R | 倒车挡 |
| N | 空挡 |
| D | 前进挡（经济模式） |
| S | 运动模式（动力更强） |
### 3.2  换挡操作
- **换挡前必须踩下制动踏板**。
- D/S切换可在行驶中直接拨动换挡旋钮完成。
### 3.3  电子手刹
- **拉起**：向上拨动电子手刹开关。
- **释放**：踩下制动踏板后向下按压开关；或挂入D/R挡后轻踩加速踏板自动解除。
###  3.4  能量回收调节
-通过方向盘左侧的“能量回收等级调节按键”可在 L1 / L2 / L3之间切换：
    - L1：滑行距离长，回收强度低
    - L2：中等回收（推荐日常使用）
    - L3：强回收（类似单踏板感觉）
---
##  四、充电操作
###  4.1  交流慢充（家用/交流桩）
1.  车辆熄火并挂入P挡。
2.  打开充电口盖（位于车头 LOGO后方或车身侧围）。
3.  将交流充电枪插入充电口直至锁止到位。
4.  充电桩侧刷卡或扫码启动充电。
5.  仪表盘显示充电状态及剩余时间。
6.  充满后自动断电，拔枪并盖好充电口盖。
###  4.2  直流快充（公共快充桩）
1.  熄火挂P挡，打开直流快充口盖。
2.  将直流快充枪插入快充接口（注意防尘塞取下）。
3.  按充电桩提示启动充电。
4.   电量从20%充至80%通常约需30~40分钟（视环境温度而定）。
5   结束充电后拔枪归位并关闭盖子。
> ⚠️ **注意**：
> -    严禁在雷雨天进行户外充电。
> -    请使用符合国标的正规充电设备。

##   五、仪表盘与指示灯解读
###   5 .1   主要显示区域
-    左侧：电机转速 /功率输出指示
-    中间：车速数字显示
-    右侧：电池电量百分比及续航里程估算
###   5 .2   常见指示灯说明
|     指示灯图标      |      含义      |      处理方式      |
|---|---|---|
| READY (绿色) |     系统就绪可行驶      |      正常      |
|       红色电池图标        |     动力电池故障      |      立即停车联系售后      |
|       黄色扳手图标        |     系统需要保养/故障      |      尽快进站检查      |
|       红色感叹号圆圈        |     制动系统故障/电子手刹异常      |      停车检查刹车及手刹状态      |
|       黄色乌龟图标        |     功率限制/限速模式      |      电量过低或系统保护，低速行驶至最近充电点 |

##    六、安全注意事项
1．    每次出车前检查轮胎气压及外观是否正常。
2．    涉水深度不得超过轮毂中心线；避免长时间浸泡电池包区域 。
3．    禁止私自改装高压电路或加装大功率电器 。
4．    长期停放建议保持电量在40%~60%之间 ，每三个月进行一次补电 。
5．    如发生碰撞导致高压系统报警 ，请勿触碰车身金属部分 ，立即远离并拨打救援电话 。
---
##    七 、应急处理
###        7 .1        车辆无法启动
-        检查智能钥匙电量是否充足 。
-        尝试将钥匙贴近感应区再按启动按钮 。
-        确认12V小电瓶是否有电 （可通过按喇叭判断）。
###        7 .2        行驶中动力中断
-        保持冷静 ，利用惯性滑行至安全区域 。
-        开启双闪警示灯 ，放置三角警示牌 。
-        尝试重新上电 ：挂P挡 →熄火 →等待10秒 →再次启动 。
###         7 .3         拖车注意事项
-         必须使用平板拖车 ，严禁直接拖拽驱动轮 。
-         如无法挂入N挡 ，需由专业人员解除电机锁定后再拖运 。

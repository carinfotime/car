# 丰田卡罗拉锐放（Corolla Cross）操作指南
  
更新时间：2026年07月10日17时06分33秒   

## 一、启动与熄火
### 启动车辆
1. **确认挡位**：确保挡位在 **P（驻车挡）**。
2. **踩下制动踏板**：必须踩住刹车。
3. **一键启动**：按下 **ENGINE START STOP** 按钮，仪表盘指示灯亮起即启动成功。
### 熄火车辆
1. **挂入P挡**：将挡位切换至 P。
2. **拉起电子手刹**（或脚刹）。
3. **按下启动按钮**关闭发动机。
---
## 二、驾驶操作
### 换挡操作
| 挡位 | 用途 |
|------|------|
| P | 驻车挡，长时间停车使用 |
| R | 倒车挡，需车辆完全停止后挂入 |
| N | 空挡，短暂停车或拖车时使用 |
| D | 前进挡，日常行驶使用 |
| M / S | 手动模式 / 运动模式，可手动加减挡 |
> ⚠️ **换挡时必须踩住制动踏板。**
### 电子手刹（EPB）
- **启用**：向上拉起电子手刹开关。
- **解除**：踩下刹车，向下按压开关；或挂D挡后轻踩油门自动解除。
### Auto Hold（自动驻车）
- **开启**：按下 Auto Hold 按钮（指示灯亮）。
- **作用**：等红绿灯时松开刹车车辆仍保持制动，轻踩油门即可解除。
---
## 三、灯光与雨刷
### 灯光控制杆
- **旋转旋钮**：
  - OFF → AUTO（自动大灯） → 示宽灯 → 近光灯
- **拨杆向前推**：远光灯常亮
- **拨杆向后拉**：远光灯闪一下（超车示意）
### 转向灯
- **向下拨**：左转向灯
- **向上拨**：右转向灯
- **轻拨不卡位**：转向灯闪三次后自动熄灭（变道辅助）
### 雨刷控制杆
- **向上推一档**：间歇刮刷（可调节间隔速度）
- **向上推二档**：低速连续刮刷
- **向上推三档**：高速连续刮刷
- **向内拉喷水+刮刷**
---
## 四、空调与多媒体
### 空调系统
|  功能   |  操作方式   |
|---------|-------------|
| A/C开关 |  按下点亮为制冷模式 |
| AUTO    |  自动恒温模式 |
| TEMP    |  旋转调节温度（左冷右热） |
| FAN     |  调节风量大小 |
| MODE    |  切换出风方向（吹面/吹脚/除雾） |
| FRONT除雾 |  快速清除前挡风玻璃雾气 |
| REAR除雾   |  后窗加热除雾 |
> 💡 *冬季取暖无需开A/C，调高温度即可。*
### Apple CarPlay / Android Auto
1. iPhone用户使用USB数据线连接扶手箱内USB接口。
2. Android用户同样通过USB连接。
3. （部分车型支持无线连接，需在设置中开启蓝牙/WiFi配对。）

## 五、安全辅助系统
### Toyota Safety Sense（TSS）
默认开启，可通过方向盘右侧按键调整：
#### LTA（车道追踪辅助）
- Lane Departure Alert + Steering Assist  
- `方向盘按键` → `设置` → `LTA` → `ON/OFF`
#### DRCC（动态雷达巡航控制）
1. `方向盘右侧`按 `巡航开关`。
2. `SET/-`设定当前速度为巡航速度。
3. `+RES`加速 / `SET/-`减速。
4. `CANCEL`取消巡航。
#### PCS（预碰撞安全系统）
默认开启。通过仪表盘菜单可调整灵敏度或关闭。

## ⚠️六、仪表盘常见指示灯说明
|   图标颜色    |    含义    |
|---------------|------------|
| 🟢绿色/蓝色     |    功能正常启用中      |
| 🟡黄色          |    警告/提醒，需尽快检查      |
| 🔴红色          |    严重故障，立即停车检查      |
常见图标：
- ⚠️ *感叹号* — 制动系统异常  
- 🛞 *胎压报警* —轮胎气压不足  
- 🔋 *电池灯* —充电系统故障  
- 🛢️ *机油灯* —机油压力过低

## 📋七、日常维护建议
### ✅每5000km或6个月检查：
- ✅机油液位  
- ✅冷却液液位  
- ✅制动液液位  
- ✅轮胎气压及磨损情况  
### ✅每10000km保养项目：
✅更换机油及机滤  
✅轮胎换位  
✅空气滤芯清洁/更换  
### ✅每20000km保养项目：
✅空调滤芯更换  
✅刹车片厚度检查

## 🧰八、应急处理
### 🔋蓄电池亏电无法启动：
1. 🔗准备搭电线。  
2. 🔴先接正极(+)→正极(+)。  
3. ⚫再接负极(-)→搭铁点(车身金属)。  
4. 🚗救援车启动后等待几分钟再尝试启动本车。  
> ❗拆卸顺序相反：先拆负极→再拆正极。
### 🛞轮胎漏气/爆胎：
1. 📍安全靠边停车。  
2. ⚠️打开双闪警示灯。  
3. 🔧取出备胎及千斤顶工具。  
4. 🔩按对角线顺序松螺丝→顶起车辆→更换备胎→拧紧螺丝。  
> 💡 *部分车型无备胎，配备补胎液和气泵，请参照随车说明书使用。*
---
## 📌九、省油驾驶技巧
1️⃣平稳加速和减速，避免急踩油门急刹车。  
2️⃣保持经济时速60~90km/h行驶最省油。  
3️⃣合理使用定速巡航功能保持匀速行驶。  
4️⃣减少不必要的怠速热车时间。  
5️⃣定期检查胎压保持在标准值范围内。

## 二、各阶段详细保养内容
### 🔧 首次保养（3个月 / 5000公里）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查变速箱油液位
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查灯光系统是否正常
- [ ] 检查雨刮器及喷水系统
- [ ] 紧固底盘螺栓
- [ ] **注意：首次保养免费，务必按时到授权服务站进行**
### 🔧 A类常规保养（每6个月 / 5000公里）
#### ✅必做项目：
1. **更换发动机机油** —推荐规格：5W-30（API SN级或以上）
2. **更换机油滤清器**
3. **空气滤清器清洁/更换** —每10000公里更换一次，中间可清洁
4. **空调滤清器清洁/更换** —每10000公里更换一次，中间可清洁
5. **全车灯光检查**
6. **轮胎气压检测与调整** —标准胎压：前轮2.3bar，后轮2.5bar
7. **底盘及悬挂系统检查**
8. **制动系统检查**（刹车片厚度≥3mm）
9. **电瓶电压检测**
10. **各种油液液位检查与补充**
#### ⚠️视情况项目：
-  四轮换位（建议每10000公里进行一次）
-  节气门清洗（出现怠速不稳时进行）
### 🔧 B类大保养（2年 / 40000公里）
在A类基础上增加以下项目：
1. ✅ **更换火花塞** —建议使用原厂型号：DK7RTC或同等规格
2. ✅ **更换制动液** —DOT4级别，约需1L
3. ✅ **更换防冻液/冷却液** —约需6L，建议使用长效型(-35℃)
4. ✅ **更换变速箱油**
   - *手动挡*：GL-4级别，约需2L，每40000km更换
   - *CVT自动挡*：专用CVT油，约需4L，每40000km更换（严禁用普通ATF代替）
5. ✅ **转向助力油检查/更换** —ATF DEXRON III级别，约需1L
6. ✅ **燃油滤清器更换**
7. ✅ **正时皮带/链条检查**
   - *正时皮带*：60000km时需重点检查，80000km建议强制更换
   - *正时链条*：一般免维护，但需定期检查张紧度及异响

## 📌四、日常自检清单（每月一次）
1️⃣ 【机油】冷车状态下拔出机油尺查看——应在上下刻度线之间  
2️⃣ 【冷却液】观察膨胀罐液面——应在MIN与MAX之间  
3️⃣ 【制动液】观察储油罐——不应低于MIN线  
4️⃣ 【轮胎】目测胎压是否明显不足、有无鼓包或异常磨损  
5️⃣ 【灯光】请他人协助查看前后灯是否全部正常工作  
6️⃣ 【雨刮水】及时补充玻璃水

## 📌五、重要注意事项
### ❗关于保修：
1. 《保修手册》规定的定期保养必须在东风风光授权服务站进行并盖章记录。
2. 《保修手册》中规定的首保必须按时完成，逾期未做将视为自动放弃保修权利。
3. 《保修手册》中规定的强制保养项目未按期执行导致的故障不在保修范围内。
### ❗关于配件：
1. 🔴强烈建议使用原厂配件或符合原厂规格的知名品牌配件。
2. 🔴劣质配件可能导致发动机严重损坏。
### ❗关于驾驶习惯对车辆的影响：
1. 🚫避免长时间低转速高档位行驶。
2. 🚫避免冷车启动后立即高转速行驶。
3. 🚫避免长期短途行驶导致电瓶亏电和积碳增加。
### ❗特殊环境下的额外维护：
#### 🌡️高温地区/夏季：
-  缩短空气滤芯和空调滤芯的清洁间隔至5000km一查。
-  注意冷却风扇运转是否正常。
#### ❄️寒冷地区/冬季：
-  入冬前务必检测防冻液的冰点值是否符合当地最低气温要求。
-  入冬前建议提前预热车辆后再起步行驶。
#### 🌪️多尘环境：
-  缩短空气滤芯的清洁间隔至3000~5000km一查。
-  缩短空调滤芯的清洁间隔至3000~5000km一查。

## 📌六、常见问题与警示信号
⚠️遇到以下情况请尽快进站检修：
🔴【仪表盘报警灯亮起】
> - ✔发动机故障灯 → ECU自检异常  
> - ✔ABS故障灯 → ABS系统异常  
> - ✔安全气囊灯 → SRS系统异常  
🔴【驾驶感受异常】
> - ✔加速无力 →可能原因:节气门积碳、火花塞老化、燃油泵压力不足  
> - ✔刹车变软或有异响 →可能原因:刹车片磨薄、刹车管路进气  
> - ✔方向盘抖动 →可能原因:轮胎动平衡失准、转向拉杆球头松旷  
🔴【漏油漏水】
> - ✔发现地面有滴漏痕迹应立即排查来源

# 迈莎锐库里南 (Mansory Rolls-Royce Cullinan) 操作指南
  
更新时间：2026年07月14日14时00分06秒   

## 一、车辆概述与注意事项
迈莎锐库里南是基于劳斯莱斯库里南进行深度定制改装的高性能豪华SUV。本指南旨在帮助您快速熟悉车辆的特殊操作与功能。
**重要提示：**
- 本车为高性能改装车型，驾驶前请确保已熟悉所有操作。
- 由于采用了更宽的车身套件和特殊轮毂，需特别注意车辆的通过性及轮距。
- 建议使用高标号无铅汽油（推荐98号或以上）。
---
## 二、启动与熄火
### 启动步骤
1.  携带智能钥匙进入车内。
2.  确保变速箱处于 **P（驻车）** 档位。
3.  脚踩刹车踏板。
4.  按下位于中控台或方向盘右侧的 **“ENGINE START/STOP”** 按钮。
5.  发动机启动后，仪表盘会进行自检并点亮。
### 熄火步骤
1.  将车辆完全停稳。
2.  挂入 **P（驻车）** 档位。
3.  再次按下 **“ENGINE START/STOP”** 按钮，发动机关闭。

## 三、驾驶模式选择
通过中控台上的驾驶模式旋钮或触摸屏选择不同的驾驶模式，以适应不同路况：
| 模式 | 适用场景 | 特点 |
| :--- | :--- | :--- |
| **舒适 (Comfort)** | 日常城市道路、长途巡航 | 悬挂最软，转向轻盈，排气声浪低沉 |
| **运动 (Sport)** | 高速公路、激烈驾驶 |悬挂变硬，油门响应更灵敏，排气阀门打开，声浪高亢 |
| **越野 (Off-Road)** |非铺装路面、轻度越野 |提升底盘高度（如配备），优化牵引力控制 |
| **自定义 (Custom)** |个性化需求 |可单独调节悬挂、转向、动力响应等参数 |
> **注意：** Mansory改装版通常拥有更激进的排气系统。在运动模式下，排气声浪会显著增大。

## 四、特殊功能操作
### （一）车门开启与关闭
- **对开门（Coach Doors）：**后门为向后开启式。从外部拉动门把手即可打开。关闭时需注意后方障碍物。
- **电动吸合门：**所有车门轻掩至接近关闭位置时，会自动吸合锁止。请勿用力推拉。
### （二）“星光顶篷”控制
如选配了Mansory定制星光顶篷：
1.  可通过后排中央扶手上的触摸屏控制器开启/关闭灯光效果。
2.  可调节亮度及颜色（如支持RGB）。
3.  部分版本支持随音乐律动功能。
### （三）后排娱乐系统
1.  前排座椅头枕后方配备多媒体显示屏。
2.  使用后排中央扶手上的遥控器或触摸屏进行操作。
3.  支持HDMI输入及流媒体播放。
### （四）空气悬挂调节
1.  通过中控屏幕进入“车辆设置” -> “悬挂”。
2.  可选择升高底盘以应对复杂路况（车速超过40km/h时自动降低）。
3.  在“运动”模式下悬挂会自动降低约20mm以提升稳定性。

## 五、仪表盘与信息显示
- **主仪表盘：**采用传统机械指针与液晶屏结合的设计。显示车速、转速、油量、水温等核心信息。
- **抬头显示 (HUD)：**将导航信息、车速投射到前挡风玻璃上。可在车辆设置中调节亮度和位置。

## 六、安全辅助系统
本车配备全套主动安全系统：
- **自适应巡航控制 (ACC)：**设定跟车距离和速度，自动加减速。
- **车道保持辅助 (LKA)：**当车辆偏离车道时提供轻微转向修正或震动提醒。
- **夜视系统：**在夜间识别前方行人及动物并预警（如配备）。
- **360度全景影像：**低速行驶或倒车时自动激活，显示车身周围环境。
> *注意：Mansory改装可能涉及电子系统的调整，部分原厂辅助功能的灵敏度可能与标准版有所不同。*

## 七、保养与维护建议
由于Mansory对动力系统和外观进行了升级，保养周期和项目有所变化：
|项目|建议周期|
|:---|:---|
|发动机机油 &机滤|每5,000公里或6个月|
|刹车片 &刹车盘检查|每5,000公里|
|轮胎检查 &胎压校准(建议冷胎前2.6bar /后2.8bar)|每月一次|
|碳纤维外观件清洁 &养护(使用中性清洁剂)|每次洗车后|
|排气系统检查(检查有无松动异响)|每10,000公里|
> *重要提示：请务必前往具备Mansory授权资质的专业服务中心进行保养维修。*

## 八、常见问题解答 (FAQ)
### Q:如何切换手动换挡模式？
A:将换挡杆向左拨至手动模式位置（M/S），然后使用方向盘后的换挡拨片进行升档 (+)和降档 (-)。
### Q:如何重置胎压监测系统？
A:在中控屏幕中找到“车辆信息” -> “轮胎压力”，选择“初始化”。确保所有轮胎气压正常后再执行此操作。
### Q:为什么排气声音突然变大？
A:Mansory车型通常配备主动式排气阀门。当切换到运动模式或深踩油门时阀门会自动打开以增强声浪；舒适模式下则相对安静。
### Q:车身太低容易刮蹭怎么办？
A:通过空气悬挂升高底盘；在通过减速带或地库斜坡时务必斜向缓慢通过；加装的前唇和后扩散器离地间隙较小需格外小心。

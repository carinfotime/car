# 雷克萨斯RZ 空间参数介绍
  
更新时间：2026年07月18日15时09分01秒   

## 车身尺寸
| 参数 | 数值 |
|------|------|
| 车长 | 4,805 mm |
| 车宽 | 1,895 mm |
| 车高 | 1,635 mm |
| 轴距 | 2,850 mm |
| 前轮距 | 1,630 mm |
| 后轮距 | 1,640 mm |
| 最小离地间隙 | 约170 mm |
## 车内乘坐空间
- **前排头部空间**：约980 mm
- **后排头部空间**：约940 mm
- **前排腿部空间（最大）**：约1,080 mm
- **后排腿部空间**：约950 mm
- **后排膝部空间**：约60 mm
- **后排座椅宽度**：约1,420 mm（可舒适乘坐3名成人）
## 座椅布局与功能
- **座椅布局**：5座（2+3）
- **前排座椅调节方式**：电动8向调节（带腰部支撑）
- **后排座椅放倒比例**：40:20:40分段折叠放倒
- **后排座椅靠背角度调节**：支持手动调节
## 后备箱空间
| 状态 | 容积 |
|------|------|
| 标准状态（5座） | **522 L**（VDA标准） |
| 后排座椅放倒后 | **1,451 L**（至车顶） |
| 后备箱开口宽度 | ~1,100 mm |
| 后备箱开口高度 | ~750 mm |
| 后备箱纵深（标准） | ~950 mm |
| 后备箱纵深（放倒后） | ~1,800 mm |
## 储物空间亮点
- **中控台杯架**：前排2个，后排2个
- **中央扶手箱容积**：约6.5 L
- **手套箱容积**：约8 L
- **门板储物格容量**：
    -  前门：可容纳500 ml水瓶 +杂物
    -  后门：可容纳500 ml水瓶 +杂物
- **手机无线充电板位置**：中控台下方开放式储物槽内
## Frunk（前备箱）
>  雷克萨斯RZ作为纯电车型，取消了传统发动机，但并未设置前备箱。前机舱盖下为电机、电控及空调等部件。
---
###  总结评价
雷克萨斯RZ在车身尺寸上属于中型纯电SUV，轴距达到2,850mm，为前后排提供了较为充裕的纵向乘坐空间。后备箱标准容积522L在同级中表现中规中矩，但放倒后可扩展至近1,451L，满足大件物品装载需求。整体而言，RZ的空间表现偏向实用与舒适取向，适合家庭日常使用及长途出行。

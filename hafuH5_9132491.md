# 哈弗H5 内饰参数介绍
  
更新时间：2026年07月12日09时48分27秒   

## 一、座椅系统
| 项目 | 参数 |
|------|------|
| 座椅材质 | 仿皮/织物（根据配置不同） |
| 驾驶座调节方式 | 手动6向调节（部分高配为电动8向） |
| 副驾驶座调节方式 | 手动4向调节 |
| 前排座椅加热 | 部分车型配备 |
| 后排座椅比例放倒 | 支持（4/6分割放倒） |
| 后排中央扶手 | 带杯架 |
## 二、方向盘
| 项目 | 参数 |
|------|------|
| 方向盘材质 | PVC塑料 / 真皮包裹（高配） |
| 方向盘调节 | **手动上下+前后调节**（四向调节） |
| 多功能方向盘 | **全系标配**（蓝牙电话、多媒体控制、定速巡航等） |
## 三、仪表盘与中控
| 项目 | 参数 |
|------|------|
| **仪表盘** | **3.5英寸单色行车电脑显示屏** +机械指针仪表组合 |
| **中控屏幕** | **12.3英寸悬浮式触控液晶屏**（部分车型为10.25英寸） |
| GPS导航系统 | **标配**（高德地图车机版） |
| CarPlay / CarLife | **支持手机互联映射**（Apple CarPlay + Baidu CarLife） |
| OTA升级 | **支持整车OTA远程升级** |
## 四、空调系统
- **自动空调**
- **后排出风口**
- PM2.5过滤装置
- AQS空气质量管理系统
- CN95级空调滤芯
## 五、音响系统
- **扬声器数量：6个**
- USB接口：前排2个 +后排1个
- Type-C接口：前排1个
- AUX音频输入接口
## ⚠️注意：以上参数基于2023款及后续改款车型，具体配置请以实际购买车型为准。

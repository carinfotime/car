## 别克GL8 动力参数介绍
  
更新时间：2026年07月15日15时54分43秒   

别克GL8作为国内MPV市场的标杆车型，其动力系统在不同代际和配置上有所差异。目前主销的车型主要搭载 **2.0T涡轮增压发动机** 与 **48V轻混系统**，并匹配 **9速HYDRA-MATIC智能变速箱**。
### 核心动力总成
#### 1. 发动机：2.0T可变缸涡轮增压发动机
-   **排量**：2.0L
-   **进气形式**：涡轮增压
-   **发动机型号**：LSY（通用集团）
-   **最大马力**：237 Ps（174 kW）
-   **最大功率转速**：5000 rpm
-   **最大扭矩**：350 N·m
-   **最大扭矩转速**：1500 - 4000 rpm
-   **供油方式**：缸内直喷
-   **燃油标号**：95号及以上无铅汽油
-   **环保标准**：国VI b
#### 2. 48V轻混系统（部分车型）
-   辅助电机在起步、加速时提供额外助力，提升平顺性。
-   支持更高效的启停功能，降低油耗与震动。
#### 3. 变速箱
-   **型号**：9速HYDRA-MATIC智能变速箱（9AT）
-   **类型**：手自一体电子换挡（ETRS）
-   **特点**：
    -   齿比绵密，换挡平顺。
    -   高速巡航时能有效降低发动机转速，提升燃油经济性。
### 性能与油耗表现
| 项目 | 数据 |
| :--- | :--- |
| **官方0-100km/h加速时间** | 约9.5秒 - 10秒 |
| **最高车速 (km/h)** | 195 |

| 项目 | 规格 |
| :--- | :--- |
| **驱动方式** | 前置前驱 (FF) |
| **前悬挂类型** | 麦弗逊式独立悬挂 |
| **后悬挂类型** | （陆上公务舱）扭力梁式非独立悬挂 / （陆尊/艾维亚）多连杆式独立悬挂 |
### 📝总结
别克GL8的动力调校并非追求极致性能，而是以“平顺、舒适、静谧”为核心目标。2.0T+9AT的组合能够提供充沛的低扭输出，满足满载或高速超车需求；48V系统的加入进一步优化了启停时的平顺性和起步响应。这套动力总成很好地平衡了商务接待的舒适性与日常使用的经济性。

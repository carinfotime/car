# 小鹏P7+ 儿童乘车安全注意事项
  
更新时间：2026年07月15日12时22分06秒   

## 一、儿童安全座椅使用规范
- **12岁以下或身高不足150cm的儿童**，必须使用符合国家标准的儿童安全座椅。
- **反向安装**：建议0-4岁（或体重18kg以下）的儿童使用反向安装的儿童安全座椅，以更好地保护头颈。
- **ISOFIX接口**：小鹏P7+后排配备ISOFIX接口，请严格按照说明书正确固定安全座椅。
- **严禁将反向安装的儿童座椅放置在前排**（即使关闭副驾气囊也不建议）。
## 二、前排乘坐禁令
- **严禁12岁以下儿童坐在副驾驶位置**。
- 副驾驶前方配备安全气囊，气囊弹出时冲击力极大，可能对儿童造成严重伤害。
- 若因特殊情况必须让儿童坐前排（如所有后排座位已满且年龄较大），请务必：
  - 将前排座椅调至最靠后位置
  - 确保安全带正确佩戴且不勒到颈部
  - 关闭副驾气囊（如有此功能）
## 三、车窗与车门安全
- **启用后排车窗锁**：防止儿童在行驶中误触车窗开关导致夹伤或意外开启。
- **启用后排车门儿童锁**：防止车辆行驶中或停车时儿童从内侧打开车门。
- 停车后切勿将儿童单独留在车内，尤其是夏季高温天气，车内温度可在短时间内升至致命水平。
## 四、安全带使用提醒
- 安全带应跨过儿童的**肩部中间位置**（而非颈部），腰带部分应紧贴**髋部下方**（而非腹部）。
- **切勿让安全带从腋下穿过或放在背后**，这会极大降低保护效果并增加受伤风险。
- 身高不足135cm的儿童不建议直接使用成人安全带，应配合增高垫使用。
## 五、智能功能相关提醒
| 功能 | 注意事项 |
|------|----------|
| **电动门 / 电吸门** | P7+配备电动门时，请勿让儿童在车门附近玩耍手指或身体部位。建议由成人操作开关门。 |
| **全景天幕 /天窗** | P7+全景天幕面积大且不可开启。但仍需教育孩子不要攀爬或站立在座位上触碰车顶。 |
| **中控大屏 /语音交互** | P7+中控屏集成大量功能。建议开启“童锁模式”或“访客模式”，防止行驶中误操作影响驾驶。 |
| **空调出风口** | P7+出风口设计较为隐蔽但仍需注意不要让小孩将异物塞入出风口内。 |
## 六、车内物品与活动管理
- **避免尖锐玩具或硬物放置在座位上/脚下空间内**——急刹车时可能飞起伤人。
- **不要在车内奔跑打闹或爬行移动身体位置**,急刹车/碰撞时极易造成严重伤害。
- **不要将头手伸出窗外/天窗外**,即使车辆静止也可能被其他车辆刮伤或被升降窗夹伤。
## ⚠️  特别警示：切勿将孩子单独留在车内！
> ❗无论时间长短、无论是否开空调——都绝对不要将孩子单独留在小鹏P7+车内！
>
> - P7+具备自动下电功能,长时间驻车可能自动关闭空调,导致缺氧/中暑风险
> -  孩子在车内可能误触方向盘/挡位等造成意外移动
> -  夏季阳光暴晒下,密闭车厢内温度可在15分钟内达到60°C以上,极其危险

# 庆铃T28 驾驶注意事项
  
更新时间：2026年07月12日08时00分00秒   

## 一、启动与预热
1.  **钥匙门操作**：插入钥匙后，先将钥匙拧到“ON”档（仪表盘灯亮），等待 **5-10秒**，让车辆电子系统完成自检（特别是柴油版车型的预热塞指示灯）。
2.  **预热功能（柴油版）**：若仪表盘上的“预热指示灯”（类似螺旋状图标）亮起，请务必等待该灯熄灭后再启动发动机。气温越低，预热时间越长。
3.  **启动**：踩下离合器踏板（手动挡车型）和刹车踏板，将钥匙拧至“START”档启动。启动后应立即松开钥匙。
4.  **冷车启动后**：不要立即大脚轰油门。建议怠速运转 **1-3分钟**（冬季适当延长），待发动机水温表指针开始上升后再缓慢行驶。
## 二、行驶中注意事项
### 1. 变速箱与换挡
-   **手动挡操作**：庆铃T28的变速箱齿比偏向载重设定，换挡时需确保转速匹配。
    -   **推荐换挡转速**：建议在发动机转速达到 **2000-2500转/分** 时进行升档。
    -   **避免拖档**：当转速低于1500转/分时，应及时降档，避免发动机抖动和积碳。
    -   **倒挡**：挂倒挡前，务必确保车辆完全停稳。
### 2. 载重与操控
-   **切勿超载**：严格遵守车辆铭牌上的最大允许总质量。超载会严重损坏悬架、轮胎和制动系统。
-   **货物固定**：货箱装载货物时，必须均匀分布并使用绳索或紧固器固定牢固。重心过高或偏移会导致侧翻风险剧增。
-   **空载行驶**：空车或轻载时，后悬架较硬，经过颠簸路面时后排跳动感较强，请适当降低车速。
### 3. 制动系统
-   **刹车距离预判**：作为皮卡车型，自重较大（尤其是满载时），刹车距离比普通轿车更长。请保持更大的安全跟车距离。
-   **长下坡路段**：
    -   利用发动机制动（挂入低速档），避免长时间连续踩踏刹车踏板导致刹车片过热失效（热衰减）。
    -   严禁空挡滑行。
### 4. 转向与视野
-   **转弯半径**：T28转弯半径较大，在狭窄道路掉头或转弯时需留出更多空间。
-   **盲区注意**：
    -   A柱较粗壮，转弯时需左右移动头部观察路况。
    -   货箱较高且无倒车影像的车型，倒车时盲区极大。建议加装倒车雷达或影像。
## 三、特殊路况驾驶
| 路况 | 操作建议 |
| :--- | :--- |
| **雨天/湿滑路面** | T28多为后轮驱动（部分四驱除外）。雨后路面湿滑时起步勿急加速以防甩尾；过弯提前减速防侧滑。 |
| **非铺装路面/烂路** | T28底盘高通过性好但非硬派越野。遇坑洼路段应低速慢行；涉水深度勿超过轮毂一半且匀速通过；陷车时可尝试关闭ESP并挂低速四驱脱困。 |
| **高速公路行驶** | T28风阻大且重心高易受横风影响。高速行驶建议时速控制在100km/h以内；超越大货车时应紧握方向盘防止气流扰动；切勿急打方向变道以防侧翻。 |
## 四、日常检查与维护
1.  **每日出车前检查**
    -   【机油】拔出机油尺检查油位是否在上下刻度之间。
    -   【冷却液】查看副水壶液位是否在MIN-MAX之间。
    -   【轮胎】检查胎压及胎面有无鼓包或扎钉。（标准胎压参考油箱盖内侧标签）
2.  【灯光与信号】确认所有灯光及喇叭功能正常。
3.  定期保养周期参照用户手册严格执行更换机油机滤等操作。
## ⚠️安全警示
1.  无论前后排驾乘人员均须系好安全带——皮卡发生碰撞翻滚风险较高不系安全带非常危险！
2.  严禁酒后驾车及疲劳驾驶——连续开车满4小时必须休息至少20分钟以上！
3.  行车途中如遇仪表盘故障灯亮起应立即靠边停车熄火并联系专业维修人员处理切勿强行继续行驶以免造成更大损伤！

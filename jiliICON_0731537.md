# 吉利ICON 车主操作指南
  
更新时间：2026年07月17日12时12分41秒   

## 一、车辆基本操作
### 1. 智能钥匙与启动
- **解锁/锁车**：携带钥匙靠近车辆，轻触门把手感应区即可解锁；离车后自动落锁。
- **一键启动**：踩下刹车踏板，按下“START/STOP”按钮即可启动发动机。
- **熄火**：车辆停稳后，挂入P挡，再次按下“START/STOP”按钮。
### 2. 挡位操作（电子换挡）
| 挡位 | 操作方式 | 说明 |
|------|----------|------|
| P（驻车） | 按下挡杆顶部P键 | 停车时使用 |
| R（倒车） | 按住左侧按键向前推 | 倒车时使用 |
| N（空挡） | 前后推动至N位置 | 短暂停车或洗车时使用 |
| D（前进） | 向后拉动一次 | 正常行驶 |
| M（手动模式） | D挡位置再向后拉一次切换至M挡，通过拨片换挡 |
> **注意**：切换R/D挡前务必确认车辆已完全停稳。

## 二、驾驶辅助系统
### 1. ACC自适应巡航
- **开启**：方向盘左侧按下巡航开关图标按键。
- **设定速度**：达到目标速度后，向下拨动“SET/-”键。
- **调整速度**：向上拨动“RES/+”加速，向下拨动减速。
- **跟车距离调节**：按方向盘上的距离调节按键（通常为车辆图标+横线）。
- **取消/恢复**：轻踩刹车取消巡航；向上拨动“RES/+”恢复上次设定速度。
### 2. LKA车道保持辅助
- **开启/关闭**：中控屏 → “设置” → “驾驶辅助” → “车道保持辅助”。
- **功能说明**：当车辆无意识偏离车道时，系统会轻微修正方向盘并发出警示。
### 3. APA自动泊车
1.  车速低于30km/h时，按下中控台自动泊车按键。
2.  系统搜索车位，找到后中控屏提示选择车位类型。
3.  按照提示挂入R挡或D挡，松开方向盘和刹车。
4.  系统自动控制转向、油门和刹车完成泊入。

## 三、中控屏与智能互联
### GKUI吉客智能生态系统
#### （1）导航
-  默认内置高德地图车机版。
-  支持语音输入目的地：“你好吉利，导航到XXX”。
-  登录账号后可同步手机端收藏地点和路线。
#### （2）语音控制
唤醒方式：
-   说“你好吉利”。
-   或按方向盘右侧语音按键。
常用指令示例：

“打开空调”
“调低温度”
“播放音乐”
“打开车窗”
“我想听周杰伦的歌”

#### （3）手机互联
-  支持百度CarLife+（有线连接）。
-  部分车型支持华为HiCar无线连接。
-  连接方式：使用原装数据线连接USB接口 → APP自动启动。

## 四、空调系统操作
### （1）自动空调
在中控屏下方触控面板或屏幕内操作：
|功能|操作|
|---|---|
|温度调节|左右旋转温度旋钮 / A/C界面滑动条|
|风量调节|旋转风量旋钮 / A/C界面滑动条|
|吹风模式|点击屏幕上的风向图标切换|
|内外循环切换|点击内循环图标（箭头在车内循环的图案）|
|前风挡除雾|点击扇形+波浪线图标|
|后风挡除雾/外后视镜加热|点击矩形+波浪线图标|
> **建议**：冬季开暖风请关闭A/C；夏季开冷风请开启A/C并设为内循环以快速降温。

## 五、灯光与雨刮
### （1）灯光控制杆
位于方向盘左侧：

OFF     →   关闭
AUTO    →   自动大灯（推荐日常使用）
示宽灯 →   位置灯
近光灯 →   手动近光
远光灯 →   向前推控制杆
雾灯    →   旋转中间环至雾灯标识

> ICON全系标配LED大灯及自动大灯功能。
### （2）雨刮控制杆
位于方向盘右侧：

MIST     →   单次刮刷
INT/AUTO→   间歇/自动雨刮（可旋转中间环调节灵敏度）
LO       →   低速连续刮刷
HI       →   高速连续刮刷

> -  向上抬控制杆喷水清洗前风挡。
> -  部分车型配备雨量感应雨刮（AUTO模式）。

## 六、仪表盘信息解读
主要显示区域：

左侧区域      ：转速表 / PHEV车型功率表
中央区域      ：车速数字显示 + 
               行车信息（续航里程、平均油耗、
               胎压监测、驾驶模式等）
右侧区域      ：燃油表 / EV车型电量表 + 
               水温表或电机温度表

可通过方向盘右侧的上下左右方向键切换中央区域显示内容：

行程信息     ：小计里程A/B、平均油耗、行驶时间等
导航信息     ：转弯箭头及道路名称  
多媒体信息   ：当前播放歌曲及专辑封面  
辅助驾驶信息 ：ACC状态及车道识别情况  
胎压监测     ：四个轮胎实时气压值

## 七、保养与维护建议
### （1）定期保养周期表

机油机滤更换         :每10,000公里或12个月  
空气滤芯             :每20,000公里更换  
空调滤芯             :每10,000公里检查/更换  
火花塞               :每30,000公里更换  
制动液               :每40,000公里更换  
变速箱油             :每60,000公里检查/更换  
防冻液               :每40,000公里或4年更换  
正时皮带(若适用)     :每80,000公里检查/更换  

### （2）轮胎维护建议
 
标准胎压(冷态)       :前轮230kPa / 
                       后轮230kPa (参考B柱标签)
轮胎换位             :每10,000公里前后交叉换位   
备胎(非全尺寸)       :最高时速不超过80km/h   

> *以上数据以官方用户手册为准。*

## ⚠️八、常见故障提示及处理办法  
 
故障灯亮起           ->   
发动机故障指示灯      ->尽快前往服务站检修   
ABS警告灯            ->制动系统可能异常，
                        谨慎驾驶并尽快检修   
安全气囊警告灯        ->立即前往服务站检修   
胎压报警              ->检查轮胎气压并及时充气   
蓄电池报警            ->发电机或电池异常，
                        尽快检修避免抛锚   
ESP OFF指示灯亮起     ->误触开关则重新开启；
                        持续亮起则需检修

## 📋九、应急操作  
 
遥控钥匙没电         ->
用机械钥匙撬开主驾门把手护盖，
插入机械钥匙开门；
将钥匙贴近启动按钮感应区仍可启动。  
电瓶亏电无法启动     ->
搭电救援：
正极接正极(+)，负极接负极(-)，
先接正极再接负极。  
轮胎漏气             ->
取出后备箱下方非全尺寸备胎及千斤顶；
按对角顺序松螺丝→顶起→换胎→拧紧；
备胎限速80km/h。  
拖车                 ->
挂N挡释放电子手刹；
优先使用平板拖车；
若用硬牵引则车速≤30km/h且距离≤50km。

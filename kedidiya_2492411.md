# 克蒂帝亚（KEDE DIA）汽车保养指南
  
更新时间：2026年07月13日12时01分36秒   

## 一、日常保养（每日/每次用车后）
| 项目 | 操作建议 |
|------|----------|
| 外观检查 | 检查车身是否有划痕、凹陷，车灯是否完好 |
| 轮胎检查 | 目测胎压是否正常，轮胎表面有无异物或鼓包 |
| 仪表盘警示灯 | 启动后观察是否有故障灯亮起 |
| 车内清洁 | 保持内饰整洁，避免食物残渣、液体洒落 |
---
## 二、每周保养
- **胎压检测**：使用胎压计检测四轮胎压，保持在厂家推荐值（通常2.3–2.5 bar）
- **机油液位**：冷车状态下拔出机油尺检查，液位应在上下刻度之间
- **冷却液液位**：检查膨胀壶液面是否在MIN–MAX之间
- **玻璃水**：及时补充，确保喷水正常
- **灯光功能**：检查远近光灯、转向灯、刹车灯、雾灯是否正常
---
## 三、每月保养
- **空气滤清器**：取出检查，若灰尘较多可吹净或更换
- **空调滤芯**：检查是否有异味或堵塞，必要时更换
- **雨刮片**：测试刮水效果，如有异响或刮不干净应及时更换
- **电瓶状态**：观察电瓶观察孔颜色（绿色为正常），检查接线柱有无腐蚀
- **底盘与悬挂**：目测底盘有无漏油、磕碰痕迹
---
## 四、每5000公里 / 半年保养（小保养）
> ✅ **建议在授权服务中心进行**
| 项目 | 说明 |
|------|------|
| ✅ 更换机油 & 机滤 | 使用厂家推荐标号的全合成机油 |
| ✅ 检查刹车系统 | 刹车片厚度 ≥3mm，刹车盘无深沟槽 |
| ✅ 轮胎换位 &动平衡 | 前后交叉换位，做动平衡检测 |
| ✅ 全车灯光 &喇叭 | — |
| ✅ ECU系统检测 | OBD读取故障码 |

## 五、每10000公里 / 1年保养（中保养）
包含小保养全部项目 +：
- 🔧 **更换空气滤清器**
- 🔧 **更换空调滤芯**
- 🔧 **火花塞检查/更换**（视车型而定）
- 🔧 **燃油滤清器更换**
- 🔧 **节气门清洗**
- 🔧 **制动液含水量检测**（含水率 >3%需更换）
- 🔧 **变速箱油液位及状态检查**

包含中保养全部项目 +：

|项目 |备注|
|---|---|
| 🛠️变速箱油更换 | AT/CVT/DCT不同变速箱类型对应专用油品 |
| 🛠️防冻液更换 | — |
| 🛠️转向助力油更换（如配备） | — |
| 🛠️刹车片/刹车盘深度检查或更换 | — |
| 🛠️正时皮带/链条状态检查 |皮带类需按里程强制更换 |
### ⚙️其他重要项目
- 💡四轮定位检测与调整
- 💡燃油系统积碳清洗（喷油嘴+进气道）
- 💡蒸发箱及空调管路杀菌消毒
- 💡全车橡胶密封条养护
---
## 📌七、特殊注意事项
### ❄️冬季养护要点
1.  提前更换冰点更低的防冻液
2.  电瓶性能下降易亏电，建议检测启动电压
3.  雨刮片如冻结切勿强行开启
4.  冷车启动后怠速热车1–2分钟再行驶
### ☀️夏季养护要点
1.  空调系统提前检测制冷效果及冷媒压力
2.  冷却系统重点排查水箱散热风扇工作状态
3.  轮胎胎压适当调低0.1–0.2 bar防止爆胎风险
### ⚠️常见警示信号及应对
> ❗发动机故障灯亮 →立即停车联系售后  
> ❗水温过高 →关闭空调靠边停车熄火等待降温  
> ❗制动异响/踏板变软 →尽快进店检修  
> ❗方向盘抖动 →做动平衡或四轮定位

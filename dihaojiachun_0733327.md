# 帝豪甲醇汽车操作指南
  
更新时间：2026年07月14日14时29分40秒   

## 一、车辆基本介绍
帝豪甲醇汽车是以甲醇燃料为主要动力的清洁能源车型，采用甲醇/汽油双燃料系统，具备经济环保、动力充沛的特点。本指南适用于吉利帝豪甲醇版车型。
---
## 二、燃料加注操作
### 2.1 燃料类型
- **主燃料**：M100 甲醇燃料（纯甲醇）
- **备用燃料**：92# 及以上无铅汽油
- **油箱容积**：约 53L（甲醇箱）+ 约 10L（汽油副箱）
### 2.2 加注步骤
1. 车辆熄火，拉起手刹，确保车辆处于驻车状态。
2. 打开油箱盖，根据加油口标识选择甲醇或汽油加注口。
3. **甲醇加注口**：绿色或蓝色标识，加注 M100 甲醇燃料。
4. **汽油加注口**：常规黄色标识，加注 92# 及以上汽油。
5. 加注完成后拧紧油箱盖，确保密封。
### 2.3 注意事项
- **禁止混加**：甲醇箱严禁加入汽油，汽油箱严禁加入甲醇。
- **加注时机**：建议甲醇燃料剩余 1/4 时及时加注，避免空箱运行。
- **汽油备用**：汽油箱主要用于冷启动辅助，建议保持 1/4 以上油量。
- **燃油品质**：请到正规甲醇加注站加注符合国家标准的甲醇燃料。
---
## 三、启动与驾驶操作
### 3.1 冷启动（冬季/低温环境）
1. 确保汽油箱有足够汽油（建议 1/4 以上）。
2. 将钥匙拧到 ON 档，等待仪表盘自检完成（约 2-3 秒）。
3. 启动发动机，系统自动使用汽油辅助启动。
4. 发动机运转 30-60 秒后，自动切换至甲醇燃料模式。
5. 待水温上升至正常工作温度后，方可正常行驶。
### 3.2 热启动（常温环境）
1. 正常拧钥匙启动即可。
2. 系统自动判断温度，优先使用甲醇燃料直接启动。
### 3.3 驾驶注意事项
- **磨合期**：新车前 1500km 为磨合期，建议避免急加速、急刹车，转速不超过 3000rpm。
- **换挡时机**（手动挡）：建议 2000-2500rpm 换挡。
- **经济时速**：60-90km/h 为经济行驶速度。
- **动力特性**：甲醇燃料动力输出平顺，加速感受与汽油版基本一致。

## 五、燃料切换操作
### 5.1 自动切换
- 系统根据发动机温度、燃料余量等参数自动选择最优燃料模式。
- 冷启动时自动使用汽油，热车后切换至甲醇。
- 甲醇燃料耗尽时自动切换至汽油备用模式。
### 5.2 手动切换（部分车型配备）
- 中控台设有 **燃料切换按钮**（标有 FUEL 或 燃料切换图标）。
- 按下按钮可在甲醇/汽油模式间手动切换。
- 建议在行驶中保持稳定车速时进行切换，避免急加速或爬坡时切换。

## 六、日常维护与保养
### 6.1 保养周期
| 保养项目 | 周期 |
|----------|------|
| 更换机油及机滤 | 每 5000km 或 6 个月 |
| 更换甲醇滤清器 | 每 10000km |
| 更换汽油滤清器 | 每 20000km |
| 检查火花塞 | 每 15000km |
| 检查燃料管路 | 每 10000km |
| 更换冷却液 | 每 40000km 或 2 年 |
### 6.2 专用保养要求
- **机油**：必须使用甲醇发动机专用机油（标号通常为 5W-30 或 10W-40）。
- **火花塞**：建议使用甲醇专用火花塞或铱金火花塞。
- **滤清器**：甲醇燃料杂质较多，需定期更换甲醇滤清器。
- **燃油系统清洗**：建议每 20000km 进行燃油系统清洗。
### 6.3 冬季保养
- 检查汽油箱油量，确保冷启动辅助正常。
- 检查冷却液冰点，防止冻裂。
- 检查蓄电池电量，低温下蓄电池性能下降。

## 七、安全注意事项
### 7.1 甲醇燃料特性
- **易燃**：甲醇为易燃液体，远离火源。
- **有毒**：甲醇有毒，切勿饮用或接触皮肤。
- **腐蚀性**：甲醇对部分橡胶件有腐蚀作用，需使用耐甲醇材料。
### 7.2 操作安全
- 加注甲醇时保持通风，避免吸入蒸汽。
- 如甲醇溅到皮肤或眼睛，立即用大量清水冲洗并就医。
- 发现燃料泄漏时，立即关闭发动机，远离车辆，联系专业人员处理。
- 禁止在车内放置甲醇燃料容器。
### 7.3 应急处理
| 情况 | 处理方式 |
|------|----------|
| 甲醇燃料耗尽 | 系统自动切换至汽油模式，尽快加注甲醇 |
| 汽油燃料耗尽 | 使用甲醇启动（需水温正常），尽快加注汽油 |
| 发动机无法启动 | 检查汽油余量，尝试多次启动 |
| 燃料泄漏 | 关闭发动机，疏散人员，拨打救援电话 |

## 八、常见问题解答（FAQ）
**Q1：甲醇车可以只加汽油吗？**
A：可以，但长期使用汽油会失去甲醇的经济优势，建议以甲醇为主。
**Q2：甲醇车冬天启动困难怎么办？**
A：确保汽油箱有油，启动时踩下离合器（手动挡），每次启动不超过 5 秒，间隔 10 秒后再试。
**Q3：甲醇对发动机有损害吗？**
A：帝豪甲醇版发动机经过专用改造，使用耐甲醇材料，正常保养下无损害。
**Q4：甲醇车续航里程多少？**
A：满箱甲醇续航约 400-500km，加上汽油备用续航约 100km。
**Q5：甲醇加注站多吗？**
A：主要分布在山西、陕西、贵州等甲醇汽车推广地区，建议提前规划加注点。

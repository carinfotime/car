# 奥迪S5 空间参数介绍
  
更新时间：2026年07月10日21时13分04秒   

## 车身尺寸
| 参数 | 数据 |
|------|------|
| 车长 | 4,762 mm |
| 车宽（含后视镜） | 2,026 mm |
| 车宽（不含后视镜） | 1,843 mm |
| 车高 | 1,384 mm（Coupé）/ 1,386 mm（Sportback） |
| 轴距 | 2,825 mm |
## 内部空间
### 前排空间
- **头部空间**：约 **980 mm**
- **腿部空间**：充裕，支持多向电动调节座椅
- **肩部空间**：约 **1,400 mm**
- **座椅调节范围**：前后可调范围约 **240 mm**
### 后排空间
- **头部空间**：约 **920 mm**（Coupé车型略低）
- **腿部空间**：约 **750 mm**
- **肩部空间**：约 **1,350 mm**
- **后排座椅形式**：双座布局（Coupé）/三座布局（Sportback）
> ⚠️ Coupé车型后排更适合短途乘坐，Sportback版本后排实用性更高。
## 储物容积
### Sportback（五门掀背版）
| 状态 | 容积 |
|------|------|
| 标准状态 | **465 L** |
| 后排座椅放倒后 | **1,300 L** |
### Coupé（双门轿跑版）
| 状态 | 容积 |
|------|------|
| 标准状态 | **375 L** |
| 后排座椅放倒后 | — |
### Cabriolet（敞篷版）
- **后备箱容积**：约 **320 L**
## 车内储物细节
- **手套箱容积**：约 **8 L**
- **中央扶手储物盒**：可容纳手机、钥匙等小件物品
- **前门板储物格**：可放置500ml水瓶
- **后门板储物格**：（仅Sportback车型配备）
- **杯架数量**：前排2个，后排2个
- **眼镜盒位置**：驾驶位头顶上方
## 乘坐人数
|  车型版本   |  座位数   |
|---|---|
| Coupé       |  4座      |
| Sportback   |  5座      |
| Cabriolet   |  4座      |

> 📌 *以上数据基于奥迪S5最新款官方参数整理，实际体验可能因配置及选装包不同略有差异。*

# 奇骏驾驶注意事项
  
更新时间：2026年07月12日16时57分08秒   

## 一、新车磨合期（前 1000-1500 公里）
- **避免急加速与急刹车**：平稳驾驶，让发动机和变速箱各部件充分磨合。
- **转速控制**：建议发动机转速不超过 4000 rpm。
- **避免定速巡航**：尽量使用变化的车速行驶，使各档位齿轮都能得到磨合。
- **不要长时间高负荷运行**：如满载爬坡或拖拽重物。
## 二、启动与热车
- **启动后短暂热车**：冬季建议热车 30 秒至 1 分钟，待转速回落至正常怠速后再起步。
- **起步后低速行驶**：在水温达到正常工作温度前，避免大油门高转速行驶。
- **一键启动操作**：先按一下启动键让车辆通电自检（仪表盘灯亮起），待自检完成后再踩刹车按启动。
## 三、CVT 变速箱特别注意事项
奇骏搭载的 CVT 变速箱在驾驶习惯上需要特别注意：
- **避免频繁急加速**：CVT 变速箱的钢带/链条承受冲击能力有限，急加速会加剧磨损。
- **长下坡利用发动机制动**：切换至手动模式（M 档）或 L 档，利用发动机转速控制车速，避免长时间踩刹车导致刹车过热。
- **避免 N 档滑行**：N 档滑行会导致变速箱油泵供油不足，严重损害变速箱。
- **定期更换变速箱油**：建议每 4-6 万公里或按厂家保养手册要求更换 CVT 油，切勿超期。
## 四、四驱系统（适时四驱）
- **铺装路面使用 2WD 模式**：日常城市或高速行驶，使用 2WD 模式更省油。
- **湿滑、雨雪路面使用 Auto 模式**：系统自动分配前后轮动力，提升稳定性。
- **非铺装或脱困使用 Lock 模式**：在沙地、泥地等低附着力路面，使用四驱锁止模式（车速超过 40km/h 会自动解除）。
- **不要在干燥硬路面使用 Lock 模式**：会导致转弯困难、传动系统磨损加剧。
## 五、日常行驶
- **视野盲区**：奇骏 A 柱较粗，转弯时注意左右摆动头部观察路况。
- **自动启停**：涉水路段务必关闭自动启停功能，防止发动机在水中二次启动损坏。
- **底盘高度**：虽然奇骏离地间隙较高（约 210mm），但仍属于城市 SUV，不要进行重度越野或冲坡。
- **ECO 模式**：城市拥堵路况可开启，但高速超车时建议关闭以获得更佳动力响应。
## 六、停车与熄火
- **停车步骤**：踩刹车停稳 → 挂 N 档 → 拉电子手刹 → 松脚刹（释放应力） → 挂 P 档 → 熄火。
- **坡道停车**：建议配合使用砖块或石头垫住车轮，减少变速箱 P 档锁止机构受力。
- **熄火前关闭空调**：避免下次启动时增加电瓶和发动机负荷。
## 七、轮胎与胎压
- **标准胎压**：一般建议 2.3-2.5 bar（具体以驾驶侧车门 B 柱标签为准）。
- **定期检查**：每月检查一次胎压，长途出行前务必检查。
- **轮胎换位**：每 1 万公里进行四轮换位，保证轮胎均匀磨损。
## 八、保养周期（参考）
| 项目 | 周期/里程 |
|------|-----------|
| 机油、机滤 | 每 5000 公里或 6 个月 |
| 空气滤芯 | 每 1 万公里检查，2 万公里更换 |
| 空调滤芯 | 每 1 万公里检查，2 万公里更换 |
| 火花塞 | 每 6-8 万公里 |
| 刹车油 | 每 2 年或 4 万公里 |
| CVT 变速箱油 | 每 4-6 万公里 |
| 防冻液 | 每 4 年或 8 万公里 |
> **注意**：以上为通用建议，具体请以您的车辆保养手册为准。
## 九、其他提示
- **钥匙不要留在车内**：奇骏具备智能感应功能，钥匙留在车内可能导致车门无法正常锁闭。
- **后备箱开启**：确认上方空间足够，避免在低矮停车场开启后备箱撞到障碍物。
- **儿童安全锁**：后排乘坐儿童时，启用儿童安全锁，防止儿童从车内误开车门。

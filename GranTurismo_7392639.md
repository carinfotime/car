# 玛莎拉蒂 GranTurismo 内饰参数介绍
  
更新时间：2026年07月12日08时19分21秒   

## 整体设计风格
- **意式豪华运动风**：融合顶级皮革、碳纤维与金属材质，兼顾 GT 跑车的优雅与运动感。
- **环绕式座舱**：中控台向驾驶员倾斜，强调驾驶者为中心的人机工程学。
## 座椅
| 项目 | 参数 / 描述 |
|------|-------------|
| **材质** | 全系标配 **Pieno Fiore 天然全粒面皮革**（可选配 Alcantara / 织物混搭） |
| **前排座椅** | 12向电动调节（含4向腰托）、加热、通风功能 |
| **后排座椅** | 2+2布局，独立式运动座椅，支持加热（选装） |
| **记忆功能** | 驾驶席带3组记忆位置（含方向盘、后视镜联动） |
## 方向盘
- **材质**：真皮包裹（可选碳纤维/木饰），底部平底运动造型
- **功能**：电动四向调节、换挡拨片、多功能按键（含启动/驾驶模式选择）
- **加热功能**：选装
## 仪表盘与显示系统
| 组件 | 规格 |
|------|------|
| **全液晶仪表盘** | 12.3英寸高清屏，支持经典/运动/舒适三种主题切换 |
| **中控触屏** | **12.3英寸 + 8.8英寸双屏设计**（上方主控娱乐导航，下方控制空调/座椅） |
| **抬头显示 (HUD)** | 选装，投射车速、导航、驾驶辅助信息 |
## 智能互联与音响
- **车机系统**：Maserati Intelligent Assistant (MIA) ，支持 OTA升级
- **手机互联**：无线 Apple CarPlay / Android Auto
- **音响系统**：
    - *标准*：10扬声器 Hi-Fi音响
    - *选装*：Sonus Faber® High-Premium Audio System （14扬声器 /  860W或更高功率）
##  空调与舒适配置
- **双区自动空调**（前排独立温控）
- **后排独立出风口**
- **空气净化系统**（PM2.5过滤 + AQS传感器）
- *选装*：四区自动空调
##  内饰材质与装饰件
|  部位      |  可选材质                                     |
|------------|-----------------------------------------------|
|  仪表台    |  真皮包裹 + Alcantara / Radica木纹 / Carbon Skin®碳纤维饰板 |
|  门板      |  全真皮覆盖 + Alcantara拼接                    |
|  顶棚      |  标准织物顶棚；选装 Alcantara / Dinamica微纤维顶棚 |
|  脚垫      |  专属刺绣天鹅绒脚垫                            |
##  储物空间与人机工程
- **中央扶手箱**:带冷藏功能的小型储物格
- **杯架**:前排2个 +后排2个隐藏式杯架
- ***行李厢容积***:约310升 (可容纳两套高尔夫球袋)
## ✨氛围灯与环境照明
- ***LED氛围灯***:7色可调 (白/蓝/红/绿/紫等)，覆盖脚部空间、门把手及中控台轮廓
---
> 📌 *注：以上参数基于最新款 GranTurismo Modena/Trofeo/Folgore版本整理，实际配置可能因市场及选装包不同而有所差异。*

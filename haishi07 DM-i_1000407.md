# 比亚迪海狮07 DM-i 操作指南
  
更新时间：2026年07月19日18时27分44秒   

## 一、车辆基本参数与功能概览
| 项目 | 说明 |
|------|------|
| 动力类型 | DM-i 超级混动（插电式混合动力） |
| 驱动模式 | EV（纯电）/ HEV（混动） |
| 能量回收 | 可调节强度 |
| 驾驶模式 | Eco / Normal / Sport / 雪地等 |

## 二、启动与熄火
### 启动车辆
1. 携带智能钥匙或手机NFC钥匙进入车辆。
2. 踩下制动踏板，此时启动按键指示灯变为绿色。
3. 按下 **启动/停止按键**，仪表盘点亮，车辆进入可行驶状态。
### 熄火车辆
1. 将车辆停稳，挡位置于 **P挡**。
2. 按下 **启动/停止按键**，仪表盘熄灭，车辆下电。
> 提示：挂P挡后开门，车辆会自动熄火（部分配置）。
---
## 三、挡位操作
| 挡位 | 操作方式 | 说明 |
|------|----------|------|
| D挡（前进） | 踩刹车，向后拉动换挡杆 | 正常前进 |
| R挡（倒车） | 踩刹车，向前推动换挡杆 | 倒车影像自动开启 |
| N挡（空挡） | 踩刹车，短促前推或后推至中间位 | 洗车/拖车时使用 |
| P挡（驻车） | 按下换挡杆顶端 P 按键 | 停车时使用 |
> 换挡时需踩住制动踏板，否则无法换挡。
---
## 四、驾驶模式切换
### 动力模式
- **EV模式**：纯电行驶，适合市区短途、电量充足时。
- **HEV模式**：混动模式，发动机与电机协同工作，适合长途或电量较低时。
切换方式：通过中控屏或挡杆旁的 **EV/HEV 按键** 切换。
### 驾驶风格模式
通过中控屏或物理按键切换：
| 模式 | 适用场景 |
|------|----------|
| Eco | 城市拥堵、追求续航 |
| Normal | 日常通勤 |
| Sport | 超车、山路、追求动力 |
| 雪地 | 湿滑/冰雪路面 |
---
## 五、能量回收系统
1. 通过中控屏设置 **能量回收强度**：
   - **标准**：松油门滑行较顺畅，回收力度轻。
   - **较大**：松油门有明显拖拽感，回收效率高。
2. 仪表盘可实时查看能量流动与回收状态。
---
## 六、充电操作
### 交流慢充（家用桩/公共慢充桩）
1. 车辆熄火，打开充电口盖（位于右后翼子板）。
2. 插入交流充电枪，听到"咔哒"声表示连接到位。
3. 充电桩启动充电，仪表盘显示充电状态。
4. 充电完成后，解锁车辆，拔下充电枪，关闭充电口盖。
### 直流快充（公共快充桩）
1. 车辆熄火，打开直流快充口盖。
2. 插入直流快充枪（较粗、较重）。
3. 扫描充电桩二维码启动充电。
4. 充电完成后，解锁车辆，拔枪归位。
> 建议：日常使用建议保持电量在 20%-80% 之间，延长电池寿命。
---
## 七、智能车机系统（DiLink）
### 中控屏操作
- **主页**：点击屏幕下方 Home 键或三指下滑。
- **空调控制**：屏幕底部常驻空调快捷栏，或点击空调图标进入详细界面。
- **车辆设置**：点击「车辆设置」图标，可调整灯光、门锁、驾驶辅助等。
### 语音控制
唤醒方式：说 **"你好，小迪"** 或按下方向盘语音按键。
常用指令示例：
- "打开空调，温度24度"
- "导航到最近的充电站"
- "播放音乐"
- "打开车窗"
### 手机远程控制（比亚迪APP）
- 远程解锁/锁车
- 开启空调（提前制冷/制热）
- 查看车辆位置、电量、续航
- 预约充电
---
## 八、驾驶辅助系统
### 自适应巡航（ACC）
1. 按下方向盘左侧 **巡航开启按键**。
2. 通过 **SET/-** 设定当前速度为巡航速度。
3. 通过 **RES/+** 恢复或增加速度。
4. 按下 **取消按键** 或踩刹车退出巡航。
### 车道保持辅助
- 默认开启，车辆偏离车道时方向盘会轻微修正。
- 可在中控屏「车辆设置-驾驶辅助」中关闭。
### 360°全景影像
- 挂 R 挡自动开启。
- 低速行驶时可手动点击屏幕上的全景影像图标开启。
- 可切换不同视角（前、后、俯视、车轮视角）。
---
## 九、灯光与雨刮
### 灯光控制
| 操作 | 功能 |
|------|------|
| 旋钮调至 AUTO | 自动大灯（推荐日常使用） |
| 旋钮调至 示宽灯 | 仅亮示宽灯 |
| 旋钮调至 近光灯 | 手动近光 |
| 拨杆向前推 | 远光灯常亮 |
| 拨杆向后拉 | 远光灯闪烁（超车提示） |
### 雨刮控制
- 右侧拨杆上下拨动调节雨刮速度。
- 向上拨一次为单次刮刷。
- 向方向盘方向拉动为喷水+刮刷。
---
## 十、应急与安全
### 紧急情况
- **双闪灯**：中控台红色三角按键，按下开启。
- **电子手刹**：P挡自动拉起，也可手动拉起/释放。
- **SOS紧急呼叫**：车内后视镜上方或车顶的SOS按键，长按触发紧急呼叫。
### 轮胎与备胎
- 车辆配备补胎液和充气泵（位于后备箱盖板下方），无备胎。
- 胎压异常时仪表盘会报警，请及时检查。
### 拖车
- 必须将挡位置于 **N挡**，并解除电子手刹。
- 拖车钩位于前后保险杠盖板内，用工具撬开盖板后安装。

## 十二、常见问题提示
| 问题 | 可能原因与解决 |
|------|----------------|
| 车辆无法启动 | 检查是否踩刹车、钥匙是否在车内、电量是否过低 |
| 充电枪拔不下来 | 先解锁车辆，再按充电枪上的解锁按钮 |
| 仪表盘亮黄灯 | 查看仪表盘提示信息，或语音询问"小迪" |
| 续航显示不准 | 续航会根据驾驶习惯动态调整，属正常现象 |

## 十三、省电/省油驾驶技巧
1. **市区短途**：优先使用 EV 模式，电量低于 25% 自动切换 HEV。
2. **高速长途**：使用 HEV 模式，保持匀速行驶更省油。
3. **能量回收**：市区可设置为"较大"回收，减少刹车使用。
4. **空调**：使用自动空调，温度设置在 24-26°C 更节能。
5. **胎压**：保持标准胎压，过低会增加能耗。
6. **轻踩轻放**：避免急加速和急刹车，平缓驾驶最省电。

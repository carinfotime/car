# 瑞风M3 PHEV 操作指南
  
更新时间：2026年07月19日17时52分22秒   

## 一、车辆启动与关闭
### 1. 启动车辆
1. **确认钥匙**：确保智能钥匙在车内。
2. **踩下制动踏板**：必须踩下制动踏板。
3. **按下一键启动按钮**：
   - 仪表盘点亮，READY指示灯亮起，表示车辆已进入可行驶状态。
   - 此时车辆静音，属于正常现象（纯电模式）。
### 2. 关闭车辆
1. **停稳车辆**：挂入P挡（驻车挡）。
2. **再次按下一键启动按钮**：车辆电源关闭。
> **注意**：下车前请确保所有车窗、天窗关闭，并携带好智能钥匙。
---
## 二、驾驶模式选择
瑞风M3 PHEV提供多种驾驶模式，可通过中控台或仪表盘附近的模式切换按键进行选择。
| 模式名称 | 适用场景 | 特点 |
| :--- | :--- | :--- |
| **EV（纯电）模式** | 城市通勤、短途行驶 | - 零油耗、安静<br>- 电量不足时自动切换 |
| **HEV（混动）模式** | 日常综合路况 | - 发动机与电机协同工作<br>- 兼顾油耗与动力 |
| **SPORT（运动）模式** | 超车、爬坡、高速行驶 | - 动力响应更积极<br>- 油耗相对较高 |
> **提示**：在电量充足时优先使用EV模式；长途或高速建议使用HEV或SPORT模式以保持动力储备。
---
## 三、充电操作
### 1. 交流慢充（家用/公共充电桩）
1. **熄火并挂P挡**。
2. **打开充电口盖板**（位于车身左后侧或右后侧）。
3. **插入充电枪**：将充电枪插入充电口，听到“咔哒”声表示连接到位。
4. **开始充电**：
   -  若使用充电桩，刷卡或扫码启动。
   -  若使用家用插座，确认接地良好后连接。
5. **结束充电**：
   -  先停止充电（APP/刷卡/拔枪）。
   -  按住解锁按钮拔出充电枪。
   -  盖好充电口盖板。
### 2. SOC设置
-   可通过中控屏设置目标SOC值（例如20%~80%），避免过度放电或过度充电。
> ⚠️ **安全警告**
> *   禁止使用破损、潮湿的充电设备。
> *   雷雨天气尽量避免户外充电。
> *   长期不用车时，建议保持电量在40%~60%之间存放。

## 四、仪表盘指示灯解读
|    指示灯图标    |     含义     |                    处理方式                    |
| :--------------: | :----------: | :--------------------------------------------: |
|      READY       |   可行驶状态    |                  正常启动标志                   |
|       红色电池图标       |   高压系统故障   |             立即停车并联系售后服务              |
|       黄色发动机图标       |   发动机故障灯   |         尽快前往维修站检查          |
|       黄色电池图标       |   电量过低警告   |              尽快为车辆充电               |
|       黄色加油机图标       |   燃油不足警告   |                及时添加燃油                |

## 五、能量回收系统
-   松开加速踏板或轻踩刹车时，系统会自动回收动能给电池充电。
-   可通过换挡拨片或中控设置调整回收强度：
    -   L1：滑行阻力小，类似燃油车滑行感
    -   L2/L3：回收强度递增，减速感明显
> 💡 *建议*：城市拥堵路段可使用较高回收等级以减少刹车磨损并增加续航；高速路段建议使用低等级以保证滑行平顺性。
---
## 六、冬季与夏季使用建议
### ❄️冬季
-   提前通过APP预约开启空调制热及电池预热功能（利用市电而非电池电量）。
-   低温会导致纯电续航缩短属正常现象；可适当增加HEV模式使用比例。
-   冰雪路面起步轻柔；避免急加速急刹车以防打滑失控风险增大！
### ☀️夏季
-    暴晒后先开窗通风再开启空调降温更快且省电！
-    避免将车辆长时间暴露于烈日下——高温会加速电池老化进程！

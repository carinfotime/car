# 凯翼江豚E5 儿童乘车安全注意事项
  
更新时间：2026年07月12日10时28分05秒   

## 🚗 基本安全须知
1. **儿童不得坐在前排**：12岁以下儿童严禁乘坐副驾驶位置，紧急刹车或碰撞时安全气囊弹出可能对儿童造成严重伤害。
2. **使用儿童安全座椅**：
   - 4岁以下儿童必须使用符合国家标准的儿童安全座椅
   - 建议将安全座椅安装在后排中间或两侧位置
   - 安装后请确认座椅牢固，晃动幅度不超过2.5厘米
3. **正确使用安全带**：
   - 身高不足140cm的儿童，安全带可能勒到颈部，请配合增高垫使用
   - 安全带应穿过肩部中间位置，不可放在腋下或背后
## 🔒 车门与车窗安全
1. **开启儿童安全锁**：后门配备儿童安全锁功能，请务必开启，防止行车中儿童误开车门。
2. **车窗防夹功能**：凯翼江豚E5配备车窗防夹功能，但仍需注意：
   - 不要让儿童将头、手伸出窗外
   - 不要让儿童独自操作车窗升降
   - 建议开启车窗锁止功能
## 🧸 车内活动注意事项
1. **车内禁食硬物**：行车过程中避免给儿童吃棒棒糖、果冻、坚果等易噎呛食物。
2. **玩具选择**：
   - 避免携带尖锐、硬质玩具
   - 玩具应固定放置，防止急刹车时飞溅伤人
   - 不建议在车内玩手机或平板电脑（易晕车）
3. **不要单独留孩子在车内**：
   - ⚠️ **严禁将儿童单独留在车内**，哪怕只是短暂离开
   - 夏季车内温度可在10分钟内升至60℃以上，极其危险
   - 冬季车内也可能因缺氧导致窒息风险
## 🌡️ 空调与环境安全
1. **空调温度设置**：建议保持在24-26℃，避免直吹儿童头部和腹部。
2. **内外循环切换**：长时间使用内循环时，每隔30分钟切换外循环换气一次。
3. **遮阳措施**：后排如有阳光直射，建议使用遮阳帘保护孩子皮肤和眼睛。
## 📋 出行前检查清单
| ✅ | 检查项目 |
|:---:|:---|
| ☐ | 儿童安全座椅安装牢固 |
| ☐ | 后门儿童锁已开启 |
| ☐ | 车窗锁止功能已启用 |
| ☐ | 车内无尖锐/危险物品 |
| ☐ | 安全带高度调整合适 |
| ☐ | ISOFIX接口连接正常 |
---
> 💡 **温馨提示**：凯翼江豚E5后排空间宽敞且配有ISOFIX接口（后排两侧座位），建议优先选择通过3C认证的0-12岁全阶段可调节式安全座椅。每次出行前花1分钟检查以上事项，让孩子的每一次出行都平安快乐！

# 传祺M6 保养指南
  
更新时间：2026年07月10日17时59分10秒   

## 一、概述
本指南适用于广汽传祺M6（含M6 PRO、M6 MAX等衍生车型）。定期保养能有效延长车辆寿命、保障行车安全并维持良好的燃油经济性。建议严格按照保养手册执行，并结合实际用车环境适当调整。
---
## 二、常规保养周期表
| 项目 | 周期（公里或时间，以先到为准） |
| :--- | :--- |
| **机油 & 机滤** | 每 **5,000公里** 或 **6个月** |
| **空气滤芯** | 每 **10,000公里** 或 **12个月** |
| **空调滤芯** | 每 **10,000公里** 或 **12个月** （建议每年换季时检查） |
| **燃油滤芯** | 每 **20,000公里** （内置式通常更长，视手册而定） |
| **火花塞** | 每 **30,000 - 40,000公里** （涡轮增压车型建议30,000km检查） |
| **制动液** | 每 **40,000公里** 或 **2年** |
| **防冻液/冷却液** | 每 **40,000公里** 或 **2年** （长效型可延长至8万/4年） |
| **变速箱油（自动）** | 每 **60,000公里** （恶劣工况建议40,000km） |
| **正时皮带/链条** | M6多为正时链条（免维护），若为皮带则按手册约80,000-100,000km更换 |
| **刹车片/刹车盘** | 视磨损情况，通常前片约30,000-50,000km，后片约50,000-70,000km |
> *注：以上为参考值，具体请以随车《保修保养手册》为准。*

- [x]  更换空调滤芯

## 四、关键部件与注意事项
### 🔧　机油选择
| M6发动机型号 | SAE粘度等级推荐 | API/ACEA标准要求 |
| :--- | :--- | :--- |
| (常见)1.5T涡轮增压 (4A15J1) | `0W‑20` (优先) / `5W‑30` (备选) | API SP / SN+ <br> ACEA C2/C3 (低灰分) |
> ⚠️ *重要提示*：必须使用符合要求的低灰分机油！使用错误的高灰分机油会导致GPF堵塞。
### 🔧　轮胎规格与胎压
*   原厂规格：215/55 R17 (常见)
*   标准胎压：
    *   前轮：2.3 bar (230 kPa)
    *   后轮：2.3 bar (230 kPa)
*   满载时可适当提高至前2.4 /后2.5 bar。
### 🔧　制动系统
*   制动液型号：DOT4。
*   注意含水量检测：超过3%必须立即更换。
*   刹车盘磨损极限通常为厚度减少1mm左右。
### 🔧　冷却系统
*   使用原厂G13型防冻液或同等品。
*   严禁混用不同颜色或品牌的防冻液。

## 五、日常自检清单（每月一次）
1. ✅　检查机油尺刻度 ——应在上下限之间。
2. ✅　查看冷却液储液罐 ——不低于MIN线。
3. ✅　查看制动液储液罐 ——不低于MIN线。
4. ✅　检查轮胎外观 ——无鼓包、裂纹；胎纹深度不低于1.6mm。
5. ✅　测试所有灯光 ——包括远近光、转向灯、刹车灯。
6. ✅　测试雨刮喷水 ——确保刮净且不划伤玻璃。
7. ✅　启动后观察仪表盘 ——无故障灯常亮。

## ⚠️六、特别提醒与常见问题
1️⃣　GPF再生问题：
M6搭载GPF的车型在市区短途行驶较多时可能触发主动再生。此时不要熄火，保持中高速行驶直至再生完成。若频繁出现“GPF已满”提示需及时进店处理。
2️⃣　双离合变速箱保护：
停车等红灯超过20秒建议挂N挡拉手刹；拥堵路段可切换至手动模式避免频繁半联动过热。
3️⃣　天窗排水孔堵塞：
定期清理天窗导轨和排水孔；雨季前务必疏通以防漏水进驾驶舱。
4️⃣　电瓶亏电预警：
长期短途行驶容易导致电瓶亏电；每周至少跑一次20分钟以上路程给电瓶充电。若启停功能失效往往是电瓶电量不足的信号。
5️⃣　空调异味处理：
每次到达目的地前提前关闭AC制冷但保持风扇吹风几分钟干燥蒸发箱可有效预防霉菌产生异味。

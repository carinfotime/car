# 远程星智 动力参数介绍
  
更新时间：2026年07月18日16时30分00秒   

## 一、动力系统概述
远程星智是吉利远程新能源商用车集团旗下的一款新能源轻卡车型，采用纯电动驱动平台，主打城配物流、快递快运等场景。
---
## 二、电机参数
| 项目 | 参数 |
|------|------|
| 电机类型 | 永磁同步电机 |
| 额定功率 | 50 kW |
| 峰值功率 | **120 kW（约163马力）** |
| 额定扭矩 | 120 N·m |
| 峰值扭矩 | **325 N·m** |
| 最高转速 | 6000 rpm |
| 驱动形式 | 后置后驱 |
> 电机由宁德时代或自家威睿电机提供，具备高效率、低噪音、长寿命等优势。

## 七、底盘与传动
- **变速箱**：单级减速，无级变速（无传统变速箱）
- **驱动桥速比**：5.125 / 5.375（可选）
- **悬挂系统**：前3后3+2钢板弹簧
- **轮胎规格**：7.00R16LT / 195/75R16LT

## 八、综合优势总结
✅ **大扭矩输出**：325N·m峰值扭矩，满载起步、爬坡轻松应对  
✅ **高效电机**：永磁同步电机效率可达 **95% 以上**  
✅ **长续航保障**：80kWh+大容量电池，满足城市配送全天需求  
✅ **快充便捷**：30分钟快充，提升运营效率  
✅ **低使用成本**：电耗约 0.3~0.5 元/公里，远低于燃油车

# 远程星智 技术参数介绍
  
更新时间：2026年07月12日15时44分51秒   

## 一、车型概述
远程星智是吉利远程新能源商用车集团旗下主打的新能源轻卡系列，主要面向城市物流、城际配送等场景。该系列提供纯电动（EV）和增程式（EREV）两种动力形式，具有高承载、长续航、智能化等特点。
---
## 二、动力系统
### 1. 纯电动版 (EV)
| 项目 | 参数 |
|------|------|
| **驱动电机类型** | 永磁同步电机 |
| **额定功率** | 50 kW |
| **峰值功率** | 120 kW |
| **峰值扭矩** | ≥325 N·m |
| **最高车速** | ≥90 km/h |
| **最大爬坡度** | ≥25% |
### 2. 增程版 (EREV)
| 项目 | 参数 |
|------|------|
| **发动机排量** | 1.5T / 2.0T（增程器） |
| **驱动电机峰值功率** | ≥100 kW |
| **综合续航里程** | ≥600 km（NEDC工况） |
---
## 三、电池系统
### EV版电池配置
- **电池类型**:磷酸铁锂电池
- **电池容量**:可选装多种电量包（如81 kWh / 86 kWh / 100 kWh等）
- **续航里程**:根据电量包不同，NEDC续航可达200 km ~ >300 km
- **充电方式**:支持快充与慢充
    - **快充时间**:约1小时（20% → 80%）
    - **慢充时间**:约6~8小时
- **电池质保**:8年或40万公里（以先到为准）
### EREV版电池配置
- **电池类型**:磷酸铁锂电池
- **电池容量**:约30~50 kWh
- **纯电续航里程**:≥100 km
- **综合续航里程**:≥600 km

## 五、底盘与悬架系统
-  前桥:2.4T级前桥
-  后桥:4T级后桥
-  悬架形式:前3后3+3钢板弹簧悬架（可选前3后5+3重载悬架）
-  制动系统:气刹制动 + ABS防抱死系统 + EBD电子制动力分配
-  轮胎规格:7.00R16LT / 7.50R16LT （根据载荷选配）

## 六、智能配置与安全辅助
### ADAS智能驾驶辅助系统(部分高配车型)
1. FCW前方碰撞预警
2. LDW车道偏离预警
3. AEB自动紧急制动(选装)
4. ACC自适应巡航(选装)
5. DMS驾驶员疲劳监测(选装)
### IVI智能网联系统
1.  12英寸/15英寸中控大屏(触控)
2.  支持语音控制导航、空调、车窗等
3.  车联网功能：实时监控车辆状态、剩余电量/油量提醒、
    路径规划充电站推荐。
4. OTA远程升级能力。
---
## 七、核心优势总结
1. ✅零排放或超低油耗：纯电零排放；增程综合油耗低于传统燃油车。
2. ✅路权优势：可享受绿牌政策及城市不限行待遇。
3. ✅高效运营：快充补能迅速；增程无里程焦虑。
4. ✅智能化程度高：L2级辅助驾驶 + OTA升级 + FOTA固件升级。
5. ✅可靠耐用：三电系统经过严苛测试，满足商用车高强度使用需求。

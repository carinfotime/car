# 🚗 亚洲龙（Avalon）儿童乘车注意事项
  
更新时间：2026年07月17日11时21分27秒   

## 1. 儿童安全座椅使用
- **必须使用符合国家标准的儿童安全座椅**，严禁怀抱儿童乘车。
- **12岁以下儿童**不得乘坐副驾驶位置。
- 建议将安全座椅安装在后排**中间或两侧位置**。
- 根据儿童年龄、体重和身高选择合适的安全座椅类型：
  - **0-1岁（约0-13kg）**：反向安装的婴儿提篮式座椅
  - **1-4岁（约9-18kg）**：正向安装的儿童安全座椅
  - **4-12岁（约15-36kg）**：增高垫+安全带
## 2. ISOFIX接口使用
亚洲龙后排配备有**ISOFIX儿童座椅固定接口**，安装时请注意：
- ISOFIX接口位于后排座椅靠背与坐垫之间的缝隙处
- 安装时听到“咔哒”声表示已锁定到位
- 配合顶部锚点（Top Tether）使用，增强稳定性
## 3. 车窗与车门安全
| ⚠️ 注意事项 | ✅ 正确做法 |
|---|---|
| ❌ 不要让儿童将头、手伸出窗外 | ✅ 开启车窗锁止功能 |
| ❌ 不要让儿童自行开关车门 | ✅ 启用后门儿童安全锁 |
| ❌ 不要在行驶中让儿童站立或爬动 | ✅ 全程系好安全带 |
## 4. 🚫车内禁止行为
1. **禁止在车内吃零食或喝饮料** — 防止呛噎或洒漏
2. **禁止玩尖锐玩具或硬物** — 急刹车时可能造成伤害
3. **禁止在行驶中解开安全带**
4. **禁止将身体探出天窗**
5. **禁止在车内打闹、蹦跳**
## 🛑特别提醒
> 🔒 **下车前务必检查后排座位，切勿将儿童单独留在车内！**
>
> 🌡️夏季车内温度可在短时间内升至60℃以上，极其危险！
## ✅家长操作清单
| # | ✔️ | 
|---|---|
| ☐ |启动车辆前确认所有乘客系好安全带 |

| ☐ |调整空调出风口，避免直吹儿童 |
| ☐ |行驶中保持平稳驾驶，避免急加速、急刹车 |

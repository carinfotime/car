# 锐程PLUS 驾驶注意事项
  
更新时间：2026年07月15日13时37分04秒   

## 一、新车磨合期（前1500公里）
1.  **避免急加速与急刹车**：平稳驾驶，让发动机和变速箱各部件充分磨合。
2.  **控制发动机转速**：建议转速不超过3000 rpm。
3.  **避免长时间匀速行驶**：适当变换车速和档位，有助于各档位齿轮的磨合。
4.  **不要超载**：减轻车辆负荷，保护悬挂和轮胎。
## 二、启动与熄火
1.  **一键启动流程**：
    -   不踩刹车，按一下启动键接通电源（让车辆自检）。
    -   等待仪表盘自检灯熄灭后，踩住刹车，再按一次启动键点火。
2.  **冷车启动**：
    -   点火后原地热车30秒至1分钟（冬季可适当延长），待转速回落至怠速状态再起步。
    -   起步后前几分钟避免高转速行驶，待水温表上升后再正常驾驶。
3.  **熄火注意事项**：
    -   **涡轮增压车型（锐程PLUS为全系涡轮）**：长途或激烈驾驶后，建议怠速运转1-2分钟再熄火，以保护涡轮增压器轴承散热。
## 三、行驶中
### 1. 双离合变速箱（DCT）使用技巧
-   **等红灯时**：若等待时间超过30秒，建议挂入N档并拉起电子手刹；短时间等待可挂D档踩刹车。不建议长时间D档踩刹车停车，以减少离合器片磨损和过热。
-   **拥堵路况**：在走走停停的极度拥堵路段，可尝试切换至手动模式或运动模式（S档），减少变速箱频繁换挡导致的顿挫和过热风险。
-   **坡道起步**：车辆标配坡道辅助功能（HHC），松开刹车后有约2秒保持力，请利用此时间从容踩油门起步。
### 2. 智能辅助驾驶系统
-   **自适应巡航（ACC）**：
    -   仅适用于高速公路或路况良好的封闭道路。
    -   雨天、雾天、夜间或弯道过急时慎用或关闭ACC。
    -   驾驶员需始终手握方向盘并观察路况，ACC仅为辅助功能。
-   **主动刹车/碰撞预警**：
    -   系统可能对金属井盖、减速带、路边金属护栏产生误判而触发警报或制动。此为正常现象。
    -   切勿依赖主动刹车代替人工制动。
### 3. 动力与操控
-   **涡轮介入特性**：锐程PLUS的涡轮介入点较早（约1500转），动力输出较为平顺。但深踩油门时仍有短暂延迟（涡轮迟滞），超车时应预留提前量。
-   **过弯速度**：SUV底盘重心相对轿车高一些（虽为轿车但悬挂偏舒适调校），过弯时请提前减速，避免高速入弯导致车身侧倾过大。
## 四、停车与驻车
1.  **自动驻车功能 (Auto Hold)**：
    -   D档状态下深踩刹车即可激活仪表盘绿色P标志亮起。轻点油门即可解除。
2.  **电子手刹使用规范**：
    停车步骤推荐顺序：
    踩住脚刹 → N档 → （拉电子手刹） → P档 → （松脚刹） →熄火
    此顺序可防止P档锁止机构因受力而卡滞难摘挡。
## 五、特殊天气与环境
| 场景 | 注意事项 |
|------|----------|
| **雨天/积水路面** | •关闭自动启停功能！涉水深度不超过排气管口。<br>•驶出积水区后轻点几次刹车排干刹车盘水分。<br>•路面湿滑时谨慎使用ACC自适应巡航。 |
| **冬季/冰雪路面** | •轮胎气压可适当调高至标准上限。<br>•冰雪路面起步若打滑可尝试关闭ESP车身稳定系统一小段距离后再开启。<br>•玻璃水务必更换为防冻型。 |
| **夏季高温/长途连续行驶** | •注意监控水温表及机油温度。<br>•连续行驶4小时左右建议休息20分钟让车辆散热。<br>•胎压监测系统报警时立即靠边停车检查。 |
## 六、定期检查要点
1.  **机油液位检查**:每两周检查一次机油尺刻度应在上下限之间偏上位置。（锐程PLUS推荐5W-30标号全合成机油）
2.  **冷却液液位**:冷车状态下观察副水箱刻度应在MAX-MIN之间。（注意不要热车开盖）
3.  **轮胎胎压**:标准胎压通常为前轮230kPa(2.3bar)，后轮210kPa(2.1bar)，具体参照驾驶员侧门框标签为准。（带胎压监测功能的可在中控屏查看数值）
4.  定期清理前风挡下方雨刮器排水槽落叶杂物防止堵塞导致车内进水。
## ⚠️重要安全提示
*严禁酒后驾车；*
*驾车途中请勿操作手机等分散注意力的行为；*
*儿童乘车必须使用符合其年龄体重的安全座椅且放置于后排；*
*以上内容基于通用驾驶规范及该车型常见特性编写具体操作请以随车用户手册为准。*

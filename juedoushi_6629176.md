# Jeep 角斗士 (Gladiator) 操作指南
  
更新时间：2026年07月20日09时07分20秒   

## 一、车辆启动与熄火
### 启动
1.  **智能钥匙**：确保智能钥匙在车内。
2.  **制动踏板**：将右脚踩住制动踏板。
3.  **启动按钮**：按下仪表台右侧的 **ENGINE START/STOP** 按钮。
4.  **等待自检**：观察仪表盘指示灯，等待系统自检完成（约2-3秒）。
5.  **松开手刹**：确认仪表盘上的红色“P”手刹灯熄灭（或手动释放电子手刹）。
### 熄火
1.  **挂入驻车挡**：将变速杆推至 **P** 挡。
2.  **拉起手刹**：按下或拉起电子手刹按钮（仪表盘显示红色“P”）。
3.  **熄火**：再次按下 **ENGINE START/STOP** 按钮。
4.  **锁车**：离开车辆后，触摸门把手感应区或按钥匙锁车键。
> **注意**：在陡坡停车时，建议先拉起手刹，再挂入P挡，以保护变速箱锁止机构。

## 二、驾驶模式选择（Selec-Trac / Command-Trac）
角斗士配备分时四驱系统，通过中控台的旋钮或挡杆选择模式。
| 模式 | 适用场景 | 操作说明 |
| :--- | :--- | :--- |
| **2H** (两驱高速) | 日常城市、高速道路 | 默认模式，后轮驱动，省油。 |
| **4H** (四驱高速) | 湿滑、沙地、碎石路、雪地 | 车速低于 88 km/h 时，直接旋动旋钮或挂入。 |
| **4L** (四驱低速) | 陡坡攀爬、深泥、岩石攀爬 | **必须停车，挂N挡**，再挂入或旋动。提供更大扭矩。 |
| **N** (空挡) | 被拖车或使用分动箱取力器 | 停车挂N挡后，再操作分动箱至N。 |
**切换步骤（以4L为例）：**
1.  将车辆停稳，踩住制动踏板。
2.  将变速箱挂入 **N** (空挡)。
3.  将分动箱操纵杆拉到底（或旋钮旋至4L）。
4.  仪表盘显示 **4L** 指示灯常亮，表示已接合。

## 三、可拆卸车顶与车门操作
### 硬顶拆卸（三件式）
1.  **拆卸前顶板**：
    -   拉下前遮阳板。
    -   旋转两侧锁扣至解锁位置（图标为开锁状态）。
    -   向上推起前顶板，从车顶取下。
2.  **拆卸后顶板**：
    -   打开尾门。
    -   拧松后顶板两侧的固定螺栓（需使用工具或手动旋钮）。
    -   将后顶板向后滑动并抬起取下。
### 车门拆卸
1.  **准备**：将车窗降至最低，拔掉门内连接线束插头（通常位于车门铰链处）。
2.  **拆卸**：使用随车工具或内六角扳手，拧松上下两个铰链固定螺栓（不要完全拧出）。
3.  **取下**：一只手托住车门底部，另一只手扶住车窗框，向上提起车门，脱离铰链。
4.  **存放**：将车门直立存放于安全位置，避免划伤。
> **警告**：拆卸车门后，车辆不再具备侧面碰撞保护，仅限低速越野或封闭场地使用。

## 四、越野功能操作
### 1. 前/后差速锁（如配备）
-   **开启条件**：必须在 **4L** 模式下。
-   **操作**：按下中控台对应的 **前桥锁** 或 **后桥锁** 按钮。
-   **顺序**：先锁后桥，再锁前桥。
-   **解除**：先解除前桥锁，再解除后桥锁；或直接切换出4L模式自动解除。
### 2. 电子稳定控制系统关闭
-   **短按**：关闭牵引力控制，保留部分车身稳定。
-   **长按（5秒以上）**：完全关闭电子稳定控制（ESC Off），适用于深沙或泥地。
### 3. 防倾杆断开（如配备）
-   **条件**：4L模式下，车速低于 32 km/h。
-   **操作**：按下中控台 **Sway Bar** 按钮。仪表盘指示灯亮起表示已断开，增加悬挂行程。

## 五、拖车与载货
### 拖车钩使用
-   **前拖车钩**：位于前保险杠两侧，红色或银色钩环，直接挂接拖车绳。
-   **后拖车钩**：位于后保险杠下方，同样为坚固钩环。
### 货箱装载
-   **最大载重**：请查阅驾驶座门框上的载重标签（通常约 550-620 kg）。
-   **固定货物**：使用货箱内的锚点（D型环）和捆绑带固定货物。
-   **货箱盖**：若配备可卷曲货箱盖，拉动尾门方向拉手即可关闭；打开时需解锁并向后卷起。
### 拖车牵引
-   **最大拖拽能力**：根据配置不同，约为 3,500 - 7,700 磅（约 1.5 - 3.5 吨）。
-   **拖车模式**：挂上拖车并连接线束后，系统自动识别。可在设置中开启 **拖车摇摆控制**。

## 七、保养与维护要点
| 项目 | 周期/标准 | 备注 |
| :--- | :--- | :--- |
| **发动机机油** | 每 10,000 km 或 1年 | 使用 0W-20 全合成机油（Pentastar发动机）。 |
| **分动箱油** | 每 80,000 km | 越野频繁时缩短至 40,000 km。 |
| **前后桥油** | 每 80,000 km | 若涉水或深度越野后应立即检查更换。 |
| **轮胎换位** | 每 10,000 km | 交叉换位，保持磨损均匀。 |
| **车顶密封条** | 每次拆装后检查 | 清洁并涂抹硅基润滑脂，防止漏水。 |
---
## 八、安全注意事项
1.  **越野安全**：永远不要单独越野，至少两辆车结伴。携带卫星电话或手台。
2.  **涉水**：角斗士最大涉水深度约 760 mm（30英寸）。涉水时保持低速匀速，切勿冲水。
3.  **车顶载物**：若拆卸前顶板，车顶行李架最大静载约 150 kg，动态行驶时减半。
4.  **儿童安全**：拆卸车门后，务必使用后排儿童锁，防止儿童意外打开车门。
5.  **燃油**：使用 91 号及以上无铅汽油（推荐辛烷值）。

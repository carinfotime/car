# 九龙C6 动力参数介绍
  
更新时间：2026年07月20日15时39分50秒   

## 发动机参数
| 项目 | 参数 |
|------|------|
| **发动机型号** | 4RB2（汽油）/ D19TCID1（柴油） |
| **发动机类型** | 直列四缸、水冷、四冲程 |
| **排量** | 2.4L（汽油）/ 1.9L（柴油） |
| **进气形式** | 自然吸气（汽油）/ 涡轮增压（柴油） |
| **最大功率** | 102 kW（约139马力）/ 5500 rpm（汽油） |
| | 82 kW（约111马力）/ 4000 rpm（柴油） |
| **最大扭矩** | 217 N·m / 4000 rpm（汽油） |
| | 300 N·m / 2000-2800 rpm（柴油） |
| **燃油供给方式** | 多点电喷（汽油）/ 高压共轨直喷（柴油） |
## 变速箱系统
- **变速箱类型**：5挡手动变速箱
- **驱动方式**：前置后驱
## 底盘与悬挂
- **前悬挂系统**：双横臂式独立悬挂
- **后悬挂系统**：变截面钢板弹簧非独立悬挂
- **转向系统**：液压助力转向
- **制动系统（前/后）**：盘式 / 鼓式
## 性能表现
- **最高车速**：≥145 km/h
- **燃油标号**：92号及以上无铅汽油（汽油版）/0号柴油（柴油版）
- **油箱容积**：70 L
- **排放标准**：国V/国VI
## *注*
九龙C6定位为轻型客车，动力调校偏向中低速扭矩输出，适合城市客运、商务通勤及短途旅游等场景使用。柴油版在低转速下扭矩充沛，爬坡及满载工况表现更优。

# 江铃福特 E福顺小卡 保养指南
  
更新时间：2026年07月12日10时05分48秒   

## 一、概述
E福顺小卡作为一款纯电动微卡，其保养核心与传统燃油车有显著区别。保养重点从发动机、变速箱、机油等转移到**三电系统（电池、电机、电控）**、高压线束及底盘结构件。正确的保养不仅能延长车辆寿命，更能保障行车安全。
## 二、核心保养周期表
| 项目 | 检查/更换周期 | 备注 |
| :--- | :--- | :--- |
| **减速器齿轮油** | **首次：** 5,000公里或3个月<br>**后续：** 每40,000公里或24个月 | 需使用专用油品 |
| **驱动电机冷却液** | **首次：** 5,000公里或3个月<br>**后续：** 每40,000公里或24个月 | 检查液位及冰点 |
| **空调滤芯** | **每10,000公里或12个月** | 视空气质量可缩短 |
| **整车高压线束及插接件** | **每10,000公里或6个月** | 检查有无松动、破损、腐蚀 |
| **制动液（刹车油）** | **每40,000公里或24个月** | DOT4级别 |
| **轮胎换位及动平衡** | **每10,000-15,000公里** | - |
| **底盘及悬架紧固件检查** | **每10,000公里或6个月** | - |
> *注：以上为建议周期，具体请以车辆随附的《用户手册》为准。若车辆用于重载、恶劣路况或频繁快充，应适当缩短保养间隔。*
## 三、关键部件详细保养说明
### （一）动力电池系统（重中之重）
1.  **日常使用建议**
    -   **浅充浅放：** SOC（荷电状态）保持在20%-90%之间使用最佳。
    -   **避免过放：** 仪表盘提示电量过低时请及时充电，长期亏电会损伤电池。
    -   **避免过充：** 充满后应及时结束充电，不建议长时间连接充电枪不拔。
2.  **定期检查**
    -   *外观检查：* （由专业技师操作）检查电池包外壳有无磕碰变形、漏液痕迹。
    -   *气密性检测：* （由专业技师操作）定期检测电池包气密性，防止进水短路。
3.  **存放注意事项**
    -   若车辆需长期停放（超过7天），建议将SOC保持在50%-60%。
    -   每隔一个月进行一次补电。
### （二）驱动电机与减速器
-   E福顺小卡采用永磁同步电机+单级减速器结构。
-   *无需更换机油*，但需要定期更换减速器齿轮油。
-   *冷却系统维护：* 确保电机冷却液充足且无泄漏。冷却液不足会导致电机过热限功率。
### （三）制动系统
-   *能量回收影响：* EV车型因有能量回收（滑行/制动回收），机械刹车片磨损通常比燃油车慢。
-   *定期检查刹车片厚度*，当厚度接近磨损极限标记时及时更换。
-   *制动液吸水性高*，务必按周期更换，防止沸点降低导致刹车失灵。
### （四）低压蓄电池（12V小电瓶）
-   负责给灯光、中控锁、BMS控制器等低压设备供电。
-   如果发现遥控钥匙不灵敏、启动困难（仪表盘亮起但无法上高压），可能是12V蓄电池亏电。
-   长期停放前应断开12V蓄电池负极接线。
### （五）空调系统
-   EV车型的空调压缩机由高压动力电池直接驱动（PTC加热）。
-   *定期更换空调滤芯*，保持车内空气清新并防止蒸发箱异味发霉。
## 四、日常自检清单（建议每周执行）
1.  检查轮胎气压及外观有无异常磨损/鼓包/扎钉。
2.  观察仪表盘有无故障灯报警（尤其是红色/黄色警示灯）。
3.  确认灯光系统全部正常工作（大灯/转向灯/刹车灯/雾灯）。
4.  目视检查底盘下方有无明显液体滴漏痕迹（冷却液为绿色或粉色）。
5.  测试驻车制动是否有效。
## ⚠️五、重要安全警告
1.  【高压危险】严禁私自拆卸橙色高压线束或维修三电部件！必须由持有电工证的授权服务站技师操作。维修前必须断电并等待电容放电完毕。
2.  洗车时避免用高压水枪直接冲洗机舱内的高压电器接口和充电座口。
3.  涉水行驶时注意水深不得超过厂家规定的安全涉水深度标准；通过积水路段后应轻踩刹车排除水分并确认制动正常。
4.  充电时应使用符合国标的合规充电桩；雷雨天气尽量避免露天充电；充电枪口保持干燥清洁无异物。

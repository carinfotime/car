## 三、动力系统
### 3.1 电机类型
- **永磁同步电机**（后驱/四驱可选）
### 3.2 驱动方式
| 版本 | 驱动形式 | 最大功率 | 最大扭矩 |
|------|----------|----------|----------|
| 后驱版 | 后置单电机 | 250kW（约340Ps） | 约450N·m |
| 四驱版 | 前后双电机 | 579kW（约787Ps） | 约800N·m |
> 注：高性能版本最大功率可达 579kW，零百加速约 3.48s。
## 四、电池与续航
| 项目 | 参数 |
|------|------|
| 电池类型 | 三元锂电池 / 磷酸铁锂 |
| 电池容量 | 71kWh / 90kWh / 100kWh（视版本而定） |
| 续航里程（CLTC） | 560km / 680km / 760km（视版本而定） |
| 快充时间 | 15分钟可充至80%（800V平台） |
| 充电平台 | **准900V高压平台**（最大工作电压875V） |
## 五、充电性能
- **最大充电功率**：约 396kW
- **充电倍率**：支持 3.75C 超快充
- **充电5分钟**：续航增加约200km
- **充电15分钟**：续航增加约500km
## 六、底盘与悬架
| 项目 | 参数 |
|------|------|
| 前悬架类型 | 双叉臂式独立悬架 |
| 后悬架类型 | 多连杆式独立悬架 |
| 驱动方式 | 后驱 / 电动四驱 |
| 制动系统 | 前通风盘 + 后通风盘 |
| 转向系统 | 电动助力转向（EPS） |
| 轮胎规格 | 235/50 R20 / 255/45 R20 或 前235/45 R21 + 后265/40 R21 |
## 七、智能座舱
- **车载芯片**：高通骁龙8295
- **屏幕配置**：
  - 26.3英寸中控+副驾一体屏
  - 10.5英寸中控下屏
  - 可选装半辐式方向盘
- **操作系统**：IM OS 智能座舱系统
- **语音交互**：四音区语音识别
- **音响系统**：可选21扬声器音响系统
## 八、智能驾驶
| 项目 | 参数 |
|------|------|
| 智驾芯片 | 英伟达Orin-X（高配双Orin-X） |
| 算力 | 254 TOPS（单）/ 508 TOPS（双） |
| 感知硬件 | 激光雷达 + 11摄像头 + 12超声波雷达 + 3毫米波雷达 |
| 智驾系统 | IM AD 智能驾驶辅助系统 |
| 功能亮点 | 城市NOA、高速NOA、自动泊车、记忆泊车、遥控泊车 |
## 九、安全配置
- **主动安全**：AEB自动紧急制动、FCW前碰撞预警、LDW车道偏离预警、BSD盲区监测、DOW开门预警
- **被动安全**：高强度钢铝混合车身、6安全气囊（气帘）、预紧式安全带
- **电池安全**：IP68防水防尘、热失控预警、云端BMS电池管理系统

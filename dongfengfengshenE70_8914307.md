# 东风风神E70 用户操作指南
  
更新时间：2026年07月16日09时45分16秒   

## 一、车辆启动与关闭
### 启动车辆
1. **携带智能钥匙**进入驾驶位，确保钥匙在车内。
2. **踩下制动踏板**（必须踩到底）。
3. **按下“一键启动”按钮**，仪表盘点亮，READY指示灯亮起，表示车辆已进入可行驶状态。
### 关闭车辆
1. 将车辆完全停稳，挡位置于 **P挡**。
2. 再次按下 **“一键启动”按钮**，仪表盘熄灭，车辆断电。
> **注意**：下车前请确认所有用电设备已关闭，避免蓄电池亏电。
---
## 二、换挡操作
| 挡位 | 操作方式 | 说明 |
|------|----------|------|
| P（驻车挡） | 按下挡杆顶端的P键 | 停车时使用 |
| R（倒车挡） | 按住挡杆侧面的解锁键，向前推 | 倒车时使用 |
| N（空挡） | 短时间停留或洗车时使用 | |
| D（前进挡） | 向后拉到底 | 正常行驶 |
| E（ECO模式） | D挡状态下再次向后拉一下 | 进入经济模式 |
- **电子旋钮式换挡**：部分车型采用旋钮换挡，顺时针旋转为D/R切换。
- **换挡时必须踩住刹车**。
---
## 三、驾驶模式选择
- **普通模式 (Normal)**：默认模式，适合日常城市路况。
- **ECO经济模式 (E)**：降低电机输出功率，延长续航里程。
- **强回收模式 (Strong Regen)**：
    - D/E模式下向左拨动换挡拨片或按下方向盘上的回收强度调节键。
    - 松开加速踏板时能量回收力度增强，实现单踏板效果。
---
## 四、仪表盘关键指示灯说明
| 图标/指示灯 | 含义 |
|-------------|------|
| READY（绿色） | 车辆已就绪可行驶 |
| EV / ECO / SPORT |当前驾驶模式|
| SOC百分比 /续航里程显示|剩余电量及预估续航|
| ⚠️红色电池图标/故障灯亮起 |高压系统故障或电池异常|
| 🔋黄色充电连接指示灯闪烁或常亮(插枪后)|充电状态指示|
> *如出现红色故障灯常亮且无法消除* → *请立即联系售后维修*

##  五、充电操作
###  交流慢充（家用/公共桩）
1.  确保车辆熄火并挂P档。
2.  打开充电口盖板（位于左前翼子板处）。
3.  插入交流充电枪 → “咔哒”一声锁止 →  仪表盘显示正在充电/呼吸灯闪烁绿色
4.  充满后自动停止 →  解锁车门后拔出充电枪 →  盖好盖子
### ⚡直流快充
1.  同样熄火挂P档 →  打开快充口盖板
2.  插入直流快充枪 →  听到锁止声即可
3.  通常30分钟内可从30%充至80%（视温度与桩功率而定）
4.  结束充电后先点击屏幕“停止充电”或在桩上点击停止→再拔枪
> ⚠️注意事项：
> -   雷雨天气尽量避免户外露天充电
> -   严禁湿手插拔充电枪
> -   长期停放建议保持电量在40%~60%

## ✅六、日常使用注意事项
1️⃣ **胎压检查**
建议每月检查一次胎压。标准值通常为：
-   前轮：2.3 bar
-   后轮：2.5 bar
具体以驾驶员侧门框标签为准。
2️⃣ **空调使用**
-   夏季开AC + AUTO即可自动调节温度与风量；
-   冬季制热建议开启ECO空调以节省电量；
-   长时间停车开空调休息注意通风安全。
3️⃣ **动能回收调节**
通过方向盘左侧拨片或中控屏设置可调整回收强度：
-   强回收：适合下坡/拥堵路段增加续航；
-   弱回收或无回收：适合高速滑行更平顺。
4️⃣ **保养周期**
每行驶12个月或20,000公里进行一次常规保养。主要项目包括：
- ✅检查高压线束及绝缘性能；
- ✅更换减速器齿轮油；
- ✅检查冷却液及制动液液位；
- ✅更新BMS及VCU软件版本（如有）。
5️⃣ **紧急情况处理**
若遇高压系统报警或无法上电：
①尝试重新下电等待5分钟后再次启动；
②若仍无法解决→拨打400服务热线寻求道路救援；
③切勿自行拆解高压部件！

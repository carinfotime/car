# 五菱龙卡 驾驶注意事项
  
更新时间：2026年07月13日13时57分33秒   

## 一、出车前检查
1.  **轮胎检查**：检查轮胎气压是否正常（参考驾驶室门框上的标准值），胎面有无破损、鼓包或异常磨损。
2.  **油液检查**：
    -   **机油**：拔出机油尺，检查油位是否在上下刻度线之间。
    -   **冷却液**：查看副水箱液位是否在“MAX”和“MIN”之间。
    -   **制动液**：检查制动液储液罐液位是否正常。
    -   **玻璃水**：确保玻璃水充足。
3.  **灯光与信号**：依次检查大灯（远近光）、转向灯、刹车灯、雾灯及双闪灯是否正常工作。
4.  **车身与货箱**：
    -   确认货箱门锁扣已锁紧，栏板无松动。
    -   检查货物是否捆绑牢固，避免行驶中散落或重心偏移。
5.  **底盘与传动**：目视检查车底有无漏油、漏水痕迹，传动轴及球头等部件有无异常。
## 二、启动与起步
1.  **挡位确认**：启动前确保挡位处于空挡（手动挡车型），并踩下离合器踏板。
2.  **预热**：冷车启动后，怠速运转30秒至1分钟，待发动机转速稳定后再起步。冬季可适当延长预热时间。
3.  **低速起步**：使用1挡平稳起步，避免猛抬离合器导致熄火或冲击传动系统。
## 三、行驶中注意事项
### （一）换挡操作
-   **转速匹配**：建议在发动机转速2000-2500rpm时换挡，避免拖挡（转速过低）或高转速运行。拖挡易造成发动机积碳和动力不足。
-   **离合操作**：换挡时离合器需踩到底，挂挡动作干脆利落。
### （二）速度控制
-   **经济时速**：五菱龙卡的经济时速通常在60-80km/h之间。保持此速度区间行驶可有效降低油耗。
-   **限速遵守**：严格遵守道路限速规定。满载时制动距离会显著增加，切勿超速。
### （三）转弯与变道
-   **提前减速**：转弯前充分减速，入弯后根据情况缓慢加速。货车重心较高，高速过弯有侧翻风险。
-   **观察盲区**：货箱较长且宽，变道或转弯时务必通过后视镜观察盲区。必要时转头确认侧方及后方情况。
### （四）制动技巧
-   **预见性驾驶**：提前观察路况，松油门利用发动机制动辅助减速，避免急刹车。
-   **点刹使用**：在湿滑路面或下长坡时，采用“点刹”方式（间歇性踩刹车），防止车轮抱死导致失控。
-   **下长坡注意**：
    -   严禁空挡滑行！必须挂入低速挡（如2或3挡），利用发动机制动控制车速。
    -   长时间连续踩刹车会导致刹车片过热、制动效能衰减（热衰退）。
### （五）载重与装载
-   **严禁超载**：严格按照车辆铭牌标定的总质量装载货物。超载会严重损坏车架、悬挂、轮胎及制动系统。
-   **均匀分布**：货物应均匀分布在货箱内，重心尽量居中靠前。避免单侧偏重或尾部过重影响操控稳定性。
## 四、特殊路况驾驶
| 路况类型 | 操作要点 |
| :--- | :--- |
| **雨天/湿滑路面** | •降低车速至干燥路面的70%以下。<br>•增大与前车的跟车距离（至少4秒以上）。<br>•避免猛打方向或急加速/急刹车。<br>•开启近光灯及前后雾灯提高辨识度。 |
| **山路/坡道** | •上坡提前降档保证动力充沛。<br>•下坡使用低速挡配合点刹。<br>•弯道鸣笛示警并靠右行驶。<br>•注意水温表变化防止开锅。 |
| **夜间行车** | •合理切换远近光灯（会车时切近光）。<br>•注意观察路面颜色变化判断坑洼或障碍物。<br>•疲劳时务必停车休息20分钟以上。 |
## 五、停车与熄火
1.  **选择平地停车位**
2. 拉紧手刹至最大行程位置
3.  手动挡车型挂入1档或倒档（上坡停车挂1档+方向朝路边；下坡停车挂倒档+方向朝路中心）
4.  熄火前怠速运转30秒左右再关闭发动机（尤其长途行驶后有助于涡轮/自然吸气发动机散热）
5.  锁好车门并带走贵重物品
## ⚠️特别警示
1.  ABS防抱死系统工作时会伴随刹车踏板弹脚异响属正常现象请勿松开刹车持续踩住即可
2.  仪表盘亮起红色警告灯时必须立即安全停靠排查故障严禁继续行驶
3.  定期保养周期严格参照用户手册执行更换机油机滤空滤等核心部件建议使用原厂配件规格

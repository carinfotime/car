# 仰望U7 用户操作指南
  
更新时间：2026年07月12日18时36分28秒   

## 一、车辆启动与关闭
### 启动车辆
1. **携带智能钥匙**或**已连接的手机**进入车辆。
2. 坐入驾驶位，系好安全带。
3. **踩下制动踏板**。
4. 仪表盘亮起，车辆进入 **“READY”** 状态（或听到提示音），表示启动完成。
5. 拨动换挡杆或旋转方向盘右侧的换挡拨轮，选择前进挡（D）或倒挡（R）。
### 关闭车辆
1. 踩下制动踏板，将车辆停稳。
2. 按下挡位操作区域的 **“P” 挡按键**（电子驻车）。
3. 解开安全带，打开车门，车辆将自动下电。
4. 如需手动下电，可在停车状态下按下中控屏上的 **“电源”** 图标。

## 二、驾驶操作
### 换挡操作
- **前进（D）**：在P挡或N挡时，将换挡拨轮向下旋转。
- **倒车（R）**：在P挡或N挡时，将换挡拨轮向上旋转。
- **空挡（N）**：在D挡或R挡时，轻拨换挡拨轮至中间位置并保持。
- **驻车（P）**：按下换挡区域顶部的 **“P”** 按键。
### 方向盘调节
- 调节拨杆位于方向盘左下方。
- 向下拨动拨杆解锁，调整方向盘高度与前后距离。
- 调整完毕后，向上推回拨杆锁定。
### 驾驶模式切换
- 通过中控屏或方向盘右侧的 **“驾驶模式”** 滚轮进行切换。
- **模式包括**：经济、舒适、运动、雪地、个性化（可自定义动力响应、悬架高度、转向力度等）。
### 易四方（e⁴）特色操作
- **原地掉头**：在车辆静止且方向盘回正状态下，通过中控屏激活“原地掉头”功能，按屏幕提示操作。
- **敏捷转向**：在低速泊车或窄路调头时，系统自动启用后轮转向（最大±10°），缩短转弯半径。
- **应急浮水脱困**：仅限意外涉水场景。车辆检测到水深超过安全阈值时，系统自动触发浮水模式，此时请保持方向盘正直，控制车速，尽快驶离深水区。

## 三、智能座舱操作
### 中控屏主界面
- **主页**：显示导航、音乐、车辆状态卡片。
- **车辆设置**：进入可调节悬架高度、转向模式、能量回收强度、灯光、门锁等。
- **空调控制**：屏幕底部常驻空调栏，可调节温度、风量、座椅加热/通风/按摩。
### 语音助手（“你好，仰望”）
- **唤醒**：说出“你好，仰望”或按下方向盘左侧的语音按键。
- **常用指令**：
    - “打开所有车窗”
    - “导航到[目的地]”
    - “空调调到24度”
    - “播放[歌曲名]”
    - “座椅按摩开启”
### 仪表盘信息
- **左侧**：当前驾驶模式、能量输出/回收状态。
- **中央**：车速、导航提示、辅助驾驶状态。
- **右侧**：电量/油量、续航里程、轮胎压力。
---
## 四、辅助驾驶系统
### 开启与关闭
- 按下方向盘左侧的 **“巡航”** 按键激活自适应巡航（ACC）。
- 再次按下 **“车道保持”** 按键（方向盘图标）激活车道居中辅助（LCC）。
- **设定速度**：向上/向下拨动左侧滚轮调整目标车速。
- **跟车距离**：按下左侧滚轮左右的“车距”按键调整（1-5档）。
### 注意事项
- 辅助驾驶仅为辅助功能，驾驶员必须时刻手握方向盘并观察路况。
- 系统可能无法应对静止障碍物、施工区域、恶劣天气等场景。
- 当系统提示“请接管”或发出警报音时，请立即人工接管。

## 五、能量补给
### 充电（纯电版）
1. 车辆解锁状态下，按压充电口盖板（左后翼子板）。
2. 插入充电枪，充电口指示灯呼吸闪烁表示连接成功。
3. 充电完成后，按压充电枪按钮解锁并拔出。
4. 关闭充电口盖板。
### 加油（插混版）
1. 车辆解锁状态下，按压加油口盖板（右后翼子板）。
2. 逆时针旋转加油口内盖。
3. 加注指定标号汽油（参见油箱盖内侧标注）。
4. 加注完成后，顺时针拧紧内盖，关闭外盖。

## 六、应急处理
### 车辆无法启动
- 检查智能钥匙是否在车内，或手机蓝牙是否连接。
- 检查制动踏板是否踩到底。
- 尝试使用中控台下方的 **“应急启动”** 按键（需用机械钥匙或手机NFC触碰感应区）。
### 轮胎气压报警
- 立即降低车速，避免急刹车。
- 检查轮胎外观是否有明显破损。
- 使用随车充气泵进行补气，或联系服务商救援。
### 紧急呼叫
- 车内后视镜上方有 **“SOS”** 按键。
- 按下后自动连接比亚迪紧急救援中心，请说明位置及情况。

## 八、安全须知
1. **儿童安全**：12岁以下儿童不得乘坐前排，必须使用合适的儿童安全座椅安装于后排。
2. **电池安全**：严禁私自拆卸或改装高压电池组件。若车辆发生碰撞，请及时联系服务商检查电池状态。
3. **涉水行驶**：虽然具备应急浮水能力，但请勿主动驶入深水区域。涉水后应检查制动效果。
4. **拖车**：必须使用平板式拖车，禁止仅抬起前轮或后轮拖行，以免损坏电机或电池。

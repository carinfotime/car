# 途观L插电混动 驾驶注意事项
  
更新时间：2026年07月14日18时29分50秒   

## 一、启动与行驶前检查
1. **充电状态确认**：出发前检查仪表盘显示的电池电量（SOC）及纯电续航里程，合理规划行程。
2. **充电枪与盖板**：确保充电枪已拔出，充电口盖板已完全关闭并锁止。
3. **轮胎与胎压**：检查轮胎外观及胎压（建议冷态胎压2.5-2.7 bar），混动车型车重较大，胎压不足会增加能耗。
4. **仪表盘警示灯**：启动后观察是否有故障灯亮起（如高压系统、驱动电机、12V蓄电池等）。
## 二、驾驶模式选择
| 模式 | 适用场景 | 注意事项 |
|------|----------|----------|
| **E-MODE（纯电）** | 市区通勤、短途行驶 | - 电量充足时优先使用<br>- 避免长时间大油门急加速<br>- 电量低于10%时自动退出 |
| **混合动力（Hybrid）** | 日常综合路况 | - 系统自动优化油电分配<br>- 高速巡航时发动机会介入更频繁 |
| **GTE（运动模式）** | 超车、山路或需要动力时 | - 油耗较高，慎用<br>- 发动机会持续运转并辅助电机输出最大扭矩 |
| **蓄电池维持/充电模式** | 预留电量给后续市区路段 | - BTM模式：保持当前电量<br>- CHG模式：发动机为电池充电（油耗较高） |
## 三、能量回收与制动
1. **滑行能量回收**：
   - D挡下松开油门，车辆滑行回收能量。
   - B挡（强制回收挡）：松开油门即进入强回收状态，适合下坡或拥堵跟车。
2. **制动踏板使用**：
   - GTE车型配备协调式再生制动系统，轻踩刹车优先回收能量。
   - 紧急情况请果断深踩刹车，机械制动会立即介入。
3. **避免空挡滑行**：N挡下无法进行能量回收且存在安全隐患。
## 四、充电注意事项
1. **充电设备选择**：
   - 支持交流慢充（最大3.6kW /7.2kW视版本而定）。
   - 不建议长期使用直流快充（除非车辆明确支持）。
2. **充电环境要求**：
   - 避免在极端高温或低温环境下立即充电。
   - ⛈️雷雨天气建议暂停室外充电。
3. **电池保护建议**：
   - SOC保持在20%-80%之间有助于延长电池寿命。
   - ⚠️长期停放建议保持50%左右电量。
## ⚠️五、安全与应急处理
1. **涉水行驶限制**
   
   最大涉水深度 ≤300mm
   涉水速度 ≤10km/h
   涉水后尽快检查高压部件是否进水
   
2. **碰撞后处理**
   - ⚡高压系统可能带电！切勿触碰橙色高压线束。
   - ✅立即熄火并远离车辆 → ✅呼叫救援 → ✅告知对方为插电混动车型。
3. **拖车规范**
   
   必须使用平板拖车！
   禁止两轮着地拖曳（会损坏电机/发电机）
   禁止N挡长距离滑行拖车
   
## 🔧六、日常维护要点
- [ ] **每5000公里或半年**:检查12V蓄电池状态及接线柱清洁度
- [ ] **每10000公里**:检查高压线束绝缘层有无破损
- [ ] **每20000公里**:更换空调滤芯（混动车空调压缩机由高压驱动）
- [ ] ❄️冬季注意:低温会降低纯电续航约20%-30%，提前开启座舱预热功能可改善体验
## 📋七、常见误区提醒
1. ❌ “混动车不需要热车” → ✅冬季建议启动后怠速30秒再行驶，让机油充分润滑发动机部件。
2. ❌ “纯电模式下可以一直开” → ✅定期让发动机运转一段时间，防止机油乳化及燃油变质。
3. ❌ “随便用家用插座就能充” → ✅必须使用可靠接地的专用插座(≥16A)，线路需满足负载要求。

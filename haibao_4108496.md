# 比亚迪海豹 驾驶注意事项
  
更新时间：2026年07月18日08时47分27秒   

## 一、启动与驾驶前检查
1.  **钥匙与启动**：确保智能钥匙在车内，踩下制动踏板，按下一键启动按钮。仪表盘亮起且“READY”指示灯点亮后，方可挂挡行驶。
2.  **电量检查**：出发前确认剩余续航里程，规划好行程，避免电量过低导致抛锚。
3.  **轮胎状态**：检查胎压是否正常（建议参考驾驶位车门 B 柱标签），观察轮胎有无明显损伤或鼓包。
4.  **充电口检查**：确认充电口盖已完全关闭并锁紧，防止行驶中意外弹开。
## 二、行驶中注意事项
1.  **驾驶模式选择**：
    -   **ECO（经济模式）**：适合城市拥堵路况，电门响应柔和，续航更长。
    -   **SPORT（运动模式）**：动力响应极快，电门敏感，超车时注意控制力度，避免突然窜出。
    -   **雪地/沙地模式**：仅在对应路况下使用，普通路面勿开启。
2.  **能量回收**：
    -   海豹配备可调能量回收强度（标准/较大）。
    -   **较大回馈**：松开电门时减速感明显，类似单踏板逻辑，需提前适应，避免乘客晕车。
    -   **雨天/湿滑路面**：建议使用“标准”回馈，防止突然减速导致车轮打滑或失控。
3.  **底盘与通过性**：
    -   海豹底盘较低，尤其是前唇位置。经过陡坡、高减速带或马路牙子时，务必 **斜向缓慢通过**，避免刮伤电池包或前保险杠。
4.  **智能驾驶辅助**：
    -   ACC（自适应巡航）和车道保持仅为辅助功能，双手 **不得长时间离开方向盘**。
    -   雨雾天气、车道线不清晰或弯道过急时，建议关闭辅助驾驶，改为人工操控。
## 三、充电注意事项
1.  **充电口位置**：海豹的快充口在右后侧（交流慢充口在左后侧），停车时注意朝向充电桩。
2.  **快充建议**：
    -   支持高压快充（如 180kW 以上），但频繁使用快充会加速电池衰减。日常建议多用慢充，长途再使用快充。
    -   快充时电量建议充至 80%-90% 即可，最后 10% 充电速度会显著变慢。
3.  **温度影响**：
    -   **冬季**：低温下电池活性降低，建议出发前使用“电池预加热”功能（可通过车机或 APP 开启），能提升充电速度和续航表现。
    -   **夏季**：高温暴晒后，建议先通风降温再充电，避免电池过热。
4.  **充电安全**：雷雨天气请勿在露天充电；充电时确保充电枪完全插入，锁止到位。
## 四、特殊路况与天气
1.  **雨天/积水路段**：
    -   虽然海豹电池防水等级高（IP67），但 **不可涉水超过轮毂中心**。
    -   涉水后轻踩刹车，蒸发刹车盘水分，恢复制动性能。
2.  **冬季/雪地**：
    -   海豹车重较大（约 2.1 吨），原厂轮胎在冰雪路面抓地力有限，建议更换雪地胎。
    -   开启 **雪地模式**，电门响应变缓，减少打滑风险。
3.  **高速行驶**：
    -   高速时风阻与能耗成正比，时速超过 110km/h 后，续航会明显下降。建议保持 100-110km/h 的经济时速。
    -   使用 **自适应巡航** 时，注意前车急刹，系统刹车力度可能偏晚或偏重，需随时准备接管。
## 五、停车与长期停放
1.  **停车技巧**：尽量选择平整地面，避免停在斜坡上长期悬挂，损害悬挂和电池包。
2.  **长期停放**：
    -   若需停放超过 7 天，建议保持电量在 **40%-60%** 之间，不要满电或亏电停放。
    -   停放期间建议每周远程启动一次空调或查看 APP 状态，防止小电瓶亏电。
3.  **洗车注意**：避免高压水枪直接冲洗前格栅（虽然封闭式，但内部有散热器）；充电口盖冲洗后及时擦干。
## 六、安全提醒
-   **紧急断电**：如遇碰撞或异常情况，长按一键启动按钮 3 秒以上可强制切断高压电。
-   **拖车**：海豹必须使用 **平板拖车** 或抬起驱动轮（后驱版为后轮）进行拖车，严禁直接拖拽，否则会损坏电机。
-   **儿童锁**：后门配备电子儿童锁，可通过车机设置或主驾门板按键控制，有儿童乘坐时请务必开启。
---
**总结：海豹动力强劲、底盘扎实，但作为电动车，需特别注意电池保养、底盘高度和能量回收的适应。安全驾驶，享受电驱乐趣！**

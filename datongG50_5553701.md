# 上汽大通G50 保养指南
  
更新时间：2026年07月17日12时35分12秒   

## 一、保养周期总表
| 保养项目 | 建议周期 |
|---------|---------|
| 首保 | 行驶3,000公里或3个月（以先到为准） |
| 常规保养 | 每7,500公里或6个月 |
| 大保养 | 每30,000公里或24个月 |
---
## 二、各阶段保养内容
### 1. 首保（3,000公里/3个月）
- 更换发动机机油
- 更换机油滤清器
- 检查冷却液液位及状态
- 检查制动液液位
- 检查转向助力油液位
- 检查变速箱油液位
- 检查轮胎气压及磨损情况
- 检查全车灯光系统
- 检查雨刮器及喷水系统
- 检查底盘及悬挂系统螺栓紧固情况
- 电脑检测车辆系统故障码
### 2. 小保养（每7,500公里/6个月）
- 更换发动机机油
- 更换机油滤清器
- 检查并补充冷却液
- 检查制动液
- 检查空气滤清器（视情况清洁或更换）
- 检查空调滤清器（视情况清洁或更换）
- 检查轮胎磨损及胎压
- 检查刹车片厚度
- 检查底盘各部件
- 检查电瓶状态
### 3. 大保养（每30,000公里/24个月）
在常规保养基础上增加以下项目：
- 更换空气滤清器芯
- 更换空调滤清器芯
- 更换火花塞（1.5T发动机）
- 更换燃油滤清器
- 更换制动液
- 更换防冻液
- 清洗节气门
- 清洗喷油嘴
- 检查并更换变速箱油（视情况）
- 检查正时系统（1.5T为链条，免维护但需检查）
---
## 三、关键油液规格
| 油液类型 | 规格要求 |
|---------|---------|
| 发动机机油 | SAE 5W-30 或 0W-20（API SN或以上，ACEA C3） |
| 机油容量 | 约4.5升（含滤芯） |
| 变速箱油 | 上汽DCT专用油（7速双离合） |
| 制动液 | DOT 4 |
| 防冻液 | 乙二醇基长效防冻液 |
| 转向助力油 | 电子助力，免维护 |
---
## 四、易损件更换参考
| 部件 | 建议更换周期 |
|------|------------|
| 机油滤清器 | 每次换油时更换 |
| 空气滤清器 | 每15,000公里或1年 |
| 空调滤清器 | 每15,000公里或1年 |
| 燃油滤清器 | 每30,000公里 |
| 火花塞 | 每30,000公里（1.5T发动机） |
| 刹车片（前） | 每30,000-40,000公里（视磨损） |
| 刹车片（后） | 每40,000-50,000公里（视磨损） |
| 蓄电池 | 每3-4年 |
| 雨刮片 | 每6-12个月 |
| 轮胎 | 每50,000-60,000公里或5年 |
---
## 五、日常检查项目（建议每月一次）
1. **机油液位**：冷车状态下检查，油位应在上下刻度之间
2. **冷却液液位**：应在MAX和MIN之间
3. **制动液液位**：应在MAX和MIN之间
4. **轮胎气压**：标准值见驾驶侧门框标签
5. **轮胎外观**：检查是否有鼓包、裂纹或异物扎入
6. **灯光系统**：检查大灯、尾灯、转向灯、刹车灯是否正常
7. **雨刮器**：检查刮水效果，如有异响或刮不净及时更换
8. **仪表盘警示灯**：启动自检后确认无异常灯亮起

## 七、保养注意事项
1. **机油选择**：建议使用上汽原厂机油或符合规格的大品牌机油，不建议混用不同品牌机油
2. **保养记录**：每次保养后保留好保养单据，质保期内需在授权服务站保养
3. **质保要求**：脱保可能导致保修失效，建议质保期内严格按4S店要求保养
4. **节气门清洗**：建议每30,000公里清洗一次，可改善怠速稳定性
5. **空调系统**：每年夏季前建议检查空调制冷效果，必要时补充冷媒
6. **颗粒捕捉器（GPF）**：大通G50国六车型配备GPF，需使用低灰分机油（ACEA C3标准）
7. **冬季保养**：寒冷地区建议提前检查防冻液冰点、电瓶状态及轮胎状况
---
## 八、常见故障及预防
| 常见问题 | 预防措施 |
|---------|---------|
| 发动机抖动 | 定期清洗节气门、更换火花塞 |
| 变速箱顿挫 | 按时更换变速箱油，避免暴力驾驶 |
| 空调制冷不足 | 定期检查冷媒压力，清洁冷凝器 |
| 电瓶亏电 | 长期停放建议每周启动运行20分钟以上 |
| 刹车异响 | 检查刹车片磨损，定期清洁刹车系统 |

# 别克君威 驾驶注意事项
  
更新时间：2026年07月15日11时26分29秒   

## 一、启动与熄火
1.  **启动前检查**：确认挡位在 P 挡或 N 挡，拉紧手刹。
2.  **一键启动**：不踩刹车按一下启动键为通电自检，待仪表盘指示灯熄灭后，踩刹车再次按下启动键点火。
3.  **冷车启动**：建议怠速热车 30-60 秒，待转速回落至 800-1000 转再缓慢行驶，避免高转速行驶。
4.  **熄火顺序**：踩刹车停稳 → 挂 N 挡 → 拉手刹 → 松脚刹（释放车身应力） → 挂 P 挡 → 熄火。
## 二、行驶中注意事项
1.  **变速箱保护**：
    -   行驶中禁止挂 N 挡滑行，会损坏变速箱。
    -   等红灯时，若时间短（30 秒内）可 D 挡踩刹车；时间长建议挂 N 挡拉手刹。
    -   倒车时必须在车辆完全停稳后再挂 R 挡。
2.  **涡轮增压车型（如 28T/GS）**：
    -   高速或长途行驶后，建议怠速 1-2 分钟再熄火，以冷却涡轮增压器，延长寿命。
    -   避免冷车时大脚油门。
3.  **自动启停**：
    -   涉水路段务必关闭自动启停，防止水中二次启动导致发动机损坏。
    -   拥堵路段建议关闭，避免频繁启停影响舒适性和电瓶寿命。
4.  **轮胎与胎压**：
    -   君威建议冷车胎压为 **240-260 kPa**。
    -   胎压过低会增加油耗和爆胎风险；胎压过高会影响舒适性和抓地力。
## 三、特殊路况
1.  **雨天/湿滑路面**：
    -   开启 ESP/ESC 车身稳定系统（默认开启，不要手动关闭）。
    -   避免急加速、急刹车、急转向，防止侧滑。
    -   打开后视镜加热和前后挡除雾功能。
2.  **高速公路**：
    -   保持安全车距，君威刹车脚感偏硬，需要提前预判。
    -   避免长时间定速巡航，适时切换人工控制。
3.  **涉水路段**：
    -   水深超过半个轮胎高度时请勿强行通过。
    -   通过时挂低速挡（M 挡手动模式），稳住油门匀速通过，中途不松油。
## 四、日常维护
1.  **机油检查**：定期检查机油尺，保持油量在上下刻度之间。推荐使用 5W-30 全合成机油。
2.  **冷却液**：定期检查冷却液液位，缺少时添加同品牌同型号冷却液。
3.  **变速箱油**：建议每 4-6 万公里检查或更换变速箱油。
4.  **燃油标号**：请按照油箱盖标注使用燃油（通常 92 号及以上，涡轮增压车型建议 95 号）。
## 五、其他提示
-   **座椅调节**：君威座椅包裹性较强，建议调整至腿部微弯、手腕可搭在方向盘顶端的位置。
-   **盲区监测**：A 柱较粗，转弯时注意左右摆头观察盲区。
-   **车机系统**：CarPlay/CarLife 建议使用原装数据线，避免连接不稳定。
-   **钥匙电池**：当遥控距离变短时，及时更换钥匙电池（型号 CR2032）。

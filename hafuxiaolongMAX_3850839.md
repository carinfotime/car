- [x] 空调滤清器清洁检查

- [ ] **更换减速器油**（混动专用变速箱油）
- [ ] **清洗节气门**
- [ ] **检查并清洁散热器及冷凝器**
- [ ] **检查驱动半轴防尘套**
### 4. C类保养（40,000公里 / 48个月）
包含B类全部项目，另增加：
- [ ] **更换冷却液**（长效型，建议4年更换）
- [ ] **更换动力电池冷却液**（如配备）
- [ ] **深度检查高压系统绝缘性能**
- [ ] **更换燃油滤清器**
- [ ] **清洗燃油系统积碳**
- [ ] **四轮定位检测与调整**
### 5. D类保养（60,000公里 / 60个月）
包含C类全部项目，另增加：
- [ ] **更换正时皮带及相关张紧轮**（1.5T发动机）
> ⚠️ *若为正时链条则仅需检查，无需定期更换*
> - ✅ *哈弗枭龙MAX搭载1.5L/1.5T混动专用发动机，通常为正时链条设计*

## 🔧 DIY日常自检项目
建议每月或长途出行前自行完成以下检查：
| # | 自检项目 | ✅ |
|---|---------|---|
| 🔹 | 【胎压】冷车胎压建议：前轮2.3bar /后轮2.3bar | ☐ |
| 🔹 | 【轮胎外观】有无鼓包、裂纹、异常磨损 | ☐ |
| 🔹 | 【灯光】大灯、转向灯、刹车灯、雾灯是否正常 | ☐ |
| 🔹 | 【玻璃水】补充至MAX刻度线 | ☐ |
| 🔹 | 【刹车盘片】目测厚度是否充足、有无沟槽异响 | ☐ |
| 🔹 | 【仪表盘】有无故障灯亮起、续航里程显示是否正常 | ☐ |

## ⚡ DHT混动系统专项注意事项
哈弗枭龙MAX搭载长城柠檬DHT混动系统，以下为特别关注点：
### 🧊 PHEV电池维护
| # |  建议内容 |
|---|----------|
| ✅ |  长期停放保持电量在40%~60%之间 |
| ✅ |  每月至少进行一次完整充放电循环 |
| ❌  避免长期亏电存放 （<20%）|
| ❌  避免长期满电存放 （>90%）|
### 🌀 DHT变速箱维护
> DHT专用变速箱油需使用长城认证的专用油品，不可用普通ATF代替。
### ⚡高压安全提示
> ❗️严禁私自拆卸橙色高压线束。涉及高压系统的维修必须由具备新能源维修资质的技师操作。

## 🧴推荐油液规格
### 🌿发动机机油
> SAE等级：0W‑20 （ACEA C5或API SP级别）  
> 💧用量：约4.0~4.5升  
> 🏷️推荐品牌：长城金吉星J700U、壳牌超凡喜力、美孚1号ESP配方  
### 🌀DHT变速箱/减速器油  
> 💧规格：长城DHT专用变速箱油  
> 💧用量：约3~4升  
### ❄️冷却液  
> 💧类型：长效有机酸型OAT冷却液  
> 💧冰点要求：‑35℃以下  
### 🛑制动液  
> 💧规格：DOT4及以上等级

## 📌省钱小贴士
1️⃣【空滤自己换】空气滤芯和空调滤芯可网购原厂件自行更换，省下工时费约80~120元。  
2️⃣【关注厂家活动】哈弗经常有“春季服务节”“秋季关爱活动”，工时打折甚至免费检测。

4️⃣【保险续保送券】续保商业险时常赠送基础保养券或洗车券。

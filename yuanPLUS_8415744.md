# 比亚迪 元PLUS 驾驶注意事项
  
更新时间：2026年07月20日13时24分18秒   

## 一、提车与初始检查
1.  **外观检查**：仔细检查车身漆面有无划痕、色差，轮毂有无刮伤。
2.  **内饰与功能**：确认座椅、方向盘、中控屏、空调、音响等所有电子设备功能正常。
3.  **随车物品**：核对随车工具（补胎液、充气泵、反光背心、三角牌）、充电设备（便携式充电枪、7kW壁挂式充电盒）是否齐全。
4.  **车辆信息**：核对车辆铭牌上的生产日期（建议不超过3个月）、VIN码是否与文件一致。
## 二、驾驶模式选择
元PLUS提供多种驾驶模式，适应不同场景：
| 模式 | 适用场景 | 特点 |
| :--- | :--- | :--- |
| **ECO (经济模式)** | 城市拥堵路况、追求长续航 | 动力输出柔和，电耗最低 |
| **NORMAL (标准模式)** | 日常通勤、综合路况 | 动力响应均衡，兼顾舒适与能耗 |
| **SPORT (运动模式)** | 超车、山路或需要强劲动力时 | 动力响应极快，推背感强，电耗较高 |
| **雪地模式** | 雨雪湿滑路面 | ESP介入更积极，限制扭矩输出防止打滑 |
> **建议**：日常使用 NORMAL 或 ECO；需要快速超车时可临时切换 SPORT；雨雪天气务必开启雪地模式。
## 三、能量回收与续航管理
1.  **能量回馈强度设置**：
    -   **标准回馈**：松油门滑行感接近燃油车，乘坐更舒适。
    -   **较大回馈**：松油门有明显拖拽感（单踏板逻辑），可回收更多能量并减少刹车使用。
    -   *新手建议先使用“标准”，适应后再尝试“较大”。*
2.  **续航预估技巧**：
    -   冬季低温（低于10℃）续航会明显下降（约打7-8折），请预留充足余量。
    -   高速行驶（时速>100km/h）电耗显著增加，规划长途出行时需提前确认沿途充电站。
3.  **电池保养提示**：
    -   **随用随充**：尽量保持电量在20%-80%之间，避免长期满电或亏电停放。
    -   **定期满充**：建议每1-2周进行一次慢充至100%，以校准电池SOC（荷电状态）。
## 四、充电安全与规范
1.  **优先使用慢充**：家用7kW交流慢充对电池寿命最友好。快充仅用于长途应急。
2.  **快充注意事项**：
    -   电量在20%-80%区间时快充功率最高；超过80%后功率会大幅下降以保护电池。
    -   极端高温或低温下快充速度会变慢，属正常现象。
3.  **禁止事项**：
    -   ❌严禁在车辆未熄火状态下拔插充电枪。
    -   ❌严禁使用破损或非国标的转接头/延长线。
    -   ❌雷雨天气避免露天充电。
## 五、特殊路况驾驶技巧
### （一）雨天/积水路段
-  元PLUS电池包位于底盘最低处，虽然防水等级较高（IP67），但切勿涉水超过半个车轮高度（约25cm）。
-  通过积水后应轻踩刹车数次，利用摩擦生热蒸发刹车盘水分。
### （二）冬季用车
-  提前通过手机APP开启“空调预约”和“电池加热”，可减少冷启动时的电量损耗并保护电池。
-  冰雪路面起步轻柔，避免大脚电门导致轮胎打滑。
### （三）长下坡路段
-  建议将能量回馈设置为“较大”，利用电机反拖制动减少刹车片磨损和过热风险。
## ⚠️六、核心安全警告
1.  **静音风险 (AVAS)**：纯电车低速行驶时极为安静。在小区或行人密集区请留意周围行人及非机动车。元PLUS标配低速行人警示音(AVAS)，请勿随意关闭此功能线束或保险丝。
2.  **底盘保护**：元PLUS的电池包位于底盘下方且无额外金属护板。经过坑洼路面或马路牙子时务必减速慢行，避免托底撞击电池包——这是最严重的安全事故之一！
3.  **紧急断电操作**：如遇碰撞或其他紧急情况导致车辆无法正常断电时：
    1.  解开安全带 → P挡亮起自动驻车；
    2.  长按P挡按钮3秒以上可强制切断高压电源；
    3.  若仍无效则打开前机盖断开12V小电瓶负极。

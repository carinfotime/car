# 骏航ES 保养指南
  
更新时间：2026年07月11日04时16分56秒   

## 一、保养周期总览
| 保养项目 | 建议周期 | 备注 |
|---------|---------|------|
| 机油 & 机滤 | 每 5,000 公里 或 6 个月 | 以先到为准 |
| 空气滤芯 | 每 10,000 公里 或 12 个月 | 视路况可提前 |
| 空调滤芯 | 每 10,000 公里 或 12 个月 | 建议换季前检查 |
| 燃油滤芯 | 每 20,000 公里 | — |
| 火花塞 | 每 30,000 公里 | 涡轮增压车型建议缩短 |
| 刹车油 | 每 40,000 公里 或 2 年 | 检测含水量超标即换 |
| 防冻液 | 每 40,000 公里 或 2 年 | — |
| 变速箱油 | 每 60,000 公里 | CVT/AT 均需更换 |
| 刹车片 | 每 30,000–50,000 公里 | 视磨损情况 |
| 轮胎换位 & 动平衡 | 每 10,000 公里 | — |
| 正时皮带（如有） | 每 80,000–100,000 公里 | 具体看发动机型号 |
| 电瓶 | 每 2–3 年检查 | 冬季前重点检测 |

## 二、日常自检项目（建议每月一次）
- 机油液位：冷车状态下拔出机油尺，液面应在上下刻度之间。
- 冷却液液位：查看膨胀壶，液面应在 MIN 与 MAX 之间。
- 制动液液位：同上。
- 轮胎胎压：标准值见驾驶侧门框标签（通常 2.3–2.5 bar）。
- 轮胎外观：检查是否有鼓包、裂纹、异常磨损。
- 灯光：全车灯光（大灯、转向灯、刹车灯、雾灯）功能正常。
- 雨刮水：加注专用玻璃水，冬季注意防冻。

## 三、机油选择建议
| 发动机类型 | 推荐粘度 | 标准 |
|-----------|---------|------|
| 自然吸气 | 5W-30 或 0W-20 | API SN 或 SP 级 |
| 涡轮增压 | 5W-30 或 0W-20 | API SP / ILSAC GF-6 |
| 高里程（>10万公里） | 5W-30 高里程配方 | — |
- 建议使用全合成机油，延长发动机寿命。
- 每次更换机油量请参考随车手册（通常 3.5–4.5 升）。

## 四、各系统保养要点
### 1. 发动机系统
- 每 5,000 公里更换机油及机滤。
- 每 10,000 公里检查或更换空气滤芯。
- 每 30,000 公里更换火花塞。
- 每 20,000 公里清洗节气门（视积碳情况）。
- 建议每 5,000 公里添加一次燃油添加剂（清除积碳）。
### 2. 制动系统
- 刹车片厚度低于 3mm 时必须更换。
- 刹车盘出现明显沟槽或跳动时需光碟或更换。
- 刹车油含水量超过 3% 必须更换。
- 每 2 年或 40,000 公里更换刹车油。
### 3. 轮胎系统
- 胎压建议：前轮 2.3 bar / 后轮 2.3 bar（以车门标签为准）。
- 每 10,000 公里进行四轮换位（交叉换位）。
- 轮胎花纹深度低于 1.6mm 必须更换。
- 轮胎使用超过 5 年（即使花纹良好）建议更换。
### 4. 变速箱
- CVT 变速箱：每 60,000 公里更换 CVT 专用油。
- AT 变速箱：每 60,000 公里更换 ATF 油。
- 严禁混用不同规格变速箱油。
- 建议同时更换变速箱滤芯（如有）。
### 5. 冷却系统
- 每 2 年或 40,000 公里更换防冻液。
- 使用与原车相同颜色的防冻液（通常为绿色或粉色）。
- 不可混用不同颜色的防冻液。
### 6. 空调系统
- 每 10,000 公里更换空调滤芯。
- 每 2 年清洗蒸发箱（防止异味）。
- 制冷效果下降时检查冷媒压力。
### 7. 电气系统
- 电瓶观察孔绿色为正常，黑色需充电，白色需更换。
- 每 2 年检查发电机皮带是否龟裂。
- 定期检查全车线束有无老化、破损。

## 五、骏航ES 常见保养注意事项
1. **机油型号确认**：骏航ES 不同年款发动机对机油粘度要求不同，请以发动机盖或随车手册为准。
2. **变速箱类型**：确认是 CVT 还是 AT，两者油品不可通用。
3. **正时系统**：部分发动机使用正时链条（免维护），部分使用皮带（需定期更换），请确认自己车辆的配置。
4. **电子手刹**：更换后刹车片时需用诊断仪进入保养模式，不可强行顶回。
5. **胎压监测**：轮胎换位后需重新学习胎压传感器位置（部分车型需到店操作）。

## 六、保养记录建议
建议每次保养后记录以下信息：
- 保养日期 & 里程数
- 更换项目及配件品牌/型号
- 下次保养里程提醒
- 保养费用
可使用手机备忘录或专门的养车 App 记录，便于后续维保参考。

## 七、简易保养计划表（按里程）
| 里程（公里） | 必做项目 | 建议检查项目 |
|------------|---------|-------------|
| 5,000 | 机油 + 机滤 | 轮胎、灯光、液位 |
| 10,000 | 机油 + 机滤、空气滤芯、空调滤芯 | 轮胎换位、刹车片 |
| 20,000 | 机油 + 机滤、燃油滤芯 | 节气门清洗、火花塞检查 |
| 30,000 | 机油 + 机滤、火花塞 | 刹车油检测、变速箱油检查 |
| 40,000 | 机油 + 机滤、刹车油、防冻液 | 电瓶检测、皮带检查 |
| 60,000 | 机油 + 机滤、变速箱油、火花塞 | 全车油液更换、底盘检查 |
| 80,000+ | 正时皮带（如有）、全车检查 | 减震器、胶套、底盘件 |
---
> **温馨提示**：以上为通用保养建议，具体保养项目及周期请以骏航ES 随车《用户手册》为准，不同年款和配置可能存在差异。如遇特殊情况（如长期短途行驶、恶劣路况、高温/严寒环境），建议适当缩短保养周期。

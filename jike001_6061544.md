# 极氪001 动力参数介绍
  
更新时间：2026年07月18日14时32分33秒   

## 一、动力系统概览
极氪001提供多种动力配置，涵盖单电机后驱与双电机四驱版本，满足不同用户需求。
---
## 二、各版本动力参数
### 1. WE版（单电机后驱）
| 参数项 | 数值 |
|--------|------|
| 驱动形式 | 后置后驱 |
| 最大功率 | **200 kW（272 Ps）** |
| 最大扭矩 | **343 N·m** |
| 0-100 km/h加速 | **6.9秒** |
| 最高车速 | 200 km/h |
### 2. WE版（双电机四驱）
| 参数项 | 数值 |
|--------|------|
| 驱动形式 | 双电机四驱 |
| **前电机**：最大功率 | **200 kW（272 Ps）** |
| **前电机**：最大扭矩 | **343 N·m** |
| **后电机**：最大功率 | **200 kW（272 Ps）** |
| **后电机**：最大扭矩 | **343 N·m** |
| **系统综合功率** | **400 kW（544 Ps）** |
| **系统综合扭矩** | **686 N·m** |
| **0-100 km/h加速** | **3.8秒（含弹射模式）** |
| 最高车速 | 240 km/h |
### 3. ME版 / YOU版（双电机四驱）
> ME版与YOU版在动力参数上与WE版双电机四驱一致。
- ✅ **综合功率：400 kW**
- ✅ **综合扭矩：686 N·m**
- ✅ **0-100 km/h加速：3.8秒**

## 三、电池与续航
### （1）电池类型
- ✅ **三元锂电池**
- ✅ **宁德时代/威睿电动供应**
### （2）电池容量及续航（CLTC标准）
#### 🅰️ WE版
- 💡搭载86 kWh电池包：
    - 🔹单电机后驱续航：约526 km
    - 🔹双电机四驱续航：未单独公布
- 💡搭载100 kWh电池包：
    - 🔹单电机后驱续航：约741 km
    - 🔹双电机四驱续航：约656 km
#### 🅱️ ME版 / YOU版
- 💡搭载100 kWh电池包：
    - 🔹双电机四驱续航：约656 km

## ⚡充电性能
### 🚀快充能力
极氪001支持最高360 kW超快充，搭配极氪极充桩可实现：
> ✅“充电5分钟，续航120公里”
具体充电时间参考：
> 💡从10%充至80%电量，最快仅需约30分钟（使用400V快充桩）
>
> 💡配合800V高压平台车型，充电速度进一步提升
---
## ⚙️底盘与操控相关动力特性
### 🛞悬挂系统
✅前悬架类型：双叉臂式独立悬架  
✅后悬架类型：多连杆式独立悬架  
✅部分车型可选装空气悬架 + CCD电磁减振系统  
### 🏎️驾驶模式
提供多种驾驶模式选择：
> 🅰️经济模式 → 🅱️舒适模式 → 🅲运动模式 → 🅳雪地模式 → 🅴沙地模式 → 🅵越野模式  
不同模式下电机的输出特性、能量回收强度、底盘高度等会相应调整。

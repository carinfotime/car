# 福特智趣烈马（Ford Bronco Sport）保养指南
  
更新时间：2026年07月11日01时53分47秒   

## 一、定期保养周期表
| 保养项目 | 建议周期 |
|---------|---------|
| 更换机油及机滤 | 每5,000公里或6个月 |
| 轮胎换位及动平衡 | 每10,000公里 |
| 空气滤清器更换 | 每20,000公里或1年 |
| 空调滤清器更换 | 每20,000公里或1年 |
| 火花塞检查/更换 | 每40,000公里 |
| 制动液更换 | 每40,000公里或2年 |
| 变速箱油检查/更换 | 每60,000公里 |
| 冷却液更换 | 每60,000公里或3年 |
| 分动箱/差速器油（四驱） | 每60,000公里 |
| 正时皮带（如适用） | 每100,000公里 |
> **注意**：以上为常规建议，具体请以车辆用户手册为准。恶劣路况或极端气候下应适当缩短保养间隔。
---
## 二、日常自检项目（建议每月一次）
- **机油液位**：冷车状态下检查，保持在上下刻度线之间
- **冷却液液位**：观察副水箱，应在MIN与MAX之间
- **制动液液位**：不低于MIN线
- **轮胎气压**：建议冷胎气压2.3–2.5 bar（具体参考驾驶侧门框标签）
- **轮胎磨损**：检查花纹深度，低于1.6mm需更换；观察有无偏磨、鼓包
- **灯光系统**：大灯、转向灯、刹车灯、雾灯是否正常
- **雨刮片及玻璃水**：雨刮片有无老化裂纹，玻璃水及时补充
---
## 三、机油选择建议
- **推荐标号**：SAE **5W-30**（全合成机油）
- **认证标准**：符合 Ford WSS-M2C946-A / B1 / BA
- **容量参考**：约5.0–5.4升（以实际车型为准）
- **品牌推荐**：
    - Mobil Super™ / Mobil Advanced Fuel Economy
    - Castrol EDGE / Magnatec
    - Shell Helix Ultra
    - Ford原厂机油
---
## 四、关键部件保养要点
### 🔧制动系统
-  前刹车片厚度低于3mm时需更换
-  刹车盘出现深沟槽或抖动时需光碟或更换
-  建议每次换机油时同步检查刹车片厚度
### 🔧轮胎系统
-  建议前后交叉换位，延长使用寿命
-  备胎也需定期检查气压和状态
-  烈马配备全地形胎时，注意胎噪和磨损均匀性
### 🔧四驱系统（4x4）
-  分动箱油和前后差速器油按周期更换
-  越野后检查底盘是否有磕碰、漏油迹象
-  涉水后检查差速器通气孔是否堵塞
### 🔧电瓶维护
-  检查电瓶接线柱是否氧化腐蚀
-  长期停放超过两周建议使用充电器维护

# 福田风景i 保养指南
## 一、概述
福田风景i 作为一款纯电动轻客/物流车，与传统燃油车的保养有较大区别。其核心在于 **三电系统（电池、电机、电控）** 的维护，同时兼顾底盘、制动、空调等常规部件的检查与更换。
> **建议保养周期：每行驶 10,000 公里 或 每6个月（以先到为准）进行一次常规检查。**
---
## 二、核心三电系统保养
### 1. 动力电池
- **外观检查**：每次保养时检查电池包外壳有无磕碰、变形或漏液。
- **电量管理**：日常使用建议保持电量在 **20% - 80%** 之间，避免长期满电或亏电停放。
- **均衡充电**：每2-3个月进行一次慢充充满至100%，以激活电池均衡系统。
- **冷却系统**：检查电池冷却液液位及管路是否泄漏（如有液冷系统）。
### 2. 驱动电机
- **无异物干涉**：检查电机表面是否有异物缠绕或油污堆积。
- **运行噪音**：启动及行驶中注意电机有无异常啸叫或异响。
- **无需更换机油**：纯电动车电机无需更换机油和机滤。
### 3. 电控系统（VCU/MCU）
- **故障码读取**：每次保养使用诊断仪读取系统故障码。
- **线束检查**：检查高压线束及低压控制线束有无磨损、松动或氧化。
---
## 三、底盘与悬挂系统
| 项目 | 检查内容 | 建议周期 |
|------|----------|----------|
| **轮胎** |胎压（标准值参考车门标签）、胎面磨损情况、有无鼓包或偏磨 |每月自查 /  每1万公里换位 |
| **制动片/制动盘** |厚度是否低于极限值（前片通常≤3mm需更换） |每2万公里检查 |
| **制动液** |含水量是否超标（＞3%需更换） |每2年或4万公里更换 |
| **减震器/悬挂衬套** |有无漏油、异响或旷量 |每1万公里检查 |
| **转向拉杆/球头** |间隙是否正常，防尘套有无破损 |每1万公里检查 |
---
## 四、空调与冷却系统
### A.  空调系统
- **空调滤芯**:  建议每10,000公里或6个月更换一次（视使用环境可缩短）。
- **制冷剂压力**:  如制冷效果下降，需检测冷媒压力并补充。
- **蒸发器/冷凝器**:  定期清理表面灰尘杂物，防止散热不良。
### B.  驱动电机冷却液
> *注意：部分车型电机采用独立冷却回路。*
-  冷却液类型:  通常为专用长效防冻液 (如G12/G13标准)。
-  更换周期:  每4年或8万公里更换一次。
-  日常检查:  观察副水壶液位在MIN-MAX之间。

## ✅六、日常自检清单（车主可自行操作）
1. [ ]  充电插头及插座是否清洁干燥
2. [ ]  轮胎胎压是否正常
3. [ ]  灯光全亮无损坏
4. [ ]  雨刮片刮拭效果良好
5. [ ]  玻璃水充足
6. [ ]  仪表盘无异常报警灯亮起
7. [ ]  驻车制动功能正常
---
## ⚡七、特别注意事项（新能源专属）
1. ✅　禁止私自改装高压电路及相关部件。
2. ✅　洗车时避免高压水枪直接冲洗充电口及机舱高压部件。
3. ✅　长时间停放(超过15天)，建议保持电量在50%-60%。
4. ✅　涉水深度不要超过轮毂中心线(约30cm)，防止电池进水。
5. ❌　不要等到续航显示为0再充电——这会加速电池衰减。

## 🛠️ 定期保养周期表
### A类保养（每5000公里或6个月，以先到为准）
- **更换机油**：建议使用 **5W-30**（四季通用）或 **10W-40**（夏季），API等级 SL及以上
- **更换机油滤清器**
- **检查并补充冷却液**
- **检查制动液液位**
- **检查转向助力油液位**
- **检查轮胎气压及磨损情况**
- **检查灯光、喇叭、雨刮器**
### B类保养（每10000公里或12个月，以先到为准）
包含A类全部项目，另增加：
- **更换空气滤清器滤芯**
- **更换空调滤清器滤芯**（如配备）
- **检查火花塞状况**
- **检查正时皮带/链条状态**
- **检查底盘螺栓紧固情况**
### C类保养（每20000公里或24个月）
包含B类全部项目，另增加：
- **更换燃油滤清器**
- **清洗节气门**（视积碳情况）
- **检查并调整离合器自由行程**
- **检查制动片厚度**
### D类保养（每40000公里或48个月）
包含C类全部项目，另增加：
- **更换火花塞**（建议使用原厂型号）
- **更换制动液**（DOT4）
- **更换变速箱油**（GL‑4 75W‑90）
- **更换后桥齿轮油**（GL‑5 80W‑90）
- **清洗喷油嘴**

## ⚡常见故障与处理建议
### ❗启动困难
1.  检查电瓶电压是否正常
2.  确认燃油量充足
3.  查看火花塞是否老化积碳
4.  清洗节气门和喷油嘴
### ❗怠速不稳/抖动
1.  清洗节气门和怠速马达
2.  检查点火线圈和高压线
3.  查看进气系统有无漏气
4.  使用燃油添加剂清除积碳
### ❗油耗异常升高
1.  胎压是否过低 → *标准胎压：前轮2.3bar /后轮2.8bar*
2.  空气滤芯是否过脏 → *及时更换*
3.  氧传感器是否失效 → *需用诊断仪读取数据流*
4.  驾驶习惯与路况影响 → *平稳驾驶可有效降低油耗*

## ✅日常自检清单 (每周/每次出车前)
1️⃣    绕车一周：观察轮胎外观、车身有无刮擦  
2️⃣    打开引擎盖：  
    -   拔出机油尺查看油位和颜色  
    -   观察冷却液储液罐液位  
    -   观察制动液储液罐液位  
    -   观察玻璃水余量  
3️⃣    进入驾驶室：  
    -   踩下刹车踏板感受力度是否正常  
    -   启动发动机听怠速声音有无异响  
    -   开启所有灯光确认功能正常

## 📌重要提醒
✅    首保时间：购车后3个月或行驶5000公里内完成首保。  
✅    质保期内务必在授权服务站进行保养并保留记录。  
✅    不同批次车辆可能存在差异，具体以随车《用户手册》为准。  
✅    恶劣工况下行驶应适当缩短保养间隔：
>     如频繁短途行驶、多尘环境、重载运输、山路爬坡等。
✅    轮胎换位建议每10000公里进行一次交叉换位。
✅    正时链条一般免维护，但若出现异响应及时检修。

# 起亚奕跑（Kia Stonic / KX1）操作指南
  
更新时间：2026年07月11日04时58分15秒   

## 一、启动与熄火
### 启动车辆
1. **智能钥匙启动**
   - 携带智能钥匙进入车辆
   - 踩下制动踏板（自动挡）或离合器踏板（手动挡）
   - 按下 **ENGINE START/STOP** 按钮，仪表盘指示灯亮起即启动成功
2. **钥匙启动（如有）**
   - 将钥匙插入点火开关
   - 拧至"ON"位置，等待仪表自检完成
   - 拧至"START"位置启动发动机
### 熄火
- 将挡位置于 **P挡**（自动挡）或 **空挡**（手动挡）
- 拉起手刹
- 按下 **ENGINE START/STOP** 按钮熄火
---
## 二、挡位操作（自动挡）
| 挡位 | 说明 |
|------|------|
| **P** | 驻车挡，停车熄火时使用 |
| **R** | 倒车挡 |
| **N** | 空挡，临时停车或拖车时使用 |
| **D** | 前进挡，正常行驶使用 |
| **+ / -** | 手动模式，可手动升降挡 |
> 换挡时必须踩下制动踏板，确保车辆完全停稳后再切换 R/P 挡。
---
## 三、仪表盘指示灯说明
| 图标 | 含义 | 处理方式 |
|------|------|----------|
| 🔴 电池灯 | 充电系统故障 | 尽快检查发电机或电瓶 |
| 🛢️ 机油灯 | 机油压力过低 | 立即停车检查机油量 |
| 🌡️ 水温灯 | 发动机过热 | 停车冷却，检查冷却液 |
| ⚠️ 发动机故障灯 | 发动机/排放系统异常 | 尽快到维修站检测 |
| 🔒 防盗灯 | 未检测到智能钥匙 | 检查钥匙是否在车内 |

## 四、驾驶辅助功能
### 1. 倒车影像
- 挂入 **R挡** 自动激活
- 屏幕显示后方画面及引导线
- 黄色/红色区域表示距离警示
### 2. 胎压监测（TPMS）
- 胎压异常时仪表盘亮起警示灯
- 检查并补充轮胎气压后，行驶一段距离自动复位
### 3. 自动启停（ISG）
- 默认开启，等红灯时发动机自动熄火
- 松开制动踏板自动重启
- 可通过中控台 **ISG OFF** 按钮关闭
### 4. 车身稳定系统（ESC）
- 默认开启，防止侧滑
- 按下 **ESC OFF** 可关闭（非必要不建议关闭）

## 五、灯光与雨刷
### 灯光控制（方向盘左侧拨杆）
| 操作 | 功能 |
|------|------|
| 旋至 **OFF** | 关闭灯光 |
| 旋至 **AUTO** | 自动大灯 |
| 旋至 **示宽灯** | 位置灯 |
| 旋至 **近光灯** | 近光灯 |
| 向前推 | 远光灯常亮 |
| 向后拉 | 远光灯闪一次 |
### 雨刷控制（方向盘右侧拨杆）
| 操作 | 功能 |
|------|------|
| 向上拨 | 单次刮刷 |
| 向下至 INT | 间歇刮刷（可调节频率） |
| 向下至 LO | 低速连续刮刷 |
| 向下至 HI | 高速连续刮刷 |
| 向自己拉动 | 喷水清洗挡风玻璃 |

## 六、空调与多媒体
### 空调系统
- **A/C**：开启压缩机（制冷）
- **温度旋钮**：蓝色为冷风，红色为热风
- **风量旋钮**：调节风速
- **内外循环**：城市拥堵用内循环，高速用外循环
- **后窗除雾**：按下后窗加热按钮
### 多媒体系统
- **音量旋钮**：调节音量 / 按下静音
- **MODE**：切换音源（收音机/USB/蓝牙）
- **蓝牙连接**：手机搜索 "Kia" 配对即可
- **USB接口**：支持 U 盘播放音乐 / 手机充电

## 七、日常维护与检查
### 定期检查项目（建议每月一次）
- ✅ 机油液位（冷车时检查油尺）
- ✅ 冷却液液位（膨胀壶刻度之间）
- ✅ 制动液液位
- ✅ 轮胎气压（标准值见驾驶侧门框标签）
- ✅ 雨刮水液位
### 保养周期
| 项目 | 周期 |
|------|------|
| 机油机滤更换 | 每 5000~7500 公里 |
| 空气滤芯 | 每 10000 公里 |
| 空调滤芯 | 每 10000 公里 |
| 火花塞 | 每 40000 公里 |
| 变速箱油 | 每 60000 公里 |
| 制动液 | 每 2 年或 40000 公里 |

## 八、应急处理
### 1. 轮胎漏气/爆胎
- 保持方向，缓慢减速
- 打开双闪灯，靠边停于平坦路面
- 拉手刹，放置三角警示牌（车后 50~100 米）
- 使用随车千斤顶和备胎/补胎工具更换
### 2. 电瓶亏电无法启动
- 使用搭电线连接另一辆车电瓶
- **正极接正极，负极接负极**（或接车身搭铁）
- 启动救援车，再启动奕跑
### 3. 发动机过热
- 靠边停车，挂 N 挡或 P 挡
- 保持发动机怠速运转，不要立即熄火
- 待水温下降后检查冷却液，不足时补充
### 4. 拖车注意事项
- 自动挡：挂 N 挡，拖车速度 ≤ 50 km/h，距离 ≤ 50 公里
- 建议使用平板拖车
---
## 九、省油驾驶技巧
- 🚦 避免急加速、急刹车
- 🛣️ 保持 60~90 km/h 匀速行驶最省油
- ⛽ 使用 92 号及以上无铅汽油
- 🔄 及时换挡（手动挡：2000~2500 转升挡）
- 🧳 减少不必要负载，车顶行李架会增加油耗
- 🔧 定期保养，保持胎压正常
---
## 十、常见问题解答
**Q：钥匙没电了怎么开门？**
- 使用机械钥匙（隐藏在智能钥匙内部），插入驾驶门把手下方钥匙孔
**Q：如何重置保养提示？**
- 关闭发动机，按住仪表盘上的复位按钮（TRIP），同时启动发动机，继续按住直到提示消失
**Q：后排座椅如何放倒？**
- 拉动座椅靠背顶部的拉绳或扳手，向前放倒即可
**Q：如何连接手机蓝牙？**
- 中控屏 → 设置 → 蓝牙 → 搜索设备 → 手机选择 "Kia" 配对

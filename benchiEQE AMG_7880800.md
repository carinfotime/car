# 奔驰EQE AMG 技术参数介绍
  
更新时间：2026年07月17日12时41分22秒   

## 车型概览
奔驰EQE AMG（Mercedes-AMG EQE）是梅赛德斯-AMG基于EVA2纯电平台打造的高性能纯电动中大型轿车，融合了AMG的运动基因与EQ系列的电动科技。
---
## 动力系统
| 参数 | 数据 |
|------|------|
| **驱动形式** | 全轮驱动（4MATIC+） |
| **电机布局** | 前后双永磁同步电机 |
| **综合最大功率** | **505 kW（687 PS）** |
| **综合最大扭矩** | **1,000 N·m** |
| **0-100 km/h加速** | **3.5秒** |
| **最高时速** | 240 km/h（电子限速） |
> *注：部分市场提供AMG EQE 43 4MATIC版本，功率约350 kW（476 PS），扭矩858 N·m，0-100 km/h加速4.2秒。*

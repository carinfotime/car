# 劳斯莱斯幻影 驾驶注意事项
  
更新时间：2026年07月18日10时37分02秒   

## 一、启动与预热
- **冷启动预热**：启动后怠速运转 **2-3分钟**，待机油充分润滑涡轮增压器
- **检查仪表盘**：确认无故障灯亮起，特别注意冷却液温度、机油压力指示
- **发动机转速限制**：水温未达到正常工作温度前，转速勿超过 **3000 rpm**
## 二、行驶操作规范
### 变速箱使用
- 挂挡前务必踩死刹车
- 从 P 挡挂出或挂入时，车辆必须完全静止
- 行驶中严禁挂 N 挡滑行（会损坏变速箱）
- 建议使用 **Comfort（舒适）模式**作为日常驾驶首选
### 转向与底盘
- 转弯半径较大，狭窄路段需预留充足空间
- 空气悬挂系统在通过减速带时建议将车速降至 **15 km/h以下**
- 避免频繁原地打方向盘，减少轮胎和转向机磨损
## 三、制动系统注意事项
| 项目 | 要求 |
|------|------|
| **刹车盘磨合期** | 前500公里避免急刹车 |
| **涉水后处理** | 轻踩刹车数次以清除刹车盘水膜 |
| **长下坡路段** | 使用发动机辅助制动，避免刹车过热 |
## 四、特殊路况驾驶
### ❄️雨雪天气
- ESP/车身稳定系统必须保持开启状态
- V12发动机扭矩极大，起步时轻踩油门防止打滑
- 建议使用冬季轮胎或雪地模式（如配备）
### 🅿️停车规范
1. **熄火前怠速1分钟**——让涡轮散热（尤其长途行驶后）
2. **先拉电子手刹 → 再挂 P挡**
3. **方向盘回正**——保护转向助力系统及轮胎均匀磨损
## ⚠️五、重要禁忌事项
> ❌ **禁止行为清单**
> - ⛽️严禁加注低于95号汽油（推荐98号）
> - 🚫切勿用自动洗车机（可能损伤漆面及轮毂）
> - 🚗不要牵引其他车辆（变速箱可能损坏）
> - 💧涉水深度不得超过 **30厘米**
> - 🔧非授权维修店不得进行任何电气系统改装
## 📋六、日常检查要点
1. **每周检查一次胎压**（标准值见驾驶座门框标签）
2. **每月检查冷却液和玻璃水液位**
3. **每半年检查一次电瓶状态**
4. *注意*：幻影自重约2.6吨，轮胎磨损速度约为普通轿车的1.5倍

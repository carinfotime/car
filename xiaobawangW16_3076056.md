## 二、定期保养周期表
### ✅ A类保养（每5000公里或6个月，以先到为准）
- [ ] 更换机油及机油滤清器
- [ ] 检查冷却液液位及浓度
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查雨刮水液位并补充
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查灯光系统（大灯、转向灯、刹车灯等）
- [ ] 清洁空气滤清器滤芯（吹尘）
- [ ] 底盘及悬挂系统目视检查
### ✅ B类保养（每10000公里或12个月，以先到为准）
包含A类全部项目，另加：
- [ ] **更换空气滤清器**
- [ ] **更换空调滤清器**
- [ ] **更换燃油滤清器**
- [ ] **检查火花塞状况**（建议每2万公里更换）
- [ ] **检查并调整皮带张紧度**
- [ ] **清洗节气门**（视积碳情况）
- [ ] **四轮换位**（交叉换位）
- [ ] **检测蓄电池电压及接线柱清洁**
### ✅ C类保养（每40000公里或24个月，以先到为准）
包含B类全部项目，另加：
- [x] **更换火花塞**
- [x] **更换变速箱油**（手动挡：2年/4万公里；CVT：4万公里）
- [x] **更换制动液**（DOT4标准）
- [x] **更换冷却液**
- [x] **清洗喷油嘴**或燃油系统积碳清洗
- [x] **检查并更换正时皮带/链条**（如为正时皮带需严格按里程更换）
### ✅ D类保养（每80000公里或48个月）
包含C类全部项目，另加：
- ⚠️ **更换正时皮带套装**（含张紧轮、惰轮）
- ⚠️ **更换水泵**（建议与正时皮带一同更换）
- ⚠️ **深度清洗发动机积碳**
- ⚠️ **检查三元催化器及氧传感器**
- ⚠️ **全车油液彻底更换**

## 三、关键部件保养建议
### 🛢️机油选择
| 季节/环境 | SAE粘度等级 | API质量等级 |
|-----------|-------------|-------------|
| -30°C ~ -10°C | SAE:0W‑30,0W‑40,5W‑30,5W‑40 | SL/SM/SN |
| -20°C ~ +40°C | SAE:10W‑30,10W‑40,15W‑40 | SL/SM/SN |
> 💡 *推荐使用 SN级及以上全合成机油，提升发动机保护性能。*
### 🔧变速箱油
|变速箱类型|推荐规格|
|---|---|
|5速手动挡|GL‑4 / GL‑5,75W90|
|CVT无级变速器|专用CVT油(NS‑2/NS‑3级别)|
### ❄️冷却液
使用乙二醇型长效防冻冷却液，冰点低于当地最低气温10℃以上。

## 四、日常自检清单
建议每周或每次加油前快速自查：
1. 🔍 **机油尺** —冷车状态下拔出擦净再插入后查看刻度
2. 🔍 **冷却液储液罐** —在MAX与MIN之间
3. 🔍 **制动液储液罐** —不低于MIN线
4. 🔍 **轮胎外观与气压** —标准胎压：前轮2.3bar /后轮2.5bar(满载可适当增加)
5. 🔍 **各灯光是否正常亮起**
6. 🔍 **雨刮片刮拭效果**

## ⚠️六、特别注意事项
1. 🚫 *切勿混用不同品牌/型号的防冻冷却液*
2. 🚫 *涡轮增压车型必须使用全合成机油*
3. 🚫 *严禁用水代替冷却液*
4. 🚫 *CVT变速箱必须使用专用油品，不可用ATF代替*
5. ✅ *每次保养后记录里程和日期，保留维修单据*
6. ✅ *冬季启动后怠速热车1~2分钟再行驶*
7. ✅ *长途出行前务必进行全车安全检查*

# 红旗H9 动力参数介绍
  
更新时间：2026年07月14日14时02分41秒   

## 动力总成概述
红旗H9提供多种动力组合，涵盖2.0T涡轮增压发动机+48V轻混系统以及3.0T V6机械增压发动机，匹配7速湿式双离合变速箱。
---
## 一、2.0T 涡轮增压发动机 + 48V轻混系统
| 参数项 | 数值 |
|--------|------|
| **发动机类型** | 2.0T 直列四缸涡轮增压 |
| **排量** | 1989 mL |
| **最大功率** | **185 kW（252 Ps）** |
| **最大功率转速** | 5500 rpm |
| **最大扭矩** | **380 N·m** |
| **最大扭矩转速** | 1800 - 4000 rpm |
| **燃油标号** | 95号及以上无铅汽油 |
| **供油方式** | 缸内直喷 |
| **排放标准** | 国VI b |
| **变速箱** | 7速湿式双离合变速箱（DCT） |
| **驱动方式** | 前置后驱 |
### 🔋48V轻混系统作用
- **起步辅助：** BSG电机辅助启动，提升平顺性
- **加速助力：** 短暂提供额外扭矩，改善响应
- **能量回收：** 制动时回收能量，降低油耗
- **启停优化：** 减少启停抖动，提升舒适性
### ✅性能表现
- **0-100 km/h加速时间：** ≈7.8秒
- **最高车速：** ≥230 km/h
- **WLTC综合油耗：** ≈8.5 L/100km

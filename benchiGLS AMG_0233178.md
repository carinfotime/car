# 奔驰GLS AMG 内饰参数介绍
  
更新时间：2026年07月11日15时20分13秒   

## 整体设计风格
奔驰GLS AMG的内饰融合了豪华、运动与科技感，采用顶级材质与精湛工艺，营造出极具战斗氛围的座舱环境。
## 座椅系统
| 项目 | 参数 |
|------|------|
| **座椅材质** | Nappa真皮（AMG专属） |
| **前排座椅** | AMG高性能运动座椅 |
| **座椅功能** | 多轮廓电动调节、记忆功能、加热、通风、按摩 |
| **后排座椅** | 电动可折叠第三排（7座布局） |
| **第二排** | 独立行政座椅（可选）或三座连排 |
| **头枕** | AMG徽标刺绣头枕 |
## 仪表盘与中控
- **双12.3英寸高清显示屏**（全液晶仪表盘 + 中控触控屏）
- **MBUX智能人机交互系统**（支持语音控制、触控、手势操作）
- **AMG专属显示风格**：运动化仪表界面，含赛道计时器、G值显示
- **方向盘**：AMG高性能方向盘，Nappa真皮包裹，带Dinamica防滑材质
- **方向盘功能**：触控按键、换挡拨片、AMG Drive Unit操控单元（驾驶模式选择）
## 内饰装饰
- **碳纤维饰板**（标配或选装）
- **铝合金/钢琴烤漆饰板**（可选）
- **AMG发光门槛条**
- **64色环境氛围灯**
- **全景滑动天窗**
## 空调系统
- THERMOTRONIC多区智能空调（5区独立温控）
- ENERGIZING舒享套装（空气净化、香氛系统）
## 音响系统
| 配置 | 说明 |
|------|------|
| Burmester®环绕立体声音响 | 13扬声器，590W功率 |
| Burmester®高端3D环绕音响（选装） | 26扬声器，1610W功率 |
## AMG专属配置
- AMG Track Pace赛道数据记录系统
- AMG Performance排气系统控制按键
- AMG专属脚垫与迎宾踏板
- AMG红色安全带（可选）
## 空间尺寸
| 项目 | 数据 |
|------|------|
| **轴距** | 3135 mm |
| **后备箱容积** | ~355 L（第三排展开）/ ~2400 L（全部放倒） |

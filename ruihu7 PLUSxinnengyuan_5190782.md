# 瑞虎7 PLUS新能源 内饰参数介绍

## 一、内饰设计风格
| 项目 | 参数 |
|------|------|
| 内饰主题 | 环抱式座舱设计 |
| 配色方案 | 黑色/棕黑双拼/蓝白双拼 |
| 装饰材质 | 软质搪塑工艺 + 皮质包覆 + 金属拉丝饰板 |
| 缝线工艺 | 双缝线设计（撞色缝线） |

| 项目 | 参数 |
|------|------|
| 座椅材质 | 仿皮（高级皮革） |
| 前排座椅功能 | 电动调节（主驾6向/副驾4向） |
| 前排座椅加热 | 支持（部分配置） |
| 前排座椅通风 | 支持（顶配） |
| 后排座椅 | 4/6比例放倒 |
| 后排中央扶手 | 带杯架 |
| 座椅记忆功能 | 主驾座椅记忆（顶配） |
## 三、中控与仪表盘
| 项目 | 参数 |
|------|------|
| 仪表盘 | 12.3英寸全液晶仪表 |
| 中控屏 | 12.3英寸触控液晶屏 |
| 屏幕布局 | 双联屏一体式设计 |
| 中控屏分辨率 | 1920×720 |
| 车机系统 | Lion 5.0雄狮智云系统 |
| 芯片 | 高通骁龙8155 |
| 语音控制 | 支持（多媒体/导航/电话/空调/天窗） |
| OTA升级 | 支持整车OTA远程升级 |
## 四、智能互联
| 项目 | 参数 |
|------|------|
| 手机互联 | Apple CarPlay（无线）/ 华为HiCar |
| 导航系统 | 高德在线导航 |
| 蓝牙 | 蓝牙5.0 |
| WiFi热点 | 支持 |
| 远程控制 | 奇瑞智云APP远程控制（门锁/空调/充电/寻车） |
| 车载KTV | 支持 |

| 项目 | 参数 |
|------|------|

| 后排空调出风口 | 标配 |
| PM2.5过滤 | 标配 |
| AQS空气质量管理系统 | 支持（部分配置） |
| 负离子空气净化 | 支持（部分配置） |

| 项目 | 参数 |
|------|------|
| 方向盘材质 | 真皮包裹 |
| 方向盘调节 | 手动上下+前后调节 |
| 多功能按键 | 标配（多媒体/巡航/语音/仪表控制） |
| 换挡拨片 | 支持 |
## 七、音响系统
| 项目 | 参数 |
|------|------|
| 扬声器数量 | 6扬声器（标配）/ 8扬声器SONY（顶配） |
| 品牌音响 | SONY（顶配） |

| 阅读灯 | 前后排LED阅读灯 |

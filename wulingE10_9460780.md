# 五菱E10 保养指南
  
更新时间：2026年07月20日13时15分16秒   

## 一、概述
五菱E10是一款纯电动微型货车，主要用于城市末端物流配送。与传统燃油车不同，其保养重点在于**三电系统（电池、电机、电控）**、底盘及车身常规检查。正确的保养不仅能延长车辆使用寿命，更能确保行车安全。
---

| **冷却液** | 每 **24个月/40,000公里** | 检查液位及冰点 |
| **制动液** | 每 **12个月/20,000公里** | 检查含水量 |
| **空调滤芯** | 每 **6个月/10,000公里** | 视使用环境缩短 |
| **轮胎检查与换位** | 每 **6个月/10,000公里** | - |
| **制动系统检查** | 每 **12个月/20,000公里** | - |
| **底盘及悬架检查** | 每 **12个月/20,000公里** | - |
> *注：以上为官方建议标准周期，若车辆使用环境恶劣（如多尘、高温、频繁重载），建议适当缩短保养间隔。*

## 三、核心保养项目详解
### （一）三电系统（核心）
1.  **动力电池**
    -   **外观检查：** 检查电池包外壳有无磕碰、变形或漏液。
    -   **连接线束：** 检查高压线束插头是否松动、氧化或磨损。
    -   **SOC校准：** 建议每月进行一次满充满放（从低电量充至100%），以校准电池管理系统（BMS）。
    -   *注意：严禁私自拆卸电池包，维修需由授权服务站进行。*
2.  **驱动电机**
    -   **清洁散热：** E10电机通常采用风冷或液冷，定期清理电机表面灰尘及杂物，确保散热良好。
    -   **异响检测：**启动及行驶中留意电机有无异常啸叫或摩擦声。
3.  **电控系统（VCU/MCU）**
    -   OBD诊断接口读取故障码，确认无历史或当前故障。
    -   确保高压继电器工作正常。
### （二）减速器（相当于燃油车的变速箱）
-   E10采用单级减速器，需定期更换专用齿轮油。
-   *首保必须更换*，后续按周期执行。
-   更换标准：放出旧油后加注至观察孔溢出为止。
### （三）冷却系统
-   E10的电机和电控需要冷却液散热。
-   *检查方法*：观察膨胀水箱液位应在“MAX”与“MIN”之间。
-   *更换标准*：使用冰点仪检测冷却液冰点是否达标；颜色变浑浊或有异味需立即更换。
### （四）制动系统
1.  **制动液**
    -   使用测试笔检测含水量。含水量超过3%必须更换；超过2%建议更换。
2.  **刹车片与刹车盘**
    -   前盘后鼓式制动。前刹车片厚度低于3mm时需更换；后刹车鼓磨损超过极限需更换蹄片。
3.  ***驻车制动***：
    -    定期调整驻车拉线松紧度，确保坡道驻车有效。
### （五）轮胎
-   E10前轮载荷较大（驾驶室位于前方），注意前轮磨损通常快于后轮。
-   *胎压标准*：
    -   前轮：约350 kPa (3.5 bar)
    -   后轮：约450 kPa (4.5 bar) *(具体以驾驶室门框标签为准)*
-   *换位方式*：前后交叉换位。
### （六）空调系统
1.  定期更换空调滤芯（位于手套箱后方）。
2.  每年夏季前检查制冷剂压力及冷凝器散热片是否堵塞。

## ⚠️四、日常自查清单（建议每日出车前执行）
1. ✅ 【灯光】大灯、转向灯、刹车灯、雾灯是否正常？
2. ✅ 【轮胎】胎压目测是否亏气？胎面有无异物或鼓包？
3. ✅ 【车身】货箱门锁扣是否锁紧？货箱有无异常晃动？
4. ✅ 【仪表】仪表盘有无故障灯亮起？（如电池报警、电机过热等）
5. ✅ 【制动】踩下刹车踏板感受行程是否正常？有无变软或过硬？
6. ✅ 【雨刮】玻璃水是否充足？雨刮片能否刮净？

## 📋六、进店保养流程参考
1. ***预约*** ：联系当地五菱授权服务站进行预约。
2. ***接车*** ：技师进行外观预检并记录里程数。
3. ***举升作业***：
    1) OBD电脑读取全车故障码；
    2)拆卸底盘护板；
    3)放掉旧齿轮油；
    4)目视检查底盘各部件紧固情况及胶套老化程度；
    5)紧固底盘螺栓力矩；
    6)加注新齿轮油；
    7)安装护板落车；
4.***常规项***：
    1)更换制动液并排气；
    2)检测并补充冷却液；
    3)拆装轮胎进行换位并做动平衡；
    4)清洁或更换空调滤芯；
5.***路试*** ：由技师进行短距离路试确认无异响无故障码后方可交车。

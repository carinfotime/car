# SS DOLPHIN 内饰参数介绍
  
更新时间：2026年07月14日12时29分10秒   

## 一、整体设计风格
- **设计理念**：科技运动风，兼顾豪华感与驾控氛围。
- **内饰色调**：提供纯黑、黑红双拼、黑蓝双拼等多种配色方案。
- **材质工艺**：大面积软性搪塑材质包覆，搭配金属拉丝/碳纤维纹路饰板。
## 二、座椅系统
| 项目 | 参数 / 说明 |
| :--- | :--- |
| **座椅材质** | 高仿皮 / Nappa 真皮（高配） |
| **前排座椅功能** | 电动 6 向调节、座椅加热、座椅通风（高配） |
| **后排座椅** | 4/6 比例放倒，中央扶手带杯架 |
| **头枕** | 前后排高度可调，后排中央独立头枕 |
| **座椅布局** | 2+3 五座布局 |
## 三、仪表盘与中控
| 项目 | 参数 / 说明 |
| :--- | :--- |
| **仪表盘** | 7 英寸 / 12.3 英寸（高配）全液晶仪表盘 |
| **中控屏幕** | 10.25 英寸 / 12.8 英寸（高配）悬浮式触控屏 |
| **车机系统** | 智能网联系统，支持 OTA 远程升级 |
| **语音控制** | 多媒体、导航、电话、空调 |
| **手机互联** | 支持 Apple CarPlay、百度 CarLife 手机映射 |
## 四、方向盘
- **类型**：D 型三辐式多功能方向盘（平底设计，运动风格）
- **材质**：真皮包裹
- **调节方式**：手动上下 + 前后调节
- **功能按键**：集成定速巡航、多媒体控制、语音唤醒、仪表盘切换
## 五、空调与空气净化
| 项目 | 参数 / 说明 |
| :--- | :--- |
| **空调类型** | 自动恒温空调 |
| **分区控制** | 双区独立控制（主 / 副驾驶） |
| **后排出风口** | 标配（带风量调节） |
| **PM2.5 过滤** | 标配 PM2.5 过滤装置 |
| **负离子发生器** | 高配车型配备 |
## 六、内饰灯光
- **氛围灯**：多色可调（64 色 / 128 色，视配置而定）
- **阅读灯**：前排 / 后排独立 LED 阅读灯
- **迎宾灯**：车门底部迎宾灯（高配）
## 七、储物空间
| 位置 | 说明 |
| :--- | :--- |
| **中央扶手箱** | 双层开启，深度约 15cm，带 USB 接口 |
| **手套箱** | 带阻尼，内部植绒 |
| **门板储物格** | 前后门均可放置 2 瓶 550ml 水瓶 |
| **杯架** | 前排 2 个，后排中央扶手 2 个 |
| **其他** | 中控台下储物槽、眼镜盒、卡片槽 |
## 八、科技与安全配置
| 项目 | 参数 / 说明 |
| :--- | :--- |
| **仪表显示** | 行车电脑、胎压监测、驾驶模式切换动画 |
| **HUD 抬头显示** | 高配可选装 |
| **行车记录仪** | 原厂隐藏式（高配） |
| **手机无线充电** | 前排 15W 快充（高配） |
| **USB 接口** | 前排 2 个（Type-A + Type-C），后排 1 个（Type-A） |
| **音响系统** | 6 扬声器 / 8 扬声器（高配品牌音响） |
## 九、尺寸参数（内部空间）

| :--- | :--- |
| **轴距** | 2700 mm |
| **前排头部空间** | 约 980 mm |
| **后排头部空间** | 约 950 mm |
| **后排腿部空间** | 约 720 mm（标准坐姿） |
| **后备箱容积** | 450 L（后排座椅放倒后可扩展至 1200 L） |

- **遮阳板**：带化妆镜 + 照明灯（主副驾）
- **地毯**：原厂环保脚垫（随车附送）

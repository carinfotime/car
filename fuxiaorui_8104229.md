# 福小瑞 内饰参数介绍
## 一、 整体设计风格
-   **设计主题**：都市胶囊 · 极简美学
-   **配色方案**：
    -   **云朵白**：主打色，温馨居家感。
    -   **燕麦杏**：低饱和度高级感，耐脏实用。
    -   **星际灰**：运动科技风，搭配荧光绿缝线。
-   **环保理念**：座舱内大面积采用**环保可再生材料**与**婴儿级亲肤织物**，新车无异味。
## 二、 座椅系统
-   **座椅布局**：2+2 四座布局（标准版）/ 2+3 五座布局（家庭版）
-   **座椅材质**：
    -   标配：环保织物座椅（透气、易清洁涂层）
    -   高配：仿皮 + 织物混搭座椅（打孔工艺）
-   **前排座椅功能**：
    -   主驾：6向手动调节（前后/靠背/高低）
    -   副驾：4向手动调节（前后/靠背）
    -   **加热功能**：前排座椅可选装（两档调节）
-   **后排座椅**：
    -   后排整体放倒（放倒后容积可扩展至 800L）
    -   后排ISOFIX儿童座椅接口（2个）
## 三、 仪表台与中控
-   **仪表盘**：
    -   类型：7英寸全液晶仪表盘
    -   显示内容：时速、电量、续航里程、驾驶模式、胎压监测
-   **中控屏**：
    -   尺寸：10.25英寸悬浮式高清触控屏
    -   分辨率：1920 x 720
    -   系统：福小瑞智能互联系统
-   **物理按键**：保留空调控制区实体按键（旋钮 + 按键），方便盲操作
## 四、 智能座舱与互联
-   **语音控制**：支持免唤醒语音交互，可控制空调、导航、音乐、车窗
-   **手机互联**：支持 CarPlay / HiCar / 原厂手机映射
-   **远程控制**：通过手机App实现远程锁车、开关空调、查看车辆状态
-   **OTA升级**：支持整车娱乐系统在线升级
## 五、 空调系统
-   **类型**：手动空调 / 自动空调（高配）
-   **过滤系统**：PM2.5 空气过滤系统 + 负离子发生器
-   **后排出风口**：标配（位于前排座椅下方）
## 六、 方向盘
-   **类型**：三幅式平底多功能方向盘
-   **材质**：超纤皮革包裹（高配）/ 发泡材质（标配）
-   **功能按键**：音量控制、语音唤醒、仪表菜单切换、定速巡航控制
-   **调节方式**：上下两向手动调节
## 七、 储物空间
-   **门板储物**：前后门板均可放置 2 瓶 500ml 水瓶
-   **中央通道**：手机无线充电面板 + 2个杯架 + 扶手箱（深度约15cm）
-   **手套箱**：带阻尼，容积约 8L
-   **其他**：副仪表台下方的镂空储物槽（可放纸巾/包包）
## 八、 灯光与氛围
-   **阅读灯**：前后排独立LED阅读灯
-   **氛围灯**：64色环绕式氛围灯（高配，支持呼吸模式与音乐律动）
-   **化妆镜**：主副驾遮阳板均带化妆镜（副驾带照明灯）
## 九、 音响系统
-   **扬声器数量**：4扬声器（标准版）/ 6扬声器（高配版）
-   **音效调校**：福小瑞定制音效（支持流行/古典/人声模式切换）
## 十、 其他实用配置
-   **充电接口**：前排 1 个 USB-A + 1 个 USB-C（支持快充），后排 1 个 USB-A
-   **手机无线充电**：前排专属区域（15W 快充）
-   **遮阳帘**：高配车型配备后排隐私玻璃 + 手动遮阳帘
-   **内后视镜**：手动防眩目

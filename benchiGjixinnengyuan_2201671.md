# 奔驰G级新能源（G 580 / G 63 e等车型）操作指南
  
更新时间：2026年07月16日12时50分22秒   

## 一、启动与熄火
### 启动车辆
1.  **携带钥匙**：确保智能钥匙在车内。
2.  **系好安全带**：这是启动车辆的前提条件之一。
3.  **踩下制动踏板**：用力踩下刹车。
4.  **按下启动按钮**：位于方向盘右侧的“START/STOP”按钮。
    -   仪表盘亮起，READY指示灯亮起，表示车辆已进入可行驶状态（电机已启动，发动机待命）。
### 熄火车辆
1.  **挂入驻车档（P）**：按下档杆上的“P”按钮。
2.  **拉起电子手刹**：通常自动拉起，也可手动操作中控台上的手刹开关。
3.  **按下启动按钮**：关闭电源，车辆下电。
> **注意**：如果仅想关闭发动机但保持车内电器供电，可在挂P档后，不踩刹车，短按启动按钮。

## 二、驾驶模式选择（Dynamic Select）
通过中控台或方向盘上的“DYNAMIC SELECT”旋钮/按键切换模式。G级新能源通常提供以下模式：
| 模式 | 图标 | 适用场景 | 特点 |
| :--- | :--- | :--- | :--- |
| **Electric** | 电池图标 | 城市通勤、短途 | 纯电驱动，零排放，安静 |
| **Comfort** | 旗帜 | 日常驾驶、长途 | 平衡动力与舒适，优先用电，适时启动发动机 |
| **Sport** | 方格旗 | 高速、超车 | 动力响应最灵敏，发动机与电机共同出力 |
| **Individual** | 齿轮+人 | 自定义 | 可单独设定动力、转向、悬架（如有） |
| **Offroad** | 山地 | 非铺装路面 | 激活低速四驱，优化悬挂和转向 |
| **Rock** | 岩石 | 攀爬、极陡坡 | 最高离地间隙，最强的扭矩输出和锁定功能 |
> 在 **Electric** 模式下，可通过中控屏选择“保持电池电量”或“电量回收优先”来管理能耗。
---
## 三、越野功能操作
G级新能源保留了强大的越野能力，操作前请确保车辆处于 **Offroad** 或 **Rock** 模式。
### 1. 三把差速锁（如果配备）
-   **操作顺序**：必须按 **1 → 2 → 3** 的顺序依次锁止。
    1.  **中差锁（1）**：增加前后轴驱动力分配稳定性。
    2.  **后差锁（2）**：锁止后轮，提升脱困能力。
    3.  **前差锁（3）**：锁止前轮，极限攀爬时使用。
-   **解锁顺序**：反向操作 **3 → 2 → 1**。
-   **指示灯**：按键上的橙色指示灯常亮表示锁止成功，闪烁表示正在锁止/解锁。
### 2. 低速四驱（Low Range）
-   **激活**：在Offroad模式下，挂N档，按下中央通道上的“LOW RANGE”按钮。
-   **退出**：挂N档，再次按下“LOW RANGE”按钮。车辆行驶速度超过设定值时会自动退出。
### 3. 透明引擎盖（Transparent Bonnet）
-   通过中控屏摄像头画面，在越野时显示车头下方及前轮位置的虚拟影像，帮助避开障碍物。
-   **开启**：挂入倒挡或按下360°全景影像按钮，在中控屏上选择“透明引擎盖”视图。
### 4. G-ROAR（模拟声浪）
-   在纯电模式下，系统可通过扬声器发出类似V8发动机的浑厚声浪。
-   **开启**：进入“车辆设置” -> “驾驶模式” -> “G-ROAR”，可选择不同音效风格。

## 四、能量回收与充电
### 1. 能量回收拨片（方向盘后方）
-   **增加回收强度**：拉动左侧“-”拨片，车辆减速感增强，回收更多电能。
-   **减小回收强度**：拉动右侧“+”拨片，车辆滑行更顺畅，类似燃油车空挡滑行。
-   **自适应回收（DAUTO）**：长按“-”拨片，车辆根据雷达探测的前车距离自动调节回收强度。
### 2. 充电方式
-   **交流慢充（AC）**：使用家用充电桩或公共交流桩。通常位于车身左后侧（传统加油口位置）。
-   **直流快充（DC）**：使用公共快充桩。通常位于车身右后侧或左后侧（视具体型号而定）。快充功率可达200kW左右（具体视电池版本）。
-   **充电进度**：通过仪表盘、中控屏或MBUX手机App查看。

## 五、常用功能操作
### 1. 空调与座椅
-   **预通风/预加热**：通过MBUX App远程开启空调或座椅加热/通风。
-   **气候预控制**：在充电时，设定出发时间，车辆会提前调节座舱温度，利用电网电能而非电池。
### 2. 车外灯光
-   **几何多光束LED大灯**：自动遮蔽对向车辆，远光常亮不晃眼。
-   **自适应远光灯辅助**：将灯光旋钮置于“Auto”档，向前推动转向灯拨杆即可激活。
### 3. 驻车与辅助
-   **主动式泊车辅助**：低于35km/h时，系统自动搜索车位。挂入倒挡后，点击中控屏“开始”即可自动泊入。
-   **360°全景影像**：挂倒挡自动激活，也可手动按下中控台摄像头图标。

## 六、仪表盘与信息显示
-   **左侧仪表盘**：默认显示功率表（电动）或转速表（混动）。可通过方向盘左侧触控按键切换显示内容（如能量流、导航、辅助驾驶信息）。
-   **右侧仪表盘**：默认显示速度、续航里程、驾驶模式等。
-   **能量流显示**：清晰展示当前是电机驱动、发动机驱动、还是两者共同驱动，以及是否在回收能量。
---
## 七、重要提示与安全
1.  **电池安全**：避免长时间浸泡在水中。越野涉水深度请参考车辆手册（通常不超过特定水深，新能源车型需注意电池组密封性）。
2.  **拖车**：必须使用平板拖车，严禁前轮或后轮单独拖行，否则可能损坏电机或电池。
3.  **紧急断电**：如果发生碰撞，车辆会自动切断高压电。如遇紧急情况，可手动拉动前机舱盖内的红色高压断电开关（位置参考手册）。
4.  **保养**：高压电池冷却液、电机油等需要定期由授权经销商检查更换。

## 八、常见问题（FAQ）
| 问题 | 解答 |
| :--- | :--- |
| **纯电续航显示不准？** | 续航受驾驶风格、温度、地形影响。系统会根据最近50km的能耗动态计算。 |
| **为什么发动机频繁启动？** | 可能原因：电池电量低、需要暖机/热车、开启暖风空调、选择了Sport模式。 |
| **如何最大化纯电续航？** | 使用Electric模式，开启DAUTO能量回收，平稳驾驶，关闭不必要的电器。 |
| **越野时电池会没电吗？** | 系统会保留一部分电量用于越野，且发动机会在必要时启动为电池充电。 |

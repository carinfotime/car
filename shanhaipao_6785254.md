# 山海炮 保养指南
  
更新时间：2026年07月20日18时36分04秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 机油 & 机滤 | 每 **5,000公里** 或 **6个月**（先到为准） |
| 空气滤清器 | 每 **10,000公里** 检查，**20,000公里**更换 |
| 空调滤清器 | 每 **10,000公里** 或 **1年** |
| 燃油滤清器 | 每 **20,000公里** |
| 火花塞 | 每 **30,000–40,000公里** |
| 变速箱油 | 每 **60,000公里** |
| 刹车油 | 每 **40,000公里** 或 **2年** |
| 防冻液 | 每 **40,000公里** 或 **2年** |
| 转向助力油 | 每 **60,000公里** |
| 前后桥齿轮油（四驱） | 每 **40,000–60,000公里** |
|分动箱油（四驱） |  每 **60,000公里**|
|正时皮带/链条 |  链条免维护，皮带视情况约 **80,000–100,000公里检查更换**
|轮胎换位 &  动平衡 |  每 **10,000公里**
|刹车片检查 |  每 **10,000公里**
|
> ⚠️ *以上为厂家推荐常规周期，恶劣路况（越野、重载、沙尘、高温等）应适当缩短保养间隔。*

##  二、日常检查项目（建议每周/每次出车前）
- [ ]  发动机机油液位（冷车状态下检查）
- [ ]  冷却液液位
- [ ]  刹车油液位
- [ ]  玻璃清洗液
- [ ]  轮胎胎压及外观（含备胎）
- [ ]  灯光系统（大灯、转向灯、刹车灯、雾灯）
- [ ]   仪表盘故障指示灯
- [ ]   雨刮器及喷水功能

##  三、各阶段详细保养内容
### 🔹 A类保养（5,000km /半年）
1. ✅更换发动机机油及机油滤清器
2. ✅检查并补充各油液（冷却液、刹车油、转向助力油）
3. ✅检查轮胎胎压及磨损情况
4. ✅检查底盘及悬挂系统螺栓紧固情况
5. ✅检查灯光及电瓶状态
6. ✅清洁或更换空调滤芯（视情况）
### 🔹 B类保养（20,000km /1年）
在A类基础上增加：
1. ✅更换空气滤清器
2. ✅更换燃油滤清器
3. ✅轮胎换位 +动平衡检测
4. ✅检查刹车片厚度
5. ✅检查传动轴及万向节润滑情况
6. ✅四驱系统功能检测
### 🔹 C类保养（40,000km /2年）
在B类基础上增加：
1. ✅更换火花塞
2. ✅更换刹车油
3. ✅更换防冻液
4. ✅清洗节气门及喷油嘴（建议）
5. ✅前后桥齿轮油更换（四驱车型）
6. ✅全车皮带张紧度检查
### 🔹 D类保养（60,000km /3年）
在C类基础上增加：
1. ✅更换自动变速箱油 +滤网/垫片
2. ✅分动箱油更换（四驱车型）
3. ✅转向助力油更换
4. ⚠️正时链条/皮带全面检测
5. ⚠️全车橡胶件老化检查（机脚胶、衬套等）

##  四、关键部件注意事项
### 🛢️机油选择
- API等级：不低于 SP级  
- SAE黏度：  
    -   常规使用：5W-30  
    -   高温/重载：5W-40  
    -   寒冷地区：0W-30  
- ACEA标准：C3/C2  
> *建议使用全合成机油以获得更好的保护效果。*
### ❄️冷却液
-    使用长城专用长效有机酸型(OAT)冷却液  
-    严禁混用不同颜色/成分的冷却液  
-    冰点要求：根据当地最低气温选择 (-25°C ~ -45°C)
### 🛞轮胎标准胎压参考
> *实际值请以驾驶侧门框标签为准*
|场景          |前轮        |后轮        |
|---|---|---|
日常行驶      |240 kPa (35 psi)|260 kPa (38 psi)|
满载/重载      |260 kPa (38 psi)|280 kPa (41 psi)|
轻度越野       |220–230 kPa (32–33 psi)|220–230 kPa|

##  五、越野后专项保养清单
山海炮作为皮卡，经常用于越野场景，每次越野后建议：
1️⃣冲洗底盘泥沙，防止腐蚀  
2️⃣检查底盘有无磕碰变形  
3️⃣清理空气滤芯/进气口沙尘  
4️⃣检查传动轴是否缠绕异物  
5️⃣差速锁功能测试  
6️⃣涉水后需重点检查：
   -     前后桥齿轮箱是否进水乳化  
   -     变速箱通气孔是否堵塞  
   -     各电器插头是否进水氧化

## ⚠️七、重要提示
1️⃣质保期内务必在授权服务站进行保养，保留全部记录凭证。  
2️⃣山海炮柴油版与汽油版部分配件不通用，请确认发动机型号后再采购配件。  
3️⃣DPF再生提醒出现时不要熄火，保持中高速行驶直至再生完成。  
4️⃣长期短途行驶会导致积碳和GPF堵塞加重，建议定期跑高速拉转速清理。  
5️⃣改装悬挂或升高车身会影响四轮定位参数，改装后需重新做定位。

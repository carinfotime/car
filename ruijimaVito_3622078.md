# 睿吉馬 Vito 动力参数介绍
  
更新时间：2026年07月14日14时39分54秒   

## 发动机规格
| 项目 | 参数 |
|------|------|
| **发动机类型** | 2.0T 涡轮增压柴油发动机 |
| **排量** | 1,950 cc |
| **气缸数** | 4缸直列 |
| **进气形式** | 涡轮增压 + 中冷器 |
| **燃油系统** | 高压共轨直喷技术 |
## 动力输出
| 项目 | 参数 |
|------|------|
| **最大功率** | **140 kW（190 hp）** @ 3,800 rpm |
| **最大扭矩** | **440 N·m** @ 1,400 - 2,400 rpm |
## 变速箱与驱动
- **变速箱类型**：9G-TRONIC — **9速自动变速箱**
- **驱动方式**：前置后驱（FR）
## 性能表现
- **最高车速**：约 **195 km/h**
- **0–100 km/h加速时间**：约 **9.5秒**
## 底盘与操控
- **前悬挂系统**：麦弗逊式独立悬挂
- **后悬挂系统**：半拖曳臂式独立悬挂
- **制动系统（前/后）**：通风盘式 / 盘式
- **转向系统**：电动助力转向（EPS）
## WLTP综合油耗与排放
- **综合油耗（WLTP）**：约 **7.5 - 8.5 L/100 km**
- **CO₂排放量（WLTP）**：约 **197 - 223 g/km**
- **排放标准**：欧6d

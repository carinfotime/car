# 庆铃五十铃轻卡保养指南
  
更新时间：2026年07月11日01时43分02秒   

## 一、保养周期总览
| 保养项目 | 保养周期 |
|---------|---------|
| 发动机机油更换 | 每5,000公里或3个月 |
| 机油滤清器更换 | 每5,000公里（与机油同步） |
| 燃油滤清器更换 | 每10,000公里 |
| 空气滤清器清洁/更换 | 每5,000公里清洁，每20,000公里更换 |
| 变速箱油更换 | 每40,000公里 |
| 后桥齿轮油更换 | 每40,000公里 |
| 冷却液更换 | 每2年或40,000公里 |
| 制动液更换 | 每2年或40,000公里 |
| 转向助力油检查/更换 | 每20,000公里检查，视需要更换 |
| 制动片检查 | 每10,000公里 |
| 轮胎换位 | 每10,000公里 |
| 蓄电池检查 | 每5,000公里 |
| 空调滤清器更换 | 每10,000公里或每年 |
| 传动轴润滑 | 每10,000公里 |
| 正时皮带检查/更换 | 每100,000公里（视车型而定） |
---
## 二、发动机系统保养
### 1. 机油与机滤
- **推荐机油型号**：15W-40 CD/CF级或以上柴油机专用机油（根据当地气温可选用10W-30）
- **机油容量**：约5.6~7.0升（视具体机型而定）
- **操作步骤**：
  1. 车辆停放在水平地面，发动机预热后熄火
  2. 拧开油底壳放油螺栓，放出旧机油
  3. 拆下旧机油滤清器，清洁安装座
  4. 在新机滤密封圈上涂抹少许机油，用手拧紧（不要用工具过紧）
  5. 装回放油螺栓并拧紧
  6. 加注新机油至油标尺上下刻度之间
  7. 启动发动机运转1~2分钟，熄火后检查油位
### 2. 燃油系统
- **柴油滤清器**：每10,000公里更换，更换后需手动泵油排气
- **排气操作**：松开滤清器上的放气螺钉，按压手油泵直到排出空气且无气泡，拧紧放气螺钉
- **燃油品质要求**：使用国标0号或-10号柴油（根据气温选择）
### 3. 空气滤清器
- 每5,000公里取出滤芯，用压缩空气从内向外吹净
- 每20,000公里更换新滤芯
- 注意：严禁用油或水清洗纸质滤芯
---
## 三、底盘与传动系统保养
### 1. 变速箱
- **润滑油型号**：GL-4 75W-90 或 80W-90
- **检查方法**：拧开加油口螺塞，油面应与加油口下沿齐平
- **更换周期**：每40,000公里
### 2. 后桥
- **润滑油型号**：GL-5 80W-90 或 85W-90
- **更换周期**：每40,000公里
- **注意**：庆铃后桥有差速器，必须使用GL-5级别齿轮油
### 3. 传动轴
- 每10,000公里向十字节加注锂基润滑脂
- 检查传动轴螺栓是否松动
- 检查十字节间隙是否过大
### 4. 轮胎
- **标准胎压**（冷胎）：
  - 前轮：4.5~5.0 kg/cm²
  - 后轮：5.5~6.5 kg/cm²（视载重调整）
- 每10,000公里进行轮胎换位（交叉换位法）
- 定期检查胎面磨损及异常磨损情况
---
## 四、制动系统保养
### 1. 制动片/蹄片
- 每10,000公里检查制动片厚度
- 前制动片厚度低于3mm、后制动蹄片厚度低于2mm时需更换
- 注意：庆铃五十铃轻卡前轮为盘式制动，后轮为鼓式制动
### 2. 制动液
- **型号**：DOT3 或 DOT4
- 每2年或40,000公里更换一次
- 制动液具有吸水性，不可混用不同类型
- 更换时注意排气，避免空气进入制动管路
### 3. 制动间隙调整
- 后轮鼓式制动需定期调整制动间隙
- 转动调整螺母至车轮转动稍有阻力为止，然后回退1/4圈
---
## 五、冷却系统保养
### 1. 冷却液
- **推荐型号**：长效乙二醇型防冻液（-25℃~-35℃规格）
- **更换周期**：每2年或40,000公里
- **容量**：约8~10升
- 更换步骤：
  1. 冷车状态下打开散热器放水阀和水箱盖
  2. 放出旧冷却液
  3. 用清水冲洗冷却系统
  4. 关闭放水阀，加注新冷却液至上限
  5. 启动发动机运转至风扇启动，补充至标准液位
### 2. 水泵与风扇
- 每次保养检查水泵是否有异响或渗漏
- 检查风扇皮带松紧度（压下皮带中间可压下10~15mm为正常）
---
## 六、电气系统保养
### 1. 蓄电池
- 检查电解液液位（需在上下刻度之间）
- 极柱清洁，涂抹凡士林防腐蚀
- 检查蓄电池固定是否牢固
- 注意：庆铃多为12V系统，部分车型为24V
### 2. 灯光与仪表
- 每次出车前检查全车灯光（大灯、转向灯、刹车灯、倒车灯）
- 检查仪表盘故障指示灯是否正常
### 3. 发电机与起动机
- 每20,000公里检查发电机皮带松紧度
- 检查起动机接线是否牢固
---
## 七、空调系统保养
- 每10,000公里或每年更换空调滤清器
- 检查制冷剂压力，制冷效果下降时及时检修
- 冬季也要定期开启空调运转10~15分钟，防止密封件老化
---
## 八、日常检查项目（出车前/收车后）
| 检查项目 | 检查内容 |
|---------|---------|
| 机油液位 | 冷车检查油标尺 |
| 冷却液液位 | 膨胀水箱液位在上下线之间 |
| 制动液液位 | 储液罐液位在上下线之间 |
| 轮胎外观 | 检查胎压、胎面磨损、有无鼓包 |
| 灯光系统 | 全车灯光功能正常 |
| 仪表报警灯 | 启动后自检，报警灯应熄灭 |
| 车身漏油 | 发动机、变速箱、后桥下方有无油迹 |
| 转向系统 | 方向盘自由行程正常，无异响 |
| 制动系统 | 制动踏板行程正常，无发软现象 |
| 喇叭、雨刮 | 功能正常 |
---
## 九、重要注意事项
1. **使用正品配件**：庆铃五十铃轻卡对配件要求较高，建议使用原厂或品牌配套滤清器、油品，劣质滤清器会严重损坏发动机。
2. **禁止超载**：庆铃轻卡标定载重需严格遵守，超载会加速底盘、轮胎、制动系统磨损，存在安全隐患。
3. **冷车启动**：柴油机冷车启动后应怠速运转3~5分钟，待水温上升后再起步行驶，切勿猛踩油门。
4. **预热装置**：冬季气温低于0℃时，启动前先使用预热塞预热（仪表盘预热指示灯熄灭后再启动）。
5. **磨合期**：新车前1,500公里为磨合期，避免高速行驶（不超过80km/h）和重载，建议在1,500~2,000公里时提前进行首保。
6. **保养记录**：每次保养后保留保养单据和更换配件记录，便于后续维修和二手车交易。
7. **DPF（颗粒捕捉器）车型**：如配备DPF的国五/国六车型，需注意再生指示灯提示，必要时进行主动再生或驻车再生操作，不可长期短途低速行驶。
8. **尿素系统（SCR）**：国五/国六车型需定期添加车用尿素溶液，确保尿素液位充足，使用正规品牌尿素。

## 十、常见故障与简易判断
| 故障现象 | 可能原因 | 处理方法 |
|---------|---------|---------|
| 启动困难 | 蓄电池亏电、预热塞故障、燃油系统进气 | 检查电瓶、预热系统、排气 |
| 发动机无力 | 空气滤清器堵塞、燃油滤清器堵塞、涡轮增压故障 | 更换滤芯、检修增压器 |
| 冒黑烟 | 喷油嘴雾化不良、供油提前角不对 | 清洗/更换喷油嘴，校对正时 |
| 水温过高 | 冷却液不足、风扇皮带松、节温器故障 | 补充冷却液、调整皮带、更换节温器 |
| 刹车偏软 | 制动液不足、制动系统进气、制动片磨损 | 补充/更换制动液、排气、更换制动片 |
| 方向盘抖动 | 轮胎动平衡不良、转向拉杆球头松动 | 做动平衡、检查转向系统 |
| 变速箱异响 | 齿轮油不足或变质、轴承磨损 | 检查油位、更换齿轮油、送修 |
---
以上保养指南适用于庆铃五十铃100P、600P、KV系列等主流轻卡车型，具体项目以车辆使用手册为准。定期规范保养可有效延长车辆使用寿命，降低故障率，保障行车安全。

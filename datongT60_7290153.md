# 上汽大通T60 驾驶注意事项
  
更新时间：2026年07月14日18时18分16秒   

## 一、启动与预热
1. **冷车启动**：柴油版车型在冷车启动后，建议怠速运转 **3-5分钟**，待机油充分润滑涡轮增压器后再行驶。
2. **冬季启动**：气温低于0℃时，请使用对应标号的柴油（如-10号或更低），避免柴油结蜡导致无法启动。
3. **预热系统**：等待仪表盘上的预热指示灯熄灭后再启动发动机。
## 二、行驶中注意事项
### 1. 变速箱操作
- **手动挡车型**：
  - 换挡转速建议控制在 **2000-2500rpm** 之间。
  - 倒挡需完全停稳后再挂入，避免打齿。
- **自动挡车型**：
  - 行驶中请勿挂入N挡滑行，会损坏变速箱。
  - 使用手动模式时注意发动机转速，避免长时间高转速运行。
### 2. 四驱系统（如配备）
- **分时四驱**：
  - **4H（高速四驱）**：可在时速80km/h以下切换。
  - **4L（低速四驱）**：必须停车挂N挡后再切换，用于越野、脱困等场景。
- **适时四驱**：铺装路面使用Auto模式即可，无需人为干预。
### 3. ESP车身稳定系统
- ESP默认开启，请勿随意关闭。
- **需要关闭的情况**：陷车脱困、沙地/泥地行驶时可按需关闭。
### 4. DPf再生（柴油颗粒捕集器）
- DPF再生期间请勿熄火或短途行驶中断再生过程。
- DPF指示灯亮起时，建议保持50km/h以上速度行驶20分钟左右完成主动再生。
## 三、载货与牵引
|项目|最大限制|
|---|---|
|额定载质量|约500kg（视具体车型）|
|牵引质量|最大3000kg（需配备拖车钩）|
1. **货物固定**：货箱内货物必须均匀分布并牢固固定，防止急刹时移位。
2. **重心控制**：装载较高货物时注意重心升高对操控的影响，转弯减速至30km/h以下。
3. **牵引拖车**：
   - T60牵引能力较强，但牵引状态下刹车距离显著增加。
   - 拖挂房车/拖车时最高时速不超过80km/h。
## 四、特殊路况驾驶
### ✅越野/非铺装路面
1. T60接近角27°，离去角24°，通过性较好但仍需注意托底风险。
2. **涉水深度建议不超过600mm**（原厂进气口高度约800mm）。
3.涉水后应检查前后桥及变速箱通气孔是否进水。
### ✅长下坡路段
1.自动挡车型应切换至手动模式低档位利用发动机制动；
2.避免长时间踩刹车导致刹车热衰减；
3.T60车身较重（约2吨），下坡惯性大需提前预判减速。
## ⚠️常见故障预防
|现象|预防措施|
|---|---|
|涡轮增压器异响/无力|熄火前怠速1分钟再熄火；使用正品机油|
|离合器沉重/异响（手动挡）|避免半联动时间过长；定期检查离合器油液|
|尿素系统报警（柴油版）|加注正规尿素溶液；不要拔掉传感器插头|
## 📌日常检查清单
1️⃣每次出车前检查：
   - [ ]轮胎气压及外观
   - [ ]机油液位
   - [ ]冷却液液位
   - [ ]刹车油液位
   - [ ]灯光信号是否正常
2️⃣每5000公里保养项目：
   - [ ]更换机油及机滤
   - [ ]检查空气滤芯清洁度
   - [ ]检查前后桥齿轮油
   - [ ]紧固底盘螺栓扭矩

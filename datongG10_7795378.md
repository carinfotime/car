# 上汽大通G10 动力参数介绍
  
更新时间：2026年07月14日09时44分28秒   

## 发动机
| 项目 | 参数 |
|------|------|
| **发动机型号** | 2.0T 涡轮增压汽油发动机 / 1.9T 涡轮增压柴油发动机 |
| **排量** | 2.0L（汽油）/ 1.9L（柴油） |
| **进气形式** | 涡轮增压 |
| **气缸排列形式** | 直列四缸（L4） |
| **每缸气门数** | 4 |
### 2.0T 汽油发动机
| 参数项 | 数值 |
|--------|------|
| **最大功率** | **165 kW（224 Ps）** |
| **最大功率转速** | 5500 rpm |
| **最大扭矩** | **345 N·m** |
| **最大扭矩转速** | 2000–4000 rpm |
| **燃油标号** | 92#及以上无铅汽油 |
| **供油方式** | 缸内直喷 |
### 1.9T 柴油发动机
| 参数项 | 数值 |
|--------|------|

| **最大扭矩转速** | 1400–2800 rpm |

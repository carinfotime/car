# AION LX 技术参数介绍
  
更新时间：2026年07月17日18时35分05秒   

## 车型概述
AION LX 是广汽埃安（GAC Aion）旗下的一款纯电动中型SUV，定位高端智能电动车型，以长续航、高性能和智能化配置为主要卖点。
---
## 动力系统
| 参数项 | 数据 |
|--------|------|
| 驱动方式 | 前驱 / 四驱 |
| 电机类型 | 永磁同步电机 |
| 最大功率（前驱版） | 150 kW（约204马力） |
| 最大功率（四驱版） | 300 kW（约408马力） |
| 最大扭矩（前驱版） | 350 N·m |
| 最大扭矩（四驱版） | 700 N·m |
| **0-100 km/h加速时间** | **3.9秒（四驱性能版）** |

## 电池与续航
| 参数项 | 数据 |
|--------|------|
| 电池类型 | **三元锂电池** / **弹匣电池** |
| 电池容量 | **93.3 kWh** / **144.4 kWh（千里版）** |
| CLTC综合续航里程 | 
- **标准版：600 km**
- **长续航版：650 km**
- **千里版：1008 km** |
> 💡 *千里版搭载144.4kWh超大容量电池包，是当时全球首款突破1000km续航的纯电SUV。*

## ADiGO智能驾驶辅助系统
AION LX搭载广汽自研的ADiGO智驾互联生态系统，包含：
### L2+级别智能驾驶辅助功能：
- ACC自适应巡航
- LKA车道保持辅助
- AEB自动紧急制动
- TJA交通拥堵辅助
- ICA集成式巡航辅助
- APA自动泊车辅助系统
###  硬件配置：
-  1个前置毫米波雷达
-  4个角毫米波雷达  
-  12个超声波雷达  
-  多个高清摄像头  
> *部分高配版本可选装更高阶的智能驾驶硬件套装。*

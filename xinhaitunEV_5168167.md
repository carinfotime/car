# 新海豚EV 驾驶注意事项
  
更新时间：2026年07月17日12时16分40秒   

## 一、启动与行驶前检查
1. **充电状态确认**：出发前检查仪表盘电量显示，确保续航里程满足行程需求。
2. **轮胎检查**：检查胎压是否正常（建议冷车胎压 2.5-2.8 bar），轮胎表面无破损、鼓包。
3. **灯光与信号**：测试大灯、转向灯、刹车灯、雾灯是否正常工作。
4. **制动系统**：轻踩刹车，确认制动响应灵敏，无异响或异常松软感。
5. **后视镜调整**：调整内外后视镜至最佳视野位置。
## 二、驾驶操作要点
1. **启动步骤**：
   - 携带智能钥匙进入车辆，踩下制动踏板。
   - 按下“启动/停止”按键，仪表盘显示“READY”后方可挂挡行驶。
2. **挡位操作**：
   - D 挡（前进）：正常行驶使用。
   - R 挡（倒车）：车辆完全停稳后再切换。
   - N 挡（空挡）：短暂停车时使用，拖车时务必挂 N 挡。
   - P 挡（驻车）：长时间停车时使用。
3. **加速与减速**：
   - 电门踏板轻踩缓放，避免急加速导致电量快速消耗。
   - 充分利用能量回收系统：松开电门踏板即可实现减速并回收电能。
4. **能量回收模式选择**：
   - **弱回收模式**：滑行距离长，适合高速或顺畅路况。
   - **强回收模式**：减速感明显，适合城市拥堵路段，可减少刹车使用频率。
## 三、续航与充电管理
1. **经济驾驶技巧**：
   - 保持匀速行驶，建议时速控制在 60-80 km/h（经济时速区间）。
   - 减少急加速和急刹车频率。
   - 合理使用空调：夏季空调温度设定在 24-26°C，冬季建议座椅加热优先于暖风。
2. **充电注意事项**：
   - **快充（直流）**：电量剩余20%-30%时进行快充，充至80%即可停止以保护电池寿命。
   - **慢充（交流）**：建议日常使用慢充充满至100%，有利于电池均衡维护。
   - 避免将电量耗尽至0%再充电，长期低电量存放会损伤电池。
3. **冬季特别提醒**：
   - 低温环境下续航里程会有所下降（约20%-30%），请预留充足余量。
   - 冬季用车后尽量立即充电，利用电池余温提高充电效率。
## 四、安全驾驶提示
1. **涉水行驶**：
   - EV车型虽有一定防水能力，但切勿强行通过水深超过轮毂中心的路段。
   - 涉水后应及时检查底盘及电池包状态。
2. **坡道起步辅助功能（HAC）**：
   新海豚EV配备坡道辅助系统，松开刹车后有约2-3秒保持时间。在此期间迅速切换至电门踏板即可平稳起步。
3. **低速行人警示音（AVAS）**
   车辆在低速行驶时会自动发出警示音提醒行人。此为正常功能，请勿误认为是故障。
4. **盲区监测与倒车影像**
   倒车时请结合倒车影像和左右后视镜观察周围环境。影像画面存在一定盲区不可完全依赖。

## ⚠️六、紧急情况处理
1. **仪表盘故障灯亮起**
   如出现红色警告灯（如动力电池故障、电机过热等），立即靠边停车并联系售后救援服务。黄色警告灯可谨慎低速驶往最近服务站检修。
2. **车辆无法启动**
   检查智能钥匙电量是否充足；尝试将钥匙靠近感应区域再启动；若仍无效请联系救援服务。
3. **碰撞事故后处理**
   发生碰撞事故后应尽快关闭车辆电源并远离车辆。即使外观无明显损伤也需前往服务站检查高压系统安全性。

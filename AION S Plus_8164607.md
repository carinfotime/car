# AION S Plus 驾驶注意事项
  
更新时间：2026年07月17日18时10分49秒   

## 一、启动与行驶前检查
1.  **钥匙与启动**：确保智能钥匙在车内，踩下制动踏板，按下一键启动按钮。仪表盘显示“READY”后方可挂挡行驶。
2.  **挡位操作**：旋转式换挡，顺时针为 D 挡（前进），逆时针为 R 挡（倒车），按下 P 挡驻车。换挡时务必踩住刹车。
3.  **电量检查**：出发前确认续航里程是否满足行程需求，建议保持电量在 20% 以上以保护电池。
4.  **胎压监测**：检查仪表盘胎压报警信息，标准胎压通常在 2.5 bar 左右。
## 二、日常驾驶技巧
1.  **能量回收**：
    -   可通过中控屏或拨片调节能量回收强度（I-Pedal / 低 / 中 / 高）。
    -   **高回收**：松油门拖拽感强，适合城市拥堵路况，可减少刹车使用。
    -   **低回收**：滑行更平顺，适合高速或乘客易晕车时使用。
2.  **单踏板模式 (I-Pedal)**：
    -   开启后仅通过油门踏板控制加速和减速（完全松开即刹停）。
    -   **注意**：需要时间适应紧急情况下的反应距离，建议先在空旷路段练习。
3.  **加速与超车**：
    -   电机响应快，动力输出直接。超车时深踩电门即可获得强劲推背感。
    -   **雨天/湿滑路面**：避免急加速或急转弯，防止轮胎打滑（ESP介入会有延迟）。
4.  **动能回收与刹车配合**：
    -   提前预判路况，利用能量回收减速可增加续航。
    -   **紧急情况**：直接踩死刹车，不要依赖能量回收。
## 三、充电注意事项
1.  **充电口位置**：快充口在车辆左后侧（慢充口在右前侧）。
2.  **充电建议**：
    -   **日常通勤**：建议使用慢充（7kW家用桩），充至80%-90%即可延长电池寿命。
    -   **长途出行**：使用快充桩时，电量从30%充至80%通常最快（约30分钟）。80%后充电功率会下降以保护电池。
3.  **极端温度充电**：
    -   **冬季低温**：刚行驶完立即充电效果最佳（电池温度较高）。若车辆长时间停放后充电速度会变慢（BMS会先加热电池）。
    -   **夏季高温**：避免暴晒后立即快充，建议等电池冷却后再充。
4.  **禁止行为**：
    -   严禁在车辆通电状态下插拔充电枪。
    -   雷雨天气尽量避免户外露天充电。
## 四、智能驾驶辅助系统 (ADAS)使用须知
1.  **自适应巡航 (ACC)**：
    -   `SET`键设定速度后车辆自动跟车。加减速较柔和但需注意加塞车辆识别可能延迟。
2.  **车道保持 (LKA)**：
    -   LKA仅辅助居中行驶而非自动驾驶。手必须轻握方向盘否则系统会报警并退出辅助功能。
3.  **自动泊车 (APA)**：
       适用于清晰标线的车位。斜列车位或极窄车位成功率较低请随时准备接管刹车。
## ⚠️五、安全与维护警告
1.	底盘保护
	-	AION S Plus的电池包位于底盘下方。遇到坑洼路面或马路牙子请减速慢行避免托底损伤电池包外壳导致质保失效甚至起火风险。
2.	涉水深度
	-	虽然IP67防水但建议积水深度不超过半个车轮高度约20厘米且车速低于10km/h匀速通过切勿冲浪式涉水以免连接器进水短路。
3.	轮胎磨损
	-	电动车自重较大扭矩大轮胎磨损比燃油车快尤其前轮每8000-10000公里建议前后轮交叉换位一次延长使用寿命。
4．空调使用省电技巧
	-	夏季开A/C+内循环比外循环省电约15%。冬季开暖气无需按A/C键直接调高温度利用PTC加热即可更省电因为压缩机不工作只耗风机电力。
5．长期停放
    若超过两周不使用请保持电量在50%-60%左右停放并断开小电瓶负极或者每周远程启动一次给12V小电瓶补电防止亏电无法解锁车门的情况发生。

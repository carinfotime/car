## 内部空间
### 前排空间
- **头部空间**：约 1,005 mm
- **腿部空间**：宽敞，支持多向电动调节
- **肩部空间**：约 1,495 mm
### 后排空间
- **头部空间**：约 985 mm
- **腿部空间**：约 950 mm（标准座椅位置）
- **肩部空间**：约 1,480 mm
- **后排座椅**：支持 40:20:40 比例放倒
### 座椅布局
- **座位数**：5座（标准版）/ 可选 7座布局（部分版本）
- **座椅材质**：温莎真皮 / 半苯胺真皮（选装）
## 后备箱容积
| 状态 | 容积 |
|------|------|
| 标准状态（5座） | 835 L |
| 后排座椅放倒 | 1,860 L |
| 7座版（第三排立起） | 260 L |
| 7座版（第三排放倒） | 835 L |
## 储物空间
- **中央扶手箱**：约 5.5 L
- **手套箱**：约 8 L
- **前门储物格**：可容纳 1.5 L 水瓶
- **后门储物格**：可容纳 1 L 水瓶
- **杯架**：前排 2个 / 后排 2个（中央扶手）
- **后排中央扶手储物**：带储物盒
## 通过性参数
| 参数 | 数据 |
|------|------|
| 接近角 | 29.5°（越野高度） |
| 离去角 | 28.5°（越野高度） |
| 纵向通过角 | 23.5°（越野高度） |
| 最大涉水深度 | 900 mm |
| 空气悬架升降范围 | +60 mm / -50 mm |
## 其他空间亮点
- **全景天窗**：双片式滑动全景天窗，面积约 1.2 m²
- **后排座椅**：电动靠背角度调节（选装）
- **后备箱**：电动尾门带脚踢感应开启
- **拖车钩**：可选电动伸缩拖车钩，最大拖曳质量 3,500 kg
> **注**：以上数据基于最新款揽胜运动版（L461），实际参数可能因不同年款、配置及选装包略有差异。

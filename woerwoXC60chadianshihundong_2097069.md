# 沃尔沃XC60插电式混动 内饰参数介绍
  
更新时间：2026年07月21日08时58分15秒   

## 一、座椅系统
| 项目 | 参数 |
|------|------|
| 座椅材质 | Nappa真皮（部分配置）/ 高级织物+皮革混搭 |
| 前排座椅功能 | 电动调节（主驾带记忆功能）、加热、通风（部分配置）、按摩（部分配置） |
| 后排座椅 | 4:6比例放倒、中央扶手带杯架 |
| 座椅布局 | 5座布局 |
| 座椅调节方式 | 前排电动12向调节（含腰部支撑） |
## 二、仪表盘与显示系统
| 项目 | 参数 |
|------|------|
| 液晶仪表盘 | 12.3英寸全液晶智能数字仪表盘 |
| 中控触摸屏 | 9英寸竖置触控屏 |
| 显示内容 | 车速、续航、电量、驾驶模式、导航信息、多媒体等 |
| HUD抬头显示 | 可选装（投射式，显示车速、导航、限速等信息） |
## 三、中控与多媒体系统
| 项目 | 参数 |
|------|------|
| 系统名称 | Sensus智能车载交互系统 |
| 支持功能 | 导航、蓝牙、CarPlay、Android Auto、语音控制 |
| 音响系统 | 标配：Harman Kardon 音响（600W，14扬声器） |
| 选装音响 | Bowers & Wilkins 殿堂级音响（1400W，15扬声器） |
| 声场技术 | 鹦鹉螺技术、空间声场优化 |
## 四、空调系统
| 项目 | 参数 |
|------|------|
| 空调类型 | 双区自动恒温空调（部分配置为四区） |
| 空气质量系统 | Clean Zone 清洁驾驶舱系统 |
| 功能 | PM2.5过滤、IAQS空气质量监测、负离子发生器 |
| 后排独立出风口 | 配备 |
## 五、方向盘
| 项目 | 参数 |
|------|------|
| 材质 | Nappa真皮包裹 |
| 功能 | 多功能方向盘（音响、语音、定速巡航控制） |
| 调节方式 | 手动4向调节（上下+前后） |
| 加热功能 | 可选装 |
## 六、内饰氛围与材质
| 项目 | 参数 |
|------|------|
| 内饰风格 | 北欧极简设计风格 |

| 氛围灯 | 可选装7色LED内饰氛围灯 |
| 车顶内衬 | 黑色织物/米色织物（依配置） |
| 地板材质 | 高级地毯 |
## 七、储物空间
| 项目 | 参数 |
|------|------|
| 中央扶手箱 | 带储物格、USB接口 |
| 门板储物格 | 前后门均配备 |
| 杯架 | 前排2个、后排中央扶手2个 |
| 手套箱 | 带照明、可锁 |
| 后备箱容积 | 常规：约468L（后排座椅放倒后约1395L） |

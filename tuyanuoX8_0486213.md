# 图雅诺X8 保养指南
  
更新时间：2026年07月18日16时18分04秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 机油 & 机滤 | 每5,000公里或6个月 |
| 空气滤清器 | 每10,000公里或12个月 |
| 燃油滤清器 | 每20,000公里或24个月 |
| 空调滤清器 | 每10,000公里或12个月 |
| 火花塞 | 每30,000公里 |
| 变速箱油 | 每40,000公里或24个月 |
| 刹车油 | 每40,000公里或24个月 |
| 防冻液 | 每40,000公里或24个月 |
| 正时皮带/链条 | 每60,000-80,000公里（视车型配置） |
| 轮胎换位 &动平衡 | 每10,000公里 |
> **注：** “先到为准”，即里程数或时间任一条件满足即应进行保养。

## 二、各阶段保养项目明细
### A级保养（5,000km / 6个月）
- [x] **更换**：发动机机油、机油滤清器
- [x] **检查**：轮胎气压及磨损、灯光系统、雨刮器、喇叭
- [x] **补充**：玻璃清洗液
- [x] **紧固**：底盘螺栓、轮胎螺母
### B级保养（10,000km /12个月）
包含A级全部项目，另增加：
- [x] **更换**：空气滤清器、空调滤清器
- [x] **检查**：刹车片厚度、制动盘磨损
- [x] **检查**：冷却液液位及浓度
- [x] **检查**：蓄电池状态及接线柱清洁度
- [x] **检查**：传动皮带松紧度及老化情况
- [x] **轮胎换位**
### C级保养（20,000km /24个月）
包含B级全部项目，另增加：
- [x] **更换**：燃油滤清器
- [x] **更换**：刹车油（制动液）
- [x] **检查/清洗**：节气门体
- [x] **检查**：转向系统间隙及助力油液位
- [x] **润滑**：车门铰链、锁扣等活动部件
### D级保养（40,000km /24–36个月）
包含C级全部项目，另增加：
- [x] **更换**：变速箱油（手动/自动）
- [x] **更换**：防冻液（冷却液）
- [x] **更换**：火花塞（汽油版）
- [x] **检查/调整**：气门间隙（柴油版适用）
- [x] **检查/更换**：前后差速器油（四驱版适用）
- [x] **深度检测**：发动机燃烧室积碳情况

## 三、关键部件专项说明
### 🛢️发动机机油
|  项目   |  规格要求   |
|---------|------------|
| API等级 | SN / SP （汽油）/ CJ‑4 / CK‑4 （柴油）|
| SAE粘度 |  5W‑30 （四季通用）或10W‑40 （高温地区）|
|  容量   |约5.0–6.0升 （以实际标尺为准）|
> ⚠️请务必使用符合国六排放标准的低灰分机油。
### 🔧变速箱油
|变速箱类型    |  规格     |容量参考|
|---          |---        |---     |
|手动变速箱    | GL‑4  75W‑90   约2.2L|
|自动变速箱    | ATF VI / Dexron VI   约7.0L|
### ❄️防冻液
> - G12/G13级别乙二醇型防冻液。
> - -35°C冰点标准。
> -建议使用原厂专用防冻液，避免混用不同颜色产品。

## 四、日常自检清单（建议每周一次）
1. ✅查看机油尺 ——冷车状态下在上下刻度之间。
2. ✅检查冷却液储液罐 ——不低于MIN线。
3. ✅观察刹车油壶 ——清澈且位于MAX与MIN之间。
4. ✅胎压检测 ——冷车标准值见驾驶侧B柱标签。
5. ✅灯光测试 ——大灯、转向灯、刹车灯、雾灯均正常。
6. ✅仪表盘警示灯 ——无异常亮起。
---
## 五、常见注意事项
1. 🚫不要超期使用机油——涡轮增压发动机对机油劣化更敏感。
2. 🚫不要忽略柴油版DPF再生提示——若仪表提示“颗粒捕集器正在再生”，请保持中高速行驶直至完成。
3. 🔔每次保养后请重置保养提醒指示灯：
   > —关闭点火开关 →按住“TRIP”键 →打开点火开关 →继续按住约5秒 →“扳手”图标闪烁后熄灭。
4. 📋保留每次保养记录单，有助于二手车保值及质保索赔。

# 锐航汽车驾驶注意事项
  
更新时间：2026年07月18日16时02分10秒   

## 一、启动前检查
1. **绕车一周**：检查轮胎气压、外观有无刮伤或异物，车底有无漏油、漏水痕迹。
2. **调整座椅与后视镜**：确保驾驶姿势舒适，左右及车内后视镜视野清晰。
3. **系好安全带**：驾驶员及所有乘员必须全程系好安全带。
4. **检查仪表盘**：启动后观察仪表盘指示灯是否正常熄灭（如发动机故障灯、机油灯、电瓶灯等）。
## 二、行驶中规范
1. **平稳驾驶**
   - 避免急加速、急刹车，以减少油耗和机械磨损。
   - 自动挡车型请勿在行驶中挂入 N 挡滑行。
2. **速度控制**
   - 严格遵守道路限速标志。
   - 雨天、雾天或夜间行驶时适当降低车速。
3. **换挡操作（手动挡）**
   - 根据车速及时换挡，避免低档高速或高档低速行驶。
   - 离合器踩到底再换挡，防止打齿。
4. **转向与变道**
   - 提前开启转向灯（至少3秒），观察后视镜确认安全后再变道。
   - 避免急打方向盘，尤其是在湿滑路面。
5. **保持车距**
   - 与前车保持至少2秒的安全距离（雨天或视线不佳时加倍）。
## 三、特殊路况应对
| 路况 | 注意事项 |
|------|----------|
| **高速公路** | 保持规定时速，远离大型车辆，避免长时间占用超车道。 |
| **山区道路** | 下坡时利用发动机制动（手动挡降档），避免长时间踩刹车导致过热。 |
| **涉水路段** | 确认水深不超过排气管高度，低速匀速通过，切勿停车或急加速。 |
| **冰雪路面** | 安装防滑链或雪地胎，轻柔操作油门和刹车，防止侧滑。 |
## 四、停车与熄火
1. **停车位置**：选择平坦、安全的地点，拉紧手刹（电子手刹确认指示灯亮起）。
2. **挂挡习惯**
   - 自动挡：挂 P 挡后再熄火。
   - 手动挡：平地挂空挡拉手刹；坡道停车挂一档或倒档并垫三角木。
3. **关闭用电设备**：熄火前关闭空调、大灯、音响等，保护电瓶寿命。
## ⚠️五、紧急情况处理
- **爆胎**：紧握方向盘，不猛踩刹车，缓慢减速至安全地带停靠。
- **刹车失灵**：逐步降档利用发动机制动，同时拉手刹（切勿一次拉死）。
- **发动机过热**：立即靠边停车怠速降温（不要立即打开水箱盖）。
- **车辆起火**：迅速熄火并撤离人员 →报警 →使用灭火器初期扑救。
## 🔧六、定期保养提醒
- ✅每5000公里或半年更换机油及机滤
- ✅每10000公里检查空气滤芯和空调滤芯
- ✅每20000公里检查火花塞和制动液
- ✅每40000公里更换变速箱油和防冻液
- ✅定期检查轮胎磨损及胎压（建议每月一次）
> 💡 *以上内容适用于锐航车型的一般驾驶场景。具体细节请以随车用户手册为准。*

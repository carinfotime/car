# 日产逍客 内饰参数介绍
  
更新时间：2026年07月14日12时12分20秒   

## 一、座椅
- **座椅材质**：
    - **入门版**：织物座椅
    - **中高配版**：皮/织物混搭座椅 或 **真皮座椅**
- **主驾驶座椅调节**：
    - **手动调节**（部分低配）：前后、靠背、高低（2向）
    - **电动调节**（中高配）：前后、靠背、高低（4向或6向），部分车型带腰部支撑调节
- **副驾驶座椅调节**：手动4向调节（前后、靠背）
- **前排座椅功能**：
    - **加热功能**：部分高配车型提供前排座椅加热
- **后排座椅**：
    - 后排座椅比例放倒（4/6分割）
    - 后排中央扶手带杯架
    - 后排中间头枕（可升降）
## 二、方向盘
- **方向盘材质**：
    - **塑料方向盘**（低配）
    - **真皮方向盘**（中高配）
- **方向盘调节**：手动上下+前后调节
- **多功能方向盘**：全系标配，集成音响控制、蓝牙电话、定速巡航/自适应巡航控制按键
- **换挡拨片**：部分高配车型配备
## 三、仪表盘
- **仪表盘类型**：
    - **传统机械指针式仪表 + 5英寸彩色行车电脑显示屏**
    - **7英寸彩色液晶仪表盘 + 机械指针组合**
    - *注：不同年款配置有所差异，最新款逐步普及更大尺寸液晶屏*
## 四、中控台与信息娱乐系统
| 项目 | 参数/描述 |
|------|-----------|
| **中控屏幕尺寸** | • 7英寸触控液晶屏（低配）<br>• 9英寸触控液晶屏（中高配）<br>• *部分新款车型搭载12.3英寸触控屏* |
| **车机系统功能** | • Nissan Connect超智联系统<br>• GPS导航系统<br>• Bluetooth蓝牙连接<br>• USB/AUX接口<br>• Wi-Fi热点<br>• OTA远程升级 |
| **手机互联/映射** | • CarPlay / CarLife / HiCar (根据年款和配置支持) |
| **语音识别控制** | • 多媒体系统<br>• 导航<br>• 电话<br>• *部分高配支持空调控制* |
| **音响系统** | • 4扬声器 (低配)<br>• 6扬声器 (中高配)<br>• *顶配可选装BOSE品牌音响* |
## 五、空调系统
-  双区独立自动空调 (大部分配置)
-  后排出风口
-  车内PM2.5过滤装置 / *部分车型带负离子空气净化器*
-  空调控制方式：物理按键 +旋钮 / *部分新款集成于触控屏*
## 六、其他内饰细节与储物空间
### 📦储物空间
|位置|描述|
|---|---|
|前排杯架|中央通道2个杯架|
|中央扶手箱|内部深度适中，可放置小件物品|
|手套箱|常规大小，带阻尼|
|门板储物格|前后门板均可放置水瓶及杂物|
|其他储物处|挡把前方储物槽、卡槽等|
### 💡内部配置亮点
-  车内氛围灯 (单色 / *部分高配多色*)
-  电动天窗 / *全景天窗 (中高配)*
-  内后视镜防眩目：手动防眩目 (*自动防眩目仅顶配*)
-  化妆镜+照明灯 (主副驾驶遮阳板)

## 📐内饰尺寸参考
*以下为近似值，具体以实车为准*
-  前排头部空间 (mm)：约980–1020 mm
-  后排头部空间 (mm)：约940–970 mm
-  后排腿部空间 (mm)：约650–720 mm (取决于前排位置)
-  后备厢容积 (L)：约486 L (*标准状态*) → *放倒后排可扩展至约1410 L*

# 东风帕拉索 动力参数介绍
  
更新时间：2026年07月20日13时30分11秒   

## 发动机
东风帕拉索搭载的是一台 **1.8T 四缸涡轮增压发动机**（型号为 CE18），该发动机源自宝马技术授权，并由绵阳新晨动力生产。
| 项目 | 参数 |
| :--- | :--- |
| **发动机型号** | CE18 |
| **排量** | 1.8 L (1765 mL) |
| **进气形式** | 涡轮增压 |
| **气缸排列形式** | 直列四缸 (L4) |
| **每缸气门数** | 4 |
| **供油方式** | 多点电喷 (MPI) |
| **燃油标号** | 92号及以上无铅汽油 |
## 性能数据
-   **最大马力：** **231 Ps** （约 170 kW）
-   **最大功率：** **170 kW**
    -   *最大功率转速：* 5200 rpm
-   **最大扭矩：** **300 N·m**
    -   *最大扭矩转速：* 2000 - 4000 rpm
> **特点说明：**
> -   这台发动机在低转速区间（2000转）即可输出峰值扭矩，保证了车辆在越野或重载情况下的低速爬坡和脱困能力。
> -   高功率调校使其在高速巡航和超车时也具备充足的动力储备。
## 变速箱与驱动系统
-   **变速箱类型：**
    -   **6速手动变速箱 (6MT)** — *部分车型*
    -   **8速自动变速箱 (8AT)** — *部分车型*
-   **驱动方式：**
    -  前置后驱 (FR)
    -   部分高配车型配备分时四驱系统（带低速四驱挡位）。
## NVH与排放
-   **排放标准：** 国VI b
-   NVH表现经过优化，采用静音正时链条等技术，提升驾驶舒适性。

### 📊 动力总成小结
东风帕拉索的动力系统兼顾了城市通勤与户外越野的双重需求。1.8T发动机提供的231匹马力和300N·m扭矩在同级中表现突出，配合手动或自动变速箱以及可选的四驱系统，使其既能满足日常装载运输，也能胜任轻度至中度的越野路况。

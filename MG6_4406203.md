# MG6 空间参数介绍
  
更新时间：2026年07月17日08时51分48秒

| 参数 | 数值 |
|------|------|
| 车长 | 4704 mm |

| 车高 | 1466 mm |
| 轴距 | 2715 mm |
| 前轮距 | 1562 mm |

## 车内空间
### 前排空间
- **头部空间**：约980mm（座椅调至最低时）
- **腿部空间**：充裕，驾驶位支持6向调节
- **肩部空间**：约1420mm
### 后排空间
- **头部空间**：约920mm
- **腿部空间**：约850mm（前排座椅调至适中位置）
- **肩部空间**：约1380mm
- **后排座椅宽度**：约1300mm，可舒适乘坐3名成年人
## 储物空间
### 后备箱
- **标准容积**：499升
- **后排座椅放倒后容积**：约1183升
- **开口宽度**：约1050mm
- **开口高度**：约500mm
- **进深（标准状态）**：约950mm
- **进深（座椅放倒）**：约1750mm
### 车内储物格
- **手套箱容积**：约8升
- **前排中央扶手箱容积**：约3升
- **前门板储物格**：可容纳2瓶500ml矿泉水 + 杂物
- **后门板储物格**：可容纳1瓶500ml矿泉水 + 杂物
- **杯架数量**
    - *前排*：2个（中央通道）
    - *后排*：1个（中央扶手翻折式）
## 乘坐体验特点
1. **后排地台凸起高度**较低，约为80mm，对中间乘客影响较小。
2. **后排座椅靠背角度适中，支持4/6比例放倒。
3. A柱倾斜角度较大，但未明显压缩前排头部空间。
4. C柱设计兼顾溜背造型与后排头部空间的平衡。
> *以上数据基于MG6主流车型版本，实际尺寸可能因具体配置和年款略有差异。*

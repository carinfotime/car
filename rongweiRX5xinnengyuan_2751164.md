# 荣威RX5新能源 技术参数介绍
  
更新时间：2026年07月12日09时57分42秒   

## 一、车型概览
荣威RX5新能源是上汽荣威旗下的一款插电式混合动力紧凑型SUV，融合了传统燃油动力与电动驱动技术，兼顾燃油经济性与续航能力。
---

| 最大扭矩（发动机） | 250 N·m |

## 七、智能科技配置
- **12.3英寸全液晶仪表盘**
- **10.4英寸中控触摸屏**
- 斑马智行互联系统（支持语音控制、在线导航、远程控制）
- 4G 网络 + Wi-Fi 热点
- OTA 远程在线升级
- 智能驾驶辅助系统（部分车型）

  - AEB 自动紧急制动
  - LDW 车道偏离预警
  - 360° 全景影像

## 九、工作模式
荣威RX5新能源提供多种驾驶模式，适应不同驾驶场景：
| 模式 | 说明 |
|------|------|
| EV 纯电模式 | 纯电驱动，适合城市短途通勤 |
| 智能混动模式 | 系统自动优化燃油与电能分配 |
| 强制充电模式 | 发动机为电池充电，保持电量 |
| ECO 经济模式 | 降低能耗，提升续航 |
| Sport 运动模式 | 动力响应更积极，加速性能更强 |
---
## 十、总结
荣威RX5新能源作为一款插电混动SUV，在动力性能、燃油经济性、智能科技配置方面表现均衡，适合对绿牌有需求、日常通勤距离适中且偶尔长途出行的用户。其纯电续航满足城市代步需求，综合续航表现优秀，是新能源紧凑型SUV市场中较有竞争力的选择。

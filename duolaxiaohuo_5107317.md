# 多拉小货车辆操作指南
  
更新时间：2026年07月18日14时37分49秒   

## 一、出车前检查
### 1. 证件与保险
-   **驾驶证**：确认准驾车型与车辆相符（如C1可驾驶蓝牌轻型货车）。
-   **行驶证**：确认车辆年检、保险在有效期内。
-   **营运证**：若从事经营性运输，需携带有效道路运输证。
### 2. 车辆外观与安全
-   **轮胎**：检查胎压是否正常，胎面有无鼓包、裂纹，备胎是否完好。
-   **灯光**：测试大灯、转向灯、刹车灯、雾灯及示廓灯是否正常。
-   **反光条**：车身侧面及尾部反光条是否完整、清洁。
-   **车牌**：前后车牌清晰无遮挡。
### 3. 货物装载检查
-   **栏板/货箱门**：确认锁扣牢固，无变形或松动。
-   **篷布/绳索**：若需遮盖防雨或固定货物，检查篷布有无破损，绳索强度足够。
## 二、货物装载规范
### 1. 载重限制
-   严格遵守行驶证上标注的**总质量**与**核定载质量**。
-   严禁超载。超载会导致刹车距离变长、轮胎爆裂风险剧增，且严重违法。
### 2. 装载原则
| 原则 | 说明 |
| :--- | :--- |
| **均匀分布** | 重物放在车厢中部靠前位置（靠近驾驶室），避免头重脚轻或后轴过重。 |
| **固定牢固** | 使用捆绑带、绳索或网兜将货物固定，防止转弯或急刹时移位。 |
| **不超高/宽/长** | - **高度**：从地面起不超过4米（集装箱车不超过4.2米）。<br>- **宽度**：不得超出车厢。<br>- **长度**：货物前端不超过车厢前部；后端超出部分需做明显警示标志（如绑红布）。 |
### 3. 特殊货物处理
-   **散货（沙石/粮食）**：必须用篷布严密遮盖，防止遗撒。
-   **液体桶装物**：桶口朝上，底部垫防滑垫，桶间用木楔塞紧。
-   **管材/长条物**：超长部分必须悬挂警示标志；严禁在市区道路违规拖拽。
## 三、驾驶操作要点
### 1. 起步与换挡
-   空车起步可用2挡；重车建议用1挡起步。
-   柴油发动机转速保持在1500—2500转之间换挡最省油且动力足。
-   离合抬起要慢半拍，避免因载重导致熄火或传动轴冲击。
### 2. 行驶中注意事项
| 场景 | 操作要点 |
| :--- | :--- |
| **转弯/变道** | -提前减速至30km/h以下。<br>-转弯半径比小车大很多（后轮会向内偏移），注意内轮差。<br>-严禁急打方向。 |
| **下长坡/陡坡** | -使用发动机制动（挂低挡位），避免长时间踩刹车导致刹车过热失效。<br>-必要时配合排气制动（如有此功能）。 |
| **雨天/湿滑路面** | -空车时后轮抓地力弱，易侧滑；重车时刹车距离大幅延长。<br>-保持比小车多50%以上的跟车距离。 |
### 3. “多拉小货”常见误区纠正
> ❌ “货车重心高没关系”——错！重心高极易侧翻，尤其是急弯或紧急避让时。
>
> ❌ “开货车就要猛踩油门”——错！平顺加速更省油且保护传动系统。
## 四、停车与卸货
### 1.停车规范
-   尽量选择平坦硬质路面停车。若停在坡道：
    -   拉紧手刹后挂入低速挡（上坡挂1挡，下坡挂倒挡）。
    -   车轮下塞三角木或石块。
-   市区临时卸货不得妨碍其他车辆行人通行；开启双闪警示灯。
### .卸货安全
-   先解开捆绑绳的一侧（不要全部解开），防止货物突然倾泻砸伤人。
-   打开栏板时站在侧面而非正后方——防止货物涌出伤人。

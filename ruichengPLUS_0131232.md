# 长安锐程PLUS 驾驶注意事项
  
更新时间：2026年07月11日15时16分47秒   

## 一、新车磨合期注意事项 (前1500公里)
- **避免急加速与急刹车**：平稳驾驶，让发动机和变速箱各部件充分磨合。
- **控制发动机转速**：建议转速不超过 **3000 rpm**，避免长时间高负荷运行。
- **避免定速巡航**：磨合期内尽量使用人工控制油门，让车速和转速有适当变化。
- **不要超载**：载重应不超过额定载荷的 **75%**，减轻悬挂和轮胎压力。
## 二、日常驾驶操作要点
### 1. 启动与预热
- 启动后等待约 **30秒至1分钟**（冬季可适当延长），待怠速稳定后再起步。
- 起步后前几分钟避免大脚油门，待水温表指针上升后再正常行驶。
### 2. 变速箱使用 (7速湿式双离合)
- **等红灯时**：若等待时间超过30秒，建议挂入 **N挡 + 拉手刹/电子手刹**；短时间可D挡踩刹车。
- **拥堵路况**：频繁走走停停时，可切换至手动模式或S挡（运动模式），减少变速箱频繁换挡导致的过热。
- **停车顺序**：车停稳 → N挡 → 拉手刹 → P挡 → 熄火。避免直接挂P挡冲击锁止机构。
### 3. 涡轮增压发动机保养
- **熄火前怠速冷却**：长途或激烈驾驶后，建议怠速运转 **1-2分钟**再熄火，保护涡轮增压器。
- **使用高品质机油**：务必使用符合厂家规格的全合成机油（通常为5W-30或0W-20）。
## 三、特殊路况驾驶
| 路况 | 注意事项 |
|------|----------|
| **雨天/湿滑路面** | - ESP车身稳定系统默认开启，请勿手动关闭。<br>- 降低车速，增大跟车距离。<br>- 积水路段确认深度不超过排气管口。 |
| **山路/长下坡** | - L挡或手动模式低档位行驶，利用发动机制动。<br>- 避免长时间踩刹车导致刹车热衰减。 |
| **高速行驶** | - IACC集成式自适应巡航适合高速使用，但需保持手握方向盘。<br>- 注意胎压监测报警。 |
## 四、智能系统使用提醒
### AR全景影像
- **狭窄路段/倒车入库时务必开启**：利用360°影像观察盲区。
- **注意摄像头清洁**：前后摄像头脏污会影响成像效果。
### IACC / ACC自适应巡航
- **仅限高速公路或封闭道路使用**
- ***切勿***在市区拥堵、弯道过急、雨雪天气下依赖此功能
- ***始终***保持对车辆的控制权
### “小安”语音助手
- *“小安你好”*唤醒后可直接控制导航、空调、车窗等
- *注意*：语音识别受车内噪音影响较大
## 五、轮胎与制动系统
|项目|检查频率/标准|
|---|---|
|胎压（冷态）|建议2.3~2.5 bar（参考B柱标签）|
|轮胎磨损标记|花纹深度低于1.6mm需更换|
|刹车片厚度|<3mm时建议更换|
## ⚠️六、常见故障警示灯处理
|警示灯亮起|应对措施|
|---|---|
|发动机故障灯（黄）|-尽快检查并前往维修站<br>-若伴随抖动/加速无力则立即停车|
|电池灯（红）|-立即关闭非必要电器<br>-尽快驶往维修点|
|胎压报警|-靠边停车检查轮胎外观及气压<br>-补气后需在系统中重置胎压监测|
## ✅七、省油驾驶技巧
1.  保持经济时速（60~90 km/h）
2.  预见性驾驶——提前松油门滑行代替急刹
3.  减少不必要的怠速热车时间
4.  定期清理车内多余负重
5.  空调温度设定在22~25℃，风速适中即可

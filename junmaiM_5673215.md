# 骏迈M 儿童乘车安全注意事项
  
更新时间：2026年07月18日08时15分20秒   

为了保障您孩子的乘车安全，请务必遵守以下事项：
## 1. 儿童安全座椅使用规范
- **强制使用**：12周岁以下或身高低于1.4米的儿童，必须使用与其年龄、体重相匹配的儿童安全座椅。
- **反向安装**：建议0-3岁（或体重低于18公斤）的婴幼儿使用反向安装的儿童安全座椅，以更好地保护头颈部。
- **正确固定**：确保安全座椅通过 ISOFIX 接口或安全带牢固固定于后排座位上，安装后不应有明显晃动。
## 2. 乘坐位置要求
- **严禁前排**：切勿让儿童坐在副驾驶位置。在发生碰撞时，副驾驶的安全气囊弹出可能对儿童造成严重伤害。
- **后排为佳**：所有儿童均应乘坐于车辆后排座位。
- **中间位置**：如果条件允许，将儿童安全座椅放置在后排中间位置，该位置离两侧车门最远，相对更安全。
## 3. 车窗与车门安全
- **开启童锁**：启动车辆前，请务必启用后车门的儿童安全锁。启用后，车门无法从车内打开。
- **车窗控制**：启用驾驶员侧的车窗锁止功能（如有），防止儿童自行升降车窗夹伤手指或探出窗外。
- **禁止探头**：严禁将头、手或身体任何部位伸出天窗或侧窗外。
## 4. 车内活动与物品管理
- **避免干扰驾驶**：教育孩子不要在车内大声喧哗、打闹或突然触碰方向盘、挡位杆等驾驶操作部件。
- **小物件管理**：收好车内的小物件（如硬币、纽扣电池、笔帽等），防止儿童误吞。
- **避免硬物**：车内不要放置尖锐、坚硬的装饰品或摆件。急刹车时这些物品可能飞起伤人。
## 5. 停车与上下车注意事项
- **观察后方**：下车前，教导孩子先通过车窗观察后方是否有来车或行人。
- **右侧下车**：尽量让孩子从右侧（靠路边一侧）下车。
- **切勿独留车内**：无论时间长短、天气如何（夏季高温/冬季寒冷），切勿将孩子单独留在车内。车厢内温度会迅速升高至危险水平。
## 6. 行车途中应急处理
| 情况 | 建议做法 |
| :--- | :--- |
| 孩子哭闹 | 尽快在安全地点停车安抚。切勿边开车边回头安抚。 |
| 晕车呕吐 | 提前准备呕吐袋和湿巾；适当开窗通风；避免行车前吃得太饱。 |
| 突发急病 | 立即靠边停车并拨打急救电话（120）。 |
---
> ⚠️ **重要提示**：
> -   “骏迈M”车型的具体配置可能因年款不同而有所差异。请务必查阅随车的《用户手册》中关于“儿童安全”的章节。
> -   驾驶员应始终保持专注驾驶，所有与儿童的互动应在车辆完全停稳后进行。

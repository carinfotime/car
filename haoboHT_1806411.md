# 昊铂HT 内饰参数介绍
  
更新时间：2026年07月20日14时05分26秒   

## 一、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | Nappa真皮（部分配置）/ 仿皮材质 |
| 前排座椅功能 | 电动调节（主驾12向/副驾10向）、加热、通风、记忆（主驾） |
| 后排座椅功能 | 电动调节（靠背角度可调）、加热、通风 |
| 后排座椅放倒 | 支持4/6比例放倒，放倒后可形成纯平空间 |
| 座椅布局 | 2+3五座布局 |
| 头枕调节 | 前后/上下可调 |
## 二、仪表与中控
| 项目 | 参数 |
|------|------|
| 仪表盘 | **8.8英寸**全液晶仪表盘 |
| 中控屏 | **14.6英寸**悬浮式中控触摸屏 |
| HUD抬头显示 | **AR-HUD**增强现实抬头显示系统（部分配置） |
| UI系统 | ADiGO智驾互联系统 |
## 三、智能座舱
- **芯片方案**：高通骁龙8155芯片
- **语音控制**：四音区语音识别，支持连续对话
- **面部识别**：驾驶员疲劳监测 / Face ID人脸识别
- **手势控制**：支持部分手势交互功能
- **OTA升级**：整车FOTA远程升级
## 四、音响系统
- **扬声器数量**：22个扬声器（高配）/ 12个扬声器（标准版）
- **音响品牌**：Premium HIFI音响系统
- **主动降噪**：支持主动降噪功能
- **音效模式**：多种预设音效场景可选
## 五、空调系统
- **空调类型**：双区自动恒温空调
- **后排独立出风口**：有
- **PM2.5过滤**：CN95级空气过滤系统
- **负离子发生器**：有
- **香氛系统**：多色氛围香氛可选（高配）
## 六、内饰材质与配色
### 🔹内饰颜色选项
1. **黑色内饰**
2. **米色内饰**
3. **棕色内饰**
### 🔹装饰面板
-  木纹饰板 /  金属拉丝饰板 /  钢琴烤漆面板（根据不同配置搭配）
### 🔹方向盘
-  真皮包裹多功能方向盘
-  电动四向调节 +记忆功能 +加热功能（高配）

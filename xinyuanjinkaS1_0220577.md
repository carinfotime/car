# 鑫源金卡S1 儿童乘车安全注意事项
  
更新时间：2026年07月18日16时56分01秒   

## ⚠️ 核心原则
- **严禁儿童坐在副驾驶位**：副驾驶是车内最危险的位置之一，一旦发生碰撞，安全气囊弹出会对儿童造成严重伤害（甚至致命）。
- **12岁以下儿童必须使用安全座椅**：鑫源金卡S1的后排座椅需正确安装符合国家标准的儿童安全座椅。
- **切勿将儿童单独留在车内**：任何情况下都不要将孩子单独留在车内，哪怕车窗留缝或天气凉爽。
## 🚗 具体注意事项
### 1. 安全座椅安装
- **适用年龄/体重**：
  - **0-15个月（约13kg以下）**：使用反向安装的婴儿提篮式安全座椅。
  - **1-4岁（9-18kg）**：使用正向安装的儿童安全座椅。
  - **4-12岁（15-36kg）**：使用增高垫配合成人安全带。
- **固定方式**：鑫源金卡S1后排通常配备ISOFIX接口或安全带固定装置，请严格按照说明书操作，确保座椅牢固不晃动。
### 2. 车门与车窗安全
- **开启儿童安全锁**：车辆后门两侧通常有儿童安全锁拨杆。启用后，车门无法从内部打开，防止孩子在行驶中误开车门。
- **车窗防夹功能**：鑫源金卡S1可能不具备全车车窗防夹功能。请勿让孩子将头、手或身体伸出窗外；建议锁定车窗升降开关（主驾位通常有总控锁）。
### 3. 车内物品与活动
- **避免尖锐/硬物**：不要在座位上放置玩具、水瓶等硬物，急刹车时可能飞溅砸伤孩子。
- **禁止在车内打闹**：行驶中严禁孩子在车厢内站立、跑动或嬉戏打闹。
- **安全带正确佩戴**：
  - 肩带应跨过肩膀和胸部中央（避开颈部）。
  - 腰带应紧贴大腿根部（避开腹部）。
  - 切勿让安全带压在孩子脖子或脸上。
### 4. 停车与上下车
- **确认周围环境**：停车后，家长应先观察后方来车及行人，再开门让孩子下车。尽量让孩子从右侧（靠路侧）上下车。
- **避免倒车盲区**：鑫源金卡S1作为货车/微卡车型，车身较高且盲区较大。倒车前务必确认孩子不在车辆后方或周围玩耍。
### 🛑 ❌绝对禁止的行为
| ❌禁止事项 | ✅正确做法 |
|-----------|-----------|
| ❌抱着孩子坐副驾或后排 | ✅必须使用专用安全座椅 |
| ❌让孩子独自在车上睡觉 | ✅家长全程陪同看护 |
| ❌行驶中给孩子喂食/喝水 | ✅停车后再进行 |
| ❌让孩子操控方向盘/挡杆 | ✅教育孩子这些是危险操作 |

# 瑞擎V6 技术参数介绍

## 🛡️六、安全与辅助配置
### ✅主动安全
- ABS防抱死 + EBD制动力分配 + ESP车身稳定控制
- FCW前方碰撞预警 + AEB自动紧急制动（带行人识别）
- LDW车道偏离预警 + LKA车道保持辅助
- BSD盲区监测 + RCTA后方交通穿行提示
- DOW开门预警
### ✅被动安全
- RQF高强度笼式车身（热成型钢比例达30%）
- SRS前排双气囊 / SRS前排侧气囊 / SRS前后排头部气帘（共8个）
- TPMS胎压监测系统（带数值显示）
## 💻七、智能座舱与科技配置
### 📱信息娱乐系统
- RQ-Link智能互联系统（12.3英寸中控触摸屏）
- AR实景导航 / OTA远程升级 / AI语音助手（支持连续对话）
- HUD抬头数字显示（投影面积达10英寸）
### 🎵音响系统
- BOSE® Performance至臻系列音响（14扬声器，含低音炮）
### 🌬️舒适与环境
- PM2.5过滤装置 + AQS空气质量管理系统 + CN95级空调滤芯
- NVH静音工程：双层隔音玻璃（前门）、ANC主动降噪技术
## 🔌八、外部灯光与视野
*   全LED自适应远近光大灯（带AFS随动转向功能）
*   贯穿式LED尾灯（动态流水转向效果）
*   全景天窗（可开启，带电动遮阳帘）
*   外后视镜电动折叠/加热/记忆/倒车自动下翻功能

# 悦ER 驾驶注意事项
  
更新时间：2026年07月11日03时16分57秒   

## 一、启动前检查
1. **绕车检查**：检查轮胎气压、车身外观、灯光是否正常。
2. **调整座椅与后视镜**：确保驾驶姿势舒适，后视镜视野清晰。
3. **系好安全带**：全车人员务必佩戴安全带。
4. **检查仪表盘**：启动后观察是否有故障灯亮起。
## 二、行驶中注意事项
1. **平稳驾驶**：避免急加速、急刹车，保护车辆传动系统。
2. **控制车速**：遵守道路限速，切勿超速行驶。
3. **保持车距**：与前车保持安全距离，预留足够反应时间。
4. **合理使用灯光**：
   - 夜间或雨雾天气开启近光灯。
   - 会车时切换近光灯，避免远光灯干扰对向车辆。
5. **注意盲区**：变道或转弯前务必观察后视镜及侧方盲区。
## 三、特殊路况驾驶
1. **雨天/湿滑路面**：
   - 降低车速，避免急转向。
   - 开启雨刮器和雾灯。
   - 避免积水路段，防止涉水熄火。
2. **山路/弯道**：
   - 提前减速，鸣笛示意。
   - 不占道行驶，弯道不超车。
3. **高速行驶**：
   - 保持匀速，避免长时间占用超车道。
   - 注意疲劳驾驶，每2小时休息一次。
## 四、停车与熄火
1. **规范停车**：停入指定车位，拉紧手刹。
2. **熄火前**：关闭空调、车灯、音响等用电设备，保护电瓶。
3. **锁车检查**：确认车窗、天窗关闭，车门锁好。
## 五、定期保养提示
| 项目 | 建议周期 |
|------|----------|
| 机油更换 | 每5000-10000公里 |
| 轮胎检查 | 每月一次 |
| 刹车系统 | 每10000公里检查 |
| 空调滤芯 | 每6个月或10000公里 |
## 六、紧急情况处理
1. **爆胎**：握紧方向盘，不猛踩刹车，缓慢减速靠边。
2. **刹车失灵**：降档利用发动机制动，逐步拉手刹。
3. **车辆自燃**：立即熄火，撤离人员，拨打119。
4. **事故处理**：开启双闪，放置三角警示牌（高速150米外），报警并联系保险公司。

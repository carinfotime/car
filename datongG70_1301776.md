# 上汽大通MAXUS G70 技术参数介绍
  
更新时间：2026年07月10日20时30分03秒   

## 一、车型定位
上汽大通MAXUS G70是一款中大型家用MPV，主打宽敞空间、舒适乘坐与智能科技。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| 发动机类型 | 2.0T涡轮增压汽油发动机 |
| 排量 | 1996 mL |
| 最大功率 | **172 kW**（约234马力） |
| 最大扭矩 | **360 N·m** |
| 变速箱 | **9速自动变速箱（9AT）** |
| 驱动方式 | **前置前驱** |
| 最高车速 | ≥180 km/h |
| WLTC综合油耗 | **约8.5 L/100km** |

## 三、车身尺寸
| 项目 | 参数 |
|------|------|
| 长 × 宽 × 高 | **4900 mm × 1885 mm × 1768 mm** |
| 轴距 | **2960 mm** |
| 座位数 | **7座**（2+2+3布局）可选6座（2+2+2） |

## 四、底盘与悬挂
- **前悬挂**：麦弗逊式独立悬挂
- **后悬挂**：多连杆式独立悬挂
- **制动系统**：前通风盘 / 后实心盘
- **转向系统**：电动助力转向（EPS）
- **轮胎规格**：225/55 R18

## 五、智能科技配置
### 🖥️智能座舱
- **双联屏设计**
    - *12.3英寸全液晶仪表盘*
    - *12.3英寸中控触摸屏*
- *支持语音控制、导航、在线音乐*
- *手机互联*（CarPlay / HiCar）
### 🚗驾驶辅助系统
- L2级智能驾驶辅助
    - ACC自适应巡航
    - AEB自动紧急制动
    - LDW车道偏离预警
    - FCW前方碰撞预警
    - TSR交通标志识别
### 📷泊车辅助
- *360°全景影像*
- *前后驻车雷达*

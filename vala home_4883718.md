## 居住舱（生活区）配置
### 1. 空间布局
- **前排驾驶区**：独立驾驶舱，可贯通至后方生活区。
- **中部会客/休息区**：
  - 航空级独立座椅（带通风、加热、按摩、旋转功能）。
  - 可折叠多功能桌板。
  - 后排座椅可完全放平，形成 **2000mm × 1400mm** 的双人床。
- **尾部功能区**：
  - 整体式厨房操作台（电磁炉、洗菜池、微波炉）。
  - 独立卫浴间（可抽拉式淋浴、电动粉碎马桶、换气扇）。
### 2. 水电系统

| **清水箱容积** | 100 L |
| **灰水箱容积** | 50 L |
| **黑水箱容积** | 20 L（电动直排/抽拉式） |
| **生活电池** | 48V / 10 kWh 磷酸铁锂电池 |
| **太阳能板** | 车顶 500W 柔性太阳能板 |
| **逆变器** | 3000W 纯正弦波逆变器 |
| **充电方式** | 行车充电、市电接口、太阳能充电 |
### 3. 舒适与智能
- **空调系统**：顶置驻车空调（冷暖两用，低噪音）。
- **影音系统**：后排 32 英寸智能投影幕布 + 环绕音响。
- **智能控制**：12.3 英寸中控屏集成车辆控制与家居控制，支持语音交互。
- **网络连接**：5G 车载模块 + Wi-Fi 热点。

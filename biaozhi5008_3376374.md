# 标致5008 操作指南
  
更新时间：2026年07月19日18时32分45秒   

## 一、车辆基本操作
### 1. 启动与熄火
- **启动**：智能钥匙在车内，踩下刹车踏板，按下“START/STOP”按钮。
- **熄火**：车辆停稳后，再次按下“START/STOP”按钮。
### 2. 电子手刹
- **启用**：向上拉起电子手刹开关。
- **解除**：踩下刹车，向下按压电子手刹开关；或挂入D挡轻踩油门自动解除。
### 3. 挡位操作（电子挡杆）
| 挡位 | 操作方式 |
|------|----------|
| P（驻车） | 按下挡杆顶部的P按钮 |
| R（倒车） | 按住挡杆侧面按钮，向前推 |
| N（空挡） | 向前或向后轻推至中间位置 |
| D（前进） | 向后拉 |
| M（手动模式） | D挡位置向右拨动，通过拨片换挡 |

## 二、仪表盘与方向盘
### i-Cockpit®座舱特色
- **小方向盘设计**：方向盘应调至较低位置，从方向盘上方看仪表盘。
- **仪表盘调节**：通过方向盘左侧滚轮调节亮度、显示模式。
### 多功能方向盘按键
#### 左侧区域
- **音量控制**：滚轮上下调节音量。
- **语音控制**：按下激活语音助手。
- **电话接听/挂断**。
#### 右侧区域
- **多媒体切换**：切换音源（收音机/USB/蓝牙）。
- **仪表盘信息切换**：滚动切换显示内容（油耗、导航、行驶数据等）。

## 三、中控触摸屏操作
### Blue-i®智能互联系统
#### 主界面功能
1. **导航**
   - 输入目的地支持拼音/手写。
   - 实时路况显示需联网。
2. **多媒体**
   - USB接口位于中央扶手箱内。
   - Bluetooth配对：设置→蓝牙→搜索设备。
3. **空调控制**
   - 双区自动空调可通过屏幕调节温度、风量。
   - “AUTO”键自动维持设定温度。
   - “SYNC”键同步主副驾驶温度。
4. **车辆设置**
   - “驾驶辅助”：车道保持、盲区监测开关。
   - “灯光”：伴我回家功能、日行灯设置。
   - “门锁”：行车自动落锁设置。

## 四、驾驶辅助系统
### ADAS功能操作
1. **定速巡航/自适应巡航**
   - CRUISE键开启巡航模式。
   - SET +/-设定速度（30km/h以上）。
   - CANCEL取消，RES恢复上次设定速度。
2. **车道保持辅助**
   - LKA按键位于灯光拨杆末端。
   - >70km/h自动激活，偏离车道时方向盘震动并修正。
3. **主动紧急制动**
   - >7km/h时工作，可检测车辆和行人。灵敏度在中控屏“驾驶辅助”中调节。
4. **盲区监测**
   - >12km/h激活，后视镜内侧亮灯提示盲区有车。
5. **自动泊车**
   - <20km/h时按下中控台P键选择泊车类型。按屏幕提示控制刹车和挡位即可。

## 五、灯光与雨刮
### Smartbeam LED大灯
- AUTO档位实现远近光自动切换。手动模式下：
    - O档关闭 / AUTO自动 / ⭕示宽灯 / ⭕近光灯
    - ⭕远光灯需先开近光再向前推拨杆
### Follow Me Home伴我回家功能
熄火后抬一下远光拨杆即可激活30秒延时照明。
### Magic Wash雨刮清洗系统
雨刮器喷水口集成在雨刮臂上。向上拉拨杆前窗喷水清洗；向下按压后窗喷水清洗。间歇档位可通过滚轮调节频率。

## 六、座椅与空间
### AGR认证座椅调节
电动座椅支持：
1.坐垫前后/高低 /2靠背角度 /3腰部支撑四向调节 /4按摩功能按键在座椅侧面
### Grand Lounge后排空间设计
第二排座椅可前后滑动160mm。第三排座椅可完全收纳至地板下：
1.拉动靠背释放拉环折叠靠背；
2.拉动底部拉环翻转坐垫；
3.整体推入地板凹槽固定；
第三排座椅也可单独拆卸作为户外临时座位使用（单张约11kg）。

## 七、实用小技巧
1.**油箱盖开启**:车辆解锁状态下按压油箱盖即可打开；加注92号及以上汽油即可满足要求；
2.**后备箱感应开启**:携带钥匙站在车尾保险杠下方踢脚动作即可感应开启后备箱；
3.**儿童安全锁**:后排车门侧面机械式儿童锁旋钮可用钥匙旋转锁定防止内部开门；
4.**保养重置**:熄火状态下长按仪表盘右侧黑色柱状按钮通电直到保养提示闪烁松开再短按确认即可重置保养里程；
5.**紧急呼叫**:车内天窗控制区域SOS按键可在紧急情况下联系标致客服中心获取救援协助；
6.**拖车模式**:需要拖车时在中控屏车辆设置中开启拖车模式避免电子手刹误触发损坏变速箱；
7.**冬季预热**:建议怠速30秒左右低速行驶预热发动机无需长时间原地热车浪费燃油且容易产生积碳；
8.**ECO模式**:中控台ECO按键可进入节能模式降低空调功率油门响应变缓适合城市拥堵路段节省燃油消耗约10%左右；

# 广汽本田缤智（VEZEL）操作指南
  
更新时间：2026年07月12日08时14分20秒   

## 一、驾驶操作
### 1. 启动与熄火
- **启动**：踩住刹车踏板，按下一键启动按钮（配备智能钥匙系统）；若钥匙电量不足，将钥匙贴近启动按钮再启动。
- **熄火**：车辆停稳后，挂入P挡，拉起电子手刹，按下启动按钮熄火。
- **注意事项**：启动前确认挡位在P挡或N挡；熄火后关闭所有用电设备。
### 2. 换挡操作
- **P（驻车挡）**：停车时使用，必须车辆完全停止后挂入。
- **R（倒车挡）**：倒车时使用，车辆停稳后挂入。
- **N（空挡）**：临时停车（如等红灯）或拖车时使用。
- **D（前进挡）**：日常行驶使用。
- **S（运动模式）**：需要更强动力输出时使用，如超车、爬坡。
- **L（低速挡）**：长下坡或陡坡时使用，利用发动机制动。
### 3. 电子手刹与自动驻车（BRAKE HOLD）
- **电子手刹**：向上拉起为锁止，向下按压为释放。系好安全带并挂D挡后，轻踩油门可自动解除。
- **自动驻车（BRAKE HOLD）**：按下BRAKE HOLD按钮（仪表盘指示灯亮起），等红灯时踩刹车至车辆停稳，松开刹车即可保持制动，轻踩油门自动解除。
### 4. 方向盘调节
- 方向盘下方有锁止手柄，向下拉开后可调节方向盘上下、前后位置，调整至舒适位置后推回锁止。

## 二、灯光操作
### 1. 灯光控制杆（方向盘左侧）
| 操作 | 功能 |
|------|------|
| 旋钮转至 AUTO | 自动大灯（根据光线自动开关） |
| 旋钮转至 示宽灯 | 示宽灯/尾灯亮起 |
| 旋钮转至 近光灯 | 近光灯常亮 |
| 控制杆向前推 | 远光灯常亮（蓝色图标亮起） |
| 控制杆向后拉 | 远光灯闪一次（超车示意） |
| 控制杆上下拨 | 左右转向灯 |
### 2. 前后雾灯
- 灯光旋钮向外拉：第一段为前雾灯，第二段为后雾灯。
### 3. 车内灯光
- 车顶灯开关：ON为常亮，DOOR为开门亮起，OFF为关闭。
---
## 三、雨刮器操作
### 1. 前雨刮器（右侧控制杆）
- **MIST**：向上推一次，单次刮扫。
- **OFF**：关闭。
- **INT**：间歇刮扫（可调节旋钮调整间隔）。
- **LO**：低速连续刮扫。
- **HI**：高速连续刮扫。
- **喷水清洗**：控制杆向驾驶员方向拉动。
### 2. 后雨刮器
- 控制杆末端旋钮旋转至相应位置开启后雨刮或喷水。
---
## 四、空调与多媒体系统
### 1. 空调控制
- **A/C按钮**：开启/关闭压缩机（制冷）。
- **温度旋钮**：蓝色为冷风，红色为暖风。
- **风量旋钮**：调节风扇风速。
- **模式选择**：吹脸、吹脚、吹前挡风玻璃等。
- **内外循环**：内循环为车内空气循环，外循环引入外界空气。堵车或空气质量差时建议使用内循环。
- **后窗除雾**：按下后窗除雾按钮（REAR），同时可加热外后视镜。
### 2. 多媒体系统
- **音量调节**：旋钮或方向盘右侧音量键。
- **蓝牙连接**：进入设置 → 蓝牙 → 配对设备 → 手机搜索并连接。
- **手机互联**：支持Apple CarPlay/百度CarLife（视车型年份而定），通过USB线连接手机后自动识别。
- **USB接口**：中央扶手箱内或中控台下方，支持U盘播放音乐及充电。

## 五、安全与辅助系统
### 1. 倒车影像
- 挂入R挡时，中控屏自动显示后方影像，随方向盘转动显示辅助轨迹线。
### 2. 车身稳定系统（VSA）
- 默认开启，仪表盘上VSA OFF灯熄灭表示正常工作。长按VSA OFF按钮可关闭（不推荐日常关闭）。
### 3. 胎压监测系统
- 胎压异常时仪表盘亮起胎压报警灯。调整胎压后，需在设置中重置胎压监测系统。
### 4. ECO节能模式
- 按下ECO按钮，仪表盘亮起绿色ECO指示灯，系统自动优化油门响应和空调输出以节省燃油。

## 六、保养与日常检查
### 1. 保养提示复位
- 保养到期后仪表盘显示扳手图标。复位方法：通电状态下，按下仪表盘上的TRIP按钮切换至保养界面，长按RESET按钮复位。
### 2. 油箱盖与加油
- 油箱盖开关在驾驶座左下方地板处（带加油图标），向上拉起即可。推荐使用92号及以上无铅汽油。
### 3. 发动机舱检查
- 打开发动机盖：拉动驾驶座左下方的发动机盖拉手，再拨动前格栅中间的锁扣即可抬起。
- 定期检查：机油尺（冷车时检查）、刹车油、冷却液、玻璃水。
### 4. 轮胎
- 标准胎压：通常为前轮2.2~2.4 bar，后轮2.2~2.3 bar（以驾驶座门框标签为准）。
- 备胎位于后备箱地板下方，使用随车千斤顶和扳手更换。
---
## 七、常见问题与提示
| 问题 | 解决方法 |
|------|---------|
| 钥匙遥控失灵 | 检查钥匙电池是否耗尽，或周围是否有强电磁干扰 |
| 启动时仪表盘亮多个故障灯 | 可能是电瓶亏电，尝试搭电启动 |
| 空调制冷效果差 | 检查是否开启A/C，切换至内循环，必要时检查制冷剂 |
| 雨刮刮不干净 | 更换雨刮胶条，或清洁挡风玻璃油膜 |
---
## 八、应急操作
- **拖车模式**：挂入N挡，松开电子手刹。
- **儿童锁**：后门侧面有儿童锁开关，拨至LOCK位置后车门无法从车内打开。
- **紧急双闪灯**：中控台红色三角按钮，按下后所有转向灯闪烁。
---
> **提示**：以上内容适用于广汽本田缤智多数年款，具体功能可能因车型配置及年份略有差异，请以随车用户手册为准。

# 启腾N50EV 内饰参数介绍
  
更新时间：2026年07月19日18时22分00秒   

## 整体设计风格
启腾N50EV内饰采用简约实用设计风格，以深色系为主色调，注重功能性与耐用性，符合新能源物流车的定位需求。
---
## 座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 织物面料（耐磨、易清洁） |
| 座椅布局 | 2座（驾驶席 + 副驾驶席） |
| 驾驶席调节 | 前后滑动 + 靠背角度手动调节 |
| 副驾驶席调节 | 靠背角度手动调节 |
| 座椅填充材料 | PU发泡海绵，支撑性良好 |
---
## 仪表台与中控
| 项目 | 参数 |
|------|------|
| **仪表盘** | |
| - 类型 | LCD液晶组合仪表 |
| - 显示内容 | 车速、电量、续航里程、电机转速、故障提示等 |
| **中控台** | |
| - CD/DVD播放器 | —（无） |
| - USB接口 | ●（支持U盘音乐播放及手机充电） |
| - AUX接口 | ●（选配） |
| - FM/AM收音机 | ●（标配） |
---
## 方向盘
- **类型**：三幅式方向盘
- **材质**：PVC塑料
- **功能按键**：无多功能按键（基础款）
- **调节方式**：不可调节（固定式）
---
## 空调系统
- **空调类型**：手动空调
- **制冷/制热**：冷暖两用
- **出风口布局**：中央出风口 ×2 + 两侧出风口 ×2
- **后舱出风口**：无独立后排出风口
---
##  储物空间
- **手套箱**：副驾前方开放式储物格
- **门板储物槽**：左右车门各设杯架及储物槽
- **中控台储物格**：换挡杆前方小型储物格
- **顶棚储物架**：部分车型可选装顶棚置物架
---
##  其他内饰配置
###  灯光系统
- **车内照明灯**：前排顶灯（手动开关）
- **阅读灯**：无独立阅读灯
###  车窗与后视镜
- **前门电动车窗** ●（主驾一键下降）
- **后门车窗形式** —（封闭式盲窗/货箱结构）
- **外后视镜调节方式**：手动调节
- **内后视镜防眩目功能** ●（手动防眩目）
###  安全与便利配置
- ISO-FIX儿童座椅接口 —（无，仅2座布局）
-  安全带未系提醒 ●（主驾位）
-  车内中控锁 ●
-  遥控钥匙 ●
---
##  货厢与后排区域说明
> ⚠️ N50EV为纯电动微面车型，采用“前置驾驶舱 + 后部封闭货厢”结构。
> 
> - **后排乘客座位数**: ❌ （无后排座位，仅驾驶舱2座）
> - **货厢地板材质**: PVC防滑地板或金属波纹板
> - **货厢内高**: ≥1200mm （具体因批次略有差异）
> - **货厢内宽**: ≥1300mm （具体因批次略有差异）

# 奥迪 RS 4 动力参数介绍
  
更新时间：2026年07月19日16时21分48秒   

## 发动机规格
| 项目 | 参数 |
|------|------|
| **发动机类型** | 2.9L V6 双涡轮增压（TFSI） |
| **排量** | 2,894 cc |
| **气缸数** | V型6缸 |
| **进气形式** | 双涡轮增压 + 电动泄压阀 |
| **燃油喷射方式** | 缸内直喷（FSI） |
## 性能数据
| 项目 | 参数 |
|------|------|
| **最大功率** | **331 kW（450 PS / 444 hp）** |
| **最大功率转速** | 5,700 – 6,700 rpm |
| **最大扭矩** | **600 N·m** |
| **最大扭矩转速** | 1,900 – 5,000 rpm |
| **最高转速** | 7,200 rpm |
## 传动与驱动系统
- **变速箱**：8速 Tiptronic（手自一体）
- **驱动方式**：quattro®全时四驱系统
- **中央差速器**：机械式自锁差速器（默认前后40:60分配）
## 加速与极速
- **0–100 km/h加速时间**：约 **4.1秒**
- **0–200 km/h加速时间**：约 **14.5秒**
- **最高时速（电子限速）**：250 km/h
- *可选装动态增强包解除限速至*：280 km/h
## RS专属调校亮点
1. 🔧 **双涡流涡轮增压器**
   - V型气缸夹角内侧布置，响应更迅速
   - 峰值扭矩平台宽广（1,900–5,000 rpm）
2. ⚙️ **48V轻度混合动力系统**
   - BSG皮带启动发电机
   - 支持滑行模式，提升启停平顺性及燃油经济性
3. 🎯 **RS运动排气系统**
   - 可变阀门控制，提供低沉浑厚的声浪
   - “RS1”和“RS2”两种驾驶模式可独立设定排气特性
4. 🛞 **动态驾驶模式选择**
   - comfort / auto / dynamic / RS1 / RS2 / individual
   - RS模式下发动机响应、换挡逻辑、悬架硬度同步调整
## 📊 动力输出曲线特点
> - ✅ 低扭充沛：1,900 rpm即可输出600 N·m峰值扭矩
> - ✅ 中段爆发力强：3,000–5,500 rpm区间持续高功率输出
> - ✅ 高转延伸性好：功率峰值可持续至6,700 rpm，断油转速7,200 rpm

# Jeep 牧马人 动力参数介绍
  
更新时间：2026年07月13日13时39分16秒   

## 发动机
| 项目 | 参数 |
|------|------|
| **发动机类型** | 2.0T 直列四缸涡轮增压发动机 |
| **排量** | 1995 mL |
| **进气形式** | 涡轮增压 |
| **气缸排列形式** | L型（直列） |
| **气缸数** | 4缸 |
| **每缸气门数** | 4个 |
| **配气机构** | DOHC（双顶置凸轮轴） |
| **燃油标号** | 95号及以上无铅汽油 |
| **供油方式** | 缸内直喷 |
## 性能参数
| 项目 | 参数 |
|------|------|
| **最大马力** | 266 Ps（约195.5 kW） |
| **最大功率** | 195.4 kW / 5000 rpm |
| **最大扭矩** | 400 N·m / 3000 rpm |
| **最高车速** | ≈180 km/h（电子限速） |
## 变速箱
- **变速箱类型**：8挡手自一体变速箱（8速自动变速箱）
- **挡位个数**：8个前进挡 + R挡
- **换挡方式**：电子换挡 +方向盘换挡拨片
## 底盘与驱动系统
### 驱动方式
- Sahara / Sport版：分时四驱（Part-time 4WD）
- Rubicon版：分时四驱 + Rock-Trac®分动箱
### 悬挂系统
- **前悬挂类型**:多连杆整体桥式非独立悬挂
- **后悬挂类型**:多连杆整体桥式非独立悬挂
- **车体结构**:非承载式车身（带大梁）
### 转向系统
- EPS电动助力转向系统
### 制动系统
- **前制动器**:通风盘式
- **后制动器**:盘式/通风盘式（视配置而定）
## 越野相关动力参数
### 四驱模式切换
- Command-Trac®分时四驱系统（Sahara/Sport）
    -2H（两驱高速）
    -4H AUTO（自动四驱高速）
    -4H PART TIME（分时四驱高速）
    -4L（低速四驱，扭矩放大约2.72倍）
-Rock-Trac®分时四驱系统（Rubicon）
    -2H、4H、N、4L
    -低速四驱扭矩放大倍数：约4:1
### 差速锁与限滑功能
-Rubicon标配前后桥电控差速锁
-Sahara/Sport可选装后桥差速锁或限滑差速器

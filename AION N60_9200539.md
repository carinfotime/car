# 广汽埃安 AION N60 驾驶注意事项
  
更新时间：2026年07月18日10时01分05秒   

## 一、启动与行驶前检查
1.  **钥匙与系统**：确保智能钥匙在车内，踩下制动踏板并按下“启动/停止”键启动车辆。确认仪表盘上“READY”指示灯亮起。
2.  **电量检查**：出发前确认剩余续航里程是否满足当日出行需求。建议长途出行前将电量充至80%-90%。
3.  **轮胎检查**：定期检查胎压（建议冷车胎压保持在2.5-2.7 bar），确保轮胎无鼓包、裂纹或异常磨损。
4.  **灯光与信号**：检查前后大灯、转向灯、刹车灯及雾灯是否正常工作。
## 二、驾驶操作规范
1.  **起步与加速**：
    -   电动车扭矩响应快，轻踩加速踏板即可获得平顺起步。
    -   避免“地板油”急加速，这会导致瞬间大电流放电，增加能耗并可能触发牵引力控制系统介入。
2.  **能量回收**：
    -   N60通常配备多级能量回收（如I-Pedal/单踏板模式或低/高回收）。
    -   **单踏板模式（I-Pedal）**：松开加速踏板即产生较强制动效果，适合城市拥堵路况。需适应其减速特性，紧急情况仍需踩刹车。
    -   **低回收模式**：滑行更接近燃油车，适合高速巡航或乘客易晕车时使用。
3.  **制动技巧**：
    -   提前预判路况，利用能量回收减速（松开电门），可有效回收电能并减少刹车片磨损。
    -   紧急情况果断深踩刹车，ABS和电子稳定系统会介入工作。
## 三、特殊路况驾驶
1.  **雨天/湿滑路面**：
    -   EV车辆自重较大且轮胎抓地力要求高，雨天应降低车速，避免急转向或急加速导致打滑。
    -   涉水深度请勿超过轮毂中心（约30cm），电池组虽防水但接口需保护。
2.  **冬季低温行驶**：
    -   **电池预热**：若车辆支持电池预热功能，建议在充电时开启或出发前使用APP预约预热，可改善续航表现并保护电池。
    -   **暖风使用**：PTC加热器耗电较高。建议合理设置空调温度（20-22℃），配合座椅加热使用更节能。
3.  **高速行驶**：
    -   N60经济时速约为60-90km/h。高速行驶（>100km/h）风阻剧增，电耗会显著上升（可能增加30%-50%）。
    -   ACC自适应巡航（如配备）在高速上可减轻疲劳，但需注意跟车距离设定。
## 四、充电注意事项
1.  **充电桩选择**：
    -   N60支持快充和慢充。日常通勤建议使用家用慢充桩（7kW），对电池寿命更友好。
    -  长途快充时尽量选择国标直流快充桩（功率≥60kW），SOC从30%充至80%通常约需30-40分钟。
2.  **充电安全**：
    -  充电前检查充电枪头是否干燥、无异物。雷雨天气尽量避免户外露天快充。
3.  **电量管理原则**：
    -  避免将电量耗尽至0%（亏电状态），这会严重损伤三元锂电池寿命。
    -  长期停放时保持电量在40%-60%，每月至少进行一次完整充放电循环以校准BMS。
## 五、安全与维护提醒
1.  **高压安全警告**
    严禁私自拆卸或改装高压部件（橙色线束）。发生碰撞后若闻到异味或看到烟雾，立即远离车辆并联系专业人员处理。
2.  **保养周期**
    纯电动车无需更换机油机滤。主要保养项目包括：
|项目|周期|
|---|---|
|空调滤芯|每12个月或10000公里|
|减速器齿轮油|每40000公里|
|冷却液检查|每20000公里|
|制动液更换|每24个月|
3.  **驻车与锁车**
    挂P挡后拉好电子手刹再下车。离开前确认车窗、天窗已关闭。长时间停放建议关闭所有用电设备。
4.  **智能辅助系统**
    请勿过度依赖L2级辅助驾驶功能（如车道保持）。驾驶员必须始终手握方向盘并对车辆拥有最终控制权。

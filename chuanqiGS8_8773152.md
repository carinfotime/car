# 传祺GS8 内饰参数介绍
  
更新时间：2026年07月11日15时39分46秒   

## 一、座椅配置
| 项目 | 参数 |
|------|------|
| **座椅材质** | 真皮/仿皮（根据配置不同） |
| **座椅布局** | 2+3+2（7座）/ 2+2+2（6座可选） |
| **前排座椅功能** | 电动调节（主驾8向/副驾4向）、加热、通风、记忆功能（高配） |
| **第二排座椅** | 前后滑动 + 靠背角度调节，支持4/6比例放倒 |
| **第三排座椅** | 可折叠放平，支持5/5比例放倒 |
## 二、仪表盘与中控屏
| 项目 | 参数 |
|------|------|
| **仪表盘** | 12.3英寸全液晶仪表盘 |
| **中控屏** | 14.6英寸高清触控屏（部分配置为12.3英寸） |
| **分辨率** | 1920×1080（中控屏） |
| **车机系统** | ADiGO智驾互联生态系统 |
## 三、方向盘
- **材质**：真皮包裹
- **调节方式**：手动上下+前后调节
- **多功能按键**：集成多媒体控制、定速巡航/自适应巡航控制、语音控制等
- **加热功能**：高配车型配备方向盘加热
## 四、空调系统
- **类型**：自动分区空调
- **分区数**：三区独立控制（驾驶位、副驾驶位、后排）
- **后排独立出风口**
- **PM2.5过滤装置**
- **负离子空气净化系统**
## 五、音响系统
| 项目 | 参数 |
|------|------|
| **扬声器数量** | 6喇叭 /10喇叭（高配Alpine阿尔派音响） |
| **品牌音响（高配）** | Alpine阿尔派 |
## 六、内饰颜色与材质
### 🔹内饰颜色选项
-  黑色内饰
-  棕色内饰
-  米色内饰（部分配置）
### 🔹装饰面板
-  木纹饰板（高配）
-  金属拉丝饰板
-  钢琴烤漆面板
## 七、其他舒适配置
### 🌟灯光系统
-  车内氛围灯：多色可调氛围灯（32色）
-  阅读灯：前后排独立LED阅读灯
### 🌟储物空间
-  中央扶手箱容积较大，支持冷藏功能
-  门板储物格可容纳水瓶
-  杯架数量：前排2个 +后排2个 +第三排1个
### 🌟便利配置
-  手机无线充电（前排）
- USB接口数量：
    - Type-C ×2（前排）
    - USB ×2 （后排）
    - Type-C ×1 （第三排）

后备箱容积      176L (7座满员) / ~880L (放倒第三排)

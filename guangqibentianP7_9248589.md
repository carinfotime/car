# 广汽本田P7 技术参数介绍
  
更新时间：2026年07月16日12时07分26秒   

## 一、车型概述
广汽本田P7是广汽本田基于全新纯电专属平台打造的首款纯电动SUV车型，定位为中型纯电SUV，采用“e:N Architecture”智能高效纯电架构，代表了本田在电动化领域的最新成果。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **驱动形式** | 后驱 / 四驱 |
| **电机类型** | 永磁同步电机 |
| **后驱版最大功率** | 约200kW（272Ps） |
| **四驱版最大功率** | 前150kW + 后200kW（综合约350kW / 476Ps） |
| **峰值扭矩（后驱版）** | 约420N·m |
| **峰值扭矩（四驱版）** | 前310N·m + 后420N·m（综合约730N·m） |
| **0-100km/h加速时间（四驱版）** | ≤4.6秒 |
| **最高车速** | ≥200km/h |

## 五、底盘与悬架
-   **前悬架类型：**麦弗逊式独立悬架
-   **后悬架类型：**多连杆式独立悬架
-   **转向系统：**EPS电动助力转向
-   **制动系统：**前通风盘 / 后实心盘 + IPB智能集成制动系统
-   **轮胎规格：**

    -   高配：255/45 R20

-   语音控制：自然语义识别 + AI语音助手
-   手机互联：Apple CarPlay / Baidu CarLife / HiCar

## 七、智能驾驶辅助 —— Honda SENSING Elite+
### 🛡️ 感知硬件
-   1个前置单目摄像头 （800万像素）
-   5个毫米波雷达 （1前 +4角）
-   12个超声波雷达

3.  交通拥堵辅助 （TJA）
4.  自动变道辅助 （ALC）
5.  自动泊车入位 &遥控泊车 （RPA/APA）
6.  前方碰撞预警 &自动紧急制动 （FCW/AEB）
7.  盲区监测 &后方交通穿行警示 （BSI/RCTA）
> *注：部分高阶功能需通过OTA后续开通。*

## ⚡八、充电与补能体系
### 🔌 充电接口标准：
国标直流快充 +交流慢充双接口。
### ⚡ 对外放电：
V2L对外放电功能，最大输出功率3.3kW，支持户外用电场景。
### 🏪 充电网络：
广汽本田联合特来电/星星充电等运营商提供专属充电权益及接入第三方超充站。

## ✅九、安全车身结构
### 🔩 车身刚性：
采用高强度钢铝混合车身结构，超高强度钢占比≥35%，热成型钢占比≥15%。
### 🧪 电池安全：
“Honda Battery Protection”多层防护结构：
1. IP67级防尘防水电池包；
2. ±30mm碰撞吸能空间；
3. BMS全天候监控热失控预警；
4. “弹匣”级隔热阻燃材料填充。

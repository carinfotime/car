# 雷驰·仁T30 技术参数介绍
  
更新时间：2026年07月20日10时27分45秒   

## 一、车型概览
**雷驰·仁T30** 是一款面向城市物流与短途运输场景打造的新能源微卡，兼具经济性、承载能力与灵活操控。该车型采用纯电驱动，旨在为用户提供低成本、高效率的运输解决方案。
## 二、核心三电系统
| 项目 | 参数/规格 |
| :--- | :--- |
| **电机类型** | 永磁同步电机 |
| **额定功率** | 30 kW |
| **峰值功率** | 60 kW |
| **峰值扭矩** | 220 N·m |
| **电池类型** | 磷酸铁锂电池 |
| **电池容量** | 41.86 kWh |
| **续航里程 (CLTC)** | ≥ 260 km |
## 三、车身尺寸与货箱
| 项目 | 参数/规格 |
| :--- | :--- |
| **整车尺寸 (长×宽×高)** | 4870 mm × 1635 mm × 2010 mm |
| **轴距** | 3050 mm |
| **货箱尺寸 (长×宽×高)** | 3010 mm × 1540 mm × 370 mm |
| **货箱容积** | ≈6.8 m³ (含栏板高度) |
## 四、底盘与承载
-   **驱动方式**：中置后驱
-   **前悬架类型**：麦弗逊式独立悬架
-   **后悬架类型**：钢板弹簧非独立悬架
-   **轮胎规格**：175/75R14LT
-   **额定载质量**：约1.2吨
-   **最大总质量 (GVW)**：约2.5吨
## 五、充电与补能
-   **充电口类型**
    -   直流快充：国标GB/T
    -   交流慢充：国标GB/T
-   **快充时间 (SOC:20%-80%)**
    -   约40分钟
-   **慢充时间 (SOC:0%-100%)**
    -   约8小时（使用6.6kW车载充电机）
## 六、智能与安全配置
-   ABS防抱死制动系统 + EBD制动力分配
-   倒车雷达（标配）
-   遥控钥匙 +中控锁
-   液晶仪表盘（显示续航、电量、故障信息）
-   电动助力转向（EPS）
## 七、适用场景
适合城市内快递配送、商超进货、家具家电运输及个体商户自用等场景。凭借其紧凑的车身和灵活的转向，可轻松进出老旧小区及狭窄街道。

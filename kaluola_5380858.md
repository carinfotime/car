# 丰田卡罗拉 - 儿童乘车安全注意事项
  
更新时间：2026年07月14日17时25分26秒   

## 一、安全座椅使用规范
### 1. 年龄与座椅类型
- **婴儿（0-1岁，体重<13kg）**：必须使用**反向安装**的婴儿提篮或儿童安全座椅。
- **幼儿（1-4岁，体重9-18kg）**：建议继续反向安装，或使用正向安装的儿童安全座椅。
- **学龄前儿童（4-8岁，体重15-36kg）**：使用带有五点式安全带的前向式安全座椅或增高垫。
- **大龄儿童（8-12岁，身高<150cm）**：使用增高垫，确保车载安全带正确穿过肩部和胯部。
### 2. ISOFIX接口
- 卡罗拉后排座椅配备有 **ISOFIX** 儿童安全座椅固定接口。
- 安装时请务必听到“咔哒”声并确认锁定指示灯变绿（如有）。
- 同时使用顶部系带（Top Tether）固定座椅上部。
## 二、乘坐位置要求
### ✅ 最佳位置：后排中间座位
> 远离两侧车门和侧面碰撞区域，且前方无手套箱等硬物。
### ✅ 次选位置：后排两侧座位
> 必须配合 ISOFIX + Top Tether 固定。
### ❌ **严禁将儿童安全座椅安装在前排副驾驶位**
> - 副驾驶位前方有安全气囊。
> - 一旦发生碰撞弹出气囊，会对儿童造成严重伤害甚至致命风险。
> - *特殊情况*：如果必须在前排安装反向座椅（如双排满载），必须先关闭副驾驶安全气囊（请查阅车辆说明书操作）。
## 三、安全带使用提醒
1. **不要让孩子直接使用成人安全带** ——身高不足150cm时容易勒到脖子或腹部。
2. **不要让安全带从腋下穿过** ——会失去保护作用并导致内脏受伤。
3. **不要让安全带压在坚硬物品上** （如手机、玩具）。
4. **确保安全带贴紧身体** ，没有扭曲或松弛。
## 四、车窗与车门锁
| 🔒 | ⚠️ |
|---|---|
| **后门儿童安全锁** | ✅ **务必开启！** |
| | - 开启后车门只能从外部打开。 |
| | -防止行驶中孩子误开车门。 |
| | -位置在后门门锁附近（拨动开关）。 |
| **电动车窗防夹功能** | ⚠️部分卡罗拉车型配备。 |
| | -但仍需注意不要让孩子单独操作车窗。 |
## 五、车内其他注意事项
### 🧸  玩具与杂物
- ❌不要放置尖锐、硬质玩具在座位上或脚下。
- ❌不要在遮阳板或后窗处挂过多装饰物——急刹车时可能飞出伤人。
### 🌡️  温度控制
- ❌切勿将孩子单独留在车内——夏季车内温度可在10分钟内升至50°C以上。
- ❌即使开着空调也不建议留孩子在车内独自等待。
### 🍽️  饮食
- ❌行驶中不要给孩子吃棒棒糖、果冻、坚果等易噎住的食物。
- ✅如需喂食请停车进行。
## 📋  日常检查清单
每次出行前快速检查：
1. [ ]  后排ISOFIX连接是否牢固？
2. [ ]  顶部系带是否拉紧？
3. [ ]  后门儿童锁是否开启？
4. [ ]  车窗升降权限是否已限制？
5. [ ]  前排副驾气囊状态是否正确？（若前排有反向座椅则必须关闭）
6. [ ]  车内是否有松散危险物品？

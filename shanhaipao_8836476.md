# 长城炮·山海炮 操作指南
  
更新时间：2026年07月11日08时14分21秒   

## 一、车辆基本介绍
山海炮是长城汽车旗下高端豪华皮卡，搭载3.0T V6涡轮增压发动机或2.4T柴油发动机，匹配9速自动变速箱，配备博格华纳智能四驱系统。本指南涵盖日常操作、驾驶模式、安全功能及保养维护等内容。
---
## 二、启动与熄火
### 2.1 启动车辆
1. 智能钥匙必须在车内（或携带钥匙进入车辆）。
2. 踩下制动踏板，档位处于 **P档** 或 **N档**。
3. 按下 **一键启动按钮**（位于方向盘右侧）。
4. 仪表盘自检完成后，发动机启动。
5. 启动后怠速运行约 **30秒**，让润滑油充分循环。
### 2.2 熄火
1. 踩下制动踏板，将档位挂入 **P档**。
2. 拉紧电子手刹（EPB）。
3. 再次按下 **一键启动按钮**，发动机熄火。
4. 熄火后建议等待 **10-15秒** 再关闭电源，让涡轮增压器冷却（特别是长途或激烈驾驶后）。
> **注意**：熄火前请确保所有车窗、天窗、灯光已关闭。
---
## 三、档位操作
山海炮采用电子换挡机构，档位排列如下：
| 档位 | 说明 |
|------|------|
| **P** | 驻车档，停车时使用，变速箱锁止 |
| **R** | 倒车档，倒车时使用 |
| **N** | 空档，短暂停车或拖车时使用 |
| **D** | 前进档，日常行驶 |
| **M** | 手动模式，可拨动换挡拨片手动加减档 |
### 换挡操作
- 踩下制动踏板，按下换挡杆侧面的 **解锁按钮** 即可换挡。
- 从 **D档向右推** 进入手动模式（M档），使用方向盘换挡拨片操作。
- 从 **D档向下拉** 可切换至运动模式（S档）。
---
## 四、驾驶模式选择
山海炮提供 **多种驾驶模式**，通过中控台驾驶模式旋钮切换：
| 模式 | 适用场景 | 说明 |
|------|----------|------|
| **经济模式（ECO）** | 城市通勤、高速巡航 | 降低油门响应，节省燃油 |
| **标准模式（NORMAL）** | 日常综合路况 | 平衡动力与油耗 |
| **运动模式（SPORT）** | 超车、山路、激烈驾驶 | 提升油门响应，延迟换挡 |
| **泥地模式** | 泥泞、湿滑路面 | 控制扭矩输出，防止打滑 |
| **沙地模式** | 沙漠、沙滩 | 保持高转速，防止陷车 |
| **岩石模式** | 崎岖山地、攀爬 | 低速高扭矩，差速锁介入 |
| **雪地模式** | 冰雪路面 | 柔和动力输出，防止侧滑 |
> **提示**：切换模式时，建议车速低于 **40km/h**，四驱系统会自动调整。
---
## 五、四驱系统操作
山海炮配备 **博格华纳智能四驱系统**，操作面板位于中控台：
### 5.1 四驱模式切换
| 模式 | 用途 | 切换方式 |
|------|------|----------|
| **2H**（高速两驱） | 铺装路面日常行驶 | 默认模式 |
| **4H**（高速四驱） | 雨雪、沙石、轻度越野 | 车速低于80km/h可切换 |
| **4L**（低速四驱） | 重度越野、陡坡攀爬 | 停车挂N档后切换 |
### 5.2 前后桥差速锁
- **后桥差速锁**：在4L模式下，按下差速锁按键启用。
- **前桥差速锁**：需先启用后桥差速锁，再按下前桥差速锁按键。
- 差速锁仅在 **低速越野** 时使用，铺装路面严禁开启。
> **警告**：差速锁开启后，方向盘转向会变重，请勿高速行驶。
---
## 六、智能辅助驾驶系统
### 6.1 自适应巡航（ACC）
1. 按下方向盘左侧 **巡航开关按键**。
2. 仪表盘显示巡航图标后，按下 **SET/-键** 设定当前速度。
3. 使用 **RES/+** 加速，**SET/-** 减速。
4. 按下 **CANCEL** 或踩刹车取消巡航。
### 6.2 车道保持辅助（LKA）
- 通过中控屏车辆设置中开启/关闭。
- 车速超过 **60km/h** 自动激活。
- 当车辆偏离车道时，系统会轻微修正方向盘。
### 6.3 360°全景影像
- 挂入 **R档** 自动激活。
- 或按下中控台 **360影像按键** 手动开启。
- 可切换视角：前视、后视、俯视、车轮视角。
---
## 七、货箱操作
### 7.1 货箱开启
1. 拉动货箱后挡板把手。
2. 向下打开后挡板（部分车型配备阻尼缓降）。
3. 关闭时向上推至锁止位置，确认卡扣锁紧。
### 7.2 货箱盖（如配备）
- 手动卷帘盖：拉动拉环，向车头方向推。
- 电动货箱盖：按下遥控钥匙或车内按键控制。
### 7.3 货箱锚点
- 货箱内部配备 **多个锚点**，用于固定货物。
- 建议使用捆绑带固定货物，防止行驶中移位。
> **注意**：货箱最大载重请参阅车辆铭牌，切勿超载。
---
## 八、拖车功能
山海炮具备 **拖车资质**，操作如下：
1. 安装拖车钩（后保险杠预留接口）。
2. 连接拖车电源插头（7针或13针接口）。
3. 在中控屏车辆设置中开启 **拖车模式**。
4. 拖车模式下，变速箱换挡逻辑自动优化，ESP系统调整。
> **注意**：拖车总质量不得超过车辆标定值（请查阅用户手册）。
---
## 九、常见功能操作
### 9.1 电子手刹（EPB）
- 自动释放：挂入D档，轻踩油门，手刹自动解除。
- 手动释放：踩刹车，按下手刹开关。
- 紧急制动：行驶中长拉手刹开关可触发紧急制动。
### 9.2 自动驻车（Auto Hold）
- 按下中控台 **AUTO HOLD** 按键激活。
- 车辆停稳后自动保持制动，轻踩油门解除。
### 9.3 陡坡缓降（HDC）
- 按下中控台陡坡缓降按键。
- 车速自动控制在 **5-30km/h** 之间。
- 可通过油门或刹车调节速度。
### 9.4 坦克转弯
- 在4L模式下，按下坦克转弯按键。
- 系统对内侧后轮施加制动，减小转弯半径。
- 适用于狭窄山路、林道掉头。

## 十、仪表盘与警告灯
| 指示灯 | 含义 | 处理方式 |
|--------|------|----------|
| 🔴 红色电池灯 | 充电系统故障 | 尽快检查发电机/电瓶 |
| 🔴 红色机油灯 | 机油压力过低 | 立即停车检查机油 |
| 🟡 黄色发动机灯 | 发动机系统故障 | 尽快前往服务站检修 |
| 🟡 黄色ESP灯 | 车身稳定系统关闭或故障 | 检查ESP开关状态 |
| 🟢 绿色巡航灯 | 巡航系统激活 | 正常 |
| 🔵 蓝色远光灯 | 远光灯开启 | 对向有车时关闭 |

## 十一、保养与维护
### 11.1 保养周期
| 项目 | 周期 |
|------|------|
| 发动机机油 & 机滤 | 每 **10,000公里** 或 **12个月** |
| 空气滤清器 | 每 **20,000公里** |
| 燃油滤清器 | 每 **30,000公里** |
| 变速箱油 | 每 **60,000公里** |
| 分动箱油 & 差速器油 | 每 **40,000公里** |
| 制动液 | 每 **2年** 或 **40,000公里** |
| 冷却液 | 每 **4年** 或 **80,000公里** |
### 11.2 轮胎
- 标准胎压：**2.4-2.6 bar**（冷胎状态）
- 越野时建议降低至 **1.5-2.0 bar**
- 每 **10,000公里** 进行轮胎换位
### 11.3 尿素添加（柴油版）
- 柴油版车型需添加 **车用尿素（AdBlue）**。
- 尿素加注口位于油箱盖旁边。
- 仪表盘提示尿素液位低时，请及时添加。

## 十二、安全注意事项
1. **安全带**：全车人员必须系好安全带，后排同样重要。
2. **儿童安全锁**：后门配备儿童安全锁，防止儿童在车内开门。
3. **货物固定**：货箱货物必须牢固固定，防止急刹车时货物前冲。
4. **涉水注意**：山海炮最大涉水深度约 **800mm**，涉水后请检查差速器油。
5. **改装须知**：改装前后保险杠、升高底盘等需合法备案。
6. **拖车安全**：拖车时务必连接安全钢丝绳。
---
## 十三、紧急情况处理
### 13.1 轮胎漏气/爆胎
1. 握紧方向盘，保持直线行驶。
2. 缓慢减速，切勿急刹车。
3. 停至安全区域，开启双闪灯。
4. 放置三角警示牌（车后50-100米）。
5. 使用随车千斤顶更换备胎或使用补胎液。
### 13.2 发动机过热
1. 立即靠边停车，挂P档，拉手刹。
2. 保持发动机怠速运转，不要立即熄火。
3. 打开发动机盖（注意烫伤）。
4. 检查冷却液液位，待冷却后再添加。
### 13.3 车辆陷车
1. 尝试倒车脱困。
2. 切换至4L模式，开启差速锁。
3. 左右轻打方向盘，寻找抓地力。
4. 如无法脱困，使用拖车绳或寻求救援。

## 十四、常见问题解答（FAQ）
**Q：山海炮加什么标号的汽油？**
A：3.0T汽油版推荐 **95号及以上** 无铅汽油；2.4T柴油版加 **0号或-10号柴油**（视气温而定）。
**Q：四驱模式可以在行驶中切换吗？**
A：2H与4H可在 **80km/h以下** 行驶中切换；切换4L需 **停车挂N档**。
**Q：如何重置胎压监测？**
A：补气后，在中控屏车辆设置中选择"胎压监测复位"，行驶几分钟后自动校准。
**Q：山海炮能加装车顶帐篷吗？**
A：可以，车顶动态承重约 **75kg**，静态承重约 **300kg**（含车顶行李架）。

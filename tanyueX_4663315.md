# 探岳X 驾驶注意事项
  
更新时间：2026年07月15日18时15分50秒   

## 一、启动与熄火
1.  **启动前检查**：确认挡位处于 **P挡**（驻车挡）或 **N挡**（空挡），并踩住制动踏板。
2.  **一键启动**：短按启动按钮，待仪表盘自检完成、发动机运转平稳后再挂挡行驶。
3.  **熄火操作**：车辆完全停稳后，挂入 **P挡**，拉紧电子手刹，再按下启动按钮熄火。**切勿在车辆未停稳时挂入P挡**，以免损坏变速箱。
## 二、行驶中注意事项
1.  **变速箱使用**：
    -   行驶中避免在 **D挡**（前进挡）和 **R挡**（倒挡）之间直接切换，必须在车辆完全静止时操作。
    -   长下坡或拥堵路段，建议使用 **S挡**（运动模式）或 **手动模式**，利用发动机制动，减少刹车系统负担。
2.  **自动启停系统**：
    -   涉水路段、严重拥堵或倒车入库时，建议 **手动关闭自动启停功能**，防止发动机频繁启停或涉水时意外熄火。
3.  **四驱系统（如配备）**：
    -   探岳X 4MOTION四驱版本具备一定的越野能力，但 **不适合极限越野或深水涉渡**。
    -   在冰雪、泥泞路面可使用 **越野模式**，但需控制车速，避免急加速和急转弯。
4.  **驾驶模式选择**：
    -   根据路况合理选择经济、标准、运动或越野模式。经济模式下动力响应较慢，超车时需提前预判。
## 三、特殊路况驾驶
1.  **雨天/湿滑路面**：
    -   降低车速，避免急刹车和急打方向。
    -   开启 **ESP（车身电子稳定系统）**，切勿手动关闭（除非陷入泥地需要脱困）。
2.  **雪地/结冰路面**：
    -   使用雪地模式（如有）或手动模式起步，避免大油门导致打滑。
    -   保持更大跟车距离，提前轻踩刹车。
3.  **山路/长下坡**：
    -   利用 **手动模式** 或 **S挡** 控制车速，避免长时间踩刹车导致刹车过热失效。
4.  **涉水路段**：
    -   **切勿强行涉水**。若必须通过，确认水深不超过排气管高度，低速匀速通过，中途不换挡、不松油门。通过后轻踩刹车排干刹车盘水分。
## 四、停车与驻车
1.  **电子手刹与Auto Hold**：
    -   等红灯时，可使用 **Auto Hold**（自动驻车）功能，解放右脚。
    -   停车熄火前，务必确认 **电子手刹** 已拉起，仪表盘有红色驻车指示灯。
2.  **坡道停车**：
    -   在坡道上停车，建议先拉紧电子手刹，再挂入P挡，可防止下次启动时P挡卡滞难以摘出。
3.  **方向盘回正**：停车熄火前，务必将方向盘回正，避免转向系统受力不均。
## 五、日常检查与维护
1.  **轮胎**：定期检查胎压（建议按驾驶侧车门标签值充气），注意轮胎磨损情况，特别是大角度转向时避免蹭到路肩。
2.  **机油与冷却液**：每两周检查一次机油尺和冷却液液位，长途出行前务必检查。
3.  **颗粒捕捉器（GPF）**：
    -   探岳X 380TSI及部分330TSI车型配备颗粒捕捉器。为预防堵塞，建议 **定期跑高速或保持发动机高转速行驶**（如使用S挡），避免长期短途、低速行驶。
    -   若仪表提示“颗粒捕捉器再生”，请按提示保持高速行驶直至再生完成。
4.  **天窗与排水**：定期清理天窗滑轨和排水孔，防止堵塞导致车内进水。
## 六、安全提醒
1.  **盲区监测**：探岳X 溜背造型导致后窗视野较窄，变道时务必结合 **后视镜** 和 **盲区监测提示灯**，不可完全依赖倒车影像。
2.  **儿童锁**：后排有儿童乘坐时，请启用后门儿童安全锁。
3.  **禁止超载**：严格按照车辆铭牌标注的载重限额，避免超载影响操控和刹车性能。
4.  **勿用手机**：驾驶过程中严禁手持接打电话或操作中控屏幕，建议使用车载语音控制。
---
> **总结**：探岳X 动力强劲、操控灵活，但溜背造型带来一定的视野局限。日常驾驶中注意变速箱操作规范、定期检查颗粒捕捉器状态，并善用驾驶模式应对不同路况，即可获得安全且富有乐趣的驾乘体验。

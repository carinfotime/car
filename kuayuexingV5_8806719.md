# 跨越星V5 驾驶注意事项
  
更新时间：2026年07月17日11时55分07秒   

## 一、启动前检查
1. **轮胎检查**：检查胎压是否正常（建议冷车胎压前轮2.3-2.5bar，后轮2.5-2.8bar），轮胎表面无破损、鼓包。
2. **油液检查**：
   - 机油液位在标尺上下限之间
   - 冷却液液位在正常范围
   - 制动液液位充足
   - 玻璃清洗液充足
3. **灯光检查**：大灯、转向灯、刹车灯、雾灯、示廓灯均正常工作。
4. **仪表盘自检**：启动后观察仪表盘是否有故障灯常亮。
## 二、驾驶中注意事项
### 1. 起步与换挡
- 手动挡车型：起步时离合要慢抬，避免熄火；换挡转速建议在2000-2500rpm之间。
- 自动挡车型：挂入D挡后松开手刹，缓慢加油起步。
### 2. 行驶速度
- 经济时速约为60-90km/h，高速行驶建议控制在100km/h以内。
- 避免长时间高速行驶，防止发动机过热。
### 3. 转弯与变道
- V5车身较高，转弯时注意减速，防止侧倾。
- 变道前提前打转向灯，观察后视镜盲区。
### 4. 刹车技巧
- 提前预判路况，避免急刹车。
- 雨天或湿滑路面加大跟车距离，轻踩刹车。
### 5. 载重注意事项
- **严禁超载**：严格按照车辆核定载质量装载货物。
- **货物固定**：货物需均匀分布并固定牢固，防止行驶中移位影响操控。
- **重心控制**：装载较高货物时转弯速度要更低，防止侧翻风险。
## 三、特殊路况驾驶
| 路况 | 注意事项 |
|------|----------|
| **雨天** |降低车速至60km/h以下；开启雾灯或近光灯；避免涉水过深（不超过排气管高度） |
| **山路/坡道** |下坡使用低挡位辅助制动；上坡提前降挡保持动力；弯道鸣笛示警 |
| **高速路** |保持安全车距（至少100米）；避免频繁变道；注意横风影响 |
| **夜间行车** |合理使用远近光灯；会车时切换近光灯；注意路面障碍物 |
## 四、停车与熄火
1. **停车位置**：选择平坦坚实地面停放，拉紧手刹。
2. **熄火前操作**：
   - （涡轮增压车型）怠速运转1-2分钟再熄火，保护涡轮
   - （非涡轮车型）可直接熄火
3. **冬季停车**：若气温低于0℃，建议添加防冻玻璃水并检查防冻液冰点。
## ⚠️五、安全警示
1. **安全带**：所有乘员必须系好安全带。
2. **分心驾驶禁止**：驾驶过程中严禁使用手机或其他电子设备。
3. **疲劳驾驶禁止**：连续驾驶不超过4小时，需休息至少20分钟。
4. **定期保养提醒**：
   - ✅每5000公里或6个月更换机油机滤（以先到为准）
   - ✅每10000公里检查空气滤芯和空调滤芯
   - ✅每20000公里更换火花塞和变速箱油（手动挡）
   - ✅每40000公里更换制动液和冷却液

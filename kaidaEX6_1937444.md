# 恺达EX6 操作指南
  
更新时间：2026年07月18日13时02分46秒   

## 一、车辆概述
恺达EX6是一款纯电动微型货车，适用于城市短途物流运输。本指南涵盖车辆基本操作、充电、安全驾驶及日常维护等内容。
---
## 二、启动与熄火
### 启动步骤
1. 确认车辆处于 **P挡（驻车挡）**
2. 智能钥匙在车内有效范围内
3. 踩下 **制动踏板**
4. 按下 **一键启动按钮**（绿色指示灯亮起，仪表盘显示“READY”）
5. 松开手刹，挂入 D挡 或 R挡 即可行驶
### 熄火步骤
1. 踩下制动踏板，将车辆停稳
2. 挂入 **P挡**
3. 拉起 **电子手刹**（或手动手刹）
4. 再次按下 **一键启动按钮**，车辆断电
---
## 三、挡位操作
| 挡位 | 功能说明 |
|------|----------|
| **P** | 驻车挡，停车时使用 |
| **R** | 倒车挡，倒车时使用 |
| **N** | 空挡，临时停车或拖车时使用 |
| **D** | 前进挡，正常行驶使用 |
| **E** | 经济模式，动力输出平顺，续航更长 |
| **S** | 运动模式，动力响应更快 |
> 切换挡位前务必确保车辆完全停稳。
---
## 四、仪表盘指示灯说明
| 指示灯 | 含义 | 处理方式 |
|--------|------|----------|
| **READY** | 车辆已就绪可行驶 | 正常状态 |
| **红色电池** | 电池故障 | 立即停车，联系售后 |
| **黄色电池** | 电量过低 | 尽快充电 |
| **红色感叹号** | 系统故障 | 检查手刹/制动系统 |
| **充电指示灯** | 正在充电 | 正常状态 |
| **ABS灯** | ABS系统故障 | 谨慎驾驶，尽快检修 |

## 五、充电操作
### 充电方式
| 类型 | 说明 | 充电时间 |
|------|------|----------|
| **交流慢充（家用220V）** | 使用随车充电枪，接家用插座 | 约 8~10 小时 |
| **直流快充** | 使用公共快充桩 | 约 1~1.5 小时（充至80%） |
### 充电步骤
1. 车辆熄火，挂 P挡
2. 打开充电口盖（位于车头 LOGO 处或车身侧方）
3. 插入充电枪，确认锁止
4. 充电桩端启动充电（刷卡/扫码/APP）
5. 仪表盘显示充电进度
6. 充电完成后，先停止充电桩，再拔下充电枪
7. 盖好充电口盖
### 充电注意事项
- 尽量使用 **专用接地插座** 充电
- 避免在 **雷雨天气** 户外充电
- 充电时不要启动车辆
- 电量建议保持在 **20%~90%** 之间，延长电池寿命
- 长期停放时，保持电量在 **50%~80%**
---
## 六、驾驶操作
### 起步
- 确认仪表盘显示 **READY**
- 挂入 D 挡，松开手刹
- 轻踩 **电门踏板** 起步
### 行驶
- 电门踏板控制加速，松踏即可减速
- 车辆具备 **能量回收** 功能，松电门时自动回收能量
- 建议使用 **E挡（经济模式）** 以获得更长续航
### 倒车
- 挂入 R 挡，注意观察后方
- 倒车影像自动开启（如配备）
### 停车
- 踩刹车停稳 → 挂 P挡 → 拉手刹 → 熄火
---
## 七、能量回收系统
- **默认开启**，松电门时自动回收能量
- 回收强度可调（通过中控或仪表设置）
- 强回收模式：松电门后减速明显，适合城市走走停停
- 弱回收模式：松电门后滑行更远，适合高速行驶
---
## 八、安全注意事项
### 行车安全
- 系好安全带，驾驶座与副驾驶座均需佩戴
- 禁止超载，最大载重量请参考车辆铭牌
- 雨天行驶注意路面湿滑，制动距离会增加
- 涉水深度不超过 **30cm**，避免电池进水
### 停放安全
- 尽量停放在 **室内或遮阴处**
- 长时间暴晒会影响电池寿命
- 远离火源和易燃物
### 紧急情况
- 如遇车辆故障，打开双闪灯，放置三角警示牌
- 紧急断电：长按 **一键启动按钮** 3秒以上可强制断电
- 如发生碰撞，第一时间断开高压电（维修人员操作）

## 九、日常维护
| 项目 | 周期 | 说明 |
|------|------|------|
| 轮胎检查 | 每月 | 胎压建议 2.4~2.6 bar |
| 制动液 | 每2年 | 检查液位及含水量 |
| 空调滤芯 | 每1年 | 更换 |
| 减速器油 | 每2年或4万公里 | 更换 |
| 电池冷却液 | 每4年 | 更换 |
| 高压系统检查 | 每1年 | 由专业技师检查 |
### 自检清单
- 充电口及线束有无破损
- 轮胎磨损及胎压
- 灯光、喇叭、雨刮是否正常
- 仪表盘有无故障灯

## 十、常见问题处理
| 问题 | 可能原因 | 解决方法 |
|------|----------|----------|
| 车辆无法启动 | 钥匙不在车内/未踩刹车/电量过低 | 检查钥匙、刹车、电量 |
| 充电无法开始 | 充电枪未插好/桩端故障/接地不良 | 重新插枪，检查插座 |
| 续航明显下降 | 驾驶模式/空调/胎压/温度影响 | 检查胎压，关闭大功率电器 |
| 车辆异响 | 轮胎异物/制动系统/悬挂问题 | 检查轮胎，联系售后 |
| 仪表盘报警 | 系统故障 | 记录报警信息，联系售后 |

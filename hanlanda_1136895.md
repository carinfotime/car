# 丰田汉兰达 技术参数介绍
  
更新时间：2026年07月20日10时09分51秒   

## 一、车型概览
丰田汉兰达（Toyota Highlander）是一款中型SUV，以其宽敞的内部空间、可靠的品质和均衡的性能著称。目前国内在售的主要为第四代车型（基于TNGA-K平台打造），提供燃油版与双擎混动版两种动力选择。
---
## 二、动力系统
### 1. 燃油版 (380T)
| 项目 | 参数 |
|------|------|
| **发动机型号** | S20A-FTS (2.0T) |
| **发动机类型** | 直列四缸、涡轮增压、D-4ST双喷射 |
| **排量** | 1997 mL |
| **最大功率** | 182 kW (248 PS) / 6000 rpm |
| **最大扭矩** | 380 N·m / 1800-4000 rpm |
| **变速箱** | Direct Shift-8AT (8速手自一体) |
| **驱动方式** | 前置四驱 (动态扭矩矢量控制) |
### 2. 双擎混动版 (智能电混双擎)
| 项目 | 参数 |
|------|------|
| **发动机型号** | A25D-FXS (2.5L) |
| **发动机类型** | 直列四缸、自然吸气、混合喷射 |
| **排量** | 2487 mL |
| **发动机最大功率** | 139 kW (189 PS) / 6000 rpm |
| **发动机最大扭矩** | 236 N·m / 4200-4700 rpm |
| **前电机功率/扭矩** | 134 kW /  270 N·m |
| **后电机功率/扭矩(四驱)** |  40 kW /  121 N·m |
| **系统综合最大功率** |  181 kW (246 PS) |
| **变速箱** | E-CVT (电子无级变速) |
| **驱动方式** | E-Four电子四驱 /  前驱 |

## 三、底盘与悬架
-   **前悬架：**  麦弗逊式独立悬架
-   **后悬架：** E型多连杆式独立悬架
-   **转向系统：** EPS电动助力转向
-   **制动系统：**
    -   前：通风盘式
    -   后：实心盘式
    -   驻车：电子手刹 + Auto Hold自动驻车
---
## 四、车身尺寸与重量
### （以2024款双擎四驱尊贵版为例）
*注：不同配置车型尺寸略有差异*
-   **长 ×宽 ×高：**
    -   4965 mm ×1930 mm ×1750 mm
-   **轴距：**
    -   2850 mm
-   **最小离地间隙（空载）：**
    -   约170 mm
-   **整备质量：**
    -   燃油版约2035 kg
    -   双擎两驱约1965 kg
    -   双擎四驱约2075 kg

## 五、座椅与空间布局
-   **座位数：**
    -   5座（部分入门款）/7座（主流配置）
-   **座椅布局形式（7座）：**
    2+3+2 （第二排可4/6分割放倒，第三排可5/5分割放倒）
-   **后备箱容积（参考值）：**
    第三排升起时约215 L；第三排放倒后约568 L；第二三排全放倒后约1150 L以上。

## 六、主要配置亮点
### （部分高配或选装项）
1.　智能驾驶辅助 —— Toyota Safety Sense智行安全套装：
     包括PCS预碰撞安全系统、DRCC动态雷达巡航控制（全速域）、LTA车道循迹辅助、AHB自动远光灯等。
2.　多媒体系统 —— Toyota Space智能座舱：
     12.3英寸中控屏 +12.3英寸全液晶仪表盘，支持CarPlay/HiCar/CarLife手机互联，语音控制。
3.　舒适便利：
     JBL高级音响（11扬声器）、前排座椅加热/通风、全景天窗、电动尾门带脚踢感应。
4.　安全气囊：
     全系标配9个SRS空气囊。

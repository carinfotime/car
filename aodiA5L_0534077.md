# 奥迪A5L 驾驶注意事项
  
更新时间：2026年07月18日13时55分20秒   

## 一、启动与预热
1. **冷启动**：车辆长时间停放后启动，建议怠速运转 **30秒至1分钟**，待机油充分润滑涡轮增压器后再行驶。
2. **避免原地热车过久**：现代发动机无需长时间原地热车，低速行驶至水温达到正常工作温度（约90°C）即可正常驾驶。
3. **一键启动操作**：踩住刹车踏板，按下启动按钮；若仅通电不启动发动机，请勿踩刹车直接按启动按钮。
## 二、行驶中注意事项
### 1. 变速箱使用
- **双离合变速箱（S tronic）**：
  - 拥堵路况建议切换至 **手动模式** 或 **S挡（运动模式）**，减少频繁换挡导致的离合器过热。
  - 等红灯时建议挂入 **N挡（空挡）** 并拉电子手刹，避免D挡踩刹车长时间等待。
- **禁止空挡滑行**：自动变速箱严禁N挡滑行，会损坏变速箱。
### 2. 驾驶模式选择
- **经济模式（Efficiency）**：适合城市巡航和高速匀速行驶，可降低油耗。
- **舒适模式（Comfort）**：日常推荐使用，悬挂偏软、转向轻盈。
- **动态模式（Dynamic/Sport）**：激烈驾驶或山路使用，油门响应更快、悬挂变硬。
- **自定义模式（Individual）**：可根据个人喜好单独设定转向、动力、悬挂等参数。
### 3. Quattro四驱系统
- A5L配备适时四驱系统（部分车型），正常路面以前驱为主；雨雪或湿滑路面会自动分配后轮动力。
- **切勿在非铺装路面进行高强度越野**，A5L定位为公路轿跑。
### 4. 制动系统
- **磨合期**：新车前500公里避免急刹车，让刹车片与刹车盘充分磨合。
- **涉水后制动**：通过积水路段后轻踩几次刹车，利用摩擦热量蒸发水分恢复制动性能。
## 三、停车与熄火
1. **熄火前操作流程**
   - 车辆停稳 → N挡 → 拉电子手刹 → P挡 → 松开脚刹 → 关闭空调/音响等电器 →熄火。
   - *注意*：涡轮增压车型熄火前最好怠速运转30秒左右再熄火（长途高速后尤其重要），保护涡轮轴承。
2. **坡道停车**
   -务必先拉手刹再挂P挡，防止P挡锁止机构承受过大压力导致下次挂挡困难。
3. **自动启停系统**
   -涉水路段务必手动关闭自动启停功能（按钮位于中控台），防止水中二次启动损坏发动机。
## 四、特殊路况驾驶
|  路况类型 |  注意事项 |
|-----------|-----------|
|  雨天/湿滑路面 |  降低车速、增大跟车距离；关闭ESP可能增加失控风险 |
|  积雪/结冰路面 |  建议使用冬季轮胎；开启ESP保持开启状态 |
|  长下坡路段 |  利用换挡拨片降档辅助制动，避免长时间踩刹车导致热衰减 |
|  高速公路 |  保持合理胎压；每2小时休息一次防止疲劳 |
## 五、日常维护提醒
1. **机油检查**
   - A5L通常使用0W-20或5W-30全合成机油。每1000公里检查一次机油尺液位（电子机油尺可在MMI系统中查看）。
2. **轮胎保养**
   - A5L原厂多为防爆胎或高性能轮胎。每月检查胎压：
     -前轮：约2.3~2.5 bar
     -后轮：约2.4~2.6 bar
     （具体数值以油箱盖内侧标签为准）
3. **燃油要求**
   -必须加注95号及以上无铅汽油。切勿加错柴油！
4. **颗粒捕捉器（GPF）再生**
   -若仪表提示颗粒捕捉器需要再生：
    保持发动机转速2000转以上行驶约20~30分钟即可完成再生。避免长期短途低速行驶。
## ⚠️特别警示
> ❌禁止事项：
> - N挡滑行
> - D/R切换时未完全停稳就换档
> -涉水时开启自动启停
> -用高压水枪直接冲洗发动机舱
> -长期短途行驶不进行高速巡航

## 一汽-大众 探岳 空间参数介绍

### 车身尺寸
| 项目 | 参数 |
|------|------|
| 车长 (mm) | 4589 / 4592 / 4595 (因配置版本不同略有差异) |
| 车宽 (mm) | 1860 |

### 内部乘坐空间
- **前排头部空间**：约 **980 mm**（身高175cm体验者实测约一拳余量）
- **后排头部空间**：约 **940 mm**
- **后排腿部空间**：约 **900 mm**（膝部距离前排座椅靠背约两拳半至三拳）
- **后排座椅**：支持前后调节（前后滑动范围约 **180 mm**）及靠背角度调节，灵活性较高
- **后排中央通道**：有一定凸起，中间乘客长途乘坐舒适性略受影响
### 储物空间
#### 常规储物格
- **前排杯架**：2个（带限位器）
- **中央扶手箱**：容积约 **5.5 L**
- **手套箱**：容积适中，带阻尼
- **门板储物格**：前后门均可放置水瓶及杂物
- **眼镜盒**：驾驶位头顶处
- **卡片/票据槽**：中控台下方
#### 后备箱
| 项目 | 参数 |
|------|------|
| 标准容积 (L) | **506 L** |
| 最大扩展容积 (L) | **1590 L**（放倒后排座椅后） |
|  后备箱开口宽度 |  约1100 mm |
|  后备箱离地高度 |  适中，搬运大件较方便 |
|  后排座椅放倒方式 |  4/6比例放倒，可形成近乎纯平的空间 |
### 📌亮点总结
1. **轴距优势明显（2731mm）**，在同级中型SUV中处于领先水平，为纵向腿部空间提供保障。
2. **后排座椅可调设计（前后+靠背角度）**，能灵活平衡乘坐与装载需求。
3. **后备箱规整度高且扩展性强（506L→1590L）**，满足家庭日常及长途出行需求。
4. *不足之处*：后排中央地台隆起较高；部分储物格深度一般。

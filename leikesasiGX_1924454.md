# 雷克萨斯GX 动力参数介绍
  
更新时间：2026年07月10日23时26分18秒   

## 发动机参数
| 项目 | 规格 |
|------|------|
| **发动机类型** | V型6缸 DOHC 24气门 |
| **排量** | 3.5L（3,456 cc） |
| **进气形式** | 双涡轮增压 |
| **最大功率** | 260 kW（354 PS）/ 5,200 rpm |
| **最大扭矩** | 649 N·m / 2,000–3,600 rpm |
| **燃油系统** | D-4ST（直喷+歧管喷射） |
| **压缩比** | 10.7 : 1 |
| **推荐燃油** | 95号及以上无铅汽油 |
## 传动系统
- **变速箱**：10速自动变速箱（Direct Shift-10AT）
- **驱动方式**：全时四驱（Full-Time AWD）
- **分动箱**：带低速挡（4L）的电子控制分动箱
- **差速锁**：后桥差速锁（部分车型标配）
## 底盘与悬挂
- **前悬挂**：双叉臂式独立悬挂
- **后悬挂**：多连杆式非独立悬挂
- **车身结构**：非承载式车身（梯形大梁）
## 性能数据
| 项目 | 数据 |
|------|------|
| **0–100 km/h加速时间** |约6.5秒|
| **最高时速** |约175 km/h|
| **最小离地间隙** |225 mm|
| **接近角/离去角/通过角** |31° /22° /25°|
## Overtrail版专属配置
Overtrail及Overtrail+版本额外配备：
- E-KDSS（电子动态悬架控制系统）
- AVS（自适应可变悬架系统）
- MTS（多地形选择系统）
- Crawl Control（蠕行控制系统）
---
> *注：以上参数基于最新一代雷克萨斯GX，具体配置可能因市场及年款不同而有所差异。*

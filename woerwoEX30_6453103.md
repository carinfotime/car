# 沃尔沃EX30 驾驶注意事项
  
更新时间：2026年07月17日16时43分10秒   

## 一、启动与驾驶前检查
1.  **钥匙与手机绑定**：确保已通过沃尔沃汽车App或钥匙卡成功绑定车辆。手机作为数字钥匙时，请保持蓝牙开启且App在后台运行。
2.  **电量检查**：出发前确认电池电量充足。建议长途出行前将电量充至80%-90%，以保护电池健康。
3.  **轮胎状态**：检查胎压是否正常（建议冷车胎压2.6-2.8 bar），并观察轮胎磨损情况。
4.  **儿童安全**：后排如需安装儿童座椅，请严格按照说明书操作，并确保副驾驶安全气囊已关闭（如反向安装儿童座椅时）。
## 二、行驶中的核心注意事项
### 1. 单踏板模式适应
- EX30默认提供“单踏板模式”（One Pedal Drive）。松开加速踏板时车辆会强力回收能量并减速，直至停止。
- **新手建议**：先在空旷路段练习松油门力度，避免因突然松开导致急刹或乘客不适。
- **可调节**：在设置中可调整回收强度，找到最适合自己的脚感。
### 2. 视野与盲区
- EX30车身较高但A柱较粗，转弯时务必左右移动头部观察盲区。
- **全景天幕注意**：虽然通透性好，但强光下建议使用遮阳帘或佩戴偏光镜。
### 3. 智能辅助驾驶（Pilot Assist）
- **非全自动驾驶**：该系统仅为辅助功能，双手必须始终放在方向盘上。
- **使用场景**：适合高速公路及清晰标线的道路。城市拥堵路况慎用，谨防加塞车辆识别不及时。
- **紧急接管**：当系统发出“请接管”提示音或仪表盘闪烁红色警告时，立即人工控制车辆。
### 4. 充电相关
- **快充限制**：直流快充（DC）下，电量从10%充至80%通常需26分钟左右。当电量超过80%后充电速度会显著下降。
- **低温影响**：冬季气温低于0℃时，建议使用“电池预热”功能（可通过App或导航到充电站时自动激活），以缩短充电时间。
- **避免过放**：不建议将电量耗尽至0%，长期低电量会损害电池寿命。
## 三、特殊路况应对
| 路况 | 操作建议 |
|------|----------|
| **雨天/湿滑路面** | EX30重心较低且为四驱/后驱版本。起步轻踩电门；关闭单踏板模式或调低回收强度以防打滑；保持更大跟车距离。 |
| **雪地/结冰路面** | - ESP系统默认开启切勿关闭。<br>- 建议使用冬季轮胎。<br>- “越野模式”仅适用于极低速脱困，不可用于高速行驶。 |
| **山路/长下坡** | -利用高动能回收代替刹车减少刹车过热风险。<br>-手动切换至B档（如配备）获得更强制动效果。 |
## 四、停车与离车
1.  **自动驻车确认**：停车后仪表盘显示“(A)”绿色图标表示自动驻车生效。坡道停车务必确认后再松刹车。
2.  **挂P档逻辑**：解开安全带并打开车门时车辆会自动挂入P档并拉起电子手刹（需在设置中开启此功能）。
3.  **锁车检查**：
    -   EX30无传统门把手锁孔。确保所有车门、后备箱关严后再离开。
    -   手机作为钥匙离车一定距离后会自动落锁（可设置落锁提示音）。
4.  长时间停放（超过一周）：建议保持电量在50%-60%，并连接慢充桩维持12V电瓶健康。
## 五、其他关键提醒
-   OTA升级期间不可驾驶车辆；升级前确保停在安全位置且电量高于20%。
-   车内大屏是主要控制中心；熟悉菜单层级后再上路以减少分心操作时间。
-   紧急情况下长按危险警示灯按钮可触发SOS紧急呼叫功能（需连接网络）。

# 帕萨特插电混动（Passat GTE）操作指南
  
更新时间：2026年07月20日17时41分26秒   

## 一、车辆启动与关闭
### 启动车辆
1. 确保智能钥匙在车内。
2. 踩下制动踏板。
3. 按下“START ENGINE STOP”按钮（一键启动按钮）。
4. 仪表盘点亮，READY指示灯亮起，表示车辆已进入可行驶状态。
### 关闭车辆
1. 将车辆完全停稳。
2. 挂入P挡（驻车挡）。
3. 再次按下“START ENGINE STOP”按钮，车辆电源关闭。
## 二、驾驶模式选择
通过中控台或中控屏幕上的 **GTE/模式按钮**，可在以下模式间切换：
| 模式 | 名称 | 适用场景 |
|------|------|----------|
| **E-MODE** | 纯电模式 | 城市通勤、短途行驶。仅由电机驱动，零油耗零排放。 |
| **混合动力 (Hybrid Auto)** | 自动混合模式 | 日常综合驾驶。系统自动优化发动机与电机的配合。 |
| **蓄电池维持 (Battery Hold)** | 保持电量模式 | 高速巡航或即将进入市区时使用。优先使用发动机，保留电池电量。 |
| **蓄电池充电 (Battery Charge)** | 充电模式 | 行驶中利用发动机为电池充电（油耗较高）。 |
| **GTE** |运动模式 |需要强劲动力时（如超车、山路）。发动机与电机同时输出最大功率。 |
> **提示**：在电量充足时，默认启动为E-MODE纯电模式。
## 三、充电操作
### 家用充电（交流慢充）
1.  打开车辆左前翼子板上的充电盖板。
2.  取下充电枪，插入车辆充电口。
3.  另一端连接至220V家用插座或专用充电桩。
4.  充电开始后，仪表盘或充电口指示灯会显示状态：
    -   **绿色闪烁**：正在充电
    -   **绿色常亮**：充电完成
    -   **红色闪烁**：故障
5.  充满后按压解锁键并拔出充电枪，盖好盖板。
### 公共快充（直流快充）
-   帕萨特插电混动通常仅支持交流慢充（具体请以车型配置为准），不支持直流快充。请查阅用户手册确认。
### 预约充电
-   可通过中控屏幕设置预约充电时间，利用夜间谷电时段降低成本。
## 四、能量回收与制动
-   **滑行能量回收**：松开加速踏板时，系统自动回收动能给电池充电。
-   **制动能量回收**：踩下制动踏板时加强回收力度。
-   仪表盘上的功率/能量指示条会显示当前是消耗能量还是回收能量。
## 五、仪表盘关键信息解读
-   **READY指示灯**：亮起表示可行驶。
-   **电量指示条/百分比**：显示高压电池剩余电量。
-   **续航里程显示**：
    -   纯电续航里程
    -   燃油续航里程
    -   总续航里程
-   **功率表/能量流图**：
    -   蓝色区域 =电机驱动 /能量回收
    -   白色区域 =发动机驱动或共同驱动
## 六、常见注意事项
1.  **保持高压线束干燥清洁**：避免水浸或碰撞底盘电池组。
2.  **长期停放建议**：
    -   若超过两周不使用，建议保持电池电量在30%-80%之间。
    -   避免长时间亏电存放（低于10%），以免影响电池寿命。
3.  **涉水提醒**：虽然高压部件有防水设计，但仍应避免深水路段行驶。
4.  **保养提醒**：
    -除常规机油机滤保养外，定期检查高压系统冷却液及线束接口状态。
5.**紧急情况处理**
    如果发生碰撞导致高压系统报警：
    立即熄火并下车远离车辆等待救援人员处理切勿自行触碰橙色高压线束

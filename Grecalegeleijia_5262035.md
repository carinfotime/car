# 玛莎拉蒂 Grecale（格雷嘉）驾驶注意事项
  
更新时间：2026年07月13日11时06分49秒   

## 一、启动与预热
1.  **钥匙识别**：确保智能钥匙在车内，踩下刹车踏板，按下方向盘右侧的“Start/Stop”按钮启动发动机。
2.  **冷车启动**：冷车启动后，建议原地怠速 **30-60秒**，待发动机转速回落至平稳怠速状态（约800rpm）后再行驶。
3.  **机油温度**：在机油温度达到正常工作范围（约90°C）前，避免急加速或让发动机转速超过 **4000rpm**。
## 二、驾驶模式选择
Grecale 提供多种驾驶模式，请根据路况和需求合理选择：
| 模式 | 适用场景 | 特点与注意事项 |
| :--- | :--- | :--- |
| **COMFORT (舒适)** | 日常通勤、城市道路、长途巡航 | - 悬架最软，转向最轻。<br>- 适合平稳驾驶。<br>- **注意**：在湿滑路面起步时建议使用此模式。 |
| **GT (标准)** | 默认模式、综合路况 | - 平衡了舒适与运动感。<br>- 车辆会自动调节悬架和动力响应。 |
| **SPORT (运动)** | 山路、高速公路超车、追求操控乐趣 | - **悬架变硬**：路感更清晰，但颠簸感增强。<br>- **排气阀门打开**：声浪更响亮。<br>- **注意**：在颠簸路面或减速带前请减速。 |
| **RACE (赛道)** *(仅限Trofeo/高性能版)* | 赛道日、封闭道路极限驾驶 | - ESP介入最少，牵引力控制放宽。<br>- **警告**：不适合公共道路日常使用。 |
| **OFF-ROAD (越野)** *(可选配)* | 非铺装路面、轻度越野、雪地/泥地 | - ESC优化，陡坡缓降功能激活。<br>- **注意**：车速超过约70km/h时自动退出。 |
## 三、行驶中关键操作
### 1. 变速箱与换挡
-   Grecale采用 ZF8速自动变速箱。
-   **D档（自动模式）**：日常驾驶首选。
-   **M档（手动模式）**：
    -   可通过方向盘后方的金属换挡拨片操作。
    -   *左侧拨片* (-) ：降档；*右侧拨片* (+) ：升档。
    -   SPORT模式下响应更快。
    -   *注意*：连续降档超车时，请留意发动机转速不要进入红线区。
### 2. “Sport”按钮与排气
-   按下中控台上的“Sport”按钮可单独激活更激进的排气声浪和更快的油门响应。
-   *提示*：在居民区或深夜行驶时，建议关闭Sport模式以避免噪音扰民。
### 3. “Launch Control”（弹射起步）
> *仅限Trofeo版本或选装运动套件的车型*
1.  关闭ESP（长按ESP OFF按钮）。
2.  将驾驶模式切换至 SPORT / RACE。
3.  左脚踩死刹车踏板。
4.  右脚迅速将油门踏板踩到底（触发强制降档开关）。
5.  仪表盘显示“Launch Control Active”。
6.  松开刹车踏板。
> ⚠️ **警告**：
> -   弹射起步会极大增加传动系统负荷。
> -   仅在车辆充分热车后使用。
> -   不建议频繁操作。
### 4. iCUP信息娱乐系统
-   所有主要功能集成于中央触控屏和下方舒适性控制屏。
-   *注意*：行驶中尽量避免深度操作屏幕菜单。建议使用语音控制或方向盘快捷键。
## 四、停车与熄火
1.  停稳车辆后，挂入 P档（驻车档）。
2.  拉起电子手刹（位于方向盘左侧下方）。
3.  按下“Start/Stop”按钮熄火。
4.  熄火前建议：
    -   如果刚进行过激烈驾驶或长途高速行驶，让发动机怠速运转约1分钟再熄火以冷却涡轮增压器。
## ⚠️五、安全与维护提醒
### A) ESP / ASR系统
-   默认情况下ESP处于开启状态以保障安全。
-   如需脱困或在雪地起步打滑时：
    1.  短暂按下ESP OFF可解除ASR牵引力限制以便脱困。
    2.  脱困后应立即重新开启ESP。
### B) Q4智能四驱系统
-   全系标配适时四驱系统。正常行驶时为后驱以节省燃油；需要抓地力时自动切换至四驱。
-   无需手动干预。
### C) “空气悬架”（可选配）
-   车身高度可调：
    - *最低* （Aero）：高速巡航降低风阻。（车速 >130km/h自动降低）
    - *标准*
    - *最高* （Off-Road）：越野或通过深水区使用。（车速 >70km/h自动恢复标准高度）
- ⚠️ *重要提示*：
    1.  举升车辆前必须将空气悬架设置为“千斤顶模式”（通过中控屏设置），否则可能损坏气包及管路！
    2.  冬季寒冷地区长时间停放可能导致气泵工作频繁属正常现象；若出现车身明显倾斜则需检修漏气问题。
### D) “轮胎”
- Grecale配备高性能轮胎（夏季胎居多），低温环境 (<7°C)下抓地力会显著下降！冬季务必更换冬季轮胎！
- TPMS胎压监测系统会在仪表盘显示胎压数值；冷态胎压建议保持在厂家推荐值（通常在前轮2 .3 bar /后轮2 .5 bar左右）。
### E) “制动系统”
新车刹车片需要磨合期（约300公里），此期间避免紧急制动！
激烈驾驶后刹车盘温度极高！切勿立即洗车或用冷水冲洗刹车盘！

## 📋六、“新手常见误区”
1️⃣ “等红灯挂P档”：错误做法！若被追尾会严重损坏变速箱锁止机构！正确做法是挂N档+拉手刹或保持D档+Auto Hold功能激活状态即可！
2️⃣ “油箱见底才加油”：Grecale燃油泵依靠汽油散热冷却；长期低油位会缩短燃油泵寿命！建议剩余续航显示80公里左右及时加油！
3️⃣ “忽略保养提示灯”：仪表盘出现黄色扳手图标表示保养到期；切勿拖延以免影响质保权益！

# 北京越野BJ30 儿童乘车注意事项
  
更新时间：2026年07月19日10时21分36秒   

为了确保您孩子的乘车安全，请务必遵守以下事项：
## 一、安全座椅使用规范
1.  **强制使用**：12周岁以下或身高低于145厘米的儿童，必须使用与其年龄、体重和身高相匹配的儿童安全座椅。
2.  **反向安装**：建议4岁以下的儿童使用反向安装的儿童安全座椅，以更好地保护其头颈部位。
3.  **正确固定**：严格按照安全座椅说明书和车辆说明书进行安装，确保座椅与车辆ISOFIX接口（如有）或安全带牢固连接，无松动。
4.  **后排安装**：儿童安全座椅必须安装在后排座位上。
## 二、乘坐位置与约束
1.  **严禁副驾驶**：绝对禁止儿童坐在副驾驶位置。在发生碰撞时，弹出的安全气囊会对儿童造成严重伤害甚至致命。
2.  **后排落座**：所有12岁以下儿童均应坐在后排座位。
3.  **安全带使用**：
    -   对于身高足以正常使用成人安全带（通常约145厘米以上）的儿童，确保安全带肩带跨过肩膀和胸部中央，腰带紧贴大腿根部，而非腹部或颈部。
    -   切勿让儿童将肩带放在手臂下方或背后。
## 三、行车过程中的注意事项
1.  **开启童锁**：上车后请务必开启两侧后门的儿童安全锁。启用后，车门无法从车内打开，只能从车外打开，防止孩子在行驶中误开车门。
2.  **车窗控制**：
    -   启用驾驶员侧的车窗锁止功能（如有），防止孩子自行操作车窗升降。
    -   切勿让孩子将头、手或身体任何部位伸出车窗外。
3.  **车内物品管理**：
    -   收好车内所有零散物品（如水瓶、玩具、手机支架等），急刹车或碰撞时这些物品可能变成抛射物伤及孩子。
    -   BJ30后备箱空间较大，请勿让孩子在行车过程中爬入后备箱玩耍。
4.  **避免干扰驾驶**：教育孩子不要在行车过程中拉扯方向盘、挡杆或触碰中控屏幕按键。
## 四、停车与上下车
1.  **观察环境**：停车后开门前，务必先通过后视镜观察后方是否有来车或行人。教育孩子养成“荷式开门法”（用远离车门的手开门）的习惯。
2.  **家长陪同下车**：年幼的孩子应由家长陪同并协助上下车，避免其独自跳下摔伤或被车门夹伤手指。
3.  **切勿独留车内**：无论天气如何（尤其是夏季高温暴晒），绝对不要将孩子单独留在车内。车内温度会迅速升高至危险水平，极易导致热射病甚至死亡。
## 五、越野路况特别提醒
1.  在非铺装路面行驶时（如沙地、泥地、碎石路），车身晃动和颠簸会加剧。请务必确认孩子的安全座椅已加固到最紧状态。
2.  穿越陡坡或涉水路段前，应提前告知并安抚孩子情绪，避免其因紧张而突然起身或在座位上大幅度移动。
3.  越野过程中尽量避免让孩子进食或饮水（特别是用吸管杯），以防呛咳。
> **总结：后排 + 安全座椅 + 童锁 + 车窗锁 + 不独留 = BJ30上最安全的童年旅程**

# 江淮iEVA50 驾驶注意事项
  
更新时间：2026年07月10日22时11分06秒   

## 一、启动与行驶前检查
1.  **钥匙与启动**：确保智能钥匙在车内，踩下制动踏板，按下“启动/停止”键，确认仪表盘上“READY”指示灯亮起，车辆方可挂挡行驶。
2.  **仪表盘检查**：启动后观察仪表盘，确认无故障灯（如电池故障、电机故障、系统警告）常亮。
3.  **充电状态**：确认剩余电量（SOC）足够本次行程。冬季或长途出行前，建议提前预热电池或充满电。
4.  **轮胎与制动**：检查胎压是否正常，制动系统是否灵敏无异响。
## 二、行驶中操作规范
1.  **挡位操作**：
    -   **D挡**：正常前进挡。
    -   **R挡**：倒车挡，必须在车辆完全停止后挂入。
    -   **N挡**：空挡，短暂停车（如等红灯）可使用，但拖车时请遵循说明书。
    -   **P挡**：驻车挡，停车熄火时必须挂入。
2.  **能量回收系统**：
    -   车辆通常配备多级能量回收（如弱、中、强）。
    -   **强回收模式**：松开电门踏板时会有明显拖拽感，可回收更多电能，但需提前适应，避免造成后车追尾风险。
    -   建议在拥堵路况或下坡时使用高回收等级，高速巡航时使用低回收等级更平顺。
3.  **加速与减速**：
    -   电动车扭矩输出直接，起步加速快。请轻踩电门，避免急加速导致电能消耗过快或轮胎打滑。
    -   预判路况，提前松电门利用能量回收减速，减少刹车片磨损并回收电能。
4.  **涉水与雨天**：
    -   虽然电池有防水设计，但**尽量避免**驶入深水区（水深超过车轮一半）。
    -   遇到积水路段，应低速匀速通过，避免激起过大水花冲击底盘高压部件。
    -   雨天行驶后，若条件允许，可检查充电口是否干燥清洁。
## 三、续航与充电管理
1.  **续航估算**：仪表显示的剩余续航为估算值，会受驾驶习惯、空调、温度、路况影响。冬季低温续航会有明显下降，需预留更多余量。
2.  **空调使用**：
    -   **制热**：PTC加热器耗电较高，冬季开暖风会显著降低续航。建议合理设置温度（如22℃左右），配合座椅加热使用更节能。
    -   **制冷**：空调制冷耗电相对较低，但高速行驶时开窗比开空调更费电。
3.  **充电建议**：
    -   **日常充电**：建议使用慢充，保持电量在20%-80%之间，有利于延长电池寿命。
    -   **快充**：长途急需时使用，频繁快充可能加速电池衰减。
    -   **冬季充电**：低温下充电速度会变慢，建议在刚行驶完（电池尚有余温）时立即充电。
    -   **充电安全**：充电前检查充电枪和接口无积水、无异物。充电过程中不要在车内长时间停留（尤其是使用快充时）。
## 四、安全与应急处理
1.  **底盘与电池**：行驶中避免托底，底盘下方的动力电池包是核心部件，一旦受到严重撞击或刮擦，应立即停车检查，必要时联系售后。
2.  **异常报警**：若仪表盘出现红色高压系统故障灯或动力电池故障灯，请立即安全靠边停车，关闭电源并联系江淮官方救援。
3.  **拖车注意事项**：
    -   **严禁**在前轮（驱动轮）着地的情况下长距离拖车，否则会损坏电机。
    -   必须使用平板拖车或抬起前轮（驱动轮）进行拖运。
4.  **长时间停放**：
    -   若车辆需停放超过一周，建议保持电量在50%-70%之间。
    -   长时间停放后，再次使用前请先慢充充满一次，让电池管理系统进行均衡。
## 五、日常保养小贴士
1.  **保养周期**：电动车保养间隔通常较长（如每1万公里或1年），以先到为准。主要检查高压线束、冷却液、减速器油、空调滤芯等。
2.  **轮胎换位**：定期进行轮胎换位，保证均匀磨损。
3.  **软件升级**：关注厂家发布的车辆控制软件（VCU/BMS）升级通知，及时到店升级可优化续航和驾驶体验。
> **核心总结**：温柔驾驶、预判减速、浅充浅放、避免托底、冬季预留余量。

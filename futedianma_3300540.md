# 福特电马（Mustang Mach-E）儿童乘车安全注意事项
  
更新时间：2026年07月13日12时54分32秒   

## 一、核心安全原则
1.  **儿童乘车必须使用合适的儿童安全座椅**，严禁怀抱或使用成人安全带。
2.  **12岁以下或身高不足150cm的儿童**，建议始终坐在后排座位。
3.  请根据孩子的体重、身高和年龄，选择符合国家3C认证的儿童座椅（反向安装、正向安装或增高垫）。
## 二、福特电马专属注意事项
### 1. 车门与车窗安全
-   **隐藏式门把手**：电马采用隐藏式门把手，请确保孩子不会在行驶中或停车时随意触碰把手。家长应开启**后门儿童安全锁**（位于后车门边缘的机械开关），防止孩子从车内打开车门。
-   **电动车窗**：开启**车窗锁止功能**（驾驶员侧车门控制面板），防止孩子误触车窗升降按钮导致夹伤。
### 2. 电动动力系统
-   **静音启动风险**：电动汽车启动时几乎无声。停车时务必确认孩子远离车辆前后方，避免因车辆突然移动发生意外。
-   **高压部件警告**：严禁让孩子触碰充电口、高压线束或前备箱下的高压部件。充电时，请将充电线缆放置在孩子接触不到的地方。
### 3. 智能座舱与屏幕
-   **中控大屏操作**：15.5英寸中控屏集成了大量功能。建议在行驶前设置好导航和娱乐内容，避免孩子在行车途中触碰屏幕影响驾驶安全。
-   **游戏与视频模式**：驻车时可以使用车机娱乐功能，但请合理控制观看时长，保护视力。
### 4. 后备箱与前备箱
-   **严禁玩耍空间**：绝对禁止孩子钻入后备箱或前备箱玩耍。一旦意外锁闭且车内无人知晓，极易导致窒息危险。
-   **电动尾门防夹**：开启后备箱时确保孩子站在安全距离外。部分车型配备防夹功能，但仍需留意。
## 三、日常使用提醒
| 项目 | 注意事项 |
| :--- | :--- |
| **安全带使用** | 所有乘员（包括后排）均需系好安全带。切勿让安全带勒住孩子的脖子或面部。 |
| **空调与通风** | 避免空调出风口直吹儿童面部或腹部；夏季暴晒后先开窗通风再上车。 |
| **车内物品摆放** | 中控台、座椅上不要放置尖锐或硬质物品；紧急刹车时可能飞溅伤及儿童。 |
| **单独留车内的风险** | <u>任何时候都不要将儿童单独留在车内</u>（尤其在夏季高温下），电马虽可远程开空调但仍有断电风险。 |
## 四、紧急情况应对
1.  如果车辆发生碰撞且无法从内部开门：
    -   拉动后排车门内侧的物理机械开门拉手（位于储物格内）。
    -   使用破窗器击碎侧窗玻璃逃生。
2.  如果车辆落水：
    -   立即解开安全带并打开车门/车窗逃生。
    -   若电子系统失效，使用机械拉手开门。
## ⚠️ “三不”原则总结
-   ❌ **不坐副驾驶**
-   ❌ **不玩门把手和车窗**
-   ❌ **不被单独留在车内**
> *以上内容仅供参考，具体操作请以福特电马车主手册为准。*

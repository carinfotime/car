# 比亚迪元UP 技术参数介绍
  
更新时间：2026年07月11日15时49分54秒   

## 车型概览
比亚迪元UP是比亚迪e平台3.0打造的首款A0级纯电SUV，定位为“年轻人的第一台车”。它融合了潮趣龙颜美学设计、高效八合一电动力总成以及智能科技配置，旨在提供高颜值、大空间、易操控和高安全性的城市通勤与轻度户外出行体验。
---
## 核心参数
### 动力系统
| 项目 | 参数 |
| :--- | :--- |
| **驱动电机类型** | 永磁同步电机 |
| **电机最大功率** | 70 kW (95 Ps) / **高功率版：** 130 kW (177 Ps) |
| **电机最大扭矩** | 180 N·m / **高功率版：** 290 N·m |
| **最高车速** | ≥150 km/h (受电子限速) |
| **0-50km/h加速时间** | **高功率版：** 约4.0秒 |
### 电池与续航
| 项目 | 参数 |
| :--- | :--- |
| **电池类型** | 磷酸铁锂刀片电池（比亚迪自研） |
| **电池容量** | **标准续航版：**约32 kWh / **长续航版/高功率版：**约45.1 kWh |
| **CLTC综合续航里程** | **标准续航版：**301 km / **长续航版：**401 km / **高功率版：**401 km |
| **快充时间 (30%-80%)** | ≥30分钟 (支持直流快充) |
### 车身尺寸
| 项目 | 参数 |
| :--- | :--- |
| **长 × 宽 × 高 (mm)** | `4310 × 1830 × 1675` |
| **轴距 (mm)** | `2620` |
| **最小离地间隙 (mm)** | ≥150 (空载) |
### 底盘与悬架
-   **驱动方式：**前置前驱
-   **前悬架类型：**麦弗逊式独立悬架
-   **后悬架类型：**扭力梁式非独立悬架
-   **转向系统：**EPS电动助力转向
-   **制动系统：**前通风盘式 / 后盘式制动器
-   **轮胎规格：**
    -   低配车型: `215/65 R16`
    -   中高配车型: `215/60 R17`

## 智能科技配置
### DiLink智能座舱
-   *10.1英寸/12.8英寸自适应旋转悬浮Pad*
-   支持智能语音交互（可控制导航、空调、多媒体等）
-   车载4G网络 + Wi-Fi热点
-   在线导航、影音娱乐应用生态（抖音、爱奇艺等）
-   OTA远程升级能力
### DiPilot智能驾驶辅助系统（部分配置搭载）
-   ACC自适应巡航控制
-   AEB自动紧急制动
-   LKA车道保持辅助
-   交通标志识别
-   全息透明影像系统（360°全景影像 +底盘透视）

## “易操控”驾驶模式选择
提供多种驾驶模式以适应不同路况：
1. ECO （经济模式） ——适合城市拥堵路况，延长续航。
2. SPORT （运动模式） ——动力响应更积极，超车更轻松。
3. SNOW （雪地模式） ——湿滑路面起步更平稳。

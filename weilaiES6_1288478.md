## 一、车辆启动与驾驶
### 1.1 启动车辆
1. **携带钥匙**：确保智能钥匙或绑定的手机（NFC/蓝牙钥匙）在车内。
2. **踩下制动踏板**：车辆将自动上电并进入“Ready”状态（仪表盘显示绿色READY指示灯）。
3. **选择挡位**：
   - **D挡（前进）**：将换挡拨杆向下拨到底。
   - **R挡（倒车）**：将换挡拨杆向上拨到底。
   - **N挡（空挡）**：短暂向上或向下拨动拨杆至中间位置保持1秒。
   - **P挡（驻车）**：按下换挡拨杆顶端的按钮。
### 1.2 驾驶模式选择
通过中控屏或方向盘右侧滚轮进入驾驶模式菜单，提供以下模式：
| 模式 | 特点 | 适用场景 |
| :--- | :--- | :--- |
| **舒适** | 悬架软、加速平缓、能量回收弱 | 日常城市通勤 |
| **运动** | 悬架硬、加速迅猛、能量回收强 | 追求驾驶乐趣 |
| **节能** | 限制空调功率、能量回收最强 | 长途续航优先 |
| **个性化** | 自定义动力、悬架、转向等参数 | 根据个人喜好调整 |
### 1.3 NIO Pilot (NOP) / Pilot辅助驾驶
- **开启ACC/Pilot**：在行驶中，向驾驶员方向拉动巡航控制拨杆两次。
- **调节车速/跟车距离**：
    - `SET +/-`：设定/微调当前速度。
    - `DIST +/-`：调节与前车的跟车距离（共5档）。
- **退出辅助驾驶**：
    - `轻点刹车`或`向外推巡航拨杆一次`可退出ACC/Pilot。
    - `向上推动巡航拨杆`可完全退出系统。
> ⚠️ *注意：辅助驾驶仅为辅助功能，驾驶员必须始终保持对车辆的掌控，双手不得离开方向盘。*

## 二、充电与补能
### 2.1 NIO Power换电
蔚来核心服务，快速补充电量。
1. **下单预约**：通过NIO App搜索附近换电站并下单排队。
2. **驶入换电站**：
    - NIO Pilot模式下车辆可自动泊入换电站（需支持自动换电的站点）。
    - 手动泊入时，请严格按照地面引导线行驶至指定位置。
3. **自动换电过程**：
    - NOMI会语音提示操作步骤。请勿踩刹车或转动方向盘。
    - *注意*：部分站点需手动点击中控屏“开始换电”按钮。
4. **完成驶出**：
    - NOMI提示“换电完成”后，挂D挡驶出。
### 2.2 AC/DC充电桩充电
- **交流慢充 (AC)**：
    1.  打开车辆左侧后方的充电口盖板。
    2.  插入交流充电枪直至听到“咔哒”锁止声。
    3.  可通过NIO App或中控屏设置预约充电时间及目标电量。
- **直流快充 (DC)**：
    1.  打开车辆右侧后方的充电口盖板（部分车型为左侧）。
    2.  插入直流快充枪。通常建议在电量20%-80%区间进行快充以保护电池。

## 三、智能座舱与NOMI
### 3.1 NOMI语音助手
唤醒方式：“Hi NOMI”或按下方向盘右侧语音键。
常用指令示例：
| 功能类别 | 语音指令示例 |
| :--- | :--- |
| ***导航*** | “导航到上海中心大厦”、“去最近的换电站” |
| ***空调*** | “空调温度调到24度”、“打开座椅通风” |
| ***车窗*** | “打开所有车窗”、“关闭天窗遮阳帘” |
| ***媒体*** | “播放周杰伦的歌”、“调大音量” |
| ***车辆控制*** | “开启哨兵模式”、“打开后备箱” |
### 3.2 HUD抬头显示
- HUD可显示车速、导航信息、限速标志及辅助驾驶状态。
- *调节*：在中控屏“设置 > HUD”中调整亮度、高度及倾斜角度。
### 3.3 NIO Radio & UWB数字钥匙
- UWB数字钥匙支持手机靠近车门自动解锁/落锁。需在手机蓝牙设置中授权NIO App使用UWB权限。

### ✅日常安全提醒
- 【儿童安全】切勿让儿童单独留在车内；12岁以下儿童应使用后排儿童安全座椅并锁定车窗升降开关(主驾门板上)。

# 铂智3X 内饰参数介绍
  
更新时间：2026年07月16日12时57分43秒   

## 整体设计风格
- **主题定位**：简约科技风，融合东方美学与现代工业设计
- **内饰配色**：提供深色/浅色双拼方案，可选装氛围灯套件
## 座椅系统
| 项目 | 参数 |
|------|------|
| 座椅材质 | Nappa真皮（高配）/ 仿皮+织物混搭（中低配） |
| 前排座椅 | 电动8向调节（主驾）+ 4向调节（副驾） |
| 加热/通风 | 前排座椅加热（全系标配），通风功能（高配） |
| 后排座椅 | 4/6比例放倒，中央扶手带杯架 |
| 头枕 | 前后排独立可调头枕 |
## 仪表盘与中控
- **仪表盘**：12.3英寸全液晶仪表盘，支持多种主题切换
- **中控屏**：14.6英寸悬浮式触控屏，分辨率1920×1080
- **车机系统**：自研智能座舱OS，支持OTA远程升级
- **语音控制**：双区语音识别，支持连续对话与免唤醒指令
## 方向盘
- **类型**：D型三辐多功能方向盘
- **材质**：真皮包裹（高配）/ PU皮革（中低配）
- **功能键**：多媒体控制、定速巡航、语音唤醒、蓝牙电话
## 空调系统
- **类型**：双区自动恒温空调
- **后排出风口**：标配，带独立风量调节
- **空气净化**：PM2.5过滤 + N95级滤芯 + AQS空气质量监测
## 储物空间
| 位置 | 容量/特点 |
|------|-----------|
| 中央扶手箱 | ≈8L，内置USB-C充电口 |
| 手套箱 | ≈12L，带阻尼开启 |
| 门板储物格 | ≥1.5L/侧，可放水瓶+杂物 |
| 杯架 |前排2个 +后排2个 |
## NVH与舒适配置
- **隔音玻璃**：前排双层夹胶隔音玻璃（高配）
- **主动降噪**：ANC主动降噪系统（选装）
- **无线充电**：前排50W无线快充面板
## Lighting & Atmosphere
- **氛围灯**：64色可调环绕氛围灯（高配），支持呼吸/律动模式
- **阅读灯**：前后排LED阅读灯，触控开关
## Other Details
- **遮阳板**：主副驾带化妆镜+照明灯
- **天窗/天幕**：
    -  全景天幕（标配）：面积≈1.2㎡，防紫外线涂层
    -  电动遮阳帘（高配）：一键开闭

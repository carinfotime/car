# 威虎汽车操作指南
  
更新时间：2026年07月15日17时20分51秒   

## 一、车辆概述
威虎（Weihu）是中兴汽车旗下的一款经典皮卡车型，以其坚固耐用、通过性强、承载能力出色而著称。本指南适用于威虎系列各主要车型，包括柴油版与汽油版、两驱与四驱版本。
---
## 二、启动与熄火
### 2.1 启动前检查
- 确认挡位处于 **空挡（N）** 或 **P挡**（自动挡车型）
- 检查手刹是否拉紧
- 确认各灯光、仪表工作正常
- 检查轮胎气压及外观
### 2.2 启动发动机
**柴油版：**
1. 将钥匙插入点火开关
2. 等待 **预热指示灯** 熄灭（冬季需预热5-10秒）
3. 踩下离合器踏板（手动挡）或刹车踏板（自动挡）
4. 将钥匙拧至"START"位置启动
5. 启动后松开钥匙，怠速运转30秒以上再行驶
**汽油版：**
1. 踩下离合器或刹车踏板
2. 将钥匙拧至"START"位置
3. 启动后松开钥匙
### 2.3 熄火操作
1. 将车辆停稳，拉紧手刹
2. 手动挡挂入空挡，自动挡挂入P挡
3. 将钥匙拧回"LOCK"位置
4. 长时间停车后，柴油版建议怠速1-2分钟再熄火，以保护涡轮增压器

## 三、驾驶操作
### 3.1 手动挡操作
| 挡位 | 用途 | 推荐换挡转速（柴油） | 推荐换挡转速（汽油） |
|------|------|---------------------|---------------------|
| 1挡 | 起步、爬陡坡 | — | — |
| 2挡 | 低速行驶 | 1500-2000 rpm | 2000-2500 rpm |
| 3挡 | 城市道路 | 1500-2000 rpm | 2000-2500 rpm |
| 4挡 | 郊区行驶 | 1500-2000 rpm | 2000-2500 rpm |
| 5挡 | 高速巡航 | 1500-2000 rpm | 2000-2500 rpm |
| R挡 | 倒车 | — | — |
> **注意：** 挂倒挡前必须完全停稳车辆，踩下离合器3秒后再挂入。
### 3.2 自动挡操作
- **P（驻车挡）：** 停车时使用，必须车辆完全停止后挂入
- **R（倒车挡）：** 倒车时使用
- **N（空挡）：** 短暂停车、拖车时使用
- **D（前进挡）：** 正常行驶使用
- **2/L/1（低速挡）：** 爬坡、重载或下长坡时使用
### 3.3 四驱系统操作（四驱版）
| 模式 | 适用场景 | 操作方式 |
|------|---------|---------|
| **2H（两驱高速）** | 正常公路行驶 | 默认模式 |
| **4H（四驱高速）** | 湿滑、泥泞、沙石路面 | 车速低于80km/h时旋钮切换 |
| **4L（四驱低速）** | 陡坡、深泥、脱困 | 停车挂N挡后切换 |
> **重要：** 干燥硬路面上请勿使用四驱模式，否则会损坏传动系统。

## 四、载货与牵引
### 4.1 货箱载货
- **最大载重量：** 请参考车辆铭牌（通常为500-800kg，视具体型号而定）
- 货物应均匀分布，重心尽量靠前
- 超长货物需开启尾门并固定警示标志
- 货物必须固定牢靠，防止行驶中移位
### 4.2 牵引拖车
- 威虎最大牵引质量请参见说明书（通常为1500-2000kg）
- 牵引时建议使用 **4H** 模式
- 牵引时适当增加轮胎气压
- 下坡时使用发动机制动，避免刹车过热

| 车速表 | 显示当前车速（km/h） |
| 转速表 | 显示发动机转速（×1000 rpm） |
| 燃油表 | 显示油箱剩余油量 |
| 水温表 | 显示发动机冷却液温度（正常：90℃左右） |
| 里程表 | 记录总行驶里程 |
| 短程里程表 | 可归零，记录单次行程 |
### 5.2 重要指示灯
| 指示灯 | 颜色 | 含义 | 处理方式 |
|--------|------|------|---------|
| 发动机故障灯 | 黄色/橙色 | 发动机系统故障 | 尽快检修 |
| 机油压力灯 | 红色 | 机油压力过低 | 立即停车检查 |
| 电池充电灯 | 红色 | 充电系统故障 | 尽快检修 |

| 安全气囊灯 | 红色 | 气囊系统故障 | 尽快检修 |
| 四驱指示灯 | 绿色 | 四驱已接通 | — |
| 预热指示灯 | 黄色 | 柴油预热中 | 等待熄灭后启动 |
| 胎压监测灯 | 黄色 | 胎压异常 | 检查轮胎 |

## 六、保养维护
### 6.1 定期保养周期
| 项目 | 周期 |
|------|------|
| 发动机机油更换 | 每5000公里或6个月 |
| 机油滤清器更换 | 每5000公里 |
| 空气滤清器检查/更换 | 每10000公里 |
| 燃油滤清器更换（柴油版） | 每10000公里 |
| 变速箱油更换 | 每40000公里 |
| 后桥齿轮油更换 | 每40000公里 |
| 分动箱油更换（四驱版） | 每40000公里 |
| 刹车油更换 | 每2年或40000公里 |
| 冷却液更换 | 每2年或60000公里 |
| 正时皮带检查/更换 | 每60000-80000公里 |
### 6.2 日常检查项目
- **每周：** 检查机油液位、冷却液液位、刹车油液位、轮胎气压
- **每月：** 检查灯光、雨刮器、玻璃水、电瓶接线柱
- **每次长途前：** 全面检查上述项目及备胎、随车工具
### 6.3 轮胎维护
- 标准胎压：前轮 **2.5 bar**，后轮 **3.0 bar**（空载）/ **3.5 bar**（满载）
- 轮胎换位：每10000公里交叉换位
- 轮胎磨损标记：花纹深度低于1.6mm时必须更换

## 七、应急处理
### 7.1 发动机无法启动
1. 检查燃油量
2. 检查电瓶连接是否松动
3. 柴油版检查预热系统是否正常工作
4. 检查启动机是否有反应
5. 如仍无法启动，联系救援
### 7.2 行驶中水温过高
1. 立即靠边停车，挂空挡，怠速运转
2. 打开暖风最大挡辅助散热
3. 切勿立即打开水箱盖（高温蒸汽危险）
4. 等待发动机冷却后检查冷却液液位
5. 如冷却液不足，待冷却后添加
### 7.3 轮胎爆胎
1. 紧握方向盘，保持直线行驶
2. 松开油门，不要急刹车
3. 让车辆自然减速，轻踩刹车
4. 驶离主干道至安全区域
5. 放置三角警示牌（车后50-100米）
6. 更换备胎或呼叫救援
### 7.4 电瓶亏电搭电
1. 关闭两车所有用电设备
2. 红色夹子连接两车正极（+）
3. 黑色夹子一端连接救援车负极（-）
4. 黑色夹子另一端连接被救车发动机搭铁点（远离电瓶）
5. 启动救援车，怠速运转2分钟
6. 启动被救车
7. 按相反顺序拆除搭电线
---
## 八、安全驾驶提示
1. **空载行驶：** 威虎空载时后悬较硬，经过颠簸路面时减速慢行
2. **重载行驶：** 刹车距离会增加，提前预判，保持更大车距
3. **雨天行驶：** 皮卡后轴较轻，湿滑路面易甩尾，避免急转弯
4. **下长坡：** 使用发动机制动（挂低速挡），避免长时间踩刹车
5. **盲区注意：** 货箱较高，倒车时注意后方低矮障碍物
6. **系好安全带：** 所有乘员必须全程系好安全带
7. **严禁超载：** 超载不仅违法，还会严重影响操控和制动
---
## 九、常见问题与解答
**Q：威虎柴油版冬季启动困难怎么办？**
A：确保使用适合当地气温的柴油（0号/-10号/-20号），检查预热系统是否正常，电瓶电量充足。
**Q：四驱模式切换时有异响？**
A：可能是在行驶中切换或车速过高，建议停车挂N挡后重新切换。如仍有异响，请到服务站检查。
**Q：货箱门关不严怎么办？**
A：检查锁扣是否对齐，可调整锁扣位置或涂抹润滑脂。
**Q：仪表盘上黄灯亮了还能继续开吗？**
A：黄色指示灯通常表示警示，可谨慎行驶至维修站；红色指示灯表示严重故障，应立即停车检查。

# 广汽埃安 AION i60 技术参数介绍
  
更新时间：2026年07月10日20时16分57秒   

## 车型概述
广汽埃安 AION i60 是广汽新能源旗下的一款纯电动紧凑型轿车，基于广汽第二代纯电专属平台 GEP 2.0 打造，定位为高性价比家用电动轿车。
---
## 基本参数
| 项目 | 参数 |
|------|------|
| 车型级别 | 紧凑型轿车 |
| 车身结构 | 4门5座三厢车 |
| 长 × 宽 × 高 (mm) | **4768 × 1880 × 1530** |
| 轴距 (mm) | **2750** |
| 整备质量 (kg) | **约1610** |
| 行李厢容积 (L) | **约453** |
---
## 动力系统
| 项目 | 参数 |
|------|------|
| **驱动电机类型** | **永磁同步电机** |
| **电机最大功率 (kW)** | **135 kW（约184马力）** |
| **电机最大扭矩 (N·m)** | **300 N·m** |
| **最高车速 (km/h)** | ≥150 km/h |
| **0-100km/h加速时间 (s)** | **约9.7秒** |

## 电池与续航
### i60（标准续航版）
| 项目 | 参数 |
|------|------|
| **电池类型** | **三元锂电池 /磷酸铁锂电池（视批次）** |
| **电池容量 (kWh)** | **约58.8 kWh（三元锂）/约55.2 kWh（磷酸铁锂）** |
| NEDC综合续航里程 (km) | ≥460 km（三元锂版）/ ≥430 km（磷酸铁锂版） |
### i60 Plus / i60 Max（长续航版）
> *注：部分版本命名可能因年份批次不同有所差异*
- NEDC综合续航里程：≥510 km
- CLTC综合续航里程：≥500 km
---
## 充电性能
- **快充时间：**
    - SOC从30%充至80%：约30分钟
    - （快充功率最高支持约100kW直流快充）
- **慢充时间：**
    - SOC从0%充至100%（使用家用交流充电桩）：约8~10小时
---
## 底盘与悬架系统
-  驱动方式：前置前驱
-  前悬架类型：麦弗逊式独立悬架
-  后悬架类型：扭力梁式非独立悬架
-  转向系统：电动助力转向（EPS）
-  制动系统：
    -  前制动器：通风盘式
    -  后制动器：盘式
    -  驻车制动类型：电子手刹（EPB）
---
## 轮胎与轮毂
-  轮胎规格：
    - i60基础版：215/55 R17
    - i60高配版/Plus版：215/50 R18
-  轮毂材质：铝合金轮毂
---

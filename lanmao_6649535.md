# 蓝猫 内饰参数介绍

| :--- | :--- |
| **座椅材质** | Nappa 真皮 + Alcantara 拼接 |
| **前排座椅功能** | - 12向电动调节（含4向腰托）<br>- 加热 / 通风 / 按摩<br>- 驾驶席记忆功能（3组） |
| **后排座椅** | - 材质与前排一致<br>- 加热功能<br>- 靠背角度电动调节（25°-40°）<br>- 中央扶手带触控屏及杯架 |

| 项目 | 参数 / 配置 |

| **类型** | D型平底运动方向盘，真皮包裹 + Alcantara防滑处理 |

## 四、氛围与照明
-   **氛围灯系统**
    -   LED光带环绕式布局
    -   256色可调
    -   支持呼吸模式/音乐律动模式/分区独立设定
-   **阅读灯**
    -   前后排独立LED阅读灯，亮度无极调节
-   *迎宾灯*
    -   车门底部投射“蓝猫”品牌标识光影
## 五、空调与空气净化
-   *空调类型*:自动双区恒温空调 (支持后排独立控制)
-   *空气净化系统*
    -   PM2.5过滤效率 ≥99%
    -   负离子发生器
    -   AQS空气质量监测自动切换内外循环
-   *香氛系统*
    -    内置3种香氛 (海洋/森林/花香)
    -    浓度三档可调
## 六、音响系统
-   *品牌*:定制Hi-Fi音响系统 (可选配Bose或Burmester)
-   *扬声器数量*:14个 (含低音炮)
-   *总功率*:800W
-   *技术特点*
    -    7.1声道环绕声
    -    主动降噪补偿 (RNC)
    -    支持无损音频格式 (FLAC/WAV)
## 七、储物空间与便利性
1.  中央扶手箱容积约8L，内置USB-C快充口 ×2 +无线充电板(50W风冷)
2.  门板储物格可容纳1L水瓶 ×2
3.  手套箱带制冷功能(可冷藏饮料)
4.    后排中央扶手集成触控屏及弹出式杯架
5.    后备箱侧壁挂钩 ×2 +网兜 ×1
## 八、其他细节配置

-     全景不可开启天幕(双层夹胶玻璃，紫外线隔绝率99%)
-     可选装电动遮阳帘(一键开合)
*车窗*
-     全车双层隔音玻璃(前挡+四门)
-     后窗隐私玻璃标配
*遮阳板*
-     主副驾均带LED柔光化妆镜+票据夹

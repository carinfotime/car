# 依维柯聚星新能源 保养指南
  
更新时间：2026年07月12日17时01分39秒   

## 一、保养周期总览
| 保养项目 | 周期 |
|---------|------|
| 首次保养 | 行驶 **3,000公里** 或 **购车后3个月**（以先到为准） |
| 常规保养 | 每 **10,000公里** 或 **6个月**（以先到为准） |
| 空调滤芯更换 | 每 **10,000公里** |
| 减速器油更换 | 每 **40,000公里** |
| 冷却液更换 | 每 **40,000公里** |
| 制动液更换 | 每 **20,000公里** |
| PTC加热器检查 | 每年入冬前 |
---
## 二、日常检查（每日/每次出车前）
- [ ] **轮胎外观与胎压**：检查有无鼓包、划伤，胎压保持在推荐值（驾驶位门框有标注）
- [ ] **灯光系统**：大灯、转向灯、刹车灯、雾灯是否正常
- [ ] **仪表盘警示灯**：启动后观察有无异常报警
- [ ] **电量与续航**：确认剩余电量满足当日行程需求
- [ ] **制动系统**：轻踩刹车感受制动力是否正常
- [ ] **雨刮与玻璃水**：确保雨刮片无老化，玻璃水充足
---
## 三、定期保养项目详解
### 🔹 A类保养（每10,000公里 / 6个月）
1. **高压系统检查**
   - 高压线束及连接器有无松动、腐蚀
   - DC-DC转换器工作状态
   - PDU（配电单元）功能检测
2. **电池系统检测**
   - OBD读取电池SOC/SOH数据
   - BMS系统升级检查
   - 电池包外观及密封性检查
3. **空调滤芯更换**
   - PM2.5过滤芯建议使用原厂件
4. **底盘与悬挂**
   - 紧固底盘螺栓
   - 检查减震器有无漏油
   -转向拉杆及球头间隙检查
5. **制动系统**
   -刹车片厚度测量（低于3mm需更换）
   -刹车盘磨损情况
6. **轮胎换位**
   -前后交叉换位，延长使用寿命
7. **润滑与紧固**
   -车门铰链、锁扣润滑
   -座椅滑轨润滑
### 🔹 B类保养（每40,000公里 /2年）
在A类基础上增加：
1. ✅减速器油更换（约1.5L，规格请参照用户手册）
2. ✅冷却液更换（建议使用原厂长效冷却液）
3. ✅制动液更换（DOT4级别）
4. ✅PTC加热器功能测试与清洁
5. ✅全车高压绝缘检测
6. ✅电机控制器冷却系统清洗

## 四、季节性保养要点
### ❄️冬季入冬前准备
- PTC加热器功能测试，确保暖风正常输出
-检查12V小电瓶电压，低于12V建议充电或更换
-轮胎花纹深度≥1.6mm，雪地地区建议换冬季胎
-玻璃水换成防冻型(-25℃以下)
### ☀️夏季高温前准备  
-空调制冷效果测试，必要时补充冷媒  
-散热风扇及冷凝器表面清洁  
-充电口防尘盖密封性检查  
-避免在高温暴晒后立即快充  
---
## 五、充电注意事项对电池寿命的影响  
|场景|建议做法|
|---|---|
|日常通勤 |保持电量在20%~80%之间使用|
|长途出行前 |充满至100%，但不要长时间保持满电存放|
|长期停放(超过7天)|电量保持在50%~60%，断开12V小电瓶负极|
|快充使用频率 |建议每周不超过3次快充，多用慢充|
|环境温度过低(<-10℃)|充电前先预热电池(部分车型自动执行)|

|部件名称 |建议更换周期|

## 七、常见故障提示与应对  
⚠️仪表显示"动力电池故障"
→立即停车熄火→联系售后救援→不要自行拆卸高压部件  
⚠️仪表显示"绝缘故障"
→停止使用车辆→联系服务站检修→可能存在漏电风险  
⚠️充电中途中断且指示灯异常闪烁  
→拔下充电枪等待30秒重新插拔→如仍失败换一个充电桩测试→若依旧报错进店检修  
⚠️行驶中突然失去动力但仪表亮着  
→打开双闪靠边停车→挂P挡关闭电源等待1分钟重启→若无法恢复呼叫救援  
---
## 八、重要提醒  
1️⃣所有涉及高压系统的维修必须由授权服务站操作。严禁私自拆解电池包或高压线束。  
2️⃣保养时务必携带《三电终身质保凭证》，按规定在授权网点进行保养方可享受质保。  
3️⃣依维柯聚星新能源的OBD诊断接口位于方向盘左下方盖板内。  
4️⃣不同年款/配置的车型参数可能略有差异，具体以随车《用户手册》为准。  
5️⃣建议每次进店保养时要求技师打印《高压系统检测报告》留存备查。

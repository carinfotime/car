# 吉奥帅豹 技术参数介绍
  
更新时间：2026年07月13日12时32分05秒   

## 一、车型概述
吉奥帅豹是广汽吉奥旗下的一款硬派SUV车型，采用非承载式车身结构，兼具城市驾驶舒适性与越野通过能力。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **发动机型号** | 4G69（三菱技术） |
| **发动机类型** | 直列四缸、水冷、四冲程、自然吸气 |
| **排量** | 2.4L（2378ml） |
| **最大功率** | 100kW / 5250rpm |
| **最大扭矩** | 200N·m / 2500-3000rpm |
| **燃油供给方式** | 多点电喷（MPI） |
| **燃油标号** | 92#及以上无铅汽油 |
| **排放标准** | 国IV / 国V |
---
## 三、传动系统
| 项目 | 参数 |
|------|------|
| **变速箱类型** | 5速手动变速箱 / 5速自动变速箱 |
| **驱动方式** | 前置后驱（FR）/ 分时四驱（4WD） |
| **分动箱** | 机械式分时四驱，带低速四驱挡 |
---
## 四、底盘与悬挂
| 项目 | 参数 |
|------|------|
| **车身结构** | 非承载式车身（带大梁） |
| **前悬挂类型** | 双横臂独立悬挂 + 扭杆弹簧 |
| **后悬挂类型** |  整体桥式非独立悬挂 +  钢板弹簧/螺旋弹簧（视配置而定） |
---
##  五、转向与制动系统
### 🔹转向系统
- **转向器类型：**  齿轮齿条式液压助力转向
- **最小转弯半径：** ≤6.5m
### 🔹制动系统
- **前制动器：**  通风盘式
- **后制动器：**  鼓式 /  盘式（高配）
- **制动辅助：** ABS防抱死 + EBD制动力分配
---
##  六、尺寸与重量
### 🔹外部尺寸
- **长 ×宽 ×高：**
    -  长：4620mm
    -  宽：1885mm
    -  高：1830mm
- **轴距：**  2720mm
- **轮距（前/后）：**
    -  前轮距：1520mm
    -  后轮距：1530mm
- **最小离地间隙：**
    - ≥210mm（空载）
- **接近角/离去角：**
    -  接近角 ≥28°
    -  离去角 ≥26°
### 🔹重量数据
- **整备质量：**
    - MT车型约1830kg
    - AT车型约1850kg

    - ≥2350kg

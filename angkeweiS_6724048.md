# 别克昂科威S 驾驶注意事项
  
更新时间：2026年07月18日09时36分41秒   

本指南旨在帮助您更安全、高效地驾驶别克昂科威S，延长车辆使用寿命，并充分发挥其性能。
## 一、 启动与驾驶前检查
1.  **绕车检查**：启动前，绕车一周检查轮胎气压、外观有无刮擦、车底有无漏油漏水迹象。
2.  **调整坐姿与后视镜**：确保座椅、方向盘、内外后视镜调整至最适合您的驾驶姿势，保证视野开阔。
3.  **系好安全带**：全车人员必须系好安全带，包括后排乘客。
4.  **检查仪表盘**：启动后，观察仪表盘是否有故障灯亮起（如发动机故障灯、胎压监测灯等）。
5.  **熟悉按键布局**：昂科威S采用按键式换挡，请熟悉 R/N/D 挡位的操作逻辑，避免行驶中误触。
## 二、 行驶中注意事项
### 1. 动力系统与变速箱
-   **48V轻混系统**：车辆带有48V轻混系统，启动和自动启停时更平顺。请勿在非专业维修店随意改装或断开48V电池线路。
-   **9速HYDRA-MATIC变速箱**：换挡平顺，但在急加速或爬坡时，变速箱需要时间降挡，请提前预判动力需求。
-   **避免空挡滑行**：自动变速箱严禁空挡滑行（N挡），这会导致变速箱油泵供油不足，严重损坏变速箱。
-   **自动启停功能**：该功能默认开启，可在中控屏幕中关闭。涉水行驶时，**务必关闭自动启停**，以防发动机在水中熄火后自动重启导致损坏。
### 2. 刹车系统
-   **磨合期**：新车的刹车片和刹车盘需要约300-500公里的磨合，此期间应避免紧急制动。
-   **刹车异响**：若刹车时有轻微“吱吱”声，可能是刹车片有硬点或表面有杂质，通常轻踩几次后消失。若持续异响或制动效果下降，请及时检查。
-   **涉水后**：通过积水路段后，应轻踩几次刹车，利用摩擦产生的热量蒸发刹车盘上的水分，恢复制动效能。
### 3. 辅助驾驶与电子系统
-   **主动安全系统**：昂科威S配备碰撞预警、主动刹车、车道保持等功能。这些系统是辅助，**不能完全替代**驾驶员的判断和操作。
-   **车道保持**：开启后，车辆会轻微修正方向盘。若您需要变道，请先打转向灯，系统会自动解除该侧的车道保持干预。
-   **360°全景影像**：倒车或低速通过狭窄区域时，请结合后视镜和影像系统，注意影像存在边缘畸变，不能完全依赖。
### 4. 特殊路况驾驶
-   **湿滑路面**：昂科威S为前驱或适时四驱。在雨雪、结冰路面，避免急加速、急刹车、急转向，防止车辆侧滑。
-   **越野/非铺装路面**：昂科威S定位城市SUV，底盘离地间隙有限。如需走烂路，请放慢车速，选择高底盘模式（若配备），避免托底或磕碰油底壳、变速箱。
-   **长下坡**：连续长下坡时，建议使用手动模式（M挡）或换挡拨片，将挡位控制在低挡位，利用发动机制动，减少刹车系统热衰减。
## 三、 停车与熄火
1.  **正确停车步骤**：踩住刹车 → 车辆停稳 → 挂入 N 挡 → 拉紧电子手刹 → 松开脚刹（确认车辆不溜车）→ 挂入 P 挡 → 熄火。
    -   *注意*：在坡道停车时，直接挂P挡再拉手刹，下次启动挂挡时可能会有很大的冲击感，且会损伤变速箱锁止机构。
2.  **熄火前**：先关闭空调、音响、车灯等用电设备，再熄火，可保护蓄电池。
3.  **电子手刹**：熄火后电子手刹会自动拉起（部分配置需手动拉起）。起步时，系好安全带并挂入D挡，轻踩油门，电子手刹会自动释放。
## 四、 保养与维护建议
1.  **机油**：请使用符合 Dexos1 认证的 0W-20 标号全合成机油，并严格按照保养手册（通常每5000公里或6个月）进行更换。
2.  **轮胎**：定期检查胎压（建议冷车胎压2.4-2.5bar），每行驶1万公里进行轮胎换位，以保证均匀磨损。
3.  **燃油**：建议使用 92 号及以上无铅汽油，避免使用劣质燃油导致发动机积碳或亮故障灯。
4.  **颗粒捕捉器**（GPF）：若车辆为国六B排放标准，搭载颗粒捕捉器。建议偶尔进行高速行驶（转速保持2000转以上行驶20分钟），帮助再生，避免堵塞导致油耗升高、动力下降。

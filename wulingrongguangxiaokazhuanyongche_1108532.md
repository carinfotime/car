# 五菱荣光小卡专用车 保养指南
  
更新时间：2026年07月16日16时10分21秒   

## 一、概述
五菱荣光小卡专用车（包括厢式运输车、仓栅车、翼开启厢式车等）基于五菱荣光小卡底盘改装，兼顾载货与耐用性。本指南适用于常规使用条件下的保养周期与项目，若车辆经常重载、长途或在恶劣环境下行驶，建议适当缩短保养间隔。

### （三）制动系统
-   **制动液液位及品质检查：**  每月或每次常规保养时。
-   **制动片厚度检查：**  每10,000公里。前片磨损至极限标记时立即更换。
-   **制动鼓/制动蹄片检查：**  每20,000公里或出现制动异响时。
-   **制动液更换：**  每2年或40,000公里。
### （四）转向与悬挂系统
-   **转向助力液（如有）：**  每月检查液位；必要时添加或按需更换。
-   **转向拉杆球头、横直拉杆间隙：**  每次保养时目视及晃动检查。
-   **前轮轴承润滑脂加注：**  每次拆装轮毂时；建议至少每年一次。
-   ***钢板弹簧/板簧销润滑：*** *对于经常重载的专用车，建议每次换油时对板簧销进行润滑脂加注。*
### （五）冷却系统
-   冷却液液位：每周或出车前目视副水箱刻度。
-   冷却液冰点检测：入冬前进行。
-   冷却液更换：2年或40,000~60,000公里。
### （六）电气与空调系统
-   蓄电池接线柱清洁及紧固：每月一次。
-   发电机皮带张紧度及裂纹检查：每次常规保养。
-   空调滤芯（如配备）：每年或按需清洁/更换。

## (三) “专用”部分的特殊注意事项
由于车辆为“专用车”（如厢式货车），以下部分需额外关注：
1. **[重要]货箱与底盘连接螺栓**
    定期（建议每次换油时）检查和紧固货箱与大梁之间的U型螺栓及连接支架螺栓。重载行驶后尤其需要复查。
2. **[重要]液压举升机构（仅限自卸类车型，若有此配置）：**
     -   每日检查液压管路有无渗漏；
     -   定期清洁液压油箱通气孔；
     -   液压油首次使用500小时后应过滤或更换；之后每年或1000小时换一次；
     -   举升臂关节处定期加注润滑脂。
3. **[重要]冷藏机组 /保温箱体(若有此配置)：**
     -   冷凝器散热片每月用压缩空气吹净灰尘；
     -   制冷剂压力观察窗定期查看气泡情况；
     -   箱体密封条保持清洁无破损。
4. **[重要]仓栅式货箱立柱及锁扣(若有此配置)：**
     -   经常性目视焊接处有无开裂；
     -   活动立柱锁扣保持灵活并涂抹少量润滑脂防锈。
5. **[重要]翼开启厢式车上装机构(若有此配置)：**
     -   两侧翼板铰链、连杆机构每周加注润滑油；
     -   液压撑杆接口处保持清洁无泥沙；

## (四)日常出车前“绕车一圈”自检清单(每日)
1. [ ]外观灯光齐全有效(大灯、转向灯、刹车灯)。
2. [ ]轮胎气压正常且无鼓包裂纹；备胎固定牢固。
3. [ ]货箱门锁扣关闭到位；栏板挂钩挂好；篷布绑扎牢固(如有)。
4. [ ]发动机机油尺抽出查看油位在上下线之间。
5. [ ]冷却液副水箱水位在MIN-MAX之间。
6. [ ]挡风玻璃清洗液充足；雨刮片能刮净玻璃。
7. [ ]启动后仪表盘无故障灯常亮(特别是ABS灯和发动机故障灯)。
8. [ ]原地转动方向盘无异响且助力正常(如有)。
9. [ ]踩下刹车踏板感觉坚实不疲软；松开手刹后车辆可自由推动一小段距离(平地)。

## (五)季节性重点维护
### 🌞夏季来临前：
1.清理水箱散热器和冷凝器表面的柳絮和污物；
2.**重点注意**:夏季高温易导致轮胎气压升高引发爆胎——请严格按标准气压充气并避免长时间超高速行驶;
3.**重点注意**:空调制冷效果变差应及时检修制冷剂泄漏;
4.**重点注意**:雨刮片老化及时换新以应对雨季暴雨天气;
### ❄️冬季来临前：
1.**重点注意**:防冻液的冰点应低于当地最低气温至少10℃以上;严禁用水代替防冻液以免冻裂缸体;
2.**重点注意**:蓄电池储电量下降明显;若启动无力应及时充电或换新;
3.**重点注意**:柴油版车主应根据气温及时切换为相应标号的低凝点柴油(-10号/-20号等);
4.**重点注意**:预热塞功能应在寒冷天气下确保正常工作;

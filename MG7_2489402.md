# MG7 技术参数介绍
  
更新时间：2026年07月15日11时01分14秒   

## 一、车型概览
MG7 是上汽名爵旗下的一款运动型轿跑车，定位为中型轿车，主打年轻化、运动化设计风格，搭载高性能动力总成与智能化配置。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **发动机类型** | 2.0T 涡轮增压发动机 / 1.5T 涡轮增压发动机 |
| **排量** | 2.0L（1986cc）/ 1.5L（1490cc） |
| **最大功率** | 2.0T：192kW（261Ps）/ 1.5T：138kW（188Ps） |
| **最大扭矩** | 2.0T：405N·m / 1.5T：300N·m |
| **变速箱** | 2.0T：9速手自一体变速箱（9AT）/ 1.5T：7速双离合变速箱（7DCT） |
| **驱动方式** | 前置前驱 |
| **最高车速** | 230 km/h（2.0T） |
| **0-100km/h加速时间** | 6.5秒（2.0T） |

## 四、底盘与悬挂
- **前悬挂类型**：麦弗逊式独立悬挂
- **后悬挂类型**：多连杆式独立悬挂
- **制动系统**：前通风盘 / 后实心盘
- **转向系统**：电动助力转向（EPS）
- **轮胎规格**：225/50 R18 /  245/40 R19（高配）
---
## 五、燃油与排放
- **燃油标号**：92号及以上无铅汽油
- **油箱容积**：约60 L
- **排放标准**：国VI b
---
## 六、智能科技与安全配置
### 🧠智能座舱
-  12.3英寸全液晶仪表盘 +  12.3英寸中控触摸屏
-  斑马维纳斯智能车机系统
-  支持语音控制、在线导航、OTA远程升级
-  手机远程控制（解锁/启动/空调等）
### 🛡️主动安全系统
-  自适应巡航控制（ACC）
-  车道保持辅助（LKA）
-  主动刹车 / AEB自动紧急制动
-  360°全景影像 +透明底盘
-  前后驻车雷达
### 🔒被动安全
-  高强度钢车身结构
-  前排双气囊 +侧气囊 +侧气帘
-  预紧式安全带 + ISOFIX儿童座椅接口

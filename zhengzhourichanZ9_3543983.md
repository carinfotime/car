# 郑州日产 Z9 空间参数介绍
  
更新时间：2026年07月12日11时30分51秒   

## 车身尺寸
- **长 × 宽 × 高 (mm)**：约 5520 × 1850 × 1835 （具体以官方配置为准）
- **轴距 (mm)**：约 3150
- **货箱尺寸 (mm)**：约 1760 × 1562 × 475
- **最小离地间隙 (mm)**：约 215
## 乘坐空间
| 项目 | 参数 |
|------|------|
| **前排头部空间** | ≥1000 mm |
| **后排腿部空间** | ≥900 mm |
| **后排头部空间** | ≥980 mm |
| **后排座椅宽度** | ≥1450 mm |
| **标准乘坐人数** | 5人 |
> *注：以上数据基于标准双排座车型，实际空间因配置版本略有差异。*
## 储物与载货能力
### 🚛 **货箱**
- **货箱类型**：长货箱 / 标准货箱（视版本而定）
- **货箱内部平整度**：纯平地板设计，便于装载大件货物
- **最大载重质量**：约800–1000 kg（具体视公告型号）
### 🧳 **车内储物**
- **手套箱容积**：约8 L
- **中央扶手储物盒**：可容纳6瓶500ml矿泉水
- **前后门板储物格**：
    - *前门*：可放置1.5L水瓶 + A4文件
    - *后门*：可放置1L水瓶 +杂物
- **杯架数量**：
    - *前排*：2个（中央通道）+2个（门板）
    - *后排*：2个（中央扶手）+2个（门板）
- **其他储物位**：
    - *中控台顶部储物盒*（适合手机、卡片等小件）
    - *副驾座椅下方储物抽屉*（部分高配车型）
## 📐  空间灵活性
| 功能 | ✅ / ❌ |
|------|---------|
| *后排座椅放倒* | ✅ （整体放倒） |
| *后排座椅翻折* | ❌ （固定式） |
| *后窗开启方式* | ✅ （手动/电动升降玻璃） |
| *货箱宝/防刮涂层* | ✅ （部分车型标配） |
## 🔍  关键亮点
1. **超长轴距（3150 mm）**
   - >带来更稳定的行驶姿态和更宽敞的后排纵向空间。
2. **纯平货箱地板**
   - >方便堆放规则货物，提高装载效率。
3. **同级领先的后排腿部空间**
   - >长途乘坐舒适性优于多数同级皮卡。
4. ***高离地间隙***
   - >兼顾通过性，同时不影响车内垂直空间。

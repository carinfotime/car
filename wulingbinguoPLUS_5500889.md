# 五菱缤果PLUS 内饰参数介绍
  
更新时间：2026年07月16日17时48分37秒   

## 一、整体设计风格
- **主题理念**：极简自然主义，融合复古与科技感。
- **配色方案**：
  - **空影绿**：低饱和度绿色，搭配米白色内饰，营造清新治愈氛围。
  - **暖沙白**：温暖米色系，搭配黑色细节，更显居家温馨。
- **材质工艺**：大面积使用软质包覆（仪表台、门板），搭配高亮镀铬饰条与钢琴烤漆面板。
## 二、座椅系统
- **座椅布局**：2+2 四座布局（部分版本为5座可选）。
- **座椅材质**：
  - 仿皮/织物混搭（高配为全仿皮）。
  - 打孔透气设计（前排）。
- **前排座椅功能**：
  - **主驾**：6向手动调节。
  - **副驾**：4向手动调节。
- **后排座椅**：
  - 支持整体放倒（放倒后后备箱容积可扩展至1450L）。
- **舒适性配置**：
  - 前排座椅头枕高度可调。
  - 后排中央扶手（带杯架）。
## 三、仪表台与中控
- **仪表盘**：
  - **全液晶仪表盘**：尺寸为7英寸或10.25英寸（视配置而定）。
    - *显示内容*：车速、电量、续航里程、驾驶模式、胎压监测等。
    - *UI风格*：简洁卡片式设计，支持主题切换。
- **中控屏幕**：
  - **悬浮式触控屏**：10.1英寸或12.3英寸（高配）。
    - *分辨率*：1920x1080 (FHD)。
    - *操作系统*：Ling OS灵犀系统。
    - *功能*：
      - GPS导航（高德地图车机版）
      - CarPlay/HiCar手机互联
      - OTA远程升级
      - AI语音助手（支持连续对话）
      - App远程控制（空调开关、车门解锁等）
- **物理按键/旋钮**：
  -  空调控制区域保留实体旋钮与按键，便于盲操作。
    （温度调节旋钮、风量旋钮、除雾按键）
## 四、方向盘
- **类型**：双辐式多功能方向盘。
- **材质**：皮质包裹（高配）/塑料（低配）。
- **调节方式**：手动上下调节。
- **功能按键集成**：
  左侧:  音量控制、语音唤醒、蓝牙电话接听/挂断
  右侧:  菜单切换、定速巡航设置
## 五、换挡机构
- **类型**:  电子怀挡 (位于方向盘右后方)
    (取代传统中控台换挡杆，释放中央通道空间)
## 六、储物空间
|  位置 |  描述 |  容积/特点 |
| :--- | :--- | :--- |
|  中央通道 |  开放式储物槽 + USB接口 + Type-C接口 |  可放置手机/钱包 |
|  手套箱 |  常规手套箱 |  带阻尼缓降 |
|  门板储物格 |  前后门板均有杯架+储物槽 |  可容纳1L水瓶 |
|  后排中央扶手杯架 |  两个杯架位 | — |
|  后备箱下沉式储物格 | （部分车型）隐藏于后备箱垫下方 | — |
## 七、灯光与氛围系统
- **车内照明灯**: LED阅读灯 (前后排独立控制)
- **氛围灯**: 
    (仅限高配车型)
    颜色:多色可选 (通常8色或以上)
    位置:中控台贯穿式灯带 +门板饰条

|    远程控制    	| App远程开关空调/车门锁/寻车      	|

   1个USB-A充电口(位于出风口下方)

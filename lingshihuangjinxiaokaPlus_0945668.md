## 菱势黄金小卡Plus 内饰参数介绍
  
更新时间：2026年07月11日08时37分37秒   

### 整体风格与布局
- **设计理念**：采用乘用车化的设计风格，注重实用性与舒适性的结合。
- **中控台**：环抱式座舱设计，偏向驾驶员一侧，便于操作。
- **配色方案**：通常以深色（如黑色、深灰）为主基调，搭配银色或仿碳纤维饰条点缀，营造简约、耐脏且具有一定质感的视觉效果。
### 座椅系统
- **座椅材质**：全系标配**织物座椅**（部分高配车型或选装皮/织物混搭材质）。
- **座椅布局**：标准2座布局（单排座），为驾驶室提供宽敞的横向空间。
- **功能配置**：
    - 主驾驶座椅支持前后滑动及靠背角度调节。
    - 副驾驶座椅支持靠背角度调节。
    - 坐垫填充物较厚实，支撑性良好，适合长时间驾驶。
### 方向盘
- **造型与材质**：三辐式多功能方向盘，采用搪塑或PVC材质包裹。
- **功能按键**：
    - 左侧集成多媒体控制（音量、曲目切换）。
    - 右侧集成蓝牙电话接听/挂断功能。
    - （部分车型）可控制仪表盘信息切换。
### 仪表盘
- **类型**：全液晶仪表盘（尺寸约7英寸至10.25英寸不等，视具体配置而定）。
- **显示内容**：
    - 传统车速、转速信息。
    - 续航里程、瞬时电耗、电池电量百分比。
    - 挡位显示（D/N/R）、驾驶模式指示。
    - 胎压监测数据、车门未关提醒等警示信息。
### 中控多媒体系统
- **屏幕尺寸**：悬浮式触控屏（主流配置为10.1英寸或12.8英寸）。
- **操作系统**：自研车机系统，界面简洁直观。
- **核心功能**：
    - **手机互联映射**：支持蓝牙连接、USB音乐/视频播放。部分车型支持CarLife/HiCar手机互联。
    - **倒车影像**：标配高清倒车影像（带动态辅助线）。
    - **导航系统**：（高配车型）内置在线导航或通过手机互联实现导航投屏。
    - **语音控制**：支持基础的语音指令（如“打开空调”、“播放音乐”等）。
### 空调系统
- **类型**：手动空调（全系标配）。
- **操作方式**：物理旋钮+按键组合，位于中控屏下方或两侧，便于盲操作。
- **功能特点**:具备制冷/制热、风量调节、吹风模式选择及内/外循环切换。部分车型配备后排出风口（位于前排座椅下方）。
### 储物空间与便利性
| 位置 | 描述 |
| :--- | :--- |
| **门板储物格** | 前后门板均设有杯架及储物槽，可放置水瓶及杂物。 |
| **中央通道区域** | -
        •   *换挡机构*:旋钮式电子换挡或传统机械挡杆(视版本)。
        •   *储物*:前方设有开放式储物格(可放手机)、两个杯架及中央扶手箱(深度适中)。 |
| **手套箱** |  副驾前方标准手套箱,容积可观,可存放文件等物品。 |
| ***其他*** | -
        •   *顶棚眼镜盒*:主驾上方设有眼镜收纳盒(部分车型)。
        •   *副驾挂钩*:副驾前方中控台处常设折叠挂钩,方便挂早餐袋等。 |
### 灯光与舒适配置
-   ***车内照明***:前排顶灯(带延时关闭功能)。
-   ***电动车窗***:四门电动车窗,主驾侧带一键降窗功能(部分车型)。
-   ***外后视镜***:电动调节(全系标配),高配版带电动加热除雾功能。

> *注:以上参数基于主流在售版本整理,具体配置可能因生产批次、选装包及地区差异略有不同,请以实车为准。*

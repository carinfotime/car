# 远程星享V 驾驶注意事项
  
更新时间：2026年07月15日08时48分02秒   

## 一、启动与预热
1. **启动前检查**：确认车辆周围无障碍物，轮胎气压正常，灯光、刹车系统工作良好。
2. **高压上电**：插入钥匙或按下启动按钮后，等待仪表盘自检完成，确认动力电池电量充足。
3. **冬季预热**：低温环境下建议先通电预热1-2分钟，待电池温度适宜后再行驶，以保护电池寿命。
## 二、行驶中注意事项
1. **平稳驾驶**：避免急加速、急刹车，保持匀速行驶有助于延长续航里程。
2. **能量回收**：合理利用能量回收系统（E/P模式切换），下坡或减速时可提前松开油门，让车辆自动回收电能。
3. **载重控制**：严禁超载，货物需均匀分布并固定牢固，避免重心偏移影响操控稳定性。
4. **车速控制**：
   - 城市道路建议时速不超过60km/h
   - 高速行驶建议时速控制在80-90km/h之间
   - 过弯时提前减速，防止侧倾
5. **跟车距离**：保持比燃油车更长的安全跟车距离（电动车制动响应特性不同）。
## 三、充电注意事项
1. **充电时机**：建议电量剩余20%-30%时进行充电，避免过度放电损伤电池。
2. **充电方式**：
   - 优先使用厂家配套的专用充电桩
   - 使用家用插座充电时确保线路容量足够（建议≥16A）
3. **充电环境**：
   - 避免在高温暴晒或雨雪天气下露天充电
   - 充电口保持干燥清洁
4. **禁止事项**：
   - 严禁在车辆通电状态下插拔充电枪
   - 严禁使用破损或非标充电设备
## 四、电池维护与保养
1. **长期停放**：
   - 若长期不用，建议将电量保持在50%-70%
   - 每两周至少启动车辆一次并检查电量
2. **温度管理**：
   - 夏季避免长时间暴晒
   - 冬季尽量停放在室内或避风处
3. **定期检查**：按照保养手册定期到授权服务站检查高压系统及电池状态。
## 五、安全驾驶提醒
1. **盲区注意**：厢式货车盲区较大，变道和转弯时务必观察后视镜并注意侧后方来车。
2. **倒车安全**：利用倒车影像和雷达辅助判断后方障碍物。
3. **夜间行车**：确保前后灯光正常工作，注意大灯高度调节避免影响对向车辆。
4. **涉水行驶**：
   - 尽量避免涉水深度超过30cm
   - 涉水后应及时检查底盘及高压部件是否进水
## 六、紧急情况处理
| 情况 | 处理方法 |
|------|----------|
| ⚠️仪表盘故障灯亮起 | →立即靠边停车 →联系售后服务中心 |
| ⚠️动力中断 | →开启双闪 →缓慢滑行至安全区域 →拨打救援电话 |
| ⚠️发生碰撞 | →切断高压电源（拔掉维修开关） →远离车辆 →报警处理 |
## 七、日常检查清单（每日出车前）
- [ ] 🅿️轮胎外观及胎压正常
- [ ] 💡灯光（大灯/转向灯/刹车灯）完好
- [ ] 🔋仪表显示电量充足且无故障码
- [ ] 🛑制动踏板行程正常无异响
- [ ] 📦货箱门锁闭牢固无松动

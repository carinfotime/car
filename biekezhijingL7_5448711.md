# 别克至境L7 技术参数介绍
  
更新时间：2026年07月10日19时24分37秒   

## 一、车型概览
别克至境L7是上汽通用别克品牌推出的中大型纯电旗舰轿车，基于通用汽车奥特能（Ultium）纯电平台打造，定位高端智能电动市场。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| 驱动形式 | 后驱 / 四驱 |
| 电机类型 | 永磁同步电机 |
| 后驱版最大功率 | 约 255 kW（347 马力） |
| 四驱版最大功率 | 约 375 kW（510 马力） |
| 后驱版峰值扭矩 | 约 450 N·m |
| 四驱版峰值扭矩 | 约 700 N·m |
| 0-100 km/h 加速（后驱） | 约 5.6 秒 |
| 0-100 km/h 加速（四驱） | 约 4.3 秒 |
| 最高车速 | 200 km/h |
---

| 项目 | 参数 |

| CLTC综合续航（99kWh 后驱） | 约 750 km |
| CLTC综合续航（99kWh 四驱） | 约 680 km |
| 快充时间（10%-80%） | 约 28 分钟（直流快充） |
| 快充功率 | 最高 200 kW |
| 对外放电（V2L） | 支持 3.3 kW / 6.6 kW |

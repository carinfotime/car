# 大众途安（Touran）动力参数介绍
  
更新时间：2026年07月14日10时03分42秒   

大众途安作为一款紧凑型MPV，其动力系统兼顾了日常使用的平顺性、燃油经济性以及偶尔的动力需求。根据不同年份款型和市场版本，途安主要搭载了以下几种动力总成。
## 1. 汽油发动机
### 1.1 1.4 TSI（涡轮增压直喷）发动机
这是途安最核心、最主流的动力配置，广泛应用于国内及欧洲市场。
-   **排量**：1.4L
-   **进气形式**：涡轮增压
-   **气缸数**：4缸
-   **最大功率**：
    -   低功率版（280 TSI）：110 kW（150 PS / 150 hp）
    -   高功率版（330 TSI）：110 kW（150 PS / 150 hp）*注：部分早期或特定版本可能调校不同*
-   **最大扭矩**：
    -   低功率版：250 N·m
    -   高功率版：250 N·m
-   **峰值扭矩转速**：1,500 - 3,500 rpm
-   **推荐燃油标号**：95号及以上无铅汽油
### 1.2 1.8 TSI（涡轮增压直喷）发动机
在早期款型或特定配置中提供，动力更为充沛。
-   **排量**：1.8L
-   **进气形式**：涡轮增压
-   **气缸数**：4缸
-   **最大功率**：
    -   约132 kW（180 PS / 177 hp）
-   **最大扭矩**：
    -   约300 N·m
-   **峰值扭矩转速**：1,450 - 4,100 rpm
### 1.3 2.0 TSI（涡轮增压直喷）发动机
常见于海外市场的运动版或高配车型。
-   **排量**：2.0L
-   **进气形式**：涡轮增压
-   **气缸数**：4缸
-   **最大功率**：
    -   约162 kW（220 PS / 217 hp）
-   **最大扭矩**：
    -   约350 N·m
## 2. 柴油发动机 (主要见于欧洲市场)
### 2.1 1.6 TDI (涡轮增压柴油直喷)
主打经济性和低排放。
| 参数 | TDI (90 PS) | TDI (115 PS) |
| :--- | :--- | :--- |
| **排量 (L)** | 1.6 | 1.6 |
| **最大功率 (kW/PS)** | 66 / 90 | 85 / 115 |
| **最大扭矩 (N·m)** | ~230 | ~250 |
| **峰值扭矩转速 (rpm)** | ~1,500 - ~3,000 | ~1,500 - ~3,000 |
### 2.2 2.0 TDI (涡轮增压柴油直喷)
提供更强的低扭和高速巡航能力。
|参数|TDI (150 PS)|TDI (190 PS)|
|:---|:---|:---|
|排量(L)|2.0|2.0|
|最大功率(kW/PS)|110 /150|140 /190|
|最大扭矩(N·m)|~340|~400|
|峰值扭矩转速(rpm)|~1,750 - ~3,000|~1,750 - ~3,000|
## 🔧传动系统匹配
途安的动力系统通常与以下变速箱匹配：
*   ⚙️手动变速箱:
    常见于入门级车型或欧洲市场柴油版本。
*   ⚙️DSG双离合变速箱:
    主流配置。根据发动机不同，匹配6速湿式DSG或7速干式DSG。
*   ⚙️自动变速箱(AT):
    部分早期车型可能配备传统的6速自动变速箱。
## 🚀性能表现参考(以主流280TSI为例)
*   0–100 km/h加速时间:约9秒左右。
*   最高车速:约200 km/h以上。
*   综合油耗(NEDC/WLTC):约5–7 L/100km之间(视驾驶习惯和路况而定)。

# 锐行者 内饰参数介绍
  
更新时间：2026年07月16日14时48分19秒   

## 一、整体设计风格
- **设计理念**：科技运动与豪华舒适融合
- **内饰色调**：提供曜石黑、星云灰、熔岩棕三种主题配色
- **材质工艺**：大面积软性材质包覆，搭配金属拉丝饰板与钢琴烤漆面板
## 二、座椅系统
| 项目 | 参数 |
|------|------|
| **座椅材质** | Nappa真皮 / 高级仿皮（视配置而定） |
| **前排座椅功能** | 电动12向调节（含腰托4向）、加热、通风、记忆功能 |
| **后排座椅** | 4/6比例放倒，中央扶手带杯架与储物盒 |
| **后排功能** | 加热（高配）、靠背角度调节（高配） |
| **头枕** | 前后排均配备可调式航空头枕 |
## 三、驾驶舱布局
- **方向盘**
    - D型运动多功能方向盘
    - 真皮包裹，带换挡拨片
    - 支持手动四向调节（上下+前后）
- **仪表盘**
    - 12.3英寸全液晶仪表盘
    - UI界面支持经典、运动、导航三种主题切换
- **中控屏**
    - 14.6英寸悬浮式高清触控屏
    - 分辨率：1920 x 1080
    - OLED面板，防指纹涂层
## 四、智能互联系统
- **车载系统**：锐行智联OS V3.0
- **芯片方案**：高通骁龙 SA8155P
- **语音控制**：双区域语音识别（主/副驾），支持连续对话与可见即可说
- **手机互联**：无线 Apple CarPlay / Android Auto
- **OTA升级**：支持整车FOTA远程升级
## 五、空调与空气管理
- **空调类型**：双区自动恒温空调（高配为三区）
- **后排出风口**：标配，带独立风量调节
- **空气净化**
    - PM2.5过滤系统
    - AQS空气质量监测传感器（高配）
    - N95级空调滤芯
## 六、音响系统（选装/高配）
| 项目 | 规格 |
|------|------|
| **品牌** | BOSE / JBL（视市场版本） |
| **扬声器数量** | 12个 / BOSE为14个 |
| **技术特点** | Centerpoint环绕声技术 / QuantumLogic沉浸式音效 |
## 七、灯光与氛围系统
- **氛围灯**
    -  64色可调环绕式氛围灯带（中控台+门板+脚窝）
    -  支持呼吸模式与音乐律动联动功能
## 八、储物空间细节 (部分)
1.  中央扶手箱容积约8L，内置USB-C充电口×2 + SD卡槽。
2.  手套箱带阻尼感且内部植绒处理。
3.  门板储物格可容纳1L水瓶。
4.  中控台下方镂空储物区带无线充电板。
## 九、其他便利配置 (部分)
*  手机无线充电面板 (最大15W快充)。
*  流媒体内后视镜 (可选装)。
*  抬头显示系统 HUD (投射面积约10英寸)。
*  前排遮阳板均带化妆镜及照明灯。

> *注：以上参数基于顶配/高配车型整理，具体配置请以实车为准。*

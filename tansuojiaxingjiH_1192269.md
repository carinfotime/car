# 上汽大通MAXUS 探索家星际H 驾驶注意事项
  
更新时间：2026年07月19日08时18分56秒   

## 一、行车前检查
1.  **车身与拓展机构**
    -   检查升顶帐篷或硬升顶机构是否完全复位并锁止。
    -   确认侧边遮阳棚、外置厨房等拓展部件已收回并固定。
    -   检查外部储物箱门是否关紧。
2.  **水电系统**
    -   **清水箱/灰水箱**：确认排水阀处于关闭状态，长途行驶建议排空或只保留少量水以减轻负载。
    -   **燃气系统**：确认燃气瓶阀门已关闭，且固定牢固无晃动。
    -   **电器设备**：行车前务必关闭所有12V直流电器（如冰箱可切换至电池供电模式），拔掉220V外接电源线。
3.  **轮胎与底盘**
    -   检查轮胎气压（建议冷态前轮约4.0-4.5bar，后轮约4.5-5.0bar，具体以车辆铭牌为准）。
    -   检查底盘有无漏油、漏水迹象。
## 二、行驶中要点
1.  **重心与操控**
    -   **重心偏高**：房车重心远高于普通乘用车，过弯时务必提前减速，严禁急打方向。
    -   **侧风敏感**：高速行驶遇强侧风或超越大型车辆时，握紧方向盘，适当减速。
    -   **刹车距离**：满载质量可达3.5吨以上，刹车距离显著增加，保持比普通轿车更长的跟车距离。
2.  **速度限制**
    -   **高速公路**：建议最高时速不超过100km/h（部分路段法规限速100km/h）。
    -   **普通公路**：严格遵守限速标志，尤其注意弯道、下坡路段限速。
    -   **不要超速**：车身大、惯性大，超速极易导致失控。
3.  **超车与变道**
    -   **盲区巨大**：车身长（约5.9米），后视镜存在较大盲区。变道前必须观察后视镜并结合盲区监测系统（如有）。
    -   **超车预留空间**：超车需预留更长的加速距离和并入空间。
4.  **倒车与泊车**
    -   **尽量有人指挥**：由于后方视野受限（尤其是升顶状态下），倒车时最好有同伴在车外指挥。
    -   **利用影像系统**：充分利用360°环视影像和倒车雷达。
    -   **注意顶部高度**：进入地下车库、限高杆、隧道前，务必确认车辆总高度（含升顶/空调外机等），通常超过2.8米甚至3米。
## 三、特殊路况应对
1.  **山路/长下坡**
    -   **使用低速挡/手动模式**：利用发动机牵引力辅助制动，避免长时间踩刹车导致刹车过热失效。
        > *探索家星际H配备采埃孚8AT变速箱及博世ESP系统。*
        > *在下长坡时建议切换到手动模式并挂入低档位。*
        > *切勿空挡滑行！*
2.  **非铺装路面/轻度越野**
    探索家星际H基于“星际”皮卡底盘打造，具备一定的越野能力：
    1. 选择合适的四驱模式（如沙地、泥地模式）。
    2.  通过交叉轴或炮弹坑时需慢速且保持方向稳定以避免磕碰底盘或车厢连接处。
    3.  涉水深度有限制（通常不超过600mm），切勿贸然涉深水以防损坏电气设备及发动机进气系统进水导致拉缸报废！
## ⚠️四、安全红线
| ❌禁止事项 | ✅正确做法 |
| :--- | :--- |
| ✋行驶中在车内走动或躺卧 | ✨所有乘客必须坐在座位上并系好安全带 |
| ✋行驶中使用厨房设备 | ✨停车熄火后再使用燃气灶具及微波炉等大功率电器 |
| ✋行驶中打开遮阳棚或升顶帐篷 | ✨到达营地驻车熄火后再进行操作 |
| ✋搭载超过核定人数 | ✨严格按行驶证核载人数乘坐 |

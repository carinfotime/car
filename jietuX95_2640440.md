# 捷途X95 技术参数介绍
  
更新时间：2026年07月15日13时32分52秒   

捷途X95是奇瑞控股旗下捷途品牌推出的一款中型SUV，主打“旅行+”概念，以大空间、高配置和性价比为核心卖点。
## 一、动力系统
| 项目 | 参数 |
| :--- | :--- |
| **发动机型号** | 1.6TGDI（涡轮增压缸内直喷） |
| **排量** | 1.6L (1598ml) |
| **进气形式** | 涡轮增压 |
| **最大马力** | 197 Ps (约145 kW) |
| **最大功率转速** | 5500 rpm |
| **最大扭矩** | 290 N·m |
| **最大扭矩转速** | 2000-4000 rpm |
| **供油方式** | 缸内直喷 |
| **燃油标号** | 92号及以上无铅汽油 |
| **环保标准** | 国VI |
## 二、传动与底盘
| 项目 | 参数 |
| :--- | :--- |
| **变速箱类型** | 7速湿式双离合变速箱 (7DCT) / 6速手动变速箱 (6MT) |
| **驱动方式** | 前置前驱 (FF) |
| **前悬挂类型** | 麦弗逊式独立悬挂 |
| **后悬挂类型** | 多连杆式独立悬挂 |
| **转向系统** | EPS电动助力转向 |
| **制动系统（前/后）** | 通风盘式 / 盘式 |
## 三、车身尺寸与空间
*注：具体尺寸因配置版本略有差异*
-   **长 × 宽 × 高 (mm)**：4875 × 1925 × 1780 (或1785)
-   **轴距 (mm)**：2855
-   **座位数**：5座 /  6座 /  7座
-   **油箱容积 (L)**：约55
-   **后备箱容积 (L)**：约380 -  2100（后排座椅放倒后）
## 四、主要配置亮点
### （一）智能科技
-   12.3英寸全液晶仪表盘 +  12.3英寸中控触控大屏（双联屏设计）
-   9.7英寸空调触控屏
-   智能语音交互系统（支持导航、电话、多媒体控制）
-   车联网（4G）、Wi-Fi热点、OTA远程升级
-   手机远程控制（启动/空调/车窗等）
### （二）安全辅助
-   360°全景影像系统
-   前后驻车雷达
-   博世ESP车身稳定控制系统
-   上坡辅助 +  陡坡缓降
-   主动刹车/主动安全系统（高配）
-   车道偏离预警（高配）
-   盲区监测预警（高配）
### （三）舒适便利
-   全景天窗（可开启）
-   电动尾门（带记忆功能，高配）
-   主/副驾驶座椅电动调节 +  加热/通风功能（高配）
-   后排独立空调出风口
-    无钥匙进入 +  一键启动
## ⚠️五、性能与油耗参考
> *以下数据为官方或工信部综合工况参考值，实际表现受驾驶习惯及路况影响。*
### （一）加速性能
> -   `0 -100 km/h`加速时间：约10秒左右
### （二）最高车速
> -    最高时速：约180 km/h
### （三）油耗表现
> -    工信部综合油耗：
>     -     手动挡车型：约8.1 L/100km  
>     -     自动挡车型：约8.3 L/100km  
>
> *注：实际市区油耗通常在9 -11 L/100km之间。*

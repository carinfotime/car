# 世极 (Century) 动力参数介绍
  
更新时间：2026年07月12日11时51分53秒   

作为丰田旗下的顶级旗舰车型，世极（Century）在动力系统上追求的是极致平顺、静谧与从容。其动力总成旨在提供如丝般顺滑的加速体验和符合顶级豪华车身份的行驶质感。
## 核心动力总成
世极搭载了一套 **3.5升 V6 插电式混合动力系统 (PHEV)**，这是丰田目前最先进的混合动力技术之一。
| 项目 | 参数 |
| :--- | :--- |
| **发动机类型** | 2GR-FXS 型 V型6缸 |
| **排量** | 3.5升 (3,456 cc) |
| **进气形式** | 自然吸气 |
| **发动机最大功率** | 约 193 kW (262 PS) / 6,000 rpm |
| **发动机最大扭矩** | 约 335 N·m / - rpm |
| **前电机最大功率** | 约 134 kW (182 PS) |
| **后电机最大功率** | 约 80 kW (109 PS) |
| **系统综合最大功率** | **约 303 kW (412 PS)** |
> *注：具体数值可能因不同市场版本而略有差异。*
## E-Four Advanced 电子四驱系统
-   **布局：** E-Four Advanced
-   **特点：**
    -   前轴由发动机和前电机驱动。
    -   后轴由独立的 eAxle（后桥电机）驱动，实现电子四驱。
    -   可根据行驶状态在前后轴之间精确分配扭矩（前后轮扭矩分配可在100:0到20:80之间变化），提供卓越的抓地力和操控稳定性。
    -   在起步、加速或湿滑路面时，后桥电机可提供额外推力，确保车辆姿态稳健。
## Hybrid & EV模式
-   **EV模式：**
    -   **纯电续航里程：** WLTC工况下约为57公里。
    -   **适用场景：** 在城市低速行驶、短途通勤或需要极致静谧的场合（如出入酒店、住宅区），可完全依靠电力驱动，实现零排放和近乎无声的移动。
-   **HYBRID模式：**
    -   **工作原理：** ECU根据油门深度、车速和电池电量，智能协调发动机与电动机的工作状态。在大多数巡航状态下优先使用电机；需要强劲动力时发动机介入并与电机协同发力；减速或制动时进行能量回收。
    -   **驾驶感受：** “油门跟脚”，响应线性且无延迟。加速过程平滑流畅，没有传统变速箱的换挡顿挫感。
## NVH控制与驾驶质感
-   **极致静谧性：**
    -   PHEV结构本身减少了发动机低效运转的时间。
    -   车身大量使用隔音、吸音材料（如多层隔音玻璃、底盘隔音护板等）。
    -   V6发动机经过特殊调校和平衡处理，运转极为平顺安静。
-   **平顺性优先：**
    -   e-CVT无级变速机构消除了换挡冲击。
    -   动力输出被刻意调校为“不急不躁”，即使深踩油门，加速力也是持续而线性地释放，而非突兀的推背感。这符合其作为“移动宫殿”的身份定位。
## Summary
世极的动力系统并非追求极限性能或运动感，而是将以下三点作为核心目标：
1.  **绝对平顺与静谧：** V6 PHEV + e-CVT的组合提供了内燃机时代无法比拟的丝滑体验。
2.  **从容不迫的动力储备：** “412马力”的系统综合功率保证了在任何速度区间都能轻松完成超车或爬坡动作。
3.  **顶级豪华车的仪式感：** EV模式的无声启动与低速滑行是其区别于普通豪华车的重要标志之一。

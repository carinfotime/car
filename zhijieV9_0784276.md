# 智界V9 驾驶注意事项
  
更新时间：2026年07月13日14时20分39秒   

## 一、启动与预热
1.  **智能钥匙确认**：确保智能钥匙在车内有效感应区域，避免因钥匙信号干扰导致无法启动。
2.  **上电自检**：踩下制动踏板，按下启动键。等待仪表盘完成自检，确认无故障报警灯常亮后再挂挡行驶。
3.  **冬季预热**：在低温环境下（低于-5°C），建议车辆上电后等待1-2分钟，让电池和动力系统达到适宜工作温度，提升能效并保护电池。
## 二、驾驶模式选择
-   **经济模式**：适合城市拥堵或长途续航优先场景。加速响应平缓，能量回收强度较高。
-   **舒适模式**：日常通勤推荐。动力输出平顺，悬挂调校偏软（若配备空气悬挂），兼顾能耗与乘坐舒适性。
-   **运动模式**：适合高速超车或需要强劲动力时。电门响应灵敏，悬挂变硬以提供更好支撑。
-   **雪地/湿滑模式**：雨雪天气务必开启。系统会限制扭矩输出，防止车轮打滑。
## 三、能量回收与制动
1.  **单踏板模式适应期**：
    -   初次使用“单踏板模式”时，请在空旷路段练习。松开电门即产生较强减速感（类似轻踩刹车）。
    -   提前预判路况，缓慢松开电门可实现平稳停车；紧急情况仍需立即踩下制动踏板。
2.  **自动驻车 (Auto Hold)**：
    -   车辆完全停稳后深踩一脚刹车即可激活。此时可松开脚刹，起步时直接轻踩电门即可解除。
3.  **再生制动限制**：
    -   当电池电量接近满电（>95%）或电池温度过低时，能量回收效果会减弱或关闭。此时需提前使用机械刹车。
## 四、智能驾驶辅助 (HUAWEI ADS)
> **重要提示**：智能驾驶为辅助功能，驾驶员必须时刻保持注意力集中并随时准备接管车辆。
1.  **NCA (领航辅助)**：
    -   **适用范围**：仅限高精地图覆盖的高速公路及城市快速路。
    -   **变道策略**：系统会自动发起变道超车。若对侧后方来车有顾虑或当前车道更优时，可通过打转向灯取消变道指令。
    -   **匝道通行**：进入匝道前请留意系统是否自动降速至合理范围（通常40-60km/h）。如感觉车速过快或路线偏离，请立即人工接管。
2.  **LCC (车道巡航)**：
    -   **红绿灯路口**：LCC无法识别红绿灯！经过有信号灯的路口时务必主动接管控制方向和速度。
3.  **AEB (自动紧急制动)**：
    -   AEB主要针对车辆、行人及骑行者。对静止的小型障碍物（如锥桶、低矮石墩）识别能力有限。切勿依赖AEB规避所有风险。
## 五、泊车与遥控
1.  **APA (自动泊车)**：
    -   *搜索车位*：低速行驶通过车位旁即可自动扫描。支持垂直、平行及斜列车位。
    -   *泊车过程*：系统会自动控制方向盘和挡位。驾驶员需随时准备刹车以防突发情况（如后方突然窜出的行人或宠物）。
2.  **RPA (遥控泊车)**：
    -   *使用场景*：适用于极窄车位（人无法开门下车的情况）。
    -   *操作*：人在车外通过手机App控制车辆前进/后退/停止。
    -   *安全距离*：确保遥控范围内无障碍物和陡坡；手机蓝牙连接稳定后再操作。
## 六、充电注意事项
1.  **快充策略**：
    -    建议将电量用到20%-30%左右再充电至80%-90%，避免频繁满充或过放以延长电池寿命。
2.  **充电枪操作**：
    -    插拔充电枪时请勿用力拉扯线缆；听到“咔哒”锁止声后再离开。
3.  **极端天气充电**：
    -    雷雨天气尽量避免户外快充；若必须充电请确保充电枪和接口干燥无积水。
## 七、特殊路况应对
| 路况类型 |     应对措施      |
| :------- | :---------------: |
| ✅涉水路段 |     谨慎通过      |
| ❌深积水区 |      禁止驶入      |
| 🛑冰雪路面 |      降低车速      |
-   *涉水深度* ：智界V9作为轿车/SUV车型（视具体版本），涉水深度不宜超过轮毂中心线高度（约25cm）。纯电车虽无排气管但底盘有高压电池包——一旦进水可能造成严重损坏甚至短路起火！
-   *托底风险* ：遇到坑洼路面务必减速慢行——底盘刮蹭可能导致电池包受损且不在常规质保范围内！

# Van宝路 内饰参数介绍
  
更新时间：2026年07月18日18时23分52秒   

## 一、座椅系统
| 项目 | 参数/配置 |
| :--- | :--- |
| **座椅材质** | Nappa真皮 / 高级仿皮（视配置版本而定） |
| **前排座椅功能** | 电动调节（8向）、座椅加热、通风、记忆功能 |
| **第二排座椅** | 独立行政座椅，支持前后滑动及靠背角度调节 |
| **第三排座椅** | 可折叠放平，支持50:50分割翻折 |
| **头枕** | 全车独立可调头枕，前排带航空睡眠头枕 |
## 二、仪表与中控
| 项目 | 参数/配置 |
| :--- | :--- |
| **仪表盘** | 12.3英寸全液晶高清仪表盘，支持多种主题切换 |
| **中控屏幕** | 15.6英寸悬浮式触控屏，分辨率1920×1080 |
| **车机系统** | VanOS智能互联系统，支持OTA远程升级 |
| **语音控制** | 双区语音识别，支持导航、空调、车窗控制 |
## 三、空调与空气质量
- **空调类型**：双区自动恒温空调
- **后排独立出风口**：B柱及顶棚出风口
- **空气净化系统**：PM2.5过滤 + 负离子发生器
- **香氛系统**：可选配车载香氛（三种香型）
## 四、音响与娱乐
- **音响品牌**：Bose® Premium音响系统（选装）
- **扬声器数量**：12个高保真扬声器
- **后排娱乐屏**：10.1英寸顶置折叠屏（选装）
- **接口配置**：
    - USB-C接口 ×4（前排2个/后排2个）
    - USB-A接口 ×2
    - 12V电源接口 ×2
    - HDMI输入接口 ×1
## 五、储物空间
| 位置 | 容积/特点 |
| :--- | :--- |
| **手套箱** | ≈8L，带照明及锁止功能 |
| **中央扶手箱** | ≈12L，内置冷藏功能（可制冷饮料） |
| **门板储物格** | ≥1.5L/侧，可放置水瓶及杂物 |
| **后备箱常规容积** | ≥500L（第三排展开状态）|
| **后备箱最大容积** | ≥1800L（第三排折叠后）|
## 六、灯光氛围
- **氛围灯颜色数**:64色
- **氛围灯区域**:中控台、门板饰条、脚窝灯
- **阅读灯**:前后排独立LED阅读灯，带触摸开关
- **迎宾灯**:车门投影迎宾灯（Van宝路Logo）
## 七、便利与舒适配置
-  电动尾门（带脚踢感应开启）
-  无钥匙进入 +一键启动
-  手机无线充电面板（15W快充）
-  全景天窗（可开启式，带电动遮阳帘）
-  后排隐私玻璃
-  方向盘材质：真皮包裹 +多功能按键 +加热功能
---
> *注：以上参数基于Van宝路旗舰版车型整理。不同年款或配置版本可能存在差异，请以实车为准。*

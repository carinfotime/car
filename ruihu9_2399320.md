# 瑞虎9 儿童乘车安全注意事项
  
更新时间：2026年07月16日14时27分10秒   

为了保障您孩子的乘车安全，请务必遵守以下事项：
## 一、儿童安全座椅使用规范
1.  **必须使用安全座椅**：12周岁以下或身高低于145厘米的儿童，乘车时必须使用符合国家标准的儿童安全座椅。
2.  **反向安装**：建议4岁及以下儿童使用反向安装的儿童安全座椅，以更好地保护头颈。
3.  **ISOFIX接口**：瑞虎9后排座椅配备ISOFIX儿童座椅接口，请严格按照安全座椅说明书和车辆说明进行安装，确保锁止牢固。
4.  **禁止使用成人安全带**：切勿让儿童直接使用成人安全带，以免勒伤颈部或腹部。
## 二、后排乘坐规定
1.  **禁止坐副驾驶**：12岁以下儿童严禁乘坐副驾驶位。如万不得已，必须关闭副驾驶位的安全气囊。
2.  **始终坐后排**：儿童应始终坐在后排座位，并正确使用安全座椅或安全带。
3.  **车窗与天窗**：
    -   启用**后排车窗锁**，防止儿童意外打开车窗。
    -   严禁儿童将头、手或身体任何部位伸出天窗或车窗外。
4.  **车门安全**：启用**儿童安全锁**，防止儿童在行驶中意外打开车门。
## 三、行驶过程中的注意事项
1.  **避免车内玩耍**：行驶中请勿让儿童在车内站立、嬉戏打闹或随意走动。
2.  **勿干扰驾驶**：教育儿童不要触碰方向盘、换挡杆、中控屏幕等驾驶相关部件。
3.  **物品摆放**：车内尖锐、易碎或重物应放置在储物箱或后备箱，避免急刹车时伤及儿童。
4.  **温度控制**：停车后切勿将儿童单独留在车内，夏季车内温度会迅速升至致命水平。
## 四、下车时的安全提醒
1.  **观察后方**：使用“荷式开门法”，教育儿童下车前先观察后方是否有来车或行人。
2.  **家长陪同**：年幼儿童下车时，应由家长陪同并协助开门。
3.  **远离车辆**：下车后，让孩子立即离开车辆，不要在车辆周围逗留或玩耍。
## 五、瑞虎9 特殊功能提醒
1.  **智能驾驶辅助**：请勿让儿童误触方向盘上的辅助驾驶按键。
2.  **空气净化系统**：如车辆配备PM2.5过滤或负离子净化功能，建议在车内有人时开启，保持空气清新。
3.  **全景天幕/天窗**：提醒儿童不要在天窗开启时站立或将身体探出。
> **总结：安全座椅 + 后排乘坐 + 锁好门窗 + 不独留车内 = 儿童乘车安全**

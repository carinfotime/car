# AION V 内饰参数介绍
  
更新时间：2026年07月12日14时33分01秒   

## 一、整体设计风格
AION V 采用“环宇式”极简科技座舱设计理念，融合未来感与豪华感，内饰线条流畅、层次分明，营造宽敞舒适的驾乘空间。
---
## 二、座椅系统
| 项目 | 参数 |
|------|------|
| **座椅材质** | Nappa真皮 / 仿皮（视配置版本而定） |
| **前排座椅功能** | 电动调节（主驾6向/副驾4向）、加热、通风、记忆功能（主驾） |
| **后排座椅** | 支持靠背角度调节（多档可调） |
| **座椅布局** | 5座布局 |
| **后排中央扶手** | 带杯架及储物盒 |
---
## 三、仪表盘与中控屏
| 项目 | 参数 |
|------|------|
| **仪表盘** | 12.3英寸全液晶仪表盘 |
| **中控屏** | 15.6英寸悬浮式中控大屏 |
| **分辨率** | 2K级别高清显示 |
| **车机系统** | ADiGO智驾互联系统（基于安卓深度定制） |
| **芯片** | 高通骁龙8155车规级芯片 |
---
## 四、方向盘
- **类型**：多功能真皮方向盘
- **调节方式**：手动上下+前后调节
- **按键功能**：音量控制、语音唤醒、定速巡航/自适应巡航控制、菜单切换等
- **加热功能**：部分高配车型提供方向盘加热
---
## 五、空调系统
- **类型**：自动双区恒温空调
- **后排独立出风口**
- **PM2.5过滤系统**
- **负离子空气净化器**
- AQS空气质量监测系统（部分配置）
---
## 六、音响系统
- **扬声器数量**：6扬声器（标准版）/10扬声器（高配版）
- **品牌音响**：部分高配搭载品牌音响系统
- **音效模式**：支持多种预设音效均衡器调节
---
## 七、灯光氛围
- **氛围灯颜色数**：32色 / 64色（视配置版本）
- **氛围灯区域**：中控台、门板饰条、脚窝灯等
- **亮度多级可调**
---
## 八、储物空间
| 位置 | 说明 |
|------|------|
| **中央扶手箱** | USB接口+储物格，容积约8L |
| **手套箱** |带阻尼开启，可存放文件及杂物 |
| **门板储物格** |前后门均可放置水瓶及小物件 |
| ***中控台下层*****  开放式储物平台，支持手机无线充电 |
> *注：“中控台下层”为镂空设计，提供额外储物空间*
---
##  九、其他内饰配置亮点
1. ✅  全景天窗 /  全景天幕（视版本而定）
2. ✅  手机无线充电面板（15W快充）
3. ✅  后排USB充电接口（Type-A + Type-C）
4. ✅  电动遮阳帘（全景天幕版配备）
5. ✅  车内摄像头（用于疲劳监测等功能）
6. ✅  隐藏式出风口设计
7. ✅  金属拉丝/钢琴烤漆装饰面板
8. ✅  软质材料包覆面积大，触感细腻

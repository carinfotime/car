## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 机油 & 机滤 | 每 **5,000公里** 或 **6个月** |
| 空气滤芯 | 每 **10,000公里** |
| 空调滤芯 | 每 **10,000公里**（建议每年换两次） |
| 燃油滤芯 | 每 **20,000公里** |
| 火花塞 | 每 **30,000公里** |
| 变速箱油 | 每 **40,000公里** |
| 刹车油 | 每 **40,000公里**（或2年） |

## 二、各阶段详细保养清单
### 🟢 A类保养（5,000km /6个月）
- [ ]更换发动机机油
- [ ]更换机油滤清器
- [ ]检查轮胎气压及磨损情况
- [ ]检查灯光系统
- [ ]检查雨刮器及玻璃水液位
- [ ]检查底盘有无异常松动或漏油
### 🟡 B类保养（10,000km /12个月）
包含A类全部项目，另加：
- [ ]更换空气滤芯
- [ ]更换空调滤芯
- [ ]检查并补充防冻液
- [ ]检查刹车片厚度
- [ ]检查蓄电池状态及接线柱清洁度
- [ ]全车润滑（门铰链、天窗导轨等）
### 🔵 C类保养（20,000km /24个月）
包含B类全部项目，另加：
- [ ]更换燃油滤芯
- [ ]清洗节气门（视积碳情况）
- [ ]四轮换位及动平衡
- [ ]四轮定位检测与调整

- [x]更换变速箱油 +变速箱滤网/垫片(如适用)
- [x]更换刹车油(制动液)
- [x]更换防冻液(冷却液)
- [x]更换转向助力油(电子助力车型可忽略)
- [x]更换前后桥齿轮油(四驱版)
- [x]更换分动箱油(四驱版)
 -[x]清洗喷油嘴/进气道积碳(建议使用专用清洗剂)
 -[x]全面底盘螺丝紧固与胶套检查

## 三、核心部件详细说明
### 1.发动机机油
*推荐规格：*
*API等级：SN或更高*
*粘度等级：5W‑30 (四季通用) /0W‑40 (严寒地区)*
*容量：约5.5L (含机滤)*
> 💡 *涡轮增压发动机建议使用全合成机油，延长涡轮寿命。*
### 2.变速箱油
*自动变速箱 (ZF8HP /长城自研9AT)：*
*规格：ZF Lifeguard8/ZF8HP专用或长城ATF认证*
*容量：约8–9L (循环机换油需更多)*
*方法：建议使用循环机换油，更彻底*
> ⚠️ *严禁混用不同品牌规格的变速箱油！*
### 3.分动箱 &前后桥齿轮油
*分动箱：ATF Dexron III或专用分动箱油*
*前桥/后桥差速器：GL‑5  75W‑90 (限滑差速器需添加LSD添加剂)*

## 四、日常自查清单 (每月一次)
1. □  机油尺刻度是否在上下限之间  
2. □  冷却液储液罐液位是否正常  
3. □  刹车踏板行程有无异常变软  
4. □  轮胎胎压是否在标准范围(通常2.3–2.5bar)  
5. □  各灯光功能是否正常  
6. □  仪表盘有无故障灯亮起  
7. □  雨刮水是否充足

## 五、越野/恶劣工况特别注意事项
若经常进行以下活动，请相应缩短保养周期：
1.**重度越野/涉水**
 -每次越野后冲洗底盘泥沙
 -涉水后立即检查差速器及分动箱是否进水乳化
 -缩短至20,000km换前后桥齿轮油和分动箱油
2.**高寒地区 (-20°C以下)**
 -使用0W‑40机油提高冷启动保护性
 -确保防冻液冰点低于当地最低气温10°C以上
 -冬季前提前检测蓄电池健康度(CAA值)
3.**多尘/沙石路面**
 -空气滤芯缩短至5,000km一换或勤吹扫清理
 -空调滤芯同样缩短周期至半年一换
4.**频繁拖挂房车/重载**
 -适当提高机油粘度至5W‑40以增强高温保护性  
 -注意变速箱温度监控，必要时加装辅助散热器

## 六、常见问题与提示
1.**关于颗粒捕捉器(GPF)**
哈弗H9部分国六b排放车型带有GPF。务必使用低灰分ACEA C2/C3认证机油，避免堵塞。
2.**关于尿素系统 (柴油版H9)**
柴油版需定期添加车用尿素溶液(DEF)，并注意尿素喷嘴结晶堵塞问题。
3.**关于天窗漏水预防**
定期清理天窗排水孔(位于天窗四个角落)，防止雨水倒灌车内。
4.**关于电瓶亏电**
H9电子设备较多，长期停放超过两周建议断开电瓶负极或用智能充电器维持电量。

# 车型空间参数介绍：E路顺V6
  
更新时间：2026年07月12日08时45分53秒   

## 一、整车尺寸参数
| 项目 | 参数 |
| :--- | :--- |
| 车长（mm） | 4,990 |
| 车宽（mm） | 1,780 |
| 车高（mm） | 1,990 |
| 轴距（mm） | 3,050 |
| 前轮距（mm） | 1,520 |
| 后轮距（mm） | 1,530 |
> **说明**：长轴距设计为车内纵向空间提供了良好基础，同时兼顾了行驶稳定性。
## 二、乘员舱空间
- **座位布局**：2+2+3 / 2+3+2 / 2+3 （可选）
- **座位数**：5座 / 7座
- **第一排**：
    -  头部空间：约 **1050 mm**
    -  腿部空间：充裕，座椅支持前后滑动及靠背调节
- **第二排**：
    -  头部空间：约 **1020 mm**
    -  腿部空间：最大可达 **850 mm** （座椅调至最后位置）
    -  座椅形式：独立座椅或连排座椅（视配置而定）
- **第三排**：
    -  头部空间：约 **980 mm**
    -  腿部空间：约 **650 mm** （适合中短途乘坐）
    -  座椅可折叠/侧挂，便于扩展储物空间
## 三、储物与载物空间
### （一）常规后备箱容积
- **7座满员状态**：约 **350 L**
- **5座状态**（第三排折叠）：约 **1,200 L**
- **最大扩展容积**（后排全部折叠/拆除）：约 **2,500 L**
### （二）车内储物细节
| 位置 | 储物设计 |
| :--- | :--- |
| 仪表台 | — USB接口旁储物槽<br>—副驾手套箱（容积约12L）<br>—中央通道杯架×2 +储物盒 |
| 车门板 | —前门板可容纳500ml水瓶×3<br>—后门板可容纳500ml水瓶×2<br>—门板下方设有长条形储物槽 |
| 第二排 | —中央扶手杯架×2<br>—座椅侧方网兜<br>—前排座椅背后地图袋 |
| 第三排 | —两侧杯架<br>—侧壁储物槽 |
### （三）特殊装载能力
- **尾门开口宽度**：约 **1,200 mm**
- **尾门开口高度**：约 **1,100 mm**
- **地板离地高度**：约 **550 mm** （便于装卸货物）
- **内部最大进深**（后排折叠后）：约 **2,400 mm**
- **内部最大宽度**：约 **1,550 mm**
- **内部最大高度**：约 **1,250 mm**
## 四、其他实用数据
-  最小转弯半径：5.8 m
-  接近角 /离去角 ：18° /22°
-  空载离地间隙 ：180 mm
-  整备质量 ：1,650 kg ~1,750 kg （视配置）
---
以上数据基于官方公布及实测信息整理，具体参数可能因配置版本不同略有差异。

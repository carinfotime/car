## 骆驼微卡 动力参数介绍
  
更新时间：2026年07月12日13时37分33秒   

骆驼微卡作为一款主打城市物流与短途运输的轻型商用车，其动力系统兼顾了燃油经济性与承载能力。以下为核心动力参数与技术特点：
### 发动机规格
-   **发动机型号**：东安DAM16KR / 五菱柳机LJ4A15Q6（根据具体配置版本）
-   **发动机类型**：直列四缸、水冷、自然吸气/涡轮增压（部分高配版）
-   **排量**：1.6L
-   **最大功率**：
    -   自然吸气版：90kW（约122马力）
    -   涡轮增压版（如适用）：105kW（约143马力）
-   **最大扭矩**：
    -   自然吸气版：158 N·m
    -   涡轮增压版：210 N·m
-   **燃油供给方式**：多点电喷（MPI）
-   **排放标准**：国六b
### 传动系统
-   **变速箱类型**：
    -   5速手动变速箱（MT） —— *主力配置*
    -   （部分高配或自动挡版本可选）CVT无级变速箱 / 6速AMT
-   **驱动方式**：中置后驱（MR）
### 性能表现
-   **最高车速**：≥120 km/h
-   **0-60km/h加速时间**：约8秒（空载状态）
-   **爬坡度**：≥30%
-   **综合油耗（WLTC工况）**：
    -   手动挡版本：约7.5 L/100km
    -   自动挡版本：约7.8 L/100km
### 核心技术与特点
| 技术亮点 | 说明 |
| :--- | :--- |
| **低扭强劲** | 发动机在2000rpm即可输出90%以上峰值扭矩，适合城市频繁启停与重载起步。 |
| **轻量化设计** | 采用铝合金缸盖与塑料进气歧管，降低自重，提升燃油经济性。 |
| **耐久性强化** | 正时链条传动、铸铁缸体，延长大修间隔里程。 |
| **智能电控系统** | Bosch ECU精确控制喷油与点火，适应不同海拔与负载工况。 |

1.  **城市快递配送**

4.  **农副产品转运**

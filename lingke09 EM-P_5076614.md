# 领克09 EM-P 保养指南
  
更新时间：2026年07月10日17时52分45秒   

本指南旨在帮助领克09 EM-P车主了解车辆的保养周期、项目及注意事项，以保持车辆最佳性能并延长使用寿命。请结合您的车辆用户手册和实际驾驶情况执行。
---
## 一、核心保养原则
-   **以用户手册为准**：本指南为通用参考，最权威的信息请以您的《领克09 EM-P用户手册》或领克官方App内的电子手册为准。
-   **时间和里程以先到者为准**：保养间隔通常以“每XX个月”或“每行驶XX公里”计算，哪个条件先达到，就应进行保养。
-   **质保与保养**：请在领克授权服务中心或具备资质的正规维修厂进行保养，并保留好记录，以免影响车辆质保权益。
---
## 二、常规保养周期与项目（建议）
> **注意**：领克09 EM-P为插电混动车型，保养逻辑与传统燃油车略有不同，尤其涉及高压系统。
### 1. 首次保养
-   **时间/里程**：购车后 **12个月** 或 **行驶里程达到 15,000 公里**（以先到者为准）。
-   **项目**：
    -   更换发动机机油、机油滤清器。
    -   检查车辆各油液液位（冷却液、制动液、玻璃水等）。
    -   检查轮胎状况及胎压。
    -   检查底盘、悬架及转向系统。
    -   检查全车灯光及电子设备。
    -   检查高压电池系统及充电接口（外观及功能）。
    -   检查空调滤清器（视情况清洁或更换）。
    -   更新车辆系统软件（如有必要）。
### 2. 常规保养（每次保养）
-   **间隔**：每 **12个月** 或每 **15,000 公里**。
-   **固定项目**：
    -   **更换**：发动机机油、机油滤清器。
    -   **检查/更换**：空调滤清器（建议每年更换一次，空气质量差地区缩短）。
    -   **检查**：
        -   制动片/制动盘磨损情况
        -   制动液含水量及液位
        -   轮胎磨损及胎压（含备胎，如有）
        -   底盘、悬架连接件及橡胶衬套
        -   转向系统
        -   全车灯光、雨刮器及喷水系统
        -   12V蓄电池状态
        -   高压系统（高压线束、连接器外观）
        -   驱动电机及电控单元工作状态
### 3. 大保养（每2年或30,000公里）
在常规保养基础上，增加以下项目：
-   **更换**：
    -   **空气滤清器**（发动机进气）
    -   **燃油滤清器**（部分车型或地区要求）
    -   **火花塞**（领克09 EM-P通常为2.0T发动机，建议每30,000-40,000公里更换）
    -   **制动液**（每2年更换，因吸湿性影响制动性能）
-   **检查/清洁**：
    -   节气门（视积碳情况清洗）
    -   发动机燃烧室积碳（视情况使用燃油添加剂或专业清洗）
### 4. 其他重要项目周期
| 项目 | 建议周期 | 备注 |
| :--- | :--- | :--- |
| **变速箱油** | 每 **60,000 - 80,000 公里** 或 4-5年 | 领克09 EM-P为混动专用变速箱，建议按用户手册执行 |
| **防冻液（冷却液）** | 每 **40,000 - 60,000 公里** 或 4-5年 | 检查液位，低于下限需添加同型号产品 |
| **高压电池冷却液** | 参考用户手册 | 混动系统独立冷却回路，需专用设备更换 |
| **轮胎换位** | 每 **10,000 - 15,000 公里** | 建议前后交叉换位，平衡磨损 |
| **四轮定位** | 出现跑偏、轮胎偏磨时 | 非定期项目，按需进行 |
| **空调系统** | 每年夏季前 | 清洁蒸发箱、更换滤芯、检查制冷剂压力 |

## 三、领克09 EM-P 特殊注意事项
1.  **高压安全**：
    -   涉及高压电池、电机、电控系统的维修保养，必须由经过领克认证的专业技师操作。
    -   禁止私自拆卸高压部件或改装电路。
    -   充电接口需保持清洁干燥，避免异物或水汽进入。
2.  **动力电池保养**：
    -   **长期停放**：若车辆需停放超过一个月，建议保持电池电量在 **40%-60%** 之间，并定期（如每两周）启动车辆或充电一次，以激活电池管理系统。
    -   **充电习惯**：尽量使用慢充桩进行日常充电。频繁使用快充（直流快充）会略微加速电池老化。建议每月至少使用慢充充满一次，以均衡电芯电压。
    -   **避免过放**：仪表盘显示电量耗尽（纯电续航为0）时，车辆仍会保留部分电量用于启动发动机和系统保护，但应尽快充电，避免长期低电量状态。
3.  **发动机保养**：
    -   虽然09 EM-P可纯电行驶，但发动机仍需定期运转。若长期不用油，建议每3-4个月主动使用“动力模式”或“运动模式”行驶一段距离，让发动机充分预热并润滑，排出湿气。
    -   机油必须使用符合领克规格（通常为0W-20 C5或5W-30）的全合成机油。
4.  **再生制动系统**：
    -   利用动能回收制动，可减少传统制动片的磨损。但制动片仍需定期检查，特别是内侧制动片。
    -   若感觉制动踏板脚感异常或制动距离变长，应立即检查。

## 四、日常自检清单（建议每月一次）
-   [ ] 检查轮胎胎压及外观（有无鼓包、划伤、异常磨损）。
-   [ ] 检查前后车灯、转向灯、刹车灯是否正常工作。
-   [ ] 检查雨刮器刮水效果，玻璃水是否充足。
-   [ ] 检查仪表盘有无故障灯亮起。
-   [ ] 检查充电枪及充电口有无损坏或异物。
-   [ ] 检查车内各功能按键、屏幕是否正常。
-   [ ] 观察底盘下方有无油液渗漏痕迹。
---
## 五、保养记录建议
建议您通过 **领克官方App** 或 **保养手册** 记录每次保养的：
-   保养日期、行驶里程
-   保养项目及更换的配件
-   下次保养的提醒时间/里程
-   维修技师的签字或工单号

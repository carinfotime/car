# 梅赛德斯-AMG GT新能源 驾驶注意事项
  
更新时间：2026年07月10日23时14分45秒   

## 一、启动与驾驶模式
1.  **启动流程**：
    -   携带钥匙进入车辆，踩下制动踏板，按下“START/STOP”按钮即可启动。
    -   仪表盘会显示“READY”指示灯，表示车辆已进入可行驶状态（电机或发动机已就绪）。
2.  **驾驶模式选择**：
    -   **Electric（纯电模式）**：城市通勤推荐。注意电量消耗，避免深踩油门导致发动机强制介入。
    -   **Comfort（舒适模式）**：日常驾驶首选。系统自动优化油电切换，兼顾平顺与能耗。
    -   **Sport / Sport+（运动/运动增强模式）**：激烈驾驶时使用。发动机与电机协同输出最大功率，悬架变硬，排气声浪增强。
    -   **Race（赛道模式）**：仅限赛道或封闭场地使用。关闭部分电子稳定程序，动力响应最激进。
    -   **Individual（个性化模式）**：可自定义动力、悬架、转向等参数。
## 二、电池与充电管理
1.  **日常充电建议**：
    -   **最佳充电区间**：建议将电量保持在 `20% - 80%` 之间，有助于延长电池寿命。
    -   **避免过放**：尽量避免电量低于10%长期停放，否则可能损坏高压电池。
    -   **充电设备**：优先使用原厂或认证的交流/直流充电桩。直流快充虽快但频繁使用会加速电池衰减。
2.  **能量回收 (Recuperation)**：
    -   可通过方向盘后的拨片调节能量回收强度（D+ / D / D-）。
    -   D-（强回收）：单踏板感较强，适合拥堵路况回收能量。
    -   D+（弱回收/滑行）：适合高速巡航减少拖拽感。
3.  **预调节功能 (Pre-conditioning)**：
    -   连接充电桩时，可通过MBUX系统预设出发时间。车辆会在出发前自动调节电池温度至最佳工作状态并预热/预冷座舱。
## 三、性能操控安全须知
1.  **动力特性警告**：
    -   AMG GT新能源拥有极高的综合扭矩（通常超过800牛·米）。在湿滑路面或低附着力路面下，切勿全油门起步。
    -   “RACE START”弹射起步功能需在特定模式下激活且车辆处于热车状态。请仅在安全封闭道路使用。
2.  **制动系统注意**：
    -   AMG高性能复合制动系统需要达到工作温度才能发挥最佳效能。刚起步时避免急刹。
    -   *再生制动*与*机械制动*的衔接可能带来脚感变化（尤其在满电状态下再生制动受限时），请提前适应。
3.  **轮胎与胎压**：
    -   AMG车型通常配备高性能夏季轮胎或半热熔轮胎。环境温度低于7°C时性能大幅下降，建议更换冬季轮胎。
    -   *胎压必须严格按照油箱盖内侧标签值调整*。过高或过低都会严重影响抓地力和安全性。
4.  **涉水与极端天气**：
    严禁涉水行驶！高压电池及电气系统位于底盘下方，一旦进水可能导致短路甚至触电风险。
## 四、智能系统操作提示
1.  **MBUX信息娱乐系统**：
    熟悉“AMG性能菜单”，可实时查看油温、水温、电机温度、涡轮压力及横向加速度等数据。
2.  **辅助驾驶系统限制**：
    虽然配备L2级辅助驾驶功能，但AMG车型的低矮姿态和宽胎可能导致车道保持在某些弯道中误判。请始终手握方向盘并保持注意力集中。
## 五、维护保养要点
|项目|周期/要求|
|---|---|
|机油更换|严格按行车电脑提示或每1万公里/1年更换AMG专用高性能机油|
|高压电池检查|每2年由授权服务中心进行绝缘检测及健康度评估|
|冷却液|电机和逆变器有独立冷却回路需专用冷却液不可混用普通防冻液|
|刹车片磨损|因动能回收减少机械刹车使用但重刹频率高需定期检查厚度|
## ⚠️特别警示
-   *维修涉及高压部件必须由经过培训的专业技师操作*切勿自行拆卸橙色高压线束或电池包
-   *长时间停放超过30天*建议连接智能充电器保持12V小电瓶电量并将高压电池充至约50%
-   *洗车注意*避免高压水枪直接冲洗发动机舱及底盘高压接口区域

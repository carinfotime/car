# 星途ES 内饰参数介绍
  
更新时间：2026年07月15日14时38分56秒   

## 一、整体设计风格
- **设计理念**：采用“凤之美学”与“曲水流韵”设计语言，环抱式座舱布局，营造简约、科技与豪华的融合氛围。
- **配色方案**：提供多种内饰主题可选，如：
    - **砚黑/瓷白**（经典双色）
    - **砚黑/雅灰**（沉稳运动）
    - **木饰板/金属拉丝**点缀（根据配置不同）
## 二、座椅系统
| 项目 | 参数 / 描述 |
| :--- | :--- |
| **座椅材质** | Nappa真皮（高配）/ 高级仿皮（中低配） |
| **前排座椅功能** | - 主驾：12向电动调节（含4向腰托）<br>- 副驾：10向电动调节<br>- 加热、通风、按摩功能（高配）<br>- 头枕4向可调 |
| **后排座椅功能** | - 加热功能（高配）<br>- 靠背角度电动调节<br>- 中央扶手带触控屏及杯架 |
| **特殊设计** | - “女王副驾”：一键舒躺模式，带电动腿托<br>- 零重力座椅概念（部分版本） |
## 三、中控台与仪表盘
- **仪表盘**：12.3英寸全液晶仪表盘
    - UI界面支持多种主题切换
    - 显示导航、驾驶辅助、能耗等信息
- **中控屏**：15.6英寸2.5K高清悬浮式中控屏
    - OLED材质（高配）/ LCD材质
    - 分辨率：2560 x 1440
- **副驾娱乐屏**：12.3英寸（部分高配车型）
- **HUD抬头显示**：50英寸AR-HUD增强现实抬头显示
    - 投影距离：7.5米
    - 显示内容：车速、导航箭头、ADAS警示等
## 四、智能座舱系统
- **车机芯片**：高通骁龙8155芯片（部分批次或升级为8295）
- **操作系统**：Lion雄狮智云系统 / EXEED OS
- **语音控制**：
    - “你好，星途”唤醒
    - 四音区识别（主驾、副驾、后排左右）
    - 支持连续对话、可见即可说
- **互联功能**：
    -  5G网络 + Wi-Fi热点
    -  蓝牙5.2
    -  手机远程控制（启动空调/车窗/门锁）
    - OTA整车在线升级
## 五、音响系统
|项目 |参数 |
| :--- | :--- |
|品牌 |Lion Melody / （部分配置为23扬声器品牌音响）|
|扬声器数量 |标准版8个 / Hi-Fi版23个 |
|特色技术 |-头枕扬声器（导航/电话独立播放）<br>-主动降噪技术<br>-7.1.4全景声布局 |
## 六、空调与空气净化
-空调类型 ：自动双区恒温空调 （高配为三区独立控制 ）
-后排出风口 ：标配 ，带独立控制面板 （高配 ）
-空气净化系统 ：AQS空气质量监测 + PM2.5过滤 +负离子发生器 +香氛系统 （多款香型可选 ）
## 七 、方向盘与换挡机构 
-**方向盘材质**:真皮包裹 
-**方向盘调节**:手动四向调节 （上下+前后 ） 
-**多功能按键**:集成巡航控制 、多媒体 、语音唤醒等 
-**换挡方式**:怀挡式电子换挡 （位于方向盘右后方 ） 
## 八 、其他内饰细节 
-**氛围灯**:256色无极变色氛围灯 ，贯穿中控台及门板 
-**天窗/天幕**:全景天幕 （不可开启 ，带可变光调光功能 ）或全景天窗 （可开启 ，视配置而定 ） 
-**储物空间**:无线充电面板 （50W快充 ，带散热 ）；中央扶手箱容积约8L ；门板储物格可容纳1L水瓶 
-**内后视镜**:自动防眩目流媒体后视镜 
---
以上参数基于星途ES主流配置整理 ，具体以实车最终配置为准 。

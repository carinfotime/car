# 雪佛兰开拓者（Chevrolet Blazer）空间参数介绍
  
更新时间：2026年07月19日10时58分19秒   

## 车身尺寸
| 参数 | 数据 |
|------|------|
| 车长 | 4,999 mm |
| 车宽 | 1,953 mm |
| 车高 | 1,722 mm（含行李架） |
| 轴距 | 2,863 mm |
| 前轮距 | 1,687 mm |

## 内部乘坐空间
### 前排空间
- **头部空间**：约 **1,020 mm**
- **腿部空间**：最大可达 **1,080 mm**
- **肩部空间**：约 **1,500 mm**
### 第二排空间
- **头部空间**：约 **990 mm**
- **腿部空间**：最大可达 **950 mm**（座椅可前后滑动调节）
- **肩部空间**：约 **1,480 mm**
- **座椅调节**：支持前后滑动 + 靠背角度调节
### 第三排空间（7座版）
- **头部空间**：约 **920 mm**
- **腿部空间**：约 **700–800 mm**（受第二排位置影响）
- **肩部空间**：约 **1,350 mm**
- *适合身高 ≤175 cm的乘客短途乘坐*
## 座椅布局
| 版本 | 座椅布局 |
|------|----------|
| RS / Redline / Standard版 | `2+3` （5座）或 `2+3+2` （7座）可选 |
## 储物能力
### 🧳后备箱容积
|  状态   |  容积   |
|---|---|
|  三排全部立起（7座）   |  约222 L   |
|  第三排放倒（5座模式）   |  约883 L   |
|  第二、三排全部放倒   |  约1,970 L（最大扩展容积） |
> ✅第二、三排座椅支持一键放平，形成近乎纯平的装载平台。
### 🧰车内储物细节
#### 🔹中控区域
- 📦中央扶手箱容积：约12 L
- 📱手机无线充电槽 + USB接口储物格
- 🥤杯架 ×4（前排2个 +后排中央扶手2个）
#### 🔹门板储物格
- 🚪前排门板可容纳 ≥3瓶500ml矿泉水
- 🚪后排门板可容纳 ≥2瓶500ml矿泉水
#### 🔹其他储物位
- 🧤手套箱标准容量，带照明灯
- 💺前排座椅背后地图袋 ×2
- 🎒第三排两侧杯架 +储物槽 ×各1
## 📐灵活性与便利性配置
### 🔑进出便利性设计
✅第二排座椅一键翻折 +滑动功能，方便第三排进出  
✅第三排独立空调出风口 +阅读灯  
✅电动尾门（高配车型支持脚踢感应开启）
### 🔄座椅折叠方式

第二排放倒 → "4/6比例放倒"
第三排放倒 → "5/5比例放倒" 
全部放平 → "纯平装载模式"

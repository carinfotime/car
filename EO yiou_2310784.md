# EO 羿欧 技术参数介绍
  
更新时间：2026年07月13日14时10分48秒   

## 品牌概述
EO（羿欧）是一个专注于高性能电动出行与智能科技融合的新锐汽车品牌。品牌致力于通过前瞻的设计语言、极致的能效管理以及先进的自动驾驶辅助系统，为全球用户提供兼具驾驶乐趣与环保理念的移动解决方案。

## 核心车型：EO 羿欧 · 星驰 (EO Stellar)
### 一、动力系统
| 项目 | 参数 |
| :--- | :--- |
| **驱动形式** | 双电机智能四驱 / 后驱 |
| **电机类型** | 永磁同步电机 (碳化硅逆变器) |
| **综合最大功率** | **400 kW (544 PS)** |
| **综合最大扭矩** | **720 N·m** |
| **0-100 km/h加速时间** | **3.8秒 (四驱版)** / **5.6秒 (后驱版)** |
| **最高时速** | 220 km/h (电子限速) |
### 二、电池与续航
| 项目 | 参数 |
| :--- | :--- |
| **电池类型** | NCM811三元锂电池 / LFP磷酸铁锂可选 |
| **电池容量** | **82 kWh (三元锂)** / **60 kWh (磷酸铁锂)** |
| **CLTC综合续航里程** | **650 km (三元锂长续航版)** / **450 km (磷酸铁锂标准版)** |
| **快充时间** | SOC 10%-80%：约25分钟 (800V超充平台) |
| **慢充时间** | SOC 0%-100%：约7小时 (家充桩7kW) |
### 三、车身与底盘
| 项目 | 参数 |
| :--- | :--- |
| **车身结构** | 钢铝混合轻量化车身，高强度热成型钢占比 >30% |
| **悬架系统（前）** | 双叉臂式独立悬架，带横向稳定杆 |
| **悬架系统（后）** | 多连杆式独立悬架，带自适应空气悬架（选装） |
| **制动系统** | Brembo四活塞卡钳 +  通风盘式刹车（前轮）|
| **转向系统** | DP-EPS双小齿轮电动助力转向，可变转向比 |

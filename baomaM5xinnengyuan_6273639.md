# 宝马M5新能源 动力参数介绍
  
更新时间：2026年07月13日10时19分45秒   

## 动力总成概述
宝马M5新能源（BMW M5 新能源）搭载了一套 **高性能插电式混合动力系统**，结合了经典的V8双涡轮增压发动机与高性能电机，实现了燃油性能与电动化的深度融合。
---
## 核心动力参数
| 项目 | 参数 |
|------|------|
| **发动机类型** | 4.4L V8 双涡轮增压发动机 |
| **电机类型** | 高性能永磁同步电机 |
| **综合最大功率** | **约 535 kW（约 727 匹马力）** |
| **综合最大扭矩** | **约 1000 N·m** |
| **变速箱** | 8速 M Steptronic 手自一体变速箱 |
| **驱动方式** | M xDrive 智能全轮驱动系统 |
---
## 发动机参数（S68 4.4L V8）
- **排量**：4,395 cc
- **进气形式**：双涡轮增压（TwinPower Turbo）
- **最大功率**：约 **430 kW（585 匹马力）**
- **最大扭矩**：约 **750 N·m**
- **缸体/缸盖材料**：铝合金
---
## 电机参数
- **电机类型**：永磁同步电机
- **最大功率**：约 **145 kW（197 匹马力）**
- **最大扭矩**：约 **280 N·m**
- **电池类型**：锂离子高电压电池组
- **电池容量**：约 **18.6 kWh（可用容量）**
- **纯电续航里程**：约 **60-80 km（WLTP工况）**
---
## 性能表现
| 项目 | 数据 |
|------|------|
| **0-100 km/h 加速** | **约 3.5 秒** |
| **0-200 km/h 加速** | **约 11.0 秒** |
| **最高时速（电子限速）** | **250 km/h** |
| **最高时速（选装M驾驶套件）** | **305 km/h** |
| **纯电最高时速** | **约 140 km/h** |
---
## 驱动与操控
- **M xDrive 智能全轮驱动系统**：支持 4WD、4WD Sport、2WD（后驱漂移模式）三种模式切换
- **后桥主动式M差速锁**
- **M 自适应悬架**（电子控制减震器）
- **M 复合制动系统**（前六活塞卡钳 / 后单活塞卡钳）
---
## 充电与能耗
- **充电方式**：支持交流慢充（AC）
- **慢充时间**：约 4-5 小时（0-100%，家用充电桩）
- **综合油耗（WLTP）**：约 **2.5-3.0 L/100km**（混动模式下）
- **纯电能耗**：约 **22-25 kWh/100km**
---
## 驾驶模式
| 模式 | 说明 |
|------|------|
| **HYBRID（混动模式）** | 默认模式，智能切换油电驱动 |
| **ELECTRIC（纯电模式）** | 纯电行驶，零排放 |
| **eCONTROL（电量保持）** | 保持当前电量，优先使用燃油 |
| **SPORT / SPORT+** | 发动机+电机全力输出，性能最大化 |
---
> **总结**：宝马M5新能源将经典的 **4.4L V8双涡轮增压发动机** 与 **高性能电机** 结合，综合功率超过 **700匹马力**，扭矩高达 **1000 N·m**，0-100km/h加速仅需 **3.5秒**，同时具备 **60-80km的纯电续航能力**，是性能与环保兼顾的旗舰级高性能轿车。

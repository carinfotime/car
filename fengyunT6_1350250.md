# 风云T6 儿童乘车安全注意事项
  
更新时间：2026年07月14日13时41分02秒   

## 🚗 基本安全须知
1. **儿童不得坐在前排**
   - 12岁以下儿童严禁乘坐副驾驶位置
   - 前排安全气囊弹出可能对儿童造成严重伤害
2. **正确使用儿童安全座椅**
   - 4岁以下儿童必须使用反向安装的儿童安全座椅
   - 4-7岁儿童应使用正向安装的儿童安全座椅
   - 8-12岁儿童可使用增高垫配合成人安全带
3. **后排落座**
   - 所有儿童应始终坐在后排座位
   - 后排中间位置相对最安全（如配备三点式安全带）
## 🔒 车门与车窗安全
1. **启用儿童安全锁**
   - 风云T6后门配备儿童安全锁，请务必启用
   - 防止行车中儿童意外打开车门
2. **车窗控制**
   - 开启车窗防夹功能（风云T6标配）
   - 切勿让儿童将头、手伸出窗外
   - 建议使用主驾位的车窗锁止功能
## ⚠️ 行车途中注意事项
1. **切勿将儿童单独留在车内**
   - 夏季车内温度可在短时间内升至致命水平
   - 冬季也存在缺氧风险
2. **车内物品管理**
   - 收起中控台、座椅上的硬物和尖锐物品
   - 避免放置小零件、硬币等易吞食物品在儿童可触及范围
3. **避免干扰驾驶**
   - 教育孩子不要在行车过程中吵闹或拉扯驾驶员
   - 可准备玩具、绘本等分散注意力
## 🧸 ISOFIX接口使用指南（风云T6）
| 项目 | 说明 |
|------|------|
| ISOFIX接口位置 | 后排左右两侧座椅靠背与坐垫连接处 |
| Top Tether锚点 | 后排座椅靠背后方或后备箱内 |
|适用座椅类型 | Group0/0+/1/2/3组别 |
**安装步骤：**
1. 找到后排座椅ISOFIX接口标识
2. 将安全座椅接口对准插入，听到"咔嗒"声确认锁定
3. 连接顶部Top Tether固定带并拉紧
4. **确认座椅晃动幅度不超过2.5cm**
## 📋 Age-by-Age Checklist
### 👶婴幼儿（0-3岁）
- [ ] ✅反向安装婴儿提篮或安全座椅
- [ ] ✅安全带松紧度适中（一指可插入为宜）
- [ ] ✅头部护翼调整至合适高度
### 🧒学龄前（4-7岁）
- [ ] ✅正向安装带五点式安全带的安全座椅
- [ ] ✅肩带高度与肩部齐平或略高
- [ ] ✅胸部夹扣位于腋窝水平
### 👦大龄儿童（8-12岁）
- [ ] ✅使用增高垫直至身高达到150cm
- [ ] ✅肩带跨过肩膀中部而非颈部或手臂下方
- [ ] ✅腰带紧贴大腿上部而非腹部
## 🌡️夏季特别提醒
> 🔥 **高温警示**：即使室外温度仅25°C，车内温度10分钟内可达40°C以上。**永远不要**将孩子单独留在车内，哪怕只是"一小会儿"。

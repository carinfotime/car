|---------|---------|

| 更换燃油滤清器 | 每20,000公里或24个月 |
| 更换变速箱油 | 每40,000公里或24个月 |
| 更换后桥齿轮油 | 每40,000公里或24个月 |
| 更换冷却液 | 每40,000公里或24个月 |
| 更换制动液 | 每40,000公里或24个月 |

## 二、定期保养项目详解
### 1. 发动机机油与机滤（每5,000公里）
- **推荐机油型号：** SAE 15W-40 或 10W-30（视当地气候）
- **机油等级：** API CI-4 或 CH-4 以上级别柴油机油
- **机油容量：** 约8.5~9.5升（视具体发动机型号）
- **操作要点：**
  - 热车状态下放油更彻底
  - 更换机滤时需在密封圈涂抹少量机油
  - 拧紧力矩：放油螺栓约35~40N·m，机滤约15~20N·m
### 2. 空气滤清器（每10,000公里）
- **检查方法：** 取出滤芯，对着光源观察是否透光，若严重堵塞或破损需更换
- **清洁方式：** 可用压缩空气从内向外吹净（压力不超过5bar）
- **更换周期：** 在多尘环境下建议缩短至5,000公里
### 3. 燃油滤清器（每20,000公里）
- **注意：** 柴油版柯斯达需定期排放燃油滤清器底部积水
- **更换后需排气：** 手动泵油直至手感变硬，启动发动机检查是否漏油
- **建议：** 每次更换后记录里程
### 4. 变速箱油（每40,000公里）
- **手动变速箱：** GL-4 75W-90 齿轮油，容量约2.5~3.0升
- **自动变速箱：** 丰田原厂ATF Type T-IV 或等同规格
- **检查方法：** 发动机怠速运行，油温正常时检查油尺刻度
### 5. 冷却液（每40,000公里）
- **推荐型号：** 丰田长效冷却液（LLC）或乙二醇基防冻液
- **混合比例：** 50%冷却液 + 50%蒸馏水
- **容量：** 约12~15升
- **注意事项：**
  - 必须在冷车状态下打开水箱盖
  - 严禁混用不同颜色的冷却液
  - 定期检查副水箱液位在FULL与LOW之间
### 6. 制动系统（每40,000公里或视磨损情况）
- **制动液型号：** DOT 3 或 DOT 4
- **制动片厚度：** 前片≥3mm，后片≥3mm，低于此值需更换
- **制动盘磨损：** 检查表面是否出现深沟槽或裂纹，跳动量≤0.05mm
- **制动液更换：** 需两人配合排气，顺序为：右后→左后→右前→左前
### 7. 轮胎（每10,000公里）
- **标准胎压（冷态）：**
  - 前轮：4.5~5.0 kg/cm²（视载重调整）
  - 后轮：5.0~6.0 kg/cm²（视载重调整）
- **花纹深度：** 不低于1.6mm（法定最低标准），建议≥3mm
- **换位方式：** 交叉换位法（左前→右后，右前→左后，左后→左前，右后→右前）

## 三、各里程节点保养清单
### 5,000公里 / 6个月（小保养）
- [ ] 更换发动机机油及机油滤清器
- [ ] 检查并补充各油液（冷却液、制动液、转向助力油、玻璃水）
- [ ] 检查轮胎胎压及外观
- [ ] 检查灯光、喇叭、雨刮器
- [ ] 检查底盘螺栓紧固情况
- [ ] 检查传动轴及万向节
### 10,000公里 / 12个月
- [ ] 包含5,000公里所有项目
- [ ] 更换空气滤清器
- [ ] 检查/更换空调滤清器
- [ ] 轮胎换位及动平衡
- [ ] 检查刹车片厚度
- [ ] 检查蓄电池电解液液位及接线柱
- [ ] 润滑车门铰链、锁扣

- [ ] 包含10,000公里所有项目

- [ ] 检查并调整气门间隙
- [ ] 检查发电机及起动机
- [ ] 检查全车皮带张紧度

- [ ] 包含20,000公里所有项目
- [ ] 更换变速箱油

- [ ] 清洗节气门及喷油嘴
- [ ] 检查空调系统制冷剂压力
- [ ] 四轮定位检查

## 四、日常检查项目（建议每周或每次出车前）
| 检查项目 | 标准 |
|---------|------|
| 机油液位 | 油尺刻度在H与L之间 |
| 冷却液液位 | 副水箱在FULL与LOW之间 |
| 制动液液位 | 在MAX与MIN之间 |
| 转向助力油液位 | 在HOT/COLD对应刻度内 |
| 轮胎外观及胎压 | 无鼓包、无异常磨损 |
| 灯光系统 | 大灯、转向灯、刹车灯正常 |
| 仪表盘警示灯 | 无异常指示灯亮起 |
| 雨刮器及喷水 | 刮净度良好，喷水正常 |
| 倒车镜及后视镜 | 调整到位，无破损 |

## 五、常见故障及处理建议
| 故障现象 | 可能原因 | 处理方法 |
|---------|---------|---------|
| 启动困难 | 蓄电池亏电、燃油滤清器堵塞、预热塞故障 | 检查蓄电池电压、更换燃油滤、检查预热系统 |
| 发动机抖动 | 喷油嘴堵塞、气门间隙异常、机脚胶老化 | 清洗喷油嘴、调整气门间隙、更换机脚胶 |
| 水温过高 | 冷却液不足、节温器故障、散热器堵塞 | 补充冷却液、更换节温器、清洗散热器 |
| 制动偏软 | 制动液不足、制动片磨损、制动系统有空气 | 补充/更换制动液、更换制动片、排气 |
| 转向沉重 | 转向助力油不足、轮胎气压低、转向机故障 | 补充助力油、补充胎压、检修转向机 |

## 六、重要注意事项
1. **使用正品配件：** 建议使用丰田纯牌零件或同等质量品牌配件，尤其是机油滤清器、燃油滤清器等关键部件。
2. **记录保养：** 每次保养后记录里程、更换项目及配件信息，便于后续维护参考。
3. **磨合期：** 新车或大修后前1,500公里为磨合期，避免高速行驶（不超过80km/h）和重载。
4. **柴油版特别提醒：**
   - 使用符合国标的合格柴油
   - 定期排放油水分离器积水
   - 注意DPF（颗粒捕集器）再生指示灯
5. **长期停放：** 若车辆停放超过1个月，建议断开蓄电池负极，并将胎压适当调高。
6. **季节性保养：**
   - **入夏前：** 检查空调系统、冷却液浓度
   - **入冬前：** 检查防冻液冰点、蓄电池状态、预热系统

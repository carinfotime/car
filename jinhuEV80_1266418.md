# 金琥EV80 内饰参数介绍
  
更新时间：2026年07月20日15时37分24秒   

## 整体内饰风格
金琥EV80内饰采用简约实用设计风格，注重功能性与舒适性的平衡，整体以深色系为主色调，搭配银色或镀铬装饰条，营造出沉稳、商务的座舱氛围。
---
## 座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 高级织物面料 / 可选配仿皮材质 |
| 座椅布局 | 2座（前排） |
| 主驾座椅调节 | 手动4向调节（前后+靠背角度） |
| 副驾座椅调节 | 手动4向调节 |
| 座椅填充 | 高密度发泡海绵，支撑性良好 |
| 座椅功能 | 头枕高度可调 |
---
## 方向盘
| 项目 | 参数 |
|------|------|
| 方向盘材质 | 聚氨酯（PU）包裹 |
| 方向盘调节 | 手动上下调节 |
| 多功能按键 | 可选配（音响控制/蓝牙电话） |
---
## 仪表盘
| 项目 | 参数 |
|------|------|
| 仪表类型 | 全液晶仪表盘 |
| 仪表尺寸 | 7英寸（部分配置） |
| 显示内容 | 车速、电量、续航里程、电机转速、故障提示、挡位信息等 |
---
## 中控系统
| 项目 | 参数 |
|------|------|
| 中控屏幕 | 7英寸触控屏（部分配置） |
| 多媒体系统 | 支持收音机、USB播放、蓝牙音乐 |
| 手机互联 | 支持蓝牙电话连接 |
| 扬声器数量 | 2个 |
---
## 空调系统
| 项目 | 参数 |
|------|------|
| 空调类型 | 手动空调 |
| 制冷/制热 | 电动空调压缩机（制冷）+ PTC加热器（制热） |
| 出风口 | 前排中央出风口 + 两侧出风口 |
---
## 储物空间
| 位置 | 描述 |
|------|------|
| 手套箱 | 副驾前方开放式储物格 |
| 门板储物 | 左右前门板杯架+储物槽 |
| 中控台储物 | 换挡旋钮前方储物格、杯架 |
| 顶棚储物 | 可选配顶置储物盒 |

# 帕拉索新能源 驾驶注意事项
  
更新时间：2026年07月20日10时35分26秒   

## 一、启动与行驶前检查
1.  **充电状态检查**：出发前确认电池电量充足，并预估续航里程是否满足当日行程需求。
2.  **轮胎检查**：检查胎压是否处于标准范围（通常可在驾驶员侧门框或油箱盖内侧找到标签），确保轮胎无异常磨损或鼓包。
3.  **仪表盘警示灯**：启动车辆后，观察仪表盘是否有故障灯亮起（如动力电池故障、电机故障等）。
4.  **充电口盖**：确认充电口盖已完全关闭并锁紧，防止行驶中意外打开。
## 二、行驶中的操作规范
1.  **平稳驾驶**：
    -   避免急加速和急刹车，这不仅有助于节省电能，还能保护电机与电池系统。
    -   建议使用经济模式（ECO），以获得更优的续航表现。
2.  **能量回收**：
    -   熟悉并合理利用能量回收系统。在减速或下坡时，松开油门即可回收能量，增加续航。
    -   可根据路况调节回收强度（如强回收适合城市拥堵路况）。
3.  **涉水与恶劣天气**：
    -   **严禁涉水行驶**：虽然电池组有防水设计，但电机及高压线束仍可能因进水损坏。遇到积水路段请绕行。
    -   **雨天/雪天**：降低车速，增大跟车距离。新能源车自重较大，制动距离可能更长。
4.  **高速行驶**：
    -   高速行驶会显著增加电耗。建议保持匀速（如90-100km/h），避免长时间极速行驶。
5.  **拖拽与负载**：
    -   **禁止私自拖拽其他车辆或重物**：除非车辆说明书明确允许且配备专用拖钩。非承载式车身虽有一定强度，但拖拽可能导致动力系统过载。
## 三、充电注意事项
1.  **使用正规充电桩**：尽量使用符合国标的公共充电桩或原厂家用充电桩。避免使用劣质或改装插座。
2.  **快充与慢充结合**：
    -   *日常通勤*：优先使用慢充（交流电），有利于延长电池寿命。
    -   *长途应急*：可使用快充（直流电），但建议电量降至20%左右再充至80%，避免频繁满充满放快充。
3.  **极端温度充电**：
    -   *冬季*：刚停车时电池温度较高，此时立即充电效率最佳。若车辆在低温下停放过久，建议先预热电池再充电（部分车型支持预约加热）。
    -   *夏季高温*：避免暴晒后立即快充，待电池温度稍降后再进行。
4.  **禁止飞线充电**：严禁从高层住宅私拉电线进行“飞线”充电，存在严重火灾隐患。
## 四、停车与长期停放
1.  **选择安全位置**：尽量停放在室内停车场或有遮阳的场所，避免长时间暴晒或雨淋高压部件。
2.  **长期停放策略**：
    -   若需停放超过一周以上，建议将电量保持在50%-60%左右（过高或过低均不利于电池健康）。
3.  **关闭所有用电设备**：熄火前确保大灯、空调、音响等已关闭。
## ⚠️紧急情况处理
-   **碰撞事故后处理流程**
    1.  立即熄火并拔出钥匙（或远离智能钥匙感应区）。
    2.  尽快撤离车辆至安全地带（至少15米外）。
    3.  拨打救援电话时明确告知“这是一辆新能源汽车”，以便消防人员采取特定高压断电措施。
-   **仪表盘出现红色高压警告灯**
    立即靠边停车熄火；切勿自行触摸任何橙色高压线束；联系售后救援。
-   *注意*：以上内容基于通用新能源车及帕拉索车型特性整理。具体操作请以随车附带的《用户手册》为准。

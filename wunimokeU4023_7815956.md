# 乌尼莫克U4023 驾驶注意事项
  
更新时间：2026年07月14日10时45分25秒   

## 一、启动前检查
1.  **胎压与轮胎状态**：检查轮胎气压是否处于标准范围（通常前轮较高，后轮视负载调整）。确认无扎钉、鼓包或异常磨损。
2.  **油液液位**：检查发动机机油、冷却液、制动液、转向助力油及尿素液位。
3.  **底盘与传动**：目视检查传动轴、差速锁气路接头、悬挂气囊（如有）有无漏气或松动。
4.  **灯光与信号**：测试所有外部灯光、转向灯及警示灯功能正常。
## 二、驾驶操作要点
### 1. 变速箱与分动箱
-   **手动挡操作**：U4023通常配备8速或16速手动/半自动变速箱。换挡时务必踩死离合器，避免打齿。
-   **高低挡切换**：在切换高低挡位（Hi/Lo）时，车辆必须完全停止，且挂入空挡后再操作分动箱开关。
-   **倒挡使用**：挂入倒挡前确保车辆完全静止。
### 2. 差速锁使用
-   **顺序锁定**：先锁后桥差速锁 → 再锁前桥差速锁 → 最后锁中央差速锁。解除时按相反顺序操作。
-   **禁止转向**：在铺装路面（柏油路/水泥路）上严禁锁定差速锁行驶，否则将严重损坏传动系统。仅在泥地、沙地或雪地等低附着力路面使用。
-   **脱困后立即解除**：车辆脱困后应立即关闭所有差速锁。
### 3. 涉水驾驶
-   **涉水深度**：标准涉水深度约80cm，加装涉水喉后可提升至120cm。切勿超过进气口高度。
-   **涉水前准备**：关闭空调外循环，检查并确保所有排水阀关闭。低速匀速通过，避免激起浪花导致发动机进水。
-   **涉水后检查**：驶出深水区后轻踩刹车以蒸发制动盘水分，并检查桥壳及变速箱通气孔是否进水。
### 4. 越野驾驶
-   **胎压调整**：进入松软沙地或泥地前适当降低胎压（建议降至1.5bar以下），增大接地面积。驶回铺装路前必须恢复标准胎压。
-   **接近角与离去角**：注意U4023的接近角约44度，离去角约51度。上下陡坡时控制车速，避免托底或刮擦保险杠。
-   **侧倾风险**：由于车身较高且重心偏上，侧向斜坡行驶时务必评估侧倾角度（建议不超过30度），避免急打方向。
## 三、特殊功能操作
### - VarioPilot®可转换方向盘
    -   如需将方向盘从左侧移至右侧（右舵国家/工地需要），必须在车辆完全停止且熄火状态下进行。严格按照随车手册的机械锁定步骤操作。
### - PTO（取力器）
    -   启动PTO驱动绞盘或液压系统前，确认车辆处于空挡且手刹已拉紧。严禁在PTO接合状态下行驶。
## 四、安全警示
1.  **视野盲区**：U4023引擎盖高耸且A柱较粗壮，前方及右前方存在较大盲区。起步和转弯时应探头观察或借助副驾指挥。
2.  **制动距离**：满载状态下（总质量可达14吨以上）制动距离远大于普通乘用车。保持足够跟车距离，预见性驾驶。
3.  **下坡控制**：
    -   长下坡时使用发动机制动（排气制动/缓速器），避免长时间踩刹车导致热衰减失效。
    -   手动模式下降档利用发动机转速牵制车速。
4.  **货物固定**：货箱装载货物时必须使用绑带或链条牢固固定。乌尼莫克越野时的颠簸幅度极大，未固定的货物可能移位造成危险。

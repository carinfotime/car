# 上汽大通 MAXUS 新途 V90 动力参数介绍
  
更新时间：2026年07月16日11时35分48秒   

## 核心动力总成
新途V90搭载了上汽集团自主研发的 **“上汽π”2.0T高性能柴油发动机**，并匹配了多种变速箱选项，以满足不同用户对动力输出和燃油经济性的需求。
### 发动机参数
| 项目 | 参数 |
| :--- | :--- |
| **发动机型号** | “上汽π” 2.0T 涡轮增压柴油发动机 |
| **排量** | 2.0L (1996ml) |
| **进气形式** | 涡轮增压 |
| **燃料形式** | 柴油 |
| **最大功率** | **110 kW (约150马力)** / @3500rpm |
| **最大扭矩** | **375 N·m** / @1500-2400rpm |
| **排放标准** | 国VI b |
### 技术亮点
*   **高效低扭：** 得益于双增压系统（部分高功率版本）或优化的涡轮匹配，发动机在 **1500转时即可输出峰值扭矩375N·m**，这意味着车辆在低速起步、爬坡及重载工况下拥有非常充沛的动力储备。
*   **静音设计：** “上汽π”发动机采用了多项静音技术（如平衡轴、高精度齿轮等），有效降低了柴油发动机的运转噪音，提升了驾乘舒适性。
*   **燃油经济性：** 通过高压共轨直喷技术和优化的燃烧室设计，实现了较低的油耗水平。根据官方数据及用户反馈，百公里综合油耗通常在7.6L - 8.5L之间（视具体车型和驾驶习惯而定）。
## 变速箱选项
新途V90提供了多种变速箱选择，以适应不同的使用场景：
1.  **6速手动变速箱 (6MT)**
    *   **特点：** 结构可靠、传动效率高、维护成本低。
    *   **适用场景：** 追求驾驶操控感、对成本较为敏感或经常行驶在复杂路况下的用户。
2.  **6速自动变速箱 (6AT)**
    *   **特点：** （由法国邦奇提供）换挡平顺、可靠性好。
    *   **适用场景：** 城市物流配送、频繁启停路况，降低驾驶疲劳度。
3.  **(部分新款/高配车型) ZF-8速自动变速箱 (8AT)**
    *   *注：需以具体配置表为准*
    *   **特点：** （采埃孚提供）齿比更绵密、换挡响应更快、高速巡航时转速更低，进一步优化燃油经济性和行驶静谧性。
    *   **适用场景：** 对舒适性和长途行驶品质有更高要求的用户。
## NVH与驾驶体验
得益于“上汽π”发动机的低振动设计和整车NVH优化：
*   怠速状态下车内噪音控制表现良好。
*   中低速加速时动力响应直接，无明显涡轮迟滞感。
*   高速巡航时（配合8AT变速箱），发动机转速维持在较低水平（约2000rpm左右），保证了车内的安静环境。
## 【总结】
新途V90的动力系统以“大扭矩、低油耗、低噪音”为核心优势。其2.0T柴油机提供的375N·m峰值扭矩在同级车型中表现突出，无论是用于城市物流运输还是商务接待/旅居车改装，都能提供充足的动力保障。搭配成熟的自动变速箱后，大幅降低了驾驶难度和疲劳度。

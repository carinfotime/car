# 巴菲特9 驾驶注意事项
  
更新时间：2026年07月19日11时44分03秒   

为确保您和乘客的安全，并充分发挥车辆性能，请仔细阅读并遵守以下驾驶注意事项。
## 一、启动与预热
1.  **钥匙/卡片识别**：上车后，请确保智能钥匙或卡片处于车内感应区域。若无法识别，请检查钥匙电池或按说明书指引进行应急启动。
2.  **静音启动**：启动车辆时，请注意仪表盘指示灯。由于新能源车（假设）启动无声，请确认车辆已进入“READY”状态再挂挡行驶。
3.  **冬季预热**：在寒冷天气下，建议启动后等待1-2分钟，让电池和电机系统达到适宜工作温度，避免立即急加速。
## 二、行驶中的核心要点
### 1. 动力与加速
-   **平顺驾驶**：起步时轻踩加速踏板，“巴菲特9”动力响应较快，避免因瞬间扭矩过大导致车辆窜出。
-   **能量回收**：熟悉车辆的能量回收制动强度。松开加速踏板时会有拖拽感（类似轻踩刹车），提前预判路况可减少刹车片磨损并增加续航。
### 2. 转向与操控
-   **方向盘握姿**：双手握持方向盘3点和9点位置。转弯时避免大幅度急打方向。
-   **底盘高度**：若车型为SUV或跨界风格（假设），注意底盘高度。通过减速带或坑洼路面时需减速慢行，保护悬挂系统。
### 3. 制动系统
-   **刹车距离**：由于车重较大（新能源车普遍较重），请保持比燃油车更长的跟车距离。
-   **紧急制动**：遇到突发情况时，果断用力踩下刹车踏板。ABS（防抱死制动系统）介入时会有弹脚感，属正常现象。
## 三、特殊路况应对
| 路况类型 | 操作建议 |
| :--- | :--- |
| **雨天/湿滑路面** | - 降低车速至干燥路面的70%以下。<br>- 避免猛打方向或急加速。<br>- ESP（车身稳定系统）默认开启，切勿手动关闭。 |
| **山路/长下坡** | - 利用能量回收拨片或低挡位控制车速。<br>- 避免长时间连续踩刹车导致热衰减（虽然电车热衰减较轻）。 |
| **积雪/结冰路面** | - 使用雪地模式（如有）。<br>-起步轻柔防止车轮空转。<br>-增大跟车距离至平时的3倍以上。 |
## 四、停车与充电
1.  **驻车步骤**：
    -   完全停稳后 → N挡 →拉起电子手刹 → P挡 →关闭电源。
    -   严禁在未停稳时直接按P挡。
2.  **充电安全**：
    -   使用符合国标的充电桩。
    -   雷雨天气尽量避免户外充电。
    -   充电口盖板需确保完全闭合后再离开。
## ⚠️五、重要警告
> ❌ **禁止事项**
> *  禁止在驾驶过程中操作中控大屏观看视频或进行复杂设置。
> *  禁止在非官方授权店改装电路或刷写ECU（发动机控制单元）。
> *  禁止将儿童单独留在车内且车辆处于通电状态。
> ✅ **必须事项**
> *   每次出行前检查轮胎胎压及外观有无异常鼓包。
> *   定期更新车载系统固件以获取最新安全优化。

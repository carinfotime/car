# 雪佛兰创酷（Trax）内饰参数介绍
  
更新时间：2026年07月17日18时38分46秒   

## 一、整体设计风格
- **设计理念**：以驾驶者为中心的环抱式座舱设计，强调运动感与年轻化
- **内饰色调**：提供纯黑、黑棕双拼等多种配色方案
- **材质用料**：中控台采用软性搪塑工艺包裹，搭配银色镀铬饰条与钢琴烤漆面板
## 二、座椅系统
| 项目 | 参数 |
|------|------|
| 座椅材质 | 织物 / 皮织混搭 / 仿皮（视配置而定） |
| 前排座椅调节 | 手动6向调节（部分高配主驾电动6向） |
| 后排座椅 | 4/6比例放倒，支持靠背角度调节 |
| 座椅功能 | 高配车型配备前排座椅加热 |
| 乘坐人数 | 5座布局 |
## 三、方向盘
- **类型**：三辐式多功能方向盘
- **材质**：真皮包裹（中高配）/ PVC塑料（低配）
- **调节方式**：手动上下+前后四向调节
- **功能按键**：集成定速巡航、多媒体控制、蓝牙电话、语音控制等
## 四、仪表盘
- **样式**：双炮筒式机械仪表 + 中间3.5英寸/8英寸彩色行车电脑显示屏
- **显示内容**：车速、转速、油耗、胎压监测、驾驶模式等信息
## 五、中控多媒体系统
| 项目 | 参数 |
|------|------|
| 屏幕尺寸 | 8英寸高清触控屏（标配） |
| 系统名称 | MyLink⁺智能互联系统 |
| CarPlay | ✅ Apple CarPlay（有线） |
| CarLife | ✅ Baidu CarLife |
| OTA升级 | ✅支持在线升级 |
| USB接口 |前排2个 +后排1个（Type-A）|
| AUX接口 | ✅（部分车型）|
| SD卡槽 | ✅ |
## 六、空调系统
- **类型**：
    - **手动空调**（低配）：旋钮式控制，带花粉过滤
    - **自动空调**（高配）：单区自动恒温空调，带PM2.5过滤功能
    - **后排出风口**：全系标配
## 七、储物空间
### 前排储物：
- **手套箱**：带阻尼开启，容积约8L
- **中央扶手箱**：双层设计，深度约15cm
- **门板储物格**：可放置水瓶及杂物（前门可放1.5L水瓶）
- **杯架**：前排2个 +后排中央扶手2个
### 后备箱：
- **标准容积**：约390L  
- **最大扩展容积**：约1334L（后排座椅放倒后）
- **开口高度**：离地约70cm，便于装卸物品
## 八、其他内饰配置亮点
### 灯光与氛围：
-  阅读灯：前后排独立LED阅读灯  
-  氛围灯：单色冰蓝氛围灯（仅高配）
### 便利配置：
-  一键启动 +无钥匙进入（中高配）
-  电动天窗 /全景天窗（视配置）
-  内后视镜防眩目：手动防眩目  
    （高配为自动防眩目）
### 音响系统：
-  扬声器数量：6喇叭  
    （部分顶配可选BOSE品牌音响）

> 📌 *注：以上参数以2023款及之后款型为主参考，不同年款或地区版本可能存在差异。*

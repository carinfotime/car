# 尚界H5 空间参数介绍
  
更新时间：2026年07月12日14时42分04秒   

## 整车尺寸参数
| 项目 | 参数 |
| :--- | :--- |
| 车长（mm） | 4,650 |
| 车宽（mm） | 1,838 |
| 车高（mm） | 1,470 |
| 轴距（mm） | 2,700 |
## 乘坐空间
### 前排空间
- **头部空间**：1,015 mm（座椅最低位置）
- **腿部空间**：1,080 mm（座椅调至最后）
- **肩部空间**：1,420 mm
### 后排空间
- **头部空间**：950 mm
- **腿部空间**：890 mm
- **肩部空间**：1,390 mm
- **膝部空间**：180 mm
### 座椅布局
- **座椅数量**：5座
- **座椅材质**：仿皮 / 真皮（高配）
- **后排座椅**：支持 4/6 比例放倒
## 储物空间
### 车内储物格
- **中控台杯架**：2 个（带限位器）
- **前门板储物格**：可容纳 1.5L 水瓶
- **后门板储物格**：可容纳 550ml 水瓶
- **手套箱容积**：12 L
- **中央扶手箱容积**：8 L
- **眼镜盒**：1 个（驾驶位上方）
### 后备箱空间
| 项目 | 参数 |
| :--- | :--- |
| 标准容积（L） | 480 |
| 最大容积（L） | 1,200（后排座椅放倒后） |
| 后备箱开口宽度（mm） | 1,080 |
| 后备箱进深（mm） | 1,020 |
| 后备箱高度（mm） | 480 |
| 备胎位置 | 后备箱地板下方 |
- **电动尾门**：支持（中高配）
- **感应开启**：支持（高配）
## 空间亮点
- 后排中央地台隆起高度约 **50 mm**，对中间乘客影响较小
- 后排座椅坐垫长度 **480 mm**，提供良好大腿支撑
- 后备箱地板平整，放倒后排后可形成接近纯平的空间
- 全车共 **18 处** 储物空间设计

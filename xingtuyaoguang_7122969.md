# 星途瑶光 动力参数介绍
  
更新时间：2026年07月14日12时19分39秒   

## 动力总成概述
星途瑶光搭载 **2.0TGDI 涡轮增压发动机**，匹配 **7速湿式双离合变速箱（7DCT）**，提供前驱与四驱两种驱动形式。
---
## 发动机参数
| 项目 | 参数 |
|------|------|
| 发动机型号 | 2.0T 四缸涡轮增压（缸内直喷） |
| 排量 | 1998 mL |
| 最大马力 | **261 Ps**（192 kW） |
| 最大功率转速 | 5500 rpm |
| 最大扭矩 | **400 N·m** |
| 最大扭矩转速 | 1750 - 4000 rpm |
| 供油方式 | 缸内直喷 |
| 燃油标号 | 92# 及以上无铅汽油 |
| 排放标准 | 国VI b |
---
## 变速箱
- **类型**：7速湿式双离合变速箱（7DCT）
- **特点**：换挡平顺、响应迅速，兼顾燃油经济性与动力输出

## 驾驶模式
提供多种驾驶模式选择，适应不同路况与驾驶需求：
- **经济模式** — 优化燃油经济性
- **标准模式** — 日常驾驶均衡调校

星途瑶光搭载的 **2.0T + 7DCT** 动力组合，拥有 **261马力 / 400N·m** 的强劲输出，在同级别车型中处于领先水平，兼顾了出色的加速性能与日常使用的燃油经济性，配合智能四驱系统，能够满足城市通勤与轻度越野的多场景需求。

# 红旗H5 PHEV 操作指南
  
更新时间：2026年07月18日08时34分49秒   

## 一、车辆启动与熄火
### 1.1 启动车辆
1. **携带钥匙**：确保智能钥匙在车内。
2. **系好安全带**：这是启动的前提条件之一。
3. **踩下制动踏板**：用力踩下刹车。
4. **按下一键启动按钮**：
   - 仪表盘亮起，READY（准备就绪）指示灯点亮，表示车辆已进入可行驶状态。
   - 此时发动机可能不工作，属于正常现象（纯电模式）。
### 1.2 熄火/关闭车辆
1. **停稳车辆**：挂入 P挡（驻车挡）。
2. **按下“一键启动”按钮**：车辆电源关闭。
3. **下车锁门**：触摸门把手感应区或按遥控钥匙锁车键。
> ⚠️ **注意**：PHEV车型熄火后，高压系统可能仍有残余电量，请勿自行拆卸高压部件。

## 二、驾驶模式选择
红旗H5 PHEV提供多种驾驶模式，通过中控台或屏幕上的“DRIVE MODE”旋钮/按键切换。
| 模式名称 | 图标/标识 | 适用场景 | 特点 |
| :--- | :--- | :--- | :--- |
| **纯电优先 (EV)** | EV | 城市通勤、短途行驶 | - 优先使用电池驱动<br>- 电量充足时发动机不介入<br>- 最经济、安静 |
| **混动模式 (HEV)** | HEV / Auto | 长途、综合路况 | - ECU自动协调电机与发动机工作<br>- 兼顾油耗与动力 |
| **运动模式 (SPORT)** | SPORT / S | 超车、激烈驾驶、山路 | - 发动机常启，动力响应最积极<br>-方向盘变重，悬架变硬（若配备）<br>-油耗较高 |
| **经济模式 (ECO)** | ECO |拥堵路段、追求极致续航 |-油门响应变缓<br>-空调功率受限<br>-能量回收增强 |
### ⚡️电量保持功能
-可在中控屏设置“电量保持”百分比（如30%、50%）。开启后发动机会主动充电或直驱，保留电池电量以备后续拥堵路段使用。

## 三、换挡操作
采用电子挡杆（类似飞机推杆）。
- **D挡（前进）**：踩刹车，向后拉挡杆。
- **R挡（倒车）**：踩刹车，向前推挡杆。
- **N挡（空挡）**：短暂停留中间位置。
- **P挡（驻车）**：按下挡杆顶端的“P”按钮。
- **手动模式/Mode切换**：
   - D挡状态下向左拨动挡杆进入手动模式；
   -可通过方向盘换挡拨片临时介入控制档位。
> ❗️紧急情况下长按“P”键可触发电子手刹制动。
---
## 四、能量回收系统
松开油门踏板时，电机反向发电回收能量并产生减速感。
### 调节回收强度
-通过中控屏或拨片调节回收等级：
   - **弱回收 (Level1)** ：滑行顺畅接近燃油车；
   - **标准回收 (Level2)** ：有轻微拖拽感；
   - **强回收 (Level3)** ：单踏板感觉明显，适合城市跟车减少刹车使用。
> 💡建议高速路用弱/标准；市区拥堵用强回收更省电省刹车片。

## 五、充电操作
### 5.1交流慢充（家用桩/公共慢充桩）
1.打开充电口盖板；
2.插入交流充电枪直至锁止声响起；
3.充电过程中仪表盘显示预计充满时间及当前SOC；
4.充满后解锁车辆→拔枪→关盖板。
### 5.2直流快充（仅限支持快充版本）
1.打开直流快充口盖板；
2.插入直流快充枪头；
3.**快充至80%通常约30分钟**；为保护电池建议充至90%即可停止长途前再充满；
### 5.3应急补能
-当电池亏电且无充电条件时：
   混动模式下发动机会自动为电池充电维持基本续航但油耗较高不建议长期亏电行驶否则影响电池寿命及动力体验；

|---|---|---|

|EV MODE|绿色/蓝色|当前为纯电驱动|
|发动机故障灯(MIL)|黄色/红色||
|-发动机系统异常请及时检修|||
|-高压系统警告灯||红色||高压漏电或绝缘故障立即停车联系售后||
|-充电连接指示灯||绿色闪烁||正在充电|

## 八冬季/夏季用车建议
❄️冬季注意事项：
    气温低于零度时纯电续航会缩减约20%~30%属正常物理现象；建议出发前插上充电桩利用电网预热电池包提升放电效率同时开启座椅加热比开暖风更省电；
☀️夏季注意事项：
    避免暴晒后立即快充高温会降低充电速度甚至触发过热保护；尽量选择阴凉处或夜间慢充为主；车内高温时可先远程开窗通风再开启空调制冷减少压缩机负载；

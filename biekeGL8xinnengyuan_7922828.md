# 别克GL8新能源 技术参数介绍
  
更新时间：2026年07月20日15时34分48秒   

## 一、车型概述
别克GL8新能源是上汽通用别克品牌推出的插电式混合动力（PHEV）中大型MPV，在经典GL8基础上融入电气化技术，兼顾商务接待与家庭出行需求。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| 发动机类型 | 1.5T 四缸涡轮增压发动机 |
| 发动机最大功率 | 132 kW（180 Ps） |
| 发动机最大扭矩 | 250 N·m |
| 驱动电机类型 | 永磁同步电机 |
| 电机最大功率 | 160 kW（218 Ps） |
| 电机最大扭矩 | 330 N·m |
| **系统综合功率** | **292 kW（397 Ps）** |
| **系统综合扭矩** | **580 N·m** |
| **0-100 km/h加速时间** | **约7.8秒** |
---
## 三、电池与续航
| 项目 | 参数 |
|------|------|
| 电池类型 | 三元锂电池 /磷酸铁锂电池（视配置） |
| **CLTC纯电续航里程** | **约138 km** |
| CLTC综合续航里程（满油满电） | >1000 km |
| WLTC馈电油耗（百公里） | <6.5 L/100km |
---
## 四、充电性能
- **交流慢充**：支持家用220V/7kW充电桩
- **直流快充**：支持最高约60kW直流快充
- **快充时间（30%-80%）**：约30分钟
- V2L外放电功能：支持对外供电，满足露营等场景需求
---
## 五、车身尺寸与空间
### （一）外观尺寸
- **车长**：5219 mm
- **车宽**：1878 mm
- **车高**：1799 mm
- **轴距**：3088 mm
- **整备质量**：约2100 kg（视配置）
### （二）座椅布局
- **座椅形式**：2+2+3七座布局
- **第二排座椅功能**
    -  航空级独立座椅
    -  电动调节 /加热 /通风 /按摩
    -  电动腿托 +靠背大角度后仰
    -  小桌板 /杯架 /USB-C接口
- **第三排座椅**
    -  三座设计，支持4/6比例放倒
    -  后备箱容积扩展能力强
### （三）后备箱容积
-  标准状态：约520 L（可放置多个登机箱）
-  第三排放倒后：>1600 L

## ⚙️六、底盘与操控系统
### （一）悬架结构
-  前悬架：麦弗逊式独立悬架
-  后悬架：多连杆式独立悬架（部分高配配备CDC可变阻尼减震器）
### （二）转向与制动
-  转向类型：EPS电子助力转向系统
-  制动系统：
    -   前通风盘 +后实心盘式制动器
    -   电子驻车制动 +自动驻车功能
    -   能量回收可调（多级强度选择）
### （三）驱动方式
前置前驱布局，配合混动专用变速箱实现平顺动力输出。

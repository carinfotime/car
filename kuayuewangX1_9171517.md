# 跨越王X1 技术参数介绍
  
更新时间：2026年07月20日14时11分39秒   

## 车型概览
跨越王X1是一款定位于城市物流、短途运输的微卡车型，以其灵活的车身尺寸、出色的承载能力和经济的使用成本，成为城乡配送领域的得力助手。
## 基本参数
| 项目 | 参数 |
| :--- | :--- |
| **车型级别** | 微卡 |
| **驱动形式** | 中置后驱 |
| **轴距 (mm)** | 2700 / 2990 (根据不同货箱长度) |
| **座位数** | 2座 (单排) / 5座 (双排) |
## 动力系统
### 发动机
- **型号**: DAM15KR / DK15C
- **类型**: 直列四缸、自然吸气、多点电喷汽油发动机
- **排量 (L)**: 1.5
- **最大功率 (kW/Ps)**: 85 kW (约116 Ps)
- **最大扭矩 (N·m)**: 150 N·m
- **排放标准**: 国VI b
### 变速箱
- **类型**: DAT18R / MR515 / MW5G15 —  5速手动变速箱
- **换挡方式**:  手动换挡
## 底盘与悬挂系统
| 项目 | 参数 |
| :--- | :--- |
| **前悬挂** |  麦弗逊式独立悬架 |
| **后悬挂** |  钢板弹簧非独立悬架（主副簧结构） |
| **前桥** |  承载式前桥 |
| **后桥** |  加强型后桥 |
## 制动与转向系统
- **制动系统**:  前盘后鼓式液压制动
- **驻车制动**:  机械手刹
- **转向系统**:  电动助力转向 (EPS)
## 货箱尺寸与承载能力
### （以单排长轴距版本为例）
- **货箱长度 (mm)**:  3050 /  3300 /  3600 (根据不同版本)
- **货箱宽度 (mm)**:  1650
- **货箱高度 (mm)**:  360 /  380 （栏板高度）
- **额定载质量 (kg)**:  约1495 -  1995 （具体视公告而定）
### （以双排版本为例）
- **货箱长度 (mm)**:  2550 /  2710
- **额定载质量 (kg)**:  约1495 -  1995 （具体视公告而定）
> *注：具体载重能力请以车辆合格证及当地法规要求为准。*

*注：不同配置（单/双排、长短轴）车身尺寸有所差异。*

## 核心配置亮点（部分高配车型）
*   空调系统:   手动空调（标配）
*   多媒体:   收音机+USB接口，可选装MP5大屏+倒车影像
*   安全配置:   ABS防抱死制动系统+EBD制动力分配（标配）、主驾安全气囊（选装）
*   便利配置:   电动车窗、中控锁、遥控钥匙、大灯高度调节

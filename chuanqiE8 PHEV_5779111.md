# 传祺E8 PHEV 技术参数介绍
  
更新时间：2026年07月16日10时00分15秒   

## 一、车型概览
传祺E8 PHEV是广汽传祺推出的一款插电式混合动力（PHEV）中大型MPV，定位家用与商务兼顾的多用途车型，搭载广汽自研的钜浪混动系统。
---
## 二、动力系统
| 项目 | 参数 |
|------|------|
| **发动机类型** | 2.0L ATK自然吸气发动机（钜浪动力） |
| **发动机最大功率** | 103 kW（140 Ps） |
| **发动机最大扭矩** | 180 N·m |
| **驱动电机类型** | 永磁同步电机 |
| **电机最大功率** | 134 kW（182 Ps） |
| **电机最大扭矩** | 300 N·m |
| **系统综合功率** | ≈175 kW |
| **系统综合扭矩** | ≈380 N·m |
| **变速箱形式** | 2挡DHT混合动力专用变速箱 |
---
## 三、电池与续航
| 项目 | 参数 |
|------|------|
| **电池类型** | 三元锂电池 / 磷酸铁锂电池（视版本而定） |
| **电池容量** | 25.57 kWh（CLTC标准） |
| **纯电续航里程（CLTC）** | ≥150 km（部分版本约116 km） |
| **综合续航里程（CLTC）** | ≥1200 km（满油满电） |
| **WLTC馈电油耗** | ≤6.05 L/100km |
---
## 四、充电性能
- **交流慢充：**支持6.6 kW家用充电桩
- **直流快充：**支持快充接口，30%~80%充电时间约30分钟
- **对外放电：**支持3.3 kW V2L对外放电功能
---
## 五、尺寸与空间
| 项目 | 参数 |
|------|------|
| **车身结构** | MPV，5门7座 / （部分版本可选6座布局）|
| **长 × 宽 × 高 (mm)** | ≈4920 × 1900 × ~1760 mm （具体以官方数据为准）|
| **轴距 (mm)** | ≈2930 mm （部分版本为3070mm左右，视具体配置而定）|
| **整备质量 (kg)** | ≈2000 kg （视配置不同略有差异）|
---
## 六、底盘与悬架
- **驱动方式：**前置前驱
- **前悬架类型：**麦弗逊式独立悬架
- **后悬架类型：**扭力梁式非独立悬架 / （高配版可能为多连杆独立悬架）
- **制动系统：**前通风盘 /  后盘式制动
- **转向系统：**电动助力转向
---
##  七、智能与科技配置
### 🖥️智能座舱
- ADiGO智驾互联生态系统
-  12.3英寸全液晶仪表盘 +  14.6英寸中控触摸屏
-  高通骁龙8155芯片车机处理器
-  语音识别控制、OTA远程升级
-  手机远程控制APP
### 🚗智能驾驶辅助
- L2级辅助驾驶功能
    - ACC自适应巡航
    - LKA车道保持辅助
    - AEB自动紧急制动
    - TJA交通拥堵辅助
    - ICA集成式巡航辅助
    - APA自动泊车辅助（部分版本）
---
## ⚠️八、安全配置
### 🛡️主动安全：
- ESP车身电子稳定系统
- ABS防抱死制动 + EBD制动力分配 + TCS牵引力控制 + HHC坡道起步辅助
- TPMS胎压监测系统（直接式）
- HDW抬头显示预警（选装/高配）
### 🛡️被动安全：
-  主/副驾安全气囊 +  前排侧气囊 +  前后排头部气帘 （共≥6气囊）
- ISOFIX儿童座椅接口（第二排）
- C-NCAP五星安全车身结构设计
---

> -  油箱容积：约50~55 L  
> -  排放标准：国VI b

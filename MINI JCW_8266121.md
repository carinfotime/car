## MINI JCW 动力参数介绍
  
更新时间：2026年07月20日17时51分54秒   

MINI John Cooper Works（JCW）是 MINI 品牌旗下的高性能运动版本，代表着极致的驾驶乐趣与赛道基因。以下是其核心动力系统参数详解。
### 发动机规格
| 项目 | 参数 |
| :--- | :--- |
| **发动机类型** | 2.0升 TwinPower Turbo 直列四缸汽油发动机 |
| **进气形式** | 双涡管单涡轮增压 |
| **排量** | 1998 cc |
| **气缸数** | 4 |
| **每缸气门数** | 4 |
### 性能数据
| 项目 | 参数 |
| :--- | :--- |
| **最大功率** | **170 kW (231 hp)** / 5200–6200 rpm |
| **最大扭矩** | **320 N·m** / 1450–4800 rpm |
| **最高车速** | **250 km/h** (电子限速) |
| **0-100 km/h加速时间** | **6.1秒** (手动挡) / **6.3秒** (自动挡) |
### 传动与驱动系统
-   **变速箱选项：**
    -   6速手动变速箱
    -   8速 Steptronic Sport自动变速箱（带换挡拨片）
-   **驱动方式：**
    -   前置前驱
-   **特色技术：**
    -   Mechanical Front Differential Lock（机械式前桥差速锁），提升弯道牵引力与操控极限。
### 燃油与排放
-   **燃油类型：**
    -   高标号无铅汽油（推荐95号及以上）
-   **综合油耗 (WLTP)：**
    -   约7.1 – 7.5 L/100km
-   **CO₂排放 (WLTP)：**
    -   约162 – 170 g/km
### JCW专属调校亮点
1.  **运动排气系统：**
    配备主动式排气阀门，可根据驾驶模式调节声浪，提供更激昂的听觉体验。
2.  **强化冷却系统：**
    针对激烈驾驶优化了发动机冷却与进气温度控制，确保持续高性能输出。
3.  **底盘调校：**
    采用更硬的弹簧、减震器及防倾杆，降低车身高度，配合 JCW专属运动悬挂或可选自适应悬挂，提供精准的弯道支撑。
4.  **制动系统：**
    前轮配备大尺寸通风刹车盘与红色卡钳（JCW标识），确保强大制动力。
> *注：以上数据基于最新款 MINI JCW（F56世代）官方信息。不同市场或年款车型可能存在细微差异。*

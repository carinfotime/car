# 易至EV3 内饰参数介绍
  
更新时间：2026年07月14日17时34分52秒   

## 整体内饰风格
易至EV3的内饰设计以简约、科技感为主，采用双色拼接设计，营造年轻、时尚的座舱氛围。中控台布局简洁，功能分区清晰。
## 座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 织物/仿皮（高配） |
| 前排座椅调节 | 手动4向/6向调节 |
| 后排座椅 | 整体放倒（支持扩展后备箱空间） |
| 座椅配色 | 双色拼接（黑/白、黑/红等） |
## 中控与多媒体系统
- **中控屏幕**：8英寸触控液晶屏（悬浮式设计）
- **仪表盘**：全液晶仪表盘（尺寸约7英寸）
- **多媒体功能**：蓝牙连接、USB接口、收音机
- **手机互联**：支持CarLife手机映射
- **语音控制**：部分车型支持语音识别控制多媒体/电话
## 方向盘
- **材质**：塑料（部分高配为皮质包裹）
- **功能**：多功能按键（音量控制、蓝牙电话等）
- **调节方式**：手动上下调节
## 空调系统
- **类型**：手动空调
- **后排出风口**：无（部分车型可选装）
- **空气净化**：PM2.5过滤功能（部分配置）
## 储物空间
| 位置 | 说明 |
|------|------|
| 手套箱 | 常规尺寸，带阻尼 |
| 门板储物格 | 前后门均有，可放置水瓶/杂物 |
| 中央通道储物 | USB接口+杯架+小储物格 |
| 后备箱容积 | ≈300L（常规）/ ≈800L（后排放倒） |
## 其他内饰配置
- **换挡机构**：旋钮式电子换挡
- **车窗控制**：四门电动车窗（主驾一键升降）
- **内后视镜**：手动防眩目
- **遮阳板**：主副驾均带化妆镜
- **阅读灯**：前排顶灯
- **USB充电口**：前排1~2个USB接口
## NVH表现
易至EV3作为纯电动车型，电机运行噪音较低，车内静谧性优于同价位燃油车。高速行驶时风噪和胎噪为主要噪音来源。
---
> 📌 *注：以上参数以2023款及之后版本为主，具体配置可能因年款、地区及选装包不同而有所差异。*

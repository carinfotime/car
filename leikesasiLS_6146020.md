# 雷克萨斯LS 操作指南
  
更新时间：2026年07月16日13时37分49秒   

## 一、启动与熄火
### 1.1 智能进入与启动
- **解锁车辆**：携带智能钥匙靠近车辆，手放入门把手感应区域即可解锁。
- **锁车**：触摸门把手上的锁车感应区（通常为凹槽或条纹区域）。
- **启动发动机**：
  1. 确保智能钥匙在车内。
  2. 踩下制动踏板。
  3. 按下“ENGINE START STOP”按钮（一键启动）。
- **熄火**：
  1. 将车辆完全停稳，挂入P挡。
  2. 按下“ENGINE START STOP”按钮。
### 1.2 紧急情况
- **强制熄火**：长按“ENGINE START STOP”按钮3秒以上，或连续快速按3次。

## 二、驾驶操作
### 2.1 换挡操作
- **换挡杆（电子挡把）**：
  - **P (驻车)**：停车时使用，需车辆完全停止。
  - **R (倒车)**：按住换挡杆侧面的解锁按钮，向前推入R挡。
  - **N (空挡)**：短暂停留或洗车时使用。
  - **D (前进)**：向后拉入D挡，正常行驶。
- **手动模式 (M挡)**：
  1. D挡状态下，将换挡杆向左拨动进入M模式。
  2. 向上推（+）升档，向下拉（-）降档。
### 2.2 AVS自适应可变悬架
- LS配备自适应可变悬架系统，可根据驾驶模式和路况自动调节软硬。
- **驾驶模式选择**：通过中控台旋钮选择以下模式：
    - ECO / NORMAL / SPORT S / SPORT S+ / CUSTOM
### 2.3 Mark Levinson音响系统控制
- **音量调节**：方向盘右侧滚轮或中控旋钮。
- **音源切换**：中控屏幕选择收音机、CD、蓝牙音频、USB等。

## 三、灯光与视野
### 3.1 LED大灯
- **自动大灯**：将灯光控制杆旋至AUTO位置。大灯会根据环境亮度自动开启/关闭。
- **远光灯自适应控制系统 (AHS)**：
    - AHS开启后，远光灯会自动规避前方或对向车辆的区域，保持其他区域照明。通过灯光控制杆末端按钮开启/关闭。
### 3.2 HUD抬头显示
- **开关与调节**：
    - HUD开关位于方向盘左下方面板。按下开启/关闭。
    - HUD高度和亮度可通过仪表盘设置菜单调整。
---
## 四、舒适与便利功能
### 4.1 “Ottoman”行政座椅（后排）
- *仅限四座版车型*
- **座椅调节**：
    - ***注意***: *请勿在行驶过程中调节至躺平状态。*
    1. ***放倒***: 
        - *第一步*:向前滑动副驾驶座椅至最前位置并折叠头枕。可通过后排中央扶手上的按键操作；若副驾有人乘坐则无法完成此步骤。
        - *第二步*:按下后排座椅放倒键。靠背倾斜角度可达48度；同时腿托会自动升起以支撑小腿；脚托会从前排座椅背后伸出承托双脚。整个过程由电动马达驱动完成且各部件联动运行以确保舒适性及安全性。 
    2. ***复位***:按下复位键即可让所有部件回归正常乘坐姿态以便下车或恢复行驶状态中的安全坐姿要求！
    3.*按摩功能*:通过中央扶手触摸屏选择不同按摩程序及强度级别享受旅途放松体验！ 
    4.*加热通风*同样在此界面控制温度设定范围覆盖冬夏需求！ 
### 4 .2空调系统 
-**双区/四区独立空调**:驾驶员侧乘客侧以及左右后排均可单独设定目标温度值互不影响！ 
-**nanoe纳米水离子发生器**:开启后能有效净化车内空气并保持适度湿润感避免干燥不适！

## 五 、安全辅助系统(LSS+ ) 
### 5 .1预碰撞安全系统(PCS ) 
-**功能**:通过雷达与摄像头检测前方车辆行人等障碍物发出警报并在必要时自动施加制动减轻碰撞伤害甚至避免事故发生！ 
-**灵敏度调节**:可在仪表盘设置菜单中调整预警时机早晚偏好！ 
### 5 .2车道偏离警示(LDA )与车道保持辅助(LKA ) 
-**LDA**:当车辆无意识偏离车道时会发出蜂鸣声并在仪表盘显示警示图标提醒驾驶员修正方向！ 
-**LKA**:在LDA基础上主动施加微小转向力帮助车辆保持在车道中央行驶减轻长途驾驶疲劳感！ 
### 5 .3自适应巡航控制(ACC )全速域跟车功能 
-**设定速度**:先按下巡航主开关再向下拨动SET/-键即可以当前速度作为巡航目标值开始运行；向上拨动RES/+则恢复上次记忆值继续工作直到再次取消为止！每次短拨增减约1．6km/h长按则连续变化直至松开手指停止调整动作生效新数值显示于仪表盘中央区域供参考确认无误后放手即可维持稳定车速前行无忧虑矣！！  
-**跟车距离调节**:通过方向盘上的车距按键选择远中近三档间距以适应不同交通状况需求确保安全前提下提升通行效率减少频繁加减速带来的顿挫感影响乘坐品质！！  
-**完全停止与再起步**:前车完全停下时本车也会随之刹停静止不动等待下一步指令下达——此时只需轻点油门踏板或者按一下RES/+按键便能重新激活跟随功能继续前进无需繁琐重设参数非常便捷实用！！

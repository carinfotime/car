# 吉利星瑞 内饰参数介绍
  
更新时间：2026年07月19日11时39分40秒   

## 一、整体设计风格
- **设计理念**：采用“时空交叠座舱”设计语言，整体风格简约、科技且富有层次感。
- **配色方案**：
    - 提供 **高级灰/游艇蓝** 双拼色内饰（高配车型）
    - **纯黑** 内饰（中低配车型）
- **材质用料**：
    - 中控台及门板大面积使用 **软性搪塑材质** 包裹
    - 高配车型配备 **Alcantara麂皮** 与 **皮质** 混搭座椅
    - 细节处辅以 **镀铬饰条**、**钢琴烤漆面板** 及 **星河镭雕3D纹理**
## 二、座椅配置
| 项目 | 参数 / 功能 |
| :--- | :--- |
| **座椅材质** | 仿皮（中低配） / Alcantara+皮质混搭（高配） |
| **主驾座椅调节** | 6向电动调节（中高配） / 手动调节（低配） |
| **副驾座椅调节** | 4向电动调节（高配） / 手动调节（中低配） |
| **前排座椅功能** | ✅加热 (高配) <br> ✅通风 (顶配) <br> ✅记忆功能 (主驾，顶配) |
| **后排座椅** | ✅比例放倒 (4/6分割) <br> ✅中央扶手带杯架 <br> ❌不支持角度调节 |
## 三、仪表盘与中控屏
### （一）全液晶仪表盘
- **尺寸与类型**：
    - **12.3英寸**（中高配车型）
    - *注：低配车型为7英寸或非全液晶仪表*
- **显示内容支持**：
    - G值显示
    - Navi导航映射
    - L2级辅助驾驶状态显示
    -胎压监测、油耗等行车信息

# 金旅大海狮 驾驶注意事项
  
更新时间：2026年07月14日08时19分20秒   

## 一、出车前检查
1. **车身与轮胎**
   - 检查轮胎气压及磨损情况，确保胎压符合标准（通常前轮4.5-5.0 bar，后轮5.0-5.5 bar，具体以车门标签为准）。
   - 检查全车灯光（大灯、转向灯、刹车灯、雾灯）是否正常。
   - 确认后视镜调整到位，消除盲区。
2. **油液与冷却**
   - 检查机油液位，应在上下刻度线之间。
   - 检查冷却液、刹车油、助力转向油液位是否正常。
   - 柴油车型注意加注符合标号的柴油（冬季需加负号柴油）。
3. **车内与载货**
   - 调整座椅及方向盘至舒适驾驶姿势。
   - 系好安全带，并提醒所有乘客系好安全带。
   - 货物均匀摆放并固定牢固，避免行驶中移位影响重心。
## 二、启动与起步
1. **预热**：柴油发动机需预热（等待预热指示灯熄灭后再启动），尤其在冬季。
2. **离合操作**：踩下离合器到底再启动，挂挡轻柔。
3. **怠速热车**：启动后怠速运转1-2分钟，待水温表开始上升后再缓慢起步。
4. **松开手刹**：确认手刹完全释放后再挂挡起步。
## 三、行驶中注意事项
1. **车速控制**
   - 金旅大海狮重心较高，转弯时务必减速（建议不超过30km/h），防止侧倾或侧翻。
   - 高速行驶时保持安全车速（建议不超过100km/h），避免急打方向。
2. **跟车距离**
   - 车身较长较重，刹车距离比轿车长。保持足够的安全跟车距离（至少3秒以上反应时间）。
3. **换挡时机**
   - 柴油发动机转速不宜过低或过高。建议在2000-2500转左右换挡，避免拖挡或高转速运行。
4. **坡道驾驶**
   - 上坡提前降挡保持动力；下坡利用发动机制动（降挡），避免长时间踩刹车导致刹车过热失效。
5. **盲区注意**
   - A柱较粗且车身长，变道或转弯时务必转头观察侧后方情况。
   - 倒车时最好有专人指挥或使用倒车影像/雷达辅助。
6. **雨天/湿滑路面**
   - 降低车速，避免急加速、急刹车和急转弯。
   - 开启近光灯及前后雾灯提高辨识度。
## 四、停车与熄火
1. **停车位置**：选择平坦坚实的地面停车；坡道停车必须拉紧手刹并挂入挡位（上坡挂1挡，下坡挂倒挡）。
2. **熄火前**：怠速运转1-2分钟再熄火（尤其长途行驶后），保护涡轮增压器（如有）。
3. **关闭电器**：熄火前关闭空调、大灯等用电设备，防止下次启动亏电。

# 得意A35 内饰参数介绍
  
更新时间：2026年07月12日17时40分24秒   

## 一、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 织物面料 / 可选配仿皮座椅 |
| 座椅布局 | 3+2 五座布局（前3后2） |
| 驾驶座调节 | 手动4向调节（前后+靠背角度） |
| 副驾驶座调节 | 手动4向调节 |
| 后排座椅 | 整体式后排座椅，支持整体折叠 |
| 座椅头枕 | 前后排独立可调头枕 |
| 座椅安全带 | 前排三点式预紧限力安全带 / 后排三点式安全带 |
## 二、仪表与中控
| 项目 | 参数 |
|------|------|
| 仪表盘 | 双圆环机械仪表 + 单色行车电脑显示屏 |
| 中控台布局 | 对称式设计，功能分区清晰 |
| 中控屏幕 | 可选装 7英寸触控屏（支持蓝牙、USB、收音机） |
| 方向盘 | 三幅式多功能方向盘（可选装音响控制按键） |
| 空调系统 | 手动空调，前排独立出风口 |
| 换挡机构 | 5速手动挡杆，中控台下方布局 |
## 三、储物空间
| 项目 | 参数 |
|------|------|
| 手套箱 | 副驾驶侧开放式储物箱 |
| 门板储物格 | 前后门板均有杯架及储物槽 |
| 中控储物 | 中控台下方储物槽 + 杯架 ×2 |
| 顶部储物 | 可选装顶置行李架或储物盒 |
| 后备箱 | 后排折叠后最大进深约 2.5m，容积约 7.8m³ |
## 四、内饰材质与配色
| 项目 | 参数 |
|------|------|
| 内饰主色调 | 深灰色 / 米色（可选） |
| 仪表台材质 | 硬质塑料，表面纹理处理 |
| 门内饰板 | 硬质塑料 + 织物拼接 |
| 地板材质 | PVC 耐磨地板垫（易清洁） |
| 顶棚材质 | 织物顶棚 |
## 五、舒适与便利配置
| 项目 | 参数 |
|------|------|
| 电动车窗 | 前排电动车窗（驾驶员侧一键下降） |
| 后视镜调节 | 手动外后视镜调节 |
| 车内照明 | 前排顶灯 + 后排顶灯 |
| 遮阳板 | 主副驾遮阳板（副驾带化妆镜） |
| 点烟器 / 电源 | 前排 12V 电源接口 ×1 |
| USB 接口 | 可选装前排 USB 充电接口 |
| 扬声器 | 2扬声器音响系统（可选装4扬声器） |
## 六、安全配置（内饰相关）
| 项目 | 参数 |
|------|------|
| 安全气囊 | 主驾驶安全气囊（副驾驶可选装） |
| 儿童座椅接口 | 后排 ISOFIX 儿童座椅固定装置 |
| 车内后视镜 | 手动防眩目内后视镜 |
| 中控锁 | 遥控中控锁 |
## 七、尺寸参数（车内空间）
| 项目 | 参数 |
|------|------|
| 车内长度 | 约 2,580 mm |
| 车内宽度 | 约 1,720 mm |
| 车内高度 | 约 1,380 mm |
| 轴距 | 3,310 mm |
---
> **注**：以上参数基于得意A35标准版车型，实际配置可能因年款、选装包及市场版本不同而有所差异，请以实车为准。

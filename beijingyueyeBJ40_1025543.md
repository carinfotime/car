# 北京越野BJ40 内饰参数介绍

## 整体设计风格
BJ40 内饰采用硬派越野风格，融合现代科技元素，整体布局方正硬朗，强调功能性与耐用性。
## 座椅配置
| 项目 | 参数 |
|------|------|
| **座椅材质** | 仿皮/真皮（高配车型） |
| **前排座椅** | - 主驾驶座椅电动调节（6向/8向）<br>- 副驾驶座椅手动/电动调节<br>- 前排座椅加热（部分配置）<br>- 前排座椅通风（高配选装） |
| **后排座椅** | - 后排座椅靠背角度可调<br>- 后排座椅比例放倒（4/6分割）<br>- 后排中央扶手带杯架 |
| **乘坐人数** | 5座布局 |
## 仪表盘与中控屏
| 项目 | 参数 |
|------|------|
| **仪表盘** | - 12.3英寸全液晶仪表盘<br>- 多种显示模式切换<br>- 越野信息显示（海拔、气压、坡度、指南针等） |
| **中控屏** | - 10.25英寸/12.3英寸悬浮式中控触摸屏<br>- GPS导航系统<br>- CarPlay/CarLife手机互联<br>- OTA远程升级 |
| **车机系统** | - BJ专属智能车机系统<br>- AI语音交互控制 |
## 方向盘
- D型多功能方向盘（真皮包裹）
- 手动上下+前后调节
- 多功能按键：音量控制、蓝牙电话、定速巡航、语音唤醒
-换挡拨片（自动挡车型）
## 空调系统
-自动双区恒温空调
-后排出风口
-PM2.5过滤装置
-空气净化系统（部分配置）

|---|---|
| **内饰配色** |黑色 /黑红双拼 /黑蓝双拼|
| **饰板材质** |仿碳纤维饰板 /金属拉丝饰板 /钢琴烤漆面板|
| **顶棚材质** |织物顶棚 /可拆卸硬顶内衬|
## 储物空间
-前排中央扶手箱（带冷藏功能）
-门板储物格（可容纳水瓶）
-手套箱
-中控台杯架×2
-后排中央杯架
-后备箱储物网兜/锚点
## 灯光与氛围
-车内氛围灯（多色可选，部分配置）
-阅读灯（前后排独立控制）
-迎宾踏板灯
## 其他特色配置
-**可拆卸车顶**:三块式模块化车顶，支持手动拆卸，营造敞篷体验
-**防滚架**:车内可见高强度防滚架结构，增强安全性及硬派视觉感
-**防水处理**:地板采用防水材质，便于冲洗清洁
-**USB接口**:前排2个USB接口 +后排1个USB接口 +12V电源接口

# 凡尔赛C5 X 驾驶注意事项
  
更新时间：2026年07月20日11时43分56秒   

## 一、启动与预热
1. **一键启动**：确保挡位在 **P挡** 或 **N挡**，踩下制动踏板后再按下一键启动按钮。
2. **冷车启动**：冬季或长时间停放后，建议怠速热车 **30秒至1分钟**，待转速回落至正常怠速（约700-800转）后再起步。
3. **起步前检查**：确认仪表盘无异常报警灯（如发动机故障灯、胎压监测灯等）。
## 二、行驶中注意事项
### 1. 变速箱使用
- 凡尔赛C5 X搭载 **爱信8速手自一体变速箱**，行驶中避免频繁在D挡和N挡之间切换。
- 等红灯时建议保持D挡并踩住刹车，或开启 **自动驻车功能**（如有配备），不建议挂N挡滑行。
- 长下坡时可切换至 **手动模式（M挡）**，利用发动机制动，减少刹车系统负担。
### 2. PHC自适应液压稳定技术
- 车辆配备雪铁龙独创的PHC减震器，悬挂偏舒适取向。过弯时请适当降低车速，避免高速急转弯导致车身侧倾较大。
- 经过减速带或坑洼路面时，建议减速通过（时速控制在20-30km/h以内），以保护悬挂系统并提升乘坐舒适性。
### 3. 制动系统
- 凡尔赛C5 X刹车脚感偏线性，初段制动力较轻。紧急制动时需果断深踩刹车踏板。
- 新车前500公里为刹车盘/片磨合期，避免急刹和连续重刹。
### 4. MCS大师级底盘调校
- 底盘离地间隙约193mm（类似SUV通过性），但重心较低。行驶在非铺装路面时仍需注意托底风险。
- 高速行驶稳定性良好，但遇到横风路段请握紧方向盘并适当降速。
## 三、特殊路况驾驶
| 路况 | 建议操作 |
|------|----------|
| **雨天/湿滑路面** | ESP车身稳定系统默认开启；避免猛打方向或急加速；轮胎抓地力下降时提前预判刹车距离 |
| **雪地/结冰路面** | 建议更换雪地胎；起步轻踩油门防止打滑；可使用雪地模式（如有） |
| **山路/连续弯道** | M挡控制转速在2000-3000转以获得更好动力响应；入弯前提前减速 |
| **高速公路** | ACC自适应巡航+车道保持辅助可减轻疲劳；注意跟车距离设定 |
## 四、智能科技使用提醒
1. **L2级辅助驾驶**
   - ACC自适应巡航：设定速度后车辆自动跟车加减速。请注意系统对静止车辆识别能力有限。
   - LKA车道保持辅助：仅作为辅助功能，双手不得离开方向盘。
2. **360°全景影像**
   - R挡自动激活；狭窄路段可手动开启。摄像头视野存在盲区（尤其是车头两侧），不可完全依赖影像倒车或泊车。
3. **HUD抬头显示**
   - HUD投射角度和亮度可在中控屏内调节。佩戴偏光太阳镜可能导致显示变暗甚至消失。
## 五、燃油与保养
1. **燃油标号**
   - 推荐使用 **92号及以上无铅汽油**。长期使用低标号燃油可能导致发动机爆震或积碳增加。
   - GTE版本（插电混动）需定期充电以维持电池健康度。
2. **机油检查**
   - PHC悬挂车型对机油品质要求较高。保养周期为每7500公里或6个月（以先到为准）。
   - C5 X部分批次存在烧机油风险提示，建议每2000公里检查一次机油尺刻度。
3. **轮胎维护**
   - C5 X原厂轮胎规格为205/55 R19（高配）。胎压标准值：前轮2.4bar /后轮2.3bar。
   - PHC悬挂特性导致轮胎磨损较均匀但仍需每10000公里前后交叉换位一次。
## ⚠️六、常见误区与禁忌
1. ❌禁止空挡滑行 —— AT变速箱空挡滑行会损坏变速箱油泵润滑系统。
2. ❌禁止未热车直接大脚油门 —— PHC减震器油液低温下流动性差易造成密封件损伤。
3. ❌禁止涉水超过轮毂中心 —— C5 X进气口高度约70cm涉水深度极限为30cm以内否则有进水熄火风险且保险公司可能拒赔发动机损失险！
4. ✅正确做法 —— P挡熄火后等待10秒再断电让涡轮散热完成再锁车离开！

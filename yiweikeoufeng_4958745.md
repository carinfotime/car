# 依维柯欧风 技术参数介绍
  
更新时间：2026年07月15日10时02分13秒   

## 一、车型概览
依维柯欧风是南京依维柯旗下的一款轻型客车/商旅车型，基于依维柯欧胜平台打造，主打旅居出行、商务接待及客货两用市场。采用非承载式车身结构，具备较强的承载能力和改装潜力。
---

| 项目 | 参数 |
|------|------|
| 发动机型号 | F1C / F1A 系列柴油发动机 |
| 排量 | 2.3T（F1A）/ 3.0T（F1C） |
| 进气形式 | 涡轮增压 + 中冷 |
| 燃料形式 | 柴油 |
| 排放标准 | 国六 b |
| 最大功率 | 2.3T：约 95kW（129Ps）/ 3600rpm |
| | 3.0T：约 125kW（170Ps）/ 3500rpm |
| 最大扭矩 | 2.3T：330N·m / 1500-2800rpm |
| | 3.0T：400N·m / 1500-3000rpm |
| 变速箱 | 6 速手动 / 8 速自动（采埃孚 8AT） |
| 驱动方式 | 前置后驱 |

# 东风风行S50EV 内饰参数介绍
  
更新时间：2026年07月12日14时26分56秒   

## 一、内饰整体风格
风行S50EV内饰采用简约科技风格设计，整体以黑色为主色调，搭配银色镀铬饰条和蓝色新能源元素点缀，营造出年轻、时尚且富有科技感的座舱氛围。
## 二、座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 仿皮材质（部分版本为织物+仿皮拼接） |
| 主驾座椅调节 | 手动6向调节（部分高配为电动6向调节） |
| 副驾座椅调节 | 手动4向调节 |
| 前排座椅功能 | 部分高配车型配备座椅加热功能 |
| 后排座椅 | 支持整体放倒（部分版本为比例放倒） |
| 后排中央扶手 | 部分车型配备带杯架的后排中央扶手 |
## 三、方向盘
| 项目 | 参数 |
|------|------|
| 方向盘材质 | 真皮包裹（部分版本为塑料材质） |
| 方向盘调节 | 手动上下调节 |
| 多功能按键 | 支持多媒体控制、蓝牙电话、定速巡航等 |
## 四、仪表盘
| 项目 | 参数 |
|------|------|
| 仪表盘类型 | 7英寸彩色液晶仪表盘（部分版本为传统机械+液晶组合） |
| 显示内容 | 电量、续航里程、车速、驾驶模式、能量回收状态等 |
## 五、中控多媒体系统
| 项目 | 参数 |
|------|------|
| 中控屏尺寸 | 8英寸（部分高配为9英寸） |
| 屏幕类型 | 触控液晶屏 |
| 系统功能 | 蓝牙/车载电话、多媒体播放、车辆设置 |
| 手机互联 | 支持CarLife手机互联映射 |
| 语音控制 | 支持语音识别控制多媒体、电话等功能 |
| USB接口 | 前排1-2个USB接口 |
## 六、空调系统
| 项目 | 参数 |
|------|------|
| 空调类型 | 自动空调（部分版本为手动空调） |
| 后排出风口 | 部分车型配备 |
| PM2.5过滤 | 部分车型配备空气净化/PM2.5过滤功能 |
## 七、车内空间参数
| 项目 | 参数 |
|------|------|
| 车身尺寸（长×宽×高） | 4700×1790×1525 mm |
| 轴距 | 2700 mm |
| 前排头部空间 | 约950 mm |
| 后排头部空间 | 约910 mm |
| 后排腿部空间 | 约900 mm |
| 后备厢容积 | 约400 L（常规状态） |
## 八、其他内饰配置
- **电动车窗**：四门一键升降（部分版本仅驾驶位支持一键升降）
- **内后视镜**：手动防眩目
- **遮阳板**：主副驾均带化妆镜
- **阅读灯**：前排顶棚阅读灯
- **储物空间**：手套箱、中央扶手箱、门板储物格、杯架等
- **内饰氛围灯**：部分高配车型配备蓝色氛围灯
## 九、内饰颜色选择
- 纯黑内饰
- 黑/米双色内饰（部分版本）
---
> **注**：以上参数因不同年款和配置版本（如 豪华型、旗舰型等）可能存在差异，具体以实车为准。

# 五菱凯捷 保养指南
  
更新时间：2026年07月17日13时03分41秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 首次保养 | **购车后3个月或行驶5000公里**（以先到为准） |
| 常规保养 | 每 **6个月或行驶10000公里**（以先到为准） |
| 大保养 | 每 **2年或行驶40000公里**（以先到为准） |

## 二、各阶段详细保养内容
### （一）首次保养（3个月 / 5000公里）
> **注意：首次保养免费，务必按时前往4S店进行，否则可能影响质保。**
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查变速箱油液位
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查全车灯光及信号系统
- [ ] 检查雨刮器及喷水系统
- [ ] 底盘螺栓紧固检查
- [ ] ECU系统检测与升级（如有）
### （二）常规保养（每6个月 / 10000公里）
**基础项目：**
- [ ] **更换发动机机油**（推荐5W-30全合成机油，约4L）
- [ ] **更换机油滤清器**
- [ ] **更换空气滤清器**（每10000公里或1年）
- [ ] **更换空调滤芯**（每10000公里或1年，视空气质量可缩短）
- [ ] **燃油添加剂**（建议每5000公里添加一次）
**检查项目：**
- [ ] 检查并补充冷却液
- [ ] 检查制动片厚度（前刹车片≤3mm需更换）
- [ ] 检查制动盘磨损情况
- [ ] 检查轮胎磨损与胎压（标准胎压：2.3~2.5 bar）
- [ ] 轮胎换位（建议每10000公里交叉换位）
- [ ] 底盘及悬挂系统检查
- [ ] ECU故障码读取与清除
### （三）大保养（2年 / 40000公里）
除常规保养全部项目外，增加以下内容：
#### 🔧  必须更换项目：
|项目|说明|
|---|---|
|火花塞|原厂铱金火花塞，寿命约4万公里|
|制动液|DOT4级别，建议2年更换一次|
|冷却液|原厂长效冷却液，建议2年或4万公里更换|
|变速箱油|CVT变速箱油，建议4万公里首次更换|
|燃油滤清器|内置式，建议4万公里更换|
#### 🔧  重点检查项目：
-  正时皮带/链条状态（1.5T发动机为链条式，一般免维护但需检查）
-  发电机皮带张紧度与裂纹
-  电瓶电压与健康状况（低于12V建议更换）
-  四轮定位参数检测与调整
-  减震器是否漏油

## 📋  四、日常自检清单（车主每月自查）
> ✅ **每月至少做一次**
1. ☐ **机油尺检查**：冷车状态下拔出擦净再插入，查看刻度是否在上下限之间。
2. ☐ **冷却液壶**：观察副水壶刻度线，不低于MIN线。
3. ☐ **玻璃水**：及时补充。
4. ☐ **胎压监测**：使用胎压表检测四轮气压。
5. ☐ **灯光测试**：大灯、转向灯、刹车灯、倒车灯是否正常。
6. ☐ **仪表盘警示灯**：启动后自检是否全部熄灭。

## ⚠️  五、常见注意事项
1. 🚫 **不要超期使用机油**
   即使里程未到10000km，超过6个月也应更换。
2. 🚫 **不要混用不同品牌/型号的冷却液**
   如需补充请使用同品牌同颜色冷却液。
3. 🚫 **CVT变速箱避免暴力驾驶**
   急加速和频繁N-D切换会加速变速箱磨损。
4. ✅ **定期添加燃油添加剂**
   五菱凯捷1.5T发动机为直喷技术，易积碳，建议每5000km使用一次PEA成分添加剂。
5. ✅ **空调滤芯勤换**
   特别是雾霾严重地区或经常开空调的车主。
6. ✅ **质保期内尽量在4S店保养**
   保留好每次的保养记录和票据。

# 锐捷汽车保养指南
  
更新时间：2026年07月13日10时31分49秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 发动机机油 & 机滤 | 每 5,000 - 10,000 km 或 6 个月 |
| 空气滤清器 | 每 10,000 - 15,000 km |
| 空调滤清器 | 每 10,000 - 15,000 km |
| 燃油滤清器 | 每 20,000 - 30,000 km |
| 火花塞 | 每 30,000 - 40,000 km |
| 制动液 | 每 2 年 或 40,000 km |
| 防冻液 | 每 2 年 或 40,000 km |
| 变速箱油（手动） | 每 40,000 - 60,000 km |
| 变速箱油（自动） | 每 60,000 - 80,000 km |
| 正时皮带/链条 | 每 60,000 - 100,000 km（视具体车型） |
| 刹车片 | 每 30,000 - 50,000 km（视磨损情况） |
| 轮胎 | 每 50,000 - 80,000 km 或 5 年 |
| 蓄电池 | 每 3 - 5 年 |
| 雨刮片 | 每 6 - 12 个月 |
---
## 二、日常检查项目（建议每月或长途前检查）
- **机油液位**：冷车状态下拔出机油尺，液面应在 MIN 与 MAX 之间
- **冷却液液位**：观察副水壶刻度，不足时添加同型号防冻液
- **制动液液位**：不低于 MIN 线，颜色清澈无浑浊
- **轮胎气压**：标准值参考驾驶位门框标签（通常 2.2 - 2.5 bar）
- **轮胎外观**：检查有无鼓包、裂纹、异常磨损
- **灯光系统**：大灯、转向灯、刹车灯、雾灯是否正常
- **雨刮及喷水**：刮拭是否清晰，玻璃水是否充足
---
## 三、机油与机滤更换
### 机油选择
| 环境温度 | 推荐标号 |
|---------|---------|
| 一般地区 | 5W-30 或 5W-40 |
| 寒冷地区（-30℃以下） | 0W-30 或 0W-40 |
| 高温/高负荷 | 10W-40 或 10W-50 |
> 建议使用 API SN 或 SP 级别及以上全合成机油
### 更换步骤
1. 热车后熄火，举升车辆
2. 拧下放油螺栓，放尽旧机油
3. 更换新的机油滤清器（密封圈涂抹少量新机油）
4. 安装放油螺栓（按规定扭矩拧紧）
5. 加注新机油至标准液位
6. 启动发动机运转 1-2 分钟，检查有无泄漏
7. 熄火 5 分钟后再次检查机油液位
---
## 四、空气滤清器更换
1. 打开发动机舱，找到空气滤清器盒（通常为黑色方形塑料盒）
2. 松开卡扣或螺丝，打开盒盖
3. 取出旧滤芯
4. 清洁盒内灰尘
5. 装入新滤芯（注意方向）
6. 盖好盒盖，卡紧卡扣
---
## 五、空调滤清器更换
1. 位置通常在副驾驶手套箱后方或发动机舱副驾侧
2. 拆下手套箱（部分车型需拆卸阻尼器）
3. 取出旧滤芯（注意箭头方向）
4. 装入新滤芯（箭头指向车内或出风方向）
5. 装回手套箱
---
## 六、火花塞更换
### 注意事项
- 务必使用原厂或同规格火花塞
- 拆卸前清理火花塞孔周围灰尘
- 安装时使用扭矩扳手，推荐扭矩 25 - 30 N·m
- 电极间隙需符合车型要求（通常 0.8 - 1.1 mm）
### 更换步骤
1. 断开点火线圈插头
2. 拧出点火线圈或高压线
3. 用火花塞套筒拧出旧火花塞
4. 对比新旧火花塞长度一致
5. 手拧新火花塞至拧不动，再用扭矩扳手拧紧
6. 装回点火线圈，接好插头
---
## 七、制动系统保养
### 刹车片检查
- 厚度低于 3 mm 时必须更换
- 注意听有无金属摩擦异响
- 左右轮刹车片磨损应均匀
### 制动液更换
1. 使用 DOT3 或 DOT4 制动液（按车辆要求）
2. 建议使用专用换油机
3. 手工换油需从四个轮子分泵处依次排气
4. 注意制动液有强腐蚀性，避免接触漆面
---
## 八、冷却系统保养
### 防冻液更换
1. 冷车状态下操作
2. 打开散热器或副水壶盖
3. 拧开放水螺栓，放尽旧液
4. 用清水冲洗冷却系统
5. 加注新防冻液至标准液位
6. 启动发动机至风扇运转，检查液位并补充
> 严禁不同类型防冻液混用，建议使用原厂或同品牌乙二醇型防冻液
---
## 九、变速箱油更换
### 自动变速箱
- 建议使用循环机更换（更彻底）
- 需使用专用 ATF 油（不可混用）
- 更换后需进行油位检查（通常在怠速、P 档状态下检查）
### 手动变速箱
- 重力放油即可
- 加注至加油口溢出为止
- 注意 GL-4 与 GL-5 级别区分
---
## 十、轮胎保养
- **胎压检查**：每月至少一次，包括备胎
- **轮胎换位**：每 10,000 km 进行前后交叉换位
- **动平衡**：更换轮胎或方向盘抖动时需做
- **四轮定位**：车辆跑偏、轮胎偏磨时需做
- **轮胎年限**：生产超过 6 年建议更换，无论花纹深浅
---
## 十一、蓄电池保养
- 定期检查电极有无氧化（白色或蓝色粉末）
- 电极氧化可用热水冲洗后涂抹防氧化脂
- 长期停放超过 2 周建议断开负极
- 冬季注意蓄电池性能下降，必要时提前更换
---
## 十二、正时皮带/链条
### 正时皮带
- 严格按保养手册周期更换
- 断裂会导致发动机严重损坏
- 更换时建议同时更换张紧轮、惰轮、水泵
### 正时链条
- 通常免维护
- 如出现异响（链条哗啦声），需及时检查
---
## 十三、空调系统保养
- 每年夏季前检查制冷效果
- 每 2 年清洗蒸发箱（防止异味、细菌滋生）
- 制冷剂不足需专业设备检漏后补充
---
## 十四、其他注意事项
| 项目 | 建议 |
|------|------|
| 燃油添加剂 | 每 5,000 km 使用一次（清洁积碳） |
| 发动机清洗 | 每 20,000 - 30,000 km 清洗节气门 |
| 底盘检查 | 每年检查底盘有无锈蚀、松动 |
| 排气管 | 检查有无锈穿、异响 |
| 车门铰链 | 每半年润滑一次 |
| 天窗导轨 | 每半年清洁并润滑 |

## 十六、温馨提示
1. **优先参考随车保养手册**，不同年款、动力配置可能有所差异
2. **使用正规渠道配件**，劣质配件会严重影响车辆性能与安全
3. **保留保养记录**，对二手车出售有重要价值
4. **发现异常及时检修**，小问题拖成大故障得不偿失
5. **选择有资质的维修店**，专业设备与经验不可替代

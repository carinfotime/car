# 飞碟U5 内饰参数介绍
  
更新时间：2026年07月20日16时05分22秒   

## 整体设计风格
飞碟U5内饰采用简约科技风格，整体布局以驾驶者为中心，强调实用性与舒适性的结合。
## 座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 高级织物 / 仿皮（高配） |
| 驾驶座调节 | 手动6向调节 |
| 副驾驶座调节 | 手动4向调节 |
| 后排座椅 | 可整体折叠放倒 |
| 座椅功能 | 前排头枕高度可调 |
## 中控台与仪表盘
| 项目 | 参数 |
|------|------|
| 中控屏幕 | 8英寸触控液晶屏（高配） |
| 仪表盘 | 传统指针式仪表 + 行车电脑显示屏 |
| 方向盘 | 三幅式多功能方向盘 |
| 方向盘调节 | 手动上下调节 |
## 智能互联系统
- 蓝牙车载电话
- USB/AUX外接音源接口
- 2扬声器音响系统
- 手动空调
## 储物空间
- 前排中央扶手箱
- 门板储物格（前后门）
- 手套箱
- 前排杯架 × 2
- 后排杯架
## 内饰颜色选项
- 深色内饰（黑色为主）
- 浅色内饰（米色为主）
## 其他配置
| 项目 | 参数 |
|------|------|
| 电动车窗 | 四门一键升降（仅驾驶座） |
| 内后视镜 | 手动防眩目 |
| 遮阳板 | 驾驶座带化妆镜 |
| 阅读灯 | 前排顶灯 |
| 行李厢容积 | 约 400L（后排座椅放倒后可扩展） |
---
> **注：** 以上参数基于飞碟U5常见配置版本，具体配置可能因年款及销售地区不同而有所差异，请以实车为准。

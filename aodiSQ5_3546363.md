|------|------|
| **发动机类型** | 3.0 TFSI V6 涡轮增压发动机 |
| **排量** | 2,995 cc |
| **进气形式** | 涡轮增压 + 电动增压（EPC） |
| **燃料类型** | 汽油 |

| 项目 | 参数 |

| **中央差速器** | 自锁式机械中央差速器 |

|------|------|

## 发动机技术亮点
- **双增压系统**：涡轮增压 + 电动增压器（EPC），有效消除涡轮迟滞，提供更线性的动力输出
- **B循环燃烧技术**：优化燃油效率，兼顾性能与经济性
- **轻量化设计**：采用铝制缸体，减轻发动机重量
- **48V 轻混系统**：支持滑行模式、启停功能及能量回收，提升燃油经济性
---
**总结：** 奥迪SQ5搭载的3.0 TFSI V6发动机配合quattro四驱系统和8速tiptronic变速箱，在保证日常驾驶舒适性的同时，提供了强劲的运动性能表现。

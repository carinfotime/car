# 小鹏 G6 技术参数介绍
  
更新时间：2026年07月17日10时05分20秒   

## 一、车型概览
小鹏 G6 是一款纯电动中型轿跑 SUV，基于 SEPA2.0「扶摇」全域智能进化架构打造。该车定位为“超智驾轿跑 SUV”，主打智能驾驶、超充能力和宽敞空间。
## 二、动力与性能
| 项目 | 参数 |
|------|------|
| **驱动方式** | 后置后驱 / 双电机四驱 |
| **电机类型** | 永磁同步电机 |
| **最大功率（单电机版）** | 218 kW (296 ps) |
| **最大功率（四驱版）** | 前：140 kW + 后：218 kW，综合功率：358 kW (487 ps) |
| **最大扭矩（单电机版）** | 440 N·m |
| **最大扭矩（四驱版）** | 660 N·m |
| **0-100 km/h加速时间（四驱版）** | ≤3.9秒 |
| **最高车速** | ≥200 km/h |
## 三、电池与续航
| 项目 | CLTC标准续航版本 |
|------|----------------|
| **电池类型** | 磷酸铁锂 / 三元锂 |

| **CLTC纯电续航里程** | — **580 km** (66 kWh, R18轮毂)<br>— **755 km** (87.5 kWh, R18轮毂)<br>— **700 km+** (87.5 kWh, R20轮毂)<br>— **660 km+** (四驱版) |
| **快充技术** | —支持800V高压快充<br>—充电10分钟，续航增加约300km<br>—最快充电功率可达280kW |

## 五、底盘与悬架

-  后悬架：五连杆式独立悬架
-  转向系统：DP-EPS（双小齿轮电动助力转向）
-  制动系统：前通风盘 /  后实心盘
-  轮胎规格：
    -   235/60 R18
    -   235/55 R19
    -   255/45 R20

## 🧠七、智能座舱
-   车机芯片：高通骁龙 SA8295P （目前主流旗舰级）
-   操作系统：Xmart OS （基于 Android Automotive）
-   屏幕配置：
    -   10.25英寸全液晶仪表盘 +14.96英寸悬浮式中控屏（分辨率达2400×1200）
-   语音助手：“小P”全场景语音，支持连续对话和可见即可说。
-   音响系统：标配8扬声器；可选配18扬声器Xopera音响系统。
-    其他特色：
     -     全景天幕（带Low-E涂层隔热玻璃）
     -     无风感智能空调系统
## 🚗八、智能驾驶辅助系统 —— XNGP
小鹏 G6全系标配 XNGP智能辅助驾驶系统硬件。
### 🔹硬件配置：
-     激光雷达×2 （部分高配车型）
-     毫米波雷达×5 （含前向角雷达）
-     超声波传感器×12 
-     环视摄像头×4 
-     高感知摄像头×7 
--    车内摄像头×1 
--    高精定位模块(GNSS+IMU)
### 🔹核心功能：
--    高速NGP （导航辅助驾驶）
--    城市NGP （城市道路导航辅助驾驶，需OTA升级及开通区域支持）
--    自动泊车(VPA)
--    记忆泊车(跨楼层)
--    远程召唤(部分场景)
--    主动安全功能(AEB/LKA/LDW等)
---
*以上数据基于官方发布信息整理，具体配置可能因车型版本及选装包不同而有所差异。*

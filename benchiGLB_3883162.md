# 梅赛德斯-奔驰 GLB 内饰参数介绍
  
更新时间：2026年07月18日13时05分54秒   

## 1. 整体设计风格
- **家族化设计**：采用梅赛德斯-奔驰最新的家族式内饰设计语言，兼具豪华感与科技感。
- **硬朗线条**：中控台和门板采用硬朗的直线条设计，呼应了GLB方正的外观造型，营造出粗犷而精致的视觉效果。
- **双屏联动**：标志性的双10.25英寸高清显示屏（仪表盘+中控屏）融为一体，构成宽屏驾驶舱。
## 2. 座椅与空间布局
- **座椅材质**：
  - 标准配置：ARTICO皮革/DINAMICA微纤维或皮革座椅。
  - 可选配置：真皮座椅（需选装）。
- **座椅功能**：
  - 前排座椅：电动调节、记忆功能、加热功能（部分为选装）。
  - 后排座椅：支持前后滑动（约140mm），靠背角度可调。
- **空间布局**：
  - **5座版**：标准5座布局，后备箱容积为560L - 1755L（后排座椅放倒）。
  - **7座版（可选）**：第三排两个独立座椅，适合儿童或短途成人乘坐。第三排配备独立杯架和USB-C充电接口。
## 3. 仪表盘与中控屏幕
| **项目** | **参数/描述** |
| :--- | :--- |
| **仪表盘尺寸** | 10.25英寸全液晶仪表盘 |
| **中控屏尺寸** | 10.25英寸触控液晶屏 |
| **分辨率** | HD高清 (1280 x 720) |
| **操作系统** | MBUX (梅赛德斯-奔驰用户体验系统) |
| **交互方式** | <br>- 触摸屏<br>- “你好，奔驰”语音助手<br>- 中央扶手触控板<br>-方向盘多功能按键 |
## 4. MBUX智能人机交互系统
- **语音控制**：“你好，奔驰”自然语音识别，可控制导航、电话、空调、车窗等功能。
- **导航系统**：
  - GPS导航 +实时路况信息。
  - AR实景穿越导航（部分车型选装）：通过摄像头画面叠加虚拟箭头指示。
- **手机互联/映射**：
    - Apple CarPlay (无线)
    - Android Auto (无线)
    - Baidu CarLife (部分市场)
- **OTA升级**:支持在线地图及系统软件远程更新。
##  5.  空调与出风口
-   “飞机涡轮”造型出风口：经典的圆形镀铬出风口，带有氛围灯照明。
    -  前排中央出风口 x2
    -  前排两侧出风口 x2
    -  后排中央出风口 x2
    -  第三排出风口 x2 (7座版)
-  自动空调：
    - THERMATIC双区自动空调（标配）
    - THERMOTRONIC多区智能空调（高配/选装）：可独立控制左/右/后三区温度。
##  6.  氛围灯系统
-  64色环境氛围照明系统：
    -  分布于中控台、门板饰条、杯架及出风口处。
    -  支持多种预设主题模式（如“热带雨林”、“日落”等）。
    -  亮度及颜色可单独调节。
##  7.  音响系统
| **类型** | **扬声器数量** | **功率(W)** |
| :--- | :---: | :---: |
| *标准音响* | *6个* (或7个) | *100W* |
| *Burmester®环绕立体声音响(选装)* | *12个* (含低音炮) | *590W* |
> Burmester®音响提供VNC噪音补偿功能和多种音效预设。
##  8.  储物空间与便利配置
### 📦储物格分布:
1. ***前门板***:可容纳1L水瓶 +杂物槽。
2. ***后门板***:可容纳500ml水瓶 +杂物槽。
3. ***中央扶手箱***:深度适中,内置USB-C接口(2个)。
4. ***手套箱***:带照明,可存放A4文件或小物件。
5. ***杯架***:
    前排中央杯架x2(带限位器)。
    后排中央扶手杯架x2(5座版)。
    第三排两侧杯架x2(7座版)。
6.**其他**:方向盘左侧储物格,中控台下方的开放式储物托盘(带无线充电位)。
### 🔌充电接口:
-**USB-C接口**:全车共4~6个(前排2个+后排2个+第三排1~2个)。
-**无线充电**:前排中控台下方可选装Qi无线充电板(15W)。
## 🎛️9 .方向盘与操控按键
-**Nappa皮饰多功能方向盘**(标配):
-**换挡拨片**(标配):手动控制变速箱档位切换 。
-**操控按键**:左侧控制仪表盘菜单 ,右侧控制多媒体及语音 ,下方定速巡航拨杆 。

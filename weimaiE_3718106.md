# 威迈E 内饰参数介绍
  
更新时间：2026年07月12日11时28分04秒   

## 一、整体设计风格
- **设计理念**：极简科技风，融合环抱式座舱设计。
- **配色方案**：提供深色（曜石黑）、浅色（云母灰/象牙白）两种内饰主题。
## 二、座椅系统
| 项目 | 参数 / 配置 |
| :--- | :--- |
| **座椅材质** | Nappa真皮（高配）/ 仿皮+织物混搭（中低配） |
| **前排座椅功能** | - 主驾：12向电动调节（含4向腰托）<br>- 副驾：10向电动调节<br>- 加热、通风、记忆功能（主驾）<br>- 老板键（副驾侧） |
| **后排座椅** | - 材质与前排一致<br>- 靠背角度可调（支持4/6比例放倒）<br>- 中央扶手带杯架及储物盒<br>- *高配可选*：后排座椅加热 |
| **头枕** | 前后排均为可调式航空头枕，侧翼支撑可调 |
## 三、仪表台与中控
- **仪表盘**：12.3英寸全液晶仪表盘，分辨率1920×720。
- **中控屏**：15.6英寸悬浮式中控触摸屏，分辨率2560×1600。
- **HUD抬头显示**：AR-HUD增强现实抬头显示，投射面积达70英寸。
- **方向盘**：
    - D型平底多功能方向盘
    - Nappa真皮包裹
    - 带加热功能
    - 触控式按键 + 物理滚轮
## 四、智能座舱系统
- **车机芯片**：高通骁龙8295芯片。
- **操作系统**：自研车载OS V2.0。
- **语音控制**：四音区语音识别，支持连续对话与可见即可说。
- **手机互联**：无线Apple CarPlay / Android Auto。
## 五、氛围与灯光
- **氛围灯**：
    - RGB多色氛围灯带
    - *高配*：256色渐变氛围灯 + “呼吸”模式联动驾驶模式
    - *低配*：64色固定分区氛围灯
- **阅读灯**：前后排LED触控阅读灯。
## 六、空调系统
| 项目 | 配置 |
| :--- | :--- |
| **类型** | AQS空气质量管理系统 + PM2.5过滤 + CN95滤芯 |
| **分区控制** |双区独立自动空调（前排左右独立）|
| *高配升级* |负离子发生器 + UVC深紫外线杀菌 + AQS空气质量监测 |
## 七、音响系统
- *标准版* ：8扬声器立体声音响系统。
- *高配版* ：14扬声器品牌音响（支持7.1声道环绕声）。
- *顶配版* ：23扬声器品牌音响 + Dirac HD音效算法。
## 八、储物空间与便利性
|位置|配置|
|---|---|
|前排中央通道|-手机无线充电板（50W风冷快充）<br>-双杯架<br>-对开式扶手箱|
|门板储物格|-前后门板均可放置1L水瓶+杂物|
|手套箱|-带阻尼开启，内置USB-C接口|
|后备箱|-标准容积500L<br>-后排放倒后最大容积1200L<br>-电动尾门+脚踢感应开启|
## 九、其他细节配置
-**遮阳板**:主副驾均带化妆镜+LED补光灯。  
-**眼镜盒**:驾驶位头顶眼镜盒。  
-**内后视镜**:自动防眩目内后视镜。  
-**车窗**:全车一键升降+防夹功能。  
-**天窗/天幕**:全景天幕或可开启全景天窗(视具体版本而定)。

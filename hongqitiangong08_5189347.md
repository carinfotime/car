# 红旗天工08 技术参数介绍
  
更新时间：2026年07月11日05时54分01秒   

## 一、车型概览
红旗天工08 是红旗品牌基于全新 **“天工”纯电平台** 打造的中大型纯电SUV，定位高端智能电动市场，融合了红旗最新的设计语言与智能科技。
---
## 二、动力系统
| 参数项 | 参数值 |
|--------|--------|
| 驱动形式 | 后驱 / 四驱 |
| 电机类型 | 永磁同步电机 |
| 后驱版最大功率 | 约 250 kW |
| 四驱版最大功率 | 约 450 kW（前+后双电机） |
| 后驱版最大扭矩 | 约 450 N·m |
| 四驱版最大扭矩 | 约 750 N·m |
| 0-100 km/h 加速时间（四驱） | ≤ 4.5 秒 |
| 最高车速 | ≥ 200 km/h |

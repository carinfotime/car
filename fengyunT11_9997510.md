# 风云T11 空间参数介绍
  
更新时间：2026年07月16日16时25分39秒   

## 车身尺寸与轴距
| 项目 | 参数 |
| :--- | :--- |
| **车长 (mm)** | 5150 |
| **车宽 (mm)** | 1995 |
| **车高 (mm)** | 1800 |
| **轴距 (mm)** | 3100 |
> **解读：** 超过5.1米的车长与3.1米的超长轴距，为风云T11提供了宽敞的纵向乘坐空间和稳定的行驶姿态。接近2米的车宽保证了车内横向空间的充裕。
## 座椅布局
- **布局形式：** 2+2+2（六座布局）
- **座椅材质：** Nappa真皮（或同等级高档皮质）
- **座椅功能：**
    - **第一排：** 电动调节、加热、通风、按摩
    - **第二排：** **独立航空座椅**，支持电动调节、加热、通风、按摩，配备腿托及扶手
    - **第三排：** 支持电动折叠/展开，具备独立空调出风口及杯架
## 内部空间数据
| 项目 | 数据 / 描述 |
| :--- | :--- |
| **第一排头部空间 (mm)** | ≥1020 |
| **第二排腿部空间 (mm)** | ≥900（可调范围大） |
| **第二排头部空间 (mm)** | ≥990 |
| **第三排腿部空间 (mm)** | ≥700（可短途乘坐） |
| **第三排头部空间 (mm)** | ≥920 |
> *注：以上数据为参考值，实际体验因坐姿调整略有差异。*
## 储物能力
### 后备箱容积
- **三排座椅全部使用状态：** ≥350 L（可容纳2-3个登机箱）
- **第三排座椅放倒状态：** ≥800 L
- **第二、三排座椅全部放倒状态：** ≥2000 L（形成近乎纯平的地台）
### 车内储物格
-  中控台下方镂空储物区
-  前后门板储物槽（可容纳1.5L水瓶）
-  中央扶手箱（深度可观，支持制冷/制热）
-  副仪表台杯架（4个）
-  第二排中央通道下方储物抽屉
-  第三排两侧杯架及储物槽
## NVH与舒适性配置对空间的加持
虽然NVH不属于直接的空间参数，但优秀的静谧性能够显著提升乘坐时的“心理空间感”：
-  全车双层隔音玻璃
-  多处空腔隔断与吸音材料填充
-  主动降噪系统（ANC）
## “大平层”概念亮点
风云T11在宣传中强调“移动的大平层”，其核心在于：
1.  3100mm轴距带来的纵向贯通感。
2.  第二排独立座椅的零重力模式，可实现近乎平躺的姿态。
3.  全景天幕或天窗带来的通透感，减少压抑。

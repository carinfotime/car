# 宝马5系 内饰参数介绍
  
更新时间：2026年07月11日06时24分26秒   

## 1. 座椅
-   **材质：**
    -   标配：Sensatec 合成皮
    -   选装：Dakota 真皮、Nappa 真皮
-   **功能：**
    -   前排座椅电动调节（带记忆功能）
    -   前排座椅加热（标配/选装）
    -   前排座椅通风（选装）
    -   后排座椅加热（选装）
    -   后排座椅电动调节（高配/选装）
-   **设计：**
    -   运动型座椅（M运动套装）
    -   舒适型座椅（豪华套装）
## 2. 方向盘
-   **类型：**
    -   M运动型多功能真皮方向盘（带换挡拨片）
-   **调节：**
    -   电动上下+前后调节
-   **功能：**
    -   多功能按键（音量、电话、巡航控制等）
## 3. 仪表盘与显示
-   **仪表盘：**
    -   BMW Live Cockpit Professional (12.3英寸全液晶数字仪表盘)
-   **中控屏：**
    -   BMW iDrive 7.0 / iDrive 8.5 (12.3英寸或14.9英寸触控显示屏)
-   **抬头显示 (HUD)：**
    -   BMW全彩平视显示系统（选装/高配标配）
## 4. 中控台与操作
-   **操作系统：** BMW iDrive (支持旋钮、触控、语音控制)
-   **空调系统：**
    -   双区/四区自动空调
        -   （带PM2.5过滤功能）
        -   （带负离子发生器，部分配置）
-   **杯架与储物格：** 前排2个杯架，中央扶手箱，门板储物格
## 5. 音响系统
-   **标配音响：** HiFi音响系统 (205W / 10扬声器)
-   **选装音响1：** Harman/Kardon高端音响系统 (464W / 16扬声器)
-   **选装音响2 (顶配)：** Bowers & Wilkins钻石环绕音响系统 (1400W / 16扬声器)
## 6. 氛围灯

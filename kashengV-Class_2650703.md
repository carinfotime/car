# 卡升V-Class 技术参数介绍
  
更新时间：2026年07月19日12时16分34秒   

## 车型概览
卡升V-Class是基于梅赛德斯-奔驰V-Class平台打造的一款高端商务MPV。它在保留奔驰V级优秀基因的同时，通过内饰升级、座椅布局和智能配置的优化，旨在满足商务接待、高端出行及家庭旅居等多种场景需求。
## 动力系统
| 项目 | 参数 |
| :--- | :--- |
| **发动机类型** | 2.0T 涡轮增压汽油发动机 |
| **发动机型号** | M274 (或同系列) |
| **最大功率** | 155 kW (211 Ps) / 5500 rpm |
| **最大扭矩** | 350 N·m / 1250-4000 rpm |
| **排放标准** | 国 VI |
| **变速箱** | 9G-TRONIC 9速手自一体变速箱 |
| **驱动方式** | 前置后驱 (FR) |
| **最高车速** | 约 185 km/h |
| **0-100km/h 加速** | 约 9.1 秒 |

## 外部配置
- **前脸设计**：双横幅镀铬进气格栅，搭配全LED智能照明系统（自适应远光灯辅助）。
- **大灯**：全LED大灯，带日间行车灯及大灯清洗功能。
- **外后视镜**：电动折叠、加热、自动防眩目及记忆功能。
- **侧滑门**：双侧电动滑门，带防夹及遥控开启功能。
- **轮毂**：18英寸或19英寸铝合金轮毂（视配置而定）。
- **尾门**：电动尾门，带位置记忆功能。
## 内部配置
### 座椅布局
- 标准布局：2+2+3 七座
- 高配布局：2+2+2 六座（航空座椅）
### 前排
- 主/副驾驶电动调节，带腰部支撑及加热/通风功能。
- 多功能Nappa真皮方向盘，带换挡拨片。
- 8.4英寸悬浮式中央显示屏，集成COMAND系统。
### 后排（核心亮点）
- **第二排**：独立航空座椅（电动调节、通风、加热、按摩、腿托、旋转功能）。
- **第三排**：电动连排座椅，可放平形成休息空间（视配置）。
- **内饰材质**：Nappa真皮包覆、Alcantara顶棚、高光岑木/碳纤维饰板。
- **氛围灯**：多色环境氛围照明系统（64色可选）。
- **隔音**：全车多层隔音玻璃及底盘隔音处理。
- **地板**：游艇级柚木地板或高光大理石纹地板（视配置）。
## 舒适与科技配置
| 类别 | 配置内容 |
| :--- | :--- |
| **空调系统** | 自动双区/三区独立空调，带后排出风口及空气净化功能 |
| **娱乐系统** | 后排15.6英寸高清吸顶娱乐屏，支持手机互联及投屏 |
| **音响系统** | 柏林之声 (Burmester) 环绕立体声音响系统 |
| **智能互联** | Mercedes me 互联，支持远程控制、导航、紧急呼叫 |
| **辅助驾驶** | 自适应巡航、主动式车道保持、盲点辅助、交通标志识别 |
| **泊车辅助** | 360°全景影像、前后驻车雷达、自动泊车系统 |
## 安全配置
- **主动安全**：主动制动辅助、注意力辅助、侧风稳定控制、胎压监测。
- **被动安全**：全车8安全气囊（正/侧/帘）、ISOFIX儿童座椅接口、高强度钢车身结构。
- **防盗系统**：发动机电子防盗、防盗报警系统。

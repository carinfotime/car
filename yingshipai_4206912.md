# 英仕派（INSPIRE）保养指南
  
更新时间：2026年07月17日09时19分20秒   

## 一、保养周期总览
| 保养项目 | 建议周期 |
|---------|---------|
| 机油 & 机滤 | 每5,000公里或6个月 |
| 空气滤芯 | 每10,000公里或12个月 |
| 空调滤芯 | 每10,000公里或12个月 |
| 燃油滤清器 | 每40,000公里 |
| 火花塞 | 每60,000公里 |
| 变速箱油（CVT） | 每40,000公里 |
| 刹车油 | 每40,000公里或2年 |
| 防冻液/冷却液 | 每60,000公里或3年 |
| 刹车片 | 视磨损情况，约30,000-50,000公里 |
| 轮胎换位/动平衡 | 每10,000公里 |
| 蓄电池检查 | 每次保养时检查 |
> **注意**：以上为常规建议，具体请以车辆用户手册为准。若经常在恶劣路况、多尘环境或短途行驶，建议适当缩短保养间隔。

## 二、各阶段详细保养内容
### A.  小保养（5,000 km /  6个月）
- [ ]  更换机油及机油滤清器
- [ ]  检查轮胎气压及磨损情况
- [ ]  检查灯光、喇叭、雨刮器
- [ ]  检查制动液液位
- [ ]  检查冷却液液位
- [ ]  检查玻璃清洗液液位
- [ ]  检查底盘有无漏油、异响
### B.  中保养（10,000 km /  12个月）
包含小保养所有项目，额外增加：
- [ ]  更换空气滤芯
- [ ]  更换空调滤芯
- [ ]  轮胎换位 &amp;动平衡
- [ ]  检查刹车片厚度
- [ ]  检查蓄电池状态及接线柱清洁度
- [ ]  检查传动皮带张紧度及老化情况
- [ ]  润滑车门铰链、天窗导轨
### C.  大保养（40,000 km /  2年）
包含中保养所有项目，额外增加：
- [ ] **更换变速箱油**（CVT专用油）
- [ ] **更换刹车油**
- [ ] **更换燃油滤清器**
- [ ] **清洗节气门**（视积碳情况）
- [ ] **清洗喷油嘴**（视积碳情况）
- [ ] **四轮定位检测**（如有跑偏则调整）
- [ ] **深度检查底盘悬挂系统**
### D.  超大型保养（60,000 km /  3年）
包含大保养所有项目，额外增加：
- [x] **更换火花塞**（铱金火花塞）
- [x] **更换防冻液/冷却液**
- [x] **清洗发动机积碳**（进气道、燃烧室）
- [x] **更换正时皮带/链条张紧器**（如为皮带机型）
> *注：英仕派多数为链条正时，通常免维护，但需检查张紧器*

## 四、日常自检清单（每月一次）
1. ✅**胎压检测**
    -   冷车标准胎压：前轮2.3 bar /后轮2.1 bar （参考值，以油箱盖标签为准）
2. ✅**机油尺查看**
    -   熄火后等待5分钟 →拔出擦净 →再次插入 →取出观察刻度在上下限之间
3. ✅**灯光自检**
    -   大灯近光/远光、转向灯、刹车灯、倒车灯是否正常亮起
4. ✅**雨刮与玻璃水**
    -   雨刮片有无老化裂纹；玻璃水及时补充
5. ✅**仪表盘警告灯**
    -   启动后所有指示灯应熄灭；若有常亮故障灯请及时检修

## 五、常见问题与注意事项
### ⚠️英仕派特别关注点
1. 🛑**1.5T发动机积碳问题**
    -   缸内直喷发动机容易产生进气门积碳。
    - ⚡建议：定期使用燃油添加剂 +每隔20,000km进行进气道清洗。
2. 🛑**CVT变速箱保护**
    - ❗严禁弹射起步和长时间激烈驾驶。
    - ❗务必使用Honda HCF‑2专用油，不可混用其他ATF。
3. 🛑**主动安全系统校准**
    -   若前挡风玻璃有裂纹需及时修复，否则影响Honda SENSING摄像头标定。
4. 🛑**电池管理**
    -   带启停功能的车型使用EFB或AGM蓄电池。
        > *普通电池寿命短且无法承受频繁大电流放电。*

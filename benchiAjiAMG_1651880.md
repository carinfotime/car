# 奔驰A级AMG 驾驶注意事项
  
更新时间：2026年07月15日13时41分49秒   

## 一、启动与预热
1.  **冷车启动**：启动后，请勿立即大脚油门。建议怠速运转 **30-60秒**，待机油充分润滑涡轮增压器和发动机内部部件后再行驶。
2.  **机油温度**：在机油温度达到 **80°C** 之前，避免发动机转速超过 **4,000 rpm** 或进行激烈驾驶。
3.  **一键启动**：正确操作是：不踩刹车按一下启动键通电自检，等待仪表盘故障灯熄灭后，再踩刹车按启动键点火。
## 二、变速箱使用（AMG SPEEDSHIFT DCT 7速/8速双离合）
1.  **模式选择**：
    -   **C（舒适模式）**：日常通勤、拥堵路段使用，换挡平顺省油。
    -   **S / S+（运动/运动增强模式）**：需要动力响应时使用，换挡时机延后，转速保持在高位。
    -   **RACE（赛道模式）**：仅在封闭赛道或完全安全的路况下使用，ESP介入最少。
2.  **手动模式**：
    -   使用方向盘换挡拨片时，建议在发动机转速接近红区前升档。
    -   **降档补油**：激烈驾驶降档时，系统会自动补油；但在低速降档时需注意转速匹配，避免顿挫。
3.  **停车顺序**：踩死刹车 → 挂N档 → 拉电子手刹 → 松脚刹（释放车辆应力）→ 推P档 → 熄火。*直接挂P档熄火在坡道可能损伤变速箱锁止机构。*
## 三、底盘与悬挂
1.  **离地间隙低**：前保险杠下方离地间隙较小。上下陡坡、通过减速带或路沿时务必 **斜着通过**，避免刮擦前唇。
2.  **自适应悬挂（若配备）**：在颠簸路面建议使用舒适模式悬挂；运动模式下悬挂变硬，过坑洼需减速慢行。
## 四、制动系统
1.  **刹车盘磨合（新车前500公里）**：避免紧急制动。轻踩刹车让刹车片与盘均匀接触。
2.  **热衰减警告**：连续激烈驾驶后（如跑山），刹车可能出现热衰减。此时应减速滑行冷却，切勿立即洗车（高温刹车盘遇水会变形）。
3.  **异响提示**：AMG高性能复合制动系统在低速轻踩时可能发出“吱吱”声属正常现象。
## 五、轮胎与轮毂
1.  **胎压监测**：务必定期检查胎压。标准胎压通常在前轮2.3-2.5 bar，后轮2.5-2.8 bar（具体见油箱盖内侧标签）。
2.  **防爆胎/普通胎区别**：
    -   若配备防爆胎（RSC），缺气后可低速行驶约80公里至维修点。
    -   若配备普通性能胎（如米其林PS4S），扎钉后切勿继续行驶以免损坏轮毂和轮胎侧壁。
3.  **轮毂保护**：A级AMG通常采用大尺寸轮毂且扁平比低，靠边停车时注意不要蹭到路肩。
## ⚠️六、核心禁忌与警告
| ❌禁止行为 | ✅正确做法 |
| :--- | :--- |
| N档滑行 | D档带挡滑行即可 |
| P档等红灯 | N档+电子手刹或Auto Hold |
| RACE模式跑市区湿滑路面 | S+或C模式更安全 |
| DCT变速箱弹射起步过于频繁 | AMG允许弹射起步但需间隔冷却 |
| E挡熄火充电/开空调长时间怠速 | AMG发动机积碳风险高 |
## ⚡七、特殊功能操作指南
### AMG动态选择
-   *Individual*:可自定义组合动力响应、悬挂硬度、转向力度和排气声浪。
### RACE START (弹射起步)
1.  车辆挂入S+或RACE模式
2.  左脚踩死刹车
3.  右脚油门到底触发“RACE START”确认提示
4.  松开刹车即可完成弹射起步
*注意:此功能对传动系统负荷较大,不建议频繁使用*
### RECUPERATION (能量回收)
-   松开油门时的拖拽感可通过调节能量回收等级改变(部分车型支持方向盘按键调节)。
## 🔧八、日常维护建议
1.  每5000公里检查机油尺(电子机油尺在中控屏查看)。
2.  涡轮增压车型熄火前最好怠速30秒再关闭发动机(尤其高速刚下来)。
3.  每20000公里建议清洗节气门和进气道积碳(GDI直喷发动机通病)。
4.  定期检查冷却液液位和颜色,确保冰点符合要求。
---
> *最后提醒:AMG是一台性能机器,请尊重它的动力输出,在公共道路遵守交通法规,安全第一。*

# 艾瑞泽8 PRO 保养指南
  
更新时间：2026年07月20日17时09分43秒   

## 一、概述
本指南适用于奇瑞艾瑞泽8 PRO车型。定期且正确的保养不仅能延长车辆的使用寿命，还能确保行车安全与驾驶性能。请严格按照以下建议进行维护。
## 二、常规保养周期表
| 项目 | 周期（公里或时间，以先到为准） |
| :--- | :--- |
| **机油 & 机滤** | 每 **5,000公里** 或 **6个月** |
| **空气滤芯** | 每 **10,000公里** 或 **1年** |
| **空调滤芯** | 每 **10,000公里** 或 **1年** （建议在换季前检查） |
| **燃油滤芯** | 每 **30,000公里** |
| **火花塞** | 每 **30,000 - 40,000公里** （涡轮增压车型建议缩短至3万公里） |
| **制动液** | 每 **40,000公里** 或 **2年** |
| **防冻液/冷却液** | 每 **40,000公里** 或 **2年** （首次更换后视情况延长） |
| **变速箱油（自动挡）** | 每 **60,000公里** （恶劣工况下建议提前至4万公里） |
| **正时皮带/链条组件（如适用）** | *正时链条*：免维护；*正时皮带*：约80,000 - 100,000公里（需确认具体发动机型号） |
> *注：以上为通用建议，具体请以随车《用户手册》为准。若经常在拥堵、多尘、高温或寒冷环境下行驶，部分项目应适当缩短周期。*
## 三、日常检查项目（每月/每次加油时）
-   [ ] **机油液位**:冷车状态下拔出机油尺，检查油位是否在上下刻度线之间。
-   [ ] **冷却液液位**:观察膨胀罐内的液面是否在MIN与MAX之间。
-   [ ] **制动液液位**:检查储液罐液位是否正常。
-   [ ] **轮胎状况**:检查胎压（含备胎）、胎面磨损及有无鼓包、裂纹。
-   [ ] **灯光系统**:确认所有外部灯光（大灯、转向灯、刹车灯等）工作正常。
-   [ ] **雨刮器 &玻璃水**:检查雨刮片是否老化，并加注足量玻璃水。
## 四、关键保养项目详解
### (一)发动机机油
-   *推荐规格*:请使用符合奇瑞认证的标号，通常为 SAE ***5W-30*** (全合成)。
-   *用量*:约4.0 -4.5升（视具体排量而定）。
-   *注意*:务必使用正品全合成机油，劣质机油会加速发动机磨损。
### (二)空气滤芯
-   位于发动机舱内一个黑色塑料盒中。
-   更换频率取决于行驶环境。沙尘地区应缩短至5,000 -7,500公里更换一次。
### (三)空调滤芯
-   通常位于副驾驶手套箱后方。
-   定期更换可防止空调异味并保证车内空气质量。
### (四)制动系统
-   刹车片厚度低于3mm时必须更换。
-   刹车盘出现明显沟槽或变形时应一同更换。
-   制动液具有吸水性，含水量超过3%时必须立即更换。
### (五)轮胎
-   标准胎压参考值通常在驾驶员侧B柱标签上标注（一般前后轮均为2.3 -2.5 bar）。
-   建议每10,000公里进行四轮换位（前后交叉对调），以均衡磨损。
-   轮胎使用超过5年或磨损至磨损标记处必须更换。
## 五、各里程阶段重点保养清单
### A.首保 (5,000 km /6个月)
1.  更换发动机机油及机油滤清器
2.  检查底盘螺栓扭矩
3.  检查各油液及管路密封性
4.  免费全车检测
### B.小保养 (10,000 km /1年)
1.  更换机油 +机滤
2.  更换空气滤芯 +空调滤芯
3.  轮胎换位 +动平衡检测
4.  全面底盘及悬挂系统检查
### C.中保养 (20,000 km /2年)
1.  执行小保养全部项目
2.  清洗节气门及喷油嘴(视积碳情况)
3.  检查火花塞状态(必要时提前更换)
4.  检测蓄电池健康度
### D.大保养 (40,000 km /4年)
1.  执行中保养全部项目
2.  更换火花塞(强制)
3.  更换制动液(强制)
4.  更换防冻液(强制)
5.  清洗燃油系统及进气道积碳(推荐)
### E.EXTRA大保 (60,000 km /6年)
1.  执行大保养全部项目
2. ***自动变速箱油***:必须彻底循环机换油(约需12升新油)
3.***燃油滤清器***:外置式则直接替换;内置式则连同泵体总成一起考虑(需专业判断)
## 六、常见问题与提示(Q&A)
Q:**仪表盘出现“扳手”图标是什么意思？**
A:这是厂家预设的“提醒保养”指示灯。完成对应里程的保养后，维修技师会用诊断仪将其复位归零。
Q:**能否自行选择非4S店进行维保？**
A:可以。根据国家法规，车主有权选择具备资质的正规修理厂进行维保。但务必保留好每次的维修工单和发票作为凭证——这不会影响整车质保权益(除非能证明故障是由非授权修理不当引起的)。
Q:**涡轮增压车型有什么特殊注意事项？**
A:
①必须使用高品质全合成机油；
②冷车启动后怠速热车30秒左右再行驶；
③长途高速行驶后不要立即熄火——让发动机怠速运转1~2分钟再熄火——有助于保护涡轮轴承散热润滑；
④定期清理中冷器管路积碳/油泥。
Q:**如何判断是否需要清洗节气门？**
A:如果出现怠速不稳、转速忽高忽低；或者加速反应迟钝有顿挫感；或者油耗莫名升高——大概率是节气门过脏了——需要拆洗匹配复位即可解决。(通常每隔20k~30k km做一次即可)

# 本田CR-V 动力参数介绍
  
更新时间：2026年07月19日18时42分35秒   

## 一、发动机参数
### 1.5T 涡轮增压发动机（主力机型）
| 项目 | 参数 |
|------|------|
| **发动机型号** | L15BZ / L15BL |
| **排量** | 1.5L（1498cc） |
| **进气形式** | 涡轮增压 |
| **气缸排列** | 直列四缸 |
| **最大马力** | **193 Ps**（142 kW） |
| **最大功率转速** | 5600 rpm |
| **最大扭矩** | **243 N·m** |
| **最大扭矩转速** | 2000–5000 rpm |
| **燃油标号** | 92# 及以上无铅汽油 |
| **供油方式** | 缸内直喷 |
### 2.0L e:PHEV / e:HEV（混动版）
> *注：部分市场或新款车型搭载本田第四代 i-MMD 双电机混动系统*
#### 🔹 e:HEV（油电混动）
- **发动机排量：** 2.0L
- **发动机形式：** L4 + DOHC + i-VTEC
- **发动机最大功率：** ~107 kW（约145 Ps）
- **驱动电机最大功率：** ~135 kW
- **驱动电机最大扭矩：** ~315 N·m
- **系统综合功率：** ~158 kW
- **变速箱形式：** E-CVT（电子无级变速）
#### 🔹 e:PHEV（插电混动）
- *基本与 e:HEV一致，但电池容量更大，支持外部充电*
- *纯电续航里程约73–85 km（WLTC工况）*
---
## 二、变速箱参数
### CVT（无级变速箱）
- *适用于1.5T燃油版*
- *模拟挡位*：支持手动模式/换挡拨片模拟7挡
- *特点*：平顺省油，响应线性
### E-CVT（电子无级变速箱）
- *适用于混动版*
- *结构特点*：通过行星齿轮组实现动力分配，无传统钢带结构
---

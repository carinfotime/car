## 快运王 动力系统介绍

### 核心动力总成
快运王搭载了高效可靠的动力系统，旨在满足城市物流及短途运输的高强度需求，兼顾动力性与燃油经济性。
### 发动机参数
| 项目 | 参数 |
| :--- | :--- |
| **发动机型号** | 1.5T 涡轮增压发动机 |
| **排量** | 1.5 L |
| **进气形式** | 涡轮增压 |
| **最大马力** | 147 Ps |
| **最大功率** | 108 kW |
| **最大功率转速** | 5500 rpm |
| **最大扭矩** | 210 N·m |
| **最大扭矩转速** | 2000 - 4000 rpm |
| **供油方式** | 多点电喷 |
| **燃油标号** | 92# 及以上无铅汽油 |
### 传动系统
-   **变速箱类型**：6速手动变速箱 / 可选配 CVT 无级变速箱
-   **驱动方式**：前置前驱
### 动力性能表现
-   **最高车速**：160 km/h
-   **0-100 km/h 加速时间**：约 12.5 秒
-   **工信部综合油耗**：7.8 L/100km  (手动挡) / 8.1 L/100km (CVT)
### 动力特点
1.  **低扭强劲**：发动机在 **2000 rpm** 即可输出 **210 N·m** 的最大扭矩，起步轻快，满载爬坡时动力储备充足，适合城市频繁启停和重载工况。
2.  **经济实用**：采用多点电喷技术，可使用 92 号汽油，有效降低运营成本。手动挡车型传动效率高，油耗表现更优。
3.  **成熟可靠**：该动力总成经过市场长期验证，结构简单，维护保养方便，可靠性高，能适应高强度运输需求。

# 欧拉芭蕾猫 —— 动力参数介绍
  
更新时间：2026年07月20日16时20分24秒   

欧拉芭蕾猫定位为纯电动复古时尚座驾，其动力系统调校更侧重于城市通勤的平顺性与静谧性，而非极致性能。
## 核心动力总成
| 项目 | 参数 |
| :--- | :--- |
| **驱动形式** | 前置单电机 (前轮驱动) |
| **电动机类型** | 永磁同步电机 |
| **最大功率** | **126 kW** (约 171 马力) |
| **最大扭矩** | **250 N·m** |

| 项目 | 参数 |
| :--- | :--- |

-   **快充时间：** 30分钟可将电量从30%充至80%
-   **慢充时间：** 约8-10小时 (使用家用交流桩)
## 行驶性能
-   **最高车速：** 155 km/h
-   **0-50km/h加速时间：** 约3.8秒 (官方未公布0-100km/h加速成绩)

# 星际X SUV 驾驶注意事项
  
更新时间：2026年07月21日08时30分58秒   

为了确保您和乘客的安全，并充分发挥星际X SUV的性能，请务必遵守以下驾驶注意事项。
## 一、启动与预热
1.  **智能钥匙检查**：确保智能钥匙在车内有效范围内。
2.  **启动程序**：踩下制动踏板，按下“一键启动”按钮。请勿在未踩刹车时误触。
3.  **冷车预热**：
    -   **燃油版**：启动后怠速运转 **30-60秒**，待转速表指针回落至正常怠速区间（约700-800转）后再行驶。避免冷车大脚油门。
    -   **新能源/混动版**：系统自检完成后即可行驶，但初期应避免急加速，让动力电池和电机达到最佳工作温度。
## 二、行驶中操作规范
### 1. 动力与传动系统
-   **四驱模式选择（如配备）**：
    -   `2H`（高速两驱）：适用于干燥铺装路面。
    -   `4H`（高速四驱）：适用于雨雪、沙石或轻度越野路面。时速建议不超过80km/h。
    -   `4L`（低速四驱）：仅用于重度越野、爬坡或脱困。切换前必须将车辆完全停稳并挂入N挡。
-   **变速箱保护**：行驶中严禁从D挡直接挂入R挡或P挡。等待红绿灯时，建议挂入N挡以减少变速箱磨损。
### 2. 制动系统
-   **新车磨合期（前1000公里）**：避免紧急制动。如遇突发情况需急刹，请用力踩死刹车踏板（ABS会介入）。
-   **涉水后处理**：涉水后应轻踩几次刹车，利用摩擦生热蒸发刹车盘上的水分，恢复制动效能。
### 3. 转向与悬挂
-   **方向盘打死**：尽量避免长时间将方向盘打到底，否则可能损坏转向助力泵。回正少许方向即可。
-   **过减速带/坑洼路**：提前减速至20km/h以下通过。高速冲撞可能导致悬挂系统变形或轮胎鼓包。
## 三、特殊路况驾驶
| 路况类型 | 核心操作要点 |
| :--- | :--- |
| **高速公路** | -保持安全车距（至少2秒法则）。<br>-避免长时间占用超车道。<br>-注意横风影响，紧握方向盘。 |
| **雨雪/湿滑路面** | -开启雾灯或近光灯。<br>-降低车速至常规的70%以下。<br>-避免猛打方向或急加速。<br>-若发生侧滑，向侧滑方向轻打方向并松油门。 |
| **越野/非铺装路** | -使用4H或4L模式。<br>-关闭车身稳定系统（ESP）以允许适度车轮打滑脱困。<br>-缓慢通过深沟或交叉轴路段。<br>-注意底盘高度极限，避免托底损伤电池包（新能源版）。 |
## 四、安全辅助系统使用须知
1.  **自适应巡航 (ACC)**：
    -   仅在高速公路或路况简单的主干道使用。
    -   驾驶员双手必须始终握住方向盘，随时准备接管车辆。
2.  **车道保持 (LKA)**：
    -   该功能是辅助工具而非自动驾驶。在车道线模糊或被积雪覆盖时可能失效。
3.  **自动紧急制动 (AEB)**：
    -   对静止物体、行人及横穿车辆的识别能力有限。切勿依赖此功能代替主动刹车。
## 五、停车与熄火
1.  **停车步骤（自动挡）**：
    1.  踩死刹车使车辆完全静止。
    2.  挂入N挡 ->拉紧电子手刹 ->松开脚刹确认车辆不溜车 ->再次踩下脚刹挂入P挡 ->熄火。
2.  *注：直接挂P挡熄火可能导致下次启动时变速箱卡滞异响。*
3.  **坡道停车**：先拉手刹再挂P挡的顺序尤为重要。
4.  **关闭电源前检查**：确认所有车窗已关闭、大灯及车内阅读灯已关闭（防止电瓶亏电）。
## ⚠️六、特别警示
-   ❌禁止酒后驾车、疲劳驾驶或在服用影响精神状态的药物后驾车。
-   ❌禁止在车内放置未固定的重物或尖锐物品；后备箱物品需使用固定网兜固定。
-   ❌禁止擅自改装电路（如加装非原厂氛围灯、音响），这可能导致车辆自燃且失去质保资格。
-   ❌禁止儿童坐在副驾驶位置；12岁以下儿童必须使用符合其体重的儿童安全座椅并置于后排。

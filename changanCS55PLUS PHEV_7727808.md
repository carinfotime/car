## 长安CS55PLUS PHEV 空间参数介绍
  
更新时间：2026年07月17日14时50分30秒   

### 一、整车尺寸参数
| 项目 | 参数 |
| :--- | :--- |
| 车长（mm） | 4539 |
| 车宽（mm） | 1860 |
| 车高（mm） | 1680 |
| 轴距（mm） | **2656** |
| 前轮距（mm） | 1595 |
| 后轮距（mm） | 1600 |
> **说明**：较长的轴距为车内纵向空间提供了良好基础，宽体车身设计提升了横向乘坐舒适度。
### 二、乘坐空间
#### 1. **前排空间**
- **头部空间**：得益于SUV的高车身设计，前排头部空间约为一拳三指（以身高175cm体验者为例）。
- **腿部空间**：座椅支持前后及高低调节，驾驶位可调范围充裕。
- **横向宽度**：中控台设计紧凑，主副驾之间留有充足间隔，不显拥挤。
#### 2. **后排空间**
- **腿部空间**：轴距2656mm带来约两拳的膝部余量，长途乘坐不易疲劳。
- **头部空间**：车顶采用内凹处理，后排头部空间约为一拳一指。
- **座椅舒适度**：
    -  后排座椅靠背角度适中。
    -  坐垫长度足够支撑大腿。
    -  中央地台隆起较低（约3-5cm），中间乘客放脚较为便利。
#### 3. **座椅布局**
-  标准5座布局。
-  后排支持 **4/6比例放倒**，可灵活扩展储物空间。
### 三、储物空间
#### 1. **常规储物格**
| 位置 | 容量/特点 |
| :--- | :--- |
| 车门板储物格 |  前后门均可放置2瓶500ml矿泉水及杂物 |
| 中央扶手箱 |  深度适中，可放置小件物品或纸巾盒 |
| 手套箱 |  标准容积，可收纳文件、手册等 |
|  杯架 (前后) |  前排2个 +后排中央扶手2个 |
#### 2. **后备箱容积**
|状态	             |容积 (L)	           |备注                        |
|---                 |---                   |---                         |
|标准状态           	|约 **475 L**         	|可轻松容纳3个24寸行李箱     |
|后排座椅放倒后     	|约 **1200 L+**       	|形成近乎纯平的地台         |
> ⚠️ *注：由于PHEV车型电池组布置于底盘下方，后备箱地板高度与燃油版基本一致，未侵占装载空间。*
### ✨四、亮点与优势总结
1. **乘坐不压抑**：全景天窗（部分配置）有效提升车内开扬感。
2. **地台平整**：后排中间隆起低，三人出行更友好。
3. **装载灵活度高**：4/6放倒后可搬运长件物品或作为临时休息使用。
4. **电池不影响后备箱**：PHEV车型保持了与燃油版一致的装载能力。

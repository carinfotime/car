# 东风小康EC36II 内饰参数介绍

## 整体内饰风格
东风小康EC36II内饰采用简洁实用的设计风格，以深色系为主调，注重功能性与耐用性，符合其商用及家用兼顾的定位。
## 座椅配置
| 项目 | 参数 |
|------|------|
| 座椅材质 | 织物座椅（部分版本可选皮革座椅） |
| 座椅布局 | 2+2+3 七座布局 |
| 前排座椅 | 手动调节，支持前后滑动及靠背角度调节 |
| 第二排座椅 | 独立座椅，支持前后滑动及靠背调节 |
| 第三排座椅 | 三人联排座椅，可整体折叠放倒 |
| 座椅填充 | 高密度海绵，支撑性良好 |
## 仪表盘与中控
| 项目 | 参数 |
|------|------|
| 仪表盘 | 传统机械指针式仪表盘+单色行车电脑显示屏 |
| 行车电脑显示 | 续航里程、电耗、档位、车速、电量等信息 |
| 中控台布局 | T字形对称设计，功能分区清晰 |
| 中控屏幕 | 7英寸触控液晶屏（部分配置） |
| 多媒体系统 | 支持蓝牙连接、USB接口、收音机功能 |
| 空调系统 | 手动空调，前后分区独立控制 |
| 空调类型 | 电动压缩机空调，制冷/制热 |
## 方向盘
| 项目 | 参数 |

| 多功能按键 | 高配车型配备（音量控制、蓝牙电话等） |

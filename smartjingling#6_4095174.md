# smart 精灵#6 保养指南
  
更新时间：2026年07月18日17时03分55秒   

## 一、概述
smart 精灵#6 是一款纯电动 SUV，与传统燃油车不同，它无需更换机油、机滤等部件。其保养核心在于 **三电系统（电池、电机、电控）**、底盘、制动系统以及空调系统的检查与维护。遵循官方保养计划，可以有效延长车辆寿命并确保行车安全。
---
## 二、常规保养周期表
| 项目 | 周期 | 说明 |
| :--- | :--- | :--- |
| **空调滤芯** | **每12个月或20,000公里** | 先到为准，建议在换季前更换 |
| **制动液** | **每24个月或40,000公里** | 检测含水量，超标需立即更换 |
| **减速器油（齿轮油）** | **每48个月或80,000公里** | 纯电车减速器专用油 |
| **冷却液（电机/电池温控）** | **每48个月或80,000公里** | 检查液位及冰点 |
| **12V蓄电池检查** | **每次保养时** | 检查电压及健康状态 |
| **轮胎换位与动平衡** | **每10,000-15,000公里** | 建议与常规检查同步进行 |
| **制动片/制动盘检查** | **每20,000公里或按需** | EV能量回收会减少刹车片磨损，但仍需定期检查 |
> *注：以上为官方推荐标准周期。若车辆长期在恶劣环境（多尘、潮湿、高温）或高强度使用（频繁快充、重载），建议适当缩短部分项目的保养间隔。*

## 三、日常自行检查项目（每月一次）
1. **轮胎状况**
   - 胎压：保持在厂家推荐值（通常在驾驶员侧门框标签上）。
   - 胎纹深度：低于1.6mm必须更换。
   - 外观：有无鼓包、裂纹或异常磨损。
2. **灯光系统**
   - 全车灯光（大灯、尾灯、转向灯、刹车灯）是否正常工作。
3. **玻璃水液位**
   - 及时添加专用玻璃水（冬季注意防冻）。
4. **充电接口**
   - 保持充电口干燥清洁，无异物或腐蚀。
5. **雨刮片**
   - 刮拭效果是否清晰，有无异响或老化开裂。

## 四、专业保养流程详解
### 🔹 A类保养（约20,000公里 /12个月）
- ✅读取整车故障码及三电系统数据
- ✅更换空调滤芯
- ✅检查并补充冷却液
- ✅检查12V蓄电池电压
- ✅底盘及悬挂紧固件力矩检查
- ✅制动管路及制动液含水量检测
- ✅轮胎气压及磨损情况记录
- ✅车门铰链/天窗滑轨润滑
### 🔹 B类保养（约40,000公里 /24个月）
包含A类全部项目 +：
- 🔧更换制动液
- 🔧深度检测高压电池组绝缘电阻
- 🔧电机控制器冷却系统清洗
- 🔧四轮定位检测与调整（如发现跑偏）
### 🔹 C类大保（约80,000公里 /48个月）
包含A+B类全部项目 +：
- ⚙️更换减速器油
- ⚙️更换驱动电机冷却液
- ⚙️全面高压系统绝缘检测报告出具

## 五、电池养护建议
smart精灵#6搭载三元锂电池或磷酸铁锂电池版本：
1. **充电习惯**
   - 📌日常使用建议电量保持在20%-80%之间。
   - 📌长途出行前可充至100%，但到达后尽快使用。
   - 📌避免长期满电停放；若需长时间停放，建议电量保持在50%-60%。
2. **温度管理**
   - ❄️冬季尽量使用预约充电功能，利用充电桩预热电池。
   - ☀️夏季避免暴晒后立即快充；优先选择地下车库或有遮阳的充电站。
3. **快慢充搭配**
   - 🏠家用慢充为主（7kW/11kW），保护电池寿命。
   - ⚡公共快充为辅；仅在长途补能时使用直流快充。
4. **BMS升级**
   - 🖥️关注官方OTA推送的电池管理系统更新，及时升级以优化管理策略。

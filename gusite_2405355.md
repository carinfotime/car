# 古思特（Ghost）内饰参数介绍
  
更新时间：2026年07月15日18时40分33秒   

## 一、座椅系统
| 项目 | 参数 |
|------|------|
| **座椅材质** | 全车顶级真皮（可选苯胺皮革或缝线皮革） |
| **前排座椅功能** | 电动调节（20向）、加热、通风、按摩功能 |
| **后排座椅功能** | 电动调节、加热、通风、按摩（长轴距版配备独立行政座椅） |
| **座椅记忆** | 带3组记忆功能，含方向盘及外后视镜联动记忆 |
| **头枕** | 可电动调节高度，带刺绣RR标识（可选） |
## 二、仪表与中控
| 项目 | 参数 |
|------|------|
| **仪表盘** | 12.3英寸全液晶数字仪表盘，经典指针式UI设计 |
| **中控屏幕** | 10.25英寸中央信息显示屏，支持触控与旋钮操作 |
| **操作系统** | Rolls-Royce专属信息娱乐系统 |
| **导航系统** | 3D卫星导航，实时路况更新 |
| **抬头显示** | 彩色投影式抬头显示系统 |
## 三、音响系统
| 项目 | 参数 |
|------|------|
| **标准音响** | Bespoke Audio定制音响系统，18扬声器，1300W功率 |
| **选装音响** | Bowers & Wilkins钻石环绕音响系统，20扬声器，1400W功率 |
## 四、空调与空气净化
- **四区独立自动空调**
- **空气净化系统**：高效微粒过滤（PM2.5过滤）
- **负离子发生器**
- **湿度传感器与自动除雾功能**
## 五、内饰材质与装饰
### 🪵木饰面板
- Open Pore开放孔木饰面板
- Canadel木饰面板（手工拼接）
-  多种木材可选：胡桃木、黑檀木、白蜡木等
### 🧵缝线与刺绣
- RR字母刺绣头枕
-  定制对比色缝线
-  星空顶缝线图案可选
### 💡氛围照明
- LED环绕氛围灯
-  自定义颜色选择
-  星空顶篷（Starlight Headliner）：1340根光纤手工编织，模拟夜空星光效果
## 六、储物空间
-  前门储物格：可容纳1L水瓶
-  中央扶手箱：带冷藏功能（可存放香槟）
-  后排中央扶手：集成杯架与储物盒
-  手套箱：带照明与绒面内衬
-  后备箱容积：约490L（标准轴距）/约520L（长轴距）
## 七、个性化定制选项（Bespoke高级定制）
> Rolls-Royce提供几乎无限的内饰定制服务：
1. **配色方案**：44,000种以上颜色可选
2. **刺绣图案**：可定制专属徽章或图案刺绣于座椅/门板/头枕
3. **木饰面板拼接图案**：客户可提供设计灵感进行专属拼花制作
4. **星光顶篷定制**：可按客户提供的夜空照片复刻星座布局
5. **金属饰件材质选择**：镀铬/镀金/镀银/钛合金等不同金属质感
---
*注：以上参数基于最新款古思特及古思特Extended长轴距版整理，具体配置以官方发布为准。*

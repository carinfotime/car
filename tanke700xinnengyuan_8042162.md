# 坦克700 Hi4-T 动力参数介绍
  
更新时间：2026年07月20日18时51分10秒   

坦克700 Hi4-T 搭载了一套由 **3.0T V6双涡轮增压发动机** 与 **P2电机** 组成的插电式混合动力系统（Hi4-T），兼顾了强大的越野性能与燃油经济性。
## 核心动力总成
### 1. 发动机
-   **型号**：E30Z
-   **类型**：3.0升 V型6缸 双涡轮增压
-   **最大功率**：265 kW（约360马力）
-   **最大扭矩**：560 N·m
-   **技术特点**：
    -   双涡轮增压技术，有效改善涡轮迟滞。
    -   配备48V轻混系统（BSG电机），辅助启动和发电。
### 2. 电动机
-   **类型**：永磁同步电机（P2架构，位于变速箱与发动机之间）
-   **最大功率**：120 kW
-   **最大扭矩**：400 N·m
### 3. 综合系统参数
-   **综合最大功率**：385 kW（约524马力）
-   **综合最大扭矩**：800 N·m （峰值扭矩，提供强劲的爆发力）
-   **传动系统**：9速手自一体混动专用变速箱（9HAT）
## 电池与续航
| 项目 | 参数 |
| :--- | :--- |
| **电池类型** | 三元锂电池 |
| **电池容量** | 37.1 kWh |
| **WLTC纯电续航里程** |约90公里 |
| **WLTC综合油耗** |约2.97 L/100km （馈电油耗约10L/100km） |
| **快充时间** |支持快充，30%-80%电量约需25分钟 |
## 越野与驱动相关动力特性
1.  **驱动形式**
    -   非解耦四驱系统（机械四驱），前后轴通过传动轴硬连接。
    -   配备MLOCK机械锁止功能，可实现前后桥差速锁的刚性锁止。
2.  **动力输出特性**
    -   P2电机位于变速箱输入端，能够直接为车轮提供扭矩，在低速越野时响应迅速。
    -   发动机与电机可并联驱动，提供高达800N·m的轮端扭矩输出。
3.  **能量管理**
    -   支持多种驾驶模式（纯电、混动、智能）。
    -   保留传统越野车的低速四驱挡位，结合电机特性实现精准的蠕行控制。
## *总结*
坦克700 Hi4-T的动力系统核心优势在于将大排量V6发动机的持续高扭特性与电机的瞬时爆发力相结合。800N·m的综合扭矩配合机械四驱结构，使其在攀爬、脱困等极限越野场景下具备显著优势；同时37kWh的电池组也提供了日常通勤所需的纯电续航能力。

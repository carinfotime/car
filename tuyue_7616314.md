# 途岳驾驶注意事项
  
更新时间：2026年07月13日11时16分08秒   

## 一、启动与熄火
1. **启动前检查**：确认座椅、后视镜已调整至合适位置，系好安全带。检查仪表盘是否有故障灯亮起。
2. **一键启动**：踩住刹车踏板，短按“START/STOP”按钮启动发动机。切勿在未踩刹车时误触。
3. **冷车启动**：冬季或长时间停放后，建议怠速热车 30-60 秒，待转速回落至正常怠速（约 700-800 rpm）后再行驶。
4. **熄火顺序**：停稳车辆 → 挂入 P 挡 → 拉紧电子手刹 → 松开脚刹 → 熄火。
## 二、行驶中操作
1. **变速箱使用**：
   - D 挡为日常前进挡；S 挡（运动模式）适合超车或爬坡，但会增加油耗。
   - 等红灯时间短（<30秒）可保持 D 挡+踩刹车；时间长建议挂 N 挡+拉手刹。
   - **严禁**在车辆未停稳时挂入 R 挡或 P 挡。
2. **自动启停系统**：
   - 默认开启，深踩刹车发动机会自动熄火以省油。
   - **涉水路段务必关闭启停**，防止发动机在水中二次启动导致损坏。
3. **涡轮增压车型注意**：
   - 长途或激烈驾驶后，建议怠速运转1-2分钟再熄火，以保护涡轮增压器散热。
4. **四驱系统（如适用）**：
   - 途岳部分车型配备4MOTION智能四驱。在雨雪、泥泞路面可手动切换至越野模式。
   - **铺装路面请勿长期使用越野模式**，以免增加传动系统磨损和油耗。
## 三、安全驾驶提醒
1. **视野盲区**：A柱较粗壮，转弯时请左右移动头部观察路况。变道时善用并线辅助提示灯（如有）。
2. **主动安全功能**：
   - Front Assist（前部辅助系统）：当检测到与前车距离过近时会发出警告或主动制动。切勿完全依赖此系统。
   - ACC自适应巡航：设定好跟车距离和速度后可在高速上解放双脚，但需随时准备接管控制权。
3. **儿童安全锁**：后排有儿童乘坐时，请启用车门上的儿童安全锁（机械拨杆），防止车内开门。
## 四、特殊路况应对
| 路况 | 建议操作 |
|------|----------|
| **雨天/湿滑路面** | ESP车身稳定系统保持开启；降低车速；避免急刹急转向 |
| **积雪/结冰路面** | TCS牵引力控制系统自动介入；起步缓加油；建议更换雪地胎 |
| **长下坡路段** | M挡手动降档利用发动机制动；避免长时间踩刹车导致过热 |
| **涉水路段** | *关闭自动启停*；低档位匀速通过；水深超过轮毂一半请绕行 |
## 五、日常维护小贴士
1. **机油检查**：每月检查一次机油尺液位（冷车状态），保持在上下刻度之间。
2. **轮胎气压**：标准胎压约2.3-2.5 bar（具体见油箱盖内侧标签）。每月检查一次。
3. **颗粒捕捉器（GPF）再生提醒（国六B车型）**
   若仪表提示“颗粒过滤器需要再生”，请以较高转速（>2000rpm）行驶20-30分钟即可完成再生。避免长期短途低速行驶导致堵塞。
---
> ⚠️ *以上内容基于途岳常见配置整理。具体操作请以您车辆的实际用户手册为准。*

## 二、内部乘坐空间
### 2.1 前排空间
- **头部空间**：前排头部空间约 **1000 mm**（座椅调至最低），身高180cm的乘客仍有一拳以上余量。
- **腿部空间**：驾驶位腿部纵向调节范围充裕，支持前后及高低调节。
- **肩部空间**：前排横向宽度约 **1500 mm**，主副驾之间互不干扰。
### 2.2 第二排空间
- **头部空间**：约 **990 mm**，无明显压抑感。
- **腿部空间**：最大膝部空间可达 **两拳以上（约80-120 mm）**，支持前后滑动调节。
- **座椅角度**：靠背角度可调（多级调节），提升长途舒适性。
- **横向宽度**：后排坐满三名成年人仍较为宽松。
### 2.3 第三排空间（7座版）
> *注：瑞虎8L提供5座和7座两种布局。*
- **头部空间**：约 **920 mm**，适合身高170cm以下的乘客。
- **腿部空间**：第二排调至最前时，第三排膝部约有一拳余量；第二排正常坐姿下，第三排适合短途应急或儿童乘坐。
- **进出便利性**：第二排座椅支持一键折叠/前移，方便第三排进出。
---
## 三、储物与后备箱
### 3.1 后备箱容积
| 座椅状态 | 容积 (L) |
| :--- | :--- |
| **5座模式（全部放倒）** | ≈1930 L （最大扩展） |
| **5座模式（正常使用）** | ≈889 L |
| **7座模式（三排放倒）** | ≈700 L |
| **7座模式（三排立起）** | ≈193 L |
### 3.2 后备箱细节
- ***开口尺寸***:  宽度约1100 mm,  高度约780 mm,  地台离地高度适中,  搬运大件物品方便。
- ***内部纵深***:  二排放倒后纵深可达近1900 mm,  可平躺或装载超长物品。
- ***储物格***:  后备箱地板下方设有隐藏储物格,  可放置工具、隐私物品等。
### 3.3车内储物分布
全车共计超过30处储物设计,主要分布如下：
1. ***门板储物格***:  前后门板均可容纳500ml水瓶+杂物。
2. ***中央扶手箱***:  深度可观,可放置6瓶500ml矿泉水或随身包。
3. ***手套箱***:  标准大小,可存放文件、手册等。
4. ***杯架***:
    - *前排*:中控台2个杯架+门板各1个杯位
    - *后排*:中央扶手2个杯架+门板各1个杯位
    - *第三排*:两侧各1个杯架/储物槽
5. ***手机/卡片槽***:
    - *中控台下方*镂空储物区(带USB接口)
    - *挡杆旁*手机专用槽(支持无线充电)
6. ***座椅背袋***:
    - *前排座椅背后*地图袋(可放平板/杂志)
    - *第二排座椅背后*(7座版)小储物袋
---
## 四、其他实用设计
- ***全景天窗***:
    采光面积约1.1m²,有效提升车内开扬感;配备电动遮阳帘。
- ***空调出风口布局***:
    全车共6个出风口(前排4个+后排独立出风口),部分车型配备第三排出风口。
- ***儿童安全座椅接口***:
    第二排标配ISOFIX接口+Top Tether锚点;部分车型第三排也预留接口。
---

## LUMMA CLR LD 挑战者 — 空间参数介绍
  
更新时间：2026年07月16日17时13分23秒   

LUMMA CLR LD 挑战者基于路虎卫士长轴距版（L663）打造，在保留原车硬派越野基因的同时，通过LUMMA专属的外观套件与内饰升级，进一步强化了其豪华与实用并存的空间表现。
### 车身尺寸（外廓）
| 参数 | 数值 |
| :--- | :--- |
| 车长 | **4,958 mm** |
| 车宽（含轮眉） | **2,008 mm** |
| 车高 | **1,967 mm** |
| 轴距 | **3,022 mm** |
> *注：因加装LUMMA宽体套件及轮眉，车身宽度相比原厂卫士110有所增加。*
### 内部乘坐空间
- **座椅布局**：标准5座布局（2+3），可选装2+2式4座行政布局。
- **前排空间**：
    - 头部空间：约 **1,020 mm**
    - 腿部空间：充裕，支持多向电动调节及记忆功能。
- **后排空间**：
    - 头部空间：约 **1,000 mm**
    - 腿部空间：约 **900 mm**（以身高175 cm乘客为标准）
    - 后排座椅支持4:2:4比例放倒，提升装载灵活性。
### 储物与行李厢容积
| 项目 | 容积 / 尺寸 |
| :--- | :--- |
| 标准行李厢容积 | **857 L**（至行李厢隔板高度） |
| 最大扩展容积 | **2,233 L**（后排座椅完全放倒后） |
| 行李厢开口宽度 | **1,200 mm** |
| 行李厢内部高度 | **950 mm** |
| 行李厢纵深（标准状态） | **1,100 mm** |
- **车内储物格**：
    -  中控台杯架 ×2
    -  中央扶手箱（带冷藏功能）
    -  门板储物槽（可容纳1.5L水瓶）
    -  手套箱容积约12 L
    -  后排中央扶手带杯架及储物盒
### “挑战者”专属空间特色
1. **碳纤维内饰套件**：中控、门板及方向盘采用LUMMA定制碳纤维饰板，在减轻重量的同时提升运动感。
2. **星空顶篷选装**：可选Alcantara材质包裹的星空顶，营造更开阔的座舱氛围。
3. **后备箱多功能组件**：提供可折叠储物托盘、锚点及12V电源接口，便于户外探险装备固定。
4. **NVH优化升级**：全车增加隔音材料，提升长途行驶时的静谧性与乘坐舒适性。
---
以上数据基于LUMMA官方公布信息及实车测量整理。实际空间感受可能因选装配置不同而略有差异。

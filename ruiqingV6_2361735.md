# 瑞擎V6 动力参数介绍
  
更新时间：2026年07月21日09时52分08秒   

## 发动机参数
| 项目 | 参数 |
|------|------|
| **发动机类型** | V型6缸涡轮增压发动机 |
| **排量** | 3.0L（2996cc） |
| **进气形式** | 双涡轮增压（Twin-Turbo） |
| **燃油供给方式** | 缸内直喷（GDI） |
| **压缩比** | 10.5 : 1 |
| **燃油标号** | 95号及以上无铅汽油 |
## 动力输出
- **最大功率**：280 kW（约380马力）/ 5,500 - 6,500 rpm
- **最大扭矩**：550 N·m / 2,000 - 4,800 rpm
- **升功率**：93.3 kW/L
## 传动系统
- **变速箱**：8速手自一体自动变速箱（8AT）
- **驱动方式**：前置四驱（适时四驱系统）
## 性能表现
- **0-100 km/h加速时间**：5.8秒
- **最高时速**：250 km/h（电子限速）
- **综合油耗**：9.2 L/100km（WLTC工况）
## 技术亮点
1. **双涡管涡轮技术** — 有效减少涡轮迟滞，提升低扭响应
2. **可变气门正时系统（VVT）** — 进排气双侧独立调节，优化燃烧效率
3. **智能热管理系统** — 快速暖机，保持最佳工作温度
4. **轻量化铝合金缸体 &amp; 缸盖** — 降低整机重量，提升操控性
5. **启停系统 +48V轻混辅助** — 提升起步平顺性与燃油经济性

> *以上参数基于瑞擎V6高功率版本，实际数据以官方最终发布为准。*

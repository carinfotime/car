# 跨越王X3 动力参数介绍
  
更新时间：2026年07月20日16时58分27秒   

## 发动机
跨越王X3搭载了长安汽车自主研发的 **1.6L自然吸气发动机**，型号为 **DAM16KR**。
-   **排量**：1.6L
-   **最大功率**：90kW（约122马力）
-   **最大扭矩**：158N·m
-   **供油方式**：多点电喷
-   **排放标准**：国六B
## 变速箱
提供两种变速箱选择，满足不同用户需求：
-   **5挡手动变速箱 (MT)**：齿比绵密，换挡手感清晰，燃油经济性出色。
-   **东安DAT18R 5挡手动变速箱 (MT)**：针对商用车工况优化，可靠性高，承载能力强。
## 驱动形式
-   **驱动方式**：中置后驱 (MR)
    -   发动机位于驾驶舱下方，重心更低。
    -   后轮驱动提供更强的爬坡能力和载重起步能力。
## 底盘与悬架
动力传递与承载系统紧密配合：
| 项目 | 参数 |
| :--- | :--- |
| **前悬架类型** | 麦弗逊式独立悬架 |
| **后悬架类型** | 钢板弹簧非独立悬架（5片或6片可选） |
| **助力类型** | 电动助力转向 (EPS) |
| **轮胎规格** | 185 R14LT（轻卡专用轮胎） |
## 性能表现（参考值）
> *注：实际表现受载重、路况、驾驶习惯等因素影响。*
-   **最高车速**：约120 km/h
-   **最大爬坡度**：≥30%
-   **百公里综合油耗（参考）**：约7.5L - 8.5L
## 核心优势总结
1.  **动力充沛**：1.6L大排量自然吸气发动机，低扭强劲，适合城市配送及乡镇道路频繁启停的工况。
2.  **可靠耐用**：采用成熟的多点电喷技术及经过市场验证的东安变速箱，维护成本低。
3.  **承载有力**：中置后驱布局配合高强度钢板弹簧后悬，确保满载状态下依然拥有良好的动力输出和稳定性。

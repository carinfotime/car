# 五菱荣光S 保养指南
  
更新时间：2026年07月15日11时58分18秒   

本指南旨在帮助车主更好地了解和执行五菱荣光S的日常保养与定期维护。正确的保养不仅能延长车辆使用寿命，还能确保行车安全与经济性。
## 一、核心保养周期表
以下为官方建议的常规保养周期，实际可根据车辆使用环境（如多尘、拥堵）适当调整。
| 项目 | 周期（公里或时间，以先到为准） | 备注 |
| :--- | :--- | :--- |
| **更换机油** | **每5,000公里或6个月** | 建议使用 API SL 级或以上，粘度5W-30或10W-40 |
| **更换机油滤清器** | **每5,000公里或6个月** | 每次换机油时同步更换 |
| **更换空气滤清器** | **每10,000公里或12个月** | 多尘环境缩短至5,000公里 |
| **更换燃油滤清器** | **每20,000公里或24个月** | - |
| **更换空调滤清器** (如有) | **每10,000公里或12个月** | - |
| **检查/更换火花塞** | **每20,000-30,000公里** | - |
| **检查/更换变速箱油** (手动) | **每40,000公里或48个月** | GL-4 75W-90 |
| **检查/更换后桥齿轮油** (差速器油) | **每40,000公里或48个月** | GL-5 80W-90 |
| **检查/更换冷却液 (防冻液)** | **每40,000公里或24个月** | - |
| **检查/更换制动液 (刹车油)** | **每40,000公里或24个月** DOT3/DOT4级 |-|
| **检查/更换正时皮带 (如适用)**|按车型具体发动机要求（通常60,000-100,000km）|-|
> *注：部分早期车型可能为正时皮带，后期多为正时链条（免维护），请以随车手册为准。*
## 二、日常例行检查（建议每周一次）
1.  **机油液位：**
    -   冷车状态下拔出机油尺，擦拭干净后重新插入再拔出。
    -   液位应在上下刻度线之间。
2.  **冷却液液位：**
    -   观察膨胀壶（副水箱），液位应在 MIN 和 MAX 之间。
    -   *注意：热车时切勿打开水箱盖！*
3.  **制动液液位：**
    -   观察制动液储液罐，应在上限与下限之间。
4.  **轮胎状况：**
    -   【胎压】标准胎压通常在2.3-2.8 bar之间（参考驾驶座门框标签）。
    -   【磨损】检查是否有鼓包、裂纹或不均匀磨损。
    -   【备胎】确保备胎气压充足。
5.  **灯光系统：**
    -   【大灯、转向灯、刹车灯、倒车灯】全部点亮测试一遍。
6.  **雨刮水：**
    -   加注专用玻璃水（冬季注意防冻）。
## 三、各阶段重点保养项目详解
### A. A类保养 (小保) — @5,000km /6个月
1.  更换发动机机油
2.  更换机油滤清器
3.  清洁空气滤清器壳（视情况吹灰）
4.  全车目视检查：
    *    底盘有无漏油、螺栓松动
    *    传动轴及万向节
    *    转向拉杆及球头间隙
5.  补充雨刮水
### B. B类保养 (大保) — @20,000km /24个月
*包含A类所有项目，并增加以下内容：*
1.  更换空气滤清器
2.  更换燃油滤清器
3.  检查火花塞状况（必要时清洁或调整间隙）
4.  检查并调整离合器踏板自由行程
5.  润滑车门铰链及锁扣
### C. C类深度保养 — @40,000km /48个月
*包含B类所有项目，并增加以下内容：*
1.  手动变速箱油和后桥齿轮油全部放空并加注新油。
2.  彻底清洗节气门体及怠速马达。
3.  清洗喷油嘴（建议使用专业设备）。
4.  全面冷却系统清洗并换新防冻液。
5.  排空旧制动液并换新制动液。（需两人配合排气）
6.  四轮交叉换位。
## D: D类全面检修 — @80,000km /96个月及以上
*包含C类所有项目，并增加以下内容：*
1.【正时皮带】如果您的发动机是正时皮带设计，此时必须强制更换！同时一并换掉张紧轮和水泵。
2.【发电机皮带与空调压缩机皮带】仔细查看有无龟裂老化。
3.【轮胎】即使花纹尚可但超过6年强烈建议整套更新。
4.【刹车片与刹车盘】拆下轮子测量厚度是否在安全极限内。

## E:季节性特别注意事项
### 🥶冬季来临前准备：
* 【防冻能力检测】：用冰点仪测冷却液的冰点是否低于当地最低气温至少10℃以上(比如零下25℃)。
* 【电瓶健康度】：低温下启动电流需求大增——若电瓶已用超三年且启动无力就及时换掉吧！
* 【玻璃水】：务必换成冬季专用型(-25℃甚至更低标号)，否则结冰冻裂管路得不偿失！
### ☀️夏季来临前准备：
* 【空调制冷效果】：开启AC后出风口温度应在8℃以内才算正常——不凉快就去查冷媒压力吧！
* 【散热系统清理】：用水枪从里向外冲洗水箱散热片上的柳絮虫尸等杂物以免水温过高开锅！

## F:省钱又省心的实用技巧(老司机经验)
✅【自己动手做简单活】
>像换空气滤芯空调滤芯这类操作极其简单成本极低(网上买也就十几二十块)——完全可以自己搞定而不用去店里花冤枉工时费！
✅【选对机油很重要】
>五菱荣光S属于高转速低扭发动机——推荐用半合成甚至全合成(比如SN级别5W30)——虽然单次贵一点但能跑更长里程且保护更好！
✅【别忽略底盘养护】
>这车拉货重载多——每次举升起来记得自己拿手电筒照照后钢板弹簧有没有断裂迹象以及传动轴过桥轴承是否旷动——这些都是典型通病！
✅【定期拉高速除积碳】
>偶尔在路况允许下用低档高转速跑几分钟(比如三档4000转)——有助于清除轻微积碳保持动力顺畅！
---
## G:常见故障预警信号(及时处理避免大修)
⚠️【怠速不稳易熄火】
→大概率是节气门太脏或者怠速马达卡滞——清洗即可解决不必慌张！
⚠️【挂挡生涩难入档】
→如果是手动挡车型先试试换好点的变速箱油(比如GL475W90全合成)——若仍不行则可能是离合器分离不清需调整拉线行程！
⚠️【行驶中嗡嗡异响随车速变化】
→十有八九是后轮轴承或者差速器齿轮磨损了——尽早去检修否则可能抛锚在路上哦~
⚠️【水温偏高报警灯亮起】
→立即靠边停车熄火等待降温！切勿马上拧开水箱盖！先查风扇转不转以及节温器是否卡死再说下一步行动方案~

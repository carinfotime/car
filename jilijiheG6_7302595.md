# 吉利几何G6 内饰参数介绍
  
更新时间：2026年07月14日10时35分50秒   

## 一、整体设计风格
吉利几何G6内饰采用**极简科技风**设计理念，以简约线条与环保材质打造未来感座舱，整体色调以浅色系为主，搭配金属质感装饰件。
## 二、座椅系统
| 项目 | 参数 |
|------|------|
| 座椅材质 | 仿皮/织物混搭（高配为全仿皮） |
| 前排座椅调节 | 主驾6向电动调节 / 副驾4向手动调节 |
| 座椅加热 | 前排座椅加热（部分配置） |
| 后排座椅 | 比例放倒（40:60） |
| 后排中央扶手 | 带杯架 |
## 三、中控与仪表
- **仪表盘**：10.2英寸全液晶仪表盘
- **中控屏**：14.6英寸悬浮式触控液晶屏
- **车机系统**：HarmonyOS（鸿蒙智能座舱）
- **分辨率**：1920×1080
- **芯片**：华为麒麟990A车规级芯片
- **语音控制**：支持连续对话、可见即可说
## 四、方向盘
- **类型**：双辐式多功能方向盘
- **材质**：皮质包裹
- **调节方式**：手动上下+前后调节
- **功能按键**：多媒体控制、语音唤醒、定速巡航等
## 五、空调系统
- **类型**：自动空调
- **分区控制**：双区独立温控
- **后排出风口**：标配
- **PM2.5过滤**：CN95级空调滤芯
- **负离子净化器**（部分配置）
## 六、储物空间
| 位置 | 说明 |
|------|------|
| 中央扶手箱 | USB接口+储物格 |
| 手套箱 |  带阻尼开启 |
| 门板储物格 |  可放置水瓶/杂物 |
|  杯架 |  前排2个 +  后排2个 |
##  七、音响系统
-  扬声器数量：4喇叭（标配）/6喇叭（高配）
-  音效模式：标准/人声/古典/流行等预设均衡器
## ⚡八、灯光与氛围
-  氛围灯颜色数量（部分配置）：72色可调
-  覆盖区域：中控台+门板饰条
-  阅读灯类型：LED前后排阅读灯
## 📐九、尺寸与空间数据
|    项目    |    数值    |
|---|---|
|    轴距    |    2700 mm    |
|    前排头部空间    |    约980 mm    |
|    后排腿部空间    |    约920 mm    |
|    后备厢容积    |    430 L    |

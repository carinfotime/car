# 江西五十铃 翼放 内饰参数介绍
  
更新时间：2026年07月11日05时57分14秒   

## 一、驾驶室整体布局
- **驾驶室类型**：单排 / 排半 / 双排（根据车型配置）
- **准乘人数**：2人（单排）/ 3人（排半）/ 5人（双排）
- **驾驶室宽度**：约1880mm（中体）/ 2050mm（宽体）
- **内部高度**：约1200mm
- **车门开启角度**：最大可达90°，方便上下车
## 二、座椅系统
| 项目 | 参数 |
|------|------|
| **座椅材质** | 高级织物面料 / PVC皮革（选装） |
| **主驾座椅调节** | 前后滑动 + 靠背角度调节 |
| **副驾座椅调节** | 靠背角度调节 |
| **座椅填充厚度** | 约120mm，高密度发泡海绵 |
| **可选配置** | 全包裹气囊减震座椅（高配车型） |
## 三、仪表台与中控
- **仪表盘类型**：组合式液晶仪表盘
- **显示屏尺寸**：7英寸彩色液晶屏（高配）/ 3.5英寸单色屏（基础款）
- **中控屏尺寸**：10英寸触控大屏（选装/高配）
- **方向盘材质**：聚氨酯方向盘 / 真皮多功能方向盘（高配）
- **方向盘调节**：上下两向调节
- **多功能按键区域**：音量控制、蓝牙电话、定速巡航控制等
## 四、空调系统
| 项目 | 参数 |
|------|------|
| **空调类型** |手动冷暖空调 /自动恒温空调（选装）|
| **出风口数量** |前排4个出风口|
| **制冷剂类型** |R134a环保冷媒|
| **后排出风口** |部分车型可选装|
## 五、储物空间
- **手套箱容积**：约8L，带阻尼开启
- **门板储物格容量**：可容纳1.5L水瓶 + A4文件
- **中控台杯架数量**：2个
- **顶棚储物盒容积**：约3L
- **副驾前方储物平台面积**：约400mm ×200mm，适合放置票据、手机等物品
- ***隐藏式储物格*** ：仪表台下方设有小型储物槽
## 六、电气与多媒体配置
### 基础配置：
 -收音机+USB接口+AUX输入
 -2扬声器音响系统
 -12V电源接口×2
### 高配升级：
 -10英寸触控大屏导航系统
 -蓝牙电话+手机互联映射功能
 -倒车影像显示支持
 -4扬声器音响系统
 -USB充电接口×3（含Type-C）
## 七、灯光与视野辅助
 -前阅读灯×1，LED光源
 -遮阳板×2，副驾带化妆镜
 -电动升降车窗玻璃，一键降窗功能(主驾)
 -电动后视镜调节+加热功能(选装)
 -后窗除霜除雾功能(选装)
## 八、安全与便利配置
### 主动安全：
 -ABS防抱死制动系统+EBD制动力分配(全系标配)
 -倒车雷达(选装/高配标配)
 -倒车影像摄像头(选装/高配标配)
### 被动安全：
 -高强度笼式驾驶室结构车身设计符合ECE R29碰撞法规要求  
 —主驾安全气囊(部分车型标配/选装)  
 —预紧式三点安全带×全车座位  
 —安全带未系提醒(主驾)  
### 便利性配置：
 —遥控钥匙+中控门锁  
 —一键启动+无钥匙进入(部分高配)  
 —定速巡航控制系统  
 —间歇式前雨刮器+后雨刮器(部分车型)

以上为江西五十铃翼放系列轻卡的主要内饰参数介绍。具体配置因车型版本及选装包不同可能存在差异，请以实际实车为准。

# 焜驰麒麟 驾驶注意事项
  
更新时间：2026年07月11日02时01分11秒   

## 一、启动前检查
1. **轮胎检查**：确认胎压正常（建议冷车胎压 2.3-2.5 bar），检查轮胎表面有无鼓包、裂纹或异物扎入。
2. **油液检查**：检查机油液位、冷却液液位、制动液液位是否在正常刻度范围内。
3. **灯光与信号**：测试大灯、转向灯、刹车灯、雾灯及危险报警灯是否正常工作。
4. **雨刮与玻璃水**：确保雨刮片无老化破损，玻璃水充足。
5. **仪表盘自检**：启动后观察仪表盘有无故障灯常亮（如发动机故障灯、ABS故障灯等）。
## 二、行驶中注意事项
1. **磨合期（首保前）**
   - 发动机转速尽量不超过 3000 rpm。
   - 避免长时间匀速行驶，适当变换车速有助于磨合。
   - 避免急加速、急刹车，减轻发动机和制动系统负荷。
2. **换挡操作**
   - 自动挡车型：挂挡前确保车辆完全停稳，严禁行驶中挂入 P 挡或 R 挡。
   - 手动模式（如有）：根据车速和转速合理换挡，避免低挡高速或高挡低速拖挡行驶。
3. **制动系统**
   - 提前预判路况，避免频繁急刹。
   - 涉水后轻踩刹车数次，利用摩擦热蒸发制动盘上的水分，恢复制动效能。
4. **转向系统**
   - 避免原地打死方向盘，减少转向助力系统负担。
   - 行驶中勿长时间将方向盘打到底。
5. **电子辅助系统**
   - ESP/ESC（车身稳定系统）默认开启，非特殊情况不建议关闭。
   - 上坡辅助/陡坡缓降功能在坡道起步和长下坡时善加利用。
## 三、特殊路况驾驶
| 路况 | 注意事项 |
|------|----------|
| **雨天/湿滑路面** | 降低车速，增大跟车距离；避免猛打方向或急刹；开启近光灯及雾灯。 |
| **积雪/结冰路面** | 使用雪地模式（如有）；轻踩油门与刹车；安装防滑链时注意车速不超过40km/h。 |
| **山路/连续弯道** | 入弯前减速，弯中保持匀速；长下坡利用发动机制动（手动降挡），避免刹车过热。 |
| **涉水路段** | 确认水深不超过排气管高度；低速匀速通过（建议1挡），通过后检查制动效果。 |
## 四、停车与熄火
1. **停车步骤**
   - 踩住刹车 → N挡 →拉手刹 →松脚刹确认车辆不溜 → P挡 →熄火。
2. **坡道停车**
   - P挡 +手刹双重保险；有条件的可将前轮朝向路边路沿石方向。
3. **长时间停放**
   - 若超过一周不使用，建议断开电瓶负极或保持每周启动一次运行15分钟以上给电瓶充电。
## ⚠️重要警示
- ❌严禁酒后驾车、疲劳驾驶。
- ❌行车途中勿操作中控屏或手机（需使用语音控制或靠边停车操作）。
- ❌儿童乘车必须使用儿童安全座椅，勿让儿童坐在副驾驶位。
- ✅建议每5000公里或6个月（以先到为准）进行定期保养。
- ✅随车携带三角警示牌、反光背心及灭火器。
> *以上内容适用于焜驰麒麟车型常规驾驶场景，具体操作请以随车用户手册为准。*

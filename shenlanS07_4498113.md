# 深蓝S07 驾驶注意事项
  
更新时间：2026年07月19日15时25分37秒   

## 一、启动与预热
1. **智能钥匙**：确保智能钥匙在车内，踩下制动踏板，车辆即可上电启动。
2. **低温预热**：在冬季或气温低于0°C时，建议启动后原地等待1-2分钟，让电池和电机系统达到适宜工作温度，避免急加速。
3. **仪表检查**：启动后观察仪表盘是否有故障灯亮起（如动力电池、电机、制动系统等）。
## 二、行驶中注意事项
### 1. 能量回收与单踏板模式
- **能量回收强度**：可根据个人习惯在车机中调节“动能回收”强度。强回收模式下松油门减速明显，适合城市拥堵路况；弱回收模式更接近燃油车滑行感。
- **单踏板模式**：开启后可通过油门踏板的踩放控制加减速，但紧急情况仍需使用制动踏板。建议新手先在空旷路段适应。
### 2. 加速与超车
- **电门响应**：电动车扭矩输出快，深踩电门可获得迅猛加速。超车时请预留足够安全距离。
- **湿滑路面**：雨天或冰雪路面避免急加速，防止车轮打滑（ESP介入会有延迟）。
### 3. 制动系统
- **刹车脚感**：深蓝S07采用线控制动（iBooster），刹车脚感与传统液压刹车略有不同。初次驾驶请轻踩适应。
- **再生制动配合**：能量回收会辅助减速，但不可完全替代机械刹车。长下坡时建议利用强回收模式减少刹车片磨损。
### 4. 智能驾驶辅助（IACC/ACC）
- **使用条件**：仅在路况良好、车道线清晰的高速或城市快速路上使用。
- **保持注意力**：即使开启辅助驾驶，双手仍需轻握方向盘，随时准备接管。
- **局限性**：
    - 无法识别静止障碍物（如事故车、锥桶）。
    - 雨雾天气、隧道出入口光线突变时可能退出或误判。
    - 弯道过急时可能降速不足。
## 三、充电与续航管理
| 场景 | 建议 |
|------|------|
| **日常通勤** | SOC保持在20%-80%之间充电，有助于延长电池寿命 |
| **长途出行前** | 充满至100%，并提前规划沿途快充站 |
| **直流快充** | 电量20%-80%区间充电速度最快；超过80%后功率下降明显 |
| **交流慢充** | 推荐过夜充电，对电池更友好 |
| **极寒天气充电** | 若车辆有电池预热功能，出发前开启可提升充电效率 |
> ⚠️ *避免长期将电量耗尽至0%，否则可能导致动力电池损坏且无法启动。*
## 四、特殊路况与天气
### 🚗雨天/积水
-  涉水深度不得超过轮毂中心线（约25cm），切勿强行通过深水区。
-  雨后及时检查高压插接件是否进水。
### ❄️冰雪路面
-  建议切换至“雪地模式”（如有），降低扭矩输出。
-  保持更大跟车距离，提前预判减速。
### 🌄山路/长下坡
-  利用强动能回收代替频繁踩刹车，防止刹车热衰减。
-  注意观察电量消耗速度（上坡耗电较快）。
## 五、停车与离车
1. **挂P挡拉手刹**：车辆停稳后挂入P挡（电子手刹自动拉起），确认仪表显示驻车状态后再下车。
2. **关闭电源**：下车前按下“电源关闭”按钮或携带钥匙离开自动断电（需在设置中开启）。
3. **锁车确认**：
    -  触摸门把手感应区或按遥控锁键。
    -  确认外后视镜折叠且车窗关闭（可通过长按锁车键一键升窗）。
4. **长时间停放**
    -  若停放超过7天，建议保持SOC在50%-60%。
    -  停放超过30天需连接慢充保养电池。
## 六、其他重要提示
| ⚠️项目 | 🔍说明 |
|---------|--------|
| OTA升级 | OTA升级期间禁止驾驶车辆；升级完成后重启车机再行驶 |
| NFC卡片钥匙 | NFC卡片放在左后视镜感应区解锁；不要与手机NFC混放以免干扰 |
| APP远程控车 | APP可远程开空调/车窗；确保车辆停在有网络信号处 |
| ETC安装位置 | ETC设备安装在前挡风玻璃黑色网格区域右侧为宜 |

# MG 4X 技术参数介绍

| 项目 | 参数 |
|------|------|
| 车型级别 | 纯电动紧凑型掀背车 |
| 生产厂商 | 上汽集团（MG 品牌） |
| 车身结构 | 5门5座两厢车 |
| 驱动方式 | 后置后驱 |

| 项目 | 参数 |
|------|------|

| 最大扭矩 | 250 N·m |
| 最高车速 | 160 km/h |
| 0-100 km/h 加速时间 | 约 7.9 秒 |
| 变速箱 | 固定齿比单速变速箱 |

| 项目 | 参数 |
|------|------|
| 电池类型 | 三元锂电池 / 磷酸铁锂（因版本而异） |
| 电池容量 | 51 kWh（标准版）/ 64 kWh（长续航版） |
| CLTC 综合续航里程 | 约 425 km（标准版）/ 约 520 km（长续航版） |
| 快充时间 | 30%-80% 约 30 分钟（直流快充） |
| 慢充时间 | 约 7-9 小时（交流慢充） |
| 电池冷却方式 | 液冷 |

| 转向系统 | 电动助力转向（EPS） |
| 轮胎规格 | 215/55 R17 或 235/45 R18（因配置而异） |

| 项目 | 参数 |

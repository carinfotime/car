# 梅赛德斯-奔驰 威霆 内饰参数介绍
  
更新时间：2026年07月18日09时16分56秒   

## 一、整体设计风格
威霆的内饰设计遵循梅赛德斯-奔驰家族式的**实用与商务**风格，强调宽敞的空间布局与多功能性。整体以深色系为主（如黑色、深灰色），材质上注重耐用性与易清洁性，满足商务接待及多人出行的需求。
## 二、座椅布局与参数
| 项目 | 参数/说明 |
| :--- | :--- |
| **座椅布局形式** | **7座**：2+2+3 布局（第二排为独立行政座椅）<br>**8座**：2+3+3 布局<br>**9座**：2+2+2+3 布局 |
| **座椅材质** | ARTICO皮革（仿皮）或高级织物（视具体配置而定） |
| **前排座椅** | - 手动调节（部分高配可选电动调节）<br>- 座椅加热功能（部分车型选装） |
| **第二排座椅** | - **7座版**：两个独立航空座椅，带双侧扶手<br>- **8/9座版**：连排座椅<br>- 支持前后滑动及靠背角度调节 |
| **第三排座椅** | - 三人连排座椅<br>- 支持整体折叠/侧挂（部分车型），以扩展后备箱空间 |
## 三、仪表台与中控区域
| 项目 | 参数/说明 |

| **空调控制区** | - THERMOTRONIC双区自动空调控制面板<br>- *高配版*：THERMOTRONIC多区智能空调 |
## 四、舒适性与便利性配置
### （一）空调系统
- **类型**: THERMOTRONIC双区自动空调
- **后舱独立控制**: 
    - *7座版*：第二排顶部出风口 + B柱出风口
    - *8/9座版*：后舱独立空调控制面板及顶部出风口
    - *高配车型*: THERMOTRONIC多区智能空调，可独立调节前/后舱温度及风量
### （二）信息娱乐系统
- MBUX智能人机交互系统 (支持语音控制)
- Apple CarPlay / Android Auto (有线连接)
- USB-C接口 (前排及后排)
- Bluetooth®蓝牙连接
- *选装配置*: Burmester®柏林之声环绕立体声音响系统 (需确认具体版本)
### （三）储物空间
-  前排中央扶手箱 (带储物格)
-  门板储物槽 (可容纳水瓶及文件)
-  副驾驶手套箱
-  车顶眼镜盒
-  后排杯架及储物网兜
### （四）灯光与氛围
-  环境氛围照明系统 (64色) — *部分高配车型配备*
-  阅读灯 (前后排独立控制)
-  迎宾灯

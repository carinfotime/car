# 巴菲特S 驾驶注意事项
  
更新时间：2026年07月12日08时08分02秒   

## 一、启动与预热
1. 启动前检查
   - 确认轮胎气压正常
   - 检查机油、冷却液、制动液液位
   - 确认灯光、信号系统工作正常
   - 调整好座椅、后视镜及方向盘位置
2. 冷车启动
   - 启动后原地怠速预热 **30-60秒**
   - 水温表达到正常工作温度前，避免高转速行驶
   - 切勿长时间原地怠速（超过3分钟）
## 二、行驶中的注意事项
1. 换挡操作
   - 自动挡：挂入D挡后松开手刹再起步
   - 手动挡：离合器踩到底，逐级换挡，转速控制在 **2000-2500rpm** 换挡最佳
   - 倒车时务必等车辆完全停稳后再挂入R挡
2. 速度控制
   - 磨合期（前 **1500公里**）车速不超过 **100km/h**
   - 磨合期内避免长时间匀速行驶，应适当变换车速
   - 切勿猛踩油门或急加速
3. 制动注意事项
   - 提前预判路况，采用 **点刹或线性制动**
   - 新刹车片需磨合 **300-500公里** 才能达到最佳制动效果
   - 涉水后应轻踩刹车数次，去除刹车盘水分
## 三、特殊路况驾驶
| 路况 | 注意事项 |
|------|----------|
| **雨天** | 降低车速，保持安全距离（至少平时2倍），开启近光灯 |
| **雪地/结冰** | 使用雪地模式（如有），轻踩油门刹车，避免急转方向 |
| **山路** | 下坡使用发动机制动（低挡位），避免长时间踩刹车 |
| **高速** | 保持车距，每2小时休息15分钟，避免疲劳驾驶 |
## 四、停车与熄火
1. 停车步骤
   - 踩住刹车 → 挂入N挡 → 拉紧手刹 → 松开脚刹 → 挂入P挡 → 熄火
   - 坡道停车时，前轮朝向路沿（上坡向左打，下坡向右打）
2. 熄火前
   - 关闭空调、音响等用电设备
   - 如果涡轮增压车型，熄火前怠速 **1-2分钟** 再关闭发动机
## 五、定期检查与保养
- **每周**：检查胎压、灯光、雨刮水
- **每月**：检查机油尺、冷却液、制动液
- **每5000公里**：更换机油和机滤
- **每10000公里**：更换空气滤芯、空调滤芯
- **每20000公里**：检查火花塞、刹车片磨损
- **每40000公里**：更换变速箱油、防冻液
## 六、安全提醒
- 驾驶员和所有乘员务必 **系好安全带**
- 行车过程中 **严禁使用手机**
- 车内不要放置易燃易爆物品（打火机、香水等）
- 儿童必须使用 **儿童安全座椅**
- 仪表盘报警灯亮起时，及时靠边停车检查
- 随车携带：三角警示牌、灭火器、反光背心、急救包
## 七、省油驾驶技巧
- 保持经济车速（通常 **60-90km/h**）
- 避免急加速和急刹车
- 减少不必要的怠速
- 定期检查胎压（胎压不足会增加油耗）
- 减轻车载负重，清理后备箱多余物品

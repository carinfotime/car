# 福田火星 PHEV 操作指南
  
更新时间：2026年07月20日10时07分09秒   

## 一、车辆启动与关闭
### 1.1 启动车辆
1.  确保智能钥匙在车内。
2.  踩下制动踏板。
3.  按下“一键启动”按钮，仪表盘点亮，车辆进入“READY”状态（此时发动机可能未启动，属于正常现象）。
4.  挂入 D 挡（前进挡）或 R 挡（倒车挡），松开电子手刹，即可行驶。
### 1.2 关闭车辆
1.  踩下制动踏板，将车辆停稳。
2.  挂入 P 挡（驻车挡）。
3.  按下“一键启动”按钮，车辆断电。
> **注意**：熄火前请确认所有车窗、天窗及用电设备已关闭。

## 二、驾驶模式选择
通过中控台或屏幕上的“驾驶模式”旋钮/按键切换：
| 模式 | 适用场景 | 特点 |
| :--- | :--- | :--- |
| **EV（纯电）** | 城市通勤、短途行驶 | 仅由电机驱动，安静、零油耗。电量不足时自动切换。 |
| **HEV（混动）** | 日常综合路况 | 系统自动优化发动机与电机工作，兼顾动力与油耗。 |
| **运动模式** | 超车、爬坡、需要强劲动力时 | 发动机持续介入，油门响应更灵敏。 |
| **经济模式** | 高速巡航、追求低油耗时 | 降低油门灵敏度，优化能量回收。 |
---
## 三、充电操作
### 3.1 AC慢充（家用/公共交流桩）
1.  车辆熄火并挂入 P挡。
2.  打开充电口盖（位于车身左/右后侧）。
3.  将交流充电枪插入充电口。
4.  充电桩端刷卡或扫码启动充电。
5.  仪表盘显示充电状态及预计充满时间。
6.  充满后自动停止，拔下充电枪并盖好盖子。
### 3.2 DC快充（公共直流快充桩）
1-2步同上。
3.  将直流快充枪插入快充口（通常为9孔大接口）。
4-5步同上。
6.  建议电量充至80%-90%即可结束快充以保护电池。
> **警告**：
> -   雷雨天气请勿进行户外充电。
> -   请使用符合国标的正规充电设备。

## 四、能量回收调节
通过方向盘换挡拨片或中控屏设置调节能量回收强度：
-   **弱回收**：滑行时拖拽感轻微，类似燃油车滑行体验。
-   **标准回收**：松开油门有适度减速感并回收能量。
-   **强回收**：松开油门有明显减速感（可实现单踏板效果），适合市区拥堵路况。

|图标|含义|
|:---|:---|
|绿色 READY灯亮起 = “可以行驶”|
|黄色发动机故障灯亮起 = “建议尽快检修”|
|红色电池报警灯亮起 = “电池系统异常”|
|绿色 EV指示灯亮起 = “当前为纯电行驶”|

## ⚠️六、安全注意事项
1. **涉水**
    -   纯电/混动车型涉水深度不宜超过轮毂中心线。若电池包进水可能导致严重损坏甚至短路风险。
2. **高压部件**
    -   橙色线束及高压部件严禁私自拆卸维修。发生碰撞后请立即切断电源并联系服务站检查高压系统。
3. **紧急断电**
    -   紧急情况下可按下位于驾驶舱内的维修开关或直接断开12V蓄电池负极以切断全车电源。
4. **拖车**
    -   建议使用平板拖车进行运输。若需应急拖行请将驱动轮抬起避免电机损坏。

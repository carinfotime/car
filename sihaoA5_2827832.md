# 思皓A5 操作指南
  
更新时间：2026年07月16日08时44分52秒   

## 一、车辆基本参数与识别
- **车型定位**：紧凑型轿车
- **动力系统**：1.5T涡轮增压发动机 + CVT无级变速箱 / 6速手动变速箱
- **燃油标号**：92号及以上无铅汽油
- **油箱容积**：约55L
- **轮胎规格**：215/50 R17
---
## 二、驾驶操作
### 1. 启动与熄火
**启动：**
1. 智能钥匙随身携带，进入车内。
2. 踩下制动踏板（自动挡）或离合器踏板（手动挡）。
3. 按下一键启动按钮，仪表盘点亮，发动机启动。
4. 松开手刹（电子手刹自动释放或手动按下释放）。
**熄火：**
1. 车辆停稳，挂入P挡（自动挡）或空挡（手动挡）。
2. 拉起电子手刹。
3. 再次按下一键启动按钮，发动机熄火。
### 2. 档位操作（CVT自动挡）
| 档位 | 说明 |
|------|------|
| **P** | 驻车档，停车时使用 |
| **R** | 倒车档 |
| **N** | 空档，短时停车使用 |
| **D** | 前进档，日常行驶 |
| **S** | 运动模式，动力响应更积极 |
| **M** | 手动模式，通过换挡拨片或档杆手动控制档位 |
> **注意**：车辆未完全停稳时，禁止挂入P档或R档。
### 3. 电子手刹与自动驻车
- **电子手刹**：向上拉起按键为拉起手刹，向下按压为释放。
- **Auto Hold（自动驻车）**：按下按钮激活，指示灯点亮。等红灯时车辆自动保持制动，轻踩油门自动解除。
### 4. 方向盘与转向
- 支持上下、前后四向手动调节。
- 调节杆位于方向盘下方，解锁后调整至合适位置，锁紧即可。
- 多功能方向盘左侧控制音量、切歌、蓝牙电话；右侧控制仪表盘信息切换、定速巡航。

## 三、灯光与雨刷
### 灯光控制（方向盘左侧拨杆）
| 操作 | 功能 |
|------|------|
| 旋钮至 **O** | 关闭灯光 |
| 旋钮至 **Auto** | 自动大灯（推荐日常使用） |
| 旋钮至 **位置灯** | 示宽灯 |
| 旋钮至 **近光灯** | 手动近光 |
| 拨杆向前推 | 远光灯常亮 |
| 拨杆向后拉 | 远光灯闪一次（超车示意） |
| 拨杆上下拨动 | 左右转向灯 |
### 雾灯
- 在大灯开启状态下，旋转雾灯开关至前/后雾灯位置。
### 雨刷控制（方向盘右侧拨杆）
| 操作 | 功能 |
|------|------|
| 向上推一档 | 单次刮刷 |
| 向下推一档 | 间歇刮刷（可调节间隔速度） |
| 向下推两档 | 低速连续刮刷 |
| 向下推三档 | 高速连续刮刷 |
| 向驾驶员方向拉 | 前挡风玻璃喷水+刮刷 |
| 拨杆顶端旋钮 | 调节后雨刷（掀背/旅行版适用） |

## 四、空调与多媒体系统
### 空调控制（中控面板）
- **温度调节**：左右旋钮分别控制主副驾温度。
- **风量调节**：旋钮调节风量大小。
- **模式选择**：吹脸、吹脚、吹挡风玻璃等。
- **A/C**：空调压缩机开关，制冷时需开启。
- **内外循环**：城市拥堵或隧道内建议开启内循环；长时间驾驶建议切换外循环保持空气新鲜。
- **后窗除雾**：按下按钮加热后窗去除雾气。
### 多媒体系统（10.4英寸竖屏）
- **主界面**：导航、音乐、车辆设置、应用等。
- **语音控制**：说出"你好，思皓"唤醒语音助手，可控制导航、空调、车窗等。
- **手机互联**：支持百度CarLife、苹果CarPlay（视配置版本而定）。
- **蓝牙连接**：设置 → 蓝牙 → 搜索设备 → 配对连接。

## 五、智能驾驶辅助系统（部分配置）
### 定速巡航
1. 按下方向盘右侧 **CRUISE** 按钮。
2. 达到目标速度后，按下 **SET/-** 键设定当前速度为巡航速度。
3. 通过 **RES/+** 或 **SET/-** 调节速度。
4. 踩刹车或按 **CANCEL** 取消巡航。
### 其他辅助功能
- **车道偏离预警**：车辆偏离车道时发出提醒。
- **主动刹车**：检测到碰撞风险时自动制动（视配置）。
- **胎压监测**：仪表盘可查看实时胎压数据。

## 六、日常养护与注意事项
### 加油
- 油箱盖位于车辆左后侧。
- 车辆解锁状态下，按压油箱盖即可打开。
- 加注92号及以上汽油。
### 轮胎
- 建议胎压：2.3-2.5 bar（夏季适当降低，冬季适当升高）。
- 每月检查一次胎压及轮胎磨损情况。
### 保养周期
- **首保**：购车后3个月或5000公里（以先到为准）。
- **常规保养**：每5000-7500公里或6个月一次。
- 保养项目：更换机油、机滤、空气滤芯、空调滤芯等。
### 常见指示灯说明
| 指示灯 | 含义 | 处理方式 |
|--------|------|----------|
| 红色电池图标 | 充电系统故障 | 尽快检查发电机或电瓶 |
| 黄色发动机图标 | 发动机故障 | 尽快到维修站检查 |
| 红色油壶图标 | 机油压力过低 | 立即停车检查机油液位 |
| 黄色胎压图标 | 胎压异常 | 检查并补充胎压 |
| 绿色P图标 | 电子手刹拉起 | 正常，行驶前释放 |

## 七、应急处理
### 车辆无法启动
1. 确认智能钥匙在车内。
2. 确认档位在P挡或N挡。
3. 检查电瓶是否亏电（仪表盘灯光暗淡）。
4. 尝试按下启动按钮时踩死刹车/离合器。
5. 如仍无法启动，联系救援。
### 爆胎/胎压报警
1. 稳住方向盘，缓慢减速。
2. 靠边停放在安全区域。
3. 开启双闪警示灯，放置三角警示牌。
4. 更换备胎或呼叫道路救援。
### 钥匙电池更换
1. 按下钥匙侧面的按钮，取出机械钥匙。
2. 用机械钥匙撬开钥匙外壳。
3. 更换CR2032纽扣电池，正极朝上。
4. 合上外壳，测试功能。
---
## 八、省油驾驶技巧
- 保持匀速行驶，避免急加速、急刹车。
- 高速行驶时关闭车窗，减少风阻。
- 合理使用空调，避免长时间开启最大风量。
- 定期检查胎压，胎压偏低会增加油耗。
- 减少不必要的负重，清理后备箱多余物品。
- 使用经济模式（如配备）或D档平顺驾驶。

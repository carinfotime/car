# 🚗 Firefly 萤火虫 儿童乘车安全注意事项
  
更新时间：2026年07月18日15时04分47秒   

## ⚠️ 基本安全规则
- **必须使用儿童安全座椅**：12 岁以下或身高不足 150 厘米的儿童，请务必使用符合国家标准的儿童安全座椅。
- **禁止坐在副驾驶**：儿童应始终坐在后排座位，切勿坐在副驾驶位置。
- **系好安全带**：无论路程多短，上车后第一时间系好安全带。
## 🧸 车内行为规范
- **不要乱摸按钮**：不要触碰车窗开关、车门把手、档位杆等操作部件。
- **头和手不要伸出窗外**：严禁将头、手或身体任何部位伸出车窗外。
- **不要在车内打闹**：车辆行驶时保持安静坐好，不站立、不蹦跳。
- **不要乱扔垃圾**：保持车内整洁，垃圾放入车载垃圾桶。
## 🔥 紧急情况应对
- **认识紧急按钮**：告诉孩子双闪灯开关的位置和用途。
- **学会开门逃生**：在家长指导下学习如何从内部打开车门。
- **记住求助电话**：让孩子记住家长电话和报警电话（110/120）。

1. [ ]  确认车辆已完全停稳
2. [ ]  观察后方是否有来车或行人
3. [ ]  从右侧车门下车（靠路边一侧）
4. [ ]  下车后立即离开车辆区域
5. [ ]  不要在停车场奔跑玩耍

# 郑州日产 Z9 技术参数介绍

## 车型概览
郑州日产 Z9 是一款定位中高端的皮卡车型，兼顾了商用载货与乘用舒适性。该车基于全新的平台打造，在外观设计、动力总成、智能配置及安全性能方面均进行了全面升级。
---
## 动力系统
| 项目 | 参数 |
| :--- | :--- |
| **发动机类型** | 2.3T / 2.0T 涡轮增压柴油发动机 |
| **排量** | 2.3L / 2.0L |
| **最大功率** | 约 120 kW - 140 kW（视具体调校版本） |
| **最大扭矩** | 380 N·m - 450 N·m |
| **变速箱** | 6速手动变速箱 / ZF 8速自动变速箱 |
| **驱动形式** | 前置后驱（两驱）/ 分时四驱（四驱） |
> *注：具体动力参数以最终上市官方数据为准。*

## 底盘与悬挂
-   **前悬挂类型：**双叉臂式独立悬挂
-   **后悬挂类型：**钢板弹簧非独立悬挂（注重承载能力）
-   **转向系统：**电动助力转向（EPS）
-   **制动系统：**前通风盘式 / 后盘式或鼓式（视配置）
-   **轮胎规格：**255/65 R17 / LT265/65 R17

-   **离去角：** ≥25°
-   **最小离地间隙：** ≥210 mm

### 🖥️智能座舱
-   12.3英寸全液晶仪表盘
-   12.3英寸或更大尺寸中控触摸屏
-   支持语音控制、在线导航、蓝牙连接
-   车联网系统（4G/5G网络）
-   OTA远程升级功能

-   ESP车身电子稳定系统
-   坡道起步辅助 & HDH陡坡缓降控制
-   360°全景影像 + BSD盲区监测
-   前碰撞预警 & AEB自动紧急制动（部分高配）
-   车道偏离预警 & LKA车道保持辅助（部分高配）

# 祥菱V 动力参数介绍
  
更新时间：2026年07月17日14时09分00秒   

祥菱V系列微卡搭载了福田汽车精心调校的动力系统，旨在平衡装载能力与燃油经济性，满足城市及城际短途物流需求。
## 发动机型号与排量
| 参数项 | 规格 |
| :--- | :--- |
| **发动机型号** | DAM15KR / DAM16KR |
| **排量** | 1.5L / 1.6L |
| **进气形式** | 自然吸气 |
| **气缸数** | 4 |
| **供油方式** | 多点电喷 (MPI) |
| **燃油标号** | 92# 及以上无铅汽油 |
## 动力输出数据
### 1.5L 发动机 (DAM15KR)
- **最大马力**：116 Ps
- **最大功率**：85 kW
- **最大功率转速**：6000 rpm
- **最大扭矩**：150 N·m
- **最大扭矩转速**：4000 - 4500 rpm
### 1.6L 发动机 (DAM16KR)
- **最大马力**：122 Ps
- **最大功率**：90 kW
- **最大功率转速**：6000 rpm
- **最大扭矩**：158 N·m
- **最大扭矩转速**：4000 - 4500 rpm
## 变速箱与驱动
- **变速箱类型**：5挡手动变速箱 (5MT)
- **驱动方式**：中置后驱
## 性能与油耗
| 项目 | 数据 |
| :--- | :--- |
| **最高车速** | 约 110 km/h |
| **综合油耗 (NEDC)** | 约 6.5 - 7.2 L/100km |
| **排放标准** | 国VI b |
## 动力特点总结
1.  **低扭表现**：发动机在低转速区间（2000-3000rpm）即可输出约 80%-90% 的峰值扭矩，起步和低速爬坡能力较强，适合重载起步。
2.  **经济性**：采用多点电喷技术，配合轻量化设计，在保证动力输出的同时，燃油经济性表现良好，有效降低运营成本。
3.  **可靠性**：DAM系列发动机经过市场长期验证，结构简单，维护成本低，故障率较低。
> **适用场景**：城市配送、商超进货、家具家电运输、农产品短途转运等。

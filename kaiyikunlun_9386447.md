| 保养项目 | 周期 |

| 常规保养 | 每 **5000公里** 或 **6个月**（以先到为准） |
| 中期保养 | 每 **20000公里** |
| 大保养 | 每 **40000公里** |

## 二、各阶段详细保养内容
### 🟢 首次保养（3个月 / 5000公里）
- [ ] 更换发动机机油
- [ ] 更换机油滤清器
- [ ] 检查冷却液液位及状态
- [ ] 检查制动液液位
- [ ] 检查转向助力油液位
- [ ] 检查变速箱油液位
- [ ] 检查轮胎气压及磨损情况
- [ ] 检查灯光系统是否正常
- [ ] 检查雨刮器及喷水系统
- [ ] 检查底盘螺栓紧固情况
- [ ] ECU系统检测与复位
> ⚠️ **注意：首保免费，务必按时进行，否则可能影响后续质保权益。**
---
### 🟡 常规小保养（每5000公里 / 6个月）
#### 🔧 A类项目（每次必做）
1. **更换机油** —推荐使用 SAE **5W-30**（全合成或半合成，视发动机型号而定）
2. **更换机油滤清器**
3. **空气滤清器清洁/更换**（视脏污程度，建议每1万公里更换一次）
4. **空调滤芯清洁/更换**（建议每年至少更换一次）
5. **轮胎胎压检测与调整** —标准胎压：前轮 **2.3 bar**，后轮 **2.5 bar**
6. **全车灯光、喇叭、雨刮功能检查**
7. **底盘及悬挂系统目视检查**
8. **电瓶电压检测**
#### 📋 B类项目（按需进行）
- [ ]  四轮换位（建议每1万公里进行一次）
- [ ]  节气门清洗（出现怠速不稳时进行）
- [ ]  喷油嘴清洗
---
### 🟠 A类中期保养（20000公里 / ±2年）
包含所有小保养项目 +：
#### 🔩 C类附加项目
1. ✅  更换空气滤清器滤芯
2. ✅  更换空调滤清器滤芯  
3. ✅  火花塞检查/更换 —推荐寿命：普通火花塞约3万公里，铱金火花塞约6万公里  
4. ✅  制动片厚度检测 —前刹车片极限厚度 ≥3mm，后刹车片 ≥2mm  
5. ✅  制动盘磨损检测  
6. ✅  燃油滤清器更换  
7. ✅  冷却液补充或部分更换  
8. ✅  转向系统及球头间隙检查  
9. ✅  传动皮带外观及张紧度检查  
---
### 🔴 B类大保养（40000公里 / ±4年）
包含所有小保 + A类中期项目 +：
#### ⚙️ D类深度项目
1. ✅  全车制动液更换 — DOT4级别，建议每2年或4万公里彻底更换一次  
2. ✅  冷却液彻底排放与加注 —推荐使用原厂长效防冻液  
3. ✅  变速箱油更换  
|变速箱类型|换油方式|用量参考|
|---|---|---|
|手动变速箱 (MT)|重力换油|约2L|
|自动变速箱 (AT/CVT)|循环机换油更彻底|约6~8L|
4. ✅  转向助力油更换  
5. ✅  正时皮带/正时链条状态检查  
> - *正时链条*：通常免维护，但需定期听诊异响。
> - *正时皮带*：若为皮带结构，建议在6~8万公里强制更换。
6. ✅  发电机、空调压缩机皮带状态评估与调整  
7. ✅  四轮定位参数检测与校正  
---
## 🔄🔟十万公里级大项提醒
当车辆行驶至：
### ⏳80000 ~100000 km
请务必增加以下专项检修：
1️⃣ 🔧   正时套件全套更新 （若为正时皮带发动机）  
2️⃣ 💧   水泵总成 （建议随正时皮带一同更新）  
3️⃣ 🔩   发电机单向轮/轴承润滑评估   
4️⃣ 🛞   前后减震器漏油情况排查   
5️⃣ 🔌   氧传感器性能检测   
6️⃣ 🧪   三元催化器清洗或评估

## 🛡️四、日常自检清单 （每月一次）
1️⃣ 🚗    冷车状态下拔出机油尺，查看刻度应在上下限之间   
2️⃣ 💧    观察冷却液膨胀罐水位在 MIN~MAX之间   
3️⃣ 💦    玻璃水及时加注   
4️⃣ 👀    目测四条轮胎有无鼓包、裂纹、异常偏磨   
5️⃣ 📏    用胎压计测量冷态胎压是否达标   
6️⃣ 🔊    启动后听有无异常金属摩擦声或啸叫声   
---
## ⚠️五、重要注意事项
🔸 *质保期内*：必须在授权服务站进行维保并保留记录。自行在外维修可能导致部分质保失效。  
🔸 *涡轮增压车型*：必须使用全合成机油；熄火前怠速30秒再熄火以保护涡轮。  
🔸 *长期短途行驶*：容易产生积碳和机油乳化风险，适当拉高转速行驶有助于清除积碳。  
🔸 *恶劣工况提前养护*：
> - 🌫️多尘环境 →缩短空滤和空调滤的换洗周期至5000km。
> - 🌧️潮湿多雨 →增加底盘防锈检查和天窗排水孔疏通频率。
> - 🚦频繁启停 →缩短火花塞和刹车片寿命。
🔸 *仪表盘警示灯亮起*：立即靠边停车并联系售后救援。

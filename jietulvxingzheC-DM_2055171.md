## 捷途旅行者 C-DM 动力系统介绍
  
更新时间：2026年07月20日09时50分21秒   

捷途旅行者 C-DM 搭载了奇瑞鲲鹏超能混动 C-DM 系统，这是一套以“3擎3挡9模11速”为核心技术的混合动力系统，兼顾了强劲的动力输出与出色的燃油经济性。
### 核心动力总成
-   **发动机**：1.5T 四缸涡轮增压发动机
    -   **最大功率**：115kW（约156马力）
    -   **最大扭矩**：220N·m
-   **驱动电机**：前置永磁同步电机
    -   **最大功率**：150kW
    -   **最大扭矩**：310N·m
-   **变速箱**：3挡DHT（混合动力专用变速箱）
### 综合性能表现
-   **系统综合功率**：280kW
-   **系统综合扭矩**：610N·m
-   **官方0-100km/h加速时间**：约7.1秒（两驱版）/ 约7.5秒（四驱版）
-   **最高车速**：180km/h
### 纯电续航与能耗
-   **电池类型**：磷酸铁锂电池 / 三元锂电池（视具体配置而定）
-   **电池容量**：
    -   低配版：约19.43kWh
    -   高配版/长续航版：约26.7kWh / 43.24kWh
-   **CLTC纯电续航里程**：
    -   低配版：约100公里
    -   高配版/长续航版：约140公里 / 208公里
-   **WLTC综合油耗（馈电状态）**：
    -   两驱版：约5.0L/100km – 6.0L/100km
    -   四驱版：约6.2L/100km – 6.8L/100km
### 驱动模式与智能管理
车辆支持多种工作模式，通过智能控制系统自动切换或由驾驶者手动选择：
1.  **纯电模式 (EV)** ：适合城市通勤，零油耗、静谧行驶。
2.  **串联模式 (HEV)** ：发动机作为增程器发电，电机驱动车轮，适合中低速行驶。
3.  **并联模式 (HEV)** ：发动机与电机共同驱动车轮，提供最强动力输出。
4.  **直驱模式 (HEV)** ：在高速巡航时，发动机直接驱动车轮，效率最高。
5.  **能量回收模式 (Regen)** ：制动或滑行时回收能量为电池充电。
### “3擎3挡9模11速”技术解析
| 技术点 | 说明 |
| :--- | :--- |
| **3擎驱动** | （1）1.5T发动机、（2）P2电机、（3）P2+P3双电机协同 |
| **3挡变速** | DHT变速箱具备物理3个挡位，使发动机在更宽泛的车速区间内高效工作 |
| **9种模式切换** | EV、串联、并联、直驱、驻车充电、行车充电、单踏板、能量回收等 |
| **11种速比组合** | DHT的3个物理挡位与不同工作模式组合出11种速比 |
这套系统的优势在于能够根据不同的驾驶场景和车速，智能匹配最优的工作模式和挡位组合。例如：
-   *起步/低速* → EV纯电或串联增程模式。
-   *中低速加速* → P2+P3双电机并联发力。
-   *中高速巡航* → DHT挂入合适挡位实现发动机直驱或并联辅助。
-   *急加速超车* → “三擎”同时发力，爆发出610N·m的综合扭矩。
### 总结
捷途旅行者 C-DM的动力系统并非简单的“油改电”，而是基于混动专用平台深度开发的产物。它通过精密的机械结构（3挡DHT）和智能的能量管理策略，实现了“要省油时有省油（长续航低油耗），要动力时有动力（7秒级加速）”的平衡表现。对于一款主打轻越野和长途旅行的方盒子SUV而言，这套C-DM混动系统很好地兼顾了城市通勤的经济性和户外穿越的动力储备需求。

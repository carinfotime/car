## 途岳动力参数介绍
  
更新时间：2026年07月12日16时05分13秒   

上汽大众途岳提供多种动力总成选择，以满足不同消费者的需求。目前主要搭载 **1.5T EVO II 净效发动机** 和 **2.0T 涡轮增压发动机**。
### 1.5T EVO II 净效发动机（300TSI）
这是途岳的主力机型，取代了之前的1.4T发动机，在动力、油耗和环保方面均有提升。
| 参数项 | 数值 |
| :--- | :--- |
| **发动机型号** | EA211 EVO II |
| **排量** | 1.5L (1498 ml) |
| **进气形式** | 涡轮增压 |
| **最大马力** | **160 Ps** (118 kW) |
| **最大功率转速** | 5500 rpm |
| **最大扭矩** | **250 N·m** |
| **最大扭矩转速** | 1750 - 4000 rpm |
| **燃油标号** | 92# 及以上无铅汽油 |
| **供油方式** | 缸内直喷 (350bar) |
| **环保标准** | 国 VI b |
#### 🔑核心技术亮点
-   **VTG可变截面涡轮增压器：** 有效缓解涡轮迟滞，低扭响应更快。
-   **米勒循环：** 通过进气门早关，提高燃油经济性。
-   **APS等离子缸壁涂层：** 降低摩擦损耗，提升耐久性。
-   **连续可变气门升程：** VVT + VVL技术，优化进排气效率。
#### ⚙️传动系统与性能
-   **变速箱：**
    -   前驱版：7速干式双离合变速箱 (DQ200)
    -   四驱版：7速湿式双离合变速箱 (DQ381)
-   **驱动方式：**
    -   前轮驱动 (300TSI)
    -   全时四驱 (300TSI R-Line / ...)
-   **官方0-100km/h加速时间：**
    -   前驱版：约9.7秒
    -   四驱版：约9.3秒
-   **WLTC综合油耗：**
    -   前驱版：约5.93 L/100km
    -   四驱版：约6.29 L/100km

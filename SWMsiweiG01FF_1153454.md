# SWM斯威G01FF 保养指南
  
更新时间：2026年07月11日01时02分03秒   

## 📋 保养周期总览
| 保养项目 | 周期（公里/时间） |
|---------|-----------------|
| 首次保养 | 3000公里或3个月（以先到为准） |
| 常规保养 | 每5000公里或6个月 |
| 中期保养 | 每20000公里或12个月 |
| 大保养 | 每40000公里或24个月 |
---
## 🔧 各阶段保养项目明细
### 1. 首次保养（3000km / 3个月）
- 更换发动机机油
- 更换机油滤清器
- 检查冷却液液位
- 检查制动液液位
- 检查转向助力油液位
- 检查轮胎气压及磨损情况
- 检查全车灯光
- 检查雨刮器及喷水系统
- 检查底盘及紧固件
- 检查电瓶状态
- 电脑检测车辆故障码
> **注意：首次保养为免费保养，请务必在规定里程或时间内完成，逾期视为自动放弃。**
---
### 2. 常规保养（每5000km / 6个月）
- **更换** 发动机机油
- **更换** 机油滤清器
- 检查并补充冷却液
- 检查并补充制动液
- 检查并补充玻璃清洗液
- 检查空气滤清器（视情况清洁或更换）
- 检查空调滤清器（视情况清洁或更换）
- 检查轮胎胎压及磨损
- 检查刹车片厚度
- 检查底盘及悬挂系统
- 检查传动皮带状况
- 检查电瓶电压及接线
- 全车灯光检查
- 路试检查车辆行驶状态
---
### 3. 中期保养（每20000km / 12个月）
在常规保养基础上增加以下项目：
- **更换** 空气滤清器
- **更换** 空调滤清器
- **更换** 燃油滤清器
- 检查火花塞（视情况更换）
- 检查并清洁节气门
- 检查刹车油含水量
- 检查转向系统
- 检查前后减震器
- 检查排气管及三元催化器
- 四轮换位及动平衡
---
### 4. 大保养（每40000km / 24个月）
在中期保养基础上增加以下项目：
- **更换** 火花塞
- **更换** 刹车油（制动液）
- **更换** 变速箱油
- **更换** 防冻液（冷却液）
- **更换** 转向助力油（如适用）
- 清洗节气门
- 清洗喷油嘴
- 检查正时皮带/链条（视车型配置）
- 检查发电机及启动电机
- 检查空调系统制冷效果
- 四轮定位检查
- 更换雨刮片

## 🛢️ 油液规格推荐
| 油液类型 | 推荐规格 |
|---------|---------|
| 发动机机油 | 5W-30 或 5W-40（API SN/SP 或以上级别） |
| 变速箱油（手动） | GL-4 75W-90 |
| 变速箱油（自动） | 专用ATF油（请参照用户手册） |
| 制动液 | DOT4 |
| 冷却液 | 乙二醇型防冻液（-35℃） |
| 转向助力油 | 专用助力油或ATF油 |
| 玻璃清洗液 | 四季通用型 |

## ⚠️ 日常自查项目（建议每月检查）
- [ ] 机油液位（冷车时检查）
- [ ] 冷却液液位
- [ ] 制动液液位
- [ ] 轮胎气压（含备胎）
- [ ] 轮胎外观（鼓包、裂纹、异物）
- [ ] 全车灯光是否正常
- [ ] 雨刮器工作状态
- [ ] 仪表盘故障灯
- [ ] 车身有无异常抖动或异响
---
## 🚨 常见保养警示信号
| 现象 | 可能原因 | 建议处理 |
|-----|---------|---------|
| 发动机故障灯亮 | 传感器/排放/点火问题 | 尽快进店检测 |
| 刹车异响/抖动 | 刹车片磨损/刹车盘变形 | 检查刹车系统 |
| 加速无力/油耗上升 | 火花塞/节气门/喷油嘴积碳 | 清洗或更换 |
| 发动机水温过高 | 冷却液不足/散热故障 | 停车检查，勿继续行驶 |
| 转向沉重/异响 | 助力油不足/转向系统故障 | 检查助力油及转向机构 |
| 变速箱换挡顿挫 | 变速箱油变质/油量不足 | 检查变速箱油 |

## 📌 保养注意事项
1. **保养记录**：每次保养后请保留保养单据，作为二手车交易时的凭证。
2. **机油选择**：建议使用全合成机油，可适当延长保养周期，但建议不超过7500km。
3. **原厂配件**：关键部件（滤清器、火花塞、刹车片等）建议使用原厂或品牌认证配件。
4. **季节性保养**：
   - 入夏前：检查空调系统、冷却液
   - 入冬前：检查防冻液、电瓶、轮胎
5. **保修期内**：建议在4S店或授权服务站保养，避免影响质保权益。
6. **保养提示灯**：车辆仪表盘保养提示灯亮起后，请及时安排保养。

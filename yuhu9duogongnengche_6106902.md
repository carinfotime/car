# 江铃域虎9 多功能车 保养指南
  
更新时间：2026年07月17日08时09分48秒   

> **适用车型：** 江铃域虎9 (柴油/汽油版)
> **建议周期：** 以车辆实际行驶里程或时间（以先到者为准）进行保养。
---
## 一、保养周期总览
| 保养项目 | 建议周期 |
| :--- | :--- |
| **机油 & 机滤** | 每 **5,000 - 7,500公里** 或 **6个月** |
| **空气滤芯** | 每 **10,000公里** （恶劣路况缩短至5,000公里） |
| **空调滤芯** | 每 **10,000公里** （或每年换季时） |
| **柴油粗滤器/细滤器 (柴油版)** | 每 **10,000 - 15,000公里** （视油品质量而定） |
| **燃油滤清器 (汽油版)** | 每 **20,000公里** |
| **变速箱油 (手动/自动)** | 每 **40,000 - 60,000公里** |
| **后桥差速器油** | 每 **40,000 - 60,000公里** （重载越野需缩短） |
| **分动箱油 (四驱版)** | 每 **60,000公里** |
| **刹车油 (制动液)** | 每 **2年** （或检测含水量超标时更换） |
| **防冻液 (冷却液)** | 每 **2年 /40,000公里** （长效型可延长至5年） |

> ⚠️ *注：具体请以随车《用户手册》为准，上述为通用建议。*

## 二、核心部件详细说明
### 🔧 A.发动机机油
-   *推荐粘度：*
    -   *柴油版（如PUMA）：* SAE `5W-30` / `5W-40`（C3级或更高）
    -   *汽油版（如Ecoboost）：* SAE `5W-30`（SN级或更高）
-   *标准容量：* ~6.0L ~7.0L（视具体发动机型号）
-   *检查方法：* 
    1.  停车熄火后等待3分钟。
    2.  拔出机油尺擦净，再插入拔出观察刻度。
    3.  应在上下限之间。
### 🔧 B.变速箱油
-   *手动挡：*
    -   推荐规格：GL-4 `75W-90`
    -   容量：约2.0~2.5L
-   *自动挡（采埃孚8AT）：*
    -   推荐规格：采埃孚 `ZF LifeguardFluid8` / `AW-1`认证
    -   ⚠️ *注意：必须使用专用自动变速箱油，不可混用。*
### 🔧 C.分动箱 &差速器
-   *分动箱油（博格华纳TOD）：*
    -   规格：ATF Dexron III / Mercon V
-   *前后桥差速器油：*
    -   规格：GL-5 `80W-90` （带限滑差速器的车型需添加LSD添加剂）
### 🔧 D.冷却系统
-   *防冻液类型：* OAT有机酸型长效防冻液
-   *颜色区分：* 
    -   原厂通常为绿色或红色。
    -   ⚠️ ***严禁混用不同颜色的防冻液。***

## 📋三、日常自检清单
建议每周出车前快速检查以下项目：
1. [ ] ✅轮胎胎压及外观磨损情况。
2. [ ] ✅全车灯光是否正常。
3. [ ] ✅机油尺刻度是否在正常范围。
4. [ ] ✅刹车踏板行程是否正常、有无异响。
5. [ ] ✅雨刮水壶是否充足。
6. [ ] ✅仪表盘有无故障灯亮起。

## 🛠️四、季节性重点保养
### ☀️夏季前准备：
1.  检查空调制冷效果及冷媒压力。
2.  清洗冷凝器表面杂物与灰尘。
3.  检查冷却风扇运转情况。
### ❄️冬季前准备：
1. ***柴油版必须加注负号柴油***（如0号→负10号→负20号）。
2. ***添加抗凝剂***作为应急辅助措施。
3. ***预热系统***测试预热塞工作状态。
4. ***电瓶***检测电压及启动电流。

## 🚙五、特殊工况加强保养
域虎9常被用于越野、工地等恶劣环境，以下情况需缩短保养间隔：
1️⃣ ***重度越野 /涉水***
> —立即更换前后桥及分动箱润滑油。<br>
> —检查并清洁底盘各通气孔。<br>
> —润滑传动轴十字节与滑动叉。
2️⃣ ***沙尘/泥泞路面***
> —缩短空气滤芯更换周期至5000km。<br>
> —每次出车后清理空滤壳体内部积尘。<br>
> —勤洗散热器和中冷器表面。
3️⃣ ***重载长途***
> —出发前重点检查刹车片厚度与刹车盘温度。<br>
> —适当增加轮胎气压至满载标准值。<br>
> —到达目的地后让涡轮增压器怠速冷却再熄火。

## 📌七、重要提示
1️⃣ ***DPF再生注意事项（柴油国六车型）：***
—避免长期短途低速行驶导致DPF堵塞。<br>
—当仪表提示“DPF正在再生”时，请保持高速行驶直至完成。<br>
—严禁使用非标低灰分机油以外的其他机油！
2️⃣ ***尿素系统维护（SCR系统）：***
—务必加注正规车用尿素溶液。<br>
—切勿让尿素罐见底再补加，以免结晶堵塞喷嘴。
3️⃣ ***四驱系统操作规范：***
—铺装路面严禁使用高速四驱模式行驶！<br>
—切换四驱前确保车辆完全静止且挂入N挡。
4️⃣ ***涡轮增压器养护口诀：“冷车怠速热车，熄火前怠速冷却”。***

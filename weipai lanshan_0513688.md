# 魏牌蓝山 驾驶注意事项
  
更新时间：2026年07月17日17时32分50秒   

## 一、启动与预热
1.  **钥匙感应**：确保智能钥匙在车内有效感应区域，否则车辆无法启动。
2.  **上电流程**：踩下制动踏板，按下一键启动按钮。仪表盘亮起且READY指示灯点亮，表示车辆已进入可行驶状态。
3.  **冬季预热**：冬季气温较低时，建议在启动后原地怠速或使用“电池加热”功能1-2分钟，让动力电池和电机达到适宜工作温度，有助于提升续航和动力性能。
## 二、行驶中操作
### 1. 动力模式选择
-   **纯电优先**：适合城市通勤、电量充足时使用，能耗最低。
-   **智能混动**：适合长途或电量不足时使用，系统自动优化油电分配。
-   **强制保电**：若需保留电量（如前往目的地需用电），可设置目标SOC（建议30%-80%），发动机会主动介入充电。
### 2. 能量回收
-   可通过方向盘拨片或中控屏调节能量回收强度（弱/标准/强）。
-   **强回收**：单踏板感较强，松油门减速明显，适合拥堵路况；**弱回收**：滑行更平顺，适合高速巡航。
### 3. 智能驾驶辅助
-   **NOH（高速领航辅助）**：在高速路段开启后，车辆可自主变道、进出匝道。驾驶员仍需手握方向盘并关注路况。
-   **自适应巡航+车道保持**：设定好速度和跟车距离后，车辆自动跟随前车。注意雨雪天气或标线不清时可能退出。
-   **紧急情况接管**：系统发出接管请求提示音或仪表盘红色警告时，请立即人工接管。
### 4. 底盘与悬挂
-   蓝山配备CDC电磁悬挂（部分配置），可在中控屏调节悬架软硬：
    -   **舒适模式**：滤震更好，适合颠簸路面。
    -   **运动模式**：支撑更强，过弯侧倾小。
-   通过减速带或坑洼路面时建议降低车速至30km/h以下。
## 三、停车与充电
### 1. 停车操作
1.  踩住刹车停稳车辆。
2.  挂入P挡（电子怀挡向上拨动）。
3.  拉起电子手刹（部分车型自动联动）。
4.  按下启动键断电下车。
### 2. 充电注意事项
-   **快充/慢充口区分**：左侧为快充（直流），右侧为慢充（交流）。请勿插错接口。
-   **充电前检查**：确认充电枪及接口干燥、无异物。雷雨天气尽量避免露天充电。
-   **充电锁止**：充电过程中车辆会自动锁枪。解锁需先停止充电并解锁车门。
-   **电池健康度**：
    -   日常建议保持电量20%-90%之间循环使用。
    -   长期停放（超过7天）建议电量保持在50%-60%。
    -   每月至少完成一次满充满放以校准电池SOC。
## 四、特殊场景应对
|场景|操作建议|
|---|---|
|涉水行驶|水深不超过轮毂一半（约30cm），低速匀速通过。纯电模式涉水后检查电池底部有无磕碰。|
|长下坡路段|利用能量回收控制车速减少刹车热衰减；必要时切换至“单踏板模式”或手动降挡利用发动机制动。|
|冰雪路面|开启“雪地模式”（ESP介入更积极）；起步轻踩油门避免打滑；增大跟车距离至平时的2倍以上。|
|拖挂房车/拖车|蓝山具备拖拽资质但需确认出厂是否选装拖车钩；拖拽状态下续航下降约30%-50%，注意提前规划补能点。|
## 五、其他关键提醒
1.  **OTA升级期间禁止驾驶**
    系统升级过程中切勿启动车辆行驶。
2.  **儿童安全**
    后排娱乐屏使用时请确保儿童保持安全坐姿；开启“儿童锁”防止误开车门。
3.  **轮胎维护**
    胎压标准值通常为2.5 bar左右；每行驶10000公里建议进行轮胎换位以保证均匀磨损。
4.  **紧急断电**
    如遇碰撞等紧急情况且车门无法打开可使用车内机械拉手开门；若发生火灾请立即远离车辆并拨打119说明是新能源车火灾。

## 一、车型概览
埃尚（AiShang）是由中国汽车制造商推出的全新智能电动轿跑车型，定位中高端市场，融合前沿电动技术与智能驾驶系统。
| 项目 | 参数 |
|------|------|
| 车型级别 | 中型智能电动轿跑 |
| 车身结构 | 5门5座溜背式轿跑 |
| 平台架构 | 全新纯电智能平台（E-Platform 3.0） |
| 驱动方式 | 后驱 / 四驱 |

## 五、智能驾驶系统

|------|------|

| 高清摄像头 | 8 颗 |
| 驾驶辅助芯片 | NVIDIA Orin-X（算力 254 TOPS） |
### 5.2 智能驾驶功能
- **L2+ 级辅助驾驶**：全速域自适应巡航（ACC）、车道居中保持（LCC）
- **自动变道辅助（ALC）**
- **自动泊车**：支持垂直/侧方/斜向泊入及泊出
- **遥控泊车**：手机App远程控制
- **记忆泊车**：支持最长 100 米路径记忆
- **高速领航辅助（选装）**：支持高速路段自动进出匝道、超车变道
- **城市领航辅助（选装）**：支持城市道路红绿灯识别、路口通行、避让行人等

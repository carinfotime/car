# 九龙C6 驾驶注意事项
  
更新时间：2026年07月11日18时31分53秒   

## 一、启动前检查
1. **油液检查**
   - 检查发动机机油液位，确保在上下刻度线之间
   - 检查冷却液液位，不足时及时补充
   - 检查制动液液位，低于下限需立即添加
   - 检查挡风玻璃清洗液
2. **轮胎检查**
   - 检查轮胎气压（建议冷态气压：前轮4.5-5.0bar，后轮5.0-5.5bar）
   - 检查轮胎磨损情况，胎面花纹深度不得低于1.6mm
   - 检查轮胎表面有无鼓包、裂纹或异物扎入
3. **灯光与信号**
   - 测试前大灯（远近光）、转向灯、刹车灯、倒车灯、雾灯是否正常
   - 检查危险报警闪光灯功能
4. **车身与玻璃**
   - 清理前后挡风玻璃及后视镜上的污物
   - 确保雨刮器工作正常，刮片无老化破损
## 二、行驶中注意事项
1. **起步与换挡**
   - 冷车启动后怠速预热1-2分钟，待水温表指针开始上升后再起步
   - 手动挡车型：离合器踩到底，挂挡轻柔，避免打齿
   - 自动挡车型：挂D挡后松开手刹，缓慢加油起步
   - **严禁**空挡滑行（手动挡），易导致制动失效风险
2. **车速控制**
   - C6车身较高大，重心偏高，转弯时提前减速，时速不超过30km/h
   - 高速行驶建议控制在90-100km/h之间，经济省油且安全
   - **严禁**超速行驶（最高限速120km/h）
3. **制动技巧**
   - 预见性驾驶，提前松油门利用发动机制动辅助减速
   - 避免急刹车（尤其是满载时），防止车辆侧滑或货物移位
   - 长下坡路段使用低档位辅助制动，防止刹车过热失效
4. **并线与超车**
   - C6盲区较大（尤其是右侧），变道前务必观察后视镜并转头确认盲区
   - 超车时预留充足加速距离，确认安全后再变道超车
5. **特殊路况**
   |路况|注意事项|
      |---|---|
      |雨天|降低车速至60km/h以下；保持与前车距离至少50米；开启近光灯和雾灯|
      |雪天/结冰|安装防滑链；车速不超过30km/h；轻踩刹车避免侧滑|
      |山路|使用低档位爬坡/下坡；弯道鸣笛示意；避免长时间半联动行驶|
## 三、停车与熄火
1. **停车操作**
   1)选择平坦坚实地面停车  
   2)拉紧手刹至完全锁止  
   3)手动挡挂入一档或倒档（上坡挂一档、下坡挂倒档）  
   4)自动挡挂P档  
   5)关闭所有用电设备（空调、灯光等）  
   6)怠速运转30秒后再熄火（尤其涡轮增压车型）
2. **夏季高温停车**
   避免长时间暴晒；车内勿放置打火机、香水等易燃物品；熄火前先关闭空调AC开关保持送风2分钟干燥蒸发器。
3. **冬季低温停车**
   若环境温度低于-10℃，建议使用冬季专用机油；长期停放应断开蓄电池负极。
## 四、日常维护要点
|项目|周期/标准|
|---|---|
|机油更换|每5000公里或半年（以先到为准）|
|空气滤清器清洁/更换|每10000公里清洁，每20000公里更换|
|燃油滤清器更换|每20000公里|
|变速箱油更换|手动挡每40000公里 /自动挡每60000公里|
|刹车片厚度检查|每次保养时检测 <3mm需立即更换|
## ⚠️特别警示⚠️
- C6为大型客车类车型（长度约7米），**C1驾照可驾驶**但需注意车身长度和转弯半径差异。
- **严禁超载**：额定载客人数不得超过行驶证核定人数。
- **严禁客货混装**：行李舱以外区域不得堆放货物。
- ABS工作时刹车踏板会有弹脚感属正常现象。
- EPS故障指示灯亮起时应立即靠边停车检修。

# 欧拉好猫 保养指南
  
更新时间：2026年07月13日09时58分47秒   

## 一、保养周期总览
欧拉好猫作为一款纯电动车型，保养项目相比燃油车简单很多，主要以检查、清洁和更换易损件为主。
| 项目 | 周期 | 备注 |
| :--- | :--- | :--- |
| **常规保养（小保）** | **每12个月或10,000公里**（以先到为准） | 主要进行三电系统检查、底盘检查、更换空调滤芯等 |
| **大保养** | **每24个月或20,000公里**（以先到为准） | 包含小保项目 + 更换减速器油、制动液等 |
| **制动液** | **每24个月或20,000公里** | - |
| **减速器油（齿轮油）** | **每24个月或20,000公里** | - |
| **空调滤芯** | **每12个月或10,000公里** | 视空气质量可缩短至6个月更换 |
| **冷却液（防冻液）** | **每48个月或40,000公里检查/更换** | - |
| **轮胎换位/动平衡** | **每10,000-15,000公里** | - |
> *注：以上为厂家建议标准周期，若车辆使用环境恶劣（如多尘、高温、频繁短途行驶），建议适当缩短保养间隔。*
---
## 二、核心保养项目详解
### 1. 三电系统检查（每次必查）
-   **电池组：** 检查电池外观有无磕碰、漏液；连接线束是否松动；通过OBD接口读取电池健康度（SOH）、单体电压均衡性。
-   **电机：** 检查电机运转有无异响，冷却系统是否正常。
-   **电控系统：** 读取故障码，升级BMS（电池管理系统）固件。
### 2. 空调滤芯更换
-   位置：副驾驶手套箱后方。
-   建议：每年至少更换一次。如果经常开空调有霉味，需立即更换。
-   类型：推荐使用带活性炭的PM2.5滤芯。
### 3. 减速器油更换
-   频率：2年/2万公里。
-   作用：润滑减速齿轮和差速器。
-   用量：约1升左右（具体请参考说明书）。
### 4. 制动系统
-   **制动液：** DOT4级别，含水量超过3%必须立即更换。
-   **刹车片/盘：** EV车型因动能回收机制，刹车片磨损通常比燃油车慢。每次保养需检查厚度。
### 5.冷却液
-   用于给电机和电控系统散热。
-   颜色通常为粉色或蓝色。不同品牌不可混加。
---
## 三、日常自查项目
车主在日常用车中可自行完成的简单检查：
1.  **轮胎气压与磨损**
    -   标准胎压通常为2.5 bar左右（主驾车门内侧有标签）。
    -   观察胎面磨损标记，注意是否有鼓包或偏磨。
2.  **玻璃水**
    -   冬季需使用防冻型玻璃水。
3.  **灯光与雨刮**
    -   定期检查大灯、转向灯、刹车灯是否正常。
    -   雨刮片刮不干净或有异响时及时更换。
4.  充电口及充电枪保持清洁干燥，避免异物堵塞。
---
## 四、省钱与避坑建议
1.  首保通常是免费的，务必按时去4S店完成首保记录，否则可能影响质保。
2.  电动车没有发动机机油和机滤，不要被忽悠做“发动机清洗”之类的无用项目。
3.  空调滤芯可以自己买来换，操作简单且价格远低于4S店工时费。
4.  如果车辆出现续航明显下降、充电异常等情况，即使没到保养周期也应尽快进店检测BMS状态。
---
## ⚠️重要提示
以上内容基于通用标准整理。不同年款和配置的欧拉好猫可能存在细微差异。请务必以您随车的《用户手册》和《保修保养手册》中的官方规定为准。

# 奔驰EQB 保养指南
  
更新时间：2026年07月15日09时29分14秒   

## 一、车型简介
奔驰EQB为纯电动SUV车型，与传统燃油车不同，其保养项目主要集中在三电系统（电池、电机、电控）、底盘、空调系统及车身部件上，无需更换机油、机滤等传统项目。

| 保养项目 | 建议周期 |
|---------|---------|
| 常规检查（小保养） | 每 **1年** 或 **20,000公里**（以先到为准） |
| 全面检查（大保养） | 每 **2年** 或 **40,000公里**（以先到为准） |
| 空调滤清器更换 | 每 **1年** 或 **20,000公里** |
| 制动液更换 | 每 **2年** |
| 减速器油更换 | 每 **4年** 或 **80,000公里** |
| 防冻液更换 | 每 **10年** 或 **250,000公里** |
| 高压电池检查 | 每 **1年** |

## 三、详细保养项目
### 1. 每年 / 20,000公里（小保养）
- 读取车辆故障码，检查电控系统
- 检查高压电池状态及冷却系统
- 检查充电接口及充电线束
- 检查电机及减速器有无异响、渗漏
- 检查制动系统（刹车片厚度、刹车盘磨损）
- 检查轮胎磨损及胎压（含备胎/补胎工具）
- 检查灯光、喇叭、雨刮器
- 检查12V低压蓄电池状态
- 检查底盘悬挂及转向系统
- 检查空调系统功能及制冷效果
- **更换空调滤清器**
- 复位保养提示
### 2. 每2年 / 40,000公里（大保养）
包含小保养全部项目，额外增加：
- **更换制动液**
- 检查并清洁制动系统
- 检查驱动轴及防尘套
- 检查车身密封条及排水孔
- 检查高压线束绝缘状况
- 检查并润滑门锁、铰链、天窗导轨
- 更换雨刮片（视磨损情况）
### 3. 每4年 / 80,000公里
- **更换减速器油**
- 检查并视情更换轮胎
- 更换空调滤清器（如未按时更换）
- 全面检查高压系统绝缘性能
### 4. 每10年 / 250,000公里
- **更换电机冷却液（防冻液）**

## 四、关键注意事项
### 高压电池保养（最重要）
| 建议 | 说明 |
|-----|------|
| 保持电量在20%-80% | 日常使用建议浅充浅放，延长电池寿命 |
| 避免频繁快充 | 优先使用家用慢充，长途再使用快充 |
| 长期停放 | 电量保持在 **50%左右**，每3个月充电一次 |
| 避免高温暴晒 | 尽量停在阴凉或地下车库 |
| 冬季预热 | 充电前可开启电池预热功能，提升充电效率 |
### 轮胎保养
- EQB自重较大，轮胎磨损相对较快，建议每10,000公里前后轮交叉换位
- 胎压建议：前轮 **2.5 bar**，后轮 **2.8 bar**（具体以驾驶侧车门标签为准）
- 冬季建议适当调高0.1-0.2 bar
### 制动系统
- 纯电车有动能回收，机械刹车使用频率较低，但仍需定期检查
- 建议每2年更换制动液，防止吸湿后制动性能下降
- 定期进行急刹车操作，清除刹车盘表面氧化层
### 空调系统
- 空调滤清器建议每年更换一次，尤其雾霾严重地区可缩短至6个月
- 每2年检查空调系统制冷剂压力

1. **质保期内建议在4S店保养**，避免影响三电系统质保
2. 每次保养后保留好保养记录和发票

4. 如仪表盘出现“保养到期”提示，请及时预约保养

## 七、常见问题（FAQ）
**Q：EQB需要换机油吗？**
A：不需要。EQB是纯电动车，没有发动机，无需更换机油和机滤。
**Q：可以不去4S店保养吗？**
A：质保期内建议去4S店。如果选择第三方，需确保使用原厂配件并保留完整记录，否则可能影响三电系统质保。
**Q：电池需要保养吗？**
A：高压电池无需定期更换，但建议每年检查电池状态，日常注意充电习惯即可延长寿命。
**Q：保养一次大概多久？**
A：小保养约1-1.5小时，大保养约2-3小时。

# 锐骐汽车儿童乘车安全注意事项
  
更新时间：2026年07月18日09时56分39秒   

为了确保您和孩子在驾驶或乘坐锐骐汽车时的安全，请务必遵守以下注意事项：
## 一、儿童安全座椅使用规范
- **强制使用**：12周岁以下或身高低于1.4米的儿童乘车时，必须使用符合国家标准的儿童安全座椅。
- **反向安装**：建议0-3岁（尤其是9个月以下）的婴幼儿使用反向安装的儿童安全座椅，以更好地保护头颈。
- **后排安装**：儿童安全座椅应安装在后排座位上，切勿安装在副驾驶位置。
- **固定检查**：每次使用前请确认安全座椅与车辆ISOFIX接口或安全带连接牢固，无松动。
## 二、乘车位置与行为
- **严禁坐副驾**：儿童禁止坐在副驾驶位置。如遇紧急情况，副驾驶气囊弹出可能对儿童造成严重伤害。
- **全程系安全带**：对于已不使用安全座椅的大龄儿童，必须全程正确佩戴后排安全带。
- **禁止探头伸手**：严禁将头、手或身体任何部位伸出车窗外、天窗外。
- **勿在车内打闹**：车辆行驶过程中，禁止孩子在车内站立、跳跃或来回走动。
## 三、车门与车窗安全
- **开启儿童锁**：锐骐后门通常配备儿童安全锁。载有儿童时，请务必启用该功能，防止行车中孩子误开车门。
- **车窗控制**：建议开启主驾位的车窗锁止功能（Window Lock），防止孩子自行升降车窗导致夹伤。
- **停车开门注意**：停车后应由成人先下车观察后方来车及行人，再为孩子开门。
## 四、停车与上下车
- **切勿留孩子在车内**：无论时间长短，绝对不要将孩子单独留在车内。夏季车内温度会迅速升高至致命水平。
- **远离后备箱**：教育孩子不要在后备箱内玩耍或躲藏。锐骐作为皮卡车型，其后斗（货箱）严禁搭载任何乘客（包括儿童）。
- **上下车引导**：
    - 教导孩子从右侧（靠路侧）上下车。
    - 上下车时注意脚下踏板及地面湿滑情况。
## 五、其他重要提醒
| 项目 | 说明 |
|------|------|
| **车内物品收纳** | 尖锐物品、小零件等应放入手套箱或储物格内固定好。急刹车时散落物品可能砸伤孩子。 |
| **避免零食饮料** | 行车中尽量避免给孩子吃棒棒糖、果冻等易噎食的食物；热饮可能因颠簸烫伤。 |
| **空调温度调节** | 注意后排空调出风口风向及温度设置，避免冷风直吹孩子面部导致不适。 |
| **驾驶员专注驾驶** | 如有哭闹等情况，应由同行的其他成人安抚处理；驾驶员应保持专注不分心。 |
> ⚠️ *以上内容适用于所有锐骐系列车型（包括经典款及新款），具体配置细节请参考随车用户手册。*

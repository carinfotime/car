# 日产轩逸动力参数介绍
  
更新时间：2026年07月17日17时28分03秒   

日产轩逸作为一款主打家用舒适性的紧凑型轿车，其动力系统调校也以平顺、节油为核心。目前市售的轩逸主要提供 **1.6L 自然吸气** 发动机，同时经典款车型也保留了更老旧的动力总成。以下是具体的动力参数介绍。
## 1. 主力动力：HR16 型 1.6L 自然吸气发动机
这是目前轩逸家族（第14代轩逸及轩逸·经典）最核心的动力单元。
-   **发动机型号**：HR16
-   **排量**：1.6L（1598ml）
-   **进气形式**：自然吸气
-   **气缸数**：4缸
-   **供油方式**：多点电喷（MPI）
-   **最大马力**：
    -   第14代轩逸：**135 匹**（99kW）
    -   轩逸·经典：**122 匹**（90kW）
-   **最大功率转速**：6300 rpm
-   **最大扭矩**：
    -   第14代轩逸：**159 N·m**
    -   轩逸·经典：**155 N·m**
-   **最大扭矩转速**：4000 rpm
> **特点**：该发动机技术成熟，可靠性高。虽然动力参数在同级中不占优势，但其主要优势在于**低扭表现平顺**，适合城市拥堵路况，且燃油经济性极佳。
## 2. 传动系统
轩逸全系未提供手动变速箱选项，全部采用自动变速箱。
-   **变速箱类型**：**CVT（无级变速箱）**
    -   具体型号：XTRONIC CVT（模拟8速）
-   **驾驶感受**：CVT变速箱与1.6L发动机匹配非常默契，换挡毫无顿挫感，加速过程线性流畅。但急加速时，发动机转速会迅速拉高，噪音较大，且缺乏传统AT或双离合的降档冲击感。
## 3. 动力性能表现
-   **官方0-100km/h加速时间**：约 **12.1秒**（第14代轩逸）
-   **最高车速**：约 180 km/h - 198 km/h（视具体配置及负载）
-   **WLTC综合油耗**：
    -   第14代轩逸：约 **5.57 - 5.94 L/100km**
    -   轩逸·经典：约 **6.0 L/100km** 左右
> **评价**：轩逸的动力表现可以用“够用”来形容。它并不追求推背感或运动性，而是将重点放在了**平顺、安静、省油**上。对于日常通勤、接送家人、城市代步来说，这套动力系统完全胜任。
## 4. 轩逸·经典（老款）动力参数
上一代轩逸（轩逸·经典）仍在使用较老的HR16DE发动机，参数略低于新款。
-   **发动机**：HR16DE
-   **最大马力**：122 匹
-   **最大扭矩**：155 N·m
-   **变速箱**：CVT（无级变速）
-   **油耗**：NEDC综合油耗约 5.2L/100km (实际综合约6.5-7L)
## 5. 总结
| 项目 | 第14代轩逸 | 轩逸·经典 |
| :--- | :--- | :--- |
| **发动机** | HR16 (1.6L) | HR16DE (1.6L) |
| **最大马力** | 135 匹 | 122 匹 |
| **最大扭矩** | 159 N·m | 155 N·m |
| **变速箱** | CVT (模拟8速) | CVT |
| **驱动方式** | 前置前驱 | 前置前驱 |
| **综合油耗** | 约 5.9 L/100km | 约 6.0 L/100km |
| **动力取向** | 平顺、省油、舒适 | 平顺、省油、可靠 |
**一句话总结**：轩逸的动力系统不追求激情，但追求极致的平顺与燃油经济性，是典型的“买菜车”定位。如果您对动力有较高要求，建议考虑其他涡轮增压车型。

# 问界M5 儿童乘车安全注意事项
  
更新时间：2026年07月20日16时24分05秒   

为了保障您孩子的乘车安全，请务必遵守以下事项：
## 一、安全座椅使用规范
1.  **强制使用**：12周岁以下或身高低于145厘米的儿童，必须使用合适的儿童安全座椅，严禁直接使用成人安全带。
2.  **反向安装**：建议4岁及以下儿童（尤其是体重低于18kg）使用反向安装的儿童安全座椅，以更好地保护头颈。
3.  **ISOFIX接口**：问界M5后排座椅配备ISOFIX儿童座椅固定装置及顶部锚点。安装时请确保接口锁死，并拉动座椅确认无松动。
4.  **禁止前座**：**严禁**将反向安装的儿童安全座椅放置在前排副驾驶位。若因特殊情况需在前排使用正向座椅，必须将副驾驶座椅调至最后位置，并手动关闭该位置的正面安全气囊。
## 二、车窗与车门安全
1.  **车窗防夹**：问界M5四门车窗均具备防夹功能。但请勿让孩子将头、手或身体其他部位伸出窗外，尤其注意防止孩子误触车窗开关。
2.  **后排车窗锁**：建议开启“后排车窗锁”功能（主驾门板控制），防止儿童在后排自行升降车窗。
3.  **儿童安全锁**：务必启用后门的“儿童安全锁”（机械式拨杆）。启用后，车门只能从外部打开，防止车辆行驶中或停车时孩子误开车门。
## 三、车内环境与物品
1.  **勿留单独车内**：无论天气冷暖或时间长短，切勿将孩子单独留在车内。车内温度会迅速升高或降低，极易导致窒息或中暑/失温。
2.  **小物件收纳**：收好车内的小物件（如硬币、瓶盖、充电线等），避免孩子吞食或玩耍时卡住手脚。
3.  **前排调整**：若孩子坐在后排中间位置（如适用），请确保前排中央扶手箱和杯架已妥善收纳/关闭。
## 四、智能功能相关提醒
1.  **语音与屏幕控制**：
    -   教育孩子不要随意触碰中控大屏和方向盘上的按键。
    -   警惕孩子通过语音助手误操作车辆功能（如打开天窗、调节空调温度等）。
2.  **空调与通风**：
    -   避免空调出风口直吹儿童面部或腹部。
    -   若开启“宠物模式”或“露营模式”，仍需有成人陪同看护儿童。
3.  **自动泊车/召唤功能**：
    -   **严禁**在车辆自动泊车或遥控移动过程中让儿童站在车辆周围或行进路线上。
## ⚠️ 紧急情况处理
-   **意外锁车怎么办？**
    如果车钥匙被误留在车内且车门上锁：
    1.  优先使用备用钥匙或手机蓝牙/NFC钥匙解锁。
    2.  拨打问界道路救援热线寻求帮助。
    3.  紧急情况下可破窗（选择远离儿童的侧窗）。
-   **安全带缠绕怎么办？**
    立即停止拉动安全带。缓慢回缩安全带释放压力；若无法解开可用剪刀剪断安全带（注意避开孩子身体）。
## ✅ 每日出发前检查清单
-   [ ] ISOFIX接口是否牢固？
-   [ ] 后门儿童安全锁是否开启？
-   [ ] “后排车窗锁”是否激活？
-   [ ] 所有松散物品是否已固定/收纳？
-   [ ] GPS定位/手机远程控制功能是否正常？（便于查看车内状况）
> 💡 *温馨提示：最好的保护是成人的全程监护。行车途中请保持注意力集中，切勿因照顾分心而影响驾驶安全。*

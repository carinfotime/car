# 奥迪Q6 保养指南

本指南旨在帮助奥迪Q6车主了解车辆的常规保养项目、周期及注意事项。请结合您的车辆用户手册和实际驾驶情况（如行驶环境、驾驶习惯）进行参考。

## 一、核心保养周期总览
以下为基于正常行驶条件下的建议保养周期。若经常在恶劣环境（如多尘、高温、短途频繁启停）下行驶，建议适当缩短保养间隔。
| 项目 | 建议周期 | 备注 |
| :--- | :--- | :--- |
| **机油与机滤** | **每10,000公里或1年** | 以先到者为准。使用符合VW标准的高品质全合成机油。 |
| **空气滤芯** | **每20,000公里或2年** | 多尘环境建议缩短至10,000-15,000公里。 |
| **空调滤芯** | **每10,000公里或1年** | 建议每年换季前检查/更换，尤其夏季使用空调前。 |
| **火花塞** | **每20,000-40,000公里** | 具体视发动机型号而定（涡轮增压发动机通常为20,000公里）。 |
| **制动液** | **每2年** | 需检测含水量，超过标准必须更换。 |
| **防冻液/冷却液** | **每3-5年或60,000公里** | 检查液位和冰点，必要时添加或更换。 |
| **变速箱油** | **每60,000-80,000公里** (DSG双离合) / **免维护或按需检查** (AT自动) | DSG变速箱强烈建议定期更换油液及滤芯。具体请参照用户手册。 |
| **刹车片/刹车盘** | **视磨损情况而定，约30,000-60,000公里** | 每次保养时检查厚度，仪表盘有磨损报警提示。 |
| **轮胎换位与动平衡** | **每10,000-15,000公里** | 延长轮胎寿命，保持行驶稳定性。四轮定位按需进行。 |
| **电瓶状态检查** | **每次常规保养时检查** (约3-5年寿命) | AGM电瓶需专用设备检测健康度。 |

## 二、详细保养项目说明
### A. 小保养 (约10,000公里 / 1年)
*   ✅ 更换发动机机油及机油滤清器
*   ✅ 检查并补充各种油液（制动液、冷却液、玻璃水等）
*   ✅ 检查轮胎气压及磨损情况
*   ✅ 检查底盘及悬挂系统有无松动或漏油
*   ✅ 检查灯光系统、雨刮器功能
*   ✅ OBD车载诊断系统检测，清除故障码
### B. 大保养 (约20,000-40,000公里 / 2-4年)
在小保养基础上增加：
*   ✅ **更换空气滤清器**
*   ✅ **更换空调滤清器**
*   ✅ （首次大保）更换火花塞（涡轮增压车型）
*   ✅ （首次大保）清洗节气门（视积碳情况）
*   ✅ （首次大保）更换燃油滤清器（部分车型集成于油箱内）
*   ✅ （首次大保）深度检查皮带张紧度及老化情况
### C. Major Service / Long Life Service (约60,000-80,000公里)
在大保养基础上增加：
*   🔧 （必做）更换变速箱油及滤网/垫片（DSG车型）
*   🔧 （必做）更换制动液
*   🔧 （必做）全面检测冷却系统，必要时更换防冻液
*   🔧 （必做）四轮定位校准
*   🔧 （必做）清洗喷油嘴和进气歧管积碳（直喷发动机常见问题）
*   🔧 （视情）更换发电机皮带/正时皮带组件（如有）

## 三、关键部件养护要点
### 🛢️ [机油选择]
> *必须使用符合大众VW502.00 / VW504.00 / VW508.00认证标准的全合成机油*
> *推荐粘度：0W-20 (最新国六B发动机) / 5W-30 /5W-40*
> *容量：约5.0 -5.5升*
### ⚡ [电瓶养护]
> *奥迪Q6通常配备AGM蓄电池*
> *避免熄火后长时间使用音响/空调*
> *冬季短途行驶注意充电不足问题*
> *长期停放建议断开负极*
### 🛞 [轮胎养护]
> *推荐胎压：前2.3 -2.5 bar /后2.4 -2.6 bar (具体见油箱盖内侧标签)*
> *定期清理胎纹中的石子异物*
> *注意观察偏磨现象*
### ❄️ [空调系统]
> *每年至少进行一次蒸发箱清洗杀菌*
> *制冷效果下降时及时补充冷媒*

## ⚠️ [重要提醒]
1. **[质保期内]** ：务必在授权经销商处按计划完成所有规定项目并盖章记录。
2. **[质保期外]** ：可选择专业连锁维修店。
3. **[仪表盘提示]** ：当出现“立即进行车辆保养”或扳手图标闪烁时请尽快预约。
4. **[配件选择]** ：尽量选用原厂配件或国际一线品牌配套件。
5. **[驾驶习惯]** ：冷车启动后等待转速回落再起步；避免长期低转速高档位行驶；定期拉高转速清除积碳。

## 📅 [简易自查清单]
您可以每月自行快速完成以下项目：
1️⃣ [ ] 【机油尺】拔出擦净再插入查看刻度是否在上下限之间  
2️⃣ [ ] 【冷却液】膨胀罐内液体是否在MAX-MIN之间  
3️⃣ [ ] 【玻璃水】是否充足  
4️⃣ [ ] 【轮胎外观】有无鼓包、裂纹  
5️⃣ [ ] 【灯光】全部点亮确认无损坏  
6️⃣ [ ] 【雨刮片】刮拭效果是否清晰

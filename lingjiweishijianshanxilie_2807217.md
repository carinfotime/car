# 凌际卫士见山系列 操作指南
  
更新时间：2026年07月18日13时08分48秒   

## 一、车辆基本操作
### 1.1 启动与熄火
- **一键启动**：携带智能钥匙进入车内，踩下制动踏板，按下“START/STOP ENGINE”按钮启动发动机。
- **熄火**：车辆停稳后，挂入P挡，再次按下“START/STOP ENGINE”按钮即可熄火。
- **紧急熄火**：长按启动按钮或连续快速按两次可强制熄火。
### 1.2 智能钥匙功能
| 按键 | 功能 |
|------|------|
| 解锁键 | 解锁全车车门，同时解除防盗 |
| 锁车键 | 锁止全车车门，同时激活防盗 |
| 尾门键 | 长按开启/关闭电动尾门 |
| 远程启动键 | 先按锁车键，再长按远程启动键可远程启动发动机 |
> **提示**：钥匙感应距离约为1.5米，超出范围可能无法正常操作。
## 二、驾驶模式选择
凌际卫士见山系列提供多种驾驶模式，通过中控台旋钮或触摸屏进行切换：
| 模式 | 适用场景 | 特点 |
|------|----------|------|
| **标准模式** | 日常城市道路 | 均衡动力与油耗，适合日常通勤 |
| **运动模式** | 高速超车、山路驾驶 | 动力响应更积极，换挡转速提高 |
| **经济模式** | 长途巡航、节油驾驶 | 降低油门灵敏度，优化燃油经济性 |
| **越野模式** | 非铺装路面、泥地、沙地 | 四驱系统介入，牵引力控制优化 |
| **雪地模式** | 雨雪湿滑路面 | 二挡起步，降低打滑风险 |
> **操作方式**：旋转中控台模式选择旋钮，仪表盘会同步显示当前模式。
## 三、四驱系统操作
### 3.1 四驱模式切换
- **自动模式**（默认）：系统根据路况自动分配前后轮扭矩。
- **高速四驱（4H）**：用于轻度越野或湿滑路面，可在行驶中切换。
- **低速四驱（4L）**：用于重度越野、攀爬或脱困，需在N挡且车速低于5km/h时切换。
### 3.2 差速锁操作（如配备）
- **中央差速锁**：按下中控台“中央差速锁”按键，指示灯亮起表示锁止。
- **后桥差速锁**：在中央差速锁锁止后，按下“后桥差速锁”按键。
- **前桥差速锁**：在前后差速锁均锁止后，按下“前桥差速锁”按键（仅限极限越野场景）。
> **注意**：差速锁仅用于越野脱困，铺装路面严禁使用，以免损坏传动系统。
## 四、底盘与悬挂系统
### 4.1 空气悬挂调节
- **高度调节**：通过中控台“底盘升降”旋钮调节车身高度。
  - **越野模式**：升高底盘，增加通过性（车速超过40km/h自动降低）
  - **装载模式**：降低底盘，方便装卸行李
  - **迎宾模式**：熄火后自动降低，方便上下车
- **软硬调节**：根据驾驶模式自动适配悬挂硬度，也可在设置菜单中手动调节。
### 4.2 涉水操作
- 最大涉水深度：约900mm（视具体配置而定）
- 涉水前建议：
  1. 切换至越野模式
  2. 关闭自动启停功能
  3. 降低车窗以便应急
  4. 缓慢匀速通过，避免激起浪花
## 五、智能座舱与娱乐系统
### 5.1 中控屏操作
- **主页**：点击屏幕上方“主页”图标返回主界面
- **空调控制**：屏幕底部常驻空调控制栏，点击可展开详细设置
- **车辆设置**：点击“车辆”图标进入车辆功能设置菜单
- **导航**：支持在线导航、实时路况、语音控制目的地
### 5.2 语音控制
唤醒方式：说出“你好，凌际”或按下方向盘语音按键
支持指令示例：
- “导航到最近的加油站”
- “播放音乐”
- “打开座椅加热”
- “空调调到24度”
- “打开天窗”
### 5.3 后排娱乐系统（如配备）
- 前排中控屏可控制后排屏幕开关与内容推送
- 后排屏幕支持独立播放、蓝牙耳机连接
- 支持手机投屏（USB-C或无线连接）
## 六、舒适与便利功能
### 6.1 座椅调节
| 功能 | 操作方式 |
|------|----------|
| 前后/高低/靠背 | 门板座椅调节按键 |
| 腰托调节 | 门板四向腰托按键 |
| 座椅通风/加热 | 中控屏空调界面或实体按键 |
| 座椅按摩 | 中控屏座椅设置菜单 |
| 座椅记忆 | 调节好位置后，按“SET”+数字键保存 |
### 6.2 天窗与遮阳帘
- **全景天窗**：通过顶棚控制面板开关或语音控制
- **遮阳帘**：电动开合，可与天窗联动或单独控制
### 6.3 电动踏板
- 车门打开时自动伸出，关闭后自动收回
- 可通过中控屏“车辆设置”中关闭自动伸缩功能
- 冬季如遇冰冻，建议先清理冰雪再使用
## 七、安全辅助系统
### 7.1 主动安全功能
- **自适应巡航（ACC）**：方向盘左侧按键开启，设定车速和跟车距离
- **车道保持辅助**：中控屏开启，车速超过60km/h自动激活
- **自动紧急制动**：默认开启，检测到碰撞风险时自动刹车
- **盲区监测**：后视镜指示灯亮起提示盲区有车辆
### 7.2 驻车辅助
- **360°全景影像**：挂入R挡自动显示，也可手动开启
- **前后雷达**：接近障碍物时蜂鸣提示，距离越近提示越急促
- **自动泊车**：中控屏选择“自动泊车”，按提示操作即可
### 7.3 越野辅助
- **透明底盘**：中控屏显示车底路面影像，辅助越野通过
- **坡度显示**：仪表盘显示当前坡度与侧倾角度
- **车轮轨迹线**：全景影像中显示前轮行驶轨迹
## 八、保养与维护
### 8.1 保养周期建议
| 项目 | 周期 |
|------|------|
| 发动机机油与机滤 | 每10,000公里或12个月 |
| 空气滤清器 | 每20,000公里 |
| 空调滤清器 | 每10,000公里 |
| 制动液 | 每40,000公里或2年 |
| 变速箱油 | 每60,000公里 |
| 分动箱/差速器油 | 每60,000公里 |
| 火花塞 | 每40,000公里 |
### 8.2 轮胎维护
- 标准胎压：前轮2.4bar / 后轮2.6bar（具体以驾驶位门框标签为准）
- 建议每月检查一次胎压
- 轮胎换位：每10,000公里进行前后交叉换位
### 8.3 仪表盘警示灯
| 图标 | 含义 | 处理方式 |
|------|------|----------|
| 红色发动机灯 | 发动机故障 | 尽快停车检查，联系售后 |
| 黄色发动机灯 | 排放系统故障 | 可继续行驶，尽快检修 |
| 红色电池灯 | 充电系统故障 | 立即停车，联系救援 |
| 黄色胎压灯 | 胎压异常 | 检查并补充胎压 |
| 红色刹车灯 | 制动系统故障 | 立即停车，联系救援 |
| 黄色四驱灯 | 四驱系统故障 | 避免越野，尽快检修 |
## 九、紧急情况处理
### 9.1 车辆无法启动
1. 确认智能钥匙在车内
2. 检查是否在P挡或N挡
3. 检查蓄电池是否亏电（仪表盘是否点亮）
4. 尝试将钥匙贴近启动按钮再启动
5. 如仍无法启动，联系道路救援
### 9.2 轮胎漏气/爆胎
- **配备备胎**：使用随车千斤顶和工具更换备胎
- **无备胎（补胎液）**：使用随车补胎液，按说明注入轮胎，然后低速行驶至最近维修点
- **缺气保用轮胎**：可继续以不超过80km/h的速度行驶约80公里
### 9.3 紧急救援呼叫
- 按下车内顶棚的“SOS”按钮，直接连接凌际救援中心
- 车辆会自动发送当前位置信息
## 十、使用注意事项
1. **磨合期**：新车前3,000公里避免急加速、急刹车，发动机转速不超过3,000转
2. **燃油要求**：推荐使用95号及以上无铅汽油
3. **洗车注意**：避免高压水枪直接冲洗发动机舱和电子元件
4. **车顶行李架**：最大承重约75kg，请均匀分布载荷
5. **拖车**：四驱车辆建议使用平板拖车，避免单轮着地拖行
6. **长期停放**：若超过30天不使用，建议断开蓄电池负极，并将胎压充至标准值以上

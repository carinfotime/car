# 长安之星5 技术参数介绍
  
更新时间：2026年07月12日08时02分19秒   

长安之星5是长安汽车旗下的一款经典微面车型，以其皮实耐用、经济实惠和空间灵活著称，广泛应用于物流配送、客运及个体经营等领域。
## 基本参数
| 项目 | 参数 |
| :--- | :--- |
| **车型级别** | 微型面包车（微面） |
| **生产厂商** | 长安汽车 |
| **车身结构** | 5门客车 / 5门货车（盲窗版） |
| **座位数** | 5座 / 7座 |
## 动力系统
### 发动机
- **发动机型号**：JL473Q (或同系列)
- **发动机类型**：1.4L自然吸气
- **气缸数**：4缸
- **供油方式**：多点电喷 (MPI)
- **最大马力**：约101 Ps
- **最大功率**：约74 kW
- **最大扭矩**：约135 N·m
- **燃油标号**：92#汽油
- **排放标准**：国VI b
### 变速箱
- **变速箱类型**：5挡手动变速箱 (MT)
